# <a name="compiler-error-cs0150"></a><span data-ttu-id="e830d-102">Derleyici hatası CS0150</span><span class="sxs-lookup"><span data-stu-id="e830d-102">Compiler Error CS0150</span></span>
<span data-ttu-id="e830d-103">Sabit değer bekleniyor</span><span class="sxs-lookup"><span data-stu-id="e830d-103">A constant value is expected</span></span>  
  
 <span data-ttu-id="e830d-104">Bir sabit beklenirken bir değişken bulundu.</span><span class="sxs-lookup"><span data-stu-id="e830d-104">A variable was found where a constant was expected.</span></span> <span data-ttu-id="e830d-105">Daha fazla bilgi için bkz. [anahtar](../language-reference/keywords/switch.md).</span><span class="sxs-lookup"><span data-stu-id="e830d-105">For more information, see [switch](../language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="e830d-106">Aşağıdaki örnek CS0150 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="e830d-106">The following sample generates CS0150:</span></span>  
  
```csharp  
// CS0150.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         int j = 0;  
  
         switch(i)  
         {  
            case j:   // CS0150, j is a variable int, not a constant int  
            // try the following line instead  
            // case 1:  
         }  
      }  
   }  
}  
```  
  
 <span data-ttu-id="e830d-107">Bu hata, bir değişken değeri ile bir dizi boyutu belirtildiğinde ve dizi başlatıcısıyla başlatıldığında üretilir.</span><span class="sxs-lookup"><span data-stu-id="e830d-107">This error is also produced when an array size is specified with a variable value and initialized with an array initializer.</span></span> <span data-ttu-id="e830d-108">Hatayı kaldırmak için, diziyi ayrı bir deyimde veya deyimlerde başlatın.</span><span class="sxs-lookup"><span data-stu-id="e830d-108">To remove the error, initialize the array in a separate statement or statements.</span></span>  
  
```csharp  
// CS0150.cs  
    namespace MyNamespace  
    {  
        public class MyClass  
        {  
            public static void Main()  
            {  
                int size = 2;  
                double[] nums = new double[size] { 46.9, 89.4 }; //CS0150  
                // Try the following lines instead  
                // double[] nums = new double[size];  
                // nums[0] = 46.9;   
                // nums[1] = 89.4;  
            }  
        }  
  
    }  
```
