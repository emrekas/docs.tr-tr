# <a name="compiler-error-cs0150"></a><span data-ttu-id="3778b-102">Derleyici Hatası CS0150</span><span class="sxs-lookup"><span data-stu-id="3778b-102">Compiler Error CS0150</span></span>
<span data-ttu-id="3778b-103">Sabit değer bekleniyor</span><span class="sxs-lookup"><span data-stu-id="3778b-103">A constant value is expected</span></span>  
  
 <span data-ttu-id="3778b-104">Bir değişken bir sabit beklenirken bulunamadı.</span><span class="sxs-lookup"><span data-stu-id="3778b-104">A variable was found where a constant was expected.</span></span> <span data-ttu-id="3778b-105">Daha fazla bilgi için [geçiş](../../csharp/language-reference/keywords/switch.md).</span><span class="sxs-lookup"><span data-stu-id="3778b-105">For more information, see [switch](../../csharp/language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="3778b-106">Aşağıdaki örnek, CS0150 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="3778b-106">The following sample generates CS0150:</span></span>  
  
```csharp  
// CS0150.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         int j = 0;  
  
         switch(i)  
         {  
            case j:   // CS0150, j is a variable int, not a constant int  
            // try the following line instead  
            // case 1:  
         }  
      }  
   }  
}  
```  
  
 <span data-ttu-id="3778b-107">Bu hata, bir dizi boyutu bir değişken değerle belirtilirse ve bir dizi Başlatıcısı ile başlatılmış da oluşturulur.</span><span class="sxs-lookup"><span data-stu-id="3778b-107">This error is also produced when an array size is specified with a variable value and initialized with an array initializer.</span></span> <span data-ttu-id="3778b-108">Hata kaldırmak için ayrı bir deyimi veya deyimler dizi başlatır.</span><span class="sxs-lookup"><span data-stu-id="3778b-108">To remove the error, initialize the array in a separate statement or statements.</span></span>  
  
```csharp  
// CS0150.cs  
    namespace MyNamespace  
    {  
        public class MyClass  
        {  
            public static void Main()  
            {  
                int size = 2;  
                double[] nums = new double[size] { 46.9, 89.4 }; //CS0150  
                // Try the following lines instead  
                // double[] nums = new double[size];  
                // nums[0] = 46.9;   
                // nums[1] = 89.4;  
            }  
        }  
  
    }  
```
