# <a name="compiler-error-cs1529"></a><span data-ttu-id="5356b-102">Derleyici Hatası CS1529</span><span class="sxs-lookup"><span data-stu-id="5356b-102">Compiler Error CS1529</span></span>
<span data-ttu-id="5356b-103">Bir using yan tümcesi extern diğer ad bildirimleri dışında ad uzayında tanımlanan diğer tüm öğelerden önce gelmelidir</span><span class="sxs-lookup"><span data-stu-id="5356b-103">A using clause must precede all other elements defined in the namespace except extern alias declarations</span></span>  
  
 <span data-ttu-id="5356b-104">A [kullanarak](../../csharp/language-reference/keywords/using.md) yan tümcesi bir ad alanındaki ilk olarak görünmelidir.</span><span class="sxs-lookup"><span data-stu-id="5356b-104">A [using](../../csharp/language-reference/keywords/using.md) clause must appear first in a namespace.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5356b-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="5356b-105">Example</span></span>  
 <span data-ttu-id="5356b-106">Aşağıdaki örnek, CS1529 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="5356b-106">The following sample generates CS1529:</span></span>  
  
```csharp  
// CS1529.cs  
namespace X  
{  
    namespace Subspace  
    {  
        using Microsoft;  
  
        class SomeClass  
        {  
        };  
  
        using Microsoft;      // CS1529, place before class definition  
    }  
  
    using System.Reflection;  // CS1529, place before namespace 'Subspace'  
}  
  
using System;                 // CS1529, place at the beginning of the file  
```
