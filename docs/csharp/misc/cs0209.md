<span data-ttu-id="51b3c-103">Bir fixed deyiminde bildirilen yerel türünde bir işaretçi türü olmalıdır</span><span class="sxs-lookup"><span data-stu-id="51b3c-103">The type of local declared in a fixed statement must be a pointer type</span></span>  
  
 <span data-ttu-id="51b3c-104">Olarak bildirdiğiniz değişkeni bir [fixed statement](../../csharp/language-reference/keywords/fixed-statement.md) bir işaretçi olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="51b3c-104">The variable that you declare in a [fixed statement](../../csharp/language-reference/keywords/fixed-statement.md) must be a pointer.</span></span> <span data-ttu-id="51b3c-105">Daha fazla bilgi için [güvenli olmayan kod ve işaretçiler](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="51b3c-105">For more information, see [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="51b3c-106">Aşağıdaki örnek, CS0209 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="51b3c-106">The following sample generates CS0209:</span></span>  
  
```csharp  
// CS0209.cs  
// compile with: /unsafe  
  
class Point  
{  
   public int x, y;  
}  
  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
  
      fixed (int i)    // CS0209  
      {  
      }  
      // try the following lines instead  
      /*  
      fixed (int* p = &pt.x)  
      {  
      }  
      fixed (int* q = &pt.y)  
      {  
      }  
      */  
   }  
}  
```
