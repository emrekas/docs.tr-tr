# <a name="compiler-error-cs0174"></a>Derleyici Hatası CS0174
Bir 'base' başvurusu için bir temel sınıf gereklidir  
  
 Bu hata, yalnızca .NET Framework ortak dil çalışma zamanı için kaynak kodu, derleme oluşur `System.Object` temel sınıfa sahip yalnızca bir sınıf olan sınıf.
