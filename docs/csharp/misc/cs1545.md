# <a name="compiler-error-cs1545"></a><span data-ttu-id="9fb4b-102">Derleyici Hatası CS1545</span><span class="sxs-lookup"><span data-stu-id="9fb4b-102">Compiler Error CS1545</span></span>
<span data-ttu-id="9fb4b-103">'Property' özelliği, dizin oluşturucusu veya olayı dil tarafından desteklenmiyor; erişimci yöntemlerini 'set erişimcisi' veya '''get erişimcisine doğrudan çağırmayı deneyin</span><span class="sxs-lookup"><span data-stu-id="9fb4b-103">Property, indexer, or event 'property' is not supported by the language; try directly calling accessor methods 'set accessor' or 'get accessor'</span></span>  
  
 <span data-ttu-id="9fb4b-104">Varsayılan olmayan bir sahip bir nesne kodu tüketiyor [dizin oluşturucu](../../csharp/programming-guide/indexers/index.md) ve dizinli söz dizimini kullanacak şekilde çalıştı.</span><span class="sxs-lookup"><span data-stu-id="9fb4b-104">The code is consuming an object that has a non-default [indexer](../../csharp/programming-guide/indexers/index.md) and tried to use the indexed syntax.</span></span> <span data-ttu-id="9fb4b-105">Bu hatayı gidermek için özelliğin çağrı `get` veya `set` erişimci yöntemi.</span><span class="sxs-lookup"><span data-stu-id="9fb4b-105">To resolve this error, call the property's `get` or `set` accessor method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="9fb4b-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="9fb4b-106">Example</span></span>  
  
```cpp  
// CPP1545.cpp  
// compile with: /clr /LD  
// a Visual C++ program  
using namespace System;  
public ref struct Employee {  
   Employee( String^ s, int d ) {}  
  
   property String^ name {  
      String^ get() {  
         return nullptr;  
      }  
   }  
};  
  
public ref struct Manager {  
   property Employee^ Report [String^] {  
      Employee^ get(String^ s) {  
         return nullptr;  
      }  
  
      void set(String^ s, Employee^ e) {}  
   }  
};  
```  
  
## <a name="example"></a><span data-ttu-id="9fb4b-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="9fb4b-107">Example</span></span>  
 <span data-ttu-id="9fb4b-108">Aşağıdaki örnek, CS1545 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="9fb4b-108">The following sample generates CS1545.</span></span>  
  
```csharp  
// CS1545.cs

class x {  
   public static void Main() {  
      Manager Ed = new Manager();  
      Employee Bob = new Employee("Bob Smith", 12);  
      Ed.Report[ Bob.name ] = Bob;   // CS1545  
      Ed.set_Report( Bob.name, Bob);   // OK  
   }  
}  
```
