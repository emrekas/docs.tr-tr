# <a name="compiler-warning-level-2-cs0440"></a>Derleyici Uyarısı (Düzey 2) CS0440
'Global' adlı bir diğer ad tanımlanması önerilmez uzayına ' global::' her zaman genel ad ve bir diğer başvuruyor  
  
 Bir diğer ad adlandırılmış tanımlarken bu uyarı genel.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS0440 oluşturur:  
  
```csharp  
// CS0440.cs  
// Compile with: /W:1  
  
using global = MyClass;   // CS0440  
class MyClass  
{  
    static void Main()  
    {  
        // Note how global refers to the global namespace  
        // even though it is redefined above.  
        global::System.Console.WriteLine();  
    }  
}  
```
