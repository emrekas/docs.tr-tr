# <a name="compiler-error-cs1662"></a><span data-ttu-id="674a1-102">Derleyici Hatası CS1662</span><span class="sxs-lookup"><span data-stu-id="674a1-102">Compiler Error CS1662</span></span>

<span data-ttu-id="674a1-103">Anonim yöntem bloğu bloktaki dönüş türlerinden bazıları örtük olarak temsilci dönüş türüne dönüştürülebilir olmadığından, temsilci türü 'type' temsilciye dönüştürülemiyor</span><span class="sxs-lookup"><span data-stu-id="674a1-103">Cannot convert anonymous method block to delegate type 'delegate type' because some of the return types in the block are not implicitly convertible to the delegate return type</span></span>

<span data-ttu-id="674a1-104">Anonim yöntem bloğu'nın dönüş ifadesi örtük olarak temsilci dönüş türüne dönüştürülebilir değil bir türe sahipse, bu hata oluşur.</span><span class="sxs-lookup"><span data-stu-id="674a1-104">This error occurs if the anonymous method block's return statement had a type that was not implicitly convertible to the return type of the delegate.</span></span>

<span data-ttu-id="674a1-105">Aşağıdaki örnek, CS1662 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="674a1-105">The following sample generates CS1662:</span></span>

```csharp
// CS1662.cs

delegate int MyDelegate(int i);

class C
{

  public static void Main()
  {
     MyDelegate d = delegate(int i) { return 1.0; };  // CS1662
     // Try this instead:
     // MyDelegate d = delegate(int i) { return (int)1.0; };
  }
}
```
