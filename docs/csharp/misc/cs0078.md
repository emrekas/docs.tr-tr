# <a name="compiler-warning-level-4-cs0078"></a><span data-ttu-id="a16a2-102">Derleyici Uyarısı (düzey 4) CS0078</span><span class="sxs-lookup"><span data-stu-id="a16a2-102">Compiler Warning (level 4) CS0078</span></span>
<span data-ttu-id="a16a2-103">'L' soneki '1' basamağı ile kolaylıkla karıştırılır--daha anlaşılır olması için 'L' kullanın</span><span class="sxs-lookup"><span data-stu-id="a16a2-103">The 'l' suffix is easily confused with the digit '1' -- use 'L' for clarity</span></span>  
  
 <span data-ttu-id="a16a2-104">Derleyici, bir dönüştürme algıladığında sizi uyarır uzun bir büyük harf l yerine bir küçük harf l kullanma</span><span class="sxs-lookup"><span data-stu-id="a16a2-104">The compiler warns when it detects a cast to long using a lowercase l instead of an uppercase L.</span></span>  
  
 <span data-ttu-id="a16a2-105">Aşağıdaki örnek, CS0078 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="a16a2-105">The following sample generates CS0078:</span></span>  
  
```csharp  
// CS0078.cs  
// compile with: /W:4  
class test {  
   public static void TestL(long i)  
   {  
   }  
  
   public static void TestL(int i)  
   {  
   }  
  
   public static void Main()  
   {  
      TestL(25l);   // CS0078  
      // try the following line instead  
      // TestL(25L);  
   }  
}  
```
