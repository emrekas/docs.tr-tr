# <a name="compiler-error-cs0315"></a><span data-ttu-id="7d919-102">Derleyici hatası CS0315</span><span class="sxs-lookup"><span data-stu-id="7d919-102">Compiler Error CS0315</span></span>
<span data-ttu-id="7d919-103">' ValueType ' türü, ' typeormethod\<T > ' genel türünde veya yönteminde ' 't ' tür parametresi olarak kullanılamaz.</span><span class="sxs-lookup"><span data-stu-id="7d919-103">The type 'valueType' cannot be used as type parameter 'T' in the generic type or method 'TypeorMethod\<T>'.</span></span> <span data-ttu-id="7d919-104">' ValueType ' öğesinden ' referenceType ' öğesine paketleme dönüştürmesi yok.</span><span class="sxs-lookup"><span data-stu-id="7d919-104">There is no boxing conversion from 'valueType' to 'referenceType'.</span></span>  
  
 <span data-ttu-id="7d919-105">Bu hata, genel bir türü belirli bir sınıfa kısıtlamak ve kendisine örtülü olarak kutulanmamış bir değer türü kullanarak bu sınıfın bir örneğini oluşturmaya çalıştığınızda oluşur.</span><span class="sxs-lookup"><span data-stu-id="7d919-105">This error occurs when you constrain a generic type to a particular class, and try to construct an instance of that class by using a value type that cannot be implicitly boxed to it.</span></span>

## <a name="example"></a><span data-ttu-id="7d919-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="7d919-108">Example</span></span>  
 <span data-ttu-id="7d919-109">Aşağıdaki örnek CS0315 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="7d919-109">The following example generates CS0315:</span></span>  
  
```csharp  
// cs0315.cs  
public class ClassConstraint { }  
public struct ViolateClassConstraint { }  
  
public class Gen<T> where T : ClassConstraint  
{         
}  
public class Test  
{  
    public static int Main()  
    {  
        Gen<ViolateClassConstraint> g = new Gen<ViolateClassConstraint>(); //CS0315  
        return 1;  
    }  
}  
```
