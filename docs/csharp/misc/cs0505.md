# <a name="compiler-error-cs0505"></a>Derleyici Hatası CS0505
'Üye1': 'üye2' bir işlev olmadığından geçersiz kılınamaz  
  
 Bir yöntem olmayan taban sınıfında geçersiz kılmak bir sınıf bildiriminde bulunuldu. Geçersiz kılmalar üye türü eşleşmelidir. Aynı ada sahip bir yöntem temel sınıfta bir yöntem gerekli olarak kullanırsanız [yeni](../../csharp/language-reference/keywords/new-modifier.md) (ve [geçersiz kılma](../../csharp/language-reference/keywords/override.md)) yöntem bildiriminde temel sınıf.  
  
 Aşağıdaki örnek, CS0505 oluşturur:  
  
```csharp  
// CS0505.cs  
// compile with: /target:library  
public class clx  
{  
   public int i;  
}  
  
public class cly : clx  
{  
   public override int i() { return 0; }   // CS0505  
}  
```
