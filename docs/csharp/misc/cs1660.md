# <a name="compiler-error-cs1660"></a><span data-ttu-id="f3a1d-102">Derleyici Hatası CS1660</span><span class="sxs-lookup"><span data-stu-id="f3a1d-102">Compiler Error CS1660</span></span>
<span data-ttu-id="f3a1d-103">Anonim yöntem bloğu'bir temsilci türü olmadığından, 'type' türüne dönüştürülemiyor</span><span class="sxs-lookup"><span data-stu-id="f3a1d-103">Cannot convert anonymous method block to type 'type' because it is not a delegate type</span></span>  
  
 <span data-ttu-id="f3a1d-104">Atayın veya aksi halde bir anonim yöntem bloğu bir temsilci türü olmayan bir türe dönüştürmeye çalışırsanız, bu hata oluşur.</span><span class="sxs-lookup"><span data-stu-id="f3a1d-104">This error occurs if you try to assign or otherwise convert an anonymous method block to a type which is not a delegate type.</span></span>  
  
 <span data-ttu-id="f3a1d-105">Aşağıdaki örnek, CS1660 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="f3a1d-105">The following sample generates CS1660:</span></span>  
  
```csharp  
// CS1660.cs  
delegate int MyDelegate();  
class C {  
   static void Main()  
   {  
     int i = delegate { return 1; };  // CS1660  
     // Try this instead:  
     // MyDelegate myDelegate = delegate { return 1; };  
     // int i = myDelegate();  
   }  
}  
```
