# <a name="compiler-error-cs0568"></a>Derleyici hatası CS0568
Yapılar, açık parametresiz oluşturucular içeremez  
  
 Her [yapının](../language-reference/keywords/struct.md) zaten nesneyi sıfıra Başlatan parametresiz bir Oluşturucusu vardır. Bu nedenle, bir yapı için oluşturabileceğiniz oluşturucuların bir veya daha fazla parametre olması gerekir.  
  
 Aşağıdaki örnek CS0568 oluşturur:  
  
```csharp  
// CS0568.cs  
public struct ClassY  
{  
   public int field1;  
   public ClassY(){}   // CS0568, cannot have no param constructor  
   // Try following instead:  
   // public ClassY(int i)  
   // {  
   //    field1 = i;  
   // }  
}  
  
public class ClassX  
{  
   public static void Main()  
   {  
   }  
}  
```
