# <a name="compiler-error-cs0568"></a><span data-ttu-id="0b8e6-102">Derleyici hatası CS0568</span><span class="sxs-lookup"><span data-stu-id="0b8e6-102">Compiler Error CS0568</span></span>

 <span data-ttu-id="0b8e6-104">Her [yapının](../language-reference/keywords/struct.md) zaten nesneyi sıfıra Başlatan parametresiz bir Oluşturucusu vardır.</span><span class="sxs-lookup"><span data-stu-id="0b8e6-104">Each [struct](../language-reference/keywords/struct.md) already has a parameterless constructor that initializes the object to zero.</span></span> <span data-ttu-id="0b8e6-105">Bu nedenle, bir yapı için oluşturabileceğiniz oluşturucuların bir veya daha fazla parametre olması gerekir.</span><span class="sxs-lookup"><span data-stu-id="0b8e6-105">Therefore, the constructors that you can create for a struct must take one or more parameters.</span></span>  
  
 <span data-ttu-id="0b8e6-106">Aşağıdaki örnek CS0568 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="0b8e6-106">The following sample generates CS0568:</span></span>

```csharp  
   // {  
   //    field1 = i;  
   // }  
}

{  
   public static void Main()  
   {  
   }  
}  
```
