# <a name="compiler-error-cs0568"></a><span data-ttu-id="1b6fe-102">Derleyici Hatası CS0568</span><span class="sxs-lookup"><span data-stu-id="1b6fe-102">Compiler Error CS0568</span></span>
<span data-ttu-id="1b6fe-103">Yapı birimleri parametresi olmayan açık oluşturucular içeremez</span><span class="sxs-lookup"><span data-stu-id="1b6fe-103">Structs cannot contain explicit parameterless constructors</span></span>  
  
 <span data-ttu-id="1b6fe-104">Her [yapı](../../csharp/language-reference/keywords/struct.md) sıfıra nesnesini başlatır ve parametresiz bir oluşturucu zaten sahip.</span><span class="sxs-lookup"><span data-stu-id="1b6fe-104">Each [struct](../../csharp/language-reference/keywords/struct.md) already has a parameterless constructor that initializes the object to zero.</span></span> <span data-ttu-id="1b6fe-105">Bu nedenle, bir yapı için oluşturabileceğiniz oluşturucular bir veya daha fazla parametre almalıdır.</span><span class="sxs-lookup"><span data-stu-id="1b6fe-105">Therefore, the constructors that you can create for a struct must take one or more parameters.</span></span>  
  
 <span data-ttu-id="1b6fe-106">Aşağıdaki örnek, CS0568 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="1b6fe-106">The following sample generates CS0568:</span></span>  
  
```csharp  
// CS0568.cs  
public struct ClassY  
{  
   public int field1;  
   public ClassY(){}   // CS0568, cannot have no param constructor  
   // Try following instead:  
   // public ClassY(int i)  
   // {  
   //    field1 = i;  
   // }  
}  
  
public class ClassX  
{  
   public static void Main()  
   {  
   }  
}  
```
