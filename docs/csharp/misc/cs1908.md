# <a name="compiler-error-cs1908"></a>Derleyici Hatası CS1908
DefaultValue özniteliği bağımsız değişkenin türü parametre türüyle eşleşmelidir  
  
 Bu hata, yanlış bir bağımsız değişken için kullandığınızda oluşturulur <xref:System.ComponentModel.DefaultValueAttribute> öznitelik değeri. Parametre türü ile eşleşen bir değer kullanın.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS1908 oluşturur.  
  
```csharp  
// CS1908.cs  
// compile with: /target:library  
using System.Runtime.InteropServices;  
  
public interface ISomeInterface  
{  
   void Bad([Optional] [DefaultParameterValue("true")] bool b);   // CS1908  
  
   void Good([Optional] [DefaultParameterValue(true)] bool b);   // OK  
}  
```
