# <a name="compiler-error-cs0121"></a>Derleyici Hatası CS0121

Arama yöntemleri veya özellikleri arasında belirsiz: 'yöntem1' ve 'method2'

Örtük dönüştürme nedeniyle, derleyici bir form aşırı yüklenmiş bir yöntemin çağrı mümkün değildi. Bu hatayı aşağıdaki yollarla çözebilirsiniz:

- Yöntem parametrelerinin örtük dönüştürme yer almayan bir yol belirtin.

- Tüm aşırı yüklemeler yöntemi kaldırın.

- Uygun bir yöntemini çağırmadan önce türüne.

## <a name="examples"></a>Örnekler

Aşağıdaki örnekler, derleyici hatası CS0121 oluştur:

```csharp
public class C
{
   void f(int i, double d)
   {
   }

   void f(double d, int i)
   {
   }

   public static void Main()
   {
      C c = new C();

      c.f(1, 1);   // CS0121

      // Try the following code instead:
      // c.f(1, 1.0);
      // or
      // c.f(1.0, 1);
      // or
      // c.f(1, (double)1);   // Cast and specify which method to call.
   }
}
```

```csharp
class Program2
{
    static int ol_invoked = 0;

    delegate int D1(int x);
    delegate T D1<T>(T x);
    delegate T D1<T, U>(U u);

    static void F(D1 d1) { ol_invoked = 1; }
    static void F<T>(D1<T> d1t) { ol_invoked = 2; }
    static void F<T, U>(D1<T, U> d1t) { ol_invoked = 3; }

    static int Test001()
    {
        F(delegate(int x) { return 1; }); // CS0121
        if (ol_invoked == 1)
            return 0;
        else
            return 1;
    }

    static int Main()
    {
        return Test001();
    }
}
```
