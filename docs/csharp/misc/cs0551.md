# <a name="compiler-error-cs0551"></a><span data-ttu-id="0c248-102">Derleyici Hatası CS0551</span><span class="sxs-lookup"><span data-stu-id="0c248-102">Compiler Error CS0551</span></span>
<span data-ttu-id="0c248-103">Açık arabirim uygulamasında 'uygulama' erişimcisi 'erişimcisi' eksik</span><span class="sxs-lookup"><span data-stu-id="0c248-103">Explicit interface implementation 'implementation' is missing accessor 'accessor'</span></span>  
  
 <span data-ttu-id="0c248-104">Bir arabirimin özelliği açıkça uygulayan bir sınıf, arabirim tanımlayan tüm erişimcileri uygulamalıdır.</span><span class="sxs-lookup"><span data-stu-id="0c248-104">A class that explicitly implements an interface's property must implement all the accessors that the interface defines.</span></span>

## <a name="example"></a><span data-ttu-id="0c248-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="0c248-106">Example</span></span>  
 <span data-ttu-id="0c248-107">Aşağıdaki örnek, CS0551 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="0c248-107">The following sample generates CS0551.</span></span>  
  
```csharp  
// CS0551.cs  
// compile with: /target:library  
interface ii  
{  
   int i  
   {  
      get;  
      set;  
   }  
}  
  
public class a : ii  
{  
   int ii.i { set {} }   // CS0551  
  
   // OK  
   int ii.i      
   {  
      set {}  
      get { return 0; }  
   }  
}  
```
