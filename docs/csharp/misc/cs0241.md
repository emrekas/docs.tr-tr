## <a name="example"></a><span data-ttu-id="1f8ab-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="1f8ab-107">Example</span></span>  
 <span data-ttu-id="1f8ab-108">Aşağıdaki örnek CS0241 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="1f8ab-108">The following sample generates CS0241.</span></span> <span data-ttu-id="1f8ab-109">Buna ek olarak, örnek, varsayılan bağımsız değişkenlerle bir yöntemi aşırı yükleme ile nasıl benzeten olduğunu gösterir.</span><span class="sxs-lookup"><span data-stu-id="1f8ab-109">In addition, the sample shows how to simulate, with overloading, a method with default arguments.</span></span>  
  
```csharp  
// CS0241.cs  
public class A  
{  
   public void Test(int i = 9) {}   // CS0241  
}  
  
public class B  
{  
   public void Test() { Test(9); }  
   public void Test(int i)  {}  
}  
  
public class C  
{  
   public static void Main()  
   {   
      B x = new B();  
      x.Test();  
   }  
}  
```
