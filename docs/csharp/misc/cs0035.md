# <a name="compiler-error-cs0035"></a>Derleyici Hatası CS0035
' Operator 'işleci 'type' türünde bir işlenen üzerinde belirsizdir  
  
Derleyici, birden fazla dönüştürme vardır ve işleç uygulanmadan önce seçebileceğiniz bilmez.
  
Aşağıdaki örnek, CS0035 oluşturur:  
  
```csharp  
// CS0035.cs  
class MyClass  
{  
   private int i;  
  
   public MyClass(int i)  
   {  
      this.i = i;  
   }  
  
   public static implicit operator double(MyClass x)  
   {  
      return (double) x.i;  
   }  
  
   public static implicit operator decimal(MyClass x)  
   {  
      return (decimal) x.i;  
   }  
}  
  
class MyClass2  
{  
   static void Main()  
   {  
      MyClass x = new MyClass(7);  
      object o = - x;   // CS0035  
      // try a cast:  
      // object o = - (double)x;  
   }  
}  
```
