# <a name="compiler-error-cs0035"></a><span data-ttu-id="28514-102">Derleyici Hatası CS0035</span><span class="sxs-lookup"><span data-stu-id="28514-102">Compiler Error CS0035</span></span>
<span data-ttu-id="28514-103">' Operator 'işleci 'type' türünde bir işlenen üzerinde belirsizdir</span><span class="sxs-lookup"><span data-stu-id="28514-103">Operator 'operator' is ambiguous on an operand of type 'type'</span></span>  
  
<span data-ttu-id="28514-104">Derleyici, birden fazla dönüştürme vardır ve işleç uygulanmadan önce seçebileceğiniz bilmez.</span><span class="sxs-lookup"><span data-stu-id="28514-104">The compiler has more than one available conversion and does not know which to choose before applying the operator.</span></span>
  
<span data-ttu-id="28514-105">Aşağıdaki örnek, CS0035 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="28514-105">The following sample generates CS0035:</span></span>  
  
```csharp  
// CS0035.cs  
class MyClass  
{  
   private int i;  
  
   public MyClass(int i)  
   {  
      this.i = i;  
   }  
  
   public static implicit operator double(MyClass x)  
   {  
      return (double) x.i;  
   }  
  
   public static implicit operator decimal(MyClass x)  
   {  
      return (decimal) x.i;  
   }  
}  
  
class MyClass2  
{  
   static void Main()  
   {  
      MyClass x = new MyClass(7);  
      object o = - x;   // CS0035  
      // try a cast:  
      // object o = - (double)x;  
   }  
}  
```
