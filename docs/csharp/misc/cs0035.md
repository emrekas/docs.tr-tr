# <a name="compiler-error-cs0035"></a><span data-ttu-id="ea028-102">Derleyici Hatası CS0035</span><span class="sxs-lookup"><span data-stu-id="ea028-102">Compiler Error CS0035</span></span>
<span data-ttu-id="ea028-103">' Operator 'işleci 'type' türünde bir işlenen üzerinde belirsizdir</span><span class="sxs-lookup"><span data-stu-id="ea028-103">Operator 'operator' is ambiguous on an operand of type 'type'</span></span>  
  
 <span data-ttu-id="ea028-104">Derleyici, birden fazla dönüştürme vardır ve işleç uygulanmadan önce seçebileceğiniz bilmez.</span><span class="sxs-lookup"><span data-stu-id="ea028-104">The compiler has more than one available conversion and does not know which to choose before applying the operator.</span></span> <span data-ttu-id="ea028-105">Daha fazla bilgi için [şablonlu kullanıcı tanımlı dönüştürmeler](/cpp/misc/templated-user-defined-conversions) ve [dönüştürme işleçleri](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span><span class="sxs-lookup"><span data-stu-id="ea028-105">For more information, see [Templated User Defined Conversions](/cpp/misc/templated-user-defined-conversions) and [Conversion Operators](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span></span>  
  
 <span data-ttu-id="ea028-106">Aşağıdaki örnek, CS0035 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="ea028-106">The following sample generates CS0035:</span></span>  
  
```csharp  
// CS0035.cs  
class MyClass  
{  
   private int i;  
  
   public MyClass(int i)  
   {  
      this.i = i;  
   }  
  
   public static implicit operator double(MyClass x)  
   {  
      return (double) x.i;  
   }  
  
   public static implicit operator decimal(MyClass x)  
   {  
      return (decimal) x.i;  
   }  
}  
  
class MyClass2  
{  
   static void Main()  
   {  
      MyClass x = new MyClass(7);  
      object o = - x;   // CS0035  
      // try a cast:  
      // object o = - (double)x;  
   }  
}  
```
