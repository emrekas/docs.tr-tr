# <a name="compiler-error-cs0690"></a>Derleyici Hatası CS0690
'Dosya' giriş dosyası geçersiz meta veriler içeriyor.  
  
 Meta veri dosyası açabilirsiniz, ancak bazı yerelleştirme sorunu nedeniyle meta veriler bozuk. Bu hata için benzer [CS0009](../../csharp/misc/cs0009.md)dışında meta veri dosyası açabilirsiniz.  
  
 (İşlem yürütülebilir) bir PE yürütülebilir bir dosyadır.
