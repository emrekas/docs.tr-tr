# <a name="compiler-warning-level-1-cs0626"></a><span data-ttu-id="db966-102">Derleyici Uyarısı (düzey 1) CS0626</span><span class="sxs-lookup"><span data-stu-id="db966-102">Compiler Warning (level 1) CS0626</span></span>

<span data-ttu-id="db966-103">Yöntemi, işleci veya erişimcisi 'method' dış olarak işaretli ve hiç özniteliği yok.</span><span class="sxs-lookup"><span data-stu-id="db966-103">Method, operator, or accessor 'method' is marked external and has no attributes on it.</span></span> <span data-ttu-id="db966-104">Dış uygulamayı belirtmek için DllImport özniteliği eklemeyi düşünün.</span><span class="sxs-lookup"><span data-stu-id="db966-104">Consider adding a DllImport attribute to specify the external implementation.</span></span>
  
 <span data-ttu-id="db966-105">Bir yöntem işaretli `extern` özniteliğiyle, örneğin da işaretlenmelidir, [DllImport](xref:System.Runtime.InteropServices.DllImportAttribute) özniteliği.</span><span class="sxs-lookup"><span data-stu-id="db966-105">A method marked `extern` should also be marked with an attribute, for example, the [DllImport](xref:System.Runtime.InteropServices.DllImportAttribute) attribute.</span></span>
  
 <span data-ttu-id="db966-106">Öznitelik burada yöntemin belirtir.</span><span class="sxs-lookup"><span data-stu-id="db966-106">The attribute specifies where the method is implemented.</span></span> <span data-ttu-id="db966-107">Çalışma zamanında, programın bu bilgilere ihtiyacınız olacaktır.</span><span class="sxs-lookup"><span data-stu-id="db966-107">At run time, the program will need this information.</span></span>  
  
 <span data-ttu-id="db966-108">Aşağıdaki örnek, CS0626 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="db966-108">The following sample generates CS0626:</span></span>  
  
```csharp
// CS0626.cs  
// compile with: /warnaserror  
using System.Runtime.InteropServices;  
  
public class MyClass  
{  
   static extern public void M(); // CS0626  
   // try the following line  
   // [DllImport("mydll.dll")] static extern public void M();  
  
   public static void Main()  
   {  
   }  
}  
```
