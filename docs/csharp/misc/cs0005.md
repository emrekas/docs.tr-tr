# <a name="compiler-error-cs0005"></a>Derleyici Hatası CS0005
Bir bağımsız değişken derleyici seçeneği 'compiler_option' gelmelidir  
  
 Derleyici Seçenekleri bazı parametreler gerektirir. Derleyici seçeneği tarafından gerekli bağımsız değişken geçirmezseniz CS0005 oluşturulur.  
  
 Daha fazla bilgi için [ C# derleyici seçenekleri](../../csharp/language-reference/compiler-options/index.md).
