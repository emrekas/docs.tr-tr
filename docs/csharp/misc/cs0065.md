# <a name="compiler-error-cs0065"></a><span data-ttu-id="11b37-102">Derleyici Hatası CS0065</span><span class="sxs-lookup"><span data-stu-id="11b37-102">Compiler Error CS0065</span></span>

<span data-ttu-id="11b37-103">'event': olay özelliğinin hem de sahip olmalıdır ekleme ve kaldırma erişimcileri</span><span class="sxs-lookup"><span data-stu-id="11b37-103">'event' : event property must have both add and remove accessors</span></span>

<span data-ttu-id="11b37-104">Bir alan değil bir olay, her iki erişim yöntemlerine sahip olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="11b37-104">An event that is not a field must have both access methods.</span></span>

<span data-ttu-id="11b37-105">Aşağıdaki örnek, CS0065 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="11b37-105">The following sample generates CS0065:</span></span>

```csharp
// CS0065.cs
using System;
public delegate void EventHandler(object sender, int e);
public class MyClass
{
   public event EventHandler Click   // CS0065,
   {
      // to fix, uncomment the add and remove definitions
      /*
      add
      {
         Click += value;
      }
      remove
      {
         Click -= value;
      }
      */
   }

   public static void Main()
   {
   }
}
```
