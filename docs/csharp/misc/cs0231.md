# <a name="compiler-error-cs0231"></a><span data-ttu-id="257a1-102">Derleyici Hatası CS0231</span><span class="sxs-lookup"><span data-stu-id="257a1-102">Compiler Error CS0231</span></span>
<span data-ttu-id="257a1-103">Bir params parametresi, biçimsel parametre listesindeki son parametre olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="257a1-103">A params parameter must be the last parameter in a formal parameter list.</span></span>  
  
 <span data-ttu-id="257a1-104">[Params](../../csharp/language-reference/keywords/params.md) parametresi değişken sayıda bağımsız değişkenler destekler ve sonra diğer tüm parametreler olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="257a1-104">The [params](../../csharp/language-reference/keywords/params.md) parameter supports a variable numbers of arguments and must be after all other parameters.</span></span> <span data-ttu-id="257a1-105">Daha fazla bilgi için [yöntemleri](../../csharp/programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="257a1-105">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="257a1-106">Aşağıdaki örnek, CS0231 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="257a1-106">The following sample generates CS0231:</span></span>  
  
```csharp  
// CS0231.cs  
class Test  
{  
   public void TestMethod(params int[] p, int i) {} // CS0231  
   // To resolve the error, use the following line instead:  
   // public void TestMethod(int i, params int[] p) {}   
  
   static void Main()   
   {  
   }  
}  
```
