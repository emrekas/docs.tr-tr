# <a name="compiler-error-cs1508"></a>Derleyici hatası CS1508
' Tanımlayıcı ' kaynak tanımlayıcısı bu derlemede zaten kullanılmış  
  
 Bir derlemede, aynı tanımlayıcı (***tanımlayıcı***) iki veya daha fazla [/Resource](../language-reference/compiler-options/resource-compiler-option.md) veya [/linkresource](../language-reference/compiler-options/linkresource-compiler-option.md) derleyici seçeneklerine geçildi.  
  
 Örneğin, aşağıdaki seçenekler CS1508 oluşturur:  
  
```console  
/resource:anyfile.bmp,DuplicatIdent /linkresource:a.bmp,DuplicatIdent  
```
