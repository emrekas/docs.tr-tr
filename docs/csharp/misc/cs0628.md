# <a name="compiler-warning-level-4-cs0628"></a><span data-ttu-id="30d61-102">Derleyici Uyarısı (düzey 4) CS0628</span><span class="sxs-lookup"><span data-stu-id="30d61-102">Compiler Warning (level 4) CS0628</span></span>
<span data-ttu-id="30d61-103">'member': sealed sınıfında yeni korunan üye bildirildi</span><span class="sxs-lookup"><span data-stu-id="30d61-103">'member' : new protected member declared in sealed class</span></span>  
  
 <span data-ttu-id="30d61-104">A [korumalı](../../csharp/language-reference/keywords/sealed.md) olamaz sınıfı ekleyin bir [korumalı](../../csharp/language-reference/keywords/protected.md) üyesi başka bir sınıf devralma mümkün olacağından `sealed` kullanın ve sınıf `protected` üye.</span><span class="sxs-lookup"><span data-stu-id="30d61-104">A [sealed](../../csharp/language-reference/keywords/sealed.md) class cannot introduce a [protected](../../csharp/language-reference/keywords/protected.md) member because no other class will be able to inherit from the `sealed` class and use the `protected` member.</span></span>  
  
 <span data-ttu-id="30d61-105">Aşağıdaki örnek, CS0628 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="30d61-105">The following sample generates CS0628:</span></span>  
  
```csharp  
// CS0628.cs  
// compile with: /W:4  
sealed class C  
{  
   protected int i;   // CS0628  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
