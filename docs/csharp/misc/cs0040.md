# <a name="compiler-error-cs0040"></a>Derleyici Hatası CS0040
Hata ayıklama bilgileri dosyası oluşturulurken beklenmeyen hata: 'açıklaması'  
  
 Bu hata kullanırken oluşabilecek [/debug](../../csharp/language-reference/compiler-options/debug-compiler-option.md) derleyici seçeneği ve derleyicinin bir .pdb dosyasına yazamıyor olduğunu gösterir. Visual Studio yeniden yüklenerek, derleyici bir dosya veya dizin yazma erişimi olduğundan emin veya/Debug ile derleme değil, bu hata için olası çözümleri içerir.
