<span data-ttu-id="172fe-103">Double türündeki sabit değer örtülü olarak ' Type ' türüne dönüştürülemez; Bu türde bir sabit değer oluşturmak için ' suffix ' sonekini kullanın</span><span class="sxs-lookup"><span data-stu-id="172fe-103">Literal of type double cannot be implicitly converted to type 'type'; use an 'suffix' suffix to create a literal of this type</span></span>

 <span data-ttu-id="172fe-107">Aşağıdaki örnek CS0664 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="172fe-107">The following sample generates CS0664:</span></span>  
  
```csharp  
// CS0664.cs  
class Example  
{  
    static void Main()  
    {  
        decimal d1 = 1.0;   // CS0664, because 1.0 is interpreted  
                            // as a double.  
  
        // Try the following line instead.  
        decimal d2 = 1.0M;  // The M tells the compiler that 1.0 is a  
                            // decimal.  
        Console.WriteLine(d2);  
    }  
}  
```
