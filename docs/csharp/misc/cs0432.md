# <a name="compiler-error-cs0432"></a>Derleyici Hatası CS0432
'Tanımlayıcı bulunamadı' diğer adı  
  
 Kullandığınızda bu hata oluşur. "::" sağ tarafındaki bir diğer ad değil bir tanımlayıcı. Hatayı gidermek için kullanma "." Bunun yerine.  
  
 Aşağıdaki örnek, CS0432 oluşturur:  
  
```csharp  
// CS0432.cs  
namespace A {  
    public class B {  
        public static void Meth() { }  
    }  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        A::B.Meth();   // CS0432  
       // To resolve, use the following line instead:  
       // A.B.Meth();  
    }  
}  
```
