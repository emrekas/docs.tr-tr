# <a name="compiler-warning-level-1-cs3010"></a>Derleyici Uyarısı (düzey 1) CS3010
'member': CLS uyumlu arabirimler yalnızca CLS uyumlu üyeler olması gerekir  
  
 İle işaretlenmiş derlemedeki `[assembly:CLCSompliant(true)]`, bir arabirim ile işaretlenen bir üye içeriyor `[CLCSompliant(false)]`. Ortak dil belirtimi (CLS) uyumluluk özniteliklerden birini kaldırın. CLS uyumluluğu hakkında daha fazla bilgi için bkz. [dil bağımsızlığı ve dilden bağımsız bileşenler](../../standard/language-independence.md).
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS3010 oluşturur:  
  
```csharp  
// CS3010.cs  
  
using System;  
  
[assembly:CLSCompliant(true)]  
public interface I  
{  
    [CLSCompliant(false)]  
    int M();   // CS3010  
}  
  
public class C : I  
{  
    public int M()  
    {  
        return 1;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
