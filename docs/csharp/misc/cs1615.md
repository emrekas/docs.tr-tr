# <a name="compiler-error-cs1615"></a><span data-ttu-id="6b52a-102">Derleyici Hatası CS1615</span><span class="sxs-lookup"><span data-stu-id="6b52a-102">Compiler Error CS1615</span></span>
<span data-ttu-id="6b52a-103">Bağımsız değişken 'number', 'anahtar sözcüğü' anahtar sözcüğüyle geçirilmemelidir</span><span class="sxs-lookup"><span data-stu-id="6b52a-103">Argument 'number' should not be passed with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="6b52a-104">Bir anahtar sözcük `ref` veya **kullanıma** işlevi değil sürdü çağrılırken bir `ref` veya **kullanıma** parametresi için bağımsız.</span><span class="sxs-lookup"><span data-stu-id="6b52a-104">One of the keywords `ref` or **out** was used when the function did not take a `ref` or **out** parameter for that argument.</span></span> <span data-ttu-id="6b52a-105">Bu hatayı gidermek için yanlış anahtar sözcüğü kaldırın ve varsa işlev bildirimi ile eşleşen uygun bir anahtar sözcüğünü kullanın.</span><span class="sxs-lookup"><span data-stu-id="6b52a-105">To resolve this error, remove the incorrect keyword and use the appropriate keyword that matches the function declaration, if any.</span></span>  
  
 <span data-ttu-id="6b52a-106">Aşağıdaki örnek, CS1615 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="6b52a-106">The following sample generates CS1615:</span></span>  
  
```csharp  
// CS1615.cs  
class C  
{  
   public void f(int i) {}  
   public static void Main()  
   {  
      int i = 1;  
      f(ref i);  // CS1615  
   }  
}  
```
