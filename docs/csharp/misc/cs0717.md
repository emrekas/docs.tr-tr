# <a name="compiler-error-cs0717"></a>Derleyici Hatası CS0717
'statik sınıf': statik sınıflar kısıtlama kullanılamaz  
  
 Statik sınıf yalnızca statik üyeleri ve örnek üyeleri içeren olarak genişletilemez. Bunlar genişletilemez çünkü türü olmadan'ın, bir statik sınıfın özelleştirmesi olduğundan bu bunları gereksiz tür parametreleri ve kısıtlamaları, getirir.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS0717 oluşturur:  
  
```csharp  
// CS0717.cs  
  
public static class SC  
{  
    public static void F()  
    {  
    }  
}  
  
public class G<T> where T : SC  // CS0717  
{  
    public static void Main()  
    {  
    }  
}  
```
