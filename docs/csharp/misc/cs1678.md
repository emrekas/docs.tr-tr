# <a name="compiler-error-cs1678"></a>Derleyici Hatası CS1678
Parametre 'number', 'type1' türü olarak bildirilmiş ancak 'type2' olmalıdır  
  
 Anonim bir yöntem parametre türü yöntemi için'nu atama temsilci bildirimi farklı olduğunda bu hata oluşur.  
  
 Aşağıdaki örnek, CS1678 oluşturur:  
  
```csharp  
// CS1678  
delegate void D(int i);  
class Errors   
{  
   static void Main()   
   {  
      D d = delegate(string s) { };   // CS1678  
      // To resolve, use the following line instead:  
      // D d = delegate(int s) { };  
   }  
}  
```
