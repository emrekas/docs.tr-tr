# <a name="compiler-warning-level-4-cs0028"></a>Derleyici Uyarısı (düzey 4) CS0028
' işlev bildirimi ', giriş noktası olmak için yanlış imzaya sahip  
  
 İçin `Main` metot bildirimi geçersizdi: geçersiz bir imzayla bildirildi. `Main`statik olarak bildirilmelidir ve [int](../language-reference/builtin-types/integral-numeric-types.md) ya da [void](../language-reference/keywords/void.md)döndürmelidir. Daha fazla bilgi için bkz. [Main () ve komut satırı bağımsız değişkenleri](../programming-guide/main-and-command-args/index.md).  
  
 Aşağıdaki örnek CS0028 oluşturur:  
  
```csharp  
// CS0028.cs  
// compile with: /W:4 /warnaserror  
public class a  
{  
    public static double Main(int i)   // CS0028  
    // Try the following line instead:  
    // public static void Main()  
    {  
    }  
}  
```
