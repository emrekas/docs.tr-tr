# <a name="compiler-warning-level-4-cs0028"></a>Derleyici Uyarısı (düzey 4) CS0028
'işlev bildirimi' giriş noktası olacak yanlış imzaya sahip  
  
 Yöntem bildiriminde için `Main` geçersizdi: geçersiz bir imza ile bildirildi. `Main` Statik ve ya da döndürmeli olarak bildirilmelidir [int](../../csharp/language-reference/builtin-types/integral-numeric-types.md) veya [void](../../csharp/language-reference/keywords/void.md). Daha fazla bilgi için [ana() ve komut satırı bağımsız değişkenleri](../../csharp/programming-guide/main-and-command-args/index.md).  
  
 Aşağıdaki örnek, CS0028 oluşturur:  
  
```csharp  
// CS0028.cs  
// compile with: /W:4 /warnaserror  
public class a  
{  
    public static double Main(int i)   // CS0028  
    // Try the following line instead:  
    // public static void Main()  
    {  
    }  
}  
```
