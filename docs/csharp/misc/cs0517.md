# <a name="compiler-error-cs0517"></a>Derleyici Hatası CS0517
'class' temel sınıfa sahip ve temel oluşturucu çağıramaz  
  
 Yalnızca .NET Framework ortak dil çalışma zamanı temel sınıfa sahip tek sınıftır nesne sınıfı için kaynak kodu derler CS0517 oluşabilir.
