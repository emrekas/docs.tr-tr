# <a name="compiler-error-cs0172"></a><span data-ttu-id="0464f-102">Derleyici Hatası CS0172</span><span class="sxs-lookup"><span data-stu-id="0464f-102">Compiler Error CS0172</span></span>
<span data-ttu-id="0464f-103">'Type1' ve 'type2' örtülü olarak birbirine dönüştüğünden koşullu ifadenin türü belirlenemiyor</span><span class="sxs-lookup"><span data-stu-id="0464f-103">Type of conditional expression cannot be determined because 'type1' and 'type2' implicitly convert to one another</span></span>  
  
<span data-ttu-id="0464f-104">Bir koşullu deyim içinde her iki tarafındaki türleri örtülü olarak dönüştüremedi olmalıdır `:` belirteci.</span><span class="sxs-lookup"><span data-stu-id="0464f-104">In a conditional statement, you must be able to implicitly convert the types on either side of the `:` token.</span></span> <span data-ttu-id="0464f-105">Ayrıca, karşılıklı örtük dönüştürmelerin olamaz; yalnızca bir dönüştürme gerekir.</span><span class="sxs-lookup"><span data-stu-id="0464f-105">Also, there cannot be mutual implicit conversions; you only need one conversion.</span></span>
  
<span data-ttu-id="0464f-106">Aşağıdaki örnek, CS0172 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="0464f-106">The following sample generates CS0172:</span></span>
  
```csharp  
// CS0172.cs  
public class Square  
{  
   public class Circle  
   {  
      public static implicit operator Circle(Square aa)  
      {  
         return null;  
      }  
  
      public static implicit operator Square(Circle aa)  
      // using explicit resolves this error  
      // public static explicit operator Square(Circle aa)  
      {  
         return null;  
      }  
   }  
  
   public static void Main()  
   {  
      Circle aa = new Circle();  
      Square ii = new Square();  
      object o = (1 == 1) ? aa : ii;   // CS0172  
      // the following cast would resolve this error  
      // (1 == 1) ? aa : (Circle)ii;  
   }  
}  
```
