# <a name="compiler-error-cs0172"></a><span data-ttu-id="0a1c3-102">Derleyici hatası CS0172</span><span class="sxs-lookup"><span data-stu-id="0a1c3-102">Compiler Error CS0172</span></span>
<span data-ttu-id="0a1c3-103">' Type1 ' ve ' type2 ' örtülü olarak birbirine dönüştürülemediğinden koşullu ifadenin türü belirlenemiyor</span><span class="sxs-lookup"><span data-stu-id="0a1c3-103">Type of conditional expression cannot be determined because 'type1' and 'type2' implicitly convert to one another</span></span>  
  
<span data-ttu-id="0a1c3-104">Koşullu bir bildirimde, `:` belirtecin her iki tarafındaki türleri örtülü olarak dönüştürebilmelisiniz.</span><span class="sxs-lookup"><span data-stu-id="0a1c3-104">In a conditional statement, you must be able to implicitly convert the types on either side of the `:` token.</span></span> <span data-ttu-id="0a1c3-105">Ayrıca, karşılıklı örtük dönüştürmeler de olamaz; yalnızca bir dönüştürmeye ihtiyacınız vardır.</span><span class="sxs-lookup"><span data-stu-id="0a1c3-105">Also, there cannot be mutual implicit conversions; you only need one conversion.</span></span>
  
<span data-ttu-id="0a1c3-106">Aşağıdaki örnek CS0172 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="0a1c3-106">The following sample generates CS0172:</span></span>
  
```csharp  
// CS0172.cs  
public class Square  
{  
   public class Circle  
   {  
      public static implicit operator Circle(Square aa)  
      {  
         return null;  
      }  
  
      public static implicit operator Square(Circle aa)  
      // using explicit resolves this error  
      // public static explicit operator Square(Circle aa)  
      {  
         return null;  
      }  
   }  
  
   public static void Main()  
   {  
      Circle aa = new Circle();  
      Square ii = new Square();  
      object o = (1 == 1) ? aa : ii;   // CS0172  
      // the following cast would resolve this error  
      // (1 == 1) ? aa : (Circle)ii;  
   }  
}  
```
