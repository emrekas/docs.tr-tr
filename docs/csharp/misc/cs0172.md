# <a name="compiler-error-cs0172"></a>Derleyici Hatası CS0172
'Type1' ve 'type2' örtülü olarak birbirine dönüştüğünden koşullu ifadenin türü belirlenemiyor  
  
Bir koşullu deyim içinde her iki tarafındaki türleri örtülü olarak dönüştüremedi olmalıdır `:` belirteci. Ayrıca, karşılıklı örtük dönüştürmelerin olamaz; yalnızca bir dönüştürme gerekir.
  
Aşağıdaki örnek, CS0172 oluşturur:
  
```csharp  
// CS0172.cs  
public class Square  
{  
   public class Circle  
   {  
      public static implicit operator Circle(Square aa)  
      {  
         return null;  
      }  
  
      public static implicit operator Square(Circle aa)  
      // using explicit resolves this error  
      // public static explicit operator Square(Circle aa)  
      {  
         return null;  
      }  
   }  
  
   public static void Main()  
   {  
      Circle aa = new Circle();  
      Square ii = new Square();  
      object o = (1 == 1) ? aa : ii;   // CS0172  
      // the following cast would resolve this error  
      // (1 == 1) ? aa : (Circle)ii;  
   }  
}  
```
