# <a name="compiler-error-cs0076"></a>Derleyici hatası CS0076
Numaralandırıcı adı ' değer__ ' ayrılmıştır ve kullanılamaz  
  
 Numaralandırmadaki bir öğe [](../language-reference/keywords/enum.md) **değer__** adlı bir tanımlayıcıya sahip olamaz.
