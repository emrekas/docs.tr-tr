# <a name="compiler-error-cs0102"></a>Derleyici Hatası CS0102
'type name' türü zaten 'identifier' için bir tanım içeriyor  
  
 Bir sınıf, tanımlayıcıların aynı kapsamda aynı ada sahip birden fazla bildirimi içerir. Hatayı düzeltmek için yinelenen tanımlayıcıları yeniden adlandırın.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS0102 oluşturur.  
  
```csharp  
// CS0102.cs  
// compile with: /target:library  
namespace MyApp  
{  
   public class MyClass  
   {  
      string s = "Hello";  
      string s = "Goodbye";   // CS0102  
  
      public void GetString()  
      {  
         string s = "Hello again";   // method scope, no error  
      }  
   }  
}  
```
