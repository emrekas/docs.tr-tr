# <a name="compiler-error-cs1671"></a>Derleyici Hatası CS1671
Bir ad alanı bildirimi değiştiriciler veya öznitelikler olamaz  
  
 Değiştiricilere izin verilmeyen biçimde bir ad alanına uygulandığında anlamlı değildir.  
  
 Aşağıdaki örnek, CS1671 oluşturur:  
  
```csharp  
// CS1671.cs  
public namespace NS // CS1671  
{  
  
}  
```
