# <a name="compiler-error-cs0629"></a><span data-ttu-id="6e9ec-102">Derleyici hatası CS0629</span><span class="sxs-lookup"><span data-stu-id="6e9ec-102">Compiler Error CS0629</span></span>
<span data-ttu-id="6e9ec-103">' Member ' koşullu üyesi ' type name ' türünde ' Base Class Member ' arabirim üyesini uygulayamaz</span><span class="sxs-lookup"><span data-stu-id="6e9ec-103">Conditional member 'member' cannot implement interface member 'base class member' in type 'Type Name'</span></span>  
  
 <span data-ttu-id="6e9ec-104">[Koşullu](../programming-guide/concepts/attributes/common-attributes.md#Conditional) öznitelik bir arabirimin uygulamasında kullanılamaz.</span><span class="sxs-lookup"><span data-stu-id="6e9ec-104">The [Conditional](../programming-guide/concepts/attributes/common-attributes.md#Conditional) attribute cannot be used on the implementation of an interface.</span></span>  
  
 <span data-ttu-id="6e9ec-105">Aşağıdaki örnek CS0629 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="6e9ec-105">The following sample generates CS0629:</span></span>  
  
```csharp  
// CS0629.cs  
interface MyInterface  
{  
   void MyMethod();  
}  
  
public class MyClass : MyInterface  
{  
   [System.Diagnostics.Conditional("debug")]  
   public void MyMethod()    // CS0629, remove the Conditional attribute  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
