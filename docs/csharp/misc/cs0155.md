 <span data-ttu-id="6a086-104">**System. Exception** 'dan bir [catch](../language-reference/keywords/try-catch.md) bloğuna türemeyen bir veri türünü geçirme girişiminde bulunuldu.</span><span class="sxs-lookup"><span data-stu-id="6a086-104">An attempt was made to pass a data type that does not derive from **System.Exception** into a [catch](../language-reference/keywords/try-catch.md) block.</span></span> <span data-ttu-id="6a086-105">Yalnızca **System. Exception** 'dan türetilen veri türleri bir **catch** bloğuna geçirilebilir.</span><span class="sxs-lookup"><span data-stu-id="6a086-105">Only data types that derive from **System.Exception** can be passed into a **catch** block.</span></span> <span data-ttu-id="6a086-106">Daha fazla bilgi için bkz.  <span data-ttu-id="6a086-107">Aşağıdaki örnek CS0155 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="6a086-107">The following sample generates CS0155:</span></span>  
  
```csharp  
// CS0155.cs  
using System;  
  
namespace MyNamespace  
{  
    public class MyClass2  
    // try the following line instead  
    // public class MyClass2 : Exception  
    {  
    }  
    public class MyClass  
    {  
        public static void Main()  
        {  
            try  
            {  
            }  
            catch (MyClass2)   // CS0155, resolves if you derive MyClass2 from Exception  
            {  
            }  
        }  
    }  
}  
```
