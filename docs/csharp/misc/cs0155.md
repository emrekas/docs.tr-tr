# <a name="compiler-error-cs0155"></a><span data-ttu-id="566ea-102">Derleyici Hatası CS0155</span><span class="sxs-lookup"><span data-stu-id="566ea-102">Compiler Error CS0155</span></span>
<span data-ttu-id="566ea-103">Yakalanan veya oluşturulan tür System.Exception'dan türetilmiş olmalıdır</span><span class="sxs-lookup"><span data-stu-id="566ea-103">The type caught or thrown must be derived from System.Exception</span></span>  
  
 <span data-ttu-id="566ea-104">Türünden türemez bir veri türü için bir girişimde bulunuldu **System.Exception** içine bir [catch](../../csharp/language-reference/keywords/try-catch.md) blok.</span><span class="sxs-lookup"><span data-stu-id="566ea-104">An attempt was made to pass a data type that does not derive from **System.Exception** into a [catch](../../csharp/language-reference/keywords/try-catch.md) block.</span></span> <span data-ttu-id="566ea-105">Öğesinden türetilen veri türleri **System.Exception** yöntemlere geçirilen bir **catch** blok.</span><span class="sxs-lookup"><span data-stu-id="566ea-105">Only data types that derive from **System.Exception** can be passed into a **catch** block.</span></span> <span data-ttu-id="566ea-106">Daha fazla bilgi için [özel durumlar ve özel durum işleme](../../csharp/programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="566ea-106">For more information, see [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="566ea-107">Aşağıdaki örnek, CS0155 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="566ea-107">The following sample generates CS0155:</span></span>  
  
```csharp  
// CS0155.cs  
using System;  
  
namespace MyNamespace  
{  
    public class MyClass2  
    // try the following line instead  
    // public class MyClass2 : Exception  
    {  
    }  
    public class MyClass  
    {  
        public static void Main()  
        {  
            try  
            {  
            }  
            catch (MyClass2)   // CS0155, resolves if you derive MyClass2 from Exception  
            {  
            }  
        }  
    }  
}  
```
