# <a name="compiler-error-cs0155"></a>Derleyici Hatası CS0155
Yakalanan veya oluşturulan tür System.Exception'dan türetilmiş olmalıdır  
  
 Türünden türemez bir veri türü için bir girişimde bulunuldu **System.Exception** içine bir [catch](../../csharp/language-reference/keywords/try-catch.md) blok. Öğesinden türetilen veri türleri **System.Exception** yöntemlere geçirilen bir **catch** blok. Daha fazla bilgi için [özel durumlar ve özel durum işleme](../../csharp/programming-guide/exceptions/index.md).  
  
 Aşağıdaki örnek, CS0155 oluşturur:  
  
```csharp  
// CS0155.cs  
using System;  
  
namespace MyNamespace  
{  
    public class MyClass2  
    // try the following line instead  
    // public class MyClass2 : Exception  
    {  
    }  
    public class MyClass  
    {  
        public static void Main()  
        {  
            try  
            {  
            }  
            catch (MyClass2)   // CS0155, resolves if you derive MyClass2 from Exception  
            {  
            }  
        }  
    }  
}  
```
