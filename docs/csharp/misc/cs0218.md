# <a name="compiler-error-cs0218"></a><span data-ttu-id="02b84-102">Derleyici hatası CS0218</span><span class="sxs-lookup"><span data-stu-id="02b84-102">Compiler Error CS0218</span></span>
<span data-ttu-id="02b84-103">Tür (' Type ') true işleci ve false işleci içeren bildirimler içermelidir</span><span class="sxs-lookup"><span data-stu-id="02b84-103">The type ('type') must contain declarations of operator true and operator false</span></span>  
  
<span data-ttu-id="02b84-104">Kullanıcı tanımlı bir tür [& işlecini](../language-reference/operators/boolean-logical-operators.md#logical-and-operator-) veya [ &#124; işlecini](../language-reference/operators/boolean-logical-operators.md#logical-or-operator-)aşırı yükse, kısa devre [& & işleci](../language-reference/operators/boolean-logical-operators.md#conditional-logical-and-operator-) veya [ &#124; &#124; işleci](../language-reference/operators/boolean-logical-operators.md#conditional-logical-or-operator-) oluşturmak için de [true ve false](../language-reference/operators/true-false-operators.md) işleçlerini tanımlamalıdır. tanımlı.</span><span class="sxs-lookup"><span data-stu-id="02b84-104">If a user-defined type overloads the [& operator](../language-reference/operators/boolean-logical-operators.md#logical-and-operator-) or [&#124; operator](../language-reference/operators/boolean-logical-operators.md#logical-or-operator-), it must also define [true and false](../language-reference/operators/true-false-operators.md) operators, in order to make short-circuiting [&& operator](../language-reference/operators/boolean-logical-operators.md#conditional-logical-and-operator-) or [&#124;&#124; operator](../language-reference/operators/boolean-logical-operators.md#conditional-logical-or-operator-) defined.</span></span>
  
 <span data-ttu-id="02b84-105">Aşağıdaki örnek CS0218 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="02b84-105">The following sample generates CS0218:</span></span>  
  
```csharp  
// CS0218.cs  
using System;  
public class MyClass  
{  
   // uncomment these operator declarations to resolve this CS0218  
   /*  
   public static bool operator true (MyClass f)  
   {  
      return false;  
   }  
  
   public static bool operator false (MyClass f)  
   {  
      return false;  
   }  
   */  
  
   public static implicit operator int(MyClass x)  
   {  
      return 0;  
   }  
  
   public static MyClass operator & (MyClass f1, MyClass f2)  
   {  
      return new MyClass();  
   }  
  
   public static void Main()  
   {  
      MyClass f = new MyClass();  
      int i = f && f;   // CS0218, requires operators true and false  
   }  
}  
```
