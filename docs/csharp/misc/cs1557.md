# <a name="compiler-error-cs1557"></a>Derleyici hatası CS1557
' Class ', farklı bir çıkış dosyasında olduğundan Main yöntemi için kullanılamaz  
  
 [/Main](../language-reference/compiler-options/main-compiler-option.md) derleyici seçeneği, çok çıktılı bir dosya derlemesinde bir çıkış dosyası için belirtildi. Ancak, sınıf/Main derlemesinin kaynak kodunda bulunamadı; derleme içindeki diğer çıkış dosyalarından biri için kaynak kodu dosyasında bulundu.
