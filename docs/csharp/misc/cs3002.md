## <a name="example"></a><span data-ttu-id="7594b-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="7594b-106">Example</span></span>

```csharp

public class a  
{  
    public ushort bad()   // CS3002, public method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    private ushort OK()   // OK, private method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
