# <a name="compiler-warning-level-1-cs3002"></a>Derleyici Uyarısı (düzey 1) CS3002
' Method ' dönüş türü CLS uyumlu değil  
  
 [Ortak](../language-reference/keywords/public.md), [korumalı](../language-reference/keywords/protected.md)veya `protected internal` Yöntem, türü ortak dil belirtimi (CLS) ile uyumlu olan bir değer döndürmelidir. CLS uyumluluğu hakkında daha fazla bilgi için bkz. [Dil bağımsızlığı ve dilden bağımsız bileşenler](../../standard/language-independence.md).
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS3002 oluşturur:  
  
```csharp  
// CS3002.cs  
  
[assembly:System.CLSCompliant(true)]  
public class a  
{  
    public ushort bad()   // CS3002, public method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    private ushort OK()   // OK, private method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
