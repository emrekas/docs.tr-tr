# <a name="compiler-error-cs1627"></a><span data-ttu-id="ff3f5-102">Derleyici Hatası CS1627</span><span class="sxs-lookup"><span data-stu-id="ff3f5-102">Compiler Error CS1627</span></span>
<span data-ttu-id="ff3f5-103">Bırakma dönüşünden sonra ifade bekleniyor</span><span class="sxs-lookup"><span data-stu-id="ff3f5-103">Expression expected after yield return</span></span>  
  
 <span data-ttu-id="ff3f5-104">Bu hata oluşur `yield` bir ifade olmadan kullanılır.</span><span class="sxs-lookup"><span data-stu-id="ff3f5-104">This error occurs if `yield` is used without an expression.</span></span> <span data-ttu-id="ff3f5-105">Bu hatayı önlemek için uygun ifade deyiminde ekleyin.</span><span class="sxs-lookup"><span data-stu-id="ff3f5-105">To avoid this error, insert the appropriate expression in the statement.</span></span>  
  
 <span data-ttu-id="ff3f5-106">Aşağıdaki örnek, CS1627 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="ff3f5-106">The following sample generates CS1627:</span></span>  
  
```csharp  
// CS1627.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      yield return;   // CS1627  
      // To resolve, add the following line:  
      // yield return 0;  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
