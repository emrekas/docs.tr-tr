 <span data-ttu-id="f6423-104">Belirtme <xref:System.Runtime.InteropServices.ComImportAttribute> özniteliği, logrequest olayını sınıfı bir COM modülü içeri aktarılamadı olduğunu anlamına gelir.</span><span class="sxs-lookup"><span data-stu-id="f6423-104">Specifying the <xref:System.Runtime.InteropServices.ComImportAttribute> attribute implies that the implementation for the class is to be imported from a COM module.</span></span> <span data-ttu-id="f6423-105">Ek yöntemleri veya temel sınıftan devralınan alanları COM modülde tanımlanmış uygulamaya eklenecek izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="f6423-105">Additional methods or fields inherited from the base class are not allowed to be added to the implementation defined in the COM module.</span></span>
