# <a name="compiler-error-cs1673"></a><span data-ttu-id="44898-102">Derleyici Hatası CS1673</span><span class="sxs-lookup"><span data-stu-id="44898-102">Compiler Error CS1673</span></span>
<span data-ttu-id="44898-103">Anonim yöntem, lambda ifadeleri ve yapılar içinde sorgu ifadeleri 'this' örnek üyelerine erişemez.</span><span class="sxs-lookup"><span data-stu-id="44898-103">Anonymous methods, lambda expressions, and query expressions inside structs cannot access instance members of 'this'.</span></span> <span data-ttu-id="44898-104">'This' anonim yöntem, lambda ifadesi veya sorgu ifadesi dışında bir yerel değişkene kopyalama ve yerel kullanmayı göz önünde bulundurun.</span><span class="sxs-lookup"><span data-stu-id="44898-104">Consider copying 'this' to a local variable outside the anonymous method, lambda expression or query expression and using the local instead.</span></span>  
  
 <span data-ttu-id="44898-105">Aşağıdaki örnek, CS1673 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="44898-105">The following sample generates CS1673:</span></span>  
  
```csharp  
// CS1673.cs  
delegate int MyDelegate();  
  
public struct S  
{  
   int member;  
  
   public int F(int i)  
   {  
       member = i;  
       // Try assigning to a local variable  
       // S s = this;  
       MyDelegate d = delegate()  
       {  
          i = this.member;  // CS1673  
          // And use the local variable instead of "this"  
          // i =  s.member;  
          return i;  
  
       };  
       return d();  
   }  
}  
  
class CMain  
{  
   public static void Main()  
   {  
   }  
}  
```
