# <a name="compiler-error-cs0709"></a>Derleyici Hatası CS0709
'derived class': 'temel sınıfı' statik sınıfından türetilemez  
  
 Statik sınıf örneği veya türetilmiş olamaz. Diğer bir deyişle, bir statik sınıf başka bir sınıfın temel sınıf olamaz.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS0709 oluşturur.  
  
```csharp  
// CS0709.cs  
// compile with: /target:library  
public static class Base {}  
public class Derived : Base {}   // CS0709  
```
