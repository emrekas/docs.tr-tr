# <a name="compiler-warning-level-3-cs0219"></a>Derleyici Uyarısı (Düzey 3) CS0219
'Variable' değişkeni atanır ancak değeri hiç kullanılmadı  
  
 Bildirme ve bir değişkene atayın ancak bunu kullanmayın, derleyici bir düzey üç uyarı verir.  
  
 Aşağıdaki örnek, CS0219 oluşturur:  
  
```csharp  
// CS0219.cs  
// compile with: /W:3  
public class MyClass  
{  
   public static void Main()  
   {  
      int a = 0;   // CS0219  
   }  
}  
```
