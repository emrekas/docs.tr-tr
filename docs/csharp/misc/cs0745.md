# <a name="compiler-error-cs0745"></a><span data-ttu-id="ca1bc-102">Derleyici Hatası CS0745</span><span class="sxs-lookup"><span data-stu-id="ca1bc-102">Compiler Error CS0745</span></span>
<span data-ttu-id="ca1bc-103">Beklenen bağlamsal anahtar sözcük 'by'</span><span class="sxs-lookup"><span data-stu-id="ca1bc-103">Expected contextual keyword 'by'</span></span>  
  
 <span data-ttu-id="ca1bc-104">Desenini `group` yan tümcesi `group...by` bir isteğe bağlı olarak izlenen `into`, aşağıdaki örnekte gösterildiği gibi:</span><span class="sxs-lookup"><span data-stu-id="ca1bc-104">The pattern for the `group` clause is `group...by` followed by an optional `into`, as shown in the following example:</span></span>

```csharp  
var query = from name in names  
            group name by name[0];  
```

```csharp

             //...additional query clauses  
```

## <a name="example"></a><span data-ttu-id="ca1bc-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="ca1bc-108">Example</span></span>  
 <span data-ttu-id="ca1bc-109">Aşağıdaki kod CS0745 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="ca1bc-109">The following code generates CS0745:</span></span>  
  
```csharp  
// cs0745.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        string[] names = { "Bob", "Bill", "Jonetta", "Mary" };  
  
        var query = from name in names  
                    group name name[0]; // CS0745  
  
        return 1;  
    }  
}  
```
