# <a name="compiler-error-cs0110"></a><span data-ttu-id="b5870-102">Derleyici Hatası CS0110</span><span class="sxs-lookup"><span data-stu-id="b5870-102">Compiler Error CS0110</span></span>
<span data-ttu-id="b5870-103">'Const bildirimi' için sabit değerin değerlendirilmesi döngüsel başvuru içeriyor</span><span class="sxs-lookup"><span data-stu-id="b5870-103">The evaluation of the constant value for 'const declaration' involves a circular definition</span></span>  
  
 <span data-ttu-id="b5870-104">Bildirimi bir [const](../../csharp/language-reference/keywords/const.md) değişkeni (`a`) başka bir const değişkeni başvuramaz (`b`) de başvuran (`a`).</span><span class="sxs-lookup"><span data-stu-id="b5870-104">The declaration of a [const](../../csharp/language-reference/keywords/const.md) variable (`a`) cannot reference another const variable (`b`) that also references (`a`).</span></span>  
  
 <span data-ttu-id="b5870-105">Aşağıdaki örnek, CS0110 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="b5870-105">The following sample generates CS0110:</span></span>  
  
```csharp  
// CS0110.cs  
namespace MyNamespace  
{  
   public class A  
   {  
      public static void Main()  
      {  
      }  
   }  
  
   public class B : A  
   {  
      public const int i = c + 1;   // CS0110, c already references i  
      public const int c = i + 1;  
      // the following line would be OK  
      // public const int c = 10;  
   }  
}  
```
