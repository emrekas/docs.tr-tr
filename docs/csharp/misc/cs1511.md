# <a name="compiler-error-cs1511"></a><span data-ttu-id="bcccd-102">Derleyici Hatası CS1511</span><span class="sxs-lookup"><span data-stu-id="bcccd-102">Compiler Error CS1511</span></span>
<span data-ttu-id="bcccd-103">'Base' anahtar sözcüğü statik yöntemde kullanılamaz</span><span class="sxs-lookup"><span data-stu-id="bcccd-103">Keyword 'base' is not available in a static method</span></span>  
  
 <span data-ttu-id="bcccd-104">[Temel](../../csharp/language-reference/keywords/base.md) anahtar sözcüğü kullanıldı bir [statik](../../csharp/language-reference/keywords/static.md) yöntemi.</span><span class="sxs-lookup"><span data-stu-id="bcccd-104">The [base](../../csharp/language-reference/keywords/base.md) keyword was used in a [static](../../csharp/language-reference/keywords/static.md) method.</span></span> <span data-ttu-id="bcccd-105">`base` yalnızca bir örnek oluşturucusu, örnek yöntemi veya örneği erişimci çağrılabilir.</span><span class="sxs-lookup"><span data-stu-id="bcccd-105">`base` can only be called in an instance constructor, instance method, or instance accessor.</span></span>  
  
## <a name="example"></a><span data-ttu-id="bcccd-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="bcccd-106">Example</span></span>  
 <span data-ttu-id="bcccd-107">Aşağıdaki örnek, CS1511 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="bcccd-107">The following sample generates CS1511.</span></span>  
  
```csharp  
// CS1511.cs  
// compile with: /target:library  
public class A  
{  
   public int j = 0;  
}  
  
class C : A  
{  
   public void Method()  
   {  
      base.j = 3;   // base allowed here  
   }  
  
   public static int StaticMethod()  
   {  
      base.j = 3;   // CS1511  
      return 1;  
   }  
}  
```
