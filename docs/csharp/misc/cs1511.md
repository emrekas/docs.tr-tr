# <a name="compiler-error-cs1511"></a><span data-ttu-id="13d5b-102">Derleyici hatası CS1511</span><span class="sxs-lookup"><span data-stu-id="13d5b-102">Compiler Error CS1511</span></span>
<span data-ttu-id="13d5b-103">' Base ' anahtar sözcüğü statik bir yöntemde kullanılamaz</span><span class="sxs-lookup"><span data-stu-id="13d5b-103">Keyword 'base' is not available in a static method</span></span>  
  
 <span data-ttu-id="13d5b-104">[Base](../language-reference/keywords/base.md) anahtar sözcüğü [statik](../language-reference/keywords/static.md) bir yöntemde kullanıldı.</span><span class="sxs-lookup"><span data-stu-id="13d5b-104">The [base](../language-reference/keywords/base.md) keyword was used in a [static](../language-reference/keywords/static.md) method.</span></span> <span data-ttu-id="13d5b-105">`base`yalnızca bir örnek oluşturucusunda, örnek yönteminde veya örnek erişimcisinde çağrılabilir.</span><span class="sxs-lookup"><span data-stu-id="13d5b-105">`base` can only be called in an instance constructor, instance method, or instance accessor.</span></span>  
  
## <a name="example"></a><span data-ttu-id="13d5b-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="13d5b-106">Example</span></span>  
 <span data-ttu-id="13d5b-107">Aşağıdaki örnek CS1511 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="13d5b-107">The following sample generates CS1511.</span></span>  
  
```csharp  
// CS1511.cs  
// compile with: /target:library  
public class A  
{  
   public int j = 0;  
}  
  
class C : A  
{  
   public void Method()  
   {  
      base.j = 3;   // base allowed here  
   }  
  
   public static int StaticMethod()  
   {  
      base.j = 3;   // CS1511  
      return 1;  
   }  
}  
```
