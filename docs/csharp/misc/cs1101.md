# <a name="compiler-error-cs1101"></a><span data-ttu-id="4d856-102">Derleyici Hatası CS1101</span><span class="sxs-lookup"><span data-stu-id="4d856-102">Compiler Error CS1101</span></span>
<span data-ttu-id="4d856-103">'Ref' parametre değiştiricisi 'this' ile kullanılamaz.</span><span class="sxs-lookup"><span data-stu-id="4d856-103">The parameter modifier 'ref' cannot be used with 'this'.</span></span>  
  
 <span data-ttu-id="4d856-104">Zaman `this` anahtar sözcüğünün değiştirdiği bir statik yöntem ilk parametresi, derleyicinin yöntemin bir genişletme yöntemi olduğunu bildirir.</span><span class="sxs-lookup"><span data-stu-id="4d856-104">When the `this` keyword modifies the first parameter of a static method, it signals to the compiler that the method is an extension method.</span></span> <span data-ttu-id="4d856-105">Diğer bir değiştiricilere gerekli veya genişletme yönteminin ilk parametresi üzerinde izin verilir.</span><span class="sxs-lookup"><span data-stu-id="4d856-105">No other modifiers are needed or allowed on the first parameter of an extension method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="4d856-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="4d856-106">Example</span></span>  
 <span data-ttu-id="4d856-107">Aşağıdaki örnek, CS1101 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="4d856-107">The following example generates CS1101:</span></span>  
  
```csharp  
// cs1101.cs  
// Compile with: /target:library  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(ref this int i) {} // CS1101  
  
    // Single type parameter.  
        public static void Test<T>(ref this T t) {}// CS1101  
  
    // Multiple type parameters.  
        public static void Test<T,U,V>(ref this U u) {}// CS1101  
}  
```
