# <a name="compiler-error-cs0715"></a>Derleyici Hatası CS0715
'statik sınıf': statik sınıflar kullanıcı tanımlı işleçler içeremez  
  
 Kullanıcı tanımlı işleçler sınıfların örnekleri üzerinde çalışır. Statik sınıflar oluşturulamıyor, alacak operatörleri için hazırlanan örneklerini oluşturmak mümkün değildir. Bu nedenle, kullanıcı tanımlı işleçler statik sınıflar için izin verilmez.  
  
 Aşağıdaki örnek, CS0715 oluşturur:  
  
```csharp  
// CS0715.cs  
public static class C  
{  
   public static C operator+(C c)  // CS0715  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
