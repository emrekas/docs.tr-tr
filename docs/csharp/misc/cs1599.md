# <a name="compiler-error-cs1599"></a><span data-ttu-id="05544-102">Derleyici Hatası CS1599</span><span class="sxs-lookup"><span data-stu-id="05544-102">Compiler Error CS1599</span></span>
<span data-ttu-id="05544-103">Yöntem veya temsilci 'type' türü döndüremez</span><span class="sxs-lookup"><span data-stu-id="05544-103">Method or delegate cannot return type 'type'</span></span>  
  
 <span data-ttu-id="05544-104">.NET Framework Sınıf Kitaplığı'nda, örneğin, bazı türleri <xref:System.TypedReference>, <xref:System.RuntimeArgumentHandle> ve <xref:System.ArgIterator> güvenli olmayan işlemleri gerçekleştirmek için büyük olasılıkla kullanılabilir olduğundan, dönüş türleri olarak kullanılamaz.</span><span class="sxs-lookup"><span data-stu-id="05544-104">Some types in the .NET Framework class library, for example, <xref:System.TypedReference>, <xref:System.RuntimeArgumentHandle> and <xref:System.ArgIterator> cannot be used as return types because they can potentially be used to perform unsafe operations.</span></span>  
  
 <span data-ttu-id="05544-105">Aşağıdaki örnek, CS1599 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="05544-105">The following sample generates CS1599:</span></span>  
  
```csharp  
// CS1599.cs  
using System;  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public TypedReference Test1()   // CS1599  
   {  
      return null;  
   }  
  
   public ArgIterator Test2()   // CS1599  
   {  
      return null;  
   }  
}  
```
