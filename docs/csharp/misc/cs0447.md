# <a name="compiler-error-cs0447"></a><span data-ttu-id="fd545-102">Derleyici Hatası CS0447</span><span class="sxs-lookup"><span data-stu-id="fd545-102">Compiler Error CS0447</span></span>
<span data-ttu-id="fd545-103">Öznitelikler tür bağımsız değişkeni tür parametreleri yalnızca üzerinde kullanılamaz</span><span class="sxs-lookup"><span data-stu-id="fd545-103">Attributes cannot be used on type arguments, only on type parameters</span></span>  
  
 <span data-ttu-id="fd545-104">Öznitelik bir çağırma deyiminde bir tür bağımsız değişkeni için geçerli olduğunda bu hata oluşur.</span><span class="sxs-lookup"><span data-stu-id="fd545-104">This error occurs when you apply an attribute to a type argument that occurs in an invocation statement.</span></span> <span data-ttu-id="fd545-105">Aşağıdaki gibi bir sınıf veya metodu bildirim deyimindeki bir tür parametresi için bir öznitelik uygulamak için kabul edilebilir:</span><span class="sxs-lookup"><span data-stu-id="fd545-105">It is acceptable to apply an attribute to a type parameter in a class or method declaration statement such as the following:</span></span>  
  
```csharp

```  
  
 <span data-ttu-id="fd545-106">Aşağıdaki kod satırını, bu hata oluşturur.</span><span class="sxs-lookup"><span data-stu-id="fd545-106">The following line of code will generate this error.</span></span> <span data-ttu-id="fd545-107">Varsayılır sınıfı `C`tanımlanan kod, önceki satıra sahip adlı statik bir yöntem `MyStaticMethod`.</span><span class="sxs-lookup"><span data-stu-id="fd545-107">It is assumed that the class `C`, defined in the previous line of code, has a static method called `MyStaticMethod`.</span></span>  
  
```csharp

```  
  
## <a name="example"></a><span data-ttu-id="fd545-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="fd545-108">Example</span></span>  
 <span data-ttu-id="fd545-109">Aşağıdaki kod, hata CS0447 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="fd545-109">The following code generates error CS0447.</span></span>  
  
```csharp  
// CS0447.cs

{  
    public interface I<A>   
    {  
        void Meth<B>();  
    }  
    public class B : I<int>   
    {  
        void I<[Test] int>.Meth<X>() { }  // CS0447  
    }  
}  
```
