# <a name="compiler-error-cs0450"></a><span data-ttu-id="61265-102">Derleyici Hatası CS0450</span><span class="sxs-lookup"><span data-stu-id="61265-102">Compiler Error CS0450</span></span>
<span data-ttu-id="61265-103">'Parametre adı type': hem bir kısıtlama sınıfı hem de 'class' veya 'struct' kısıtlaması belirtilemez</span><span class="sxs-lookup"><span data-stu-id="61265-103">'Type Parameter Name': cannot specify both a constraint class and the 'class' or 'struct' constraint</span></span>  
  
 <span data-ttu-id="61265-104">Yapı türü kısıtlaması tarafından bir tür parametresi kısıtlı, yapı ve sınıf birbirini dışlayan kategorileri olduğundan Ayrıca belirli bir sınıf türü tarafından kısıtlı olabilir için mantıksal olarak çelişkili olur.</span><span class="sxs-lookup"><span data-stu-id="61265-104">If a type parameter is constrained by the struct type constraint, it is logically contradictory for it to also be constrained by a specific class type, since struct and class are mutually exclusive categories.</span></span> <span data-ttu-id="61265-105">Belirli bir sınıf tür kısıtlaması tarafından bir tür parametresi kısıtlı tanımı sınıf türü kısıtlaması tarafından kısıtlanmış olarak ve bu nedenle sınıf türü kısıtlaması belirtme gereksizdir.</span><span class="sxs-lookup"><span data-stu-id="61265-105">If a type parameter is constrained by a specific class type constraint, then it is by definition constrained by the class type constraint, and so specifying the class type constraint is redundant.</span></span>  
  
## <a name="example"></a><span data-ttu-id="61265-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="61265-106">Example</span></span>  
  
```csharp  
// CS0450.cs  
// compile with: /t:library  
public class GenericsErrors   
{  
    public class B { }  
    public class G3<T> where T : struct, B { } // CS0450  
// To resolve, use the following line instead:  
// public class G3<T> where T : B { }  
}  
```
