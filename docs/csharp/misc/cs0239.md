 <span data-ttu-id="adc76-104">Üye, [korumalı](../language-reference/keywords/sealed.md) devralınmış bir üyeyi [geçersiz kılamaz](../language-reference/keywords/override.md) .</span><span class="sxs-lookup"><span data-stu-id="adc76-104">A member cannot [override](../language-reference/keywords/override.md) a [sealed](../language-reference/keywords/sealed.md) inherited member.</span></span> <span data-ttu-id="adc76-105">Daha fazla bilgi için bkz.  <span data-ttu-id="adc76-106">Aşağıdaki örnek CS0239 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="adc76-106">The following sample generates CS0239:</span></span>  
  
```csharp  
// CS0239.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public override sealed void f()  
   {  
   }  
}

{  
   public override void f()   // CS0239  
   // Try the following definition instead:  
   // public new void f()  
   {  
   }  
}  
```
