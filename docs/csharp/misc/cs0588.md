# <a name="compiler-error-cs0588"></a>Derleyici Hatası CS0588
Derleyici iç hatası: Aşama 'LEX'  
  
 Derleyici, bağlanamama beklenmeyen söz dizimi ayrıştırma nedeniyle başarısız olduğunu anlamaya çalışın. Durum bu değilse, deneyin [konuşmak bize](/visualstudio/ide/talk-to-us).
