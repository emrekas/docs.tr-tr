# <a name="compiler-warning-level-1-cs3005"></a>Derleyici Uyarısı (düzey 1) CS3005
Yalnızca büyük küçük farklı'identifier ' tanımlayıcısı CLS uyumlu değil  
  
 A [genel](../../csharp/language-reference/keywords/public.md), [korumalı](../../csharp/language-reference/keywords/protected.md), veya `protected internal` birbirinden farklı olan tanımlayıcı `public`, `protected`, veya `protected internal` yalnızca söz konusu olduğunda harf, bir veya daha fazla tanımlayıcı uyumlu değil ortak dil belirtimi (CLS) ile. CLS uyumluluğu hakkında daha fazla bilgi için bkz. [dil bağımsızlığı ve dilden bağımsız bileşenler](../../standard/language-independence.md).
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS3003 oluşturur:  
  
```csharp  
// CS3005.cs  
  
using System;  
  
[assembly:CLSCompliant(true)]  
public class a  
{  
    public static int a1 = 0;  
    public static int A1 = 1;   // CS3005  
  
    public static void Main()  
    {  
        Console.WriteLine(a1);  
        Console.WriteLine(A1);  
    }  
}  
```
