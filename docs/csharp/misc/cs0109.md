# <a name="compiler-warning-level-4-cs0109"></a>Derleyici Uyarısı (düzey 4) CS0109
'Member' üyesi devralınmış bir üyeyi gizlemez. Yeni anahtar sözcük gerekli değil  
  
 Dahil edilen bir sınıf bildirimi [yeni](../../csharp/language-reference/keywords/new-modifier.md) bildirimi var olan bir temel sınıf bildiriminde geçersiz kılmaz olsa bile anahtar sözcüğü. Silebilirsiniz **yeni** anahtar sözcüğü.  
  
 Aşağıdaki örnek, CS0109 oluşturur:  
  
```csharp  
// CS0109.cs  
// compile with: /W:4  
namespace x  
{  
   public class a  
   {  
      public int i;  
   }  
  
   public class b : a  
   {  
      public new int i;  
      public new int j;   // CS0109  
      public static void Main()  
      {  
      }  
   }  
}  
```
