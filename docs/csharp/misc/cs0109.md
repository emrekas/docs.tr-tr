# <a name="compiler-warning-level-4-cs0109"></a><span data-ttu-id="fba28-102">Derleyici Uyarısı (düzey 4) CS0109</span><span class="sxs-lookup"><span data-stu-id="fba28-102">Compiler Warning (level 4) CS0109</span></span>
<span data-ttu-id="fba28-103">'Member' üyesi devralınmış bir üyeyi gizlemez.</span><span class="sxs-lookup"><span data-stu-id="fba28-103">The member 'member' does not hide an inherited member.</span></span> <span data-ttu-id="fba28-104">Yeni anahtar sözcük gerekli değil</span><span class="sxs-lookup"><span data-stu-id="fba28-104">The new keyword is not required</span></span>  
  
 <span data-ttu-id="fba28-105">Dahil edilen bir sınıf bildirimi [yeni](../../csharp/language-reference/keywords/new-modifier.md) bildirimi var olan bir temel sınıf bildiriminde geçersiz kılmaz olsa bile anahtar sözcüğü.</span><span class="sxs-lookup"><span data-stu-id="fba28-105">A class declaration included the [new](../../csharp/language-reference/keywords/new-modifier.md) keyword even though the declaration does not override an existing declaration in a base class.</span></span> <span data-ttu-id="fba28-106">Silebilirsiniz **yeni** anahtar sözcüğü.</span><span class="sxs-lookup"><span data-stu-id="fba28-106">You can delete the **new** keyword.</span></span>  
  
 <span data-ttu-id="fba28-107">Aşağıdaki örnek, CS0109 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="fba28-107">The following sample generates CS0109:</span></span>  
  
```csharp  
// CS0109.cs  
// compile with: /W:4  
namespace x  
{  
   public class a  
   {  
      public int i;  
   }  
  
   public class b : a  
   {  
      public new int i;  
      public new int j;   // CS0109  
      public static void Main()  
      {  
      }  
   }  
}  
```
