 <span data-ttu-id="a6abf-105">Dahil edilen bir sınıf bildirimi [yeni](../../csharp/language-reference/keywords/new.md) bildirimi var olan bir temel sınıf bildiriminde geçersiz kılmaz olsa bile anahtar sözcüğü.</span><span class="sxs-lookup"><span data-stu-id="a6abf-105">A class declaration included the [new](../../csharp/language-reference/keywords/new.md) keyword even though the declaration does not override an existing declaration in a base class.</span></span> <span data-ttu-id="a6abf-106">Silebilirsiniz **yeni** anahtar sözcüğü.</span><span class="sxs-lookup"><span data-stu-id="a6abf-106">You can delete the **new** keyword.</span></span>

```csharp  
{  
   public class a  
   {  
      public int i;  
   }  
  
   public class b : a  
   {  
      public new int i;

      {  
      }  
   }  
}  
```
