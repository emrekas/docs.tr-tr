# <a name="compiler-error-cs1613"></a>Derleyici hatası CS1613
' Interface ' arabirimi için ' class ' yönetilen coclass sarmalayıcı sınıfı bulunamıyor (bir derleme başvurunuz mu eksik?)  
  
 Bir arabirimden COM nesnesi örneğini oluşturma girişiminde bulunuldu. Arabirim **ComImport** ve `CoClass` özniteliklerine sahiptir, ancak derleyici `CoClass` öznitelik için verilen türü bulamıyor.  
  
 Bu hatayı çözmek için aşağıdakilerden birini deneyebilirsiniz:  
  
- Coclass 'a sahip olan derlemeye bir başvuru ekleyin (çoğu zaman arabirimin ve coclass 'ın aynı derlemede olması gerekir). Bilgi için bkz. [/Reference](../language-reference/compiler-options/reference-compiler-option.md) veya [başvuru ekleme iletişim kutusu](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) .  
  
- `CoClass` Arabirimdeki özniteliği onarın.  
  
 Aşağıdaki örnek, **CoClassAttribute**'in doğru kullanımını gösterir:  
  
```csharp  
// CS1613.cs  
using System;  
using System.Runtime.InteropServices;  
  
[Guid("1FFD7840-E82D-4268-875C-80A160C23296")]  
[ComImport()]  
[CoClass(typeof(A))]  
public interface IA{}  
public class A : IA {}  
  
public class AA  
{  
   public static void Main()  
   {  
      IA i;  
      i = new IA(); // This is equivalent to new A().  
                    // because of the CoClass attribute on IA  
   }  
}  
```
