# <a name="compiler-error-cs1613"></a><span data-ttu-id="b8f74-102">Derleyici Hatası CS1613</span><span class="sxs-lookup"><span data-stu-id="b8f74-102">Compiler Error CS1613</span></span>
<span data-ttu-id="b8f74-103">'İnterface' arabirimi için ' class' yönetilen coclass sarmalayıcı sınıfı bulunamıyor (bir derleme başvurunuz mu eksik?)</span><span class="sxs-lookup"><span data-stu-id="b8f74-103">The managed coclass wrapper class 'class' for interface 'interface' cannot be found (are you missing an assembly reference?)</span></span>  
  
 <span data-ttu-id="b8f74-104">Bir arabirimden bir COM nesnesi örneklemek için girişimde bulunuldu.</span><span class="sxs-lookup"><span data-stu-id="b8f74-104">An attempt was made to instantiate a COM object from an interface.</span></span> <span data-ttu-id="b8f74-105">Arabiriminin **ComImport** ve `CoClass` öznitelikler, ancak derleyici için verilen türü bulunamıyor `CoClass` özniteliği.</span><span class="sxs-lookup"><span data-stu-id="b8f74-105">The interface has the **ComImport** and `CoClass` attributes, but the compiler cannot find the type given for the `CoClass` attribute.</span></span>  
  
 <span data-ttu-id="b8f74-106">Bu hatayı gidermek için şunlardan birini deneyebilirsiniz:</span><span class="sxs-lookup"><span data-stu-id="b8f74-106">To resolve this error, you can try one of the following:</span></span>  
  
- <span data-ttu-id="b8f74-107">(Çoğu zaman arabirim ve coclass'ı aynı bütünleştirilmiş kodda olmalıdır) coclass'ı içeren derlemeye bir başvuru ekleyin.</span><span class="sxs-lookup"><span data-stu-id="b8f74-107">Add a reference to the assembly that has the coclass (most of the time the interface and coclass should be in the same assembly).</span></span> <span data-ttu-id="b8f74-108">Bkz: [/reference](../../csharp/language-reference/compiler-options/reference-compiler-option.md) veya [başvuru iletişim kutusunu](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) bilgi.</span><span class="sxs-lookup"><span data-stu-id="b8f74-108">See [/reference](../../csharp/language-reference/compiler-options/reference-compiler-option.md) or [Add Reference Dialog Box](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) for information.</span></span>  
  
- <span data-ttu-id="b8f74-109">Düzeltme `CoClass` arabirimde özniteliği.</span><span class="sxs-lookup"><span data-stu-id="b8f74-109">Fix the `CoClass` attribute on the interface.</span></span>  
  
 <span data-ttu-id="b8f74-110">Aşağıdaki örnek doğru kullanımını gösterir **ComImportAttribute**:</span><span class="sxs-lookup"><span data-stu-id="b8f74-110">The following sample demonstrates correct usage of **CoClassAttribute**:</span></span>  
  
```csharp  
// CS1613.cs  
using System;  
using System.Runtime.InteropServices;  
  
[Guid("1FFD7840-E82D-4268-875C-80A160C23296")]  
[ComImport()]  
[CoClass(typeof(A))]  
public interface IA{}  
public class A : IA {}  
  
public class AA  
{  
   public static void Main()  
   {  
      IA i;  
      i = new IA(); // This is equivalent to new A().  
                    // because of the CoClass attribute on IA  
   }  
}  
```
