# <a name="compiler-error-cs0171"></a><span data-ttu-id="b3741-102">Derleyici Hatası CS0171</span><span class="sxs-lookup"><span data-stu-id="b3741-102">Compiler Error CS0171</span></span>
<span data-ttu-id="b3741-103">Yedekleme alanını otomatik olarak uygulanan 'name' özelliği için Denetim çağırana döndürülmeden önce tamamen atanmalıdır.</span><span class="sxs-lookup"><span data-stu-id="b3741-103">Backing field for automatically implemented property 'name' must be fully assigned before control is returned to the caller.</span></span> <span data-ttu-id="b3741-104">Parametresiz bir oluşturucu başlatıcısından çağırmayı düşünün.</span><span class="sxs-lookup"><span data-stu-id="b3741-104">Consider calling the parameterless constructor from a constructor initializer.</span></span>  
  
 <span data-ttu-id="b3741-105">Bir oluşturucuda bir [yapı](../../csharp/language-reference/keywords/struct.md) yapı içindeki tüm alanlarda başlatmanız gerekir.</span><span class="sxs-lookup"><span data-stu-id="b3741-105">A constructor in a [struct](../../csharp/language-reference/keywords/struct.md) must initialize all fields in the struct.</span></span> <span data-ttu-id="b3741-106">Daha fazla bilgi için [oluşturucular](../../csharp/programming-guide/classes-and-structs/constructors.md).</span><span class="sxs-lookup"><span data-stu-id="b3741-106">For more information, see [Constructors](../../csharp/programming-guide/classes-and-structs/constructors.md).</span></span>  
  
 <span data-ttu-id="b3741-107">Aşağıdaki örnek, CS0171 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="b3741-107">The following sample generates CS0171:</span></span>  
  
```csharp  
// CS0171.cs  
struct MyStruct  
{  
   MyStruct(int initField)   // CS0171  
   {  
      // i = initField;      // uncomment this line to resolve this error  
   }  
   public int i;  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      MyStruct aStruct = new MyStruct();  
   }  
}  
```
