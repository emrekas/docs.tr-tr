# <a name="compiler-error-cs0416"></a><span data-ttu-id="dfbce-102">Derleyici Hatası CS0416</span><span class="sxs-lookup"><span data-stu-id="dfbce-102">Compiler Error CS0416</span></span>
<span data-ttu-id="dfbce-103">'tür parametresi': öznitelik bağımsız değişkeni tür parametreleri kullanamaz</span><span class="sxs-lookup"><span data-stu-id="dfbce-103">'type parameter': an attribute argument cannot use type parameters</span></span>  
  
 <span data-ttu-id="dfbce-104">Bir tür parametresine izin verilmiyor bir öznitelik bağımsız kullanıldı.</span><span class="sxs-lookup"><span data-stu-id="dfbce-104">A type parameter was used as an attribute argument, which is not allowed.</span></span> <span data-ttu-id="dfbce-105">Genel olmayan bir tür kullanın.</span><span class="sxs-lookup"><span data-stu-id="dfbce-105">Use a non-generic type.</span></span>  
  
 <span data-ttu-id="dfbce-106">Aşağıdaki örnek, CS0416 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="dfbce-106">The following sample generates CS0416:</span></span>  
  
```csharp  
// CS0416.cs

{  
   public MyAttribute(System.Type t)  
   {  
   }  
}

{  
  
   [MyAttribute(typeof(G<T>))]  // CS0416  
   public void F()  
   {  
   }  
  
}  
```
