## <a name="example"></a><span data-ttu-id="f6842-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="f6842-106">Example</span></span>

```csharp

{  
    public class b : Exception  
    {  
    }  
  
    public class a  
    {  
        public static void Main()  
        {  
            try  
            {  
            }  
  
            catch   // CS1017, must be last catch

            }  
  
            catch(b)  
            {  
                throw;  
            }  
        }  
    }  
}  
```
