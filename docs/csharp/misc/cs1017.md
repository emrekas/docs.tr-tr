# <a name="compiler-error-cs1017"></a>Derleyici Hatası CS1017
Catch yan tümceleri, try deyiminin genel bir catch yan tümcesini izleyemez.  
  
 A `catch` herhangi bir parametre almaz blok, son bir dizi olmalıdır `catch` engeller. Özel durumları hakkında daha fazla bilgi için bkz. [özel durumlar ve özel durum işleme](../../csharp/programming-guide/exceptions/index.md).  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS1017 oluşturur:  
  
```csharp  
// CS1017.cs  
using System;  
  
namespace x  
{  
    public class b : Exception  
    {  
    }  
  
    public class a  
    {  
        public static void Main()  
        {  
            try  
            {  
            }  
  
            catch   // CS1017, must be last catch  
            {  
            }  
  
            catch(b)  
            {  
                throw;  
            }  
        }  
    }  
}  
```
