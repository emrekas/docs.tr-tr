# <a name="compiler-error-cs0013"></a>Derleyici Hatası CS0013
'ModelStore.dll'--dosya meta veriler yazılırken beklenmeyen bir hata ' mantıksal boşluk sol daha fazla kullanıcı dizeleri oluşturmak için.  
  
 .NET Framework ortak dil çalışma zamanı meta verileri yayma değil. Yolun doğru olduğundan ve diskin dolu olmadığından emin olun. Sorun devam ederse, Visual Studio ve/veya .NET Framework yeniden yükleyin veya onarın gerekebilir.
