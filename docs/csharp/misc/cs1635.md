# <a name="compiler-warning-level-1-cs1635"></a><span data-ttu-id="8fddd-102">Derleyici Uyarısı (düzey 1) CS1635</span><span class="sxs-lookup"><span data-stu-id="8fddd-102">Compiler Warning (level 1) CS1635</span></span>
<span data-ttu-id="8fddd-103">Bu genel olarak devre dışı bırakıldığından uyarı uyarı 'code' geri yüklenemiyor</span><span class="sxs-lookup"><span data-stu-id="8fddd-103">Cannot restore warning 'warning code' because it was disabled globally</span></span>  
  
 <span data-ttu-id="8fddd-104">Kullanırsanız, bu uyarı oluşur **/nowarn** satırı seçeneğini veya tüm derleme birimi için bir uyarı devre dışı bırakma ayarı proje komutu, ancak kullandığınız `#pragma warning restore` bu uyarı geri yüklemeyi denemek için.</span><span class="sxs-lookup"><span data-stu-id="8fddd-104">This warning occurs if you use the **/nowarn** command line option or project setting to disable a warning for the entire compilation unit, but you use `#pragma warning restore` to attempt to restore that warning.</span></span> <span data-ttu-id="8fddd-105">Bu hatayı gidermek için / nowarn komut satırı seçeneği veya proje ayarı kaldırmak veya kaldırmak `#pragma warning restore` , devre dışı bırakma komut satırı veya proje ayarları tüm uyarılar için.</span><span class="sxs-lookup"><span data-stu-id="8fddd-105">To resolve this error, remove the /nowarn command line option or project setting, or remove the `#pragma warning restore` for any warnings you are disabling via the command line or project settings.</span></span> <span data-ttu-id="8fddd-106">Daha fazla bilgi için [#pragma Uyarısı](../../csharp/language-reference/preprocessor-directives/preprocessor-pragma-warning.md) konu.</span><span class="sxs-lookup"><span data-stu-id="8fddd-106">For more information, see the [#pragma warning](../../csharp/language-reference/preprocessor-directives/preprocessor-pragma-warning.md) topic.</span></span>  
  
 <span data-ttu-id="8fddd-107">Aşağıdaki örnek, CS1635 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="8fddd-107">The following sample generates CS1635:</span></span>  
  
```csharp  
// CS1635.cs  
// compile with: /w:1 /nowarn:162  
  
enum MyEnum {one=1,two=2,three=3};  
  
class MyClass  
{  
    public static void Main()  
    {  
#pragma warning disable 162  
  
    if (MyEnum.three == MyEnum.two)  
        System.Console.WriteLine("Duplicate");  
  
#pragma warning restore 162  
    }  
}  
```
