# <a name="compiler-error-cs1949"></a><span data-ttu-id="acea1-102">Derleyici Hatası CS1949</span><span class="sxs-lookup"><span data-stu-id="acea1-102">Compiler Error CS1949</span></span>
<span data-ttu-id="acea1-103">'Var' bağlamsal anahtar sözcüğü bir aralık değişkeni bildiriminde kullanılamaz.</span><span class="sxs-lookup"><span data-stu-id="acea1-103">The contextual keyword 'var' cannot be used in a range variable declaration.</span></span>

 <span data-ttu-id="acea1-109">Aşağıdaki örnek, CS1949 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="acea1-109">The following example generates CS1949:</span></span>
