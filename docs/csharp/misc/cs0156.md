# <a name="compiler-error-cs0156"></a><span data-ttu-id="14b5c-102">Derleyici Hatası CS0156</span><span class="sxs-lookup"><span data-stu-id="14b5c-102">Compiler Error CS0156</span></span>
<span data-ttu-id="14b5c-103">Bağımsız değişken içermeyen bir throw deyimi içinde izin verilmiyor bir finally yan tümcesi catch en yakın kapsayan iç içe geçmiş yan tümcesi</span><span class="sxs-lookup"><span data-stu-id="14b5c-103">A throw statement with no arguments is not allowed in a finally clause that is nested inside the nearest enclosing catch clause</span></span>  
  
 <span data-ttu-id="14b5c-104">A [throw](../../csharp/language-reference/keywords/throw.md) parametresiz deyimi içinde yalnızca görünebilir bir **catch** parametre almayan yan tümcesi.</span><span class="sxs-lookup"><span data-stu-id="14b5c-104">A [throw](../../csharp/language-reference/keywords/throw.md) statement with no parameters can only appear in a **catch** clause that takes no parameters.</span></span>  
  
 <span data-ttu-id="14b5c-105">Daha fazla bilgi için [özel durum işleme deyimleri](../../csharp/language-reference/keywords/exception-handling-statements.md) ve [özel durumlar ve özel durum işleme](../../csharp/programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="14b5c-105">For more information, see [Exception Handling Statements](../../csharp/language-reference/keywords/exception-handling-statements.md) and [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="14b5c-106">Aşağıdaki örnek, CS0156 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="14b5c-106">The following sample generates CS0156:</span></span>  
  
```csharp  
// CS0156.cs  
using System;  
  
namespace MyNamespace  
{  
   public class MyClass2 : Exception  
   {  
   }  
  
   public class MyClass  
   {  
      public static void Main()  
      {  
         try  
         {  
            throw;   // CS0156  
         }  
  
         catch(MyClass2)  
         {  
            throw;   // this throw is valid  
         }  
      }  
   }  
}  
```
