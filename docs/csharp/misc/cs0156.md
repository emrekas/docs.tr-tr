# <a name="compiler-error-cs0156"></a><span data-ttu-id="9e3f7-102">Derleyici hatası CS0156</span><span class="sxs-lookup"><span data-stu-id="9e3f7-102">Compiler Error CS0156</span></span>
<span data-ttu-id="9e3f7-103">Bağımsız değişken içermeyen bir throw deyimi, en yakın kapsayan catch yan tümcesinin içinde iç içe geçmiş bir finally yan tümcesinde kullanılamaz</span><span class="sxs-lookup"><span data-stu-id="9e3f7-103">A throw statement with no arguments is not allowed in a finally clause that is nested inside the nearest enclosing catch clause</span></span>  
  
 <span data-ttu-id="9e3f7-104">Parametreleri olmayan bir [throw](../language-reference/keywords/throw.md) deyimi yalnızca parametre içermeyen bir **catch** yan tümcesinde yer alabilir.</span><span class="sxs-lookup"><span data-stu-id="9e3f7-104">A [throw](../language-reference/keywords/throw.md) statement with no parameters can only appear in a **catch** clause that takes no parameters.</span></span>  
  
 <span data-ttu-id="9e3f7-105">Daha fazla bilgi için bkz. [özel durumlar ve özel durum işleme](../programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="9e3f7-105">For more information, see [Exceptions and Exception Handling](../programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="9e3f7-106">Aşağıdaki örnek CS0156 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="9e3f7-106">The following sample generates CS0156:</span></span>  
  
```csharp  
// CS0156.cs  
using System;  
  
namespace MyNamespace  
{  
   public class MyClass2 : Exception  
   {  
   }  
  
   public class MyClass  
   {  
      public static void Main()  
      {  
         try  
         {  
            throw;   // CS0156  
         }  
  
         catch(MyClass2)  
         {  
            throw;   // this throw is valid  
         }  
      }  
   }  
}  
```
