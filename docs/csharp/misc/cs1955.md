# <a name="compiler-error-cs1955"></a>Derleyici Hatası CS1955
'Name' çağrılamayan üyesi yöntem gibi kullanılamaz.  
  
 Yalnızca yöntemler ve temsilciler çağrılabilir. Bu hata, bir yöntem dışında bir şey çağrı yapma veya temsilci seçme için boş parantez kullanmaya çalıştığınızda oluşturulur.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Parantezler ifadeden kaldırın.  
  
## <a name="example"></a>Örnek  
 Yöntem çağrısı işlecini kullanarak bir alan ve bir özellik çağırmak kod çalıştığı için aşağıdaki kod CS1955 oluşturur [()](../../csharp/language-reference/operators/invocation-operator.md). Bir alan veya özellik çağrısı yapamazsınız, ancak değerin depolar üye erişimi işleci kullanarak erişebilirsiniz ( [.](../../csharp/language-reference/operators/member-access-operator.md) ).  
  
```csharp  
// cs1955.cs  
class A  
{  
    public int x = 0;  
    public int X  
    {  
        get { return x; }  
        set { x = value; }  
    }  
}  
  
class Test  
{  
    static int Main()  
    {  
        A a = new A();  
        a.x(); // CS1955  
        a.X(); // CS1955  
        // Try this line instead:  
        // int num = a.x;  
    }  
}  
```
