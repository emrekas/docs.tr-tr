## <a name="example"></a><span data-ttu-id="bfc88-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="bfc88-108">Example</span></span>

<span data-ttu-id="bfc88-109">Kod kullanarak bir alan ve bir özellik çağrılacak çalıştığı için aşağıdaki kod CS1955 oluşturur [çağırma işleci `()` ](../language-reference/operators/member-access-operators.md#invocation-operator-).</span><span class="sxs-lookup"><span data-stu-id="bfc88-109">The following code generates CS1955 because the code is trying to invoke a field and a property by using the [invocation operator `()`](../language-reference/operators/member-access-operators.md#invocation-operator-).</span></span> <span data-ttu-id="bfc88-110">Bir alan veya özellik çağrısı yapamazsınız.</span><span class="sxs-lookup"><span data-stu-id="bfc88-110">You cannot call a field or a property.</span></span> <span data-ttu-id="bfc88-111">Kullanım [üye erişimi işleci `.` ](../language-reference/operators/member-access-operators.md#member-access-operator-) depoladığı değere erişmek için.</span><span class="sxs-lookup"><span data-stu-id="bfc88-111">Use the [member access operator `.`](../language-reference/operators/member-access-operators.md#member-access-operator-) to access the value it stores.</span></span>
  
```csharp  
// cs1955.cs  
class A  
{  
    public int x = 0;  
    public int X  
    {  
        get { return x; }  
        set { x = value; }  
    }  
}  
  
class Test  
{  
    static int Main()  
    {  
        A a = new A();  
        a.x(); // CS1955  
        a.X(); // CS1955  
        // Try this line instead:  
        // int num = a.x;  
    }  
}  
```
