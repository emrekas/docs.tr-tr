# <a name="compiler-warning-level-2-cs0162"></a><span data-ttu-id="9713a-102">Derleyici Uyarısı (Düzey 2) CS0162</span><span class="sxs-lookup"><span data-stu-id="9713a-102">Compiler Warning (level 2) CS0162</span></span>
<span data-ttu-id="9713a-103">Erişilemeyen kod algılandı</span><span class="sxs-lookup"><span data-stu-id="9713a-103">Unreachable code detected</span></span>  
  
 <span data-ttu-id="9713a-104">Derleyici, hiçbir zaman yürütülecek kodu algıladı.</span><span class="sxs-lookup"><span data-stu-id="9713a-104">The compiler detected code that will never be executed.</span></span>  
  
 <span data-ttu-id="9713a-105">Aşağıdaki örnek, CS0162 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="9713a-105">The following sample generates CS0162:</span></span>  
  
```csharp  
// CS0162.cs  
// compile with: /W:2  
public class A  
{  
   public static void Main()  
   {  
      goto lab1;  
      {  
         // The following statements cannot be reached:  
         int i = 9;   // CS0162   
         i++;  
      }  
      lab1:  
      {  
      }  
   }  
}  
```
