# <a name="compiler-error-cs0529"></a><span data-ttu-id="da928-102">Derleyici Hatası CS0529</span><span class="sxs-lookup"><span data-stu-id="da928-102">Compiler Error CS0529</span></span>
<span data-ttu-id="da928-103">Devralınan arabirimi 'arabirimi1', 'interface2' arabirim hiyerarşisinde döngüye neden oluyor</span><span class="sxs-lookup"><span data-stu-id="da928-103">Inherited interface 'interface1' causes a cycle in the interface hierarchy of 'interface2'</span></span>  
  
 <span data-ttu-id="da928-104">Devralma listesini bir [arabirimi](../../csharp/language-reference/keywords/interface.md) kendisine doğrudan veya dolaylı bir başvuru içerir.</span><span class="sxs-lookup"><span data-stu-id="da928-104">The inheritance list for an [interface](../../csharp/language-reference/keywords/interface.md) includes a direct or indirect reference to itself.</span></span> <span data-ttu-id="da928-105">Bir arabirimi kendi kendinden devralamaz.</span><span class="sxs-lookup"><span data-stu-id="da928-105">An interface cannot inherit from itself.</span></span>  
  
 <span data-ttu-id="da928-106">Aşağıdaki örnek, CS0529 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="da928-106">The following sample generates CS0529:</span></span>  
  
```csharp  
// CS0529.cs  
namespace x  
{  
   public interface a  
   {  
   }  
  
   public interface b : a, c  
   {  
   }  
  
   public interface c : b   // CS0529, b inherits from c  
   {  
   }  
}  
```
