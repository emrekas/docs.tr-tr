<span data-ttu-id="61118-103">Devralınan ' arabirimi1 ' arabirimi, ' interface2 ' arabirim hiyerarşisinde bir döngüye neden oluyor</span><span class="sxs-lookup"><span data-stu-id="61118-103">Inherited interface 'interface1' causes a cycle in the interface hierarchy of 'interface2'</span></span>  
  
 <span data-ttu-id="61118-104">Bir [arabirimin](../language-reference/keywords/interface.md) devralma listesi, kendine doğrudan veya dolaylı bir başvuru içerir.</span><span class="sxs-lookup"><span data-stu-id="61118-104">The inheritance list for an [interface](../language-reference/keywords/interface.md) includes a direct or indirect reference to itself.</span></span> <span data-ttu-id="61118-105">Arabirim kendisinden devralma yapılamaz.</span><span class="sxs-lookup"><span data-stu-id="61118-105">An interface cannot inherit from itself.</span></span>  
  
 <span data-ttu-id="61118-106">Aşağıdaki örnek CS0529 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="61118-106">The following sample generates CS0529:</span></span>  
  
```csharp  
// CS0529.cs

{  
   public interface a  
   {  
   }

   {  
   }

```
