# <a name="compiler-error-cs1622"></a>Derleyici Hatası CS1622
Bir yineleyiciden değer döndüremez. Bir değer döndürmek için yield return deyimini kullanın veya yinelemeyi sonlandırmak için yield break.  
  
 Bir yineleyici return deyiminin yerine yield deyimi bir değer döndüren özel bir işlevdir. Daha fazla bilgi için **yineleyiciler**.  
  
 Aşağıdaki örnek, CS1622 oluşturur:  
  
```csharp  
// CS1622.cs  
// compile with: /target:library  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      return (IEnumerator) this;  // CS1622  
      yield return this;   // OK  
   }  
}  
```
