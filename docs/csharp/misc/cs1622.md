# <a name="compiler-error-cs1622"></a><span data-ttu-id="e487e-102">Derleyici Hatası CS1622</span><span class="sxs-lookup"><span data-stu-id="e487e-102">Compiler Error CS1622</span></span>
<span data-ttu-id="e487e-103">Bir yineleyiciden değer döndüremez.</span><span class="sxs-lookup"><span data-stu-id="e487e-103">Cannot return a value from an iterator.</span></span> <span data-ttu-id="e487e-104">Bir değer döndürmek için yield return deyimini kullanın veya yinelemeyi sonlandırmak için yield break.</span><span class="sxs-lookup"><span data-stu-id="e487e-104">Use the yield return statement to return a value, or yield break to end the iteration.</span></span>  
  
 <span data-ttu-id="e487e-105">Bir yineleyici return deyiminin yerine yield deyimi bir değer döndüren özel bir işlevdir.</span><span class="sxs-lookup"><span data-stu-id="e487e-105">An iterator is a special function that returns a value via the yield statement rather than the return statement.</span></span> <span data-ttu-id="e487e-106">Daha fazla bilgi için **yineleyiciler**.</span><span class="sxs-lookup"><span data-stu-id="e487e-106">For more information, see **iterators**.</span></span>  
  
 <span data-ttu-id="e487e-107">Aşağıdaki örnek, CS1622 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="e487e-107">The following sample generates CS1622:</span></span>  
  
```csharp  
// CS1622.cs  
// compile with: /target:library  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      return (IEnumerator) this;  // CS1622  
      yield return this;   // OK  
   }  
}  
```
