# <a name="compiler-error-cs0527"></a>Derleyici hatası CS0527
Arabirim listesindeki ' Type ' türü bir arabirim değil  
  
 Bir [yapının](../language-reference/keywords/struct.md) veya [arabirimin](../language-reference/keywords/interface.md) başka bir arabirimden devralması, ancak başka bir türden olmaması mümkündür.  
  
 Aşağıdaki örnek CS0527 oluşturur:  
  
```csharp  
// CS0527.cs  
// compile with: /target:library  
public struct clx : int {}   // CS0527 int not an interface  
```
