<span data-ttu-id="13f47-103">Verilen ifadenin adresi alınamaz</span><span class="sxs-lookup"><span data-stu-id="13f47-103">Cannot take the address of the given expression</span></span>  
  
 <span data-ttu-id="13f47-104">Alanların adresini, yerel değişkenleri ve işaretçi yöneltme 'sini alabilirsiniz, ancak örneğin iki yerel değişkenin toplamının adresini kullanamazsınız.</span><span class="sxs-lookup"><span data-stu-id="13f47-104">You can take the address of fields, local variables, and indirection of pointers, but you cannot take, for example, the address of the sum of two local variables.</span></span> <span data-ttu-id="13f47-105">Daha fazla bilgi için bkz. [güvenli olmayan kod ve işaretçiler](../programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="13f47-105">For more information, see [Unsafe Code and Pointers](../programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="13f47-106">Aşağıdaki örnek CS0211 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="13f47-106">The following sample generates CS0211:</span></span>  
  
```csharp  
// CS0211.cs  
// compile with: /unsafe  
  
public class MyClass  
{  
   unsafe public void M()  
   {  
      int a = 0, b = 0;  
      int *i = &(a + b);   // CS0211, the addition of two local variables  
      // try the following line instead  
      // int *i = &a;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
