# <a name="compiler-error-cs0306"></a>Derleyici Hatası CS0306
'type' türü bir tür bağımsız değişkeni kullanılamaz  
  
 Bir tür parametresi kullanılan türüne izin verilmiyor. Türü bir işaretçi türü olduğu için bu olabilir.  
  
 Aşağıdaki örnek, CS0306 oluşturur:  
  
```csharp  
// CS0306.cs  
class C<T>  
{  
}  
  
class M  
{  
    // CS0306 – int* not allowed as a type parameter  
     C<int*> f;  
}  
```
