 <span data-ttu-id="bbb00-104">Geçirdiğiniz herhangi bir bağımsız değişken <xref:System.Diagnostics.ConditionalAttribute> veya <xref:System.Runtime.CompilerServices.IndexerNameAttribute> öznitelikleri geçerli bir tanımlayıcı olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="bbb00-104">Any argument that you pass to the <xref:System.Diagnostics.ConditionalAttribute> or <xref:System.Runtime.CompilerServices.IndexerNameAttribute> attributes must be a valid identifier.</span></span> <span data-ttu-id="bbb00-105">Bunun anlamı olan gibi karakterler içeremez "+" tanımlayıcıları ortaya çıktığında, geçersizdir.</span><span class="sxs-lookup"><span data-stu-id="bbb00-105">This means that it may not contain characters such as "+" that are illegal when they occur in identifiers.</span></span>  
  
## <a name="example"></a><span data-ttu-id="bbb00-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="bbb00-106">Example</span></span>  
 <span data-ttu-id="bbb00-107">Bu örnekte, CS0633 kullanarak gösterilmektedir <xref:System.Diagnostics.ConditionalAttribute>.</span><span class="sxs-lookup"><span data-stu-id="bbb00-107">This example illustrates CS0633 using the <xref:System.Diagnostics.ConditionalAttribute>.</span></span> <span data-ttu-id="bbb00-108">Aşağıdaki örnek, CS0633 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="bbb00-108">The following sample generates CS0633.</span></span>  
  
```csharp  
// CS0633a.cs  
#define DEBUG  
using System.Diagnostics;  
public class Test  
{  
   [Conditional("DEB+UG")]   // CS0633  
   // try the following line instead  
   // [Conditional("DEBUG")]  
   public static void Main() { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="bbb00-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="bbb00-109">Example</span></span>  
 <span data-ttu-id="bbb00-110">Bu örnekte, CS0633 kullanarak gösterilmektedir <xref:System.Runtime.CompilerServices.IndexerNameAttribute>.</span><span class="sxs-lookup"><span data-stu-id="bbb00-110">This example illustrates CS0633 using the <xref:System.Runtime.CompilerServices.IndexerNameAttribute>.</span></span>  
  
```csharp  
// CS0633b.cs  
// compile with: /target:module  
#define DEBUG  
using System.Runtime.CompilerServices;  
public class Test  
{  
   [IndexerName("Invalid+Identifier")]   // CS0633  
   // try the following line instead  
   // [IndexerName("DEBUG")]  
   public int this[int i]   
   {   
      get { return i; }   
   }  
}  
```
