## <a name="example"></a><span data-ttu-id="9fa90-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="9fa90-109">Example</span></span>  
 <span data-ttu-id="9fa90-110">Aşağıdaki örnek, CS1113 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="9fa90-110">The following example generates CS1113:</span></span>  
  
```csharp

using System;  
public static class Extensions  
{  
    public static S ExtMethod(this S s)  
    {  
        return s;  
    }  
}

{  
}

{  
    static int Main()  
    {

    }
```  
