# <a name="compiler-error-cs1559"></a>Derleyici Hatası CS1559
İçeri aktarıldığından Main yöntemi için 'object' kullanamazsınız  
  
 Geçersiz bir sınıf için belirtilen [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) derleyici seçeneği; sınıf için bir konum olarak kullanılamaz [ana](../../csharp/programming-guide/main-and-command-args/index.md) yöntemi.
