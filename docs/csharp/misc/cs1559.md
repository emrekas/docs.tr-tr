# <a name="compiler-error-cs1559"></a>Derleyici hatası CS1559
İçeri aktarıldığından Main yöntemi için ' Object ' kullanılamaz  
  
 [/Main](../language-reference/compiler-options/main-compiler-option.md) derleyici seçeneği için geçersiz bir sınıf belirtildi; sınıf, [Main](../programming-guide/main-and-command-args/index.md) yöntemi için bir konum olarak kullanılamaz.
