<span data-ttu-id="76d39-103">Bildirim geçerli değil; Kullan '\<tür > işleç op (...' Bunun yerine</span><span class="sxs-lookup"><span data-stu-id="76d39-103">Declaration is not valid; use '\<type> operator op (...' instead</span></span>  
  
 <span data-ttu-id="76d39-104">Kullanıcı tanımlı için dönüş türü [işleci](../../csharp/language-reference/keywords/operator.md) anahtar sözcüğü işlecinden önce yer almalıdır.</span><span class="sxs-lookup"><span data-stu-id="76d39-104">The return type for a user-defined [operator](../../csharp/language-reference/keywords/operator.md) must appear before the keyword operator.</span></span>

```csharp

   // try the following line instead

   }  
  
   public static void Main()  
   {  
   }  
}  
```
