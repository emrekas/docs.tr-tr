# <a name="compiler-error-cs1554"></a><span data-ttu-id="67885-102">Derleyici hatası CS1554</span><span class="sxs-lookup"><span data-stu-id="67885-102">Compiler Error CS1554</span></span>

<span data-ttu-id="67885-104">[Aşırı yüklenmiş bir işlecin](../language-reference/operators/operator-overloading.md) dönüş türü, `operator` anahtar sözcüğünden önce gelmelidir.</span><span class="sxs-lookup"><span data-stu-id="67885-104">The return type of an [overloaded operator](../language-reference/operators/operator-overloading.md) must appear before the `operator` keyword.</span></span>

   }
