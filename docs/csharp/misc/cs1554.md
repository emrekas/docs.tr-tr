# <a name="compiler-error-cs1554"></a><span data-ttu-id="52094-102">Derleyici Hatası CS1554</span><span class="sxs-lookup"><span data-stu-id="52094-102">Compiler Error CS1554</span></span>
<span data-ttu-id="52094-103">Bildirim geçerli değil; Kullan '\<tür > işleç op (...' Bunun yerine</span><span class="sxs-lookup"><span data-stu-id="52094-103">Declaration is not valid; use '\<type> operator op (...' instead</span></span>  
  
<span data-ttu-id="52094-104">Dönüş türü bir [işleci aşırı](../../csharp/language-reference/operators/operator-overloading.md) önce görünmelidir `operator` anahtar sözcüğü.</span><span class="sxs-lookup"><span data-stu-id="52094-104">The return type of an [overloaded operator](../../csharp/language-reference/operators/operator-overloading.md) must appear before the `operator` keyword.</span></span>
  
<span data-ttu-id="52094-105">Aşağıdaki örnek, CS1554 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="52094-105">The following sample generates CS1554:</span></span>  
  
```csharp  
// CS1554.cs  
class MyClass  
{  
   public static operator ++ MyClass (MyClass f)    // CS1554  
   // try the following line instead  
   // public static MyClass operator ++ (MyClass f)  
   {  
      return new MyClass ();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
