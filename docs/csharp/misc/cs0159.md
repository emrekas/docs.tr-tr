# <a name="compiler-error-cs0159"></a><span data-ttu-id="103c0-102">Derleyici Hatası CS0159</span><span class="sxs-lookup"><span data-stu-id="103c0-102">Compiler Error CS0159</span></span>
<span data-ttu-id="103c0-103">Böyle bir etiket yoktur 'etiketi' goto deyiminin kapsamı içinde</span><span class="sxs-lookup"><span data-stu-id="103c0-103">No such label 'label' within the scope of the goto statement</span></span>  
  
 <span data-ttu-id="103c0-104">Tarafından başvurulan etiket [goto](../../csharp/language-reference/keywords/goto.md) deyimi bulunamadı kapsamında `goto` deyimi.</span><span class="sxs-lookup"><span data-stu-id="103c0-104">The label referenced by the [goto](../../csharp/language-reference/keywords/goto.md) statement could not be found within the scope of the `goto` statement.</span></span>  
  
 <span data-ttu-id="103c0-105">Aşağıdaki örnek, CS0159 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="103c0-105">The following sample generates CS0159:</span></span>  
  
```csharp  
// CS0159.cs  
public class Class1  
{  
   public static void Main()  
   {  
      int i = 0;  
  
      switch (i)  
      {  
         case 1:  
            goto case 3;   // CS0159, case 3 label does not exist  
         case 2:  
            break;  
      }  
      goto NOWHERE;   // CS0159, NOWHERE label does not exist  
   }  
}  
```
