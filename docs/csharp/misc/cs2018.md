# <a name="compiler-error-cs2018"></a>Derleyici Hatası CS2018
Dosya 'cscmsgs.dll' iletileri bulunamıyor  
  
 Derleyicinin hata ve uyarı iletilerini içeren .dll dosyası bulunamadı. Bu dosya, bir derleyici destek dosyaları ile aynı dizinde bulunmalıdır.
