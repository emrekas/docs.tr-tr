# <a name="compiler-error-cs1011"></a>Derleyici Hatası CS1011
Boş karakter sabiti  
  
 A [char](../../csharp/language-reference/keywords/char.md) bildirildi ve null değerine başlatıldı. Başlatma bir `char` bir karakter belirtmeniz gerekir.  
  
 Aşağıdaki örnek, CS1011 oluşturur:  
  
```csharp  
// CS1011.cs  
class Sample  
{  
   public char CharField = '';   // CS1011  
}  
```
