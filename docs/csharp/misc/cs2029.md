# <a name="compiler-warning-level-1-cs2029"></a>Derleyici Uyarısı (düzey 1) CS2029
Geçersiz değeri '/ define'; 'identifier' geçerli bir tanımlayıcı değil  
  
 Bu hata ortaya çıkar kullanılır değer [/ define](../../csharp/language-reference/compiler-options/define-compiler-option.md) seçeneği bazı geçersiz karakterler içeriyor.  
  
 Bu uyarı tarafından gizlenen olamaz [/nowarn](../../csharp/language-reference/compiler-options/nowarn-compiler-option.md) seçeneği.
