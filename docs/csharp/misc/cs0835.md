# <a name="compiler-error-cs0835"></a><span data-ttu-id="5ef79-102">Derleyici Hatası CS0835</span><span class="sxs-lookup"><span data-stu-id="5ef79-102">Compiler Error CS0835</span></span>
<span data-ttu-id="5ef79-103">Lambda, tür bağımsız değişkeni 'type' bir temsilci türü değil. bir ifade ağacına dönüştürülemiyor.</span><span class="sxs-lookup"><span data-stu-id="5ef79-103">Cannot convert lambda to an expression tree whose type argument 'type' is not a delegate type.</span></span>  
  
 <span data-ttu-id="5ef79-104">İfade ağacı, bir lambda ifadesi ifade ağacına dönüştürülürse, bağımsız değişkeni için bir temsilci türü olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="5ef79-104">If a lambda expression is converted to an expression tree, the expression tree must have a delegate type for its argument.</span></span> <span data-ttu-id="5ef79-105">Ayrıca, lambda ifadesi, temsilci türüne dönüştürülebilir olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="5ef79-105">Furthermore, the lambda expression must be convertible to the delegate type.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="5ef79-106">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="5ef79-106">To correct this error</span></span>  
  
1. <span data-ttu-id="5ef79-107">Tür parametresi ile değiştirme `int` , örneğin bir temsilci türüne `Func<int,int>`.</span><span class="sxs-lookup"><span data-stu-id="5ef79-107">Change the type parameter from `int` to a delegate type, for example `Func<int,int>`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5ef79-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="5ef79-108">Example</span></span>  
 <span data-ttu-id="5ef79-109">Aşağıdaki örnek, CS0835 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="5ef79-109">The following example generates CS0835:</span></span>  
  
```csharp  
// cs0835.cs  
using System;  
using System.Linq;  
using System.Linq.Expressions;  
  
public class C  
{  
    public static int Main()  
    {  
        Expression<int> e = x => x + 1; // CS0835  
  
        // Try the following line instead.  
       // Expression<Func<int,int>> e2 = x => x + 1;  
  
        return 1;  
    }  
}  
```
