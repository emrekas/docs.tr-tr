# <a name="compiler-error-cs0143"></a><span data-ttu-id="3bc10-102">Derleyici Hatası CS0143</span><span class="sxs-lookup"><span data-stu-id="3bc10-102">Compiler Error CS0143</span></span>

 <span data-ttu-id="3bc10-104">Uygun hiçbir oluşturucu yok.</span><span class="sxs-lookup"><span data-stu-id="3bc10-104">There is no appropriate constructor available.</span></span> <span data-ttu-id="3bc10-105">Bu yalnızca bir değer atayarak başlatılır yerleşik sayısal değer türleri için geçerlidir.</span><span class="sxs-lookup"><span data-stu-id="3bc10-105">This is the case for built-in numeric value types, which are initialized simply by assigning a value to them.</span></span>

```csharp  
   static public void Main ()

```
