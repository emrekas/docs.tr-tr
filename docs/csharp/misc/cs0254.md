 <span data-ttu-id="2af3c-104">[Sabit](../language-reference/keywords/fixed-statement.md) ifadenin sağ tarafı bir tür dönüştürme kullanamaz.</span><span class="sxs-lookup"><span data-stu-id="2af3c-104">The right side of a [fixed](../language-reference/keywords/fixed-statement.md) expression may not use a cast.</span></span> <span data-ttu-id="2af3c-105">Daha fazla bilgi için bkz. [güvenli olmayan kod ve işaretçiler](../programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="2af3c-105">For more information, see [Unsafe Code and Pointers](../programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="2af3c-106">Aşağıdaki örnek CS0254 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="2af3c-106">The following sample generates CS0254:</span></span>  
  
```csharp  
// CS0254.cs  
// compile with: /unsafe  
class Point  
{  
   public uint x, y;  
}  
  
class FixedTest  
{  
   unsafe static void SquarePtrParam (int* p)  
   {  
      *p *= *p;  
   }  
  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
      pt.x = 5;  
      pt.y = 6;  
  
      fixed (int* p = (int*)&pt.x)   // CS0254  
      // try the following line instead  
      // fixed (uint* p = &pt.x)  
      {  
         SquarePtrParam ((int*)p);  
      }  
   }  
}  
```
