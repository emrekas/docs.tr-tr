# <a name="compiler-warning-level-3-cs1717"></a><span data-ttu-id="9c202-102">Derleyici Uyarısı (Düzey 3) CS1717</span><span class="sxs-lookup"><span data-stu-id="9c202-102">Compiler Warning (level 3) CS1717</span></span>
<span data-ttu-id="9c202-103">Atama aynı değişkene yapıldı; başka bir şey atamak istiyordunuz?</span><span class="sxs-lookup"><span data-stu-id="9c202-103">Assignment made to same variable; did you mean to assign something else?</span></span>  
  
 <span data-ttu-id="9c202-104">Bu uyarı, bir değişken kendisine gibi atadığınızda oluşur `a = a`.</span><span class="sxs-lookup"><span data-stu-id="9c202-104">This warning occurs when you assign a variable to itself, such as `a = a`.</span></span>  
  
 <span data-ttu-id="9c202-105">Bazı yaygın hatalar, bu uyarı oluşturabilirsiniz:</span><span class="sxs-lookup"><span data-stu-id="9c202-105">Several common mistakes can generate this warning:</span></span>  
  
- <span data-ttu-id="9c202-106">Yazma `a = a` koşulu olarak bir **varsa** ifadesi gibi `if (a = a)`.</span><span class="sxs-lookup"><span data-stu-id="9c202-106">Writing `a = a` as the condition of an **if** statement, such as `if (a = a)`.</span></span> <span data-ttu-id="9c202-107">Büyük olasılıkla söyleyin belirttiğinizi `if (a == a)`, hangi her zaman true ise daha, bu yazabilmesi için kısaca olarak `if (true)`.</span><span class="sxs-lookup"><span data-stu-id="9c202-107">You probably meant to say `if (a == a)`, which is always true, so you could write this more concisely as `if (true)`.</span></span>  
  
- <span data-ttu-id="9c202-108">Yanlış yazmanız.</span><span class="sxs-lookup"><span data-stu-id="9c202-108">Mistyping.</span></span> <span data-ttu-id="9c202-109">Büyük olasılıkla söyleyin belirttiğinizi `a = b`.</span><span class="sxs-lookup"><span data-stu-id="9c202-109">You probably meant to say `a = b`.</span></span>  
  
- <span data-ttu-id="9c202-110">Parametre alan aynı ada bulunduğu bir oluşturucuda kullanmayan **bu** anahtar sözcüğü: büyük olasılıkla söyleyin belirttiğinizi `this.a = a`.</span><span class="sxs-lookup"><span data-stu-id="9c202-110">In a constructor where the parameter has the same name as the field, not using the **this** keyword: you probably meant to say `this.a = a`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="9c202-111">Örnek</span><span class="sxs-lookup"><span data-stu-id="9c202-111">Example</span></span>  
 <span data-ttu-id="9c202-112">Aşağıdaki örnek, CS1717 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="9c202-112">The following sample generates CS1717.</span></span>

   public static void Main()  
   {
