# <a name="compiler-error-cs0004"></a>Derleyici Hatası CS0004
Uyarı hata olarak değerlendirildi  
  
 İle derlenmiş `/warnaserror` derleyici seçeneği, derleyicinin uyarıları hata olarak oluşturmasına neden olur.
