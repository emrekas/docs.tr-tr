# <a name="compiler-warning-level-1-cs0602"></a>Derleyici Uyarısı (düzey 1) CS0602
' % S'özelliği 'old_feature' kullanımdan kaldırılmıştır. Lütfen bunun yerine 'new_feature' kullanın  
  
 Kodunuzda kullanılan bir dil özelliği (*old_feature*) destek gelecekteki bir sürümde kaldırılmış olabilir, ancak yine de, desteklenir. Bunun yerine, önerilen sözdizimini kullanmanız gerekir (*new_feature*).
