# <a name="compiler-warning-level-1-cs2014"></a>Derleyici Uyarısı (düzey 1) CS2014
Derleyici seçeneği 'eski seçeneği' artık kullanılmıyor, Lütfen bunun yerine 'Yeni seçeneğini' kullanın  
  
 Derleyici seçeneği biçiminde kullanım dışı bırakılmıştır. Bkz: [ C# derleyici seçenekleri](../../csharp/language-reference/compiler-options/index.md) daha fazla bilgi için.
