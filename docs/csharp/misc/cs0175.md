# <a name="compiler-error-cs0175"></a><span data-ttu-id="afb05-102">Derleyici Hatası CS0175</span><span class="sxs-lookup"><span data-stu-id="afb05-102">Compiler Error CS0175</span></span>
<span data-ttu-id="afb05-103">'Base' anahtar sözcüğünün kullanımı bu bağlamda geçerli değil</span><span class="sxs-lookup"><span data-stu-id="afb05-103">Use of keyword 'base' is not valid in this context</span></span>  
  
 <span data-ttu-id="afb05-104">[Temel](../../csharp/language-reference/keywords/base.md) anahtar sözcüğü, temel sınıfın belirli bir üye belirtmek için kullanılmalıdır.</span><span class="sxs-lookup"><span data-stu-id="afb05-104">The [base](../../csharp/language-reference/keywords/base.md) keyword must be used to specify a particular member of the base class.</span></span> <span data-ttu-id="afb05-105">Daha fazla bilgi için [oluşturucular](../../csharp/programming-guide/classes-and-structs/constructors.md).</span><span class="sxs-lookup"><span data-stu-id="afb05-105">For more information, see [Constructors](../../csharp/programming-guide/classes-and-structs/constructors.md).</span></span>  
  
 <span data-ttu-id="afb05-106">Aşağıdaki örnek, CS0175 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="afb05-106">The following sample generates CS0175:</span></span>  
  
```csharp  
// CS0175.cs  
using System;  
class BaseClass  
{  
    public int TestInt = 0;  
}  
  
class MyClass : BaseClass  
{  
    public static void Main()  
    {  
        MyClass aClass = new MyClass();  
        aClass.BaseTest();  
    }  
  
    public void BaseTest()  
    {  
        Console.WriteLine(base); // CS0175  
        // Try the following line instead:  
        // Console.WriteLine(base.TestInt);  
        base = 9;   // CS0175  
  
        // Try the following line instead:  
        // base.TestInt = 9;  
    }  
}  
```
