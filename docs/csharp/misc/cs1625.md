# <a name="compiler-error-cs1625"></a><span data-ttu-id="225f1-102">Derleyici Hatası CS1625</span><span class="sxs-lookup"><span data-stu-id="225f1-102">Compiler Error CS1625</span></span>
<span data-ttu-id="225f1-103">Gövdesinde yield ile dönülemez bir finally yan tümcesi</span><span class="sxs-lookup"><span data-stu-id="225f1-103">Cannot yield in the body of a finally clause</span></span>  
  
 <span data-ttu-id="225f1-104">Yield deyimi gövdesinde izin verilmiyor bir finally yan tümcesi.</span><span class="sxs-lookup"><span data-stu-id="225f1-104">A yield statement is not allowed in the body of a finally clause.</span></span> <span data-ttu-id="225f1-105">Yield deyimi dışında bu hatadan kaçınmak için taşıma finally yan tümcesi.</span><span class="sxs-lookup"><span data-stu-id="225f1-105">To avoid this error, move the yield statement out of the finally clause.</span></span>  
  
 <span data-ttu-id="225f1-106">Aşağıdaki örnek, CS1625 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="225f1-106">The following sample generates CS1625:</span></span>  
  
```csharp  
// CS1625.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
      }  
      finally  
      {  
        yield return this;  // CS1625  
      }  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
