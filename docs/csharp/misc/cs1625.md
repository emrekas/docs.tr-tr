# <a name="compiler-error-cs1625"></a>Derleyici Hatası CS1625
Gövdesinde yield ile dönülemez bir finally yan tümcesi  
  
 Yield deyimi gövdesinde izin verilmiyor bir finally yan tümcesi. Yield deyimi dışında bu hatadan kaçınmak için taşıma finally yan tümcesi.  
  
 Aşağıdaki örnek, CS1625 oluşturur:  
  
```csharp  
// CS1625.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
      }  
      finally  
      {  
        yield return this;  // CS1625  
      }  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
