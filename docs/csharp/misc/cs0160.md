# <a name="compiler-error-cs0160"></a><span data-ttu-id="6921d-102">Derleyici Hatası CS0160</span><span class="sxs-lookup"><span data-stu-id="6921d-102">Compiler Error CS0160</span></span>
<span data-ttu-id="6921d-103">Önceki catch yan tümcesi bunun veya bir süper türün ('type') tüm özel durumları zaten yakalıyor</span><span class="sxs-lookup"><span data-stu-id="6921d-103">A previous catch clause already catches all exceptions of this or of a super type ('type')</span></span>  
  
<span data-ttu-id="6921d-104">Bir dizi [catch](../../csharp/language-reference/keywords/try-catch.md) deyimleri türetme sırasına göre azalan düzende olması gerekir.</span><span class="sxs-lookup"><span data-stu-id="6921d-104">A series of [catch](../../csharp/language-reference/keywords/try-catch.md) statements needs to be in decreasing order of derivation.</span></span> <span data-ttu-id="6921d-105">Örneğin, en çok türetilen nesneler ilk yer almalıdır.</span><span class="sxs-lookup"><span data-stu-id="6921d-105">For example, the most derived objects must appear first.</span></span>
  
 <span data-ttu-id="6921d-106">Daha fazla bilgi için [özel durumlar ve özel durum işleme](../../csharp/programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="6921d-106">For more information, see [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="6921d-107">Aşağıdaki örnek, CS0160 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="6921d-107">The following sample generates CS0160:</span></span>  
  
```csharp  
// CS0160.cs  
public class MyClass2 : System.Exception {}  
public class MyClass  
{  
   public static void Main()  
   {  
      try {}  
  
      catch(System.Exception) {}   // Second-most derived; should be second catch  
      catch(MyClass2) {}   // CS0160  Most derived; should be first catch  
   }  
}  
```
