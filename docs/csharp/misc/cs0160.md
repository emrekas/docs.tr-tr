# <a name="compiler-error-cs0160"></a>Derleyici Hatası CS0160
Önceki catch yan tümcesi bunun veya bir süper türün ('type') tüm özel durumları zaten yakalıyor  
  
Bir dizi [catch](../../csharp/language-reference/keywords/try-catch.md) deyimleri türetme sırasına göre azalan düzende olması gerekir. Örneğin, en çok türetilen nesneler ilk yer almalıdır.
  
 Daha fazla bilgi için [özel durumlar ve özel durum işleme](../../csharp/programming-guide/exceptions/index.md).  
  
 Aşağıdaki örnek, CS0160 oluşturur:  
  
```csharp  
// CS0160.cs  
public class MyClass2 : System.Exception {}  
public class MyClass  
{  
   public static void Main()  
   {  
      try {}  
  
      catch(System.Exception) {}   // Second-most derived; should be second catch  
      catch(MyClass2) {}   // CS0160  Most derived; should be first catch  
   }  
}  
```
