# <a name="compiler-error-cs0160"></a><span data-ttu-id="0d416-102">Derleyici hatası CS0160</span><span class="sxs-lookup"><span data-stu-id="0d416-102">Compiler Error CS0160</span></span>
<span data-ttu-id="0d416-103">Önceki catch yan tümcesi, bu ya da bir süper türün (' Type ') tüm özel durumlarını zaten yakalar</span><span class="sxs-lookup"><span data-stu-id="0d416-103">A previous catch clause already catches all exceptions of this or of a super type ('type')</span></span>  
  
<span data-ttu-id="0d416-104">Bir dizi [catch](../language-reference/keywords/try-catch.md) deyimi, türetme sırasında azalan sırada olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="0d416-104">A series of [catch](../language-reference/keywords/try-catch.md) statements needs to be in decreasing order of derivation.</span></span> <span data-ttu-id="0d416-105">Örneğin, ilk olarak en türetilmiş nesneler görünmelidir.</span><span class="sxs-lookup"><span data-stu-id="0d416-105">For example, the most derived objects must appear first.</span></span>
  
 <span data-ttu-id="0d416-106">Daha fazla bilgi için bkz. [özel durumlar ve özel durum işleme](../programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="0d416-106">For more information, see [Exceptions and Exception Handling](../programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="0d416-107">Aşağıdaki örnek CS0160 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="0d416-107">The following sample generates CS0160:</span></span>  
  
```csharp  
// CS0160.cs  
public class MyClass2 : System.Exception {}  
public class MyClass  
{  
   public static void Main()  
   {  
      try {}  
  
      catch(System.Exception) {}   // Second-most derived; should be second catch  
      catch(MyClass2) {}   // CS0160  Most derived; should be first catch  
   }  
}  
```
