# <a name="compiler-error-cs0729"></a>Derleyici Hatası CS0729
'Type' türü bu derlemede tanımlanır, ancak bunun için bir tür ileticisi belirtilir  
  
 Aynı derlemede tanımlanan bir tür için tür ileticisi'ni kullanamazsınız.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS0729 oluşturur.  
  
```csharp  
// CS0729.cs  
// compile with: /target:library  
using System.Runtime.CompilerServices;  
[assembly:TypeForwardedTo(typeof(TestClass))]   // CS0729  
class TestClass {}  
```
