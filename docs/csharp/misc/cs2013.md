# <a name="compiler-error-cs2013"></a>Derleyici Hatası CS2013
Geçersiz görüntü tabanı numarası 'value'  
  
 Geçersiz bir değer (sayı değil) geçildi [/baseaddress](../../csharp/language-reference/compiler-options/baseaddress-compiler-option.md) derleyici seçeneği.  
  
 Aşağıdaki örnek, CS2013 oluşturur:  
  
```csharp  
// CS2013.cs  
// compile with: /target:library /baseaddress:x  
// CS2013 expected  
class MyClass  
{  
}  
```
