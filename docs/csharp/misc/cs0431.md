# <a name="compiler-error-cs0431"></a><span data-ttu-id="17dba-102">Derleyici Hatası CS0431</span><span class="sxs-lookup"><span data-stu-id="17dba-102">Compiler Error CS0431</span></span>
<span data-ttu-id="17dba-103">'İdentifier' diğer adı kullanamazsınız '::' bir tür başvurduğundan.</span><span class="sxs-lookup"><span data-stu-id="17dba-103">Cannot use alias 'identifier' with '::' since the alias references a type.</span></span> <span data-ttu-id="17dba-104">Kullanım '.' yerine.</span><span class="sxs-lookup"><span data-stu-id="17dba-104">Use '.' instead.</span></span>  
  
 <span data-ttu-id="17dba-105">Kullanılan, "::" içeren bir türe başvuran bir diğer ad.</span><span class="sxs-lookup"><span data-stu-id="17dba-105">You used "::" with an alias that references a type.</span></span> <span data-ttu-id="17dba-106">Bu hatayı gidermek için "." işleci.</span><span class="sxs-lookup"><span data-stu-id="17dba-106">To resolve this error, use the "." operator.</span></span>  
  
 <span data-ttu-id="17dba-107">Aşağıdaki örnek, CS0431 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="17dba-107">The following sample generates CS0431:</span></span>  
  
```csharp  
// CS0431.cs  
using A = Outer;  
  
public class Outer   
{  
   public class Inner   
   {  
      public static void Meth() {}  
   }  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      A::Inner.Meth();   // CS0431  
      A.Inner.Meth();   // OK  
   }  
}  
```
