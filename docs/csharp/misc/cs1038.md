# <a name="compiler-error-cs1038"></a>Derleyici hatası CS1038
\#endregion yönergesi bekleniyor  
  
 [#Region](../language-reference/preprocessor-directives/preprocessor-region.md) yönergesinin eşleşen bir [#endregion](../language-reference/preprocessor-directives/preprocessor-endregion.md) yönergesi yoktu.  
  
 Aşağıdaki örnek CS1038 oluşturur:  
  
```csharp  
// CS1038.cs  
#region testing  
  
public class clx  
{  
   public static void Main()  
   {  
   }  
}  
// CS1038  
// uncomment the next line to resolve  
// #endregion  
```
