# <a name="compiler-error-cs1038"></a>Derleyici Hatası CS1038
\#endregion yönergesi bekleniyor  
  
 A [#region](../../csharp/language-reference/preprocessor-directives/preprocessor-region.md) yönergesi eşleşen bir yoktu [#endregion](../../csharp/language-reference/preprocessor-directives/preprocessor-endregion.md) yönergesi.  
  
 Aşağıdaki örnek, CS1038 oluşturur:  
  
```csharp  
// CS1038.cs  
#region testing  
  
public class clx  
{  
   public static void Main()  
   {  
   }  
}  
// CS1038  
// uncomment the next line to resolve  
// #endregion  
```
