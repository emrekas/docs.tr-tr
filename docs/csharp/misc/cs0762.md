# <a name="compiler-error-cs0762"></a>Derleyici Hatası CS0762
Uygulama bildirimi olmayan bir kısmi yöntem olduğundan yönteminden 'method' temsilci oluşturulamıyor  
  
 Kısmi yönteminin uygulama bildirimi için gerekli değildir. Ancak, bir temsilci, kapsüllenmiş yöntem uygulaması sahip olmasını gerektirir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Bir uygulama için temsilci başlatmak için kullanılan yöntemi sağlar.  
  
## <a name="example"></a>Örnek  
  
```csharp  
public delegate void TestDel();  
  
    public partial class C  
    {  
        partial void Part();  
  
        public static int Main()  
        {  
            C c = new C();  
            TestDel td = new TestDel(c.Part); // CS0762  
            return 1;  
        }  
  
    }  
```
