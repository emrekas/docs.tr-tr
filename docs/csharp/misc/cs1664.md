# <a name="compiler-error-cs1664"></a>Derleyici Hatası CS1664
Sabit boyutlu arabellek uzunluğu 'length' ve type 'türü çok büyük  
  
 (Öğe boyutuyla çarpılır uzunluğu tarafından belirlendiği şekilde) sabit boyutlu arabellek boyutu olan 2 ^ 31 = 268435455.
