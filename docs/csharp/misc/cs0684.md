# <a name="compiler-warning-level-1-cs0684"></a><span data-ttu-id="e70e2-102">Derleyici Uyarısı (düzey 1) CS0684</span><span class="sxs-lookup"><span data-stu-id="e70e2-102">Compiler Warning (level 1) CS0684</span></span>
<span data-ttu-id="e70e2-103">'interface' arabirimi ' CoClassAttribute' ile işaretlenmiş'ComImportAttribute ' ile işaretli</span><span class="sxs-lookup"><span data-stu-id="e70e2-103">'interface' interface marked with 'CoClassAttribute' not marked with 'ComImportAttribute'</span></span>  
  
 <span data-ttu-id="e70e2-104">Belirtirseniz **ComImportAttribute** bir arabirimde de belirtmeniz gerekir **CoClassAttribute**.</span><span class="sxs-lookup"><span data-stu-id="e70e2-104">If you specify **CoClassAttribute** on an interface, you must also specify **ComImportAttribute**.</span></span>  
  
 <span data-ttu-id="e70e2-105">Aşağıdaki örnek, CS0684 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="e70e2-105">The following sample generates CS0684:</span></span>  
  
```csharp  
// CS0684.cs  
// compile with: /W:1  
using System;  
using System.Runtime.InteropServices;  
  
[CoClass(typeof(C))] // CS0684  
// try the following line instead  
// [CoClass(typeof(C)), ComImport]  
interface I  
{  
}  
  
class C  
{  
   static void Main() {}  
}  
```
