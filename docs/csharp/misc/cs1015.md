# <a name="compiler-error-cs1015"></a>Derleyici Hatası CS1015
Bir nesne, dize veya sınıf türü bekleniyor  
  
 Önceden tanımlı veri türü uygulamasına geçirmek için bir girişimde bulunuldu bir [catch](../../csharp/language-reference/keywords/try-catch.md) blok. Öğesinden türetilen veri türleri <xref:System.Exception?displayProperty=nameWithType> yöntemlere geçirilen bir `catch` blok. Özel durumları hakkında daha fazla bilgi için bkz. [özel durumlar ve özel durum işleme](../../csharp/programming-guide/exceptions/index.md).  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS1015 oluşturur:  
  
```csharp  
// CS1015.cs  
class Sample  
{  
    static void Main()  
    {  
        try   
        {  
        }  
        catch(int)   // CS1015, int is not derived from System.Exception  
        {  
        }  
    }  
}  
```
