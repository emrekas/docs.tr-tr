# <a name="compiler-error-cs0148"></a>Derleyici Hatası CS0148
' % S'temsilci 'temsilci' geçerli bir oluşturucusu yok  
  
 Alınan ve yönetilen bir program (.NET Framework ortak dil çalışma zamanı kullanan bir) kullanılan başka bir derleyici ile oluşturulmuş. Bu derleyici tarafından izin verilen hatalı oluşturulmuş bir [temsilci](../../csharp/language-reference/keywords/delegate.md) Oluşturucusu. Daha fazla bilgi için [Temsilciler](../../csharp/programming-guide/delegates/index.md).
