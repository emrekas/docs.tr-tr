# <a name="compiler-error-cs0148"></a>Derleyici hatası CS0148
' Delegate ' temsilcisinin geçerli bir oluşturucusu yok  
  
 Başka bir derleyici ile oluşturulmuş bir yönetilen program (.NET Framework ortak dil çalışma zamanı kullanan) içeri aktardınız ve kullandınız. Bu derleyici, hatalı oluşturulmuş bir [temsilci](../language-reference/keywords/delegate.md) oluşturucusuna izin verilir. Daha fazla bilgi için bkz. [Temsilciler](../programming-guide/delegates/index.md).
