# <a name="compiler-error-cs0577"></a><span data-ttu-id="918eb-102">Derleyici Hatası CS0577</span><span class="sxs-lookup"><span data-stu-id="918eb-102">Compiler Error CS0577</span></span>
<span data-ttu-id="918eb-103">Oluşturucu, yıkıcı, işleç veya açık arabirim uygulaması olduğundan Conditional özniteliği 'function' üzerinde geçerli değil</span><span class="sxs-lookup"><span data-stu-id="918eb-103">The Conditional attribute is not valid on 'function' because it is a constructor, destructor, operator, or explicit interface implementation</span></span>  
  
 <span data-ttu-id="918eb-104">`Conditional` Belirtilen bir yönteme uygulanamaz.</span><span class="sxs-lookup"><span data-stu-id="918eb-104">`Conditional` cannot be applied to the specified methods.</span></span>  
  
 <span data-ttu-id="918eb-105">Örneğin, bir açık arabirim tanımı üzerinde bazı öznitelikler kullanamazsınız.</span><span class="sxs-lookup"><span data-stu-id="918eb-105">For example, you cannot use some attributes on an explicit interface definition.</span></span> <span data-ttu-id="918eb-106">Aşağıdaki örnek, CS0577 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="918eb-106">The following sample generates CS0577:</span></span>  
  
```csharp  
// CS0577.cs  
// compile with: /target:library  
interface I  
{  
   void m();  
}  
  
public class MyClass : I  
{  
   [System.Diagnostics.Conditional("a")]   // CS0577  
   void I.m() {}  
}  
```
