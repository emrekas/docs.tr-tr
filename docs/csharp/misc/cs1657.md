 <span data-ttu-id="9c02f-104">Bu hata, değişkenin salt okunur olduğu bir bağlamda [ref](../language-reference/keywords/ref.md) veya [Out](../language-reference/keywords/out-parameter-modifier.md) bağımsız değişkeni olarak geçirildiğinde oluşur.</span><span class="sxs-lookup"><span data-stu-id="9c02f-104">This error occurs when a variable is passed as a [ref](../language-reference/keywords/ref.md) or [out](../language-reference/keywords/out-parameter-modifier.md) argument in a context in which that variable is readonly.</span></span> <span data-ttu-id="9c02f-105">ReadOnly bağlamları, değişkenleri ve `fixed` değişkenleri [kullanarak](../language-reference/keywords/using-statement.md) [foreach](../language-reference/keywords/foreach-in.md) yineleme değişkenlerini içerir.</span><span class="sxs-lookup"><span data-stu-id="9c02f-105">Readonly contexts include [foreach](../language-reference/keywords/foreach-in.md) iteration variables, [using](../language-reference/keywords/using-statement.md) variables, and `fixed` variables.</span></span> <span data-ttu-id="9c02f-106">Bu hatayı `foreach`çözmek için, veya `ref` `using` `fixed` `using` `fixed` değişkenini bloklar, `out` deyimlerve'debirparametreolarakalanişlevleriçağırmayınve`foreach` deyimler.</span><span class="sxs-lookup"><span data-stu-id="9c02f-106">To resolve this error, do not call functions that take the `foreach`, `using` or `fixed` variable as a `ref` or `out` parameter in `using` blocks, `foreach` statements, and `fixed` statements.</span></span>  
  
## <a name="example"></a><span data-ttu-id="9c02f-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="9c02f-107">Example</span></span>  
 <span data-ttu-id="9c02f-108">Aşağıdaki örnek CS1657 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="9c02f-108">The following sample generates CS1657:</span></span>  
  
```csharp  
// CS1657.cs  
using System;  
class C : IDisposable  
{  
    public int i;  
    public void Dispose() {}  
}  
  
class CMain  
{  
    static void f(ref C c)  
    {  
    }  
    static void Main()  
    {  
        using (C c = new C())  
        {  
            f(ref c);  // CS1657  
        }  
    }  
}  
```
