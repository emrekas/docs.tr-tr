# <a name="compiler-error-cs0712"></a>Derleyici Hatası CS0712
'Statik sınıf' statik sınıfının örneği oluşturulamıyor  
  
 Statik sınıfların örneklerini oluşturmak mümkün değildir. Statik sınıflar statik alanlar ve yöntemler içerecek şekilde tasarlanmıştır ancak örneği değil.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS0712 oluşturur:  
  
```csharp  
// CS0712.cs  
public static class SC  
{  
}  
  
public class CMain  
{  
    public static void Main()  
    {  
        SC sc = new SC();  // CS0712  
    }  
}  
```
