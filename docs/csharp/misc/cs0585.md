# <a name="compiler-error-cs0585"></a>Derleyici Hatası CS0585
Derleyici iç hatası: aşama 'aşama'  
  
 Derleyici, bağlanamama ayrıştırma beklenmeyen söz dizimi nedeniyle başarısız olup olmadığını belirlemek bu seçeneği deneyin. Durum bu değilse, deneyin [konuşmak bize](/visualstudio/ide/talk-to-us).
