# <a name="compiler-error-cs0243"></a><span data-ttu-id="28a18-102">Derleyici Hatası CS0243</span><span class="sxs-lookup"><span data-stu-id="28a18-102">Compiler Error CS0243</span></span>

<span data-ttu-id="28a18-103">Bir geçersiz kılma yöntemi olduğundan Conditional özniteliği 'metodunda' geçerli değil</span><span class="sxs-lookup"><span data-stu-id="28a18-103">The Conditional attribute is not valid on 'method' because it is an override method</span></span>  
  
 <span data-ttu-id="28a18-104">[Koşullu](../../csharp/programming-guide/concepts/attributes/common-attributes.md#Conditional) özniteliği ile işaretlenmiş bir yöntemde verilmez [geçersiz kılma](../../csharp/language-reference/keywords/override.md) anahtar sözcüğü.</span><span class="sxs-lookup"><span data-stu-id="28a18-104">The [Conditional](../../csharp/programming-guide/concepts/attributes/common-attributes.md#Conditional) attribute is not allowed on a method that is marked with the [override](../../csharp/language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="28a18-105">Daha fazla bilgi için [bilerek, kullanım geçersiz kılma ve yeni anahtar sözcüklerle](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span><span class="sxs-lookup"><span data-stu-id="28a18-105">For more information, see [Knowing When to Use Override and New Keywords](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
 <span data-ttu-id="28a18-106">Derleyici yöntemleri geçersiz kılmak için hiçbir zaman bağlanmaz; yalnızca bir taban yöntemine bağlanır ve ortak dil çalışma zamanı geçersiz kılma uygun olarak çağırır.</span><span class="sxs-lookup"><span data-stu-id="28a18-106">The compiler never binds to override methods; it only binds to the base method and the common language runtime calls the override, as appropriate.</span></span>  
  
 <span data-ttu-id="28a18-107">Aşağıdaki örnek, CS0243 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="28a18-107">The following sample generates CS0243:</span></span>  

```csharp
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
