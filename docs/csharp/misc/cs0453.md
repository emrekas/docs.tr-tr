# <a name="compiler-error-cs0453"></a>Derleyici Hatası CS0453
' % S'türü 'Type Name' null olmayan bir değer türü 'Parametre adı' parametresi kullanılabilmesi için genel türünde veya yönteminde 'Genel tanımlayıcı' içinde olmalıdır  
  
 Değer olmayan tür bağımsız değişkeni olarak bir genel tür veya yöntemi olan örnekleme kullandığınızda bu hata oluşur **değer** üzerindeki sınırlama. Boş değer atanabilen değer tür bağımsız değişkeni kullandığınızda da meydana gelebilir. Aşağıdaki örnek kodda son iki satırlık bakın.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki kod bu hatayı üretir.  
  
```csharp  
// CS0453.cs  
using System;  
public class HV<S> where S : struct { }  
public class H1 : HV<string> { }                   // CS0453  
public class H2 : HV<H1> { }                       // CS0453  
public class H3<S> : HV<S> where S : class { }     // CS0453  
public class H4 : HV<int?> { }                     // CS0453  
public class H5 : HV<Nullable<Nullable<int>>> { }  // CS0453  
```
