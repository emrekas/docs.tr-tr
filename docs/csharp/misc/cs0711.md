# <a name="compiler-error-cs0711"></a><span data-ttu-id="2a4a4-102">Derleyici Hatası CS0711</span><span class="sxs-lookup"><span data-stu-id="2a4a4-102">Compiler Error CS0711</span></span>
<span data-ttu-id="2a4a4-103">Statik sınıflar yıkıcı içeremez</span><span class="sxs-lookup"><span data-stu-id="2a4a4-103">Static classes cannot contain destructors</span></span>  
  
 <span data-ttu-id="2a4a4-104">Bir statik sınıfın oluşturulamadığını, bu nedenle, oluşturucuları veya yıkıcıları gerek yoktur.</span><span class="sxs-lookup"><span data-stu-id="2a4a4-104">A static class cannot be instantiated, hence it has no need for constructors or destructors.</span></span> <span data-ttu-id="2a4a4-105">Bu hatayı önlemek için statik sınıflardan herhangi bir yıkıcı kaldırın veya gerçekten oluşturmak ve örneklerini yok etmek istiyorsanız statik olmayan sınıf olun.</span><span class="sxs-lookup"><span data-stu-id="2a4a4-105">To avoid this error, remove any destructors from static classes, or, if you really want to construct and destroy instances, make the class non-static.</span></span>  
  
 <span data-ttu-id="2a4a4-106">Aşağıdaki örnek, CS0711 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="2a4a4-106">The following sample generates CS0711:</span></span>  
  
```csharp  
// CS0711.cs

{  
   ~C()  // CS0711  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
