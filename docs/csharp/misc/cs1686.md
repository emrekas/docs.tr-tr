<span data-ttu-id="bde4d-103">Yerel 'değişkeni' veya üyelerinin adresleri alınıp sahip olamaz ve anonim bir yöntem veya lambda ifadesinde kullanılamaz</span><span class="sxs-lookup"><span data-stu-id="bde4d-103">Local 'variable' or its members cannot have their address taken and be used inside an anonymous method or lambda expression</span></span>  
  
 <span data-ttu-id="bde4d-104">Bu hata, bir değişken kullanın ve adresini erişmeyi ve bu eylemlerden biri, anonim bir yöntem içinde yapılır oluşturulur.</span><span class="sxs-lookup"><span data-stu-id="bde4d-104">This error is generated when you use a variable, and attempt to take its address, and one of these actions is done inside an anonymous method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="bde4d-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="bde4d-105">Example</span></span>  
 <span data-ttu-id="bde4d-106">Aşağıdaki örnek, CS1686 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="bde4d-106">The following sample generates CS1686.</span></span>  
  
```csharp  
// CS1686.cs  
// compile with: /unsafe /target:library  
class MyClass  
{  
   public unsafe delegate int * MyDelegate();  
  
   public unsafe int * Test()  
   {  
      int j = 0;  
      MyDelegate d = delegate { return &j; };   // CS1686  
      return &j;   // OK  
   }  
}  
```
