# <a name="compiler-error-cs0030"></a><span data-ttu-id="b3d48-102">Derleyici Hatası CS0030</span><span class="sxs-lookup"><span data-stu-id="b3d48-102">Compiler Error CS0030</span></span>
<span data-ttu-id="b3d48-103">Türü 'type' için ' type' türüne dönüştürülemez</span><span class="sxs-lookup"><span data-stu-id="b3d48-103">Cannot convert type 'type' to 'type'</span></span>  
  
<span data-ttu-id="b3d48-104">Önceden tanımlanmış türler arasında dönüştürme yoktur.</span><span class="sxs-lookup"><span data-stu-id="b3d48-104">There is no predefined conversion between types.</span></span> <span data-ttu-id="b3d48-105">Bu türler arasında özel bir dönüştürme tanımlayabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="b3d48-105">You can define a custom conversion between those types.</span></span> <span data-ttu-id="b3d48-106">Daha fazla bilgi için [kullanıcı tanımlı dönüştürme işleçleri](../../csharp/language-reference/operators/user-defined-conversion-operators.md).</span><span class="sxs-lookup"><span data-stu-id="b3d48-106">For more information, see [User-defined conversion operators](../../csharp/language-reference/operators/user-defined-conversion-operators.md).</span></span>  
  
 <span data-ttu-id="b3d48-107">Aşağıdaki örnek, CS0030 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="b3d48-107">The following sample generates CS0030:</span></span>  
  
```csharp  
// CS0030.cs  
namespace x  
{  
   public class iii  
   {  
      /*  
      public static implicit operator iii(int aa)  
      {  
         return null;  
      }  
  
      public static implicit operator int(iii aa)  
      {  
         return 0;  
      }  
      */  
  
      public static iii operator ++(iii aa)  
      {  
         return (iii)0;   // CS0030  
         // uncomment the conversion routines to resolve CS0030  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
