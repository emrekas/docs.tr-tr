# <a name="compiler-error-cs1620"></a><span data-ttu-id="471f9-102">Derleyici hatası CS1620</span><span class="sxs-lookup"><span data-stu-id="471f9-102">Compiler Error CS1620</span></span>
<span data-ttu-id="471f9-103">' Number ' bağımsız değişkeninin ' anahtar sözcüğü ' anahtar sözcüğüyle geçirilmesi gerekir</span><span class="sxs-lookup"><span data-stu-id="471f9-103">Argument 'number' must be passed with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="471f9-104">Bu hata, [ref](../language-reference/keywords/ref.md) veya [Out](../language-reference/keywords/out-parameter-modifier.md) parametresi alan bir işleve bir bağımsız değişken geçirmeniz ve çağrı noktasına `ref` veya `out` anahtar sözcüğünü eklemezseniz ya da yanlış anahtar sözcüğünü dahil etmeniz durumunda oluşur.</span><span class="sxs-lookup"><span data-stu-id="471f9-104">This error occurs if you are passing an argument to a function that takes a [ref](../language-reference/keywords/ref.md) or [out](../language-reference/keywords/out-parameter-modifier.md) parameter and you don't include the `ref` or `out` keyword at the point of call, or you include the wrong keyword.</span></span> <span data-ttu-id="471f9-105">Hata metni kullanılacak uygun anahtar sözcüğü ve hataya neden olan bağımsız değişkeni gösterir.</span><span class="sxs-lookup"><span data-stu-id="471f9-105">The error text indicates the appropriate keyword to use and which argument caused the failure.</span></span>  
  
 <span data-ttu-id="471f9-106">Aşağıdaki örnek CS1620 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="471f9-106">The following sample generates CS1620:</span></span>  
  
```csharp  
// CS1620.cs  
class C  
{  
    void f(ref int i) {}  
    public static void Main()  
    {  
        int x = 1;  
        f(out x);  // CS1620 – f takes a ref parameter, not an out parameter  
        // Try this line instead:  
        // f(ref x);  
    }  
}  
```
