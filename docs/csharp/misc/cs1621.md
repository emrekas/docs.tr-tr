# <a name="compiler-error-cs1621"></a><span data-ttu-id="a90b2-102">Derleyici Hatası CS1621</span><span class="sxs-lookup"><span data-stu-id="a90b2-102">Compiler Error CS1621</span></span>
<span data-ttu-id="a90b2-103">Yield deyimi bir anonim yöntem veya lambda ifadesinde kullanılamaz</span><span class="sxs-lookup"><span data-stu-id="a90b2-103">The yield statement cannot be used inside an anonymous method or lambda expression</span></span>  
  
 <span data-ttu-id="a90b2-104">Yield deyimi bir anonim yöntem bloğu içinde bir yineleyici olamaz.</span><span class="sxs-lookup"><span data-stu-id="a90b2-104">The yield statement cannot be in an anonymous method block in an iterator.</span></span>  
  
## <a name="example"></a><span data-ttu-id="a90b2-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="a90b2-105">Example</span></span>  
 <span data-ttu-id="a90b2-106">Aşağıdaki örnek, CS1621 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="a90b2-106">The following sample generates CS1621:</span></span>  
  
```csharp  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
