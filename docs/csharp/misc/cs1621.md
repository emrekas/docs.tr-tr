# <a name="compiler-error-cs1621"></a>Derleyici hatası CS1621
Yield deyimi bir anonim yöntem veya lambda ifadesi içinde kullanılamaz  
  
Bir [anonim yöntemde](../language-reference/operators/delegate-operator.md) veya [lambda ifadesinde](../../csharp/programming-guide/statements-expressions-operators/lambda-expressions.md) [yield](../../csharp/language-reference/keywords/yield.md) deyimini kullanamazsınız.
  
Aşağıdaki örnek CS1621 oluşturur:
  
```csharp  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
