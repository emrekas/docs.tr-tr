## <a name="example"></a><span data-ttu-id="61e70-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="61e70-108">Example</span></span>  
 <span data-ttu-id="61e70-109">Aşağıdaki örnek CS1930 oluşturur çünkü tanımlayıcı `num` aralık değişkeni için kullanılan `from` yan tümcesi ve tarafından tanıtılan aralık değişkeni için `let` yan tümcesi.</span><span class="sxs-lookup"><span data-stu-id="61e70-109">The following example generates CS1930 because the identifier `num` is used for the range variable in the `from` clause and for the range variable introduced by the `let` clause.</span></span>  
  
```csharp  
// cs1930.cs  
using System.Linq;  
class Program  
{  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        var query = from num in nums  
                    let num = 3 // CS1930  
                    select num;   
    }  
}  
```
