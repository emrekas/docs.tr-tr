# <a name="compiler-error-cs0077"></a><span data-ttu-id="e0235-102">Derleyici hatası CS0077</span><span class="sxs-lookup"><span data-stu-id="e0235-102">Compiler Error CS0077</span></span>
<span data-ttu-id="e0235-103">As işleci bir başvuru türüyle veya null olabilen bir türle birlikte kullanılmalıdır (' int ', null olamayan bir değer türüdür).</span><span class="sxs-lookup"><span data-stu-id="e0235-103">The as operator must be used with a reference type or nullable type ('int' is a non-nullable value type).</span></span>  
  
 <span data-ttu-id="e0235-104">[As](../../csharp/language-reference/operators/type-testing-and-cast.md#as-operator) işleci bir [değer türü](../../csharp/language-reference/keywords/value-types.md)geçirdi.</span><span class="sxs-lookup"><span data-stu-id="e0235-104">The [as](../../csharp/language-reference/operators/type-testing-and-cast.md#as-operator) operator was passed a [value type](../../csharp/language-reference/keywords/value-types.md).</span></span> <span data-ttu-id="e0235-105">Null döndürebildiğinden yalnızca [başvuru türleri](../../csharp/language-reference/keywords/reference-types.md) veya null yapılabilir tür geçirilebilir. [](../../csharp/language-reference/keywords/null.md) `as`</span><span class="sxs-lookup"><span data-stu-id="e0235-105">Because `as` can return [null](../../csharp/language-reference/keywords/null.md), it can only be passed [reference types](../../csharp/language-reference/keywords/reference-types.md) or nullable type.</span></span> <span data-ttu-id="e0235-106">Null yapılabilir türler hakkında daha fazla bilgi için bkz. [Nullable türler](../../csharp/programming-guide/nullable-types/index.md).</span><span class="sxs-lookup"><span data-stu-id="e0235-106">For more information about nullable types, see [Nullable Types](../../csharp/programming-guide/nullable-types/index.md).</span></span>  
  
 <span data-ttu-id="e0235-107">Aşağıdaki örnek CS0077 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="e0235-107">The following sample generates CS0077:</span></span>  
  
```csharp  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```
