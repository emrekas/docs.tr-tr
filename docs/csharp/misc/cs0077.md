# <a name="compiler-error-cs0077"></a>Derleyici hatası CS0077
As işleci bir başvuru türüyle veya null olabilen bir türle birlikte kullanılmalıdır (' int ', null olamayan bir değer türüdür).  
  
 [As](../../csharp/language-reference/operators/type-testing-and-cast.md#as-operator) işleci bir [değer türü](../../csharp/language-reference/keywords/value-types.md)geçirdi. Null döndürebildiğinden yalnızca [başvuru türleri](../../csharp/language-reference/keywords/reference-types.md) veya null yapılabilir tür geçirilebilir. [](../../csharp/language-reference/keywords/null.md) `as` Null yapılabilir türler hakkında daha fazla bilgi için bkz. [Nullable türler](../../csharp/programming-guide/nullable-types/index.md).  
  
 Aşağıdaki örnek CS0077 oluşturur:  
  
```csharp  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```
