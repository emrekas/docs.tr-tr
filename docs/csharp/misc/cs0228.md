# <a name="compiler-error-cs0228"></a>Derleyici Hatası CS0228
'type', 'member' tanımı içermiyor veya erişilebilir değil  
  
 Sistem sürümü yerine **System.Hashtable**, **System.String**, veya **System.Array**, ve değişiklik içermiyor `member`.  
  
 Aksi takdirde, Visual Studio'yu yeniden yükleyin veya onarın gerekebilir.
