# <a name="compiler-error-cs1013"></a>Derleyici Hatası CS1013
Geçersiz sayı  
  
 Derleyici, hatalı biçimlendirilmiş bir numara algıladı. Tam sayı türleri hakkında daha fazla bilgi için bkz. [tam sayı türleri tablosu](../../csharp/language-reference/builtin-types/integral-numeric-types.md).  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS1013 oluşturur:  
  
```csharp  
// CS1013.cs  
class Sample  
{  
    static void Main()  
    {  
        int i = 0x;   // CS1013  
    }  
}  
```
