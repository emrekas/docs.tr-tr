# <a name="compiler-error-cs1102"></a><span data-ttu-id="6e100-102">Derleyici Hatası CS1102</span><span class="sxs-lookup"><span data-stu-id="6e100-102">Compiler Error CS1102</span></span>
<span data-ttu-id="6e100-103">'Out' parametre değiştiricisi 'this' ile kullanılamaz.</span><span class="sxs-lookup"><span data-stu-id="6e100-103">The parameter modifier 'out' cannot be used with 'this'.</span></span>  
  
 <span data-ttu-id="6e100-104">Zaman `this` anahtar sözcüğünün değiştirdiği bir statik yöntem ilk parametresi, derleyicinin yöntemin bir genişletme yöntemi olduğunu bildirir.</span><span class="sxs-lookup"><span data-stu-id="6e100-104">When the `this` keyword modifies the first parameter of a static method, it signals to the compiler that the method is an extension method.</span></span> <span data-ttu-id="6e100-105">Diğer bir değiştiricilere gerekli veya genişletme yönteminin ilk parametresi üzerinde izin verilir.</span><span class="sxs-lookup"><span data-stu-id="6e100-105">No other modifiers are needed or allowed on the first parameter of an extension method.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="6e100-106">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="6e100-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="6e100-107">Yetkisiz değiştiriciler ilk parametresinden kaldırın.</span><span class="sxs-lookup"><span data-stu-id="6e100-107">Remove the unauthorized modifiers from the first parameter.</span></span>  
  
## <a name="example"></a><span data-ttu-id="6e100-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="6e100-108">Example</span></span>  
 <span data-ttu-id="6e100-109">Aşağıdaki örnek, CS1102 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="6e100-109">The following example generates CS1102:</span></span>  
  
```csharp  
// cs1102.cs  
// Compile with: /target:library.  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(this out int i) {} // CS1102  
  
    //Single type parameter  
        public static void Test<T>(this out T t) {}// CS1102  
  
    //Multiple type parameters  
        public static void Test<T,U,V>(this out U u) {}// CS1102  
}  
```
