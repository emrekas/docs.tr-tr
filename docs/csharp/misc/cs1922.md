<span data-ttu-id="de2bd-103">Koleksiyon Başlatıcısı System. Collections. IEnumerable öğesini uygulamak için ' Type ' türünü gerektirir.</span><span class="sxs-lookup"><span data-stu-id="de2bd-103">Collection initializer requires its type 'type' to implement System.Collections.IEnumerable.</span></span>

<span data-ttu-id="de2bd-104">Bir koleksiyon başlatıcısı türünü bir tür ile kullanmak için, türün uygulanması `IEnumerable`gerekir.</span><span class="sxs-lookup"><span data-stu-id="de2bd-104">In order to use a collection initializer with a type, the type must implement `IEnumerable`.</span></span> <span data-ttu-id="de2bd-105">Bu hata, bir nesne Başlatıcısı kullanmak istediğinizde yanlışlıkla koleksiyon başlatıcısı sözdizimini kullanırsanız oluşabilir.</span><span class="sxs-lookup"><span data-stu-id="de2bd-105">This error can occur if you accidentally use collection initializer syntax when you meant to use an object initializer.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="de2bd-106">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="de2bd-106">To correct this error</span></span>

- <span data-ttu-id="de2bd-107">Tür bir koleksiyonu temsil ediyorsa, koleksiyon başlatıcısı sözdizimi yerine nesne Başlatıcısı sözdizimini kullanın.</span><span class="sxs-lookup"><span data-stu-id="de2bd-107">If the type does not represent a collection, use object initializer syntax instead of collection initializer syntax.</span></span>

- <span data-ttu-id="de2bd-109">Tür bir koleksiyonu temsil ediyorsa ve kaynak koda erişiminiz yoksa, kendi sınıf oluşturucularını veya diğer başlatma yöntemlerini kullanarak yalnızca öğelerini başlatın.</span><span class="sxs-lookup"><span data-stu-id="de2bd-109">If the type represents a collection and you do not have access to the source code, just initialize its elements by using their class constructors or other initialization methods.</span></span>

## <a name="example"></a><span data-ttu-id="de2bd-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="de2bd-110">Example</span></span>

<span data-ttu-id="de2bd-111">Aşağıdaki kod CS1922 üretir:</span><span class="sxs-lookup"><span data-stu-id="de2bd-111">The following code produces CS1922:</span></span>

```csharp
// cs1922.cs
public class Test
{
    public static void Main()
    {
        // Collection initializer.
        var tc = new TestClass  {1,"hello"} ; // CS1922

        // Object initializer.
        var tc2 = new TestClass { memberA = 1, memberB = "hello" }; // OK
    }
}

public class TestClass
{
    public int memberA { get; set; }
    public string memberB { get; set; }
}
```
