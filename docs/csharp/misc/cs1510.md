# <a name="compiler-error-cs1510"></a>Derleyici Hatası CS1510
Bir ref veya bağımsız değişkeni, atanabilir bir değişken olmalıdır  
  
 Yalnızca bir değişken olarak geçirilen bir [ref](../../csharp/language-reference/keywords/ref.md) yöntem çağrısında parametresi. A `ref` değeri olan bir işaretçi geçirme gibi.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS1510 oluşturur:  
  
```csharp  
// CS1510.cs  
public class C  
{  
   public static int j = 0;  
  
   public static void M(ref int j)  
   {  
      j++;  
   }  
  
   public static void Main ()  
   {  
      M (ref 2);   // CS1510, can't pass a number as a ref parameter  
      // try the following to resolve the error  
      // M (ref j);  
   }  
}  
```
