# <a name="compiler-error-cs0549"></a><span data-ttu-id="33db8-102">Derleyici hatası CS0549</span><span class="sxs-lookup"><span data-stu-id="33db8-102">Compiler Error CS0549</span></span>
<span data-ttu-id="33db8-103">' function ', ' class ' Sealed sınıfındaki yeni bir sanal üyedir</span><span class="sxs-lookup"><span data-stu-id="33db8-103">'function' is a new virtual member in sealed class 'class'</span></span>  
  
 <span data-ttu-id="33db8-104">[Sealed](../language-reference/keywords/sealed.md)bir[sınıf](../language-reference/keywords/class.md) temel sınıf olarak kullanılamaz.</span><span class="sxs-lookup"><span data-stu-id="33db8-104">A [sealed](../language-reference/keywords/sealed.md)[class](../language-reference/keywords/class.md) cannot be used as a base class.</span></span>  <span data-ttu-id="33db8-105">Bu nedenle, korumalı sınıfta sanal bir metoda sahip olmak işe yaramaz.</span><span class="sxs-lookup"><span data-stu-id="33db8-105">Therefore, it is useless to have a virtual method in sealed class.</span></span>  
  
 <span data-ttu-id="33db8-106">Aşağıdaki örnek CS0549 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="33db8-106">The following sample generates CS0549:</span></span>  
  
```csharp  
// CS0549.cs  
// compile with: /target:library  
sealed public class MyClass  
{  
   virtual public void TestMethod() {}   // CS0549  
   public void TestMethod2() {}   // OK  
}  
```
