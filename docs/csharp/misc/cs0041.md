# <a name="compiler-error-cs0041"></a>Derleyici Hatası CS0041
'Type' için tam nitelikli adı hata ayıklama bilgileri için çok uzun. Olmadan derleme '/ debug' seçeneği.  
  
 Bu hata kullanırken oluşabilecek [/debug](../../csharp/language-reference/compiler-options/debug-compiler-option.md) derleyici seçeneği. Bu hatayla karşılaşırsanız, bin dizinindeki PDB dosyaları silmeyi deneyin ve yeniden derlemeden. Hala bu hatayla karşılaşıyorsanız, Visual Studio'yu yeniden yükleyin veya onarın gerekebilir.
