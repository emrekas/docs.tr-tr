# <a name="compiler-error-cs0641"></a>Derleyici Hatası CS0641
'attribute': öznitelik System.Attribute türevi olan sınıflarda yalnızca  
  
 Bir öznitelik kullanıldı, yalnızca kullanılabilir, türetilen bir sınıfta **System.Attribute**.  
  
 Aşağıdaki örnek, CS0641 oluşturur:  
  
```csharp  
// CS0641.cs  
using System;  
  
[AttributeUsage(AttributeTargets.All)]  
public class NonAttrClass   // CS0641  
// try the following line instead  
// public class NonAttrClass : Attribute  
{  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
