# <a name="compiler-error-cs0308"></a>Derleyici Hatası CS0308
Genel olmayan türü-veya-method 'identifier' tür bağımsız değişkenleri ile kullanılamaz.  
  
 Yöntem veya türü genel değil, ancak tür bağımsız değişkenleri ile kullanıldı. Bu hatadan kaçınmak için açılı ayraçlar kaldırın ve tür bağımsız değişkenleri veya yöntem veya tür, bir genel yöntem veya tür olarak yeniden bildirmek.  
  
 Aşağıdaki örnek, CS0308 oluşturur:  
  
```csharp  
// CS0308a.cs  
class MyClass  
{  
   public void F() {}  
   public static void Main()  
   {  
      F<int>();  // CS0308 – F is not generic.  
      // Try this instead:  
      // F();  
   }  
}  
```  
  
 Aşağıdaki örnek, aynı zamanda CS0308 oluşturur. Hatayı gidermek için "System.Collections.Generic kullanma" yönergesini kullanın.  
  
```csharp  
// CS0308b.cs  
// compile with: /t:library  
using System.Collections;  
// To resolve, uncomment the following line:  
// using System.Collections.Generic;  
public class MyStack<T>  
{  
    // Store the elements of the stack:  
    private T[] items = new T[100];  
    private int stack_counter = 0;  
  
    // Define the iterator block:  
    public IEnumerator<T> GetEnumerator()   // CS0308  
    {  
        for (int i = stack_counter - 1 ; i >= 0; i--)  
        yield return items[i];  
    }  
}  
```
