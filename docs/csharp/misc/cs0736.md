# <a name="compiler-error-cs0736"></a><span data-ttu-id="4ac66-102">Derleyici Hatası CS0736</span><span class="sxs-lookup"><span data-stu-id="4ac66-102">Compiler Error CS0736</span></span>
<span data-ttu-id="4ac66-103">'type name', 'üyesi adı' arabirim üyesini uygulamıyor.</span><span class="sxs-lookup"><span data-stu-id="4ac66-103">'type name' does not implement interface member 'member name'.</span></span> <span data-ttu-id="4ac66-104">statik olduğundan, 'yöntem adı' arabirim üyesini uygulayamaz.</span><span class="sxs-lookup"><span data-stu-id="4ac66-104">'method name' cannot implement an interface member because it is static.</span></span>  
  
 <span data-ttu-id="4ac66-105">Bu hata, bir statik yöntem örtük veya açık bir arabirim üyesi uygulaması bildirildiğinde oluşturulur.</span><span class="sxs-lookup"><span data-stu-id="4ac66-105">This error is generated when a static method is either implicitly or explicitly declared as an implementation of an interface member.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="4ac66-106">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="4ac66-106">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="4ac66-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="4ac66-110">Example</span></span>  
 <span data-ttu-id="4ac66-111">Aşağıdaki kodu CS0736 oluşturur çünkü `Program.testMethod` statik olarak bildirilir:</span><span class="sxs-lookup"><span data-stu-id="4ac66-111">The following code generates CS0736 because `Program.testMethod` is declared as static:</span></span>  
  
```csharp  
// cs0736.cs  
namespace CS0736  
{     
  
    interface ITest  
    {  
        int testMethod(int x);  
    }  
  
    class Program : ITest // CS0736  
    {  
        public static int testMethod(int x) { return 0; }  
        // Try the following line instead.  
        // public int testMethod(int x) { return 0; }  
        public static void Main() { }  
    }      
}  
```
