## <a name="example"></a><span data-ttu-id="14cf0-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="14cf0-110">Example</span></span>  
 <span data-ttu-id="14cf0-111">Aşağıdaki kod, statik olarak bildirildiği `Program.testMethod` için CS0736 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="14cf0-111">The following code generates CS0736 because `Program.testMethod` is declared as static:</span></span>

```csharp  
namespace CS0736  
{     
  
    interface ITest  
    {  
        int testMethod(int x);  
    }  
  
    class Program : ITest // CS0736  
    {  
        public static int testMethod(int x) { return 0; }  
        // Try the following line instead.  
        // public int testMethod(int x) { return 0; }  
        public static void Main() { }  
    }      
}  
```
