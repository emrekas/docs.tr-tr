# <a name="compiler-error-cs1593"></a><span data-ttu-id="0292d-102">Derleyici hatası CS1593</span><span class="sxs-lookup"><span data-stu-id="0292d-102">Compiler Error CS1593</span></span>
<span data-ttu-id="0292d-103">' Del ' temsilcisi ' number ' bağımsız değişkeni almaz</span><span class="sxs-lookup"><span data-stu-id="0292d-103">Delegate 'del' does not take 'number' arguments</span></span>  
  
 <span data-ttu-id="0292d-104">Bir [temsilci](../language-reference/keywords/delegate.md) çağrısına geçirilen bağımsız değişkenlerin sayısı, temsilci bildirimindeki parametre sayısıyla kabul etmez.</span><span class="sxs-lookup"><span data-stu-id="0292d-104">The number of arguments passed to a [delegate](../language-reference/keywords/delegate.md) invocation does not agree with the number of parameters in the delegate declaration.</span></span>  
  
 <span data-ttu-id="0292d-105">Aşağıdaki örnek CS1593 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="0292d-105">The following sample generates CS1593:</span></span>  
  
```csharp  
// CS1593.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello(8, 9);   // CS1593  
      // try the following line instead  
      // hello(8);  
   }  
}  
```
