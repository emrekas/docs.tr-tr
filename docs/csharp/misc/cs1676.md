# <a name="compiler-error-cs1676"></a>Derleyici Hatası CS1676
'Number' parametresi 'anahtar sözcüğü' anahtar sözcüğü ile bildirilmelidir  
  
 Parametre türü değiştiricisi bir anonim yöntem, yönteme atama metot temsilcisinin bildirimde kullanılan farklı olduğunda bu hata oluşur.  
  
 Aşağıdaki örnek, CS1676 oluşturur:  
  
```csharp  
// CS1676.cs  
delegate void E(ref int i);  
class Errors   
{  
   static void Main()  
   {  
      E e = delegate(out int i) { };   // CS1676  
      // To resolve, use the following line instead:  
      // E e = delegate(ref int i) { };  
   }  
}  
```
