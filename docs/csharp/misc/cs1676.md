# <a name="compiler-error-cs1676"></a><span data-ttu-id="cafde-102">Derleyici Hatası CS1676</span><span class="sxs-lookup"><span data-stu-id="cafde-102">Compiler Error CS1676</span></span>
<span data-ttu-id="cafde-103">'Number' parametresi 'anahtar sözcüğü' anahtar sözcüğü ile bildirilmelidir</span><span class="sxs-lookup"><span data-stu-id="cafde-103">Parameter 'number' must be declared with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="cafde-104">Parametre türü değiştiricisi bir anonim yöntem, yönteme atama metot temsilcisinin bildirimde kullanılan farklı olduğunda bu hata oluşur.</span><span class="sxs-lookup"><span data-stu-id="cafde-104">This error occurs when the parameter type modifier in an anonymous method is different from that used in the declaration of the delegate you are casting the method to.</span></span>  
  
 <span data-ttu-id="cafde-105">Aşağıdaki örnek, CS1676 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="cafde-105">The following sample generates CS1676:</span></span>  
  
```csharp  
// CS1676.cs  
delegate void E(ref int i);  
class Errors   
{  
   static void Main()  
   {  
      E e = delegate(out int i) { };   // CS1676  
      // To resolve, use the following line instead:  
      // E e = delegate(ref int i) { };  
   }  
}  
```
