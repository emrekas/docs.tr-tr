# <a name="compiler-error-cs1639"></a>Derleyici Hatası CS1639
'İnterface' arabirimi için ' imzası' yönetilen coclass sarmalayıcı sınıfı imzası geçerli bir sınıf adı imzası değildir  
  
 Arabirimi için meta verileri geçersiz, bu hata meydana gelir. Arabirimi kullanılarak oluşturulduysa bu asla gerçekleşmelidir C# veya desteklenen bir .NET dili. Arabirimin satıcısına başvurun veya arabirimi bütünleştirilmiş kod başarıyla oluşturulduğunu doğrulayın.
