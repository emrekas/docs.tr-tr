## <a name="example"></a><span data-ttu-id="015ad-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="015ad-109">Example</span></span>  
 <span data-ttu-id="015ad-110">Aşağıdaki örnek, CS0755 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="015ad-110">The following example generates CS0755:</span></span>

    {  
        static partial void Part(this C c); //Extension method

    }

        }  
    }
