 <span data-ttu-id="ab576-104">Koleksiyon Başlatıcısı öğesi başlatıcıları bir dizi oluşur.</span><span class="sxs-lookup"><span data-stu-id="ab576-104">A collection initializer consists of a sequence of element initializers.</span></span> <span data-ttu-id="ab576-105">Öğe başlatıcıları bir atama ifadesinin içerdikleri sürece ayraç içine alınması gerekmez.</span><span class="sxs-lookup"><span data-stu-id="ab576-105">The element initializers do not have to be enclosed in braces unless they contain an assignment expression.</span></span> <span data-ttu-id="ab576-106">Küme ayraçları sağlarsanız, ancak bunlar boş olamaz.</span><span class="sxs-lookup"><span data-stu-id="ab576-106">However, if you do supply braces, they cannot be empty.</span></span> <span data-ttu-id="ab576-107">Nesne Başlatıcı öğe Başlatıcısı ise yeni bir nesne oluşturma ifadesi Başlatıcı içerdiği sürece küme ayraçları boş olabilir.</span><span class="sxs-lookup"><span data-stu-id="ab576-107">If the element initializer is an object initializer, the braces may be empty as long as the initializer contains a new object creation expression.</span></span>

## <a name="example"></a><span data-ttu-id="ab576-111">Örnek</span><span class="sxs-lookup"><span data-stu-id="ab576-111">Example</span></span>  
 <span data-ttu-id="ab576-112">Aşağıdaki örnek, CS1920 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="ab576-112">The following example generates CS1920:</span></span>  
  
```csharp  
  // cs1920.cs  
using System.Collections.Generic;  
public class Test  
{  
    public static int Main()  
    {  
        // Error. Empty initializer   
        // for inner list.  
        List<List<int>> collection =  
            new List<List<int>>() { { } }; // CS1920  
  
        // OK. No initializer for inner list.  
        List<List<int>> collection2 =  
            new List<List<int>>() {  };  
  
        // OK. Inner list is initialized   
        // to one List<int> with zero elements.  
        List<List<int>> collection3 =  
            new List<List<int>>() { new List<int> { } };  
        return 0;  
    }  
}  
```
