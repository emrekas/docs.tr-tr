# <a name="compiler-error-cs0754"></a><span data-ttu-id="374fa-102">Derleyici hatası CS0754</span><span class="sxs-lookup"><span data-stu-id="374fa-102">Compiler Error CS0754</span></span>

## <a name="example"></a><span data-ttu-id="374fa-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="374fa-107">Example</span></span>  
 <span data-ttu-id="374fa-108">Aşağıdaki kod CS0754 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="374fa-108">The following code generates CS0754:</span></span>  
  
```csharp  
// cs0754.cs  
using System;  
  
    public interface IF  
    {  
        void Part();  
    }  
  
    public partial class C : IF  
    {  
        partial void IF.Part(); //CS0754  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
