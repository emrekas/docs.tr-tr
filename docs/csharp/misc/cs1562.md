# <a name="compiler-error-cs1562"></a>Derleyici Hatası CS1562
/ Out seçeneği kaynağı olmayan çıkışlar olmalıdır  
  
 Derleme bir çıktı dosyası oluşturabilirsiniz, ancak çıktı dosyası adını örtük giriş olarak hiçbir kaynak kodu dosyasının oluştu. Örneğin, bir meta - veya -yalnızca kaynak dosyasını derlemek çalışıyor olabilir.  
  
 Kullanım [/out](../../csharp/language-reference/compiler-options/out-compiler-option.md) derleyici seçeneğini çıktı dosyası adını belirtin.
