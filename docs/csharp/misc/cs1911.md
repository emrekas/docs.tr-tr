# <a name="compiler-warning-level-1-cs1911"></a><span data-ttu-id="1f134-102">Derleyici Uyarısı (düzey 1) CS1911</span><span class="sxs-lookup"><span data-stu-id="1f134-102">Compiler Warning (level 1) CS1911</span></span>
<span data-ttu-id="1f134-103">Anonim yöntem, lambda ifadesi, sorgu ifadesi ya da yineleyici aracılığıyla bir 'base' anahtar sözcüğü ' name' üyesine erişim doğrulanamayan kodu sonuçlanır.</span><span class="sxs-lookup"><span data-stu-id="1f134-103">Access to member 'name' through a 'base' keyword from an anonymous method, lambda expression, query expression, or iterator results in unverifiable code.</span></span> <span data-ttu-id="1f134-104">Erişim kapsayan türdeki bir yardımcı yöntem içinde taşımayı düşünün.</span><span class="sxs-lookup"><span data-stu-id="1f134-104">Consider moving the access into a helper method on the containing type.</span></span>  
  
 <span data-ttu-id="1f134-105">İle sanal işlevleri çağırma `base` anahtar sözcüğü bir yineleyici veya anonim yöntemler yöntem gövdesi içindeki doğrulanamaz kod oluşturulmasına neden olur.</span><span class="sxs-lookup"><span data-stu-id="1f134-105">Calling virtual functions with the `base` keyword inside the method body of an iterator or anonymous methods will result in unverifiable code.</span></span> <span data-ttu-id="1f134-106">Kısmi güven ortamında çalıştırmak amacıyla doğrulanamaz kod başarısız olur.</span><span class="sxs-lookup"><span data-stu-id="1f134-106">Unverifiable code will fail to run in a partial trust environment.</span></span>  
  
 <span data-ttu-id="1f134-107">Bir çözüm CS1911 için yardımcı işlevini sanal işlev çağrısı taşımaktır.</span><span class="sxs-lookup"><span data-stu-id="1f134-107">One resolution for CS1911 is to move the virtual function call to a helper function.</span></span>  
  
## <a name="example"></a><span data-ttu-id="1f134-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="1f134-108">Example</span></span>  
 <span data-ttu-id="1f134-109">Aşağıdaki örnek, CS1911 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="1f134-109">The following sample generates CS1911.</span></span>  
  
```csharp  
// CS1911.cs  
// compile with: /W:1  
using System;  
  
delegate void D();  
delegate D RetD();  
  
class B {  
   protected virtual void M() {  
      Console.WriteLine("B.M");  
   }  
}  
  
class Der : B {  
   protected override void M() {  
      Console.WriteLine("D.M");  
   }  
  
   void Test() { base.M(); }  
   D Test2() { return new D(base.M); }  
  
   public D CallBaseM() {  
      return delegate () { base.M(); };   // CS1911  
  
      // try the following line instead  
      // return delegate () { Test(); };  
   }  
  
   public RetD DelToBaseM() {  
      return delegate () { return new D(base.M); };   // CS1911  
  
      // try the following line instead  
      // return delegate () { return Test2(); };  
   }  
}  
  
class Program {  
   public static void Main() {  
      Der der = new Der();  
      D d = der.CallBaseM();  
      d();  
      RetD rd = der.DelToBaseM();  
      rd()();  
   }  
}  
```
