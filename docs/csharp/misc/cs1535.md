# <a name="compiler-error-cs1535"></a>Derleyici Hatası CS1535
Aşırı yüklenmiş birli'operator 'işleci bir parametre alır  
  
 Bir birli tanımını [işleci](../../csharp/language-reference/operators/operator-overloading.md) bir parametre almalıdır.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS1535 oluşturur:  
  
```csharp  
// CS1535.cs  
class MyClass  
{  
    // uncomment the method parameter to resolve CS1535  
    public static MyClass operator ++ (/*MyClass MC1*/)   // CS1535  
    {  
        return new MyClass();  
    }  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
