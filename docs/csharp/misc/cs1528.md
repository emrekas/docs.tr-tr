# <a name="compiler-error-cs1528"></a>Derleyici Hatası CS1528
Beklenen; veya = (bildirimde oluşturucu bağımsız değişkenleri belirtilemez)  
  
 Bir nesne sınıfı için oluşturulan gibi bir sınıfa bir başvuru oluşturulmuş. Örneğin, bir değişken bir oluşturucuya geçirilecek girişiminde vardı. Kullanım [yeni](../../csharp/language-reference/operators/new-operator.md) bir sınıfın bir nesnesi oluşturmak için işleç.  
  
 Aşağıdaki örnek, CS1528 oluşturur:  
  
```csharp  
// CS1528.cs  
using System;  
  
public class B  
{  
   public B(int i)  
   {  
      _i = i;  
   }  
  
   public void PrintB()  
   {  
      Console.WriteLine(_i);  
   }  
  
   private int _i;  
}  
  
public class mine  
{  
   public static void Main()  
   {  
      B b(3);   // CS1528, reference is not an object  
      // try one of the following  
      // B b;  
      // or  
      // B bb = new B(3);  
      // bb.PrintB();  
   }  
}  
```
