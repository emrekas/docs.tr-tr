# <a name="compiler-error-cs0247"></a>Derleyici Hatası CS0247
Stackalloc ile negatif boyut kullanılamaz  
  
 Negatif bir sayı geçildi bir [stackalloc](../../csharp/language-reference/keywords/stackalloc.md) deyimi.  
  
 Aşağıdaki örnek, CS0247 oluşturur:  
  
```csharp  
// CS0247.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int *p = stackalloc int [-30];   // CS0247  
   }  
}  
```
