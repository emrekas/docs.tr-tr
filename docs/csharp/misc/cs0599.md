# <a name="compiler-error-cs0599"></a>Derleyici Hatası CS0599
Adlandırılmış öznitelik bağımsız değişkeni 'bağımsız değişkeni' için geçersiz değer  
  
 Bir öznitelik için geçersiz bağımsız değişken geçirildi.
