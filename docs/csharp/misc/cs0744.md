# <a name="compiler-error-cs0744"></a><span data-ttu-id="0dedb-102">Derleyici hatası CS0744</span><span class="sxs-lookup"><span data-stu-id="0dedb-102">Compiler Error CS0744</span></span>
<span data-ttu-id="0dedb-103">' Equals ' bağlamsal anahtar sözcüğü bekleniyordu</span><span class="sxs-lookup"><span data-stu-id="0dedb-103">Expected contextual keyword 'equals'</span></span>  
  
 <span data-ttu-id="0dedb-104">Yan tümce için bir `join` model ...`join` `in`... `on`... `equals`aşağıdaki örnekte gösterildiği gibi:</span><span class="sxs-lookup"><span data-stu-id="0dedb-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>

            join y in array2 on x equals y  
            select x;

## <a name="example"></a><span data-ttu-id="0dedb-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="0dedb-107">Example</span></span>  
 <span data-ttu-id="0dedb-108">Aşağıdaki kod CS0744 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="0dedb-108">The following code generates CS0744:</span></span>  
  
```csharp  
// cs0744.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3 ,4, 5, 6 };  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 on x y // CS0744  
                select x;  
        return 1;  
    }  
}  
```
