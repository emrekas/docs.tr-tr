# <a name="compiler-error-cs0617"></a><span data-ttu-id="d67db-102">Derleyici hatası CS0617</span><span class="sxs-lookup"><span data-stu-id="d67db-102">Compiler Error CS0617</span></span>
<span data-ttu-id="d67db-103">' Reference ' geçerli bir öznitelik parametresi türü olmadığından geçerli bir adlandırılmış öznitelik bağımsız değişkeni değil</span><span class="sxs-lookup"><span data-stu-id="d67db-103">'reference' is not a valid named attribute argument because it is not a valid attribute parameter type</span></span>  
  
 <span data-ttu-id="d67db-104">Öznitelik sınıfının [özel](../language-reference/keywords/private.md) üyesine erişme girişiminde bulunuldu.</span><span class="sxs-lookup"><span data-stu-id="d67db-104">An attempt was made to access a [private](../language-reference/keywords/private.md) member of an attribute class.</span></span>  
  
## <a name="example"></a><span data-ttu-id="d67db-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="d67db-105">Example</span></span>  
 <span data-ttu-id="d67db-106">Aşağıdaki örnek CS0617 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="d67db-106">The following sample generates CS0617.</span></span>  
  
```csharp  
// CS0617.cs  
using System;  
  
[AttributeUsage(AttributeTargets.Struct |   
                AttributeTargets.Class |  
                AttributeTargets.Interface)]  
public class MyClass : Attribute  
{  
   public int Name;  
  
   public MyClass (int sName)  
   {  
      Name = sName;  
      Bad = -1;  
      Bad2 = -1;  
   }  
  
   public readonly int Bad;  
   public int Bad2;  
}  
  
[MyClass(5, Bad=0)] class Class1 {}   // CS0617  
[MyClass(5, Bad2=0)] class Class2 {}  
```
