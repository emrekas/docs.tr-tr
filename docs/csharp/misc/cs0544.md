# <a name="compiler-error-cs0544"></a><span data-ttu-id="09acf-102">Derleyici hatası CS0544</span><span class="sxs-lookup"><span data-stu-id="09acf-102">Compiler Error CS0544</span></span>
<span data-ttu-id="09acf-103">' özellik geçersiz kılma ': ' Özellik dışı ' bir özellik olmadığından geçersiz kılınamaz</span><span class="sxs-lookup"><span data-stu-id="09acf-103">'property override': cannot override because 'non-property' is not a property</span></span>  
  
 <span data-ttu-id="09acf-104">Özellik olmayan bir veri türünü [özellik](../programming-guide/classes-and-structs/properties.md)olarak geçersiz kılma girişiminde bulunuldu; buna izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="09acf-104">An attempt was made to override a nonproperty data type as a [property](../programming-guide/classes-and-structs/properties.md), which is not allowed.</span></span>  
  
 <span data-ttu-id="09acf-105">Aşağıdaki örnek CS0544 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="09acf-105">The following sample generates CS0544:</span></span>  
  
```csharp  
// CS0544.cs  
// compile with: /target:library  
public class a  
{  
   public int i;  
}  
  
public class b : a  
{  
   public override int i {   // CS0544  
   // try the following line instead  
   // public new int i {  
      get  
      {  
         return 0;  
      }  
   }  
}  
```
