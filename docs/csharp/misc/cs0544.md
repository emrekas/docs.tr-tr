# <a name="compiler-error-cs0544"></a><span data-ttu-id="50cb9-102">Derleyici Hatası CS0544</span><span class="sxs-lookup"><span data-stu-id="50cb9-102">Compiler Error CS0544</span></span>
<span data-ttu-id="50cb9-103">'özelliği geçersiz kılma': 'özelliği olmayan' bir özellik olmadığından geçersiz kılınamaz</span><span class="sxs-lookup"><span data-stu-id="50cb9-103">'property override': cannot override because 'non-property' is not a property</span></span>  
  
 <span data-ttu-id="50cb9-104">Nonproperty veri türü olarak geçersiz kılmak için bir girişimde bulunuldu bir [özelliği](../../csharp/programming-guide/classes-and-structs/properties.md), hangi izin verilmiyor.</span><span class="sxs-lookup"><span data-stu-id="50cb9-104">An attempt was made to override a nonproperty data type as a [property](../../csharp/programming-guide/classes-and-structs/properties.md), which is not allowed.</span></span>  
  
 <span data-ttu-id="50cb9-105">Aşağıdaki örnek, CS0544 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="50cb9-105">The following sample generates CS0544:</span></span>  
  
```csharp  
// CS0544.cs  
// compile with: /target:library  
public class a  
{  
   public int i;  
}  
  
public class b : a  
{  
   public override int i {   // CS0544  
   // try the following line instead  
   // public new int i {  
      get  
      {  
         return 0;  
      }  
   }  
}  
```
