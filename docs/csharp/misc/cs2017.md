# <a name="compiler-error-cs2017"></a>Derleyici Hatası CS2017
Modül veya kitaplık oluşturuluyorsa/Main belirtilemez  
  
 Oluşturmakta olduğunuz ana girdi noktası belirtilemiyor bir [/target: library](../../csharp/language-reference/compiler-options/target-library-compiler-option.md).  
  
 Aşağıdaki örnek, CS2017 oluşturur:  
  
```csharp  
// CS2017.cs  
// compile with: /main:MyClass /target:library  
// CS2017 expected  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
