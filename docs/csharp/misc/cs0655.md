## <a name="example"></a><span data-ttu-id="e2443-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="e2443-105">Example</span></span>  
 <span data-ttu-id="e2443-106">Aşağıdaki örnek CS0655 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="e2443-106">The following sample generates CS0655:</span></span>  
  
```csharp  
// CS0655.cs

{  
    // decimal is not valid attribute parameter type  
    public decimal d = 0;

}  
  
[My(d = 0)]   // CS0655  
// Try the following line instead:  
// [My(e = 0)]

{  
    public static void Main()  
    {  
    }  
}  
```
