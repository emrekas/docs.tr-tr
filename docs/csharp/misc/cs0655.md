# <a name="compiler-error-cs0655"></a><span data-ttu-id="c14b3-102">Derleyici Hatası CS0655</span><span class="sxs-lookup"><span data-stu-id="c14b3-102">Compiler Error CS0655</span></span>
<span data-ttu-id="c14b3-103">Geçerli bir öznitelik parametresi türü olmadığından 'parametresi' geçerli bir adlandırılmış öznitelik bağımsız değişkeni değil</span><span class="sxs-lookup"><span data-stu-id="c14b3-103">'parameter' is not a valid named attribute argument because it is not a valid attribute parameter type</span></span>  
  
 <span data-ttu-id="c14b3-104">Bkz: [öznitelikleri](../../csharp/programming-guide/concepts/attributes/index.md) bir öznitelik için geçerli parametre türleri hakkında ayrıntılı bilgi için.</span><span class="sxs-lookup"><span data-stu-id="c14b3-104">See [Attributes](../../csharp/programming-guide/concepts/attributes/index.md) for a discussion of valid parameter types for an attribute.</span></span>  
  
## <a name="example"></a><span data-ttu-id="c14b3-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="c14b3-105">Example</span></span>  
 <span data-ttu-id="c14b3-106">Aşağıdaki örnek, CS0655 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="c14b3-106">The following sample generates CS0655:</span></span>  
  
```csharp  
// CS0655.cs  
using System;  
  
class MyAttribute : Attribute  
{  
    // decimal is not valid attribute parameter type  
    public decimal d = 0;  
    public int e = 0;  
}  
  
[My(d = 0)]   // CS0655  
// Try the following line instead:  
// [My(e = 0)]  
class C  
{  
    public static void Main()  
    {  
    }  
}  
```
