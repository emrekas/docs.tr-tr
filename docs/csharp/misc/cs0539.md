# <a name="compiler-error-cs0539"></a><span data-ttu-id="5d6c4-102">Derleyici hatası CS0539</span><span class="sxs-lookup"><span data-stu-id="5d6c4-102">Compiler Error CS0539</span></span>
<span data-ttu-id="5d6c4-103">Açık arabirim bildirimindeki ' Member ', arabirimin üyesi değil</span><span class="sxs-lookup"><span data-stu-id="5d6c4-103">'member' in explicit interface declaration is not a member of interface</span></span>  
  
 <span data-ttu-id="5d6c4-104">Varolmayan bir [arabirim](../language-reference/keywords/interface.md) üyesini açıkça bildirmek için bir girişimde bulunuldu.</span><span class="sxs-lookup"><span data-stu-id="5d6c4-104">An attempt was made to explicitly declare an [interface](../language-reference/keywords/interface.md) member that does not exist.</span></span> <span data-ttu-id="5d6c4-105">Bildirimi silmeli ya da geçerli bir arabirim üyesine başvurması için değiştirmelisiniz.</span><span class="sxs-lookup"><span data-stu-id="5d6c4-105">You should either delete the declaration or change it so that it refers to a valid interface member.</span></span>  
  
 <span data-ttu-id="5d6c4-106">Aşağıdaki örnek CS0539 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="5d6c4-106">The following sample generates CS0539:</span></span>  
  
```csharp

   interface I  
   {  
      void m();  
   }  
  
   public class clx : I  
   {  
      void I.x()   // CS0539  
      {  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
