# <a name="compiler-error-cs0539"></a>Derleyici Hatası CS0539
Açık arabirim bildirimindeki ' member' arabirimin bir üyesi değil  
  
 Açıkça bildirmek için bir girişimde bulunuldu bir [arabirimi](../../csharp/language-reference/keywords/interface.md) var olmayan üye. Bildirimi Sil veya geçerli bir arabirim üyesi başvurduğu için değiştirin.  
  
 Aşağıdaki örnek, CS0539 oluşturur:  
  
```csharp  
// CS0539.cs  
namespace x  
{  
   interface I  
   {  
      void m();  
   }  
  
   public class clx : I  
   {  
      void I.x()   // CS0539  
      {  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
