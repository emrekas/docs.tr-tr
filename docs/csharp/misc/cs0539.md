<span data-ttu-id="09309-103">Açık arabirim bildirimindeki ' member' arabirimin bir üyesi değil</span><span class="sxs-lookup"><span data-stu-id="09309-103">'member' in explicit interface declaration is not a member of interface</span></span>  
  
 <span data-ttu-id="09309-104">Açıkça bildirmek için bir girişimde bulunuldu bir [arabirimi](../../csharp/language-reference/keywords/interface.md) var olmayan üye.</span><span class="sxs-lookup"><span data-stu-id="09309-104">An attempt was made to explicitly declare an [interface](../../csharp/language-reference/keywords/interface.md) member that does not exist.</span></span> <span data-ttu-id="09309-105">Bildirimi Sil veya geçerli bir arabirim üyesi başvurduğu için değiştirin.</span><span class="sxs-lookup"><span data-stu-id="09309-105">You should either delete the declaration or change it so that it refers to a valid interface member.</span></span>  
  
 <span data-ttu-id="09309-106">Aşağıdaki örnek, CS0539 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="09309-106">The following sample generates CS0539:</span></span>

```csharp  
      void m();

      }

      {  
         return 0;  
      }  
   }  
}  
```
