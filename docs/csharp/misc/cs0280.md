<span data-ttu-id="03d99-103">'type', 'desenini ad' desenini uygulamıyor.</span><span class="sxs-lookup"><span data-stu-id="03d99-103">'type' does not implement the 'pattern name' pattern.</span></span> <span data-ttu-id="03d99-104">'yöntem adı' yanlış imzaya sahip.</span><span class="sxs-lookup"><span data-stu-id="03d99-104">'method name' has the wrong signature.</span></span>  
  
 <span data-ttu-id="03d99-105">İki deyimlerinde C#, **foreach** ve **kullanarak**, önceden tanımlanmış desenler, "koleksiyonu" ve "kaynak" sırasıyla kullanır.</span><span class="sxs-lookup"><span data-stu-id="03d99-105">Two statements in C#, **foreach** and **using**, rely on predefined patterns, "collection" and "resource" respectively.</span></span> <span data-ttu-id="03d99-106">Bu uyarı, derleyici bir ifade desenine nedeniyle yanlış yöntemin imzası için eşleşemez oluşur.</span><span class="sxs-lookup"><span data-stu-id="03d99-106">This warning occurs when the compiler cannot match one of these statements to its pattern due to a method's incorrect signature.</span></span> <span data-ttu-id="03d99-107">Örneğin, "koleksiyon" deseni var olması adlı bir yöntem gerektiren <xref:System.Collections.IEnumerator.MoveNext%2A> hiçbir parametre almayan ve döndüren bir `boolean`.</span><span class="sxs-lookup"><span data-stu-id="03d99-107">For example, the "collection" pattern requires that there be a method called <xref:System.Collections.IEnumerator.MoveNext%2A> which takes no parameters and returns a `boolean`.</span></span> <span data-ttu-id="03d99-108">Kodunuzu içerebilir bir <xref:System.Collections.IEnumerator.MoveNext%2A> yönteminin bir parametre içeriyor veya belki de bir nesne döndürür.</span><span class="sxs-lookup"><span data-stu-id="03d99-108">Your code might contain a <xref:System.Collections.IEnumerator.MoveNext%2A> method that has a parameter or perhaps returns an object.</span></span>  
  
 <span data-ttu-id="03d99-109">"Kaynak" deseni ve `using` başka bir örnek sağlar.</span><span class="sxs-lookup"><span data-stu-id="03d99-109">The "resource" pattern and `using` provide another example.</span></span> <span data-ttu-id="03d99-110">"Kaynak" deseni gerektirir <xref:System.IDisposable.Dispose%2A> yöntemi; aynı ada sahip bir özellik tanımlarsanız, bu uyarı alırsınız.</span><span class="sxs-lookup"><span data-stu-id="03d99-110">The "resource" pattern requires the <xref:System.IDisposable.Dispose%2A> method; if you define a property with the same name, you will get this warning.</span></span>  
  
 <span data-ttu-id="03d99-111">Bu uyarıyı çözmek için türünüz yöntem imzaları karşılık gelen yöntemleri desenindeki imzalarını eşleşen ve hiçbir özellik deseni tarafından gerekli yöntemi olarak aynı ada sahip olduğundan emin olun emin olun.</span><span class="sxs-lookup"><span data-stu-id="03d99-111">To resolve this warning, ensure that the method signatures in your type match the signatures of the corresponding methods in the pattern, and ensure that you have no properties with the same name as a method required by the pattern.</span></span>  
  
## <a name="example"></a><span data-ttu-id="03d99-112">Örnek</span><span class="sxs-lookup"><span data-stu-id="03d99-112">Example</span></span>  
 <span data-ttu-id="03d99-113">Aşağıdaki örnek, CS0280 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="03d99-113">The following sample generates CS0280.</span></span>  
  
```csharp  
// CS0280.cs  
using System;  
using System.Collections;  
  
public class ValidBase: IEnumerable  
{  
   IEnumerator IEnumerable.GetEnumerator()  
   {  
      yield return 0;  
   }  
  
   internal IEnumerator GetEnumerator()  
   {  
      yield return 0;  
   }  
}  
  
class Derived : ValidBase  
{  
   // field, not method  
   new public int GetEnumerator;  
}  
  
public class Test  
{  
   public static void Main()  
   {  
      foreach (int i in new Derived()) {}   // CS0280  
   }  
}  
```
