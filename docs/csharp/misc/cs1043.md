# <a name="compiler-error-cs1043"></a>Derleyici Hatası CS1043
{veya; bekleniyor  
  
 Bir özellik erişimcisi yanlış bildirildi. Daha fazla bilgi için [kullanarak özelliklerini](../../csharp/programming-guide/classes-and-structs/using-properties.md).  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS1043 oluşturur.  
  
```csharp  
// CS1043.cs  
// compile with: /target:library  
public class MyClass  
{  
   public int DoSomething  
   {  
      get return 1;   // CS1043  
      set {}  
   }  
  
   // OK  
   public int DoSomething2  
   {  
      get { return 1;}  
   }  
}  
```
