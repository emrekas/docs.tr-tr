# <a name="compiler-error-cs0502"></a><span data-ttu-id="4e720-102">Derleyici hatası CS0502</span><span class="sxs-lookup"><span data-stu-id="4e720-102">Compiler Error CS0502</span></span>
<span data-ttu-id="4e720-103">' Member ' hem abstract hem de Sealed olamaz</span><span class="sxs-lookup"><span data-stu-id="4e720-103">'member' cannot be both abstract and sealed</span></span>  
  
 <span data-ttu-id="4e720-104">Bir sınıf üyesi hem [abstract](../language-reference/keywords/abstract.md) hem de [Sealed](../language-reference/keywords/sealed.md)olamaz.</span><span class="sxs-lookup"><span data-stu-id="4e720-104">A class member cannot be both [abstract](../language-reference/keywords/abstract.md) and [sealed](../language-reference/keywords/sealed.md).</span></span>  
  
 <span data-ttu-id="4e720-105">Aşağıdaki örnek CS0502 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="4e720-105">The following sample generates CS0502:</span></span>  
  
```csharp  
// CS0502.cs  
public class B  
{  
   abstract public void F();  
}  
  
public class C : B  
{  
   abstract sealed override public void F()   // CS0502  
   {  
   }  
}  
  
public class CMain  
{  
    public static void Main()  
    { }  
}  
```
