# <a name="compiler-error-cs0502"></a><span data-ttu-id="a5537-102">Derleyici Hatası CS0502</span><span class="sxs-lookup"><span data-stu-id="a5537-102">Compiler Error CS0502</span></span>
<span data-ttu-id="a5537-103">'member' hem soyut hem korumalı olamaz</span><span class="sxs-lookup"><span data-stu-id="a5537-103">'member' cannot be both abstract and sealed</span></span>  
  
 <span data-ttu-id="a5537-104">Bir sınıf üyesinin her ikisi birden olamaz [soyut](../../csharp/language-reference/keywords/abstract.md) ve [korumalı](../../csharp/language-reference/keywords/sealed.md).</span><span class="sxs-lookup"><span data-stu-id="a5537-104">A class member cannot be both [abstract](../../csharp/language-reference/keywords/abstract.md) and [sealed](../../csharp/language-reference/keywords/sealed.md).</span></span>  
  
 <span data-ttu-id="a5537-105">Aşağıdaki örnek, CS0502 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="a5537-105">The following sample generates CS0502:</span></span>  
  
```csharp  
// CS0502.cs  
public class B  
{  
   abstract public void F();  
}  
  
public class C : B  
{  
   abstract sealed override public void F()   // CS0502  
   {  
   }  
}  
  
public class CMain  
{  
    public static void Main()  
    { }  
}  
```
