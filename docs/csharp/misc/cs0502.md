# <a name="compiler-error-cs0502"></a>Derleyici Hatası CS0502
'member' hem soyut hem korumalı olamaz  
  
 Bir sınıf üyesinin her ikisi birden olamaz [soyut](../../csharp/language-reference/keywords/abstract.md) ve [korumalı](../../csharp/language-reference/keywords/sealed.md).  
  
 Aşağıdaki örnek, CS0502 oluşturur:  
  
```csharp  
// CS0502.cs  
public class B  
{  
   abstract public void F();  
}  
  
public class C : B  
{  
   abstract sealed override public void F()   // CS0502  
   {  
   }  
}  
  
public class CMain  
{  
    public static void Main()  
    { }  
}  
```
