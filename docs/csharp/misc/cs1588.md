# <a name="compiler-error-cs1588"></a>Derleyici Hatası CS1588
Ortak Dil Çalışma Zamanı Modülü dizini--'açıklaması' belirlenemiyor  
  
 Derleyici, .NET Framework ortak dil çalışma zamanı yüklü dizine alınamıyor. Bu hata, ortak dil çalışma zamanının düzgün yüklendiğini gösterir.
