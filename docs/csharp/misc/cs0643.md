# <a name="compiler-error-cs0643"></a>Derleyici hatası CS0643
' arg ' yinelenen adlandırılmış öznitelik bağımsız değişkeni  
  
 Kullanıcı tanımlı bir `arg`öznitelikte bir parametre iki kez belirtildi. Daha fazla bilgi için bkz. [öznitelikler](../programming-guide/concepts/attributes/index.md).  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS0643 oluşturur:  
  
```csharp  
// CS0643.cs  
using System;  
using System.Runtime.InteropServices;  
  
[AttributeUsage(AttributeTargets.Class)]  
public class MyAttribute : Attribute  
{  
    public MyAttribute()  
    {  
    }  
  
    public int x;  
}  
  
[MyAttribute(x = 5, x = 6)]   // CS0643, error setting x twice  
// try the following line instead  
// [MyAttribute(x = 5)]  
class MyClass  
{  
}  
  
public class MainClass  
{  
    public static void Main ()  
    {  
    }  
}  
```
