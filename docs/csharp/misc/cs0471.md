# <a name="compiler-error-cs0471"></a>Derleyici Hatası CS0471
'Name' yöntemi genel bir yöntem değildir. Bir ifade listesi istiyorsanız, parantez içinde kullanın < ifade.  
  
 Bu hata, kodunuzu içeren bir ifade listesi parantez olmadan oluşturulur.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS0471 oluşturur.  
  
```csharp  
// CS0471.cs  
// compile with: /t:library  
class Test  
{  
    public void F(bool x, bool y) {}  
    public void F1()  
    {  
        int a = 1, b = 2, c = 3;  
        F(a<b, c>(3));    // CS0471  
        // To resolve, try the following instead:  
        // F((a<b), c>(3));  
    }  
}  
```
