# <a name="compiler-error-cs0152"></a>Derleyici Hatası CS0152
'Etiketi' etiketi bu switch deyiminde zaten  
  
 Bir etiket içinde yinelendi bir [geçiş](../../csharp/language-reference/keywords/switch.md) deyimi. Daha fazla bilgi için [geçiş](../../csharp/language-reference/keywords/switch.md).  
  
 Aşağıdaki örnek, CS0152 oluşturur:  
  
```csharp  
// CS0152.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
  
         switch (i)  
         {  
            case 1:  
               i++;  
               return;  
  
            case 1:   // CS0152, two case 1 statements  
               i++;  
               return;  
         }  
      }  
   }  
}  
```
