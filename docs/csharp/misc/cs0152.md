<span data-ttu-id="fd1f2-103">' Label ' etiketi bu Switch ifadesinde zaten geçiyor</span><span class="sxs-lookup"><span data-stu-id="fd1f2-103">The label 'label' already occurs in this switch statement</span></span>  
  
 <span data-ttu-id="fd1f2-104">[Switch](../language-reference/keywords/switch.md) ifadesinde bir etiket tekrarlandı.</span><span class="sxs-lookup"><span data-stu-id="fd1f2-104">A label was repeated in a [switch](../language-reference/keywords/switch.md) statement.</span></span> <span data-ttu-id="fd1f2-105">Daha fazla bilgi için bkz. [anahtar](../language-reference/keywords/switch.md).</span><span class="sxs-lookup"><span data-stu-id="fd1f2-105">For more information, see [switch](../language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="fd1f2-106">Aşağıdaki örnek CS0152 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="fd1f2-106">The following sample generates CS0152:</span></span>  
  
```csharp  
// CS0152.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
  
         switch (i)  
         {  
            case 1:  
               i++;  
               return;  
  
            case 1:   // CS0152, two case 1 statements  
               i++;  
               return;  
         }  
      }  
   }  
}  
```
