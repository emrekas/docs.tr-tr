# <a name="compiler-error-cs1020"></a>Derleyici Hatası CS1020
Aşırı yüklenebilir ikili işleç bekleniyor  
  
 Operatör aşırı yüklemesi tanımlamak için bir girişimde bulunuldu ancak işleci iki parametre alan bir aşırı yüklenebilir ikili işleç değil. Fazla yüklenebilir işleçler listesi için bkz. [fazla yüklenebilir işleçler](../../csharp/language-reference/operators/operator-overloading.md#overloadable-operators) bölümünü [işleci aşırı yüklemesi](../../csharp/language-reference/operators/operator-overloading.md) makalesi.
  
 Aşağıdaki örnek, CS1020 oluşturur:  
  
```csharp  
// CS1020.cs  
public class iii  
{  
   public static int operator ++(iii aa, int bb)   // CS1020, change ++ to +  
   {  
      return 0;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
