# <a name="compiler-warning-level-3-cs1702"></a>Derleyici Uyarısı (Düzey 3) CS1702
Bütünleştirilmiş kod başvurusu "Derleme adı #1" "Derleme adı #2" ile eşleşen varsayıldığında, çalışma zamanı ilkesini sağlamanız gerekebilir  
  
 İki derleme başvurularını farklı derleme ve/veya düzeltme numaralarını değil otomatik olarak birleştirecektir şekilde vardır. Birleştirme uygulamanın .config dosyasında yönergeleri kullanarak zorlamak için çalışma zamanı ilkesini sağlamanız gerekebilir.
