# <a name="compiler-error-cs0208"></a>Derleyici hatası CS0208
Adresinin adresi alınamaz, boyutu alınamaz veya bir işaretçi, yönetilen bir türe (' Type ') göre bildirilemez  
  
 [Güvenli olmayan](../../csharp/language-reference/keywords/unsafe.md) anahtar sözcükle birlikte kullanıldığında, yönetilen bir nesnenin adresini alma, yönetilen bir nesnenin boyutunu alma veya yönetilen bir türe işaretçi bildirme için de izin verilmez. Yönetilen bir tür:  
  
- herhangi bir başvuru türü  
  
- bir alan veya özellik olarak başvuru türü içeren herhangi bir struct  
  
 Daha fazla bilgi için bkz. [yönetilmeyen türler](../../csharp/language-reference/builtin-types/unmanaged-types.md).  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS0208 oluşturur:  
  
```csharp  
// CS0208.cs  
// compile with: /unsafe  
  
class myClass  
{  
    public int a = 98;  
}  
  
struct myProblemStruct  
{  
    string s;  
    float f;  
}  
  
struct myGoodStruct  
{  
    int i;  
    float f;  
}  
  
public class MyClass  
{  
    unsafe public static void Main()  
    {  
        // myClass is a class, a managed type.  
        myClass s = new myClass();    
        myClass* s2 = &s;    // CS0208  
  
        // The struct contains a string, a managed type.  
        int i = sizeof(myProblemStruct); //CS0208  
  
        // The struct contains only value types.  
        i = sizeof(myGoodStruct); //OK  
  
    }  
}  
```
