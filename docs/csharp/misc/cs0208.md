<span data-ttu-id="efeed-103">Boyutunu alma adresi alamaz, veya yönetilen türün ('type') bir işaretçi bildirme</span><span class="sxs-lookup"><span data-stu-id="efeed-103">Cannot take the address of, get the size of, or declare a pointer to a managed type ('type')</span></span>  
  
 <span data-ttu-id="efeed-104">Hatta ile kullanıldığında [güvenli](../../csharp/language-reference/keywords/unsafe.md) anahtar sözcüğü, yönetilen bir nesnenin adresini almak, yönetilen bir nesnenin boyutunu alma veya bildirme bir yönetilen türün bir işaretçiye izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="efeed-104">Even when used with the [unsafe](../../csharp/language-reference/keywords/unsafe.md) keyword, taking the address of a managed object, getting the size of a managed object, or declaring a pointer to a managed type is not allowed.</span></span> <span data-ttu-id="efeed-105">Yönetilen bir türdür:</span><span class="sxs-lookup"><span data-stu-id="efeed-105">A managed type is:</span></span>  
  
- <span data-ttu-id="efeed-106">herhangi bir başvuru türü</span><span class="sxs-lookup"><span data-stu-id="efeed-106">any reference type</span></span>

 <span data-ttu-id="efeed-108">Daha fazla bilgi için [güvenli olmayan kod ve işaretçiler](../../csharp/programming-guide/unsafe-code-pointers/index.md) ve [sizeof](../../csharp/language-reference/keywords/sizeof.md).</span><span class="sxs-lookup"><span data-stu-id="efeed-108">For more information, see [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md) and [sizeof](../../csharp/language-reference/keywords/sizeof.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="efeed-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="efeed-109">Example</span></span>  
 <span data-ttu-id="efeed-110">Aşağıdaki örnek, CS0208 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="efeed-110">The following sample generates CS0208:</span></span>  
  
```csharp  
// CS0208.cs  
// compile with: /unsafe  
  
class myClass  
{  
    public int a = 98;  
}  
  
struct myProblemStruct  
{  
    string s;  
    float f;  
}  
  
struct myGoodStruct  
{  
    int i;  
    float f;  
}  
  
public class MyClass  
{  
    unsafe public static void Main()  
    {  
        // myClass is a class, a managed type.  
        myClass s = new myClass();    
        myClass* s2 = &s;    // CS0208  
  
        // The struct contains a string, a managed type.  
        int i = sizeof(myProblemStruct); //CS0208  
  
        // The struct contains only value types.  
        i = sizeof(myGoodStruct); //OK  
  
    }  
}  
```
