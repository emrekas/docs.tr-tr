# <a name="compiler-error-cs2001"></a>Derleyici Hatası CS2001
Kaynak dosya 'dosyası bulunamadı  
  
 Kaynak dosya adı için derleyici geçirilen, ancak bulunamadı. Dosya adının yazımını ve dosya konumunu denetleyin.
