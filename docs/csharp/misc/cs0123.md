# <a name="compiler-error-cs0123"></a>Derleyici Hatası CS0123
'Method' eşleşme temsilci 'temsilci' için hiçbir aşırı yüklemesi  
  
 Doğru imza kullanılmadığı için bir temsilci oluşturma girişimi başarısız oldu. Bir temsilci örneğini temsilci bildirimi aynı imzaya sahip bildirilmelidir.  
  
 Yöntem ya da ayarlayarak bu hatayı gidermek veya temsilci imzası. Daha fazla bilgi için [Temsilciler](../../csharp/programming-guide/delegates/index.md).  
  
 Aşağıdaki örnek, CS0123 oluşturur.  
  
```csharp  
// CS0123.cs  
delegate void D();  
delegate void D2(int i);  
  
public class C  
{  
   public static void f(int i) {}  
  
   public static void Main()  
   {  
      D d = new D(f);   // CS0123  
      D2 d2 = new D2(f);   // OK  
   }  
}  
```
