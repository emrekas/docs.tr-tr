# <a name="compiler-error-cs0123"></a><span data-ttu-id="7ff89-102">Derleyici Hatası CS0123</span><span class="sxs-lookup"><span data-stu-id="7ff89-102">Compiler Error CS0123</span></span>
<span data-ttu-id="7ff89-103">'Method' eşleşme temsilci 'temsilci' için hiçbir aşırı yüklemesi</span><span class="sxs-lookup"><span data-stu-id="7ff89-103">No overload for 'method' matches delegate 'delegate'</span></span>  
  
 <span data-ttu-id="7ff89-104">Doğru imza kullanılmadığı için bir temsilci oluşturma girişimi başarısız oldu.</span><span class="sxs-lookup"><span data-stu-id="7ff89-104">An attempt to create a delegate failed because the correct signature was not used.</span></span> <span data-ttu-id="7ff89-105">Bir temsilci örneğini temsilci bildirimi aynı imzaya sahip bildirilmelidir.</span><span class="sxs-lookup"><span data-stu-id="7ff89-105">Instances of a delegate must be declared with the same signature as the delegate declaration.</span></span>  
  
 <span data-ttu-id="7ff89-106">Yöntem ya da ayarlayarak bu hatayı gidermek veya temsilci imzası.</span><span class="sxs-lookup"><span data-stu-id="7ff89-106">You can resolve this error by adjusting either the method or delegate signature.</span></span> <span data-ttu-id="7ff89-107">Daha fazla bilgi için [Temsilciler](../../csharp/programming-guide/delegates/index.md).</span><span class="sxs-lookup"><span data-stu-id="7ff89-107">For more information, see [Delegates](../../csharp/programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="7ff89-108">Aşağıdaki örnek, CS0123 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="7ff89-108">The following sample generates CS0123.</span></span>  
  
```csharp  
// CS0123.cs  
delegate void D();  
delegate void D2(int i);  
  
public class C  
{  
   public static void f(int i) {}  
  
   public static void Main()  
   {  
      D d = new D(f);   // CS0123  
      D2 d2 = new D2(f);   // OK  
   }  
}  
```
