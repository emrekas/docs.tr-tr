# <a name="compiler-error-cs0541"></a><span data-ttu-id="7f29c-102">Derleyici hatası CS0541</span><span class="sxs-lookup"><span data-stu-id="7f29c-102">Compiler Error CS0541</span></span>
<span data-ttu-id="7f29c-103">' declaration ': açık arabirim bildirimi yalnızca bir sınıf veya yapı içinde bildirilemez</span><span class="sxs-lookup"><span data-stu-id="7f29c-103">'declaration' : explicit interface declaration can only be declared in a class or struct</span></span>  
  
 <span data-ttu-id="7f29c-104">Bir [sınıf](../language-reference/keywords/class.md) veya [Yapı](../language-reference/keywords/struct.md)dışında açık bir [arabirim](../language-reference/keywords/interface.md) bildirimi bulundu.</span><span class="sxs-lookup"><span data-stu-id="7f29c-104">An explicit [interface](../language-reference/keywords/interface.md) declaration was found outside a [class](../language-reference/keywords/class.md) or [struct](../language-reference/keywords/struct.md).</span></span>  
  
 <span data-ttu-id="7f29c-105">Aşağıdaki örnek CS0541 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="7f29c-105">The following sample generates CS0541:</span></span>  
  
```csharp  
// CS0541.cs  
namespace x  
{  
   interface IFace  
   {  
      void F();  
   }  
  
   interface IFace2 : IFace  
   {  
      void IFace.F();   // CS0541  
   }  
}  
```
