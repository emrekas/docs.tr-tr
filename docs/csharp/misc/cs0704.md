<span data-ttu-id="5fae2-103">Bir tür parametresi olduğundan 'type' içinde üye araması yapılamıyor</span><span class="sxs-lookup"><span data-stu-id="5fae2-103">Cannot do member lookup in 'type' because it is a type parameter</span></span>  
  
 <span data-ttu-id="5fae2-104">Bir iç türü ile bir tür parametresi belirtilemez.</span><span class="sxs-lookup"><span data-stu-id="5fae2-104">An inner type cannot be specified through a type parameter.</span></span> <span data-ttu-id="5fae2-105">İstenen türe açıkça kullanmayı deneyin.</span><span class="sxs-lookup"><span data-stu-id="5fae2-105">Try using the desired type explicitly.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5fae2-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="5fae2-106">Example</span></span>  
 <span data-ttu-id="5fae2-107">Aşağıdaki örnek, CS0704 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="5fae2-107">The following sample generates CS0704:</span></span>  
  
```csharp  
// CS0704.cs  
class B  
{  
    public class I { }  
}  
  
class C<T> where T : B  
{  
    T.I f;  // CS0704 – member lookup on type parameter  
    // Try this instead:  
    // B.I f;  
}  
  
class CMain  
{  
    public static void Main() {}  
}  
```
