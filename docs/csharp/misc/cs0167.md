# <a name="compiler-error-cs0167"></a>Derleyici Hatası CS0167
' % S'temsilci 'temsilci' Invoke yöntemi yok  
  
 Alınan ve yönetilen bir program (.NET Framework ortak dil çalışma zamanı kullanan bir) kullanılan başka bir derleyici ile oluşturulmuş. Bu derleyici tarafından izin verilen hatalı oluşturulmuş bir [temsilci](../../csharp/language-reference/keywords/delegate.md). Bu nedenle, `Invoke` yöntemi kullanılabilir değildi. Daha fazla bilgi için [Temsilciler](../../csharp/programming-guide/delegates/index.md).
