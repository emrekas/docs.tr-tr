# <a name="compiler-error-cs1521"></a>Derleyici Hatası CS1521
Geçersiz temel tür  
  
 A [temel](../../csharp/language-reference/keywords/base.md) sınıfı belirtimi ill biçimlendirilmiş.  
  
 Aşağıdaki örnek, CS1521 oluşturur:  
  
```csharp  
// CS1521.cs  
class CMyClass  
{  
   public static void Main()  
   {  
   }  
}  
  
class CMyClass1 : CMyClass     // OK  
{  
}  
  
class CMyClass2 : CMyClass[]   // CS1521  
{  
}  
  
class CMyClass3 : CMyClass*    // CS1521  
{  
}  
```
