 <span data-ttu-id="8b386-104">[Temel](../language-reference/keywords/base.md) sınıf belirtimi hatalı biçimlendirilmiş.</span><span class="sxs-lookup"><span data-stu-id="8b386-104">A [base](../language-reference/keywords/base.md) class specification was ill formed.</span></span>  
  
 <span data-ttu-id="8b386-105">Aşağıdaki örnek CS1521 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="8b386-105">The following sample generates CS1521:</span></span>  
  
```csharp  
// CS1521.cs

{  
   public static void Main()  
   {  
   }  
}

{  
}

{  
}

{  
}  
```
