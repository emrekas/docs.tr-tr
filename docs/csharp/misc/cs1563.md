# <a name="compiler-error-cs1563"></a>Derleyici Hatası CS1563
'Çıkış dosya' çıktı kaynak dosya yok  
  
 [/Out](../../csharp/language-reference/compiler-options/out-compiler-option.md) derleyici seçeneği belirtildi, ancak hiçbir kaynak kodu dosyalarını izleyin. Ya da kullanmamalısınız **/out**, veya aşağıdaki kaynak kodu dosyaları belirtmelidir **/out**.
