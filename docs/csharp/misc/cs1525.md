# <a name="compiler-error-cs1525"></a>Derleyici Hatası CS1525
Geçersiz ifade terimi 'character'  
  
 Derleyici bir ifadede geçersiz karakter algılandı.  
  
 Aşağıdaki örnek, CS1525 oluşturur:  
  
```csharp  
// CS1525.cs  
class x  
{  
   public static void Main()  
   {  
      int i = 0;  
      i = i +   // OK - identifier  
      'c' +     // OK - character  
      (5) +     // OK - parenthesis  
      [ +       // CS1525, operator not a valid expression element  
      throw +   // CS1525, keyword not allowed in expression  
      void;     // CS1525, void not allowed in expression  
   }  
}  
```  
  
 Boş bir etiket, aşağıdaki örnekte olduğu gibi CS1525 da oluşturabilirsiniz:  
  
```csharp  
// CS1525b.cs  
using System;  
public class MyClass  
{  
   public static void Main()  
   {  
      goto FoundIt;  
      FoundIt:      // CS1525  
      // Uncomment the following line to resolve:  
      // System.Console.Write("Hello");  
   }  
}  
```
