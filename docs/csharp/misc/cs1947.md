## <a name="example"></a><span data-ttu-id="9a7d3-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="9a7d3-109">Example</span></span>  
 <span data-ttu-id="9a7d3-110">Aşağıdaki kod CS1947 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="9a7d3-110">The following code generates CS1947:</span></span>  
  
```csharp  
// cs1947.cs  
using System.Linq;  
class Test  
{  
    static void Main()  
    {  
        int[] array = new int[] { 1, 2, 3, 4, 5 };  
        var x = from i in array  
                let k = i  
                select i = 5; // CS1947  
        x.ToList();  
    }  
}  
```
