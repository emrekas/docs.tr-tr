# <a name="compiler-error-cs0555"></a><span data-ttu-id="f09fb-102">Derleyici Hatası CS0555</span><span class="sxs-lookup"><span data-stu-id="f09fb-102">Compiler Error CS0555</span></span>

<span data-ttu-id="f09fb-103">Kullanıcı tanımlı işleç kapsayan türdeki bir nesne almak ve kapsayan türdeki bir nesneye dönüştürmek</span><span class="sxs-lookup"><span data-stu-id="f09fb-103">User-defined operator cannot take an object of the enclosing type and convert to an object of the enclosing type</span></span>

<span data-ttu-id="f09fb-104">Kapsayan sınıfın kullanıcı tanımlı dönüşümler değerlere izin verilmiyor; Böyle bir işleç gerekmez.</span><span class="sxs-lookup"><span data-stu-id="f09fb-104">User-defined conversions to values of the enclosing class are not allowed; you do not need such an operator.</span></span>

<span data-ttu-id="f09fb-105">Aşağıdaki örnek, CS0555 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="f09fb-105">The following sample generates CS0555:</span></span>

```csharp
// CS0555.cs
public class MyClass
{
   // delete the following operator to resolve this CS0555
   public static implicit operator MyClass(MyClass aa)   // CS0555
   {
      return new MyClass();
   }

   public static void Main()
   {
   }
}
```
