# <a name="compiler-error-cs1906"></a>Derleyici Hatası CS1906
Geçersiz seçenek 'seçeneği'; Kaynak görünürlüğü 'public' veya 'private' olmalıdır  
  
 Bu hata, geçersiz bir gösterir [/Resource (kaynak dosyayı çıktıya)](../../csharp/language-reference/compiler-options/resource-compiler-option.md) veya [/linkresource (.NET Framework kaynağına bağlantı)](../../csharp/language-reference/compiler-options/linkresource-compiler-option.md) komut satırı seçeneği. Sözdizimini denetleyin **/Resource** veya **/linkresource** komut satırı seçeneği ve kullanılan erişilebilirlik değiştiricisi ya da olduğundan emin olun **genel** veya `private`.
