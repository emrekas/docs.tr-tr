# <a name="compiler-error-cs1715"></a><span data-ttu-id="0d8f3-102">Derleyici Hatası CS1715</span><span class="sxs-lookup"><span data-stu-id="0d8f3-102">Compiler Error CS1715</span></span>
<span data-ttu-id="0d8f3-103">'Type1': türü, 'Type2' olmalıdır 'MemberName' geçersiz kılınan üyesiyle eşleştirmek için</span><span class="sxs-lookup"><span data-stu-id="0d8f3-103">'Type1': type must be 'Type2' to match overridden member 'MemberName'</span></span>  
  
 <span data-ttu-id="0d8f3-104">Bu hatayla aynı şeydir [derleyici hatası CS0508](../../csharp/misc/cs0508.md)dışında CS0508 artık yalnızca CS1715 'dönüş türleri yerine' özellikleri ve yalnızca 'types' sahip bir dizin oluşturucular için uygularken dönüş türlerine sahip yöntemleri için geçerlidir.</span><span class="sxs-lookup"><span data-stu-id="0d8f3-104">This error is the same as [Compiler Error CS0508](../../csharp/misc/cs0508.md), except that CS0508 now only applies to methods that have return types, while CS1715 applies to properties and indexers that only have 'types' instead of 'return types'.</span></span>  
  
## <a name="example"></a><span data-ttu-id="0d8f3-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="0d8f3-105">Example</span></span>  
 <span data-ttu-id="0d8f3-106">Aşağıdaki kod CS1715 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="0d8f3-106">The following code generates CS1715.</span></span>  
  
```csharp  
// CS1715.cs  
abstract public class Base  
{  
    abstract public int myProperty  
    {  
        get;  
        set;  
    }  
}  
  
public class Derived : Base  
{  
    int myField;  
    public override double myProperty  // CS1715  
    // try the following line instead  
    // public override int myProperty  
    {  
        get { return myField; }  
        set { myField;= value; }  
    }  
  
    public static void Main()  
    {  
        Derived d = new Derived();  
        d.myProperty = 5;  
    }  
}  
```
