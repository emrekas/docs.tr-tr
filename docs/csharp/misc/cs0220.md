# <a name="compiler-error-cs0220"></a><span data-ttu-id="0a15c-102">Derleyici Hatası CS0220</span><span class="sxs-lookup"><span data-stu-id="0a15c-102">Compiler Error CS0220</span></span>
<span data-ttu-id="0a15c-103">İşlem derleme zamanında işaretlenmiş modunda taşıyor</span><span class="sxs-lookup"><span data-stu-id="0a15c-103">The operation overflows at compile time in checked mode</span></span>  
  
 <span data-ttu-id="0a15c-104">Bir işlem tarafından algılandı [işaretli](../../csharp/language-reference/keywords/checked.md), varsayılan değer olan ve bir veri kaybı sonuçlandı.</span><span class="sxs-lookup"><span data-stu-id="0a15c-104">An operation was detected by [checked](../../csharp/language-reference/keywords/checked.md), which is the default, and a data loss resulted.</span></span> <span data-ttu-id="0a15c-105">Ya da atama veya kullanım girişleri düzeltmek [denetlenmeyen](../../csharp/language-reference/keywords/unchecked.md) bu hatayı gidermek için.</span><span class="sxs-lookup"><span data-stu-id="0a15c-105">Either correct the inputs to the assignment or use [unchecked](../../csharp/language-reference/keywords/unchecked.md) to resolve this error.</span></span> <span data-ttu-id="0a15c-106">Daha fazla bilgi için [Checked ve Unchecked](../../csharp/language-reference/keywords/checked-and-unchecked.md).</span><span class="sxs-lookup"><span data-stu-id="0a15c-106">For more information, see [Checked and Unchecked](../../csharp/language-reference/keywords/checked-and-unchecked.md).</span></span>  
  
 <span data-ttu-id="0a15c-107">Aşağıdaki örnek, CS0220 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="0a15c-107">The following sample generates CS0220:</span></span>  
  
```csharp  
// CS0220.cs  
using System;  
  
class TestClass  
{  
   const int x = 1000000;  
   const int y = 1000000;  
  
   public int MethodCh()  
   {  
      int z = (x * y);   // CS0220  
      return z;  
   }  
  
   public int MethodUnCh()  
   {  
      unchecked  
      {  
         int z = (x * y);  
         return z;  
      }  
   }  
  
   public static void Main()  
   {  
      TestClass myObject = new TestClass();  
      Console.WriteLine("Checked  : {0}", myObject.MethodCh());  
      Console.WriteLine("Unchecked: {0}", myObject.MethodUnCh());  
   }  
}  
```
