# <a name="compiler-error-cs0220"></a><span data-ttu-id="fd87d-102">Derleyici hatası CS0220</span><span class="sxs-lookup"><span data-stu-id="fd87d-102">Compiler Error CS0220</span></span>
<span data-ttu-id="fd87d-103">İşlem, denetlenen modda derleme zamanında taşıyor</span><span class="sxs-lookup"><span data-stu-id="fd87d-103">The operation overflows at compile time in checked mode</span></span>  
  
 <span data-ttu-id="fd87d-104">Bir işlem [denetlenen](../language-reference/keywords/checked.md)tarafından algılandı, bu varsayılan değer, bir veri kaybı ile sonuçlandı.</span><span class="sxs-lookup"><span data-stu-id="fd87d-104">An operation was detected by [checked](../language-reference/keywords/checked.md), which is the default, and a data loss resulted.</span></span> <span data-ttu-id="fd87d-105">Atama girdilerini düzeltin ya da bu hatayı çözmek için [işaretlenmemiş](../language-reference/keywords/unchecked.md) kullanın.</span><span class="sxs-lookup"><span data-stu-id="fd87d-105">Either correct the inputs to the assignment or use [unchecked](../language-reference/keywords/unchecked.md) to resolve this error.</span></span> <span data-ttu-id="fd87d-106">Daha fazla bilgi için bkz. [Checked ve unchecked](../language-reference/keywords/checked-and-unchecked.md).</span><span class="sxs-lookup"><span data-stu-id="fd87d-106">For more information, see [Checked and Unchecked](../language-reference/keywords/checked-and-unchecked.md).</span></span>  
  
 <span data-ttu-id="fd87d-107">Aşağıdaki örnek CS0220 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="fd87d-107">The following sample generates CS0220:</span></span>  
  
```csharp  
// CS0220.cs  
using System;  
  
class TestClass  
{  
   const int x = 1000000;  
   const int y = 1000000;  
  
   public int MethodCh()  
   {  
      int z = (x * y);   // CS0220  
      return z;  
   }  
  
   public int MethodUnCh()  
   {  
      unchecked  
      {  
         int z = (x * y);  
         return z;  
      }  
   }  
  
   public static void Main()  
   {  
      TestClass myObject = new TestClass();  
      Console.WriteLine("Checked  : {0}", myObject.MethodCh());  
      Console.WriteLine("Unchecked: {0}", myObject.MethodUnCh());  
   }  
}  
```
