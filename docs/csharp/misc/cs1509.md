# <a name="compiler-error-cs1509"></a>Derleyici Hatası CS1509
Başvurulan dosya 'dosyası bir derleme değil; Kullan ' / addmodule' bunun yerine seçeneği  
  
 Bir çıktı dosyası (çıktı dosyası 1) kullanılan bir derlemede üretilen [/target: Module](../../csharp/language-reference/compiler-options/target-module-compiler-option.md) (bir derleme bildirimi yok), için belirtilen [/reference](../../csharp/language-reference/compiler-options/reference-compiler-option.md). Bu nedenle, geçerli program derlemeye bir derleme ekleme yerine meta veri bilgilerini çıktı dosyasına 1 için geçerli program derlemeye eklenir.
