# <a name="compiler-error-cs1509"></a>Derleyici hatası CS1509
Başvurulan ' File ' dosyası bir derleme değil; Bunun yerine '/addmodule ' seçeneğini kullanın  
  
 [/Target: Module](../language-reference/compiler-options/target-module-compiler-option.md) (derleme bildirimine sahip değil) kullanan bir derlemede üretilen bir çıkış dosyası (çıkış dosyası 1), [/Reference](../language-reference/compiler-options/reference-compiler-option.md)olarak belirtilmiş. Bu nedenle, geçerli program için derlemeye bir derleme eklemek yerine, çıkış dosyası 1 ' deki meta veri bilgileri geçerli program için derlemeye eklenecektir.
