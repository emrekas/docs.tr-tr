# <a name="compiler-error-cs0214"></a>Derleyici Hatası CS0214
İşaretçiler ve sabit boyutlu arabellekler yalnızca güvenli olmayan bir bağlamda kullanılabilir  
  
 İşaretçiler yalnızca kullanılabilir ile [güvenli](../../csharp/language-reference/keywords/unsafe.md) anahtar sözcüğü. Daha fazla bilgi için [güvenli olmayan kod ve işaretçiler](../../csharp/programming-guide/unsafe-code-pointers/index.md).  
  
 Aşağıdaki örnek, CS0214 oluşturur:  
  
```csharp  
// CS0214.cs  
// compile with: /target:library /unsafe  
public struct S  
{  
   public int a;  
}  
  
public class MyClass  
{  
   public static void Test()  
   {  
      S s = new S();  
      S * s2 = &s;    // CS0214  
      s2->a = 3;      // CS0214  
      s.a = 0;  
   }  
  
   // OK  
   unsafe public static void Test2()  
   {  
      S s = new S();  
      S * s2 = &s;  
      s2->a = 3;  
      s.a = 0;  
   }  
}  
```
