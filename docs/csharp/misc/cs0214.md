# <a name="compiler-error-cs0214"></a><span data-ttu-id="fd24b-102">Derleyici Hatası CS0214</span><span class="sxs-lookup"><span data-stu-id="fd24b-102">Compiler Error CS0214</span></span>
<span data-ttu-id="fd24b-103">İşaretçiler ve sabit boyutlu arabellekler yalnızca güvenli olmayan bir bağlamda kullanılabilir</span><span class="sxs-lookup"><span data-stu-id="fd24b-103">Pointers and fixed size buffers may only be used in an unsafe context</span></span>  
  
 <span data-ttu-id="fd24b-104">İşaretçiler yalnızca kullanılabilir ile [güvenli](../../csharp/language-reference/keywords/unsafe.md) anahtar sözcüğü.</span><span class="sxs-lookup"><span data-stu-id="fd24b-104">Pointers can only be used with the [unsafe](../../csharp/language-reference/keywords/unsafe.md) keyword.</span></span> <span data-ttu-id="fd24b-105">Daha fazla bilgi için [güvenli olmayan kod ve işaretçiler](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="fd24b-105">For more information, see [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="fd24b-106">Aşağıdaki örnek, CS0214 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="fd24b-106">The following sample generates CS0214:</span></span>  
  
```csharp  
// CS0214.cs  
// compile with: /target:library /unsafe  
public struct S  
{  
   public int a;  
}  
  
public class MyClass  
{  
   public static void Test()  
   {  
      S s = new S();  
      S * s2 = &s;    // CS0214  
      s2->a = 3;      // CS0214  
      s.a = 0;  
   }  
  
   // OK  
   unsafe public static void Test2()  
   {  
      S s = new S();  
      S * s2 = &s;  
      s2->a = 3;  
      s.a = 0;  
   }  
}  
```
