# <a name="compiler-error-cs0068"></a>Derleyici Hatası CS0068
'event': arabirimdeki olayın başlatıcısı olamaz  
  
 Bir arabirimdeki bir olayın bir başlatıcıya sahip olamaz. Daha fazla bilgi için [arabirimleri](../../csharp/programming-guide/interfaces/index.md).  
  
 Aşağıdaki örnek, CS0068 oluşturur:  
  
```csharp  
// CS0068.cs  
  
delegate void MyDelegate();  
  
interface I  
{  
   event MyDelegate d = new MyDelegate(M.f);   // CS0068  
   // try the following line instead  
   // event MyDelegate d2;  
}  
  
class M  
{  
   event MyDelegate d = new MyDelegate(M.f);  
  
   public static void f()  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
