# <a name="compiler-error-cs0068"></a><span data-ttu-id="a251c-102">Derleyici Hatası CS0068</span><span class="sxs-lookup"><span data-stu-id="a251c-102">Compiler Error CS0068</span></span>
<span data-ttu-id="a251c-103">'event': arabirimdeki olayın başlatıcısı olamaz</span><span class="sxs-lookup"><span data-stu-id="a251c-103">'event': event in interface cannot have initializer</span></span>  
  
 <span data-ttu-id="a251c-104">Bir arabirimdeki bir olayın bir başlatıcıya sahip olamaz.</span><span class="sxs-lookup"><span data-stu-id="a251c-104">An event in an interface cannot have an initializer.</span></span> <span data-ttu-id="a251c-105">Daha fazla bilgi için [arabirimleri](../../csharp/programming-guide/interfaces/index.md).</span><span class="sxs-lookup"><span data-stu-id="a251c-105">For more information, see [Interfaces](../../csharp/programming-guide/interfaces/index.md).</span></span>  
  
 <span data-ttu-id="a251c-106">Aşağıdaki örnek, CS0068 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="a251c-106">The following sample generates CS0068:</span></span>  
  
```csharp  
// CS0068.cs  
  
delegate void MyDelegate();  
  
interface I  
{  
   event MyDelegate d = new MyDelegate(M.f);   // CS0068  
   // try the following line instead  
   // event MyDelegate d2;  
}  
  
class M  
{  
   event MyDelegate d = new MyDelegate(M.f);  
  
   public static void f()  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
