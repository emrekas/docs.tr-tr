 <span data-ttu-id="0217c-106">Aşağıdaki örnek CS0068 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="0217c-106">The following sample generates CS0068:</span></span>  
  
```csharp

{  
   event MyDelegate d = new MyDelegate(M.f);   // CS0068

{  
   event MyDelegate d = new MyDelegate(M.f);

   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
