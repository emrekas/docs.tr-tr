# <a name="compiler-error-cs0176"></a>Derleyici Hatası CS0176
Örnek başvurusuyla 'member' statik üyesi erişilemez; Bunun yerine bir tür adıyla niteleyin  
  
 Yalnızca bir sınıf adı nitelemek için kullanılan bir [statik](../../csharp/language-reference/keywords/static.md) değişken; örnek adı, bir niteleyici olamaz. Daha fazla bilgi için [statik sınıflar ve statik sınıf üyeleri](../../csharp/programming-guide/classes-and-structs/static-classes-and-static-class-members.md).  
  
 Aşağıdaki örnek, CS0176 oluşturur:  
  
```csharp  
// CS0176.cs  
public class MyClass2  
{  
    public static int num;  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        MyClass2 mc2 = new MyClass2();  
        int i = mc2.num;   // CS0176  
        // try the following line instead  
        // int i = MyClass2.num;  
    }  
}  
```
