 <span data-ttu-id="ad9d4-104">Bir taban sınıfın kullanıcı tanımlı dönüşümler değerlere izin verilmiyor; Böyle bir işleç gerekmez.</span><span class="sxs-lookup"><span data-stu-id="ad9d4-104">User-defined conversions to values of a base class are not allowed; you do not need such an operator.</span></span>

```csharp  
// CS0553.cs  
namespace x  
{  
   public class ii  
   {  
   }  
  
   public class a : ii  
   {  
      // delete the conversion routine to resolve CS0553

         return new ii();  
      }

      {  
      }  
   }  
}  
```
