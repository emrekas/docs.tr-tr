# <a name="compiler-error-cs0010"></a>Derleyici Hatası CS0010
Beklenmeyen önemli hata--'error'.  
  
 Bu hata, derlemeyi durdurmak için tamamen beklenmeyen bir sorun ortaya çıktığında oluşturulur.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Başka bir projeyi yeniden derleyin. Aynı hata alırsanız, Visual Studio'yu yeniden yüklemeyi deneyin. Bu hatayı alırsanız, hata raporunu Microsoft'a gönderin.
