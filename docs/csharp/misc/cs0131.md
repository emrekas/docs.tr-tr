# <a name="compiler-error-cs0131"></a>Derleyici Hatası CS0131
Atamanın sol tarafında bir değişken, özellik veya dizin oluşturucusu olmalıdır  
  
 Atama deyiminin sol tarafı için sağ tarafı değeri atanır. Sol tarafı bir değişken, özellik veya dizin oluşturucu olması gerekir.  
  
 Bu hatayı düzeltmek için tüm işleçleri sağ tarafta olduğunu ve sol tarafı bir değişken, özellik veya dizin oluşturucu olduğundan emin olun. Daha fazla bilgi için [deyimler, ifadeler ve işleçler](../../csharp/programming-guide/statements-expressions-operators/index.md).  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS0131 oluşturur.  
  
```csharp  
// CS0131.cs  
public class MyClass  
{  
    public int i = 0;  
    public void MyMethod()  
    {  
        i++ = 1;   // CS0131  
        // try the following line instead  
        // i = 1;  
    }  
    public static void Main() { }  
}  
```  
  
## <a name="example"></a>Örnek  
 Bu hata, aşağıdaki örnekte olduğu gibi bir atama işlecinin sol tarafında aritmetik işlemleri yapmayı denerseniz de oluşabilir.  
  
```csharp  
// CS0131b.cs  
public class C  
{  
    public static int Main()  
    {  
        int a = 1, b = 2, c = 3;  
        if (a + b = c) // CS0131  
        // try this instead  
        // if (a + b == c)  
            return 0;  
        return 1;  
    }  
}  
```
