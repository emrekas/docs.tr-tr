 <span data-ttu-id="4af5f-104">Atama deyiminin sol tarafı için sağ tarafı değeri atanır.</span><span class="sxs-lookup"><span data-stu-id="4af5f-104">In an assignment statement, the value of the right-hand side is assigned to the left-hand side.</span></span> <span data-ttu-id="4af5f-105">Sol tarafı bir değişken, özellik veya dizin oluşturucu olması gerekir.</span><span class="sxs-lookup"><span data-stu-id="4af5f-105">The left-hand side must be a variable, property, or indexer.</span></span>

## <a name="example"></a><span data-ttu-id="4af5f-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="4af5f-108">Example</span></span>  
 <span data-ttu-id="4af5f-109">Aşağıdaki örnek, CS0131 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="4af5f-109">The following sample generates CS0131.</span></span>  
  
```csharp  
// CS0131.cs  
public class MyClass  
{  
    public int i = 0;  
    public void MyMethod()  
    {  
        i++ = 1;   // CS0131  
        // try the following line instead  
        // i = 1;  
    }  
    public static void Main() { }  
}  
```

    public static int Main()
