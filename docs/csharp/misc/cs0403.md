# <a name="compiler-error-cs0403"></a><span data-ttu-id="fc8a9-102">Derleyici Hatası CS0403</span><span class="sxs-lookup"><span data-stu-id="fc8a9-102">Compiler Error CS0403</span></span>
<span data-ttu-id="fc8a9-103">Null, NULL olmayan bir değer türü olabileceğinden parametresinin 'name' türüne dönüştürülemiyor.</span><span class="sxs-lookup"><span data-stu-id="fc8a9-103">Cannot convert null to type parameter 'name' because it could be a non-nullable value type.</span></span> <span data-ttu-id="fc8a9-104">Default('T') kullanmayı düşünün.</span><span class="sxs-lookup"><span data-stu-id="fc8a9-104">Consider using default('T') instead.</span></span>  
  
 <span data-ttu-id="fc8a9-105">Null atama izin vermeyen bir değer türü olabileceğinden adlı bilinmeyen tür için null atanamaz.</span><span class="sxs-lookup"><span data-stu-id="fc8a9-105">You cannot assign null to the unknown type named because it might be a value type, which does not allow null assignment.</span></span> <span data-ttu-id="fc8a9-106">Değer türleri kabul etmek için genel sınıf planlanmıyorsa, sınıf türü kısıtlaması kullanın.</span><span class="sxs-lookup"><span data-stu-id="fc8a9-106">If your generic class is not intended to accept value types, use the class type constraint.</span></span> <span data-ttu-id="fc8a9-107">Yerleşik türler gibi değer türleri kabul edebiliyorsa null atamaya ifadeyle değiştirin mümkün olabilir `default(T)`, aşağıdaki örnekte gösterildiği gibi.</span><span class="sxs-lookup"><span data-stu-id="fc8a9-107">If it can accept value types, such as the built-in types, you may be able to replace the assignment to null with the expression `default(T)`, as shown in the following example.</span></span>  
  
## <a name="example"></a><span data-ttu-id="fc8a9-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="fc8a9-108">Example</span></span>  
 <span data-ttu-id="fc8a9-109">Aşağıdaki örnek, CS0403 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="fc8a9-109">The following sample generates CS0403.</span></span>  
  
```csharp  
// CS0403.cs  
// compile with: /target:library  
class C<T>  
{  
   public void f()  
   {  
      T t = null;  // CS0403  
      T t2 = default(T);   // OK  
    }  
}  
  
class D<T> where T : class   
{  
   public void f()  
   {  
      T t = null;  // OK  
    }  
}  
```
