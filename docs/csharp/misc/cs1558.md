# <a name="compiler-error-cs1558"></a><span data-ttu-id="8fbfa-102">Derleyici Hatası CS1558</span><span class="sxs-lookup"><span data-stu-id="8fbfa-102">Compiler Error CS1558</span></span>
<span data-ttu-id="8fbfa-103">'class' uygun bir statik Main yöntemine sahip değil</span><span class="sxs-lookup"><span data-stu-id="8fbfa-103">'class' does not have a suitable static Main method</span></span>  
  
 <span data-ttu-id="8fbfa-104">[/Main](../../csharp/language-reference/compiler-options/main-compiler-option.md) derleyici seçeneği belirtildi, aranacak bir sınıfta bir **ana** yöntemi.</span><span class="sxs-lookup"><span data-stu-id="8fbfa-104">The [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) compiler option specified a class in which to look for a **Main** method.</span></span> <span data-ttu-id="8fbfa-105">Ancak, [ana](../../csharp/programming-guide/main-and-command-args/index.md) yöntemi bölümlü tanımlanmamış doğru.</span><span class="sxs-lookup"><span data-stu-id="8fbfa-105">However, the [Main](../../csharp/programming-guide/main-and-command-args/index.md) method was not defined correctly.</span></span>  
  
 <span data-ttu-id="8fbfa-106">Aşağıdaki örnek, geçersiz dönüş türü nedeniyle CS1558 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="8fbfa-106">The following example generates CS1558 because of invalid return type.</span></span>  
  
```csharp  
// CS1558.cs  
// compile with: /main:MyNamespace.MyClass  
  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static float Main()   
      {  
         return 0.0; // CS1558 because the return type is a float.  
      }  
   }  
}  
```
