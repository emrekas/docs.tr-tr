# <a name="compiler-error-cs1647"></a>Derleyici Hatası CS1647
Çok uzun veya karmaşık 'code' derlemek için ifade  
  
 Kodunuzu işleme derleyicide yığın taşması oluştu. Bu hatayı gidermek için kodunuzu basitleştirin. Kodunuzu geçerliyse, Yardım [Geliştirici topluluğu](https://developercommunity.visualstudio.com/spaces/61/index.html) site veya [StackOverflow](https://stackoverflow.com/).
