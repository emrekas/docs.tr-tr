# <a name="compiler-error-cs1537"></a>Derleyici Hatası CS1537
Diğer adı 'alias' kullanarak bu ad alanında daha önce göründü  
  
 Bir sembol için bir ad alanı diğer ad olarak iki kez tanımlanmış. Bir simge yalnızca bir kez tanımlanabilir.  
  
 Aşağıdaki örnek, CS1537 oluşturur:  
  
```csharp  
// CS1537.cs  
namespace x  
{  
   using System;  
   using Object = System.Object;  
   using Object = System.Object;   // CS1537, delete this line to resolve  
   using System = System;  
}  
```
