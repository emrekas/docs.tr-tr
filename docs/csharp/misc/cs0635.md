# <a name="compiler-error-cs0635"></a>Compiler Error CS0635
'attribute': Adlandırılmış bağımsız değişkenler ComType ve sıralama System.Interop.UnmanagedType.CustomMarshaller gerektirir  
  
 **ComType** ve **sıralama** sıralama biçim olduğunda, bağımsız değişkenleri belirtilmelidir **System.Interop.UnmanagedType.CustomMarshaller**.
