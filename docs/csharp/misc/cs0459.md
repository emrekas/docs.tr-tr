# <a name="compiler-error-cs0459"></a><span data-ttu-id="d6fad-102">Derleyici Hatası CS0459</span><span class="sxs-lookup"><span data-stu-id="d6fad-102">Compiler Error CS0459</span></span>
<span data-ttu-id="d6fad-103">Salt okunur yerel değişkenin adresi alınamaz</span><span class="sxs-lookup"><span data-stu-id="d6fad-103">Cannot take the address of a read-only local variable</span></span>  
  
 <span data-ttu-id="d6fad-104">İçinde üç yaygın senaryo vardır C# salt okunur yerel değişkenleri oluşturun dil: `foreach`, `using`, ve `fixed`.</span><span class="sxs-lookup"><span data-stu-id="d6fad-104">There are three common scenarios in the C# language that generate read-only local variables: `foreach`, `using`, and `fixed`.</span></span> <span data-ttu-id="d6fad-105">Her durumda, salt okunur yerel değişkenin yazma ya da adresini almak için izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="d6fad-105">In each of these cases, you are not allowed to write to the read-only local variable, or to take its address.</span></span> <span data-ttu-id="d6fad-106">Derleyici, salt okunur yerel değişkenin adresini almak denediğiniz fark etti. Bu hata oluşturulur.</span><span class="sxs-lookup"><span data-stu-id="d6fad-106">This error is generated when the compiler realizes you are trying to take the address of a read-only local variable.</span></span>  
  
## <a name="example"></a><span data-ttu-id="d6fad-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="d6fad-107">Example</span></span>  
 <span data-ttu-id="d6fad-108">Aşağıdaki örnek bir salt okunur yerel değişkende adresini almak için bir girişimde CS0459 oluşturur bir `foreach` döngü ve bir `fixed` deyim bloğu.</span><span class="sxs-lookup"><span data-stu-id="d6fad-108">The following example generates CS0459 when an attempt is made to take the address of a read-only local variable in a `foreach` loop and in a `fixed` statement block.</span></span>  
  
```csharp  
// CS0459.cs  
// compile with: /unsafe  
  
class A  
{  
    public unsafe void M1()  
    {  
        int[] ints = new int[] { 1, 2, 3 };  
        foreach (int i in ints)  
        {  
            int *j = &i;  // CS0459  
        }  
  
        fixed (int *i = &_i)  
        {  
            int **j = &i;  // CS0459  
        }  
    }  
  
    private int _i = 0;  
}  
```
