 <span data-ttu-id="da773-104">Bu hataya neden olabilir bir ad alanı içinde bir başvuruya göre söylüyor bir türe başvuran ikinci bir ad alanı içinde var, ancak türü yok.</span><span class="sxs-lookup"><span data-stu-id="da773-104">This error can be caused by a reference inside one namespace referring to a type that it says exists inside a second namespace, but the type does not exist.</span></span> <span data-ttu-id="da773-105">Örneğin, bu tür mydll.dll diyor `A` yourdll.dll içinde var ancak böyle bir türü yourdll.dll içinde yok.</span><span class="sxs-lookup"><span data-stu-id="da773-105">For example, mydll.dll says that type `A` exists inside yourdll.dll, but no such type exists inside yourdll.dll.</span></span> <span data-ttu-id="da773-106">Bu hatanın olası bir nedeni olduğundan kullanmakta olduğunuz yourdll.dll sürümü çok eski olduğunu ve `A` henüz tanımlanmadı.</span><span class="sxs-lookup"><span data-stu-id="da773-106">One possible cause of this error is that the version of yourdll.dll you are using is too old and `A` has not yet been defined.</span></span>  
  
 <span data-ttu-id="da773-107">Aşağıdaki örnek, CS1684 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="da773-107">The following sample generates CS1684.</span></span>  
  
## <a name="example"></a><span data-ttu-id="da773-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="da773-108">Example</span></span>  
  
```csharp

// compile with: /target:library /keyfile:CS1684.key  
public class A {  
   public void Test() {}  
}  
  
public class C2 {}  
```  
  
## <a name="example"></a><span data-ttu-id="da773-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="da773-109">Example</span></span>

```csharp  
   public static A GetA() { return new A(); }

}  
```
