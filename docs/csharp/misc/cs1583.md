# <a name="compiler-error-cs1583"></a>Derleyici hatası CS1583
' File ' geçerli bir Win32 kaynak dosyası değil  
  
 Bu hata, [/win32res](../language-reference/compiler-options/win32res-compiler-option.md) derleyici seçeneğiyle geçerli veya doğru şekilde biçimlendirilen bir Win32 kaynak dosyası olmayan bir dosya adı belirttiğinizde oluşur. Visual Studio 'da dosya adı, proje Tasarımcısı 'nın uygulama bölmesinde belirtilir.
