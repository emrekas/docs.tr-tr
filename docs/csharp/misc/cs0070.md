# <a name="compiler-error-cs0070"></a><span data-ttu-id="84a31-102">Derleyici hatası CS0070</span><span class="sxs-lookup"><span data-stu-id="84a31-102">Compiler Error CS0070</span></span>
<span data-ttu-id="84a31-103">' Event ' olayı + = veya-= öğesinin yalnızca sol tarafında görünebilir (' Type ' türü içinden kullanılması dışında)</span><span class="sxs-lookup"><span data-stu-id="84a31-103">The event 'event' can only appear on the left hand side of += or -= (except when used from within the type 'type')</span></span>  
  
 <span data-ttu-id="84a31-104">İçinde tanımlandığı sınıfın dışında, bir [olay](../language-reference/keywords/event.md) yalnızca başvuru ekleyebilir veya çıkartabilir.</span><span class="sxs-lookup"><span data-stu-id="84a31-104">Outside of the class it is defined in, an [event](../language-reference/keywords/event.md) can only add or subtract references.</span></span> <span data-ttu-id="84a31-105">Daha fazla bilgi için bkz. [Olaylar](../programming-guide/events/index.md).</span><span class="sxs-lookup"><span data-stu-id="84a31-105">For more information, see [Events](../programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="84a31-106">Aşağıdaki örnek CS0070 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="84a31-106">The following sample generates CS0070:</span></span>  
  
```csharp  
// CS0070.cs  
using System;  
public delegate void EventHandler();  
  
public class A  
{  
   public event EventHandler Click;  
  
   public static void OnClick()  
   {  
      EventHandler eh;  
      A a = new A();  
      eh = a.Click;  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public class B  
{  
   public int Foo ()  
   {  
      EventHandler eh = new EventHandler(A.OnClick);  
      A a = new A();  
      eh = a.Click;   // CS0070  
      // try the following line instead  
      // a.Click += eh;  
      return 1;  
   }  
}  
```
