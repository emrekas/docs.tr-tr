# <a name="compiler-error-cs0070"></a><span data-ttu-id="5817a-102">Derleyici Hatası CS0070</span><span class="sxs-lookup"><span data-stu-id="5817a-102">Compiler Error CS0070</span></span>
<span data-ttu-id="5817a-103">'Event' olayı += veya-= (öğesinden 'type' türü içinden kullanılması dışında) sol tarafındaki yalnızca görünebilir</span><span class="sxs-lookup"><span data-stu-id="5817a-103">The event 'event' can only appear on the left hand side of += or -= (except when used from within the type 'type')</span></span>  
  
 <span data-ttu-id="5817a-104">Tanımlanan, sınıf dışında bir [olay](../../csharp/language-reference/keywords/event.md) yalnızca ekleyebilir veya başvuruları kaldırabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="5817a-104">Outside of the class it is defined in, an [event](../../csharp/language-reference/keywords/event.md) can only add or subtract references.</span></span> <span data-ttu-id="5817a-105">Daha fazla bilgi için [olayları](../../csharp/programming-guide/events/index.md).</span><span class="sxs-lookup"><span data-stu-id="5817a-105">For more information, see [Events](../../csharp/programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="5817a-106">Aşağıdaki örnek, CS0070 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="5817a-106">The following sample generates CS0070:</span></span>  
  
```csharp  
// CS0070.cs  
using System;  
public delegate void EventHandler();  
  
public class A  
{  
   public event EventHandler Click;  
  
   public static void OnClick()  
   {  
      EventHandler eh;  
      A a = new A();  
      eh = a.Click;  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public class B  
{  
   public int Foo ()  
   {  
      EventHandler eh = new EventHandler(A.OnClick);  
      A a = new A();  
      eh = a.Click;   // CS0070  
      // try the following line instead  
      // a.Click += eh;  
      return 1;  
   }  
}  
```
