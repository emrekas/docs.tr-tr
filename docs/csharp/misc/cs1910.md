 <span data-ttu-id="30b96-104">Nesne türü olan parametreler için bağımsız değişkeni <xref:System.Runtime.InteropServices.DefaultParameterValueAttribute> olmalıdır `null`, integral türünde, bir kayan nokta `bool`, `string`, `enum`, veya `char`.</span><span class="sxs-lookup"><span data-stu-id="30b96-104">For parameters whose type is object, the argument of the <xref:System.Runtime.InteropServices.DefaultParameterValueAttribute> must be `null`, an integral type, a floating point, `bool`, `string`, `enum`, or `char`.</span></span> <span data-ttu-id="30b96-105">Bağımsız değişken türü olamaz <xref:System.Type> veya herhangi bir dizi türü.</span><span class="sxs-lookup"><span data-stu-id="30b96-105">The argument can not be of type <xref:System.Type> or any array type.</span></span>
