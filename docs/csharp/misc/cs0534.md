# <a name="compiler-error-cs0534"></a><span data-ttu-id="83765-102">Derleyici hatası CS0534</span><span class="sxs-lookup"><span data-stu-id="83765-102">Compiler Error CS0534</span></span>
<span data-ttu-id="83765-103">' işlev1 ' devralınan ' function2 ' soyut üyesini uygulamıyor</span><span class="sxs-lookup"><span data-stu-id="83765-103">'function1' does not implement inherited abstract member 'function2'</span></span>  
  
 <span data-ttu-id="83765-104">Sınıf da soyut olmadığı takdirde, temel sınıftaki tüm [soyut](../language-reference/keywords/abstract.md) üyeleri uygulamak için bir sınıf gereklidir.</span><span class="sxs-lookup"><span data-stu-id="83765-104">A class is required to implement all the [abstract](../language-reference/keywords/abstract.md) members in the base class, unless the class is also abstract.</span></span>  
  
 <span data-ttu-id="83765-105">Aşağıdaki örnek CS0534 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="83765-105">The following sample generates CS0534:</span></span>  
  
```csharp  
// CS0534.cs  
namespace x  
{  
   abstract public class clx  
   {  
      public abstract void f();  
   }  
  
   public class cly : clx   // CS0534, no override for clx::f  
   {  
      // uncomment the following sample override to resolve CS0534  
      // override public void f()  
      // {  
      // }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
