 <span data-ttu-id="cf667-104">A [salt okunur](../../csharp/language-reference/keywords/readonly.md) değişken aynı olmalıdır [statik](../../csharp/language-reference/keywords/static.md) başlatmak istediğiniz Oluşturucu olarak kullanımı.</span><span class="sxs-lookup"><span data-stu-id="cf667-104">A [readonly](../../csharp/language-reference/keywords/readonly.md) variable must have the same [static](../../csharp/language-reference/keywords/static.md) usage as the constructor in which you want to initialize it.</span></span> <span data-ttu-id="cf667-105">Daha fazla bilgi için [statik oluşturucular](../../csharp/programming-guide/classes-and-structs/static-constructors.md).</span><span class="sxs-lookup"><span data-stu-id="cf667-105">For more information, see [Static Constructors](../../csharp/programming-guide/classes-and-structs/static-constructors.md).</span></span>

   }
