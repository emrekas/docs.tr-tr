# <a name="compiler-error-cs1530"></a>Derleyici Hatası CS1530
Ad alanında tanımlanan öğelerde 'new' anahtar sözcüğü kullanılamaz  
  
 Belirtmek gerekli değildir [yeni](../../csharp/language-reference/keywords/new-modifier.md) olan herhangi bir yapısı anahtar sözcüğü bir [ad alanı](../../csharp/language-reference/keywords/namespace.md).  
  
 Aşağıdaki örnek, CS1530 oluşturur:  
  
```csharp  
// CS1530.cs  
namespace a  
{  
   new class i   // CS1530  
   {  
   }  
  
   // try the following instead  
   class ii  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
