# <a name="compiler-error-cs0112"></a><span data-ttu-id="9d750-102">Derleyici Hatası CS0112</span><span class="sxs-lookup"><span data-stu-id="9d750-102">Compiler Error CS0112</span></span>
<span data-ttu-id="9d750-103">'Function' statik üyesi override, virtual veya abstract işaretlenemez</span><span class="sxs-lookup"><span data-stu-id="9d750-103">A static member 'function' cannot be marked as override, virtual or abstract</span></span>  
  
 <span data-ttu-id="9d750-104">Kullanan herhangi bir yöntem bildiriminde `override`, **sanal**, veya **soyut** anahtar sözcüğü de kullanamaz **statik** anahtar sözcüğü.</span><span class="sxs-lookup"><span data-stu-id="9d750-104">Any method declaration that uses the `override`, **virtual**, or **abstract** keyword cannot also use the **static** keyword.</span></span>  
  
 <span data-ttu-id="9d750-105">Daha fazla bilgi için [yöntemleri](../../csharp/programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="9d750-105">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="9d750-106">Aşağıdaki örnek, CS0112 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="9d750-106">The following sample generates CS0112:</span></span>  
  
```csharp  
// CS0112.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
   public class MyClass2 : MyClass  
   {  
      override public static void Foo()   // CS0112, remove static keyword  
      {  
      }  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
