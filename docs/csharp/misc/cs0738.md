## <a name="example"></a><span data-ttu-id="ad1bb-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="ad1bb-108">Example</span></span>  
 <span data-ttu-id="ad1bb-109">Aşağıdaki kod, CS0738 oluşturur, çünkü sınıf yöntemi döndürür `void` ve aynı ada sahip bir arabirim üyesini döndürür `int`:</span><span class="sxs-lookup"><span data-stu-id="ad1bb-109">The following code generates CS0738 because the class method returns `void` and the interface member of the same name returns `int`:</span></span>

```csharp  
{  
    int TestMethod();  
}

```
