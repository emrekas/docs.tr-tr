## <a name="example"></a><span data-ttu-id="ecd44-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="ecd44-106">Example</span></span>  
 <span data-ttu-id="ecd44-107">Aşağıdaki örnek, CS1642 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="ecd44-107">The following sample generates CS1642.</span></span>  
  
```csharp  
// CS1642.cs  
// compile with: /unsafe /target:library  
unsafe class C  
{  
   fixed int a[10];   // CS1642  
}  
  
unsafe struct D  
{  
    fixed int a[10];  
}  
  
unsafe class E  
{  
   public int[] a = null;  
}  
```
