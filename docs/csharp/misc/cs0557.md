# <a name="compiler-error-cs0557"></a>Derleyici Hatası CS0557
Yinelenen tür 'class' kullanıcı tanımlı dönüştürme  
  
 Yinelenen dönüştürme rutinleri bir sınıf içinde izin verilmez.  
  
 Aşağıdaki örnek, CS0557 oluşturur:  
  
```csharp  
// CS0557.cs  
namespace x  
{  
    public class ii  
    {  
        public class iii  
        {  
        public static implicit operator int(iii aa)  
        {  
            return 0;  
        }  
  
    // CS0557, delete duplicate  
        public static explicit operator int(iii aa)  
        {  
            return 0;  
        }  
        }  
  
        public static void Main()  
        {  
        }  
    }  
}  
```
