# <a name="compiler-error-cs0042"></a>Derleyici Hatası CS0042
Hata ayıklama bilgileri 'dosyasından' oluşturulurken beklenmeyen hata: 'açıklaması'  
  
 Derleyici hata ayıklama bilgileri oluşturulamadı; *neden* hata durumu hakkında ek bilgiler içerir.
