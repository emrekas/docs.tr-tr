# <a name="compiler-error-cs0716"></a>Derleyici Hatası CS0716
'Type' statik türüne dönüştürülemiyor  
  
 Kodunuzu statik bir türe dönüştürmek için bir yayın kullanıyorsa, bu hata oluşur. Bir nesne için bir statik türünün bir örneği olması mümkün olmadığından, bir statik türü atamayı hiçbir zaman anlamlı bir tür dönüştürme olabilir.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS0716 oluşturur:  
  
```csharp  
// CS0716.cs  
  
public static class SC  
{  
    static void F() { }  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        object o = new object();  
        System.Console.WriteLine((SC)o);  // CS0716  
    }  
}  
```
