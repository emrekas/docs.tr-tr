# <a name="compiler-error-cs1575"></a>Derleyici Hatası CS1575
Bir stackalloc ifadesi türden sonra [] gerektirir  
  
 İstenen ayırma boyutu ile [stackalloc](../../csharp/language-reference/keywords/stackalloc.md), köşeli ayraç içinde belirtilmelidir.  
  
 Aşağıdaki örnek, CS1575 oluşturur:  
  
```csharp  
// CS1575.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int *p = stackalloc int (30);   // CS1575  
      // try the following line instead  
      // int *p = stackalloc int [30];  
   }  
}  
```
