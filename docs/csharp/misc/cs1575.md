# <a name="compiler-error-cs1575"></a><span data-ttu-id="7ac13-102">Derleyici Hatası CS1575</span><span class="sxs-lookup"><span data-stu-id="7ac13-102">Compiler Error CS1575</span></span>
<span data-ttu-id="7ac13-103">Bir stackalloc ifadesi türden sonra [] gerektirir</span><span class="sxs-lookup"><span data-stu-id="7ac13-103">A stackalloc expression requires [] after type</span></span>  
  
 <span data-ttu-id="7ac13-104">İstenen ayırma boyutu ile [stackalloc](../../csharp/language-reference/keywords/stackalloc.md), köşeli ayraç içinde belirtilmelidir.</span><span class="sxs-lookup"><span data-stu-id="7ac13-104">The size of the requested allocation, with [stackalloc](../../csharp/language-reference/keywords/stackalloc.md), must be specified in square brackets.</span></span>  
  
 <span data-ttu-id="7ac13-105">Aşağıdaki örnek, CS1575 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="7ac13-105">The following sample generates CS1575:</span></span>  
  
```csharp  
// CS1575.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int *p = stackalloc int (30);   // CS1575  
      // try the following line instead  
      // int *p = stackalloc int [30];  
   }  
}  
```
