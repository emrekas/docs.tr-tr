# <a name="compiler-error-cs1665"></a>Derleyici Hatası CS1665
Sabit boyutlu arabellekler sıfırdan büyük bir uzunlukta olmalıdır  
  
 Sabit boyutlu arabellek sıfır ile bildirilirse, bu hata oluşur veya negatif boyutu. Sabit boyutlu arabellek uzunluğu, pozitif bir tamsayı olmalıdır.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS1665 oluşturur.  
  
```csharp  
// CS1665.cs  
// compile with: /unsafe /target:library  
struct S  
{   
   public unsafe fixed int A[0];   // CS1665  
}  
```
