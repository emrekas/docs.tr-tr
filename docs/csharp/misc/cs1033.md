# <a name="compiler-error-cs1033"></a>Derleyici Hatası CS1033
Kaynak dosyası PDB'de PDB'de görüntülenebilecek 16.707.565 satır sınırını aştı; hata ayıklama bilgileri hatalı olacak  
  
 Bir kaynak kodu dosyası derleyici işleyebilir satırların verilen sınırını aştı. Bu hatayı çözmek için iki veya daha fazla kaynak kodu dosyaları özgün dosyayı oluşturun. Maksimum satır sayısı 268,435,454 satırdır. Kullanıyorsanız **/debug**, birden fazla 16,707,566 kullanarak bozuk hata ayıklama bilgilerini neden olur.
