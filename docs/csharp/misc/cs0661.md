# <a name="compiler-warning-level-3-cs0661"></a><span data-ttu-id="74844-102">Derleyici Uyarısı (Düzey 3) CS0661</span><span class="sxs-lookup"><span data-stu-id="74844-102">Compiler Warning (level 3) CS0661</span></span>
<span data-ttu-id="74844-103">'class' tanımlar işleç == veya işleç! = ancak Object.GetHashCode() öğesini geçersiz kılmaz</span><span class="sxs-lookup"><span data-stu-id="74844-103">'class' defines operator == or operator != but does not override Object.GetHashCode()</span></span>  
  
 <span data-ttu-id="74844-104">Derleyici kullanıcı tanımlı eşitlik ya da eşitsizlik işleci, ancak geçersiz kılma için algılanan **GetHashCode** işlevi.</span><span class="sxs-lookup"><span data-stu-id="74844-104">The compiler detected the user-defined equality or inequality operator, but no override for the **GetHashCode** function.</span></span> <span data-ttu-id="74844-105">Kullanıcı tanımlı bir eşitlik ve eşitsizlik işleci ayrıca geçersiz kılmak istediğiniz gelir **GetHashCode** işlevi.</span><span class="sxs-lookup"><span data-stu-id="74844-105">A user-defined equality or inequality operator implies that you also want to override the **GetHashCode** function.</span></span>  
  
 <span data-ttu-id="74844-106">Aşağıdaki örnek, CS0661 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="74844-106">The following sample generates CS0661:</span></span>  
  
```csharp  
// CS0661.cs  
// compile with: /W:3  
class Test   // CS0661  
{  
   public static bool operator == (object o, Test t)  
   {  
      return true;  
   }  
  
   public static bool operator != (object o, Test t)  
   {  
      return true;  
   }  
  
   public override bool Equals(object o)  
   {  
      return true;  
   }  
  
   // uncomment the GetHashCode function to resolve  
   // public override int GetHashCode()  
   // {  
   //    return 0;  
   // }  
  
   public static void Main()  
   {  
   }  
}  
```
