# <a name="compiler-error-cs1606"></a>Derleyici Hatası CS1606
Derleme imzalama başarısız oldu; Çıkış olmayabilir imzalı--nedeni  
  
 Üretilen derleme, ancak derleyicinin imzalamadan son kurmaya çalışırken bir hata oluştu.
