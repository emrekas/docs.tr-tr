 <span data-ttu-id="6a769-104">Bir yöntem gövdesi olmadığından, [soyut](../language-reference/keywords/abstract.md) bir yöntem çağrılamaz.</span><span class="sxs-lookup"><span data-stu-id="6a769-104">You cannot call an [abstract](../language-reference/keywords/abstract.md) method because it does not have a method body.</span></span> <span data-ttu-id="6a769-105">Daha fazla bilgi için bkz.  <span data-ttu-id="6a769-106">Aşağıdaki örnek CS0205 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="6a769-106">The following sample generates CS0205:</span></span>  
  
```csharp  
// CS0205.cs  
abstract public class MyClass  
{  
   abstract public void M();  
}  
  
public class MyClass2 : MyClass  
{  
   public override void M()  
   {  
      base.M();   // CS0205, delete this line  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
