# <a name="compiler-warning-level-2-cs0472"></a><span data-ttu-id="f876b-102">Derleyici Uyarısı (Düzey 2) CS0472</span><span class="sxs-lookup"><span data-stu-id="f876b-102">Compiler Warning (level 2) CS0472</span></span>
<span data-ttu-id="f876b-103">'Value2' türünde bir değer hiçbir zaman eşit 'null' türü 'value3' olmadığından ifadenin sonucu her zaman 'value1' olan</span><span class="sxs-lookup"><span data-stu-id="f876b-103">The result of the expression is always 'value1' since a value of type 'value2' is never equal to 'null' of type 'value3'</span></span>  
  
 <span data-ttu-id="f876b-104">Derleyici, sabit bir null değeri ile bir işleç kullanıyorsanız bildirmelisiniz.</span><span class="sxs-lookup"><span data-stu-id="f876b-104">The compiler should warn if you use an operator with a constant null value.</span></span>  
  
## <a name="example"></a><span data-ttu-id="f876b-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="f876b-105">Example</span></span>  
 <span data-ttu-id="f876b-106">Aşağıdaki örnek, CS0472 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="f876b-106">The following sample generates CS0472.</span></span>  
  
```csharp  
public class Test  
{  
    public static int Main()  
    {  
        int i = 5;  
        int counter = 0;  
  
        // Comparison:  
        if (i == null)  // CS0472  
        // To resolve, use a valid value for i.  
            counter++;   
        return counter;  
    }  
}  
```
