# <a name="compiler-warning-level-2-cs0472"></a>Derleyici Uyarısı (Düzey 2) CS0472
'Value2' türünde bir değer hiçbir zaman eşit 'null' türü 'value3' olmadığından ifadenin sonucu her zaman 'value1' olan  
  
 Derleyici, sabit bir null değeri ile bir işleç kullanıyorsanız bildirmelisiniz.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS0472 oluşturur.  
  
```csharp  
public class Test  
{  
    public static int Main()  
    {  
        int i = 5;  
        int counter = 0;  
  
        // Comparison:  
        if (i == null)  // CS0472  
        // To resolve, use a valid value for i.  
            counter++;   
        return counter;  
    }  
}  
```
