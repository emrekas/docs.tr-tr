## <a name="example"></a><span data-ttu-id="46530-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="46530-107">Example</span></span>  
 <span data-ttu-id="46530-108">Aşağıdaki örnek, CS3019 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="46530-108">The following sample generates CS3019:</span></span>  
  
```csharp  
// CS3019.cs

[assembly: CLSCompliant(true)]

{  
    [CLSCompliant(false)]  // CS3019

    {  
    }

    {  
    }  
}  
```
