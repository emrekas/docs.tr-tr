<span data-ttu-id="527c5-103">Aşırı yüklenen kaydırma işlecinin ilk işleneni aynı türü kapsayan tür olmalıdır ve ikinci işlenenin türü de tamsayı olmalıdır</span><span class="sxs-lookup"><span data-stu-id="527c5-103">The first operand of an overloaded shift operator must have the same type as the containing type, and the type of the second operand must be int</span></span>

```csharp  
   }

   {  
   }  
}  
```
