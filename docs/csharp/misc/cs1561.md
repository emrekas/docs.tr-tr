# <a name="compiler-error-cs1561"></a>Derleyici Hatası CS1561
Çıkış dosyası adı çok uzun veya geçersiz  
  
 Çıkış dosyası adı 256 karakterden uzun olamaz ve Avrupa simgesi, soru işareti ya da, diğerlerinin yanı sıra ters eğik çizgi gibi geçersiz karakterler içeremez.
