# <a name="compiler-error-cs1577"></a>Derleyici Hatası CS1577
Derleme oluşturma başarısız oldu; nedeni  
  
 C# derleyicisi kaynak kod başarıyla derlenen sonra derleme, derleme oluşturma parçası başarısız oldu. ALINK yardımcı programı hataları belgelerine bakın ([Al.exe araç hataları ve Uyarıları](../../framework/tools/al-exe-assembly-linker.md#errors-and-warnings)) daha fazla bilgi için.
