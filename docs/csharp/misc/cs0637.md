# <a name="compiler-error-cs0637"></a><span data-ttu-id="6aed9-102">Derleyici hatası CS0637</span><span class="sxs-lookup"><span data-stu-id="6aed9-102">Compiler Error CS0637</span></span>

<span data-ttu-id="6aed9-103">Statik veya const alanlarında FieldOffset özniteliğine izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="6aed9-103">The FieldOffset attribute is not allowed on static or const fields.</span></span>
  
 <span data-ttu-id="6aed9-104">[FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) özniteliği [static](../language-reference/keywords/static.md) veya [const](../language-reference/keywords/const.md)ile işaretlenmiş alanlarda kullanılamaz.</span><span class="sxs-lookup"><span data-stu-id="6aed9-104">The [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute cannot be used on fields marked with [static](../language-reference/keywords/static.md) or [const](../language-reference/keywords/const.md).</span></span>  
  
 <span data-ttu-id="6aed9-105">Aşağıdaki örnek CS0637 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="6aed9-105">The following sample generates CS0637:</span></span>  
  
```csharp  
// CS0637.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
public class MainClass  
{  
   [FieldOffset(3)]   // CS0637  
   public static int i;  
   public static void Main ()  
   {  
   }  
}  
```
