# <a name="compiler-error-cs0021"></a><span data-ttu-id="b4f81-102">Derleyici Hatası CS0021</span><span class="sxs-lookup"><span data-stu-id="b4f81-102">Compiler Error CS0021</span></span>

 <span data-ttu-id="b4f81-104">Dizin oluşturucunun desteklemediği bir veri türünde arasında bir değere erişmek için bir girişimde bulunuldu [dizin oluşturucular](../../csharp/programming-guide/indexers/index.md).</span><span class="sxs-lookup"><span data-stu-id="b4f81-104">An attempt was made to access a value through an indexer on a data type that does not support [Indexers](../../csharp/programming-guide/indexers/index.md).</span></span>

## <a name="example"></a><span data-ttu-id="b4f81-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="b4f81-108">Example</span></span>  
 <span data-ttu-id="b4f81-109">Bu dosya bir .dll dosyasına derler — ile `DefaultMember` özniteliği geçersiz kılınan kullanıma — hata oluşturmak için.</span><span class="sxs-lookup"><span data-stu-id="b4f81-109">This file compiles to a .dll file—with the `DefaultMember` attribute commented out—in order to generate the error.</span></span>  
  
```cpp  
// CPP0021.cpp  
// compile with: /clr /LD  
using namespace System::Reflection;  
// Uncomment the following line to resolve  
//[DefaultMember("myItem")]  
public ref class MyClassMC  
{  
        public:  
        property int myItem[int]  
        {  
            int get(int i){  return 5; }  
            void set(int i, int value) {}  
        }  
};  
```
