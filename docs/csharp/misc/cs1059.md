## <a name="example"></a><span data-ttu-id="44900-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="44900-110">Example</span></span>  
 <span data-ttu-id="44900-111">Aşağıdaki örnek, bir değişken değil `i` bir sabit olduğu ve `E` `Enum` bir tür olduğundan, öğeleri de sabit değerler olan bir CS1059 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="44900-111">The following example generates CS1059 because `i` is a constant, not a variable, and `E` is an `Enum` type, whose elements are also constant values.</span></span>  
  
```csharp  
// CS1059.cs  
    class Program  
    {  
        public enum E : sbyte  
        {  
            a = 1,  
            b = 2  
        }  
  
        static void Main(string[] args)  
        {  
            const int i = 0;  
            i++;            // CS1059  
            E test = E.a++; // CS1059  
        }  
    }  
```
