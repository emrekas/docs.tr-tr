# <a name="compiler-error-cs1713"></a>Derleyici Hatası CS1713
Typename1 türü için meta veri adı oluşturulurken beklenmeyen hata: 'Açıklaması'  
  
 Bu hata genellikle bir iç derleyici hatası kaynaklanır. Kodunuzda adının uzunluğu kısaltmayı ve ardından yeniden derlemeden gibi bazı küçük değişiklikler yapmak bu hatayı çözebilir.
