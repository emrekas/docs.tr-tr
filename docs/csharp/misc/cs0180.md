 <span data-ttu-id="67954-104">[Soyut](../language-reference/keywords/abstract.md) ve [extern](../language-reference/keywords/extern.md) anahtar sözcükleri birbirini dışlıyor.</span><span class="sxs-lookup"><span data-stu-id="67954-104">The [abstract](../language-reference/keywords/abstract.md) and [extern](../language-reference/keywords/extern.md) keywords are mutually exclusive.</span></span> <span data-ttu-id="67954-105">Anahtar sözcüğü, üyenin dosya dışında tanımlandığı anlamına gelir ve soyut, uygulamanın türetilmiş bir sınıfta sağlandığı anlamına gelir. `extern`</span><span class="sxs-lookup"><span data-stu-id="67954-105">The `extern` keyword means that the member is defined outside the file, and **abstract** means that the implementation is provided in a derived class.</span></span> <span data-ttu-id="67954-106">Daha fazla bilgi için bkz. [Yöntemler](../programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="67954-106">For more information, see [Methods](../programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="67954-107">Aşağıdaki örnek CS0180 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="67954-107">The following sample generates CS0180:</span></span>  
  
```csharp

      }  
   }  
}  
```
