# <a name="compiler-error-cs0180"></a>Derleyici Hatası CS0180
'member' hem dış hem soyut olamaz  
  
 [Soyut](../../csharp/language-reference/keywords/abstract.md) ve [extern](../../csharp/language-reference/keywords/extern.md) anahtar sözcükleri karşılıklı olarak birbirini dışlar. `extern` Anahtar sözcüğü anlamına gelir üye dosyanın dışında tanımlanır ve **soyut** uygulaması türetilen bir sınıfta sağlanan anlamına gelir. Daha fazla bilgi için [yöntemleri](../../csharp/programming-guide/classes-and-structs/methods.md).  
  
 Aşağıdaki örnek, CS0180 oluşturur:  
  
```csharp  
// CS0180.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public extern abstract int Foo(int a);   // CS0180  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
