 <span data-ttu-id="750d1-104">[Soyut](../../csharp/language-reference/keywords/abstract.md) ve [extern](../../csharp/language-reference/keywords/extern.md) anahtar sözcükleri karşılıklı olarak birbirini dışlar.</span><span class="sxs-lookup"><span data-stu-id="750d1-104">The [abstract](../../csharp/language-reference/keywords/abstract.md) and [extern](../../csharp/language-reference/keywords/extern.md) keywords are mutually exclusive.</span></span> <span data-ttu-id="750d1-105">`extern` Anahtar sözcüğü anlamına gelir üye dosyanın dışında tanımlanır ve **soyut** uygulaması türetilen bir sınıfta sağlanan anlamına gelir.</span><span class="sxs-lookup"><span data-stu-id="750d1-105">The `extern` keyword means that the member is defined outside the file, and **abstract** means that the implementation is provided in a derived class.</span></span> <span data-ttu-id="750d1-106">Daha fazla bilgi için [yöntemleri](../../csharp/programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="750d1-106">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>

```csharp  
   {  
      public extern abstract int Foo(int a);   // CS0180

      }  
   }  
}  
```
