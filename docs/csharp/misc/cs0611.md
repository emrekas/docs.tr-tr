# <a name="compiler-error-cs0611"></a><span data-ttu-id="e6251-102">Derleyici Hatası CS0611</span><span class="sxs-lookup"><span data-stu-id="e6251-102">Compiler Error CS0611</span></span>

 <span data-ttu-id="e6251-104">Bir dizi türü olarak kullanılamaz bazı türleri vardır.</span><span class="sxs-lookup"><span data-stu-id="e6251-104">There are some types that cannot be used as the type of an array.</span></span> <span data-ttu-id="e6251-105">Bu türler **System.TypedReference** ve **System.ArgIterator**.</span><span class="sxs-lookup"><span data-stu-id="e6251-105">These types include **System.TypedReference** and **System.ArgIterator**.</span></span>

```csharp  
{  
   public static void Main()  
   {

   }

```
