# <a name="compiler-error-cs0685"></a><span data-ttu-id="7774f-102">Derleyici Hatası CS0685</span><span class="sxs-lookup"><span data-stu-id="7774f-102">Compiler Error CS0685</span></span>
<span data-ttu-id="7774f-103">'Member' koşullu üyesinde out parametresi olamaz</span><span class="sxs-lookup"><span data-stu-id="7774f-103">Conditional member 'member' cannot have an out parameter</span></span>  
  
 <span data-ttu-id="7774f-104">Kullanırken <xref:System.Diagnostics.ConditionalAttribute> yöntemin out parametresi olamaz, bir yöntem özniteliği.</span><span class="sxs-lookup"><span data-stu-id="7774f-104">When using the <xref:System.Diagnostics.ConditionalAttribute> attribute on a method, that method may not have an out parameter.</span></span> <span data-ttu-id="7774f-105">Out parametresi için kullanılan bir değişkenin değerini bir şey yöntem çağrısı derlenir durumda tanımlanmıştır değil olmasıdır.</span><span class="sxs-lookup"><span data-stu-id="7774f-105">This is because the value of the variable used for the out parameter would not be defined in the case that the method call is compiled to nothing.</span></span> <span data-ttu-id="7774f-106">Bu hatadan kaçınmak için koşullu yöntem bildiriminden out parametresi kaldırın veya Conditional özniteliği kullanmayın.</span><span class="sxs-lookup"><span data-stu-id="7774f-106">To avoid this error, remove the out parameter from the conditional method declaration, or don't use the Conditional Attribute.</span></span>  
  
## <a name="example"></a><span data-ttu-id="7774f-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="7774f-107">Example</span></span>  
 <span data-ttu-id="7774f-108">Aşağıdaki örnek, CS0685 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="7774f-108">The following sample generates CS0685:</span></span>
