# <a name="compiler-error-cs0574"></a>Derleyici Hatası CS0574
Yıkıcının adı sınıfın adı eşleşmelidir  
  
 Bir yok edici adını bir tilde (~) tarafından öncesinde sınıf adı olmalıdır.  
  
 Aşağıdaki örnek, CS0574 oluşturur:  
  
```csharp  
// CS0574.cs  
namespace x  
{  
   public class iii  
   {  
      ~iiii()   // CS0574  
      {  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
