# <a name="compiler-error-cs1630"></a>Derleyici hatası CS1630
/Errorreport için ' Option ' geçersiz seçeneği; Prompt, Send, Queue veya None olmalıdır  
  
 Komut satırı seçeneği [/errorreport](../language-reference/compiler-options/errorreport-compiler-option.md) , bir iç derleyici hatası oluştuğunda hangi eylemin yapılacağını belirten **Prompt**, **Send**, **Queue**veya **none**gelmelidir.
