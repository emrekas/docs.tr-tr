# <a name="compiler-error-cs1630"></a>Derleyici Hatası CS1630
Geçersiz seçenek 'seçeneği' / errorreport için; istemi, gönderme, kuyruk veya none olmalıdır  
  
 Komut satırı seçeneği [/errorreport](../../csharp/language-reference/compiler-options/errorreport-compiler-option.md) gelmelidir **istemi**, **Gönder**, **kuyruk**, veya **hiçbiri**, derleyici iç hatası oluştuğunda hangi eylemin uygulanan istediğiniz belirtme.
