# <a name="compiler-error-cs1503"></a>Derleyici Hatası CS1503
'Number' bağımsız değişkeni için TypeB TypeA dönüştürülemiyor  
  
 Yöntemde bir bağımsız değişken türünü, sınıfı oluşturulduğunda geçirilen türü eşleşmiyor. Bu hata genellikle CS1502 birlikte görüntülenir. Bkz: [CS1502](../../csharp/language-reference/compiler-messages/cs1502.md) bu hatanın nasıl düzeltileceği hakkında ayrıntılı bilgi için.
