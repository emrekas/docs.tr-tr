# <a name="compiler-error-cs0017"></a><span data-ttu-id="04a2e-102">Derleyici hatası CS0017</span><span class="sxs-lookup"><span data-stu-id="04a2e-102">Compiler Error CS0017</span></span>
<span data-ttu-id="04a2e-103">' Çıkış dosyası adı ' programında tanımlanmış birden fazla giriş noktası yok.</span><span class="sxs-lookup"><span data-stu-id="04a2e-103">Program 'output file name' has more than one entry point defined.</span></span> <span data-ttu-id="04a2e-104">Giriş noktasını içeren türü belirtmek için/Main ile derleyin.</span><span class="sxs-lookup"><span data-stu-id="04a2e-104">Compile with /main to specify the type that contains the entry point.</span></span>  
  
 <span data-ttu-id="04a2e-105">Programda yalnızca bir tane [ana](../programming-guide/main-and-command-args/index.md) yöntem olabilir.</span><span class="sxs-lookup"><span data-stu-id="04a2e-105">A program can only have one [Main](../programming-guide/main-and-command-args/index.md) method.</span></span>  
  
 <span data-ttu-id="04a2e-106">Bu hatayı çözmek için, kodunuzda bulunan tüm ana yöntemleri silebilir ya da hangi ana yöntemi kullanmak istediğinizi belirtmek için [/Main](../language-reference/compiler-options/main-compiler-option.md) derleyici seçeneğini kullanabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="04a2e-106">To resolve this error, you can either delete all Main methods in your code, except one, or you can use the [/main](../language-reference/compiler-options/main-compiler-option.md) compiler option to specify which Main method you want to use.</span></span>  
  
 <span data-ttu-id="04a2e-107">Aşağıdaki örnek CS0017 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="04a2e-107">The following sample generates CS0017:</span></span>  
  
```csharp  
// CS0017.cs  
// compile with: /target:exe  
public class clx  
{  
   static public void Main()  
   {  
   }  
}  
  
public class cly  
{  
   public static void Main()   // CS0017, delete one Main or use /main  
   {  
   }  
}  
```
