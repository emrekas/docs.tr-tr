<span data-ttu-id="3f5bf-103">Program 'çıkış dosyası adı' tanımlı birden fazla giriş noktası vardır.</span><span class="sxs-lookup"><span data-stu-id="3f5bf-103">Program 'output file name' has more than one entry point defined.</span></span> <span data-ttu-id="3f5bf-104">Giriş noktasını içeren türü belirtmek için/Main ile derleyin.</span><span class="sxs-lookup"><span data-stu-id="3f5bf-104">Compile with /main to specify the type that contains the entry point.</span></span>  
  
 <span data-ttu-id="3f5bf-105">Bir programın yalnızca bir sağlayabilirsiniz [ana](../../csharp/programming-guide/main-and-command-args/index.md) yöntemi.</span><span class="sxs-lookup"><span data-stu-id="3f5bf-105">A program can only have one [Main](../../csharp/programming-guide/main-and-command-args/index.md) method.</span></span>  
  
 <span data-ttu-id="3f5bf-106">Bu hatayı gidermek için kodunuzdaki biri dışında tüm Main metotları silebilir veya kullanabilirsiniz [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) hangi Main yöntemini kullanmak istediğinizi belirtmek için derleyici seçeneği.</span><span class="sxs-lookup"><span data-stu-id="3f5bf-106">To resolve this error, you can either delete all Main methods in your code, except one, or you can use the [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) compiler option to specify which Main method you want to use.</span></span>  
  
 <span data-ttu-id="3f5bf-107">Aşağıdaki örnek, CS0017 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="3f5bf-107">The following sample generates CS0017:</span></span>  
  
```csharp  
// CS0017.cs  
// compile with: /target:exe  
public class clx  
{  
   static public void Main()  
   {  
   }  
}  
  
public class cly  
{  
   public static void Main()   // CS0017, delete one Main or use /main  
   {  
   }  
}  
```
