# <a name="compiler-error-cs1552"></a>Derleyici Hatası CS1552
Dizi türü Belirleyicisi [], parametre adından önce gelmelidir  
  
 Değişken adında dizi bildirimi dizi türü Belirleyicisi konumunu sonradır.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS1552 oluşturur:  
  
```csharp  
// CS1552.cs  
public class C  
{  
    public static void Main(string args[])   // CS1552  
    // try the following line instead  
    // public static void Main(string [] args)  
    {  
    }  
}  
```
