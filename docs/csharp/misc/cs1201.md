# <a name="compiler-warning-level-1-cs1201"></a>Derleyici Uyarısı (düzey 1) CS1201
'Geçersiz özellik' özelliği kullanımdan kaldırılmıştır. Bunun yerine 'özelliği geçerli' kullanın.  
  
 Kullanmaya çalıştığınız özellik artık kullanılmıyor. Bunun yerine geçerli bu özelliği kullanmak için kodunuzu güncelleştirin.
