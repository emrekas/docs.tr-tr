# <a name="compiler-error-cs0113"></a>Derleyici Hatası CS0113
'Function' üyesi new veya virtual olarak işaretlenemez geçersiz kılma olarak işaretlendi  
  
 Bir yöntemi işaretlemek için birbirini dışlayan [yeni](../../csharp/language-reference/keywords/new.md) ve [geçersiz kılma](../../csharp/language-reference/keywords/override.md) anahtar sözcükleri.  
  
 Aşağıdaki örnek, CS0113 oluşturur:  
  
```csharp  
// CS0113.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
  
   public class MyClass2 : MyClass  
   {  
      override new public void Foo()   // CS0113, remove new keyword  
      {  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
