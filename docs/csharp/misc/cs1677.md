# <a name="compiler-error-cs1677"></a><span data-ttu-id="16185-102">Derleyici Hatası CS1677</span><span class="sxs-lookup"><span data-stu-id="16185-102">Compiler Error CS1677</span></span>
<span data-ttu-id="16185-103">Parametre 'number', 'anahtar sözcüğü' anahtar sözcüğü ile bildirilmemelidir</span><span class="sxs-lookup"><span data-stu-id="16185-103">Parameter 'number' should not be declared with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="16185-104">Parametre türü değiştiricisi bir anonim yöntemde yöntemi atama temsilci bildiriminde kullanılan eşleşmediğinde bu hata oluşur.</span><span class="sxs-lookup"><span data-stu-id="16185-104">This error occurs when the parameter type modifier in an anonymous method does not match that used in the declaration of the delegate, to which you are casting the method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="16185-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="16185-105">Example</span></span>  
 <span data-ttu-id="16185-106">Aşağıdaki örnek, CS1677 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="16185-106">The following sample generates CS1677:</span></span>  
  
```csharp  
// CS1677.cs  
delegate void D(int i);  
class Errors  
{  
    static void Main()   
    {  
        D d = delegate(out int i) { };   // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int i) { };  
  
        D d = delegate(ref int j){}; // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int j){};  
    }  
}  
```
