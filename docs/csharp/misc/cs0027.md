# <a name="compiler-error-cs0027"></a><span data-ttu-id="10823-102">Derleyici Hatası CS0027</span><span class="sxs-lookup"><span data-stu-id="10823-102">Compiler Error CS0027</span></span>
<span data-ttu-id="10823-103">Anahtar sözcüğü 'this' geçerli bağlamda kullanılamaz</span><span class="sxs-lookup"><span data-stu-id="10823-103">Keyword 'this' is not available in the current context</span></span>  
  
 <span data-ttu-id="10823-104">[Bu](../../csharp/language-reference/keywords/this.md) anahtar sözcüğü, bir özellik, yöntem veya Oluşturucu dışında bulundu.</span><span class="sxs-lookup"><span data-stu-id="10823-104">The [this](../../csharp/language-reference/keywords/this.md) keyword was found outside of a property, method, or constructor.</span></span>  
  
 <span data-ttu-id="10823-105">Bu hatayı düzeltmek için kullanımı ortadan kaldırmak için deyim ya da değiştirme `this` anahtar sözcüğü ve/veya taşıma kısmını veya tamamını deyimi içinde bir özellik, yöntem veya Oluşturucu.</span><span class="sxs-lookup"><span data-stu-id="10823-105">To fix this error, either modify the statement to eliminate use of the `this` keyword, and/or move part or all of the statement inside a property, method, or constructor.</span></span>  
  
 <span data-ttu-id="10823-106">Aşağıdaki örnek, CS0027 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="10823-106">The following example generates CS0027:</span></span>  
  
```csharp  
using System;  
using System.Collections.Generic;  
using System.Text;  
  
namespace ConsoleApplication3  
{  
    class MyClass  
    {  
  
        int err1 = this.Fun() + 1;  // CS0027   
  
        public int Fun()  
        {  
            return 10;  
        }  
  
        public void Test()  
        {  
            // valid use of this  
            int err = this.Fun() + 1;  
            Console.WriteLine(err);  
        }  
  
        public static void Main()  
        {  
            MyClass c = new MyClass();  
            c.Test();  
        }  
    }  
}  
```
