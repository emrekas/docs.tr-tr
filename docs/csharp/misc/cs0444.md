# <a name="compiler-warning-level-2-cs0444"></a>Derleyici Uyarısı (Düzey 2) CS0444
Önceden tanımlanmış türü ' tür adı 1' bulunamadı ' sistem ad alanı 1' içinde bulundu ancak ' sistem ad alanı 2'  
  
 Önceden tanımlanmış bir nesne gibi <xref:System.Int32> bulmak, ancak bunun yerine kaynakta bulunan derleme beklenirken bulunamadı ' sistem ad alanı 2'.  
  
 Hata, .NET Framework yanlış yüklü olduğunu gösteriyor olabilir. Bu sorunu gidermek için .NET Framework yeniden yükleyin.  
  
 Kendi taban sınıfı kitaplıkları yazıyorsanız, ayrıca bu hatayla karşılaşabilirsiniz. Bu durumda, hatayı gidermek için mscorlib yeniden oluşturun.
