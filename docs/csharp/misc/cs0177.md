# <a name="compiler-error-cs0177"></a><span data-ttu-id="bb992-102">Derleyici Hatası CS0177</span><span class="sxs-lookup"><span data-stu-id="bb992-102">Compiler Error CS0177</span></span>
<span data-ttu-id="bb992-103">Denetim geçerli yöntemi terk etmeden önce 'parametresi' out parametresi atanmalıdır</span><span class="sxs-lookup"><span data-stu-id="bb992-103">The out parameter 'parameter' must be assigned to before control leaves the current method</span></span>  
  
 <span data-ttu-id="bb992-104">Bir parametre ile işaretlenen [kullanıma](../../csharp/language-reference/keywords/out-parameter-modifier.md) anahtar sözcüğü, yöntem gövdesini bir değer değil atandı.</span><span class="sxs-lookup"><span data-stu-id="bb992-104">A parameter marked with the [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) keyword was not assigned a value in the method body.</span></span> <span data-ttu-id="bb992-105">Daha fazla bilgi için [parametreleri geçirme](../../csharp/programming-guide/classes-and-structs/passing-parameters.md)</span><span class="sxs-lookup"><span data-stu-id="bb992-105">For more information, see [Passing Parameters](../../csharp/programming-guide/classes-and-structs/passing-parameters.md)</span></span>  
  
 <span data-ttu-id="bb992-106">Aşağıdaki örnek, CS0177 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="bb992-106">The following sample generates CS0177:</span></span>  
  
```csharp  
// CS0177.cs  
public class MyClass  
{  
   public static void Foo(out int i)   // CS0177  
   {  
   // uncomment the following line to resolve this error  
   //   i = 0;  
   }  
  
   public static void Main()  
   {  
       int x = -1;  
       Foo(out x);  
   }  
}  
```
