# <a name="compiler-error-cs0225"></a>Derleyici hatası CS0225
Params parametresi tek boyutlu bir dizi olmalıdır  
  
 [Params](../language-reference/keywords/params.md) anahtar sözcüğünü kullanırken, veri türünde tek boyutlu bir dizi belirtmeniz gerekir. Daha fazla bilgi için bkz. [Yöntemler](../programming-guide/classes-and-structs/methods.md).  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS0225 oluşturur:  
  
```csharp  
// CS0225.cs  
public class MyClass  
{  
    public static void TestParams(params int a)   // CS0225  
    // try the following line instead  
    // public static void TestParams(params int[] a)  
    {  
    }  
  
    public static void Main()  
    {  
        TestParams(1);  
    }  
}  
```
