# <a name="compiler-error-cs1555"></a>Derleyici Hatası CS1555
Main yöntemi için belirtilen'class ' bulunamadı.  
  
 Bir sınıf için belirtilen [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) derleyici seçeneği, ancak sınıf adı, kaynak kodunda bulunamadı.
