# <a name="compiler-warning-level-1-cs3000"></a>Derleyici Uyarısı (düzey 1) CS3000
Değişken bağımsız değişkenleri olan yöntemleri CLS uyumlu değildir  
  
 Yönteminde kullanılan bağımsız değişkenler, ortak dil özellikleri (CLS) olmayan özellikleri kullanıma sunar. CLS uyumluluğu hakkında daha fazla bilgi için bkz. [dil bağımsızlığı ve dilden bağımsız bileşenler](../../standard/language-independence.md).
  
 Aşağıdaki örnek, CS3000 uyarı oluşturur.  
  
```csharp  
// CS3000.cs  
// compile with: /target:library  
// CS3000 expected  
[assembly:System.CLSCompliant(true)]  
  
public class Test  
{  
   public void AddABunchOfInts( __arglist ) {}   // CS3000  
}  
```
