# <a name="compiler-warning-level-1-cs0197"></a><span data-ttu-id="a4cce-102">Derleyici Uyarısı (düzey 1) CS0197</span><span class="sxs-lookup"><span data-stu-id="a4cce-102">Compiler Warning (level 1) CS0197</span></span>
<span data-ttu-id="a4cce-103">Başvuruya göre sıralama sınıfının bir alanı olduğu için 'bağımsız değişkeni' ref veya out olarak geçirerek ya da adresini almak çalışma zamanı özel durumuna neden olabilir</span><span class="sxs-lookup"><span data-stu-id="a4cce-103">Passing 'argument' as ref or out or taking its address may cause a runtime exception because it is a field of a marshal-by-reference class</span></span>  
  
 <span data-ttu-id="a4cce-104">Doğrudan veya dolaylı olarak türeyen bir sınıfı gelen <xref:System.MarshalByRefObject> bir başvuruya göre sınıfıdır.</span><span class="sxs-lookup"><span data-stu-id="a4cce-104">Any class that derives, directly or indirectly, from <xref:System.MarshalByRefObject> is a marshal-by-reference class.</span></span> <span data-ttu-id="a4cce-105">Bu sınıf, işlem ve makine sınırlarında başvuruya göre sıralanabilir.</span><span class="sxs-lookup"><span data-stu-id="a4cce-105">Such a class can be marshaled by reference across process and machine boundaries.</span></span> <span data-ttu-id="a4cce-106">Bu nedenle, bu sınıfın örneklerinin uzak nesnelere proxy'leri olabilir.</span><span class="sxs-lookup"><span data-stu-id="a4cce-106">Thus, instances of this class could be proxies to remote objects.</span></span> <span data-ttu-id="a4cce-107">Bir alan bir proxy nesnesi olarak geçirilemez [ref](../../csharp/language-reference/keywords/ref.md) veya [kullanıma](../../csharp/language-reference/keywords/out-parameter-modifier.md). Bu nedenle, bu tür bir sınıf olarak alanlarının geçirilemez `ref` veya `out`, örneği olmadığı sürece [bu](../../csharp/language-reference/keywords/this.md), bir proxy nesnesi olamaz.</span><span class="sxs-lookup"><span data-stu-id="a4cce-107">You cannot pass a field of a proxy object as [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out-parameter-modifier.md). So, you cannot pass fields of such a class as `ref` or `out`, unless the instance is [this](../../csharp/language-reference/keywords/this.md), which can not be a proxy object.</span></span>  
  
## <a name="example"></a><span data-ttu-id="a4cce-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="a4cce-108">Example</span></span>  
 <span data-ttu-id="a4cce-109">Aşağıdaki örnek, CS0197 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="a4cce-109">The following sample generates CS0197.</span></span>  
  
```csharp  
// CS0197.cs  
// compile with: /W:1  
class X : System.MarshalByRefObject  
{  
   public int i;  
}  
  
class M  
{  
   public int i;  
   static void AddSeventeen(ref int i)  
   {  
      i += 17;  
   }  
  
   static void Main()  
   {  
      X x = new X();  
      x.i = 12;  
      AddSeventeen(ref x.i);   // CS0197  
  
      // OK  
      M m = new M();  
      m.i = 12;  
      AddSeventeen(ref m.i);  
   }  
}  
```
