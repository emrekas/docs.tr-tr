<span data-ttu-id="c2fc6-103">Başvuruya göre sıralama sınıfının bir alanı olduğundan, ' Argument ' öğesini ref veya out olarak geçirmek veya adresini almak çalışma zamanı özel durumuna neden olabilir</span><span class="sxs-lookup"><span data-stu-id="c2fc6-103">Passing 'argument' as ref or out or taking its address may cause a runtime exception because it is a field of a marshal-by-reference class</span></span>  
  
 <span data-ttu-id="c2fc6-104">Doğrudan veya dolaylı olarak türetilmiş tüm sınıflar, <xref:System.MarshalByRefObject> başvuruya göre sıralama sınıfıdır.</span><span class="sxs-lookup"><span data-stu-id="c2fc6-104">Any class that derives, directly or indirectly, from <xref:System.MarshalByRefObject> is a marshal-by-reference class.</span></span> <span data-ttu-id="c2fc6-105">Bu tür bir sınıf, işlem ve makine sınırları genelinde başvuruya göre sıralanabilir.</span><span class="sxs-lookup"><span data-stu-id="c2fc6-105">Such a class can be marshaled by reference across process and machine boundaries.</span></span> <span data-ttu-id="c2fc6-106">Bu nedenle, bu sınıfın örnekleri uzak nesnelere proxy olabilir.</span><span class="sxs-lookup"><span data-stu-id="c2fc6-106">Thus, instances of this class could be proxies to remote objects.</span></span> <span data-ttu-id="c2fc6-107">Ara sunucu nesnesinin alanını [ref](../language-reference/keywords/ref.md) veya [Out](../language-reference/keywords/out-parameter-modifier.md)olarak geçiremezsiniz. Bu nedenle, `ref` [Bu](../language-reference/keywords/this.md)tür bir sınıfın alanlarını veya `out`gibi bir ara sunucu nesnesi olmayan bir sınıf olarak geçiremezsiniz.</span><span class="sxs-lookup"><span data-stu-id="c2fc6-107">You cannot pass a field of a proxy object as [ref](../language-reference/keywords/ref.md) or [out](../language-reference/keywords/out-parameter-modifier.md). So, you cannot pass fields of such a class as `ref` or `out`, unless the instance is [this](../language-reference/keywords/this.md), which can not be a proxy object.</span></span>  
  
## <a name="example"></a><span data-ttu-id="c2fc6-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="c2fc6-108">Example</span></span>  
 <span data-ttu-id="c2fc6-109">Aşağıdaki örnek CS0197 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="c2fc6-109">The following sample generates CS0197.</span></span>  
  
```csharp  
// CS0197.cs  
// compile with: /W:1  
class X : System.MarshalByRefObject  
{  
   public int i;  
}  
  
class M  
{  
   public int i;  
   static void AddSeventeen(ref int i)  
   {  
      i += 17;  
   }  
  
   static void Main()  
   {  
      X x = new X();  
      x.i = 12;  
      AddSeventeen(ref x.i);   // CS0197  
  
      // OK  
      M m = new M();  
      m.i = 12;  
      AddSeventeen(ref m.i);  
   }  
}  
```
