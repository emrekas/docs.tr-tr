# <a name="compiler-error-cs0014"></a>Derleyici Hatası CS0014
Gerekli 'dosya' dosyası bulunamadı  
  
 Bir dosya, derleyici tarafından gerekli ancak sistem üzerinde değil. Yolun doğru olduğundan emin olun. Ardından dosyayı bir Visual Studio sistem dosyası ise, yüklemenizi onarın veya kaldırın ve Visual Studio tamamen yeniden gerekebilir.
