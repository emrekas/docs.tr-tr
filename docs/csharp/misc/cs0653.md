 <span data-ttu-id="1d87b-104">Bir [soyut](../../csharp/language-reference/keywords/abstract.md) özel öznitelik sınıfı bir öznitelik olarak kullanılamaz.</span><span class="sxs-lookup"><span data-stu-id="1d87b-104">An [abstract](../../csharp/language-reference/keywords/abstract.md) custom attribute class cannot be used as an attribute.</span></span>

{  
}
