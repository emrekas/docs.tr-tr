# <a name="compiler-error-cs0653"></a>Derleyici Hatası CS0653
Soyut olduğu için 'class' öznitelik sınıfı uygulanamaz  
  
 Bir [soyut](../../csharp/language-reference/keywords/abstract.md) özel öznitelik sınıfı bir öznitelik olarak kullanılamaz.  
  
 Aşağıdaki örnek, CS0653 oluşturur:  
  
```csharp  
// CS0653.cs  
using System;  
  
public abstract class MyAttribute : Attribute  
{  
}  
  
public class My2Attribute : MyAttribute  
{  
}  
  
[My]   // CS0653  
// try the following line instead  
// [My2]  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
