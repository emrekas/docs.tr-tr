# <a name="compiler-error-cs0653"></a><span data-ttu-id="c91a0-102">Derleyici hatası CS0653</span><span class="sxs-lookup"><span data-stu-id="c91a0-102">Compiler Error CS0653</span></span>

 <span data-ttu-id="c91a0-104">[Soyut](../language-reference/keywords/abstract.md) özel öznitelik sınıfı öznitelik olarak kullanılamaz.</span><span class="sxs-lookup"><span data-stu-id="c91a0-104">An [abstract](../language-reference/keywords/abstract.md) custom attribute class cannot be used as an attribute.</span></span>  
  
 <span data-ttu-id="c91a0-105">Aşağıdaki örnek CS0653 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="c91a0-105">The following sample generates CS0653:</span></span>  
  
```csharp

{  
}  
  
public class My2Attribute : MyAttribute  
{  
}
```
