<span data-ttu-id="c97e2-103">'Class' öğesinde ComImport özniteliği olduğundan, 'method' extern veya abstract olmalıdır</span><span class="sxs-lookup"><span data-stu-id="c97e2-103">Since 'class' has the ComImport attribute, 'method' must be extern or abstract</span></span>  
  
 <span data-ttu-id="c97e2-104">ComImport özniteliğine belirterek uygulama sınıfı için bir COM modülü içeri aktarılamadı olduğunu belirtir.</span><span class="sxs-lookup"><span data-stu-id="c97e2-104">Specifying the ComImport attribute implies that the implementation for the class is to be imported from a COM module.</span></span> <span data-ttu-id="c97e2-105">Ek yöntemleri tanımlı değil.</span><span class="sxs-lookup"><span data-stu-id="c97e2-105">Additional methods may not be defined.</span></span>  
  
 <span data-ttu-id="c97e2-106">Aşağıdaki örnek, CS0423 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="c97e2-106">The following sample generates CS0423:</span></span>  
  
```csharp  
// CS0423.cs  
  
using System.Runtime.InteropServices;  
  
[  
  ComImport,  
  Guid("7ab770c7-0e23-4d7a-8aa2-19bfad479829")  
]  
class ImageProperties  
{  
  public static void Main()  // CS0423  
  {  
    ImageProperties i = new ImageProperties();  
  }  
}  
```
