# <a name="compiler-error-cs0423"></a>Derleyici Hatası CS0423
'Class' öğesinde ComImport özniteliği olduğundan, 'method' extern veya abstract olmalıdır  
  
 ComImport özniteliğine belirterek uygulama sınıfı için bir COM modülü içeri aktarılamadı olduğunu belirtir. Ek yöntemleri tanımlı değil.  
  
 Aşağıdaki örnek, CS0423 oluşturur:  
  
```csharp  
// CS0423.cs  
  
using System.Runtime.InteropServices;  
  
[  
  ComImport,  
  Guid("7ab770c7-0e23-4d7a-8aa2-19bfad479829")  
]  
class ImageProperties  
{  
  public static void Main()  // CS0423  
  {  
    ImageProperties i = new ImageProperties();  
  }  
}  
```
