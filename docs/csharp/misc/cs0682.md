# <a name="compiler-error-cs0682"></a>Derleyici Hatası CS0682
'type1' dil tarafından desteklenmediğinden 'type2' uygulayamaz  
  
 Başka bir dilde yazılmış bir arabirim uygulamak deneyin ve derleyici arabirimi desteklemiyor bu hata oluşur.
