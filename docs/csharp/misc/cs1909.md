# <a name="compiler-error-cs1909"></a><span data-ttu-id="e1b5c-102">Derleyici Hatası CS1909</span><span class="sxs-lookup"><span data-stu-id="e1b5c-102">Compiler Error CS1909</span></span>
<span data-ttu-id="e1b5c-103">DefaultValue özniteliği, 'type' türünde bir parametre üzerinde geçerli değildir.</span><span class="sxs-lookup"><span data-stu-id="e1b5c-103">The DefaultValue attribute is not applicable on parameters of type 'type'</span></span>  
  
 <span data-ttu-id="e1b5c-104">CS1909 kullandığınızda oluşturulan bir `DefaultValue` parametre türüne uygulanamaz özniteliği.</span><span class="sxs-lookup"><span data-stu-id="e1b5c-104">CS1909 is generated when you use a `DefaultValue` attribute that is not applicable on the parameter type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="e1b5c-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="e1b5c-105">Example</span></span>  
 <span data-ttu-id="e1b5c-106">Aşağıdaki örnek, CS1909 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="e1b5c-106">The following sample generates CS1909.</span></span>  
  
```csharp  
// CS1909.cs  
// compile with: /target:library  
using System.Runtime.InteropServices;  
  
public interface ISomeInterface  
{  
   void Test1([DefaultParameterValue(new int[] {1, 2})] int[] arr1);   // CS1909  
  
   void Test2([DefaultParameterValue("Test String")] string s);   // OK  
}  
```
