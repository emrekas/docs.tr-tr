# <a name="compiler-error-cs0619"></a>Derleyici hatası CS0619
' Member ' artık kullanılmıyor: ' text '  
  
 Bir sınıf üyesi [kullanılmıyor] ile işaretlendi ((.. sınıf üyesine başvuruluyorsa bir hata verilecek şekilde/TS-TES/Concepts/Attributes/Common-Attributes.exe) özniteliği.  
  
 Aşağıdaki örnek CS0619 oluşturur:  
  
```csharp  
// CS0619.cs  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", true)]   // generates an error on use  
   public static void m()  
   {  
   }  
  
   // this is the method you should be using  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m();   // CS0619  
   }  
}  
```
