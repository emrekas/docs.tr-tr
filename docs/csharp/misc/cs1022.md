# <a name="compiler-error-cs1022"></a>Derleyici Hatası CS1022
Tür veya ad alanı tanımını veya dosya sonu bekleniyor  
  
 Küme ayraçları eşleşen bir dizi kaynak kodu dosyası yok.  
  
 Aşağıdaki örnek, CS1022 oluşturur:  
  
```csharp  
// CS1022.cs  
namespace x  
{  
}  
}   // CS1022  
```
