# <a name="compiler-error-cs1565"></a>Derleyici Hatası CS1565
Çakışan seçenekler belirtildi: Win32 kaynak dosyası; Win32 simgesi  
  
 Her ikisini de belirtmek için geçerli değil [/win32res](../../csharp/language-reference/compiler-options/win32res-compiler-option.md) derleyici seçeneği ve [/win32ico](../../csharp/language-reference/compiler-options/win32icon-compiler-option.md) aynı derlemede derleyici seçeneği.
