# <a name="compiler-error-cs0003"></a>Derleyici Hatası CS0003
Bellek yetersiz  
  
 Derleyici, derleme tamamlamak için yeterli sanal bellek ayıramadı. Gereksiz tüm uygulamaları kapatın ve yeniden derleyin.  
  
 Disk belleği boyutunu artırın ve boş disk alanı olduğundan emin olmak isteyebilirsiniz.  
  
 Bu hata ayrıca sürümleri Windows Yazılım Geliştirme Seti (SDK) eşleşmeyen sonucu olarak görünebilir ve C# derleyici veya bir veya daha fazla bozuk dosya destekleyen C# derleyici; Visual Studio'yu yeniden yükleyin.
