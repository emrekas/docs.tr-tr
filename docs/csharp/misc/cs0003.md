# <a name="compiler-error-cs0003"></a>Derleyici hatası CS0003
Yetersiz bellek  
  
 Derleyici, derlemeyi tamamlamaya yetecek sayıda sanal bellek ayıramadı. Tüm gereksiz uygulamaları kapatın ve yeniden derleyin.  
  
 Ayrıca, disk belleği boyutunu artırmak ve boş disk alanınız olduğundan emin olmak isteyebilirsiniz.  
  
 Bu hata ayrıca Windows SDK ve C# derleyicisinin eşleşmeyen sürümlerinin ve derleyicinin veya C# derleyicisini destekleyen bir veya daha fazla bozuk dosyanın bulunması sonucu olarak da görünebilir; Visual Studio 'Yu yeniden yükleyin.
