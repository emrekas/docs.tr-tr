<span data-ttu-id="4e9c3-103">İçinde birden fazla tür kullanılamaz bir for, using, fixed veya bildirim deyimi</span><span class="sxs-lookup"><span data-stu-id="4e9c3-103">Cannot use more than one type in a for, using, fixed, or declaration statement</span></span>  
  
 <span data-ttu-id="4e9c3-104">Derleyici, geçersiz bir deyim bulundu.</span><span class="sxs-lookup"><span data-stu-id="4e9c3-104">The compiler found an invalid statement.</span></span>  
  
 <span data-ttu-id="4e9c3-105">Aşağıdaki örnek, CS1044 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="4e9c3-105">The following sample generates CS1044:</span></span>  
  
```csharp  
// CS1044.cs

public class MyClass : IDisposable  
{  
   public void Dispose()  
   {  
      Console.WriteLine("Res1.Dispose()");  
   }  
  
   public static void Main()  
   {  
      using (MyClass mc1 = new MyClass(),

      {  
      }  
   }  
}  
```
