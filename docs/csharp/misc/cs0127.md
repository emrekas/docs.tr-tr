# <a name="compiler-error-cs0127"></a>Derleyici Hatası CS0127
'Function' void döndürdüğünden, return anahtar sözcüğünü bir object ifadesi izlememelidir  
  
 Bir yöntemle bir [void](../../csharp/language-reference/keywords/void.md) dönüş türü, değer döndüremez. Daha fazla bilgi için [yöntemleri](../../csharp/programming-guide/classes-and-structs/methods.md).  
  
 Aşağıdaki örnek, CS0127 oluşturur:  
  
```csharp  
// CS0127.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public int hiddenMember2  
      {  
         get  
         {  
            return 0;  
         }  
         set   // CS0127, set has an implicit void return type  
         {  
            return 0;   // remove return statement to resolve this CS0127  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
