# <a name="compiler-error-cs1536"></a><span data-ttu-id="871ed-102">Derleyici Hatası CS1536</span><span class="sxs-lookup"><span data-stu-id="871ed-102">Compiler Error CS1536</span></span>

 <span data-ttu-id="871ed-104">Bu gerekli veya belirtmek için geçerli değil bir [void](../../csharp/language-reference/keywords/void.md) parametresi dışında void bir işaretçi.</span><span class="sxs-lookup"><span data-stu-id="871ed-104">It is not necessary or valid to specify a [void](../../csharp/language-reference/keywords/void.md) parameter other than a void pointer.</span></span>

```csharp  
// CS1536.cs

{  
   public static int x( void )   // CS1536  
   // try the following line instead  
   // public static int x()

      return 0;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
