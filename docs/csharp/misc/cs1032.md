# <a name="compiler-error-cs1032"></a><span data-ttu-id="335b8-102">Derleyici Hatası CS1032</span><span class="sxs-lookup"><span data-stu-id="335b8-102">Compiler Error CS1032</span></span>
<span data-ttu-id="335b8-103">Tanımlama/önişlemci sembolleri dosyadaki ilk belirteçten sonra tanımları kaldırılamaz</span><span class="sxs-lookup"><span data-stu-id="335b8-103">Cannot define/undefine preprocessor symbols after first token in file</span></span>  
  
 <span data-ttu-id="335b8-104">`#define` Ve `#undef` [önişlemci yönergeleri](../../csharp/language-reference/preprocessor-directives/index.md) ad alanı bildiriminde Kullanılanlar gibi diğer tüm anahtar önce bir program başlangıcında kullanılmalıdır.</span><span class="sxs-lookup"><span data-stu-id="335b8-104">The `#define` and `#undef` [preprocessor directives](../../csharp/language-reference/preprocessor-directives/index.md) must be used at the beginning of a program, before any other keywords, such as those used in the namespace declaration.</span></span>  
  
 <span data-ttu-id="335b8-105">Aşağıdaki örnek, CS1032 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="335b8-105">The following sample generates CS1032:</span></span>  
  
```csharp  
// CS1032.cs  
namespace x  
{  
   public class clx  
   {  
      #define a   // CS1032, put before namespace  
      public static void Main()  
      {  
      }  
   }  
}  
```
