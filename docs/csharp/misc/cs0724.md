## <a name="example"></a><span data-ttu-id="8fc36-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="8fc36-105">Example</span></span>  
 <span data-ttu-id="8fc36-106">Aşağıdaki örnek, CS0724 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="8fc36-106">The following example generates CS0724.</span></span>  
  
```csharp  
// CS0724.cs  
using System;  
  
class X  
{  
    static void Test()  
    {  
        try  
        {  
            throw new Exception();  
        }  
        catch  
        {  
            try  
            {  
            }  
            finally  
            {  
                throw; // CS0724  
            }  
        }  
    }  
  
    static void Main()  
    {  
    }  
}  
```
