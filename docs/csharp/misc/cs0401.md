## <a name="example"></a><span data-ttu-id="f93fe-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="f93fe-105">Example</span></span>  
 <span data-ttu-id="f93fe-106">Aşağıdaki örnek, CS0401 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="f93fe-106">The following sample generates CS0401.</span></span>  
  
```csharp

 class C<T> where T : new(), IDisposable {}  // CS0401
```
