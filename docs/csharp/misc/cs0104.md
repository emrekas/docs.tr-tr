# <a name="compiler-error-cs0104"></a><span data-ttu-id="96949-102">Derleyici hatası CS0104</span><span class="sxs-lookup"><span data-stu-id="96949-102">Compiler Error CS0104</span></span>
<span data-ttu-id="96949-103">' Reference ', ' Identifier ' ve ' Identifier ' arasında belirsiz bir başvuru</span><span class="sxs-lookup"><span data-stu-id="96949-103">'reference' is an ambiguous reference between 'identifier' and 'identifier'</span></span>  
  
 <span data-ttu-id="96949-104">Programınız iki ad alanı için [kullanım](../language-reference/keywords/using.md) yönergeleri içerir ve kodunuz her iki ad alanında da görüntülenen bir ada başvurur.</span><span class="sxs-lookup"><span data-stu-id="96949-104">Your program contains [using](../language-reference/keywords/using.md) directives for two namespaces and your code references a name that appears in both namespaces.</span></span>  
  
 <span data-ttu-id="96949-105">Aşağıdaki örnek CS0104 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="96949-105">The following sample generates CS0104:</span></span>  
  
```csharp  
// CS0104.cs  
using x;  
using y;  
  
namespace x  
{  
   public class Test  
   {  
   }  
}  
  
namespace y  
{  
   public class Test  
   {  
   }  
}  
  
public class a  
{  
   public static void Main()  
   {  
      Test test = new Test();   // CS0104, is Test in x or y namespace?  
      // try the following line instead  
      // y.Test test = new y.Test();  
   }  
}  
```
