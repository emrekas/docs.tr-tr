# <a name="compiler-error-cs0104"></a><span data-ttu-id="43fdf-102">Derleyici Hatası CS0104</span><span class="sxs-lookup"><span data-stu-id="43fdf-102">Compiler Error CS0104</span></span>
<span data-ttu-id="43fdf-103">'identifier' ve 'identifier' arasında belirsiz bir başvuru 'reference' olan</span><span class="sxs-lookup"><span data-stu-id="43fdf-103">'reference' is an ambiguous reference between 'identifier' and 'identifier'</span></span>  
  
 <span data-ttu-id="43fdf-104">Programınızı içeren [kullanarak](../../csharp/language-reference/keywords/using.md) iki ad alanı ve kodunuzu yönergeleri her iki ad alanlarında görünen bir ad başvuruyor.</span><span class="sxs-lookup"><span data-stu-id="43fdf-104">Your program contains [using](../../csharp/language-reference/keywords/using.md) directives for two namespaces and your code references a name that appears in both namespaces.</span></span>  
  
 <span data-ttu-id="43fdf-105">Aşağıdaki örnek, CS0104 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="43fdf-105">The following sample generates CS0104:</span></span>  
  
```csharp  
// CS0104.cs  
using x;  
using y;  
  
namespace x  
{  
   public class Test  
   {  
   }  
}  
  
namespace y  
{  
   public class Test  
   {  
   }  
}  
  
public class a  
{  
   public static void Main()  
   {  
      Test test = new Test();   // CS0104, is Test in x or y namespace?  
      // try the following line instead  
      // y.Test test = new y.Test();  
   }  
}  
```
