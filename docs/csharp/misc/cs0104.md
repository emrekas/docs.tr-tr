# <a name="compiler-error-cs0104"></a>Derleyici Hatası CS0104
'identifier' ve 'identifier' arasında belirsiz bir başvuru 'reference' olan  
  
 Programınızı içeren [kullanarak](../../csharp/language-reference/keywords/using.md) iki ad alanı ve kodunuzu yönergeleri her iki ad alanlarında görünen bir ad başvuruyor.  
  
 Aşağıdaki örnek, CS0104 oluşturur:  
  
```csharp  
// CS0104.cs  
using x;  
using y;  
  
namespace x  
{  
   public class Test  
   {  
   }  
}  
  
namespace y  
{  
   public class Test  
   {  
   }  
}  
  
public class a  
{  
   public static void Main()  
   {  
      Test test = new Test();   // CS0104, is Test in x or y namespace?  
      // try the following line instead  
      // y.Test test = new y.Test();  
   }  
}  
```
