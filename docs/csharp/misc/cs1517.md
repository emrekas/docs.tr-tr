# <a name="compiler-error-cs1517"></a>Derleyici Hatası CS1517
Geçersiz önişlemci ifadesi  
  
 Derleyici, bir Geçersiz önişlemci ifadesi karşılaşıldı.  
  
 Daha fazla bilgi için [önişlemci yönergeleri](../../csharp/language-reference/preprocessor-directives/index.md).  
  
 Aşağıdaki örnek, bazı geçerli ve Geçersiz önişlemci ifadeleri gösterir:  
  
```csharp  
// CS1517.cs  
#if symbol      // OK  
#endif  
#if !symbol     // OK  
#endif  
#if (symbol)    // OK  
#endif  
#if true        // OK  
#endif  
#if false       // OK  
#endif  
#if 1           // CS1517  
#endif  
#if ~symbol     // CS1517  
#endif  
#if *           // CS1517  
#endif  
  
class x  
{  
   public static void Main()  
   {  
   }  
}  
```
