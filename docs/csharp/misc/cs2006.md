# <a name="compiler-error-cs2006"></a>Derleyici hatası CS2006
Komut satırı sözdizimi hatası: ' Option ' seçeneği için ' text ' eksik  
  
 *Seçenek* için sözdizimi ek metin gerektirir. Bilgi için bkz. [derleyici seçenekleri](../language-reference/compiler-options/index.md).
