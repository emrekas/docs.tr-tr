# <a name="compiler-error-cs0558"></a><span data-ttu-id="d553e-102">Derleyici Hatası CS0558</span><span class="sxs-lookup"><span data-stu-id="d553e-102">Compiler Error CS0558</span></span>
<span data-ttu-id="d553e-103">Kullanıcı tanımlı işleç 'operator' static ve public olarak bildirilmelidir</span><span class="sxs-lookup"><span data-stu-id="d553e-103">User-defined operator 'operator' must be declared static and public</span></span>  
  
 <span data-ttu-id="d553e-104">Her iki **statik** ve **genel** erişim [değiştiriciler](../../csharp/language-reference/keywords/modifiers.md) kullanıcı tanımlı işleçlerde belirtilmesi gerekir.</span><span class="sxs-lookup"><span data-stu-id="d553e-104">Both the **static** and **public** access [modifiers](../../csharp/language-reference/keywords/modifiers.md) must be specified on user-defined operators.</span></span>  
  
 <span data-ttu-id="d553e-105">Aşağıdaki örnek, CS0558 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="d553e-105">The following sample generates CS0558:</span></span>  
  
```csharp  
// CS0558.cs  
namespace x  
{  
   public class ii  
   {  
      public class iii  
      {  
         static implicit operator int(iii aa)   // CS0558, add public  
         {  
            return 0;  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
