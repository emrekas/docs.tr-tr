 <span data-ttu-id="c7547-109">Aşağıdaki örnek, CS0672 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="c7547-109">The following sample generates CS0672:</span></span>

```csharp  
{  
   [System.Obsolete]  
   public virtual void ObsoleteMethod()  
   {  
   }  
}

{  
   public override void ObsoleteMethod()   // CS0672

   }

{  
   static public void Main()  
   {  
   }  
}  
```
