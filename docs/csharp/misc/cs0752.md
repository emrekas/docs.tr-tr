# <a name="compiler-error-cs0752"></a><span data-ttu-id="f733f-102">Derleyici Hatası CS0752</span><span class="sxs-lookup"><span data-stu-id="f733f-102">Compiler Error CS0752</span></span>

 <span data-ttu-id="f733f-104">Kısmi bir yöntemin olamaz bir [kullanıma](../../csharp/language-reference/keywords/out-parameter-modifier.md) parametresi.</span><span class="sxs-lookup"><span data-stu-id="f733f-104">A partial method cannot have an [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parameter.</span></span> <span data-ttu-id="f733f-105">Out Parametreleri olduğundan kısmi yöntem derleyici tarafından kaldırıldıktan sonra out parametresi hiç atanan bir garanti yoktur izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="f733f-105">Out parameters are not allowed because if the partial method is removed by the compiler then there is no guarantee that the out parameter is ever assigned.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="f733f-106">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="f733f-106">To correct this error</span></span>
