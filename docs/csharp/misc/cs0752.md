# <a name="compiler-error-cs0752"></a><span data-ttu-id="c3af1-102">Derleyici hatası CS0752</span><span class="sxs-lookup"><span data-stu-id="c3af1-102">Compiler Error CS0752</span></span>
<span data-ttu-id="c3af1-103">Kısmi bir yöntemin out parametreleri olamaz</span><span class="sxs-lookup"><span data-stu-id="c3af1-103">A partial method cannot have out parameters</span></span>  
  
 <span data-ttu-id="c3af1-104">Kısmi bir yöntemin [Out](../language-reference/keywords/out-parameter-modifier.md) parametresi olamaz.</span><span class="sxs-lookup"><span data-stu-id="c3af1-104">A partial method cannot have an [out](../language-reference/keywords/out-parameter-modifier.md) parameter.</span></span> <span data-ttu-id="c3af1-105">Kısmi yöntem derleyici tarafından kaldırılacağından Out parametrelerine izin verilmez, aksi takdirde Out parametresinin atandığı garanti yoktur.</span><span class="sxs-lookup"><span data-stu-id="c3af1-105">Out parameters are not allowed because if the partial method is removed by the compiler then there is no guarantee that the out parameter is ever assigned.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="c3af1-106">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="c3af1-106">To correct this error</span></span>  
  
1. <span data-ttu-id="c3af1-107">Parametresinden Out değiştiricisini kaldırın ve bunun yerine yöntemin dönüş değerini kullanın veya yöntem bildiriminden kısmi değiştiricisini kaldırın.</span><span class="sxs-lookup"><span data-stu-id="c3af1-107">Remove the out modifier from the parameter and use the return value of the method instead, or else remove the partial modifier from the method declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="c3af1-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="c3af1-108">Example</span></span>  
 <span data-ttu-id="c3af1-109">Aşağıdaki kod CS0752 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="c3af1-109">The following code generates CS0752:</span></span>

    partial void Part(out int num); // CS0752

    }
