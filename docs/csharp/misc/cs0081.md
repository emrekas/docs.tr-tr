# <a name="compiler-error-cs0081"></a><span data-ttu-id="65f5e-102">Derleyici Hatası CS0081</span><span class="sxs-lookup"><span data-stu-id="65f5e-102">Compiler Error CS0081</span></span>
<span data-ttu-id="65f5e-103">Tür parametresi bildirimi bir tür değil bir tanımlayıcı olmalıdır</span><span class="sxs-lookup"><span data-stu-id="65f5e-103">Type parameter declaration must be an identifier not a type</span></span>  
  
 <span data-ttu-id="65f5e-104">Bir genel yöntem veya tür bildirdiğinizde, tür parametresi örneğin "T" veya "Inputtype" bir tanımlayıcı belirtin.</span><span class="sxs-lookup"><span data-stu-id="65f5e-104">When you declare a generic method or type, specify the type parameter as an identifier, for example "T" or "inputType".</span></span> <span data-ttu-id="65f5e-105">İstemci kodu yöntemini çağırdığında, metot veya sınıflardaki gövdesinde tanımlayıcının her örneği değiştirir türü sağlar.</span><span class="sxs-lookup"><span data-stu-id="65f5e-105">When client code calls the method, it supplies the type, which replaces each occurrence of the identifier in the method or class body.</span></span> <span data-ttu-id="65f5e-106">Daha fazla bilgi için [genel tür parametreleri](../../csharp/programming-guide/generics/generic-type-parameters.md).</span><span class="sxs-lookup"><span data-stu-id="65f5e-106">For more information, see [Generic Type Parameters](../../csharp/programming-guide/generics/generic-type-parameters.md).</span></span>  
  
```csharp  
// CS0081.cs  
class MyClass  
{  
   public void F<int>() {}   // CS0081  
   public void F<T>(T input) {}   // OK  
  
   public static void Main()  
   {  
      MyClass a = new MyClass();  
      a.F<int>(2);  
      a.F<double>(.05);  
   }  
}  
```
