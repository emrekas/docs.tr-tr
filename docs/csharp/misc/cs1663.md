# <a name="compiler-error-cs1663"></a>Derleyici Hatası CS1663
Sabit boyutlu arabellek türü aşağıdakilerden biri olmalıdır: bool, byte, short, int, long, char, sbyte, ushort, uint, ulong, float veya double  
  
 Sabit boyutlu arabellek bu listelenen dışındaki herhangi bir türü olamaz. Bu hatayı önlemek için başka bir tür kullanın veya sabit bir dizi kullanmayın.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS1663 oluşturur.  
  
```csharp  
// CS1663.cs  
// compile with: /unsafe /target:library  
  
unsafe struct C  
{  
   fixed string ab[10];   // CS1663  
}  
```
