# <a name="compiler-error-cs1100"></a><span data-ttu-id="94be5-102">Derleyici Hatası CS1100</span><span class="sxs-lookup"><span data-stu-id="94be5-102">Compiler Error CS1100</span></span>

 <span data-ttu-id="94be5-104">`this` Değiştiricisi, derleyicinin yöntemin bir genişletme yöntemi olduğunu belirten bir yöntem yalnızca ilk parametresinin verilir.</span><span class="sxs-lookup"><span data-stu-id="94be5-104">The `this` modifier is allowed only on the first parameter of a method, which indicates to the compiler that the method is an extension method.</span></span>
