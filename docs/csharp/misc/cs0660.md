# <a name="compiler-warning-level-3-cs0660"></a><span data-ttu-id="5506a-102">Derleyici Uyarısı (Düzey 3) CS0660</span><span class="sxs-lookup"><span data-stu-id="5506a-102">Compiler Warning (level 3) CS0660</span></span>

<span data-ttu-id="5506a-103">'class' tanımlar işleç == veya işleç! = ancak Object.Equals (nesne o) geçersiz kılmaz</span><span class="sxs-lookup"><span data-stu-id="5506a-103">'class' defines operator == or operator != but does not override Object.Equals(object o)</span></span>  
  
<span data-ttu-id="5506a-104">Derleyici kullanıcı tanımlı eşitlik ya da eşitsizlik işleci, ancak geçersiz kılma için algılanan <xref:System.Object.Equals%2A?displayProperty=nameWithType> yöntemi.</span><span class="sxs-lookup"><span data-stu-id="5506a-104">The compiler detected the user-defined equality or inequality operator, but no override for the <xref:System.Object.Equals%2A?displayProperty=nameWithType> method.</span></span> <span data-ttu-id="5506a-105">Kullanıcı tanımlı bir eşitlik ve eşitsizlik işleci ayrıca geçersiz kılmak istediğiniz gelir <xref:System.Object.Equals%2A> yöntemi.</span><span class="sxs-lookup"><span data-stu-id="5506a-105">A user-defined equality or inequality operator implies that you also want to override the <xref:System.Object.Equals%2A> method.</span></span> <span data-ttu-id="5506a-106">Daha fazla bilgi için [nasıl yapılır: Tür için değer eşitliği tanımlama](../programming-guide/statements-expressions-operators/how-to-define-value-equality-for-a-type.md).</span><span class="sxs-lookup"><span data-stu-id="5506a-106">For more information, see [How to: Define Value Equality for a Type](../programming-guide/statements-expressions-operators/how-to-define-value-equality-for-a-type.md).</span></span>
  
<span data-ttu-id="5506a-107">Aşağıdaki örnek, CS0660 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="5506a-107">The following sample generates CS0660:</span></span>  

```csharp
// CS0660.cs  
// compile with: /W:3 /warnaserror  
class Test   // CS0660  
{  
   public static bool operator == (object o, Test t)  
   {  
      return true;  
   }  
  
   // uncomment the Equals function to resolve  
   // public override bool Equals(object o)  
   // {  
   //    return true;  
   // }  
  
   public override int GetHashCode()  
   {  
      return 0;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
