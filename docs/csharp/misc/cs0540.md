# <a name="compiler-error-cs0540"></a>Derleyici Hatası CS0540
'arabirim üyesini': içeren tür 'interface' arabirimini uygulamıyor  
  
 Bir arabirim üyesi içindeki uygulamak çalıştığınız bir [sınıfı](../../csharp/language-reference/keywords/class.md) , türemiyor gelen [arabirimi](../../csharp/language-reference/keywords/interface.md). Arabirim üyesini uygulamasını silin veya arabirimi sınıfın temel sınıf listesine ekleyin.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS0540 oluşturur.  
  
```csharp  
// CS0540.cs  
interface I  
{  
   void m();  
}  
  
public class Clx  
{  
   void I.m() {}   // CS0540  
}  
  
// OK  
public class Cly : I  
{  
   void I.m() {}  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS0540 oluşturur.  
  
```csharp  
// CS0540_b.cs  
using System;  
class C {  
   void IDisposable.Dispose() {}   // CS0540  
}  
  
class D : IDisposable {  
   void IDisposable.Dispose() {}  
   public void Dispose() {}  
  
   static void Main() {  
      using (D d = new D()) {}  
   }  
}  
```
