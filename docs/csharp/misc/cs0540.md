# <a name="compiler-error-cs0540"></a><span data-ttu-id="731d7-102">Derleyici hatası CS0540</span><span class="sxs-lookup"><span data-stu-id="731d7-102">Compiler Error CS0540</span></span>
<span data-ttu-id="731d7-103">' arabirim üyesi ': içeren tür ' Interface ' arabirimini uygulamıyor</span><span class="sxs-lookup"><span data-stu-id="731d7-103">'interface member' : containing type does not implement interface 'interface'</span></span>  
  
 <span data-ttu-id="731d7-104">Arabirimden türemeyen bir [sınıfta](../language-reference/keywords/class.md) bir arabirim üyesini uygulamaya çalıştınız. ## <a name="example"></a><span data-ttu-id="731d7-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="731d7-106">Example</span></span>  
 <span data-ttu-id="731d7-107">Aşağıdaki örnek CS0540 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="731d7-107">The following sample generates CS0540.</span></span>  
  
```csharp  
// CS0540.cs  
interface I  
{  
   void m();  
}  
  
public class Clx  
{  
   void I.m() {}   // CS0540  
}  
  
// OK  
public class Cly : I  
{  
   void I.m() {}  
   public static void Main() {}  
}  
```
