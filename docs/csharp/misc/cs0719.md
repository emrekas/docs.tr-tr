# <a name="compiler-error-cs0719"></a><span data-ttu-id="00bbe-102">Derleyici Hatası CS0719</span><span class="sxs-lookup"><span data-stu-id="00bbe-102">Compiler Error CS0719</span></span>
<span data-ttu-id="00bbe-103">'type': dizi öğeleri statik türünde olamaz</span><span class="sxs-lookup"><span data-stu-id="00bbe-103">'type': array elements cannot be of static type</span></span>  
  
 <span data-ttu-id="00bbe-104">Statik tip bir dizide dizi öğeleri örneklerdir, ancak statik türlerin örneklerini oluşturmak mümkün değil olduğundan mantıklı değildir.</span><span class="sxs-lookup"><span data-stu-id="00bbe-104">An array of static type does not make sense since array elements are instances, but it is not possible to create instances of static types.</span></span>  
  
 <span data-ttu-id="00bbe-105">Aşağıdaki örnek, CS0719 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="00bbe-105">The following sample generates CS0719:</span></span>  
  
```csharp  
// CS0719.cs  
public static class SC  
{  
   public static void F()  
   {  
   }  
}  
  
public class CMain  
{  
   public static void Main()  
   {  
      SC[] sca = new SC[10];  // CS0719  
   }  
}  
```
