 <span data-ttu-id="7b824-104">Statik tür, parametre olarak anlamı yoktur.</span><span class="sxs-lookup"><span data-stu-id="7b824-104">A static type is not meaningful as a parameter.</span></span> <span data-ttu-id="7b824-105">Statik türler örneği oluşturulabilir olduğundan, hiçbir örneği hiç parametre olarak geçirilebilir.</span><span class="sxs-lookup"><span data-stu-id="7b824-105">Since no instances of static types may be created, no instance could ever be passed as a parameter.</span></span>

{  
}
