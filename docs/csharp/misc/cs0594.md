# <a name="compiler-error-cs0594"></a>Derleyici Hatası CS0594
Kayan noktalı sabit 'type' türü aralığının dışında olduğundan  
  
 Bu veri türündeki değişkenler için çok büyük bir kayan nokta değişkeni değeri atandı. Bkz: [tam sayı türleri tablosu](../../csharp/language-reference/keywords/integral-types-table.md) veri türlerinde izin verilen değer aralığı hakkında bilgi için.  
  
 Aşağıdaki örnek, CS0594 oluşturur:  
  
```csharp  
// CS0594.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         float f = 6.77777777777E400;   // CS0594, value too large  
      }  
   }  
}  
```
