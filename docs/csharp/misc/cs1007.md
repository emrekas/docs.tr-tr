# <a name="compiler-error-cs1007"></a><span data-ttu-id="d9fa2-102">Derleyici Hatası CS1007</span><span class="sxs-lookup"><span data-stu-id="d9fa2-102">Compiler Error CS1007</span></span>
<span data-ttu-id="d9fa2-103">Özellik erişimcisi zaten tanımlı</span><span class="sxs-lookup"><span data-stu-id="d9fa2-103">Property accessor already defined</span></span>  
  
 <span data-ttu-id="d9fa2-104">Bildirdiğinizde bir [özelliği](../../csharp/programming-guide/classes-and-structs/using-properties.md), erişimci metotlarını de belirtmesi gerekir.</span><span class="sxs-lookup"><span data-stu-id="d9fa2-104">When you declare a [property](../../csharp/programming-guide/classes-and-structs/using-properties.md), you must also declare its accessor methods.</span></span> <span data-ttu-id="d9fa2-105">Ancak, bir özelliği birden fazla olamaz `get` erişimci yöntemini veya birden fazla `set` erişimci yöntemi.</span><span class="sxs-lookup"><span data-stu-id="d9fa2-105">However, a property cannot have more than one `get` accessor method or more than one `set` accessor method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="d9fa2-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="d9fa2-106">Example</span></span>  
 <span data-ttu-id="d9fa2-107">Aşağıdaki örnek, CS1007 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="d9fa2-107">The following sample generates CS1007:</span></span>  
  
```csharp  
// CS1007.cs  
public class clx  
{  
    public int MyProperty  
    {  
        get  
        {  
            return 0;  
        }  
        get   // CS1007, this is the second get method  
        {  
            return 0;  
        }  
    }  
  
    public static void Main() {}  
}  
```
