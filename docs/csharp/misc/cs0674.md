# <a name="compiler-error-cs0674"></a>Derleyici Hatası CS0674
'System.ParamArrayAttribute' kullanmayın. Bunun yerine 'params' anahtar sözcüğünü kullanın.  
  
 C# Derleyici kullanımı için izin verme <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; kullanma [params](../../csharp/language-reference/keywords/params.md) yerine.  
  
 Aşağıdaki örnek, CS0674 oluşturur:  
  
```csharp  
// CS0674.cs  
using System;  
public class MyClass   
{  
  
   public static void UseParams([ParamArray] int[] list)   // CS0674  
   // try the following line instead  
   // public static void UseParams(params int[] list)   
   {  
      for ( int i = 0 ; i < list.Length ; i++ )  
         Console.WriteLine(list[i]);  
      Console.WriteLine();  
   }  
  
   public static void Main()   
   {  
      UseParams(1, 2, 3);  
   }  
}  
```
