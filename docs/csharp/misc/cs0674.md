# <a name="compiler-error-cs0674"></a><span data-ttu-id="943b0-102">Derleyici Hatası CS0674</span><span class="sxs-lookup"><span data-stu-id="943b0-102">Compiler Error CS0674</span></span>
<span data-ttu-id="943b0-103">'System.ParamArrayAttribute' kullanmayın.</span><span class="sxs-lookup"><span data-stu-id="943b0-103">Do not use 'System.ParamArrayAttribute'.</span></span> <span data-ttu-id="943b0-104">Bunun yerine 'params' anahtar sözcüğünü kullanın.</span><span class="sxs-lookup"><span data-stu-id="943b0-104">Use the 'params' keyword instead.</span></span>  
  
 <span data-ttu-id="943b0-105">C# Derleyici kullanımı için izin verme <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; kullanma [params](../../csharp/language-reference/keywords/params.md) yerine.</span><span class="sxs-lookup"><span data-stu-id="943b0-105">The C# compiler does not allow for the use of <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; use [params](../../csharp/language-reference/keywords/params.md) instead.</span></span>  
  
 <span data-ttu-id="943b0-106">Aşağıdaki örnek, CS0674 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="943b0-106">The following sample generates CS0674:</span></span>  
  
```csharp  
// CS0674.cs  
using System;  
public class MyClass   
{  
  
   public static void UseParams([ParamArray] int[] list)   // CS0674  
   // try the following line instead  
   // public static void UseParams(params int[] list)   
   {  
      for ( int i = 0 ; i < list.Length ; i++ )  
         Console.WriteLine(list[i]);  
      Console.WriteLine();  
   }  
  
   public static void Main()   
   {  
      UseParams(1, 2, 3);  
   }  
}  
```
