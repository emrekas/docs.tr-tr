# <a name="compiler-error-cs0008"></a>Derleyici Hatası CS0008
'Dosya' dosyasından meta veriler okunurken beklenmeyen hata: 'description'  
  
 Bir DLL meta verileri almak için başarıyla açıldı, ancak bozuk sağlayacak şekilde verileri CİHAZDAN okunamadı. Daha fazla bilgi için [/Reference (C# Derleyici Seçenekleri)](../../csharp/language-reference/compiler-options/reference-compiler-option.md).
