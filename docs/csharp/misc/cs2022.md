# <a name="compiler-error-cs2022"></a>Derleyici hatası CS2022
'/Out ' ve '/Target ' seçenekleri kaynak dosya adlarından önce gelmelidir  
  
 Komut satırında belirtildiğinde [/Out (çıktı dosya adını ayarla)](../language-reference/compiler-options/out-compiler-option.md) ve [/target (çıktı dosyası biçimi belirt)](../language-reference/compiler-options/target-compiler-option.md) derleyici seçeneklerini kaynak kodu dosyalarından önce gelmelidir.
