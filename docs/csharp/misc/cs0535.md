# <a name="compiler-error-cs0535"></a>Derleyici Hatası CS0535
'class', 'member' arabirim üyesini uygulamıyor.  
  
 A [sınıfı](../../csharp/language-reference/keywords/class.md) türetilen bir [arabirimi](../../csharp/language-reference/keywords/interface.md), ancak bir veya daha fazla arabirimin üyeleri sınıf uygulamadı. Bir sınıf içinden, türetilen; Aksi takdirde bildirilmesi arabirimin tüm üyelerini uygulamalıdır `abstract`.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS0535 oluşturur.  
  
```csharp  
// CS0535.cs  
public interface A  
{  
   void F();  
}  
  
public class B : A {}   // CS0535 A::F is not implemented  
  
// OK  
public class C : A {  
   public void F() {}  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS0535 oluşturur.  
  
```csharp  
// CS0535_b.cs  
using System;  
class C : IDisposable {}   // CS0535  
  
// OK  
class D : IDisposable {  
   void IDisposable.Dispose() {}  
   public void Dispose() {}  
  
   static void Main() {  
      using (D d = new D()) {}  
   }  
}  
```
