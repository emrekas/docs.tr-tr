 <span data-ttu-id="c4ba3-104">A [sınıfı](../../csharp/language-reference/keywords/class.md) türetilen bir [arabirimi](../../csharp/language-reference/keywords/interface.md), ancak bir veya daha fazla arabirimin üyeleri sınıf uygulamadı.</span><span class="sxs-lookup"><span data-stu-id="c4ba3-104">A [class](../../csharp/language-reference/keywords/class.md) derived from an [interface](../../csharp/language-reference/keywords/interface.md), but the class did not implement one or more of the interface's members.</span></span> <span data-ttu-id="c4ba3-105">Bir sınıf içinden, türetilen; Aksi takdirde bildirilmesi arabirimin tüm üyelerini uygulamalıdır `abstract`.</span><span class="sxs-lookup"><span data-stu-id="c4ba3-105">A class must implement all members of interfaces from which it derives or else be declared `abstract`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="c4ba3-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="c4ba3-106">Example</span></span>  
 <span data-ttu-id="c4ba3-107">Aşağıdaki örnek, CS0535 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="c4ba3-107">The following sample generates CS0535.</span></span>  
  
```csharp  
// CS0535.cs  
public interface A  
{  
   void F();  
}  
  
public class B : A {}   // CS0535 A::F is not implemented  
  
// OK  
public class C : A {  
   public void F() {}  
   public static void Main() {}  
}  
```
