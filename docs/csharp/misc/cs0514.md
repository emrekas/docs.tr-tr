## <a name="example"></a><span data-ttu-id="30e8f-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="30e8f-107">Example</span></span>  
 <span data-ttu-id="30e8f-108">Aşağıdaki örnek, CS0514 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="30e8f-108">The following example generates CS0514:</span></span>  
  
```csharp

    {  
    }

    }  
  
    public B(object o)

    }  
}  
```
