# <a name="compiler-error-cs1055"></a>Derleyici Hatası CS1055
Ekleme veya kaldırma erişimcisi bekleniyor  
  
 Varsa, [olay](../../csharp/language-reference/keywords/event.md) bildirilmedi bir alan olarak bunu tanımlamalıdır hem **ekleme** ve **Kaldır** erişimci işlevleri.  
  
 Aşağıdaki örnek, CS1055 oluşturur:  
  
```csharp  
// CS1055.cs  
delegate void del();  
class Test  
{  
   public event del MyEvent  
   {  
      int i;   // CS1055  
      // uncomment accessors and delete previous line to resolve  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      // remove  
      // {  
      //    MyEvent -= value;  
      // }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
