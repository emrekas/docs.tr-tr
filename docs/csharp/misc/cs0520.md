# <a name="compiler-error-cs0520"></a>Derleyici Hatası CS0520
'Name' önceden tanımlanmış türü hatalı bildirilmiş  
  
 Derleyici tarafından gereken bir veya daha fazla dosya bulunamadı. .NET Framework ortak dil çalışma zamanı yeniden yükleyin.
