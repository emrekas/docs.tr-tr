# <a name="compiler-error-cs1578"></a>Derleyici Hatası CS1578
Dosya adı, tek satırlık açıklama veya satır sonu bekleniyor  
  
 Sonra bir [#line](../../csharp/language-reference/preprocessor-directives/preprocessor-line.md) yönergesi, yalnızca dosya adında (çift tırnak işareti) veya bir tek satır açıklama izin verilmiyor.  
  
 Aşağıdaki örnek, CS1578 oluşturur:  
  
```csharp  
// CS1578.cs  
class MyClass  
{  
   static void Main()  
   {  
      #line 101 abc.cs   // CS1578  
      // try the following line instead  
      //#line 101 "abc.cs"  
      intt i;          // error will be reported on line 101  
   }  
}  
```
