 <span data-ttu-id="04e21-104">Sonra bir [#line](../../csharp/language-reference/preprocessor-directives/preprocessor-line.md) yönergesi, yalnızca dosya adında (çift tırnak işareti) veya bir tek satır açıklama izin verilmiyor.</span><span class="sxs-lookup"><span data-stu-id="04e21-104">After a [#line](../../csharp/language-reference/preprocessor-directives/preprocessor-line.md) directive, only a file name (in double quotation marks) or a single-line comment is allowed.</span></span>  
  
 <span data-ttu-id="04e21-105">Aşağıdaki örnek, CS1578 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="04e21-105">The following sample generates CS1578:</span></span>  
  
```csharp

   static void Main()  
   {  
      #line 101 abc.cs   // CS1578

   }  
}  
```
