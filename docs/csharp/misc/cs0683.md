# <a name="compiler-error-cs0683"></a><span data-ttu-id="71bc8-102">Derleyici Hatası CS0683</span><span class="sxs-lookup"><span data-stu-id="71bc8-102">Compiler Error CS0683</span></span>
<span data-ttu-id="71bc8-103">'explicitmethod' açık yöntem uygulaması, bir erişimci olduğundan 'method' uygulayamaz</span><span class="sxs-lookup"><span data-stu-id="71bc8-103">'explicitmethod' explicit method implementation cannot implement 'method' because it is an accessor</span></span>  
  
 <span data-ttu-id="71bc8-104">Aşağıdaki örnek, CS0683 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="71bc8-104">The following sample generates CS0683:</span></span>  
  
```csharp  
// CS0683.cs

{  
   int Test { get; }  
}  
  
class CExample : IExample  
{  
   int IExample.get_Test() { return 0; } // CS0683  
   int IExample.Test { get{ return 0; } } // correct  
}  
```
