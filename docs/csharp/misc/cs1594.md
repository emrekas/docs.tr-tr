# <a name="compiler-error-cs1594"></a>Derleyici Hatası CS1594
Temsilci 'temsilci' bazı geçersiz bağımsız değişkenlere sahip  
  
 Geçirilen bir bağımsız değişken türünü bir [temsilci](../../csharp/language-reference/keywords/delegate.md) çağırma temsilci bildirimi parametresinde türüyle kabul değil.  
  
 Aşağıdaki örnek, CS1594 oluşturur:  
  
```csharp  
// CS1594.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello("8");   // CS1594  
      // try the following line instead  
      // hello(8);  
   }  
}  
```
