# <a name="compiler-error-cs1594"></a><span data-ttu-id="01261-102">Derleyici hatası CS1594</span><span class="sxs-lookup"><span data-stu-id="01261-102">Compiler Error CS1594</span></span>
<span data-ttu-id="01261-103">' Delegate ' temsilcisinin bazı geçersiz bağımsız değişkenleri var</span><span class="sxs-lookup"><span data-stu-id="01261-103">Delegate 'delegate' has some invalid arguments</span></span>  
  
 <span data-ttu-id="01261-104">Bir [temsilci](../language-reference/keywords/delegate.md) çağrısına geçirilen bir bağımsız değişkenin türü, temsilci bildirimindeki parametrenin türünü kabul etmez.</span><span class="sxs-lookup"><span data-stu-id="01261-104">The type of an argument passed to a [delegate](../language-reference/keywords/delegate.md) invocation does not agree with the type of the parameter in the delegate declaration.</span></span>  
  
 <span data-ttu-id="01261-105">Aşağıdaki örnek CS1594 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="01261-105">The following sample generates CS1594:</span></span>  
  
```csharp  
// CS1594.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello("8");   // CS1594  
      // try the following line instead  
      // hello(8);  
   }  
}  
```
