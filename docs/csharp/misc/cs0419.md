# <a name="compiler-warning-level-3-cs0419"></a><span data-ttu-id="b5a5c-102">Derleyici Uyarısı (Düzey 3) CS0419</span><span class="sxs-lookup"><span data-stu-id="b5a5c-102">Compiler Warning (level 3) CS0419</span></span>
<span data-ttu-id="b5a5c-103">Cref özniteliğinde belirsiz başvuru: 'Yöntemi Name1'.</span><span class="sxs-lookup"><span data-stu-id="b5a5c-103">Ambiguous reference in cref attribute: 'Method Name1'.</span></span>  <span data-ttu-id="b5a5c-104">'Yöntemi Name2' varsayılıyor, ancak 'Yöntemi Name3' dahil diğer aşırı yüklerle de eşleşebilirdi.</span><span class="sxs-lookup"><span data-stu-id="b5a5c-104">Assuming 'Method Name2', but could have also matched other overloads including 'Method Name3'.</span></span>  
  
 <span data-ttu-id="b5a5c-105">Kod'da bir XML belgesi açıklamasında bir başvuru çözümlenemedi.</span><span class="sxs-lookup"><span data-stu-id="b5a5c-105">In an XML documentation comment in the code, a reference could not be resolved.</span></span> <span data-ttu-id="b5a5c-106">Bu yöntem aşırı yüklüyse ya da aynı ada sahip iki farklı tanımlayıcılar bulunursa ortaya çıkabilir.</span><span class="sxs-lookup"><span data-stu-id="b5a5c-106">This could occur if the method is overloaded, or if two different identifiers with the same name are found.</span></span> <span data-ttu-id="b5a5c-107">Uyarıyı çözmek için başvuru ayırt etmek için bir tam adı kullanın veya parantez içinde belirli bir aşırı yükleme içerir.</span><span class="sxs-lookup"><span data-stu-id="b5a5c-107">To resolve the warning, use a qualified name to disambiguate the reference, or include the specific overload in parentheses.</span></span>  
  
 <span data-ttu-id="b5a5c-108">Aşağıdaki örnek, CS0419 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="b5a5c-108">The following sample generates CS0419.</span></span>  
  
```csharp  
// cs0419.cs  
// compile with: /doc:x.xml /W:3  
interface I  
{  
   /// text for F(void)  
   void F();  
   /// text for F(int)  
   void F(int i);  
}  
/// text for class MyClass  
public class MyClass  
{  
   /// <see cref="I.F"/>  
   public static void MyMethod(int i)  
   {  
   }  
/* Try this instead:  
   /// <see cref="I.F(int)"/>  
   public static void MyMethod(int i)  
   {  
   }  
*/  
   /// text for Main  
   public static void Main ()  
   {  
   }  
}  
```
