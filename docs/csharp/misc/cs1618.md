# <a name="compiler-error-cs1618"></a><span data-ttu-id="8b181-102">Derleyici Hatası CS1618</span><span class="sxs-lookup"><span data-stu-id="8b181-102">Compiler Error CS1618</span></span>
<span data-ttu-id="8b181-103">Conditional özniteliği olduğundan 'method' ile temsilci oluşturulamıyor</span><span class="sxs-lookup"><span data-stu-id="8b181-103">Cannot create delegate with 'method' because it has a Conditional attribute</span></span>  
  
 <span data-ttu-id="8b181-104">Yöntemi bazı derlemeler var olmayabilir çünkü koşullu bir yöntemle bir temsilci oluşturulamıyor.</span><span class="sxs-lookup"><span data-stu-id="8b181-104">You cannot create a delegate with a conditional method because the method might not exist in some builds.</span></span>  
  
 <span data-ttu-id="8b181-105">Aşağıdaki örnek, CS1618 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="8b181-105">The following sample generates CS1618:</span></span>  
  
```csharp  
// CS1618.cs  
using System;  
using System.Diagnostics;  
  
delegate void del();  
  
class MakeAnError {  
   public static void Main() {  
      del d = new del(ConditionalMethod);   // CS1618  
      // Invalid because on builds where DEBUG is not set,   
      // there will be no "ConditionalMethod".  
   }  
   // To fix the error, remove the next line:  
   [Conditional("DEBUG")]  
   public static void ConditionalMethod()   
   {  
      Console.WriteLine("Do something only in debug");  
   }  
}  
```
