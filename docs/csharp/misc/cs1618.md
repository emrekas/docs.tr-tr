# <a name="compiler-error-cs1618"></a>Derleyici Hatası CS1618
Conditional özniteliği olduğundan 'method' ile temsilci oluşturulamıyor  
  
 Yöntemi bazı derlemeler var olmayabilir çünkü koşullu bir yöntemle bir temsilci oluşturulamıyor.  
  
 Aşağıdaki örnek, CS1618 oluşturur:  
  
```csharp  
// CS1618.cs  
using System;  
using System.Diagnostics;  
  
delegate void del();  
  
class MakeAnError {  
   public static void Main() {  
      del d = new del(ConditionalMethod);   // CS1618  
      // Invalid because on builds where DEBUG is not set,   
      // there will be no "ConditionalMethod".  
   }  
   // To fix the error, remove the next line:  
   [Conditional("DEBUG")]  
   public static void ConditionalMethod()   
   {  
      Console.WriteLine("Do something only in debug");  
   }  
}  
```
