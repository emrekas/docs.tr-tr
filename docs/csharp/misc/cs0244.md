# <a name="compiler-error-cs0244"></a>Derleyici Hatası CS0244
'İs' ve 'as' işaretçi türlerinde geçerli değildir  
  
 [Olduğu](../../csharp/language-reference/operators/type-testing-and-conversion-operators.md#is-operator) ve [olarak](../../csharp/language-reference/operators/type-testing-and-conversion-operators.md#as-operator) işleçleri işaretçi türleri üzerinde kullanım için geçerli değildir. Daha fazla bilgi için [güvenli olmayan kod ve işaretçiler](../../csharp/programming-guide/unsafe-code-pointers/index.md).  
  
 Aşağıdaki örnek, CS0244 oluşturur:  
  
```csharp  
// CS0244.cs  
// compile with: /unsafe  
  
class UnsafeTest  
{  
   unsafe static void SquarePtrParam (int* p)  
   {  
      bool b = p is object;   // CS0244 p is pointer  
   }  
  
   unsafe public static void Main()  
   {  
      int i = 5;  
      SquarePtrParam (&i);  
   }  
}  
```
