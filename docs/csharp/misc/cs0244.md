# <a name="compiler-error-cs0244"></a><span data-ttu-id="bfb02-102">Derleyici Hatası CS0244</span><span class="sxs-lookup"><span data-stu-id="bfb02-102">Compiler Error CS0244</span></span>
<span data-ttu-id="bfb02-103">'İs' ve 'as' işaretçi türlerinde geçerli değildir</span><span class="sxs-lookup"><span data-stu-id="bfb02-103">Neither 'is' nor 'as' is valid on pointer types</span></span>  
  
 <span data-ttu-id="bfb02-104">[Olduğu](../../csharp/language-reference/keywords/is.md) ve [olarak](../../csharp/language-reference/keywords/as.md) anahtar sözcükleri işaretçi türleri üzerinde kullanım için geçerli değildir.</span><span class="sxs-lookup"><span data-stu-id="bfb02-104">The [is](../../csharp/language-reference/keywords/is.md) and [as](../../csharp/language-reference/keywords/as.md) keywords are not valid for use on pointer types.</span></span> <span data-ttu-id="bfb02-105">Daha fazla bilgi için [güvenli olmayan kod ve işaretçiler](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="bfb02-105">For more information, see [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="bfb02-106">Aşağıdaki örnek, CS0244 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="bfb02-106">The following sample generates CS0244:</span></span>  
  
```csharp  
// CS0244.cs  
// compile with: /unsafe  
  
class UnsafeTest  
{  
   unsafe static void SquarePtrParam (int* p)  
   {  
      bool b = p is object;   // CS0244 p is pointer  
   }  
  
   unsafe public static void Main()  
   {  
      int i = 5;  
      SquarePtrParam (&i);  
   }  
}  
```
