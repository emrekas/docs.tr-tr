# <a name="compiler-error-cs0669"></a><span data-ttu-id="bb87f-102">Derleyici Hatası CS0669</span><span class="sxs-lookup"><span data-stu-id="bb87f-102">Compiler Error CS0669</span></span>

<span data-ttu-id="bb87f-103">ComImport özniteliğine sahip sınıf bir kullanıcı tanımlı oluşturucusu olamaz.</span><span class="sxs-lookup"><span data-stu-id="bb87f-103">A class with the ComImport attribute cannot have a user-defined constructor.</span></span>

<span data-ttu-id="bb87f-104">COM birlikte çalışma katmanı ortak dil çalışma zamanı için oluşturucu sağlayan [ComImport](xref:System.Runtime.InteropServices.ComImportAttribute) sınıfları.</span><span class="sxs-lookup"><span data-stu-id="bb87f-104">The COM interop layer in the common language runtime supplies the constructor for [ComImport](xref:System.Runtime.InteropServices.ComImportAttribute) classes.</span></span> <span data-ttu-id="bb87f-105">Sonuç olarak, bir COM nesnesi olarak yönetilen bir nesnenin çalışma zamanında kullanılabilir.</span><span class="sxs-lookup"><span data-stu-id="bb87f-105">Consequently, a COM object can be used as a managed object in the runtime.</span></span>

<span data-ttu-id="bb87f-106">Aşağıdaki örnek, CS0669 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="bb87f-106">The following sample generates CS0669:</span></span>

```csharp
// CS0669.cs
using System.Runtime.InteropServices;
[ComImport, Guid("00000000-0000-0000-0000-000000000001")]
class TestClass
{
   TestClass()   // CS0669, delete constructor to resolve
   {
   }

   public static void Main()
   {
   }
}
```
