# <a name="compiler-error-cs0470"></a>Derleyici Hatası CS0470
Yöntemi 'method' type 'türü için ' erişimcisi' arabirim erişimcisini uygulayamaz. Açık arabirim uygulaması kullanın.  
  
 Bir arabirim uygulamak bir erişimci çalışırken bu hatayı oluşturulur. Açık arabirim uygulaması kullanılmalıdır.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS0470 oluşturur.  
  
```csharp  
// CS0470.cs  
// compile with: /target:library  
  
interface I  
{  
   int P { get; }  
}  
  
class MyClass : I  
{  
   public int get_P() { return 0; }   // CS0470  
   public int P2 { get { return 0;} }   // OK  
}  
```
