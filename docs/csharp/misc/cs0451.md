# <a name="compiler-error-cs0451"></a>Derleyici Hatası CS0451
'New()' kısıtlaması 'struct' kısıtlamasıyla kullanılamaz  
  
 Bir genel tür kısıtlamaları belirtirken `new()` kısıtlaması, yalnızca sınıf türü kısıtlamaları, arabirim tür kısıtlamaları, başvuru türü kısıtlamaları ve tür parametresi kısıtlamaları, ancak değil değeri tür kısıtlamaları ile kullanılabilir.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS0451 oluşturur.  
  
```csharp  
// CS0451.cs  
using System;  
public class C4   
{  
   public void F4<T>() where T : struct, new() {}   // CS0451  
}  
  
// OK  
public class C5  
{  
   public void F5<T>() where T : struct {}  
}  
  
public class C6  
{  
   public void F6<T>() where T : new() {}  
}  
```
