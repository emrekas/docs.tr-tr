# <a name="compiler-error-cs0161"></a>Derleyici hatası CS0161
' Method ': tüm kod yolları bir değer döndürmüyor  
  
 Değer döndüren bir yöntem, tüm kod yollarında bir `return` ifadeye sahip olmalıdır. Daha fazla bilgi için bkz. [Yöntemler](../programming-guide/classes-and-structs/methods.md).  
  
 Aşağıdaki örnek CS0161 oluşturur:  
  
```csharp  
// CS0161.cs  
public class Test  
{  
   public static int Main() // CS0161  
   {  
      int i = 10;  
      if (i < 10)  
      {  
         return i;  
      }  
      else  
      {  
         // uncomment the following line to resolve  
         // return 1;  
      }  
   }  
}  
```
