# <a name="compiler-error-cs0161"></a>Derleyici Hatası CS0161
'method': tüm kod yolları bir değer döndürmez  
  
 Bir değer döndüren bir yöntem olmalıdır bir `return` deyiminde tüm kod yolları. Daha fazla bilgi için [yöntemleri](../../csharp/programming-guide/classes-and-structs/methods.md).  
  
 Aşağıdaki örnek, CS0161 oluşturur:  
  
```csharp  
// CS0161.cs  
public class Test  
{  
   public static int Main() // CS0161  
   {  
      int i = 10;  
      if (i < 10)  
      {  
         return i;  
      }  
      else  
      {  
         // uncomment the following line to resolve  
         // return 1;  
      }  
   }  
}  
```
