# <a name="compiler-error-cs0213"></a><span data-ttu-id="9f0f5-102">Derleyici Hatası CS0213</span><span class="sxs-lookup"><span data-stu-id="9f0f5-102">Compiler Error CS0213</span></span>
<span data-ttu-id="9f0f5-103">Zaten sabitlenmiş bir ifadenin adresini almak için fixed deyimini kullanamazsınız</span><span class="sxs-lookup"><span data-stu-id="9f0f5-103">You cannot use the fixed statement to take the address of an already fixed expression</span></span>  
  
 <span data-ttu-id="9f0f5-104">Yerel değişken bir [güvenli olmayan](../../csharp/language-reference/keywords/unsafe.md) yöntemi veya bir parametre zaten sabit (stack) ya da bu iki değişkenin adresi alınamaz şekilde bir [sabit](../../csharp/language-reference/keywords/fixed-statement.md) ifade.</span><span class="sxs-lookup"><span data-stu-id="9f0f5-104">A local variable in an [unsafe](../../csharp/language-reference/keywords/unsafe.md) method or a parameter is already fixed (on the stack), so you cannot take the address of either of these two variables in a [fixed](../../csharp/language-reference/keywords/fixed-statement.md) expression.</span></span> <span data-ttu-id="9f0f5-105">Daha fazla bilgi için [güvenli olmayan kod ve işaretçiler](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="9f0f5-105">For more information, see [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="9f0f5-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="9f0f5-106">Example</span></span>  
 <span data-ttu-id="9f0f5-107">Aşağıdaki örnek, CS0213 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="9f0f5-107">The following sample generates CS0213.</span></span>  
  
```csharp  
// CS0213.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int i = 45;  
      fixed (int *j = &i) { }  // CS0213  
      // try the following line instead  
      // int* j = &i;  
  
      int[] a = new int[] {1,2,3};  
      fixed (int *b = a)  
      {  
         fixed (int *c = b) { }  // CS0213  
         // try the following line instead  
         // int *c = b;  
      }  
   }  
}  
```
