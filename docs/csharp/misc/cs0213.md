# <a name="compiler-error-cs0213"></a>Derleyici Hatası CS0213
Zaten sabitlenmiş bir ifadenin adresini almak için fixed deyimini kullanamazsınız  
  
 Yerel değişken bir [güvenli olmayan](../../csharp/language-reference/keywords/unsafe.md) yöntemi veya bir parametre zaten sabit (stack) ya da bu iki değişkenin adresi alınamaz şekilde bir [sabit](../../csharp/language-reference/keywords/fixed-statement.md) ifade. Daha fazla bilgi için [güvenli olmayan kod ve işaretçiler](../../csharp/programming-guide/unsafe-code-pointers/index.md).  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS0213 oluşturur.  
  
```csharp  
// CS0213.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int i = 45;  
      fixed (int *j = &i) { }  // CS0213  
      // try the following line instead  
      // int* j = &i;  
  
      int[] a = new int[] {1,2,3};  
      fixed (int *b = a)  
      {  
         fixed (int *c = b) { }  // CS0213  
         // try the following line instead  
         // int *c = b;  
      }  
   }  
}  
```
