# <a name="compiler-error-cs0522"></a><span data-ttu-id="717a7-102">Derleyici Hatası CS0522</span><span class="sxs-lookup"><span data-stu-id="717a7-102">Compiler Error CS0522</span></span>
<span data-ttu-id="717a7-103">'Oluşturucu': yapılar temel sınıf oluşturucuları çağıramaz</span><span class="sxs-lookup"><span data-stu-id="717a7-103">'constructor' : structs cannot call base class constructors</span></span>  
  
 <span data-ttu-id="717a7-104">A [yapı](../../csharp/language-reference/keywords/struct.md) edilemez bir temel sınıf oluşturucusunu çağırın; temel sınıf oluşturucusunu çağrısını kaldırın.</span><span class="sxs-lookup"><span data-stu-id="717a7-104">A [struct](../../csharp/language-reference/keywords/struct.md) cannot call a base class constructor; remove the call to the base class constructor.</span></span>  
  
 <span data-ttu-id="717a7-105">Aşağıdaki örnek, CS0522 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="717a7-105">The following sample generates CS0522:</span></span>  
  
```csharp  
// CS0522.cs  
public class clx  
{  
   public clx(int i)  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public struct cly  
{  
   public cly(int i):base(0)   // CS0522  
   // try the following line instead  
   // public cly(int i)  
   {  
   }  
}  
```
