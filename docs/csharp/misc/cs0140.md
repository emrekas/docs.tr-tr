# <a name="compiler-error-cs0140"></a>Derleyici Hatası CS0140
'Etiketi' etiketi yineleniyor  
  
 Aynı ada sahip bir etiket iki kez görüldü. Daha fazla bilgi için [goto](../../csharp/language-reference/keywords/goto.md).  
  
 Aşağıdaki örnek, CS0140 oluşturur:  
  
```csharp  
// CS0140.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         label1: int i = 0;  
         label1: int j = 0;   // CS0140, comment this line to resolve  
         goto label1;  
      }  
   }  
}  
```
