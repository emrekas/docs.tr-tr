# <a name="compiler-error-cs0210"></a><span data-ttu-id="f4cb5-102">Derleyici Hatası CS0210</span><span class="sxs-lookup"><span data-stu-id="f4cb5-102">Compiler Error CS0210</span></span>
<span data-ttu-id="f4cb5-103">Bir fixed veya using deyimi bildiriminde bir başlatıcı sağlamalısınız</span><span class="sxs-lookup"><span data-stu-id="f4cb5-103">You must provide an initializer in a fixed or using statement declaration</span></span>  
  
 <span data-ttu-id="f4cb5-104">Bildirmek ve başlatmak değişkeninde bir [fixed statement](../../csharp/language-reference/keywords/fixed-statement.md).</span><span class="sxs-lookup"><span data-stu-id="f4cb5-104">You must declare and initialize the variable in a [fixed statement](../../csharp/language-reference/keywords/fixed-statement.md).</span></span> <span data-ttu-id="f4cb5-105">Daha fazla bilgi için [güvenli olmayan kod ve işaretçiler](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="f4cb5-105">For more information, see [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="f4cb5-106">Aşağıdaki örnek, CS0210 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="f4cb5-106">The following sample generates CS0210:</span></span>  
  
```csharp  
// CS0210a.cs  
// compile with: /unsafe  
  
class Point  
{  
   public int x, y;  
}  
  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
  
      fixed (int i)    // CS0210  
      {  
      }  
      // try the following lines instead  
      /*  
      fixed (int* p = &pt.x)  
      {  
      }  
      fixed (int* q = &pt.y)  
      {  
      }  
      */  
   }  
}  
```  
  
 <span data-ttu-id="f4cb5-107">Aşağıdaki örnek aynı zamanda CS0210 oluşturur çünkü [using deyimi](../../csharp/language-reference/keywords/using-statement.md) Başlatıcı içermiyor.</span><span class="sxs-lookup"><span data-stu-id="f4cb5-107">The following sample also generates CS0210 because the [using statement](../../csharp/language-reference/keywords/using-statement.md) has no initializer.</span></span>  
  
```csharp  
// CS0210b.cs  
  
using System.IO;  
class Test   
{  
   static void Main()   
   {  
      using (StreamWriter w) // CS0210  
      // Try this line instead:  
      // using (StreamWriter w = new StreamWriter("TestFile.txt"))   
      {  
         w.WriteLine("Hello there");  
      }  
   }  
}  
```
