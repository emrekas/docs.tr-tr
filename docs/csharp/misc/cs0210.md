# <a name="compiler-error-cs0210"></a><span data-ttu-id="68fb6-102">Derleyici hatası CS0210</span><span class="sxs-lookup"><span data-stu-id="68fb6-102">Compiler Error CS0210</span></span>
<span data-ttu-id="68fb6-103">Fixed veya using ifade bildiriminde bir başlatıcı sağlamalısınız</span><span class="sxs-lookup"><span data-stu-id="68fb6-103">You must provide an initializer in a fixed or using statement declaration</span></span>  
  
 <span data-ttu-id="68fb6-104">Değişkeni [fixed ifadesinde](../language-reference/keywords/fixed-statement.md)bildirmelidir ve başlatmalısınız.</span><span class="sxs-lookup"><span data-stu-id="68fb6-104">You must declare and initialize the variable in a [fixed statement](../language-reference/keywords/fixed-statement.md).</span></span> <span data-ttu-id="68fb6-105">Daha fazla bilgi için bkz. [güvenli olmayan kod ve işaretçiler](../programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="68fb6-105">For more information, see [Unsafe Code and Pointers](../programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="68fb6-106">Aşağıdaki örnek CS0210 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="68fb6-106">The following sample generates CS0210:</span></span>  
  
```csharp  
// CS0210a.cs  
// compile with: /unsafe  
  
class Point  
{  
   public int x, y;  
}  
  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
  
      fixed (int i)    // CS0210  
      {  
      }  
      // try the following lines instead  
      /*  
      fixed (int* p = &pt.x)  
      {  
      }  
      fixed (int* q = &pt.y)  
      {  
      }  
      */  
   }  
}  
```  
  
 <span data-ttu-id="68fb6-107">Ayrıca, [using ifadesinin](../language-reference/keywords/using-statement.md) başlatıcısı olmadığından aşağıdaki örnek de CS0210 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="68fb6-107">The following sample also generates CS0210 because the [using statement](../language-reference/keywords/using-statement.md) has no initializer.</span></span>  
  
```csharp  
// CS0210b.cs  
  
using System.IO;  
class Test   
{  
   static void Main()   
   {  
      using (StreamWriter w) // CS0210  
      // Try this line instead:  
      // using (StreamWriter w = new StreamWriter("TestFile.txt"))   
      {  
         w.WriteLine("Hello there");  
      }  
   }  
}  
```
