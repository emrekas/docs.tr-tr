# <a name="compiler-error-cs0186"></a>Derleyici Hatası CS0186
Null kullanımı bu bağlamda geçerli değil  
  
 Aşağıdaki örnek, CS0186 oluşturur:  
  
```csharp  
// CS0186.cs  
using System;  
using System.Collections;  
  
class MyClass   
{  
   static void Main()   
   {  
      // Each of the following lines generates CS0186:  
      foreach (int i in null) {}   // CS0186  
      foreach (int i in (IEnumerable) null) { };   // CS0186  
   }  
}  
```
