 <span data-ttu-id="80e2c-104">[Olay](../language-reference/keywords/event.md) tanımındaki bir **Add** veya **Remove** anahtar sözcüğünün gövdesi olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="80e2c-104">An **add** or **remove** keyword in an [event](../language-reference/keywords/event.md) definition must have a body.</span></span> <span data-ttu-id="80e2c-105">Daha fazla bilgi için bkz.  <span data-ttu-id="80e2c-106">Aşağıdaki örnek CS0073 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="80e2c-106">The following sample generates CS0073:</span></span>  
  
```csharp  
// CS0073.cs  
delegate void del();

{  
   public event del MyEvent  
   {  
      add;   // CS0073  
      // try the following lines instead  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      remove  
      {  
         MyEvent -= value;  
      }  
  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
