# <a name="compiler-error-cs2033"></a>Derleyici Hatası CS2033
Aynı kısa dosya adlı uzun dosya adı zaten varken 'filename' kısa dosya adı oluşturulamıyor  
  
 Tüm derleme C# sekiz karakterden daha uzun bir ada sahip dosya. Ardından kısa sürüm adı ilk altı karakterinin yanı sıra "~ 1." gibi önceki dosya adının, başka bir dosyaya derleme İkinci derleme, bu hata oluşturur.  
  
 Bu hatayı gidermek için uzun dosya adıyla çakışmayacak bir kısa dosya adını yeniden adlandırın.
