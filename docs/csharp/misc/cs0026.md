## <a name="example"></a><span data-ttu-id="632ae-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="632ae-107">Example</span></span>  
 <span data-ttu-id="632ae-108">Aşağıdaki örnek CS0026 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="632ae-108">The following example generates CS0026:</span></span>  
  
```csharp  
// CS0026.cs  
public class A  
{  
   public static int i = 0;  
  
   public static void Main()  
   {  
// CS0026  
      this.i = this.i + 1;     
      // Try the following line instead:  
      // i = i + 1;  
   }  
}  
```
