# <a name="compiler-error-cs1534"></a><span data-ttu-id="305ee-102">Derleyici hatası CS1534</span><span class="sxs-lookup"><span data-stu-id="305ee-102">Compiler Error CS1534</span></span>

 <span data-ttu-id="305ee-104">İkili [işleç](../language-reference/operators/operator-overloading.md) tanımının iki parametre olması gerekir.</span><span class="sxs-lookup"><span data-stu-id="305ee-104">The definition of a binary [operator](../language-reference/operators/operator-overloading.md) must take two parameters.</span></span>

   }

      return 1;  
   }
