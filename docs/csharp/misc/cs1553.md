# <a name="compiler-error-cs1553"></a>Derleyici Hatası CS1553
Bildirim geçerli değil; Kullan ' değiştiricisi işleci \<hedef-tür > (...' Bunun yerine  
  
İçin dönüş türü bir [dönüştürme işleci](../../csharp/language-reference/operators/user-defined-conversion-operators.md) parametre listesinde hemen önce gelmelidir ve *değiştiricisi* ya da `implicit` veya `explicit`.
  
Aşağıdaki örnek, CS1553 oluşturur:  
  
```csharp  
// CS1553.cs  
class MyClass  
{  
   public static int implicit operator (MyClass f)   // CS1553  
   // try the following line instead  
   // public static implicit operator int (MyClass f)  
   {  
      return 6;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
