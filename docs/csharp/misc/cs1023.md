 <span data-ttu-id="849b3-104">Deyimlerini aşağıdaki gibi bir gömülü deyim bir **varsa** bildirimleri ne etiketli deyimler deyimi içerebilir.</span><span class="sxs-lookup"><span data-stu-id="849b3-104">An embedded statement, such as the statements following an **if** statement, can contain neither declarations nor labeled statements.</span></span>

   public static void Main()  
   {  
      if (1)  
         int i;      // CS1023, declaration is not valid here

         xx : i++;   // CS1023, labeled statement is not valid here  
   }  
}
