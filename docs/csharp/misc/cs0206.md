## <a name="example"></a><span data-ttu-id="6cde2-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="6cde2-106">Example</span></span>  
 <span data-ttu-id="6cde2-107">Aşağıdaki örnek CS0206 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="6cde2-107">The following sample generates CS0206:</span></span>  
  
```csharp  
// CS0206.cs  
public class MyClass

    public static int P  
    {  
        get  
        {  
            return 0;  
        }  
        set

        }  
    }

    }

    {  
        MyMeth(ref P);   // CS0206  
        // try the following line instead  
        // MyMeth(P);   // CS0206  
    }  
}  
```
