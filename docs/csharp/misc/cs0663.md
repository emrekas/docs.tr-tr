# <a name="compiler-error-cs0663"></a><span data-ttu-id="7a436-102">Derleyici hatası CS0663</span><span class="sxs-lookup"><span data-stu-id="7a436-102">Compiler Error CS0663</span></span>
<span data-ttu-id="7a436-103">Yalnızca ref ve out ile farklı olan aşırı yüklenmiş yöntemler tanımlanamaz.</span><span class="sxs-lookup"><span data-stu-id="7a436-103">Cannot define overloaded methods that differ only on ref and out.</span></span>  
  
 <span data-ttu-id="7a436-104">Yalnızca [içinde](../language-reference/keywords/in-parameter-modifier.md)kullanımları, [ref](../language-reference/keywords/ref.md) ve [Out](../language-reference/keywords/out-parameter-modifier.md) için bir parametre üzerinde farklı olan yöntemlere izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="7a436-104">Methods that differ only on their use of [in](../language-reference/keywords/in-parameter-modifier.md), [ref](../language-reference/keywords/ref.md) and [out](../language-reference/keywords/out-parameter-modifier.md) on a parameter are not allowed.</span></span>  
  
 <span data-ttu-id="7a436-105">Aşağıdaki örnek CS0663 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="7a436-105">The following sample generates CS0663:</span></span>  
  
```csharp  
// CS0663.cs  
class TestClass  
{  
   public static void Main()  
   {  
   }  
  
   public void Test(ref int i)  
   {  
   }  
  
   public void Test(out int i)   // CS0663  
   {  
   }  
}  
```
