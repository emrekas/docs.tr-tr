# <a name="compiler-error-cs0434"></a><span data-ttu-id="c3657-102">Derleyici Hatası CS0434</span><span class="sxs-lookup"><span data-stu-id="c3657-102">Compiler Error CS0434</span></span>

 <span data-ttu-id="c3657-104">İçeri aktarılmış bir tür olduğunda bu hata oluşur ve içeri aktarılan bir iç içe geçmiş ad alanı, aynı tam ada sahip.</span><span class="sxs-lookup"><span data-stu-id="c3657-104">This error occurs when an imported type and an imported nested namespace have the same fully qualified name.</span></span> <span data-ttu-id="c3657-105">Bu ada başvurulduğunda, ikisi arasında ayrım yapmak derleyici silemiyor.</span><span class="sxs-lookup"><span data-stu-id="c3657-105">When that name is referenced, the compiler is unable to distinguish between the two.</span></span> <span data-ttu-id="c3657-106">İçeri aktarılan kaynak kodu değiştirebilirsiniz, hem de derleme içinde benzersiz olmasını sağlamak, türü veya ad alanı adını değiştirerek bu hatayı çözebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="c3657-106">If you can change the imported source code, you can resolve the error by changing the name of either the type or the namespace so that both are unique within the assembly.</span></span>  
  
 <span data-ttu-id="c3657-107">Aşağıdaki kod, hata CS0434 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="c3657-107">The following code generates error CS0434.</span></span>  
  
## <a name="example"></a><span data-ttu-id="c3657-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="c3657-108">Example</span></span>  
 <span data-ttu-id="c3657-109">Bu kod ile aynı tam ada türünün ilk kopyasını oluşturur.</span><span class="sxs-lookup"><span data-stu-id="c3657-109">This code creates the first copy of the type with the identical fully qualified name.</span></span>  
  
```csharp  
// CS0434_1.cs  
// compile with: /t:library  
namespace TypeBindConflicts   
{  
    namespace NsImpAggPubImp   
    {  
        public class X { }  
    }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="c3657-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="c3657-110">Example</span></span>
