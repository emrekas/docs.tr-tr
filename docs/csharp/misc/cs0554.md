# <a name="compiler-error-cs0554"></a><span data-ttu-id="d568b-102">Derleyici Hatası CS0554</span><span class="sxs-lookup"><span data-stu-id="d568b-102">Compiler Error CS0554</span></span>
<span data-ttu-id="d568b-103">'dönüştürme yordamına': kullanıcı tanımlı dönüştürme/türetilmiş sınıf</span><span class="sxs-lookup"><span data-stu-id="d568b-103">'conversion routine' : user defined conversion to/from derived class</span></span>  
  
 <span data-ttu-id="d568b-104">Türetilmiş bir sınıfın kullanıcı tanımlı dönüşümler değerlere izin verilmiyor; Böyle bir işleç gerekmez.</span><span class="sxs-lookup"><span data-stu-id="d568b-104">User-defined conversions to values of a derived class are not allowed; you do not need such an operator.</span></span>  
  
 <span data-ttu-id="d568b-105">Bölüm 6 bkz C# kullanıcı tanımlı dönüştürmeler hakkında daha fazla bilgi için dil belirtimi.</span><span class="sxs-lookup"><span data-stu-id="d568b-105">See chapter 6 in the C# language specification for more information on user-defined conversions.</span></span>  
  
 <span data-ttu-id="d568b-106">Aşağıdaki örnek, CS0554 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="d568b-106">The following sample generates CS0554:</span></span>  
  
```csharp  
// CS0554.cs  
namespace x  
{  
   public class ii  
   {  
      // delete the conversion routine to resolve CS0554  
      public static implicit operator ii(a aa) // CS0554  
      {  
         return new ii();  
      }  
   }  
  
   public class a : ii  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
