# <a name="compiler-error-cs0132"></a><span data-ttu-id="276b2-102">Derleyici Hatası CS0132</span><span class="sxs-lookup"><span data-stu-id="276b2-102">Compiler Error CS0132</span></span>
<span data-ttu-id="276b2-103">'Oluşturucu': statik Oluşturucusu parametresiz olmalıdır</span><span class="sxs-lookup"><span data-stu-id="276b2-103">'constructor' : a static constructor must be parameterless</span></span>  
  
 <span data-ttu-id="276b2-104">A [statik](../../csharp/language-reference/keywords/static.md) bir veya daha fazla parametrelerle Oluşturucusu bildirilemez.</span><span class="sxs-lookup"><span data-stu-id="276b2-104">A [static](../../csharp/language-reference/keywords/static.md) constructor cannot be declared with one or more parameters.</span></span> <span data-ttu-id="276b2-105">Daha fazla bilgi için [oluşturucular](../../csharp/programming-guide/classes-and-structs/constructors.md).</span><span class="sxs-lookup"><span data-stu-id="276b2-105">For more information, see [Constructors](../../csharp/programming-guide/classes-and-structs/constructors.md).</span></span>  
  
 <span data-ttu-id="276b2-106">Aşağıdaki örnek, CS0132 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="276b2-106">The following sample generates CS0132:</span></span>  
  
```csharp  
// CS0132.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public MyClass(int i)  
      {  
      }  
   }  
  
   public class MyClass2 : MyClass  
   {  
      static MyClass2(int i)   // CS0132  
      {  
      }  
   }  
}  
```
