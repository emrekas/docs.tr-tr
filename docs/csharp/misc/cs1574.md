# <a name="compiler-warning-level-1-cs1574"></a>Derleyici Uyarısı (düzey 1) CS1574
'Yapısı' üzerindeki XML açıklamasının sözdizimsel olarak yanlış cref 'name' özniteliği var  
  
 Bir dize geçirilen cref etiket, örneğin bir \<özel durum > etiketi, geçerli yapı ortamı içinde kullanılabilir olmayan bir üyeye başvuru. Cref etiket geçen dize sözdizimsel olarak doğru bir üye veya alan adı olmalıdır.  
  
 Daha fazla bilgi için [belge açıklamaları için önerilen etiketler](../../csharp/programming-guide/xmldoc/recommended-tags-for-documentation-comments.md).  
  
 Aşağıdaki örnek, CS1574 oluşturur:  
  
```csharp  
// CS1574.cs  
// compile with: /W:1 /doc:x.xml  
using System;  
  
/// <exception cref="System.Console.WriteLin">An exception class.</exception>   // CS1574  
// instead, uncomment and try the following line  
// /// <exception cref="System.Console.WriteLine">An exception class.</exception>  
class EClass : Exception  
{  
}  
  
class TestClass  
{  
   public static void Main()  
   {  
      try  
      {  
      }  
      catch(EClass)  
      {  
      }  
   }  
}  
```
