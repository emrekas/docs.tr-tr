# <a name="compiler-warning-level-1-cs1200"></a>Derleyici Uyarısı (düzey 1) CS1200
'Geçersiz özellik' özelliği kullanımdan kaldırılmıştır. Lütfen bunun yerine 'özelliği geçerli' kullanın '.  
  
 Kullanmaya çalıştığınız özellik artık kullanılmıyor. Bunun yerine geçerli bu özelliği kullanmak için kodunuzu güncelleştirin.
