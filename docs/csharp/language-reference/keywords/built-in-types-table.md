<span data-ttu-id="e815e-123">Tüm türleri tabloda dışında `object` ve `string`, basit türler olarak adlandırılır.</span><span class="sxs-lookup"><span data-stu-id="e815e-123">All of the types in the table, except `object` and `string`, are referred to as simple types.</span></span>

```csharp

```

<span data-ttu-id="e815e-126">Kullanım [typeof](typeof.md) almak için işleci <xref:System.Type?displayProperty=nameWithType> belirtilen türünü temsil eden örneği:</span><span class="sxs-lookup"><span data-stu-id="e815e-126">Use the [typeof](typeof.md) operator to get the <xref:System.Type?displayProperty=nameWithType> instance that represents the specified type:</span></span>

```csharp
Console.WriteLine(stringType.FullName);

Type doubleType = typeof(System.Double);

// Output:
// System.String
// System.Double
```
