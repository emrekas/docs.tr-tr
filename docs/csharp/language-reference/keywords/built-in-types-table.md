## <a name="remarks"></a><span data-ttu-id="cc381-122">Açıklamalar</span><span class="sxs-lookup"><span data-stu-id="cc381-122">Remarks</span></span>

<span data-ttu-id="cc381-123">Tüm türleri tabloda dışında `object` ve `string`, basit türler olarak adlandırılır.</span><span class="sxs-lookup"><span data-stu-id="cc381-123">All of the types in the table, except `object` and `string`, are referred to as simple types.</span></span>  
  
<span data-ttu-id="cc381-124">C# anahtar sözcükleri yazın ve diğer adlarını birbirinin yerine kullanılabilir.</span><span class="sxs-lookup"><span data-stu-id="cc381-124">The C# type keywords and their aliases are interchangeable.</span></span> <span data-ttu-id="cc381-125">Örneğin, aşağıdaki bildirimleri birini kullanarak bir tam sayı değişkeni bildirebilirsiniz:</span><span class="sxs-lookup"><span data-stu-id="cc381-125">For example, you can declare an integer variable by using either of the following declarations:</span></span>  

```csharp
int x = 123;
System.Int32 y = 123;
```

<span data-ttu-id="cc381-126">Kullanım [typeof](../operators/type-testing-and-conversion-operators.md#typeof-operator) almak için işleci <xref:System.Type?displayProperty=nameWithType> belirtilen türünü temsil eden örneği:</span><span class="sxs-lookup"><span data-stu-id="cc381-126">Use the [typeof](../operators/type-testing-and-conversion-operators.md#typeof-operator) operator to get the <xref:System.Type?displayProperty=nameWithType> instance that represents the specified type:</span></span>

```csharp

Console.WriteLine(stringType.FullName);

Type doubleType = typeof(System.Double);
Console.WriteLine(doubleType.FullName);

// Output:
// System.String
// System.Double
```
