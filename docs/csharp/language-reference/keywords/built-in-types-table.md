## <a name="remarks"></a><span data-ttu-id="13494-122">Açıklamalar</span><span class="sxs-lookup"><span data-stu-id="13494-122">Remarks</span></span>

```csharp

```

<span data-ttu-id="13494-126">Belirtilen türü [](../operators/type-testing-and-cast.md#typeof-operator) temsil eden <xref:System.Type?displayProperty=nameWithType> örneği almak için typeof işlecini kullanın:</span><span class="sxs-lookup"><span data-stu-id="13494-126">Use the [typeof](../operators/type-testing-and-cast.md#typeof-operator) operator to get the <xref:System.Type?displayProperty=nameWithType> instance that represents the specified type:</span></span>

```csharp

Type doubleType = typeof(System.Double);

// Output:
// System.String
// System.Double
```
