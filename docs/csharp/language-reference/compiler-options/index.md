# <a name="c-compiler-options"></a>C# Derleyici Seçenekleri
Yürütülebilir (.exe) dosyaları, dinamik bağlantı kitaplığı (.dll) veya kod modüllerini derleyici üretir (.netmodule).  
  
 Her derleyici seçeneğinin iki biçimde sağlanır: **-seçeneği** ve **/seçeneği**. Yalnızca belgelerde **-seçeneği** formu.  
  
 Visual Studio'da web.config dosyasında derleyici seçeneklerini ayarlayın. Daha fazla bilgi için [ \<derleyici > öğesi](../../../framework/configure-apps/file-schema/compiler/compiler-element.md).
