# <a name="amp-operator-c-reference"></a>&amp;= İşleci (C# Başvurusu)

AND atama işleci.

Bir ifade kullanarak `&=` işleci gibi

```csharp
x &= y
```

eşdeğerdir

```csharp
x = x & y
```

dışında `x` yalnızca bir kez değerlendirilir.
