<span data-ttu-id="91a9f-103">'Method' için en iyi eşleşen Aşırı yüklenen yöntem, başlatıcı öğesi için yanlış imza içeriyor.</span><span class="sxs-lookup"><span data-stu-id="91a9f-103">The best overloaded method match for 'method' has wrong signature for the initializer element.</span></span> <span data-ttu-id="91a9f-104">Başlatılabilir Add, erişilebilir bir örnek yöntemi olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="91a9f-104">The initializable Add must be an accessible instance method.</span></span>  
  
 <span data-ttu-id="91a9f-105">Koleksiyon Başlatıcısı ile hiçbir genel statik olmayan sınıf kullanmaya çalıştığınızda bu hatayı oluşturulan `Add` yöntemi.</span><span class="sxs-lookup"><span data-stu-id="91a9f-105">This error is generated when you try to use a collection initializer with a class that has no public non-static `Add` method.</span></span> <span data-ttu-id="91a9f-106">Varsa `Add` yöntemi erişilebilir durumda değil, koruma düzeyi nedeniyle (`private`, `protected`, `internal`) Bu hata büyük olasılıkla yöntemi olarak tanımlanır anlamına CS0122, olacağı `static`.</span><span class="sxs-lookup"><span data-stu-id="91a9f-106">If the `Add` method is not accessible because of its protection level (`private`, `protected`, `internal`) then you will get CS0122, so that this error probably means that the method is defined as `static`.</span></span>
