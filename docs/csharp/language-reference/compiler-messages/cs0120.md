# <a name="compiler-error-cs0120"></a><span data-ttu-id="fb6f8-102">Derleyici Hatası CS0120</span><span class="sxs-lookup"><span data-stu-id="fb6f8-102">Compiler Error CS0120</span></span>

<span data-ttu-id="fb6f8-103">Bir nesne başvurusu statik olmayan alanı, yöntemi veya özelliği 'üyesinin' gereklidir</span><span class="sxs-lookup"><span data-stu-id="fb6f8-103">An object reference is required for the nonstatic field, method, or property 'member'</span></span>  
  
 <span data-ttu-id="fb6f8-104">Statik olmayan alanı, yöntemi veya özelliği kullanmak için bir nesne örneği oluşturun.</span><span class="sxs-lookup"><span data-stu-id="fb6f8-104">In order to use a non-static field, method, or property, you must first create an object instance.</span></span> <span data-ttu-id="fb6f8-105">Statik yöntemleri hakkında daha fazla bilgi için bkz. [statik sınıflar ve statik sınıf üyeleri](../../../csharp/programming-guide/classes-and-structs/static-classes-and-static-class-members.md).</span><span class="sxs-lookup"><span data-stu-id="fb6f8-105">For more information about static methods, see [Static Classes and Static Class Members](../../../csharp/programming-guide/classes-and-structs/static-classes-and-static-class-members.md).</span></span> <span data-ttu-id="fb6f8-106">Sınıfları oluşturma hakkında daha fazla bilgi için bkz. [örnek oluşturucuları](../../../csharp/programming-guide/classes-and-structs/instance-constructors.md).</span><span class="sxs-lookup"><span data-stu-id="fb6f8-106">For more information about creating instances of classes, see [Instance Constructors](../../../csharp/programming-guide/classes-and-structs/instance-constructors.md).</span></span>  
  
 <span data-ttu-id="fb6f8-107">Aşağıdaki örnek, CS0120 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="fb6f8-107">The following sample generates CS0120:</span></span>  

```csharp
// CS0120_1.cs  
public class MyClass  
{  
   // Non-static field  
   public int i;
   // Non-static method  
   public void f(){}  
   // Non-static property  
   int Prop  
   {  
      get
      {  
         return 1;
      }  
   }  
  
   public static void Main()  
   {  
      i = 10;   // CS0120  
      f();   // CS0120  
      int p = Prop;   // CS0120  
      // try the following lines instead  
      // MyClass mc = new MyClass();  
      // mc.i = 10;  
      // mc.f();  
      // int p = mc.Prop;  
   }  
}  
```

 <span data-ttu-id="fb6f8-108">CS0120 ayrıca varsa statik olmayan bir yönteme bir çağrı statik bir yöntemden şu şekilde oluşturulur:</span><span class="sxs-lookup"><span data-stu-id="fb6f8-108">CS0120 will also be generated if there is a call to a non-static method from a static method, as follows:</span></span>  

```csharp
// CS0120_2.cs  
// CS0120 expected  
using System;  
  
public class MyClass  
{  
   public static void Main()  
   {  
      TestCall();   // CS0120  
      // To call a non-static method from Main,  
      // first create an instance of the class.  
      // Use the following two lines instead:  
      // MyClass anInstanceofMyClass = new MyClass();  
      // anInstanceofMyClass.TestCall();  
   }  
  
   public void TestCall()  
   {  
   }  
}  
```

 <span data-ttu-id="fb6f8-109">Benzer şekilde, açıkça bu sınıfının bir örneğini aşağıdaki şekilde size sürece bir statik yöntem bir örnek yöntemi çağrılamıyor:</span><span class="sxs-lookup"><span data-stu-id="fb6f8-109">Similarly, a static method cannot call an instance method unless you explicitly give it an instance of the class, as follows:</span></span>  

```csharp
// CS0120_3.cs  
using System;  
  
public class MyClass  
{  
   public static void Main()  
   {  
      do_it("Hello There");   // CS0120  
   }  
  
   private void do_it(string sText)  
   // You could also add the keyword static to the method definition:  
   // private static void do_it(string sText)  
   {  
      Console.WriteLine(sText);  
   }  
}  
```
