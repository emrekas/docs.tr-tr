# <a name="compiler-error-cs0233"></a><span data-ttu-id="064e7-102">Derleyici Hatası CS0233</span><span class="sxs-lookup"><span data-stu-id="064e7-102">Compiler Error CS0233</span></span>
<span data-ttu-id="064e7-103">'identifier' önceden tanımlı boyuta sahip değil ve bu nedenle sizeof yalnızca unsafe bağlamında kullanılabilir (System.Runtime.InteropServices.Marshal.sizeof kullanmayı düşünün)</span><span class="sxs-lookup"><span data-stu-id="064e7-103">'identifier' does not have a predefined size, therefore sizeof can only be used in an unsafe context (consider using System.Runtime.InteropServices.Marshal.SizeOf)</span></span>  
  
 <span data-ttu-id="064e7-104">[Sizeof](../../../csharp/language-reference/keywords/sizeof.md) işleci yalnızca derleme zamanı sabit değerlerdir türleri için kullanılabilir.</span><span class="sxs-lookup"><span data-stu-id="064e7-104">The [sizeof](../../../csharp/language-reference/keywords/sizeof.md) operator can only be used for types that are compile-time constants.</span></span> <span data-ttu-id="064e7-105">Bu hatayı alıyorsanız, tanımlayıcının boyutunu derleme zamanında belirlenebilir emin olun.</span><span class="sxs-lookup"><span data-stu-id="064e7-105">If you are getting this error, make sure that the size of the identifier can be determined at compile time.</span></span> <span data-ttu-id="064e7-106">Bunun ardından kullanamıyorsanız <xref:System.Runtime.InteropServices.Marshal.SizeOf%2A> yerine `sizeof`.</span><span class="sxs-lookup"><span data-stu-id="064e7-106">If it cannot, then use <xref:System.Runtime.InteropServices.Marshal.SizeOf%2A> instead of `sizeof`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="064e7-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="064e7-107">Example</span></span>  
 <span data-ttu-id="064e7-108">Aşağıdaki örnek, CS0233 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="064e7-108">The following example generates CS0233:</span></span>  
  
```csharp  
// CS0233.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Sequential)]  
public struct S  
{  
    public int a;  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        S myS = new S();  
        Console.WriteLine(sizeof(S));   // CS0233  
        // Try the following line instead:  
        // Console.WriteLine(Marshal.SizeOf(myS));  
   }  
}  
```
