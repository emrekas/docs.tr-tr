# <a name="compiler-error-cs0233"></a>Derleyici Hatası CS0233
' Identifier ' için önceden tanımlanmış bir boyut yok, bu nedenle sizeof yalnızca güvenli olmayan bir bağlamda kullanılabilir
  
 [Güvenli olmayan](../keywords/unsafe.md) bağlam olmadan, [sizeof](../../../csharp/language-reference/operators/sizeof.md) işleci yalnızca boyut derleme zamanı sabiti olan türler için kullanılabilir. Bu hatayı alıyorsanız güvenli olmayan bir bağlam kullanın.
  
Aşağıdaki örnek CS0233 oluşturur:
  
```csharp  
// CS0233.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Sequential)]  
public struct S  
{  
    public int a;  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        S myS = new S();  
        Console.WriteLine(sizeof(S));   // CS0233  
        // Try the following instead:  
        // unsafe
        // {
        //     Console.WriteLine(sizeof(S));
        // }
   }  
}  
```
