# <a name="compiler-error-cs0233"></a><span data-ttu-id="eb6aa-102">Derleyici Hatası CS0233</span><span class="sxs-lookup"><span data-stu-id="eb6aa-102">Compiler Error CS0233</span></span>
<span data-ttu-id="eb6aa-103">' Identifier ' için önceden tanımlanmış bir boyut yok, bu nedenle sizeof yalnızca güvenli olmayan bir bağlamda kullanılabilir</span><span class="sxs-lookup"><span data-stu-id="eb6aa-103">'identifier' does not have a predefined size, therefore sizeof can only be used in an unsafe context</span></span>
  
 <span data-ttu-id="eb6aa-104">[Güvenli olmayan](../keywords/unsafe.md) bağlam olmadan, [sizeof](../operators/sizeof.md) işleci yalnızca boyut derleme zamanı sabiti olan türler için kullanılabilir.</span><span class="sxs-lookup"><span data-stu-id="eb6aa-104">Without [unsafe](../keywords/unsafe.md) context, the [sizeof](../operators/sizeof.md) operator can only be used for types which size is a compile-time constant.</span></span> <span data-ttu-id="eb6aa-105">Bu hatayı alıyorsanız güvenli olmayan bir bağlam kullanın.</span><span class="sxs-lookup"><span data-stu-id="eb6aa-105">If you are getting this error, use an unsafe context.</span></span>
  
<span data-ttu-id="eb6aa-106">Aşağıdaki örnek CS0233 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="eb6aa-106">The following example generates CS0233:</span></span>
  
```csharp  
// CS0233.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Sequential)]  
public struct S  
{  
    public int a;  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        S myS = new S();  
        Console.WriteLine(sizeof(S));   // CS0233  
        // Try the following instead:  
        // unsafe
        // {
        //     Console.WriteLine(sizeof(S));
        // }
   }  
}  
```
