# <a name="compiler-warning-level-1-cs0465"></a><span data-ttu-id="f1cdb-102">Derleyici Uyarısı (düzey 1) CS0465</span><span class="sxs-lookup"><span data-stu-id="f1cdb-102">Compiler Warning (level 1) CS0465</span></span>
<span data-ttu-id="f1cdb-103">'Finalize' yönteminin sunulması yıkıcı çağrılmasını engelleyebilir.</span><span class="sxs-lookup"><span data-stu-id="f1cdb-103">Introducing a 'Finalize' method can interfere with destructor invocation.</span></span> <span data-ttu-id="f1cdb-104">Bir yıkıcı bildirmek mi istiyordunuz?</span><span class="sxs-lookup"><span data-stu-id="f1cdb-104">Did you intend to declare a destructor?</span></span>  
  
 <span data-ttu-id="f1cdb-105">İmzaya sahip, bir yöntemle bir sınıf oluşturduğunuzda bu uyarı oluşur `public virtual void Finalize`.</span><span class="sxs-lookup"><span data-stu-id="f1cdb-105">This warning occurs when you create a class with a method whose signature is `public virtual void Finalize`.</span></span>  
  
 <span data-ttu-id="f1cdb-106">Böyle bir sınıfın temel sınıf olarak kullanılan ve türetilen sınıf bir yıkıcı tanımlarsa, yıkıcı temel sınıfı geçersiz kılacak olan `Finalize` yöntemi değil <xref:System.Object.Finalize%2A>.</span><span class="sxs-lookup"><span data-stu-id="f1cdb-106">If such a class is used as a base class and if the deriving class defines a destructor, the destructor will override the base class `Finalize` method, not <xref:System.Object.Finalize%2A>.</span></span>  
  
## <a name="example"></a><span data-ttu-id="f1cdb-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="f1cdb-107">Example</span></span>  
 <span data-ttu-id="f1cdb-108">Aşağıdaki örnek, CS0465 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="f1cdb-108">The following sample generates CS0465.</span></span>  
  
```csharp  
// CS0465.cs  
// compile with: /target:library  
class A  
{  
   public virtual void Finalize() {}   // CS0465  
}  
  
// OK  
class B  
{  
   ~B() {}  
}  
```
