# <a name="compiler-error-cs1674"></a><span data-ttu-id="7370e-102">Derleyici Hatası CS1674</span><span class="sxs-lookup"><span data-stu-id="7370e-102">Compiler Error CS1674</span></span>
<span data-ttu-id="7370e-103">'T': bir using deyiminde kullanılan tür açıkça 'System.IDisposable' öğesine dönüştürülebilir olmalıdır</span><span class="sxs-lookup"><span data-stu-id="7370e-103">'T': type used in a using statement must be implicitly convertible to 'System.IDisposable'</span></span>  
  
 <span data-ttu-id="7370e-104">[Using deyimi](../../../csharp/language-reference/keywords/using-statement.md) sonunda bir nesnenin elden emin olmak için kullanılması amaçlanmıştır `using` engelleyin, bu nedenle, böyle bir deyimde yalnızca atılabilir türler kullanılabilir.</span><span class="sxs-lookup"><span data-stu-id="7370e-104">The [using Statement](../../../csharp/language-reference/keywords/using-statement.md) is intended to be used to ensure the disposal of an object at the end of the `using` block, thus, only types which are disposable may be used in such a statement.</span></span> <span data-ttu-id="7370e-105">Örneğin, değer türler, atılabilir değildir ve tür parametreleri, bir sınıf olarak kısıtlı değil atılabilir olması varsayılabilir değil.</span><span class="sxs-lookup"><span data-stu-id="7370e-105">For example, value types are not disposable, and type parameters which are not constrained to be classes may not be assumed to be disposable.</span></span>  
  
## <a name="example"></a><span data-ttu-id="7370e-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="7370e-106">Example</span></span>  
 <span data-ttu-id="7370e-107">Aşağıdaki örnek, CS1674 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="7370e-107">The following sample generates CS1674.</span></span>  
  
```csharp  
// CS1674.cs

   {  
      int a = 0;  
      a++;  
  
      using (a) {}   // CS1674  
   }

```  
  
## <a name="example"></a><span data-ttu-id="7370e-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="7370e-108">Example</span></span>  
 <span data-ttu-id="7370e-109">Aşağıdaki örnek, CS1674 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="7370e-109">The following sample generates CS1674.</span></span>  
  
```csharp  
// CS1674_b.cs  
using System;  
class C {  
   public void Test() {  
      using (C c = new C()) {}   // CS1674  
   }  
}  
  
// OK  
class D : IDisposable {  
   void IDisposable.Dispose() {}  
   public void Dispose() {}  
  
   public static void Main() {  
      using (D d = new D()) {}  
   }  
}  
```
