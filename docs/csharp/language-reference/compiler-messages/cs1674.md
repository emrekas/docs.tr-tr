# <a name="compiler-error-cs1674"></a><span data-ttu-id="a9063-102">Derleyici Hatası CS1674</span><span class="sxs-lookup"><span data-stu-id="a9063-102">Compiler Error CS1674</span></span>
<span data-ttu-id="a9063-103">'T ': Using ifadesinde kullanılan tür örtülü olarak ' System. IDisposable ' öğesine dönüştürülebilir olmalıdır</span><span class="sxs-lookup"><span data-stu-id="a9063-103">'T': type used in a using statement must be implicitly convertible to 'System.IDisposable'</span></span>  
  
 <span data-ttu-id="a9063-104">[Using ifadesinin](../keywords/using-statement.md) , `using` bloğunun sonundaki bir nesnenin aktiften çıkarılmasının sağlamak için kullanılması amaçlanmıştır, bu nedenle yalnızca atılabilir olan türler böyle bir bildirimde kullanılabilir.</span><span class="sxs-lookup"><span data-stu-id="a9063-104">The [using Statement](../keywords/using-statement.md) is intended to be used to ensure the disposal of an object at the end of the `using` block, thus, only types which are disposable may be used in such a statement.</span></span> <span data-ttu-id="a9063-105">Örneğin, değer türleri atılabilir değildir ve sınıfları olarak Kısıtlanmamış tür parametrelerinin atılabilir olduğu varsayılmayabilir.</span><span class="sxs-lookup"><span data-stu-id="a9063-105">For example, value types are not disposable, and type parameters which are not constrained to be classes may not be assumed to be disposable.</span></span>  
  
## <a name="example"></a><span data-ttu-id="a9063-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="a9063-106">Example</span></span>  
 <span data-ttu-id="a9063-107">Aşağıdaki örnek CS1674 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="a9063-107">The following sample generates CS1674.</span></span>

## <a name="example"></a><span data-ttu-id="a9063-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="a9063-108">Example</span></span>  
 <span data-ttu-id="a9063-109">Aşağıdaki örnek CS1674 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="a9063-109">The following sample generates CS1674.</span></span>  
  
```csharp  
// CS1674_b.cs  
using System;  
class C {  
   public void Test() {  
      using (C c = new C()) {}   // CS1674  
   }  
}  
  
// OK  
class D : IDisposable {  
   void IDisposable.Dispose() {}  
   public void Dispose() {}  
  
   public static void Main() {  
      using (D d = new D()) {}  
   }  
}  
```
