# <a name="compiler-error-cs0311"></a><span data-ttu-id="26761-102">Derleyici Hatası CS0311</span><span class="sxs-lookup"><span data-stu-id="26761-102">Compiler Error CS0311</span></span>

<span data-ttu-id="26761-103">'Type1' türü kullanılamaz, 'T tür parametresi olarak ' genel türünde veya yönteminde'içinde '\<adı >'.</span><span class="sxs-lookup"><span data-stu-id="26761-103">The type 'type1' cannot be used as type parameter 'T' in the generic type or method '\<name>'.</span></span> <span data-ttu-id="26761-104">'Type1' öğesinden 'type2' öğesine örtük bir başvuru dönüştürmesi yoktur.</span><span class="sxs-lookup"><span data-stu-id="26761-104">There is no implicit reference conversion from 'type1' to 'type2'.</span></span>  
  
 <span data-ttu-id="26761-105">Genel tür parametresi için bir kısıtlama uygulandığında, bir örtük kimlik veya başvuru dönüştürme sabit bağımsız değişkenden kısıtlama türü için mevcut olması gerekir.</span><span class="sxs-lookup"><span data-stu-id="26761-105">When a constraint is applied to a generic type parameter, an implicit identity or reference conversion must exist from the concrete argument to the type of the constraint.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="26761-106">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="26761-106">To correct this error</span></span>

2. <span data-ttu-id="26761-108">Sınıf sahipseniz, sınırlamayı kaldırmak veya başka bir örtük bir başvuru veya kimlik dönüştürme etkinleştirmek için bir şey.</span><span class="sxs-lookup"><span data-stu-id="26761-108">If you own the class, you can remove the constraint or else do something to enable an implicit reference or identity conversion.</span></span> <span data-ttu-id="26761-109">Örneğin, ikinci tür birinciden yapabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="26761-109">For example, you can make the second type inherit from the first.</span></span>  
  
## <a name="example"></a><span data-ttu-id="26761-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="26761-110">Example</span></span>  
  
```csharp  
// cs0311.cs  
class B {}  
class C {}  
class Test<T> where T : C  
{ }  
  
class Program  
{  
    static void Main()  
    {  
        Test<B> test = new Test<B>(); //CS0311  
    }  
}  
```  
  
 <span data-ttu-id="26761-111">Bir değer türü bağımsız değişkeni kullanmaya çalışırken bu hata meydana gelirse, dikkat örneğin örtük bir sayısal dönüştürme `short` için `int`, genel tür parametresi karşılamaz.</span><span class="sxs-lookup"><span data-stu-id="26761-111">If this error occurs when trying to use a value-type argument, notice that an implicit numeric conversion, for example from `short` to `int`, does not satisfy a generic type parameter.</span></span>
