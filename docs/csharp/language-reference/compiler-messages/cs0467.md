# <a name="compiler-warning-level-2-cs0467"></a><span data-ttu-id="d05ff-102">Derleyici Uyarısı (düzey 2) CS0467</span><span class="sxs-lookup"><span data-stu-id="d05ff-102">Compiler Warning (level 2) CS0467</span></span>
<span data-ttu-id="d05ff-103">'Method' yöntemi ile yöntem olmayan 'yöntem olmayan' arasında belirsizlik var.</span><span class="sxs-lookup"><span data-stu-id="d05ff-103">Ambiguity between method 'method' and non-method 'non-method'.</span></span> <span data-ttu-id="d05ff-104">Yöntem grubu kullanılıyor.</span><span class="sxs-lookup"><span data-stu-id="d05ff-104">Using method group.</span></span>

## <a name="example"></a><span data-ttu-id="d05ff-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="d05ff-106">Example</span></span>  
 <span data-ttu-id="d05ff-107">Aşağıdaki örnek, CS0467 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="d05ff-107">The following example generates CS0467.</span></span>  
  
```csharp  
// CS0467.cs  
interface IList   
{  
    int Count { get; set; }  
}  
  
interface ICounter  
{  
    void Count(int i);  
}  
  
interface IListCounter : IList, ICounter {}  
  
class Driver   
{  
    void Test(IListCounter x)  
    {  
        // The following line causes the warning. The assignment also  
        // causes an error because you can't assign a value to a method.  
        x.Count = 1;  
        x.Count(3);     
        // To resolve the warning, you can change the name of the method or   
        // the property.  
  
        // You also can disambiguate by specifying IList or ICounter.  
        ((IList)x).Count = 1;  
        ((ICounter)x).Count(3);  
    }  
  
    static void Main()   
    {  
    }  
}  
```
