<span data-ttu-id="74f69-103">'Type2' iç içe türü yoluyla 'type1' dış türündeki statik olmayan üye erişemiyor</span><span class="sxs-lookup"><span data-stu-id="74f69-103">Cannot access a nonstatic member of outer type 'type1' via nested type 'type2'</span></span>  
  
 <span data-ttu-id="74f69-104">Bir alan bir sınıf içinde iç içe geçmiş bir sınıf için otomatik olarak kullanılabilir değil.</span><span class="sxs-lookup"><span data-stu-id="74f69-104">A field in a class is not automatically available to a nested class.</span></span> <span data-ttu-id="74f69-105">İç içe geçmiş bir sınıf için kullanılabilir olması için alan olmalıdır [statik](../../../csharp/language-reference/keywords/static.md).</span><span class="sxs-lookup"><span data-stu-id="74f69-105">To be available to a nested class, the field must be [static](../../../csharp/language-reference/keywords/static.md).</span></span> <span data-ttu-id="74f69-106">Aksi takdirde, dış sınıfının bir örneğini oluşturmanız gerekir.</span><span class="sxs-lookup"><span data-stu-id="74f69-106">Otherwise, you must create an instance of the outer class.</span></span> <span data-ttu-id="74f69-107">Daha fazla bilgi için [iç içe türler](../../../csharp/programming-guide/classes-and-structs/nested-types.md).</span><span class="sxs-lookup"><span data-stu-id="74f69-107">For more information, see [Nested Types](../../../csharp/programming-guide/classes-and-structs/nested-types.md).</span></span>  
  
 <span data-ttu-id="74f69-108">Aşağıdaki örnek, CS0038 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="74f69-108">The following sample generates CS0038:</span></span>  
  
```csharp  
// CS0038.cs  
class OuterClass  
{  
   public int count;  
   // try the following line instead  
   // public static int count;  
  
   class InnerClass  
   {  
      void func()  
      {  
         // or, create an instance  
         // OuterClass class_inst = new OuterClass();  
         // int count2 = class_inst.count;  
         int count2 = count;   // CS0038  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
