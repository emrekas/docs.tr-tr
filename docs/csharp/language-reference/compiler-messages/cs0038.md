 <span data-ttu-id="4889d-104">Sınıftaki bir alan, iç içe yerleştirilmiş bir sınıfta otomatik olarak kullanılamaz.</span><span class="sxs-lookup"><span data-stu-id="4889d-104">A field in a class is not automatically available to a nested class.</span></span> <span data-ttu-id="4889d-105">İç içe yerleştirilmiş bir sınıfta kullanılabilmesi için alanın [statik](../keywords/static.md)olması gerekir.</span><span class="sxs-lookup"><span data-stu-id="4889d-105">To be available to a nested class, the field must be [static](../keywords/static.md).</span></span> <span data-ttu-id="4889d-106">Aksi takdirde, dış sınıfın bir örneğini oluşturmanız gerekir.</span><span class="sxs-lookup"><span data-stu-id="4889d-106">Otherwise, you must create an instance of the outer class.</span></span> <span data-ttu-id="4889d-107">Daha fazla bilgi için bkz. [Iç Içe türler](../../programming-guide/classes-and-structs/nested-types.md).</span><span class="sxs-lookup"><span data-stu-id="4889d-107">For more information, see [Nested Types](../../programming-guide/classes-and-structs/nested-types.md).</span></span>  
  
 <span data-ttu-id="4889d-108">Aşağıdaki örnek CS0038 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="4889d-108">The following sample generates CS0038:</span></span>  
  
```csharp  
// CS0038.cs  
class OuterClass  
{  
   public int count;  
   // try the following line instead  
   // public static int count;  
  
   class InnerClass  
   {  
      void func()  
      {  
         // or, create an instance  
         // OuterClass class_inst = new OuterClass();  
         // int count2 = class_inst.count;  
         int count2 = count;   // CS0038  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
