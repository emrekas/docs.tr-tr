# <a name="compiler-error-cs0504"></a><span data-ttu-id="557d0-102">Derleyici Hatası CS0504</span><span class="sxs-lookup"><span data-stu-id="557d0-102">Compiler Error CS0504</span></span>
<span data-ttu-id="557d0-103">'Variable' sabiti static olarak işaretlenemez</span><span class="sxs-lookup"><span data-stu-id="557d0-103">The constant 'variable' cannot be marked static</span></span>  
  
 <span data-ttu-id="557d0-104">Bir değişken ise [const](../../../csharp/language-reference/keywords/const.md), ayrıca [statik](../../../csharp/language-reference/keywords/static.md).</span><span class="sxs-lookup"><span data-stu-id="557d0-104">If a variable is [const](../../../csharp/language-reference/keywords/const.md), it is also [static](../../../csharp/language-reference/keywords/static.md).</span></span> <span data-ttu-id="557d0-105">İsterseniz bir **const** ve **statik** değişkeni, yalnızca o değişken olarak bildirin **const**; tüm olan istiyorsanız bir **statik** değişkeni, yalnızca olarak işaretleyin **statik**.</span><span class="sxs-lookup"><span data-stu-id="557d0-105">If you want a **const** and **static** variable, just declare that variable as **const**; if all you want is a **static** variable, just mark it **static**.</span></span>  
  
 <span data-ttu-id="557d0-106">Aşağıdaki örnek, CS0504 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="557d0-106">The following sample generates CS0504:</span></span>  
  
```csharp  
// CS0504.cs  
namespace x  
{  
   abstract public class clx  
   {  
      static const int i = 0;   // CS0504, cannot be both static and const  
      abstract public void f();  
   }  
}  
```
