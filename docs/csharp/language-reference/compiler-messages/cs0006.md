# <a name="compiler-error-cs0006"></a>Derleyici Hatası CS0006
Meta veri dosyası 'dll_name' bulunamadı.  
  
 Program derlendi ve meta verileri içeren bir dosya adı açıkça geçirilen; Ancak, .dll bulunamadı. Daha fazla bilgi için [/Reference (C# Derleyici Seçenekleri)](../../../csharp/language-reference/compiler-options/reference-compiler-option.md).
