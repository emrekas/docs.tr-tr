# <a name="compiler-error-cs0571"></a><span data-ttu-id="992d9-102">Derleyici Hatası CS0571</span><span class="sxs-lookup"><span data-stu-id="992d9-102">Compiler Error CS0571</span></span>
<span data-ttu-id="992d9-103">'function': açıkça işleç veya erişimci çağıramaz</span><span class="sxs-lookup"><span data-stu-id="992d9-103">'function' : cannot explicitly call operator or accessor</span></span>  
  
 <span data-ttu-id="992d9-104">Belirli işleçleri iç adları vardır.</span><span class="sxs-lookup"><span data-stu-id="992d9-104">Certain operators have internal names.</span></span> <span data-ttu-id="992d9-105">Örneğin, **op_Increment** iç adı ++ işleci.</span><span class="sxs-lookup"><span data-stu-id="992d9-105">For example, **op_Increment** is the internal name of the ++ operator.</span></span> <span data-ttu-id="992d9-106">Değil kullanın veya bu yöntem adları açıkça çağırmanız gerekir.</span><span class="sxs-lookup"><span data-stu-id="992d9-106">You should not use or explicitly call such method names.</span></span>  
  
 <span data-ttu-id="992d9-107">Aşağıdaki örnek CS0571 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="992d9-107">The following sample generates CS0571:</span></span>  
  
```csharp  
// CS0571.cs  
public class MyClass  
{  
   public static MyClass operator ++ (MyClass c)  
   {  
      return null;  
   }  
  
   public static int prop  
   {  
      get  
      {  
         return 1;  
      }  
      set  
      {  
      }  
   }  
  
   public static void Main()  
   {  
      op_Increment(null);   // CS0571  
      // use the increment operator as follows  
      // MyClass x = new MyClass();  
      // x++;  
  
      set_prop(1);      // CS0571  
      // try the following line instead  
      // prop = 1;  
   }  
}  
```
