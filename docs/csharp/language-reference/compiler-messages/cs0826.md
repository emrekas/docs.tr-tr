 <span data-ttu-id="3cdfe-104">Dizi öğeleri, derleyicinin kullandığı tür çıkarımı kurallarına göre aynı türde veya örtülü olarak dönüştürülebilir olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="3cdfe-104">Array elements must all be the same type or implicitly convertible to the same type according to the type inference rules used by the compiler.</span></span> <span data-ttu-id="3cdfe-105">En iyi tür, dizi ifadesinde bulunan türlerden biri olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="3cdfe-105">The best type must be one of the types present in the array expression.</span></span> <span data-ttu-id="3cdfe-106">Öğeler gibi yeni bir türe `object`dönüştürülmeyecektir.</span><span class="sxs-lookup"><span data-stu-id="3cdfe-106">Elements will not be converted to a new type such as `object`.</span></span> <span data-ttu-id="3cdfe-107">Örtük olarak yazılan bir dizi için, derleyicinin atanmış öğelerin türüne göre dizi türünü çıkarması gerekir.</span><span class="sxs-lookup"><span data-stu-id="3cdfe-107">For an implicitly typed array, the compiler must infer the array type based on the type of elements assigned to it.</span></span>

## <a name="example"></a><span data-ttu-id="3cdfe-112">Örnek</span><span class="sxs-lookup"><span data-stu-id="3cdfe-112">Example</span></span>  
 <span data-ttu-id="3cdfe-113">Aşağıdaki kod CS0826 oluşturur, çünkü dizi öğeleri aynı türde değildir ve derleyicinin tür çıkarımı mantığı tek bir en iyi tür bulmaz:</span><span class="sxs-lookup"><span data-stu-id="3cdfe-113">The following code generates CS0826 because the array elements are not all the same type, and the compiler's type inference logic does not find a single best type:</span></span>  
  
```csharp  
// cs0826.cs  
public class C  
{  
    public static int Main()  
    {  
        var x = new[] { 1, "str" }; // CS0826  
  
        char c = 'c';  
        short s1 = 0;  
        short s2 = -0;  
        short s3 = 1;  
        short s4 = -1;  
  
        var array1 = new[] { s1, s2, s3, s4, c, '1' }; // CS0826  
        return 1;  
    }  
}  
```
