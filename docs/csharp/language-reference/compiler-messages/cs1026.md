# <a name="compiler-error-cs1026"></a>Derleyici Hatası CS1026
) bekleniyor  
  
 Tamamlanmamış bir deyim bulunamadı.  
  
 Bu hatanın yaygın bir nedeni, bir ASP.NET sayfasında bir satır içi ifade içindeki bir ifadenin yerine bir deyim yerleştiriyor. Örneğin, yanlış verilmiştir:  
  
```aspx-csharp  
<%=new TimeSpan(DateTime.Now.Ticks - new DateTime(2001, 1, 1).Ticks).Days;%>  
```  
  
 Doğru verilmiştir:  
  
```aspx-csharp  
<%=new TimeSpan(DateTime.Now.Ticks - new DateTime(2001, 1, 1).Ticks).Days %>  
```  
  
 Şu şekilde yorumlanır:  
  
```aspx-csharp  
<% Response.Write(new TimeSpan(DateTime.Now.Ticks - new DateTime(2001, 1, 1).Ticks).Days); %>  
```  
  
 Aşağıdaki örnek, CS1026 oluşturur:  
  
```csharp  
// CS1026.cs  
#if (a == b   // CS1026, add closing )  
#endif  
  
class x  
{  
   public static void Main()  
   {  
   }  
}  
```
