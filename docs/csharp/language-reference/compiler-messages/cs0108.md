# <a name="compiler-warning-level-2-cs0108"></a>Derleyici Uyarısı (düzey 2) CS0108

'Üye1', 'üye2' devralınan üyesini gizler. Gizlemek istiyorsanız new anahtar sözcüğünü kullanmak üzere tasarlanmıştır.  
  
 Bir değişken bir temel sınıf olarak aynı ada sahip bir değişken bildirildi. Ancak, [yeni](../../../csharp/language-reference/keywords/new-modifier.md) anahtar sözcüğü değil kullanıldı. Bu uyarı, kullanmanız gerektiğini bildirir **yeni**; değişkenin bildirildiği gibi **yeni** bildiriminde kullanılır.  
  
 Aşağıdaki örnek, CS0108 oluşturur:  

```csharp
// CS0108.cs  
// compile with: /W:2  
using System;  
  
namespace x  
{  
   public class clx  
   {  
      public int i = 1;  
   }  
  
   public class cly : clx  
   {  
      public static int i = 2;   // CS0108, use the new keyword  
      // Use the following line instead:  
      // public static new int i = 2;  
  
      public static void Main()  
      {  
         Console.WriteLine(i);  
      }  
   }  
}  
```
