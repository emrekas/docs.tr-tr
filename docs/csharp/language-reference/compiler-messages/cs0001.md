# <a name="compiler-error-cs0001"></a>Derleyici Hatası CS0001
Derleyici iç hatası  
  
 Derleyici, bağlanamama beklenmeyen söz dizimi ayrıştırma nedeniyle başarısız olduğunu anlamaya çalışın. Bu hata tekrarlanarak alırsanız, lütfen Microsoft'a başvurun.
