# <a name="compiler-error-cs0201"></a>Derleyici Hatası CS0201

Yalnızca atama, çağrı, artırma, azaltma ve yeni nesne ifadeleri deyim olarak kullanılabilir  
  
 Geçersiz bir deyim ile karşılaştığında derleyici bir hata oluşturur. Herhangi bir satırı veya satırları atama göstermiyor bir noktalı virgül ile biten bir dizi geçersiz bir deyim olduğu ([=](../operators/assignment-operator.md)), yöntem çağrısının [()](../operators/invocation-operator.md), [yeni](../keywords/new.md), [ -- ](../operators/arithmetic-operators.md#decrement-operator---) veya [ ++ ](../operators/arithmetic-operators.md#increment-operator-) işlemi. Daha fazla bilgi için [deyimler, ifadeler ve işleçler](../../programming-guide/statements-expressions-operators/index.md).  
  
## <a name="example"></a>Örnek

 Aşağıdaki örnek, 2 * 3 deyim olmayan bir ifade olduğundan CS0201 oluşturur. Kodu derlemek için ifade değeri bir değişkene atayarak deneyin.  

```csharp
// CS0201.cs  
public class MainClass  
{  
   public static void Main()  
   {  
      2 * 3;   // CS0201  
      // Try the following line instead.  
      //   int i = 2 * 3;  
   }  
}  
```

## <a name="example"></a>Örnek

 Aşağıdaki örnek CS0201 oluşturur çünkü tarafından denetlenen bir Artım işlemi tarafından parametre haline rağmen kendisini bir deyimi değil.  

```csharp
// CS0201_b.cs  
// compile with: /target:library  
public class MyList<T>
{  
   public void Add(T x)  
   {  
      int i = 0;  
      if ( (object)x == null)  
      {  
         checked(i++);   // CS0201  
  
         // OK  
         checked {  
            i++;
         }  
      }  
   }  
}  
```
