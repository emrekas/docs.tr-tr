# <a name="compiler-error-cs1019"></a><span data-ttu-id="20eb8-102">Derleyici Hatası CS1019</span><span class="sxs-lookup"><span data-stu-id="20eb8-102">Compiler Error CS1019</span></span>
<span data-ttu-id="20eb8-103">Aşırı yüklenebilir birli işleç bekleniyor</span><span class="sxs-lookup"><span data-stu-id="20eb8-103">Overloadable unary operator expected</span></span>  
  
 <span data-ttu-id="20eb8-104">Aşırı yüklenmiş birli operatör gibi görünen bir şey bildirilmiştir, ancak işleç eksik veya İmzada yanlış konumda.</span><span class="sxs-lookup"><span data-stu-id="20eb8-104">Something that looks like an overloaded unary operator has been declared, but the operator is missing or is in the wrong location in the signature.</span></span>  
  
 <span data-ttu-id="20eb8-105">*Birli işleç* , tek bir işlenen üzerinde çalışan bir işleçtir.</span><span class="sxs-lookup"><span data-stu-id="20eb8-105">A *unary operator* is an operator that operates on a single operand.</span></span> <span data-ttu-id="20eb8-106">Örneğin, `++` birli bir işleçtir.</span><span class="sxs-lookup"><span data-stu-id="20eb8-106">For example, `++` is a unary operator.</span></span> <span data-ttu-id="20eb8-107">`operator` Anahtar sözcüğünü kullanarak ve işlecin üzerinde çalıştığı türün tek bir parametresini belirterek bazı birli işleçleri aşırı yükleyebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="20eb8-107">You can overload some unary operators by using the `operator` keyword and specifying a single parameter of the type that the operator operates on.</span></span> <span data-ttu-id="20eb8-108">Örneğin, yazmak `++` `Temperature` üzere`Temperature++`Kullanıcı tanımlı bir sınıfın işlecini aşırı yüklemek istiyorsanız, bu şekilde tanımlayabilirsiniz:</span><span class="sxs-lookup"><span data-stu-id="20eb8-108">For example, if you want to overload the operator `++` for a user-defined class `Temperature` so that you can write `Temperature++`, you can define it in this way:</span></span>  
  
```csharp  
public static  Temperature operator ++ (Temperature temp)  
{  
    temp.Degrees++;  
    return temp;  
}  
```  
  
 <span data-ttu-id="20eb8-109">Bu hatayı aldığınızda, bir aşırı yüklenmiş birli işleç gibi görünen bir şey, işlecin kendisi eksik olması veya İmzada yanlış konumda olması gerekir.</span><span class="sxs-lookup"><span data-stu-id="20eb8-109">When you receive this error, you have declared something that looks like an overloaded unary operator, except that the operator itself is missing or is in the wrong location in the signature.</span></span> <span data-ttu-id="20eb8-110">Önceki örnekteki imzadan kaldırırsanız `++` , CS1019 oluşturacaksınız.</span><span class="sxs-lookup"><span data-stu-id="20eb8-110">If you remove the `++` from the signature in the previous example, you will generate CS1019.</span></span>  
  
 <span data-ttu-id="20eb8-111">Aşağıdaki kod CS1019 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="20eb8-111">The following code generates CS1019:</span></span>  
  
```csharp  
// CS1019.cs  
public class ii  
{  
   int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
}  
  
public class a  
{  
    public int i;  
// Generates CS1019: "ii" is not a unary operator.  
   public static a operator ii(a aa)     
  
   // Use the following line instead:  
   //public static a operator ++(a aa)  
   {  
      aa.i++;  
      return aa;   
   }  
  
   public static void Main()  
   {  
   }  
}  
```
