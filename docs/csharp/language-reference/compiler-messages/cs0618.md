# <a name="compiler-warning-level-2-cs0618"></a><span data-ttu-id="c4d34-102">Derleyici Uyarısı (düzey 2) CS0618</span><span class="sxs-lookup"><span data-stu-id="c4d34-102">Compiler Warning (level 2) CS0618</span></span>
<span data-ttu-id="c4d34-103">'member' artık kullanılmıyor: 'text'</span><span class="sxs-lookup"><span data-stu-id="c4d34-103">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="c4d34-104">İle işaretlenmiş bir sınıf üyesinin `Obsolete` özniteliği sağlayacak şekilde sınıf üyesi atıfta bulunulduğunda bir uyarı verilir.</span><span class="sxs-lookup"><span data-stu-id="c4d34-104">A class member was marked with the `Obsolete` attribute, such that a warning will be issued when the class member is referenced.</span></span> <span data-ttu-id="c4d34-105">Daha fazla bilgi için [ortak öznitelikleri](../../programming-guide/concepts/attributes/common-attributes.md).</span><span class="sxs-lookup"><span data-stu-id="c4d34-105">For more information, see [Common Attributes](../../programming-guide/concepts/attributes/common-attributes.md).</span></span>  
  
 <span data-ttu-id="c4d34-106">Aşağıdaki örnek, CS0618 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="c4d34-106">The following sample generates CS0618:</span></span>  
  
```csharp  
// CS0618.cs  
// compile with: /W:2  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", false)]   // warn if referenced  
   public static void m2()  
   {  
   }  
  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m2();  // CS0618  
   }  
}  
```
