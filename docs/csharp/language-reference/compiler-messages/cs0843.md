# <a name="compiler-error-cs0843"></a>Derleyici Hatası CS0843
Yedekleme alanını otomatik olarak uygulanan 'name' özelliği için Denetim çağırana döndürülmeden önce tamamen atanmalıdır. Bir oluşturucu başlatıcısından varsayılan oluşturucuyu çağırmayı düşünün.  
  
 Oluşturucudan otomatik olarak uygulanan bir özellik için bir değer atamak için nesneyi oluşturmak için varsayılan oluşturucu çağırmanız gerekir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Aşağıdaki örnekte gösterildiği gibi bir oluşturucu başlatıcı varsayılan oluşturucusuna bir çağrı ekleyin. Kullanımına dikkat edin `: this()`. Daha fazla bilgi için [bu](../../../csharp/language-reference/keywords/this.md).  
  
## <a name="example"></a>Örnek  
 Aşağıdaki kod, CS0843 oluşturur:  
  
```csharp  
// cs0843.cs  
struct S  
{  
    public int AIProp { get; set; }  
    public S(int i){} //CS0843  
    // Try the following lines instead.  
   // public S(int i) : this()  
   // {  
   //     AIProp = i;  
   // }  
}  
  
class Test  
{  
    static int Main()  
    {  
        return 1;  
    }  
}  
```
