# <a name="compiler-error-cs0445"></a><span data-ttu-id="fbca5-102">Derleyici Hatası CS0445</span><span class="sxs-lookup"><span data-stu-id="fbca5-102">Compiler Error CS0445</span></span>
<span data-ttu-id="fbca5-103">Bir paket açma dönüşümünün sonucu değiştirilemez</span><span class="sxs-lookup"><span data-stu-id="fbca5-103">Cannot modify the result of an unboxing conversion</span></span>  
  
 <span data-ttu-id="fbca5-104">Bir paket açma dönüşümünün sonucu geçici bir değişkendir.</span><span class="sxs-lookup"><span data-stu-id="fbca5-104">The result of an unboxing conversion is a temporary variable.</span></span> <span data-ttu-id="fbca5-105">Derleyicinin geçici değişken kaybolduğunda değişiklikler kaybolacağından gibi değişkenleri değiştirmesini engeller.</span><span class="sxs-lookup"><span data-stu-id="fbca5-105">The compiler prevents you from modifying such variables because any modification would go away when the temporary variable goes away.</span></span> <span data-ttu-id="fbca5-106">Bu sorunu gidermek için ara ifade depolamak için yeni bir değer türündeki değişken bildirimini ve bu değişken için kutudan çıkarma dönüştürme sonucu olarak atayın.</span><span class="sxs-lookup"><span data-stu-id="fbca5-106">To fix this, declare a new value-type variable to store the intermediate expression, and assign the result of the unboxing conversion to that variable.</span></span>  
  
 <span data-ttu-id="fbca5-107">Aşağıdaki kodu CS0455 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="fbca5-107">The following code generates CS0455.</span></span>  
  
```csharp  
// CS0445.CS  
class UnboxingTest  
{  
    public static void Main()  
    {  
        Point p;  
        p.x = 1;  
        p.y = 2;  
        object obj = p;  
        // The following line generates CS0445, because the result  
        // of unboxing obj is a temporary variable.  
        ((Point)obj).x = 2;  
  
        // The following lines resolve the error.  
  
        // Store the result of the unboxing conversion in p2.  
        Point p2;       
        p2 = (Point)obj;  
        // Then you can modify the unboxed value.  
        p2.x = 2;  
    }  
}  
  
struct Point  
{  
    public int x, y;  
}  
```
