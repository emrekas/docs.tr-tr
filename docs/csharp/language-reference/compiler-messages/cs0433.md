## <a name="example"></a><span data-ttu-id="0391d-114">Örnek</span><span class="sxs-lookup"><span data-stu-id="0391d-114">Example</span></span>  
 <span data-ttu-id="0391d-115">Aşağıdaki örnek, CS0433 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="0391d-115">The following example generates CS0433.</span></span>
