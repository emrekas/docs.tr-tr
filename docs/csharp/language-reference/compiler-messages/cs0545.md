# <a name="compiler-error-cs0545"></a><span data-ttu-id="56641-102">Derleyici Hatası CS0545</span><span class="sxs-lookup"><span data-stu-id="56641-102">Compiler Error CS0545</span></span>

 <span data-ttu-id="56641-104">Temel sınıfı, geçersiz kılmak için tür tanımı varsa, bir özellik erişimcisi için bir geçersiz kılma tanımlamak için bir deneme yapıldı.</span><span class="sxs-lookup"><span data-stu-id="56641-104">A try was made to define an override for a property accessor when the base class has no such definition to override.</span></span> <span data-ttu-id="56641-105">Tarafından bu hatayı çözebilirsiniz:</span><span class="sxs-lookup"><span data-stu-id="56641-105">You can resolve this error by:</span></span>  
  
- <span data-ttu-id="56641-106">Ekleme bir `set` erişimci temel sınıf.</span><span class="sxs-lookup"><span data-stu-id="56641-106">Adding a `set` accessor in the base class.</span></span>  
  
- <span data-ttu-id="56641-107">Kaldırma `set` türetilmiş sınıftaki erişimcisi.</span><span class="sxs-lookup"><span data-stu-id="56641-107">Removing the `set` accessor from the derived class.</span></span>  
  
- <span data-ttu-id="56641-108">Temel sınıf özelliğini ekleyerek gizleme [yeni](../../../csharp/language-reference/keywords/new-modifier.md) türetilen bir sınıfta bir özellik için anahtar sözcüğü.</span><span class="sxs-lookup"><span data-stu-id="56641-108">Hiding the base class property by adding the [new](../../../csharp/language-reference/keywords/new-modifier.md) keyword to a property in a derived class.</span></span>

 <span data-ttu-id="56641-110">Daha fazla bilgi için [kullanarak özelliklerini](../../../csharp/programming-guide/classes-and-structs/using-properties.md).</span><span class="sxs-lookup"><span data-stu-id="56641-110">For more information, see [Using Properties](../../../csharp/programming-guide/classes-and-structs/using-properties.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="56641-111">Örnek</span><span class="sxs-lookup"><span data-stu-id="56641-111">Example</span></span>  
 <span data-ttu-id="56641-112">Aşağıdaki örnek, CS0545 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="56641-112">The following sample generates CS0545.</span></span>  
  
```csharp  
// CS0545.cs  
// compile with: /target:library  
// CS0545  
public class a  
{  
   public virtual int i  
   {  
      set {}  
  
      // Uncomment the following line to resolve.  
      // get { return 0; }  
   }  
}  
  
public class b : a  
{  
   public override int i  
   {  
      get { return 0; }  
      set {}   // OK  
   }  
}  
```
