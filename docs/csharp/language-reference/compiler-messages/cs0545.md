 <span data-ttu-id="04284-104">Temel sınıfın geçersiz kılmak için böyle bir tanımı olmadığında, özellik erişimcisi için bir geçersiz kılma tanımlamak için bir deneme yapıldı.</span><span class="sxs-lookup"><span data-stu-id="04284-104">A try was made to define an override for a property accessor when the base class has no such definition to override.</span></span> <span data-ttu-id="04284-105">Bu hatayı şu şekilde çözebilirsiniz:</span><span class="sxs-lookup"><span data-stu-id="04284-105">You can resolve this error by:</span></span>

## <a name="example"></a><span data-ttu-id="04284-111">Örnek</span><span class="sxs-lookup"><span data-stu-id="04284-111">Example</span></span>  
 <span data-ttu-id="04284-112">Aşağıdaki örnek CS0545 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="04284-112">The following sample generates CS0545.</span></span>  
  
```csharp  
// CS0545.cs  
// compile with: /target:library  
// CS0545  
public class a  
{  
   public virtual int i  
   {  
      set {}  
  
      // Uncomment the following line to resolve.  
      // get { return 0; }  
   }  
}  
  
public class b : a  
{  
   public override int i  
   {  
      get { return 0; }  
      set {}   // OK  
   }  
}  
```
