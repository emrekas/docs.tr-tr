 <span data-ttu-id="dafba-104">Tüm alanları bir `struct` Oluşturucusu bir yöntemin çağrı yapmadan önce Oluşturucu tarafından atanacak olan `struct`.</span><span class="sxs-lookup"><span data-stu-id="dafba-104">All fields in a `struct` have to be assigned by a constructor before the constructor can call a method in the `struct`.</span></span>  
  
 <span data-ttu-id="dafba-105">Bir yapı Oluşturucu özelliğinde başlatmaya çalışırken bu hatayı görürseniz, çözüm özellik yerine destek alanı belirtmek için oluşturucu parametresi değiştirmektir.</span><span class="sxs-lookup"><span data-stu-id="dafba-105">If you see this error when trying to initialize a property in a struct constructor, the solution is to change the constructor parameter to specify the backing field instead of the property itself.</span></span> <span data-ttu-id="dafba-106">Hiçbir destek alanı olan ve bu nedenle herhangi bir şekilde oluşturucudan başlatılamaz çünkü otomatik uygulanan özellikler yapılarda kaçınılmalıdır.</span><span class="sxs-lookup"><span data-stu-id="dafba-106">Auto-implemented properties should be avoided in structs because they have no backing field and therefore cannot be initialized in any way from the constructor.</span></span>

## <a name="example"></a><span data-ttu-id="dafba-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="dafba-108">Example</span></span>

 <span data-ttu-id="dafba-109">Aşağıdaki örnek, CS0188 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="dafba-109">The following sample generates CS0188:</span></span>  

```csharp
// CS0188.cs  
// compile with: /t:library  
namespace MyNamespace  
{  
    class MyClass  
    {  
        struct S  
        {  
            public int a;  
  
            void MyMethod()  
            {  
            }  
  
            S(int i)  
            {  
                // a = i;  
                MyMethod();  // CS0188  
            }  
        }  
        public static void Main()  
        { }  
  
    }  
}  
```
