 <span data-ttu-id="71284-104">Oluşturucusunun içindeki bir yöntemi `struct` çağırabilmesi için, `struct`içindeki tüm alanların bir Oluşturucu tarafından atanması gerekir.</span><span class="sxs-lookup"><span data-stu-id="71284-104">All fields in a `struct` have to be assigned by a constructor before the constructor can call a method in the `struct`.</span></span>  
  
 <span data-ttu-id="71284-105">Bir struct yapıcısında bir özelliği başlatmaya çalışırken bu hatayı görürseniz, çözüm, özelliğin kendisi yerine, destek alanını belirtmek için Oluşturucu parametresini değiştirmek olur.</span><span class="sxs-lookup"><span data-stu-id="71284-105">If you see this error when trying to initialize a property in a struct constructor, the solution is to change the constructor parameter to specify the backing field instead of the property itself.</span></span> <span data-ttu-id="71284-106">Hiçbir yedekleme alanı olmadığından ve bu nedenle oluşturucudan herhangi bir şekilde başlatılamadığından, otomatik olarak uygulanan özelliklerin yapılarda kaçınılması gerekir.</span><span class="sxs-lookup"><span data-stu-id="71284-106">Auto-implemented properties should be avoided in structs because they have no backing field and therefore cannot be initialized in any way from the constructor.</span></span>

## <a name="example"></a><span data-ttu-id="71284-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="71284-108">Example</span></span>

 <span data-ttu-id="71284-109">Aşağıdaki örnek CS0188 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="71284-109">The following sample generates CS0188:</span></span>  

```csharp
// CS0188.cs  
// compile with: /t:library  
namespace MyNamespace  
{  
    class MyClass  
    {  
        struct S  
        {  
            public int a;  
  
            void MyMethod()  
            {  
            }  
  
            S(int i)  
            {  
                // a = i;  
                MyMethod();  // CS0188  
            }  
        }  
        public static void Main()  
        { }  
  
    }  
}  
```
