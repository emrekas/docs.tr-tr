<span data-ttu-id="5b728-103">'Struct2 field' türü 'struct1' yapı üyesi yapı düzeninde döngüye neden oluyor</span><span class="sxs-lookup"><span data-stu-id="5b728-103">Struct member 'struct2 field' of type 'struct1' causes a cycle in the struct layout</span></span>

 <span data-ttu-id="5b728-108">Aşağıdaki örnek, CS0523 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="5b728-108">The following sample generates CS0523:</span></span>

```csharp  
   public RecursiveLayoutStruct1 field;   // CS0523  
}  
```
