# <a name="compiler-warning-level-1-cs1690"></a><span data-ttu-id="0c815-102">Derleyici Uyarısı (düzey 1) CS1690</span><span class="sxs-lookup"><span data-stu-id="0c815-102">Compiler Warning (level 1) CS1690</span></span>
<span data-ttu-id="0c815-103">Bir başvuruya göre sıralama sınıfının alanı olduğundan, 'member' üzerindeki bir üyeye erişilmesi çalışma zamanı özel durumuna neden olabilir</span><span class="sxs-lookup"><span data-stu-id="0c815-103">Accessing a member on 'member' may cause a runtime exception because it is a field of a marshal-by-reference class</span></span>  
  
 <span data-ttu-id="0c815-104">Bu uyarı, türetilen bir sınıfın üyesi bir yöntem, özellik veya dizin oluşturucu çağırmak çalıştığınızda oluşur <xref:System.MarshalByRefObject>, ve üye bir değer türüdür.</span><span class="sxs-lookup"><span data-stu-id="0c815-104">This warning occurs when you try to call a method, property, or indexer on a member of a class that derives from <xref:System.MarshalByRefObject>, and the member is a value type.</span></span> <span data-ttu-id="0c815-105">Devralınan nesnelerin `MarshalByRefObject` genellikle uygulama etki alanı başvuruya göre sıralanması yöneliktir.</span><span class="sxs-lookup"><span data-stu-id="0c815-105">Objects that inherit from `MarshalByRefObject` are typically intended to be marshaled by reference across an application domain.</span></span> <span data-ttu-id="0c815-106">Herhangi bir kod hiç olmadığı kadar değer türü üyesine böyle bir nesnenin uygulama etki alanı arasında doğrudan erişmeyi denerse bir çalışma zamanı özel durumu oluşur.</span><span class="sxs-lookup"><span data-stu-id="0c815-106">If any code ever attempts to directly access the value-type member of such an object across an application domain, a runtime exception will occur.</span></span> <span data-ttu-id="0c815-107">Uyarıyı çözmek için öncelikle üyeyi yerel bir değişkene kopyalayın ve bu değişkende yöntemi çağırın.</span><span class="sxs-lookup"><span data-stu-id="0c815-107">To resolve the warning, first copy the member into a local variable and call the method on that variable.</span></span>  
  
 <span data-ttu-id="0c815-108">Aşağıdaki örnek, CS1690 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="0c815-108">The following sample generates CS1690:</span></span>  
  
```csharp  
// CS1690.cs  
using System;  
  
class WarningCS1690: MarshalByRefObject  
{  
   int i = 5;  
  
   public static void Main()   
   {  
     WarningCS1690 e = new WarningCS1690();  
     e.i.ToString();   // CS1690  
  
     // OK  
     int i = e.i;  
     i.ToString();  
     e.i = i;  
   }  
}  
```
