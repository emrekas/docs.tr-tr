# <a name="compiler-warning-level-1-cs3003"></a>Derleyici Uyarısı (düzey 1) CS3003
'Değişkeninin' türü CLS uyumlu değil  
  
 A [genel](../../../csharp/language-reference/keywords/public.md), [korumalı](../../../csharp/language-reference/keywords/protected.md), veya `protected internal` değişken ortak dil belirtimi (CLS) ile uyumlu bir türde olması gerekir. CLS uyumluluğu hakkında daha fazla bilgi için bkz. [dil bağımsızlığı ve dilden bağımsız bileşenler](../../../standard/language-independence-and-language-independent-components.md).  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS3003 oluşturur:  
  
```csharp  
// CS3003.cs  
  
[assembly:System.CLSCompliant(true)]  
public class a  
{  
    public ushort a1;   // CS3003, public variable  
    public static void Main()  
    {  
    }  
}  
```
