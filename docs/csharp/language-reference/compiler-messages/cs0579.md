# <a name="compiler-error-cs0579"></a><span data-ttu-id="c6d61-102">Derleyici Hatası CS0579</span><span class="sxs-lookup"><span data-stu-id="c6d61-102">Compiler Error CS0579</span></span>
<span data-ttu-id="c6d61-103">Yinelenen 'öznitelik' özniteliği</span><span class="sxs-lookup"><span data-stu-id="c6d61-103">Duplicate 'attribute' attribute</span></span>  
  
 <span data-ttu-id="c6d61-104">Aynı öznitelik birden çok kez öznitelik belirtmediği sürece belirlemek mümkün değil **AllowMultiple = true** içinde kendi [AttributeUsage](../../../csharp/programming-guide/concepts/attributes/attributeusage.md).</span><span class="sxs-lookup"><span data-stu-id="c6d61-104">It is not possible to specify the same attribute more than once unless the attribute specifies **AllowMultiple=true** in its [AttributeUsage](../../../csharp/programming-guide/concepts/attributes/attributeusage.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="c6d61-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="c6d61-105">Example</span></span>  
 <span data-ttu-id="c6d61-106">Aşağıdaki örnek, CS0579 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="c6d61-106">The following example generates CS0579.</span></span>  
  
```csharp  
// CS0579.cs  
using System;  
public class MyAttribute : Attribute  
{  
}  
  
[AttributeUsage(AttributeTargets.All,AllowMultiple=true)]  
public class MyAttribute2 : Attribute  
{  
}  
  
public class z  
{  
    [MyAttribute, MyAttribute]     // CS0579  
    public void zz()  
    {  
    }  
  
    [MyAttribute2, MyAttribute2]   // OK  
    public void zzz()  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
