# <a name="compiler-error-cs0579"></a><span data-ttu-id="a4785-102">Derleyici Hatası CS0579</span><span class="sxs-lookup"><span data-stu-id="a4785-102">Compiler Error CS0579</span></span>
<span data-ttu-id="a4785-103">Yinelenen ' Attribute ' özniteliği</span><span class="sxs-lookup"><span data-stu-id="a4785-103">Duplicate 'attribute' attribute</span></span>  
  
 <span data-ttu-id="a4785-104">Öznitelik, [AttributeUsage](../../programming-guide/concepts/attributes/attributeusage.md)Içinde **AllowMultiple = true** belirtmediği müddetçe aynı özniteliğin birden çok kez belirtilmesi mümkün değildir.</span><span class="sxs-lookup"><span data-stu-id="a4785-104">It is not possible to specify the same attribute more than once unless the attribute specifies **AllowMultiple=true** in its [AttributeUsage](../../programming-guide/concepts/attributes/attributeusage.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="a4785-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="a4785-105">Example</span></span>  
 <span data-ttu-id="a4785-106">Aşağıdaki örnek CS0579 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="a4785-106">The following example generates CS0579.</span></span>  
  
```csharp  
// CS0579.cs  
using System;  
public class MyAttribute : Attribute  
{  
}  
  
[AttributeUsage(AttributeTargets.All,AllowMultiple=true)]  
public class MyAttribute2 : Attribute  
{  
}  
  
public class z  
{  
    [MyAttribute, MyAttribute]     // CS0579  
    public void zz()  
    {  
    }  
  
    [MyAttribute2, MyAttribute2]   // OK  
    public void zzz()  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
