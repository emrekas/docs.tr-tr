# <a name="compiler-error-cs1579"></a><span data-ttu-id="c630e-102">Derleyici Hatası CS1579</span><span class="sxs-lookup"><span data-stu-id="c630e-102">Compiler Error CS1579</span></span>
<span data-ttu-id="c630e-103">'type2' bir 'identifier' ortak tanım içermediğinden foreach deyimi 'type1' türündeki değişkenlerde çalışamaz</span><span class="sxs-lookup"><span data-stu-id="c630e-103">foreach statement cannot operate on variables of type 'type1' because 'type2' does not contain a public definition for 'identifier'</span></span>

<span data-ttu-id="c630e-104">Kullanarak koleksiyon üzerinden yineleme yapmak için [foreach](../keywords/foreach-in.md) deyimi, koleksiyon aşağıdaki gereksinimleri karşılaması gerekir:</span><span class="sxs-lookup"><span data-stu-id="c630e-104">To iterate through a collection using the [foreach](../keywords/foreach-in.md) statement, the collection must meet the following requirements:</span></span>

- <span data-ttu-id="c630e-105">Genel parametresiz türünü içermelidir `GetEnumerator` yöntemi, dönüş türü olan sınıf, yapı veya arabirim türü.</span><span class="sxs-lookup"><span data-stu-id="c630e-105">Its type must include a public parameterless `GetEnumerator` method whose return type is either class, struct, or interface type.</span></span>
- <span data-ttu-id="c630e-106">Dönüş türünü `GetEnumerator` yöntemi adlı ortak bir özellik içermelidir `Current` ve adlı ortak bir parametresiz yöntemin `MoveNext` , dönüş türü olan <xref:System.Boolean>.</span><span class="sxs-lookup"><span data-stu-id="c630e-106">The return type of the `GetEnumerator` method must contain a public property named `Current` and a public parameterless method named `MoveNext` whose return type is <xref:System.Boolean>.</span></span>

## <a name="example"></a><span data-ttu-id="c630e-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="c630e-107">Example</span></span>

<span data-ttu-id="c630e-108">Aşağıdaki örnek CS1579 oluşturur çünkü `MyCollection` sınıfı, ortak içermediğinden `GetEnumerator` yöntemi:</span><span class="sxs-lookup"><span data-stu-id="c630e-108">The following sample generates CS1579 because the `MyCollection` class doesn't contain the public `GetEnumerator` method:</span></span>

```csharp  
// CS1579.cs  
using System;  
public class MyCollection   
{  
   int[] items;  
   public MyCollection()   
   {  
      items = new int[5] {12, 44, 33, 2, 50};  
   }  
  
   // Delete the following line to resolve.  
   MyEnumerator GetEnumerator()  
  
   // Uncomment the following line to resolve:  
   // public MyEnumerator GetEnumerator()   
   {  
      return new MyEnumerator(this);  
   }  
  
   // Declare the enumerator class:  
   public class MyEnumerator   
   {  
      int nIndex;  
      MyCollection collection;  
      public MyEnumerator(MyCollection coll)   
      {  
         collection = coll;  
         nIndex = -1;  
      }  
  
      public bool MoveNext()   
      {  
         nIndex++;  
         return (nIndex < collection.items.Length);  
      }  
  
      public int Current => collection.items[nIndex];
   }  
  
   public static void Main()   
   {  
      MyCollection col = new MyCollection();  
      Console.WriteLine("Values in the collection are:");  
      foreach (int i in col)   // CS1579  
      {  
         Console.WriteLine(i);  
      }  
   }  
}  
```
