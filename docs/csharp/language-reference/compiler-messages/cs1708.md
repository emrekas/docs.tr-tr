# <a name="compiler-error-cs1708"></a><span data-ttu-id="e1a3a-102">Derleyici Hatası CS1708</span><span class="sxs-lookup"><span data-stu-id="e1a3a-102">Compiler Error CS1708</span></span>
<span data-ttu-id="e1a3a-103">Sabit boyutlu arabellekler yalnızca yerel öğeler veya alanlar üzerinden erişilebilir</span><span class="sxs-lookup"><span data-stu-id="e1a3a-103">Fixed size buffers can only be accessed through locals or fields</span></span>  
  
 <span data-ttu-id="e1a3a-104">Yeni bir özellik C# 2.0 içinde satır içi dizisi tanımlama yeteneği olan bir `struct`.</span><span class="sxs-lookup"><span data-stu-id="e1a3a-104">A new feature in C# 2.0 is the ability to define an in-line array inside of a `struct`.</span></span> <span data-ttu-id="e1a3a-105">Bu tür diziler yalnızca yerel değişkenler veya alanlar erişilebilir ve ara değerler sol tarafında bir ifade olarak başvurulan değil.</span><span class="sxs-lookup"><span data-stu-id="e1a3a-105">Such arrays can only be accessed via local variables or fields, and may not be referenced as intermediate values on the left-hand side of an expression.</span></span> <span data-ttu-id="e1a3a-106">Ayrıca, diziler alanları tarafından erişilemez `static` veya `readonly`.</span><span class="sxs-lookup"><span data-stu-id="e1a3a-106">Also, the arrays cannot be accessed by fields that are `static` or `readonly`.</span></span>  
  
 <span data-ttu-id="e1a3a-107">Bu hatayı gidermek için bir dizi değişkeni tanımlayın ve satır içi dizi değişkenine atayın.</span><span class="sxs-lookup"><span data-stu-id="e1a3a-107">To resolve this error, define an array variable, and assign the in-line array to the variable.</span></span> <span data-ttu-id="e1a3a-108">Ya da kaldırma `static` veya `readonly` temsil eden satır içi dizi alanından değiştiricisi.</span><span class="sxs-lookup"><span data-stu-id="e1a3a-108">Or, remove the `static` or `readonly` modifier from the field representing the in-line array.</span></span>  
  
## <a name="example"></a><span data-ttu-id="e1a3a-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="e1a3a-109">Example</span></span>  
 <span data-ttu-id="e1a3a-110">Aşağıdaki örnek, CS1708 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="e1a3a-110">The following sample generates CS1708.</span></span>  
  
```csharp  
// CS1708.cs  
// compile with: /unsafe  
using System;  
  
unsafe public struct S  
{  
    public fixed char name[10];  
}  
  
public unsafe class C  
{  
    public S UnsafeMethod()  
    {  
        S myS = new S();  
        return myS;  
    }  
  
    static void Main()  
    {  
        C myC = new C();  
        myC.UnsafeMethod().name[3] = 'a';  // CS1708  
        // Uncomment the following 2 lines to resolve:  
        // S myS = myC.UnsafeMethod();  
        // myS.name[3] = 'a';  
  
        // The field cannot be static.  
        C._s1.name[3] = 'a';  // CS1708  
  
        // The field cannot be readonly.  
        myC._s2.name[3] = 'a';  // CS1708  
    }  
  
    static readonly S _s1;  
    public readonly S _s2;  
}  
```
