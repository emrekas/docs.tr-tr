# <a name="compiler-error-cs0731"></a>Derleyici Hatası CS0731
Derleme 'derlemesi' ' type' türü için tür ileticisi bir döngüye neden oluyor  
  
 Bu hata yalnızca Düzensiz biçimlendirilmiş alınan meta verilerini oluşabilir. Bu ile yalnızca oluşamaz C# kaynak.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS0731 oluşturur. Örneğin, üç dosyalardan oluşur:  
  
 1. Circular.IL  
  
 2. Circular2.IL  
  
 3. CS0731.cs  
  
 Önce derleyin. Kitaplıklar ve ardından derleme iki dosyanın .cs kodun IL dosyaları.
