 <span data-ttu-id="3f214-104">Bir arabirimde bildirildiği bir [olayı](../keywords/event.md) açıkça uygularken, genellikle derleyicinin sağladığı `add` ve `remove` olay erişimcilerini el ile sağlamanız gerekir.</span><span class="sxs-lookup"><span data-stu-id="3f214-104">When explicitly implementing an [event](../keywords/event.md) that was declared in an interface, you must use manually provide the `add` and `remove` event accessors that are typically provided by the compiler.</span></span> <span data-ttu-id="3f214-105">Erişimci kodu, arabirim olayını sınıfınıza (Bu konunun ilerleyen kısımlarında gösterilmektedir) veya kendi temsilci türüne bağlayabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="3f214-105">The accessor code can connect the interface event to another event in your class (shown later in this topic), or to its own delegate type.</span></span> <span data-ttu-id="3f214-106">Daha fazla bilgi için [nasıl yapılır:  Arabirim olaylarını](../../programming-guide/events/how-to-implement-interface-events.md)uygulayın.</span><span class="sxs-lookup"><span data-stu-id="3f214-106">For more information, see [How to:  Implement Interface Events](../../programming-guide/events/how-to-implement-interface-events.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="3f214-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="3f214-107">Example</span></span>  
 <span data-ttu-id="3f214-108">Aşağıdaki örnek CS0071 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="3f214-108">The following sample generates CS0071.</span></span>  
  
```csharp  
// CS0071.cs  
public delegate void MyEvent(object sender);  
  
interface ITest  
{  
    event MyEvent Clicked;  
}  
  
class Test : Itest  
{  
    event MyEvent ITest.Clicked;  // CS0071  
  
    // try the following code instead  
/*  
private MyEvent clicked;  
  
    event MyEvent Itest.Clicked  
    {  
        add  
        {  
            clicked += value;  
        }  
        remove  
        {  
            clicked -= value;  
        }  
    }  
*/  
    public static void Main() { }  
}  
```
