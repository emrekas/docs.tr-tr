# <a name="compiler-error-cs0016"></a>Derleyici Hatası CS0016
'Dosya' çıkış dosyasına yazılamadı: 'açıklaması'  
  
 Derleyici, bir çıkış dosyasına yazılamadı. Var olduğundan emin olmak için dosya yolunu kontrol edin. Önceden oluşturulmuş bir dosya konumunda ise, yazılabilir olduğundan ve hiçbir işlem kilit dosyanın şu anda sahip olduğundan emin olun. Örneğin, yapı denediğinizde yürütülebilir dosyanızın çalışmıyor emin olun.
