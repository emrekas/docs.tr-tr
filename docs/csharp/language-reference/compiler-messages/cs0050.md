## <a name="example"></a><span data-ttu-id="1326b-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="1326b-106">Example</span></span>  
 <span data-ttu-id="1326b-107">Aşağıdaki örnek, CS0050 oluşturur, çünkü hiçbir erişilebilirlik değiştiricisi için sağlanan `MyClass` ve erişilebilirliğini bu nedenle varsayılan olarak `private`.</span><span class="sxs-lookup"><span data-stu-id="1326b-107">The following sample generates CS0050 because no accessibility modifier is supplied for `MyClass` and its accessibility therefore defaults to `private`.</span></span>  
  
```csharp  
// CS0050.cs  
class MyClass //accessibility defaults to private  
// try the following line instead  
// public class MyClass   
{  
}  
  
public class MyClass2  
{  
    public static MyClass MyMethod()   // CS0050  
    {  
        return new MyClass();  
    }  
  
    public static void Main() { }  
}  
```
