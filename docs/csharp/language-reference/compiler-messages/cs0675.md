# <a name="compiler-warning-level-3-cs0675"></a><span data-ttu-id="9e6cf-102">Derleyici Uyarısı (düzey 3) CS0675</span><span class="sxs-lookup"><span data-stu-id="9e6cf-102">Compiler Warning (level 3) CS0675</span></span>
<span data-ttu-id="9e6cf-103">Bitwise- or işleci işaret genişletilmiş işlenen üzerinde; kullanılan önce daha küçük bir işaretsiz türüne atmayı düşünün</span><span class="sxs-lookup"><span data-stu-id="9e6cf-103">Bitwise-or operator used on a sign-extended operand; consider casting to a smaller unsigned type first</span></span>  
  
 <span data-ttu-id="9e6cf-104">Örtülü olarak genişletti ve işaret genişletilmiş derleyici bir değişkeni ve ardından sonuç değerini bir bit düzeyinde OR işleminde kullanılır.</span><span class="sxs-lookup"><span data-stu-id="9e6cf-104">The compiler implicitly widened and sign-extended a variable, and then used the resulting value in a bitwise OR operation.</span></span> <span data-ttu-id="9e6cf-105">Bu, beklenmeyen davranışlara neden olabilir.</span><span class="sxs-lookup"><span data-stu-id="9e6cf-105">This can result in unexpected behavior.</span></span>  
  
 <span data-ttu-id="9e6cf-106">Aşağıdaki örnek, CS0675 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="9e6cf-106">The following sample generates CS0675:</span></span>  
  
```csharp  
// CS0675.cs  
// compile with: /W:3  
using System;  
  
public class sign  
{  
   public static void Main()  
   {  
      int hi = 1;  
      int lo = 1;  
      long value = (((long)hi) << 32) | lo;              // CS0675  
      // try the following line instead  
      // long value = (((long)hi) << 32) | ((uint)lo);   // correct  
   }  
}  
```
