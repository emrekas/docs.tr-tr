# <a name="compiler-error-cs1540"></a><span data-ttu-id="39d6d-102">Derleyici Hatası CS1540</span><span class="sxs-lookup"><span data-stu-id="39d6d-102">Compiler Error CS1540</span></span>
<span data-ttu-id="39d6d-103">Korumalı üye 'member' Niteleyici 'type1' türünün aracılığıyla erişilemiyor Niteleyici 'type2' türünde olması gerekir (veya bundan türetilmiş)</span><span class="sxs-lookup"><span data-stu-id="39d6d-103">Cannot access protected member 'member' via a qualifier of type 'type1'; the qualifier must be of type 'type2' (or derived from it)</span></span>  
  
 <span data-ttu-id="39d6d-104">Türetilmiş [sınıfı](../../../csharp/language-reference/keywords/class.md) olduğu temel sınıfın korumalı üyeleri temel sınıfın örneği erişilemiyor.</span><span class="sxs-lookup"><span data-stu-id="39d6d-104">A derived [class](../../../csharp/language-reference/keywords/class.md) cannot access protected members of its base class through an instance of the base class.</span></span> <span data-ttu-id="39d6d-105">Temel sınıfın bir örneği, türetilmiş bir sınıf içindeki çalışma zamanında değil ancak aynı temel türetilmiş başka bir türde bir örnek yoksa türetilmiş sınıf için ilgili olabilir bildirilmiş.</span><span class="sxs-lookup"><span data-stu-id="39d6d-105">An instance of the base class declared in the derived class might, at run time, be an instance of another type that is derived from the same base but is not otherwise related to the derived class.</span></span> <span data-ttu-id="39d6d-106">Korumalı üyeler yalnızca türetilen türler tarafından erişilebilir olduğundan, çalışma zamanında geçerli olmayabilir korumalı üyelere erişim girişimleri geçersiz derleyici tarafından işaretlenir.</span><span class="sxs-lookup"><span data-stu-id="39d6d-106">Because protected members can be accessed only by derived types, any attempts to access protected members that might not be valid at run time are marked by the compiler as not valid.</span></span>  
  
 <span data-ttu-id="39d6d-107">İçinde `Employee` aşağıdaki örnekte, sınıf `emp2` ve `emp3` hem türüne sahip `Person` derleme zamanında ancak `emp2` türünde `Manager` çalışma zamanında.</span><span class="sxs-lookup"><span data-stu-id="39d6d-107">In the `Employee` class in the following example, `emp2` and `emp3` both have type `Person` at compile time, but `emp2` has type `Manager` at run time.</span></span> <span data-ttu-id="39d6d-108">Çünkü `Employee` türünden türetilmediğinden `Manager`, temel sınıfın korumalı üyeleri erişemiyor `Person`, örneği üzerinden `Manager` sınıfı.</span><span class="sxs-lookup"><span data-stu-id="39d6d-108">Because `Employee` is not derived from `Manager`, it cannot access the protected members of the base class, `Person`, through an instance of the `Manager` class.</span></span> <span data-ttu-id="39d6d-109">Derleyici iki hangi çalışma zamanı türü belirlenemiyor `Person` nesneleri olacaktır.</span><span class="sxs-lookup"><span data-stu-id="39d6d-109">The compiler cannot determine what the run-time type of the two `Person` objects will be.</span></span> <span data-ttu-id="39d6d-110">Bu nedenle, her iki çağrısından `emp2` ve çağrısından `emp3` derleyici hatası CS1540 neden olur.</span><span class="sxs-lookup"><span data-stu-id="39d6d-110">Therefore, both the call from `emp2` and the call from `emp3` cause compiler error CS1540.</span></span>  
  
```csharp  
using System;  
using System.Text;  
  
namespace CS1540  
{  
    class Program1  
    {  
        static void Main()  
        {  
            Employee.PreparePayroll();  
        }  
    }  
  
    class Person  
    {  
        protected virtual void CalculatePay()   
        {  
            Console.WriteLine("CalculatePay in Person class.");  
        }  
    }  
  
    class Manager : Person  
    {  
        protected override void CalculatePay()   
        {  
            Console.WriteLine("CalculatePay in Manager class.");   
  
        }  
    }  
  
    class Employee : Person  
    {  
        public static void PreparePayroll()  
        {  
            Employee emp1 = new Employee();  
            Person emp2 = new Manager();  
            Person emp3 = new Employee();  
            // The following line calls the method in the Employee base class,  
            // Person.  
            emp1.CalculatePay();   
  
            // The following lines cause compiler error CS1540. The compiler   
            // cannot determine at compile time what the run-time types of   
            // emp2 and emp3 will be.  
            //emp2.CalculatePay();   
            //emp3.CalculatePay();  
  
        }  
    }  
}  
```
