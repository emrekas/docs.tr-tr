## <a name="example"></a><span data-ttu-id="b58e7-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="b58e7-107">Example</span></span>  

 <span data-ttu-id="b58e7-108">Aşağıdaki örnek CS0417 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="b58e7-108">The following example generates CS0417:</span></span>

    // The following line causes CS0417.
