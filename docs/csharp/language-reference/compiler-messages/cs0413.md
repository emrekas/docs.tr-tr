# <a name="compiler-error-cs0413"></a><span data-ttu-id="46d5a-102">Derleyici Hatası CS0413</span><span class="sxs-lookup"><span data-stu-id="46d5a-102">Compiler Error CS0413</span></span>
<span data-ttu-id="46d5a-103">'Tür parametresi' tür parametresinde sınıf tür kısıtlaması veya bir 'class' kısıtlaması olmadığından 'as' işleciyle birlikte kullanılamaz</span><span class="sxs-lookup"><span data-stu-id="46d5a-103">The type parameter 'type parameter' cannot be used with the 'as' operator because it does not have a class type constraint nor a 'class' constraint</span></span>  
  
 <span data-ttu-id="46d5a-104">Genel tür kullanıyorsa, bu hata oluşur [olarak](../../../csharp/language-reference/keywords/as.md) işleci, ancak genel tür bir sınıf olmadığından tür kısıtlaması.</span><span class="sxs-lookup"><span data-stu-id="46d5a-104">This error occurs if a generic type uses the [as](../../../csharp/language-reference/keywords/as.md) operator, but that generic type does not have a class type constraint.</span></span> <span data-ttu-id="46d5a-105">`as` İşleci yalnızca izin verilen başvuru türleri ile bir değer türü olmadığını güvence altına almak için tür parametresi kısıtlı gerekir böylece.</span><span class="sxs-lookup"><span data-stu-id="46d5a-105">The `as` operator is only allowed with reference types, so the type parameter must be constrained to guarantee that it is not a value type.</span></span> <span data-ttu-id="46d5a-106">Bu hatayı önlemek için sınıf tür kısıtlaması veya bir başvuru türü kısıtlaması'nı kullanın.</span><span class="sxs-lookup"><span data-stu-id="46d5a-106">To avoid this error, use a class type constraint or a reference type constraint.</span></span>  
  
 <span data-ttu-id="46d5a-107">Bunun nedeni, `as` işleci döndürebilir `null`, bir değer türü için olası bir değer değil ve tür parametresi sınıf tür kısıtlaması veya bir başvuru türü kısıtlaması olmadığı sürece bir değer türü değerlendirilmesi gerekir.</span><span class="sxs-lookup"><span data-stu-id="46d5a-107">This is because the `as` operator could return `null`, which is not a possible value for a value type, and the type parameter must be treated as a value type unless it is a class type constraint or a reference type constraint.</span></span>  
  
## <a name="example"></a><span data-ttu-id="46d5a-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="46d5a-108">Example</span></span>  
 <span data-ttu-id="46d5a-109">Aşağıdaki örnek, CS0413 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="46d5a-109">The following sample generates CS0413.</span></span>  
  
```csharp  
// CS0413.cs  
// compile with: /target:library  
class A {}  
class B : A {}  
  
class CMain  
{  
   A a = null;  
   public void G<T>()  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  // CS0413  
   }  
  
   // OK  
   public void H<T>() where T : A  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  
   }  
}  
```
