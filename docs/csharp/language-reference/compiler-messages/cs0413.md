# <a name="compiler-error-cs0413"></a>Derleyici Hatası CS0413
'Tür parametresi' tür parametresinde sınıf tür kısıtlaması veya bir 'class' kısıtlaması olmadığından 'as' işleciyle birlikte kullanılamaz  
  
Genel tür kullanıyorsa, bu hata oluşur [olarak](../../language-reference/operators/type-testing-and-conversion-operators.md#as-operator) işleci, ancak genel tür bir sınıf olmadığından tür kısıtlaması. `as` İşleci yalnızca izin verilen boş değer atanabilen değer türleri ve başvuru ile için bir değer türü olmadığını güvence altına almak için tür parametresi kısıtlı gerekir. Bu hatayı önlemek için sınıf tür kısıtlaması veya bir başvuru türü kısıtlaması'nı kullanın.
  
Bunun nedeni, `as` işleci döndürebilir `null`, bir değer türü için olası bir değer değil ve tür parametresi sınıf tür kısıtlaması veya bir başvuru türü kısıtlaması olmadığı sürece bir değer türü değerlendirilmesi gerekir.
  
## <a name="example"></a>Örnek

Aşağıdaki örnek, CS0413 oluşturur.  
  
```csharp  
// CS0413.cs  
// compile with: /target:library  
class A {}  
class B : A {}  
  
class CMain  
{  
   A a = null;  
   public void G<T>()  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  // CS0413  
   }  
  
   // OK  
   public void H<T>() where T : A  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  
   }  
}  
```
