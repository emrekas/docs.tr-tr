# <a name="compiler-error-cs1942"></a><span data-ttu-id="49145-102">Derleyici Hatası CS1942</span><span class="sxs-lookup"><span data-stu-id="49145-102">Compiler Error CS1942</span></span>

<span data-ttu-id="49145-105">Bu hata genellikle Aralık değişkenine yanlış bir açık tür verildiğinde oluşturulur.</span><span class="sxs-lookup"><span data-stu-id="49145-105">This error is typically generated when the range variable has been given an incorrect explicit type.</span></span>  

## <a name="to-correct-this-error"></a><span data-ttu-id="49145-106">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="49145-106">To correct this error</span></span>

1. <span data-ttu-id="49145-107">Aralık değişkeni açıkça yazılmışsa, türün, yineleme yaptığı koleksiyondaki öğelerin türü ile aynı ya da örtülü olarak dönüştürülebilir olduğundan emin olun.</span><span class="sxs-lookup"><span data-stu-id="49145-107">If the range variable is explicitly typed, make sure that the type is either the same as, or implicitly convertible from, the type of the elements in the collection it iterates.</span></span> <span data-ttu-id="49145-108">Aralık değişkenine önünde `var` anahtar sözcüğü varsa, öğesini kaldırın `var`.</span><span class="sxs-lookup"><span data-stu-id="49145-108">If the range variable is preceded with the `var` keyword, remove `var`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="49145-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="49145-109">Example</span></span>

<span data-ttu-id="49145-110">Aşağıdaki kod CS1942 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="49145-110">The following code generates CS1942:</span></span>  

```csharp
// cs1942.cs  
class Program  
    {  
        static void Main(string[] args)  
        {  
            var x = from var i in Enumerable.Range(1, 100) // CS1949  
                    select i; //CS1942  
        }  
    }  
```

<span data-ttu-id="49145-111">CS1942, CS1949 `var` ile ilgilidir çünkü bir Aralık değişkeni ile kullanılması temeldeki `Cast<T>` işlemin başarısız olmasına neden olur çünkü `var` bir tür değildir.</span><span class="sxs-lookup"><span data-stu-id="49145-111">CS1942 is related to CS1949 because the use of `var` with a range variable causes the underlying `Cast<T>` operation to fail because `var` is not a type.</span></span>
