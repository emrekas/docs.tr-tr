# <a name="compiler-error-cs0304"></a><span data-ttu-id="a7b4c-102">Derleyici Hatası CS0304</span><span class="sxs-lookup"><span data-stu-id="a7b4c-102">Compiler Error CS0304</span></span>

<span data-ttu-id="a7b4c-103">New() kısıtlaması olmadığından 'type' değişken türünün örneği oluşturulamıyor</span><span class="sxs-lookup"><span data-stu-id="a7b4c-103">Cannot create an instance of the variable type 'type' because it does not have the new() constraint</span></span>  
  
 <span data-ttu-id="a7b4c-104">Ne zaman genel bir sınıf uygulamak ve kullanmak istediğiniz `new` anahtar sözcüğü bir tür parametresi için sağlanan her türlü yeni bir örneğini oluşturmak için `T`, uygulamalısınız [new() kısıtlaması](../../../csharp/language-reference/keywords/new.md) için `T` içinde Aşağıdaki örnekte gösterildiği gibi sınıf bildirimi.</span><span class="sxs-lookup"><span data-stu-id="a7b4c-104">When you implement a generic class, and you want to use the `new` keyword to create a new instance of any type that is supplied for a type parameter `T`, you must apply the [new() constraint](../../../csharp/language-reference/keywords/new.md) to `T` in the class declaration, as shown in the following example.</span></span>  
  
```csharp  
class C<T> where T : new()  
```  
  
 <span data-ttu-id="a7b4c-105">`new()` Kısıtlaması tarafından belirtilmesine herhangi için sağlanan türü somut tür güvenliği zorlar `T` varsayılan, parametresiz bir oluşturucusu vardır.</span><span class="sxs-lookup"><span data-stu-id="a7b4c-105">The `new()` constraint enforces type safety by guaranteeing that any concrete type that is supplied for `T` has a default, parameterless constructor.</span></span> <span data-ttu-id="a7b4c-106">CS0304 kullanmaya çalışırsa gerçekleşir `new` sınıfın gövdesi işlecinde tür parametresi bir örneğini oluşturmak için `T` olduğunda `T` belirttiğinde `new()` kısıtlaması.</span><span class="sxs-lookup"><span data-stu-id="a7b4c-106">CS0304 occurs if you attempt to use the `new` operator in the body of the class to create an instance of type parameter `T` when `T` does not specify the `new()` constraint.</span></span> <span data-ttu-id="a7b4c-107">Genel sınıf ile kod oluşturacağı hiçbir parametresiz oluşturucusu olan bir türü örneklemek kod çalışırsa istemci tarafında [Derleyici Hatası CS0310](../../../csharp/language-reference/compiler-messages/cs0310.md).</span><span class="sxs-lookup"><span data-stu-id="a7b4c-107">On the client side, if code attempts to instantiate the generic class with a type that has no parameterless constructor, that code will generate [Compiler Error CS0310](../../../csharp/language-reference/compiler-messages/cs0310.md).</span></span>  
  
 <span data-ttu-id="a7b4c-108">Aşağıdaki örnek, CS0304 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="a7b4c-108">The following example generates CS0304.</span></span>  
  
```csharp  
// CS0304.cs  
// Compile with: /target:library.  
class C<T>  
{  
    // The following line generates CS0304.  
    T t = new T();  
}  
```  
  
 <span data-ttu-id="a7b4c-109">`new` İşleci ayrıca izin verilmiyor sınıfının yöntemleri.</span><span class="sxs-lookup"><span data-stu-id="a7b4c-109">The `new` operator also is not allowed in methods of the class.</span></span>  
  
```csharp  
// Compile with: /target:library.  
class C<T>  
{  
    public void ExampleMethod()  
    {  
        // The following line generates CS0304.  
        T t = new T();  
    }  
}  
```  
  
 <span data-ttu-id="a7b4c-110">Hatayı önlemek için kullanarak bir sınıf bildirme `new()` kısıtlaması, aşağıdaki örnekte gösterildiği gibi.</span><span class="sxs-lookup"><span data-stu-id="a7b4c-110">To avoid the error, declare the class by using the `new()` constraint, as shown in the following example.</span></span>  
  
```csharp  
// Compile with: /target:library.  
class C<T> where T : new()  
{  
    T t = new T();  
  
    public void ExampleMethod()  
    {  
        T t = new T();  
    }  
}  
```
