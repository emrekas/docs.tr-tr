# <a name="compiler-error-cs0304"></a><span data-ttu-id="4e3ed-102">Derleyici Hatası CS0304</span><span class="sxs-lookup"><span data-stu-id="4e3ed-102">Compiler Error CS0304</span></span>

<span data-ttu-id="4e3ed-103">New () kısıtlamasına sahip olmadığından ' Type ' değişken türünün bir örneği oluşturulamıyor</span><span class="sxs-lookup"><span data-stu-id="4e3ed-103">Cannot create an instance of the variable type 'type' because it does not have the new() constraint</span></span>  
  
 <span data-ttu-id="4e3ed-104">Genel bir sınıf uyguladığınızda ve bir `new` tür parametresi `T`için sağlanan herhangi bir türün yeni bir örneğini oluşturmak için anahtar sözcüğünü kullanmak istiyorsanız, [New () kısıtlamasını](../keywords/new-constraint.md) `T` sınıf bildiriminde ' a uygulamanız gerekir, örneğin Aşağıdaki örnekte gösterilmiştir.</span><span class="sxs-lookup"><span data-stu-id="4e3ed-104">When you implement a generic class, and you want to use the `new` keyword to create a new instance of any type that is supplied for a type parameter `T`, you must apply the [new() constraint](../keywords/new-constraint.md) to `T` in the class declaration, as shown in the following example.</span></span>  
  
```csharp  
class C<T> where T : new()  
```  
  
 <span data-ttu-id="4e3ed-105">Kısıtlama `new()` , için `T` sağlanan herhangi bir somut türün varsayılan, parametresiz bir oluşturucuya sahip olduğunu garanti ederek tür güvenliğini zorlar.</span><span class="sxs-lookup"><span data-stu-id="4e3ed-105">The `new()` constraint enforces type safety by guaranteeing that any concrete type that is supplied for `T` has a default, parameterless constructor.</span></span> <span data-ttu-id="4e3ed-106">CS0304, `new` kısıtlamabelirtmeyen`new()` bir tür `T` parametresi `T` örneği oluşturmak için sınıfının gövdesinde işleci kullanmayı denerseniz oluşur.</span><span class="sxs-lookup"><span data-stu-id="4e3ed-106">CS0304 occurs if you attempt to use the `new` operator in the body of the class to create an instance of type parameter `T` when `T` does not specify the `new()` constraint.</span></span> <span data-ttu-id="4e3ed-107">İstemci tarafında, kod parametresiz oluşturucusu olmayan bir türle genel sınıf örneğini oluşturmaya çalışırsa, bu kod [derleyici hatası CS0310](./cs0310.md)oluşturacaktır.</span><span class="sxs-lookup"><span data-stu-id="4e3ed-107">On the client side, if code attempts to instantiate the generic class with a type that has no parameterless constructor, that code will generate [Compiler Error CS0310](./cs0310.md).</span></span>  
  
 <span data-ttu-id="4e3ed-108">Aşağıdaki örnek CS0304 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="4e3ed-108">The following example generates CS0304.</span></span>  
  
```csharp  
// CS0304.cs  
// Compile with: /target:library.  
class C<T>  
{  
    // The following line generates CS0304.  
    T t = new T();  
}  
```  
  
 <span data-ttu-id="4e3ed-109">Ayrıca `new` , işlecine sınıf yöntemlerinde de izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="4e3ed-109">The `new` operator also is not allowed in methods of the class.</span></span>  
  
```csharp  
// Compile with: /target:library.  
class C<T>  
{  
    public void ExampleMethod()  
    {  
        // The following line generates CS0304.  
        T t = new T();  
    }  
}  
```  
  
 <span data-ttu-id="4e3ed-110">Hatayı önlemek için, aşağıdaki örnekte gösterildiği gibi, `new()` kısıtlamasını kullanarak sınıfı bildirin.</span><span class="sxs-lookup"><span data-stu-id="4e3ed-110">To avoid the error, declare the class by using the `new()` constraint, as shown in the following example.</span></span>  
  
```csharp  
// Compile with: /target:library.  
class C<T> where T : new()  
{  
    T t = new T();  
  
    public void ExampleMethod()  
    {  
        T t = new T();  
    }  
}  
```
