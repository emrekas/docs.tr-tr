## <a name="example"></a><span data-ttu-id="0efda-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="0efda-108">Example</span></span>

<span data-ttu-id="0efda-109">Aşağıdaki kod CS1941 oluşturur, çünkü `equals` işlecine bir `int` ile `string`karşılaştırılacağı sorulur.</span><span class="sxs-lookup"><span data-stu-id="0efda-109">The following code generates CS1941 because the `equals` operator is being asked to compare an `int` to a `string`.</span></span>  

```csharp
// cs1941.cs  
using System.Collections;  
using System.Linq;  
class Test  
{  
    static int Main()  
    {  
        var nums = new[] { 1, 2, 3, 4, 5, 6 };  
        var words = new string[] { "lake", "mountain", "sky" };  
        IEnumerable e = from n in nums  
                        join w in words on n equals w // CS1941  
                        select w;  
        return 0;  
    }  
}  
```

<span data-ttu-id="0efda-110">Tür çıkarımı başarısız olan yöntem sorgu yan tümcesinin derleme zamanında çevrilme yöntemidir.</span><span class="sxs-lookup"><span data-stu-id="0efda-110">The method in which type inference fails is the method that the query clause is translated to at compile time.</span></span>
