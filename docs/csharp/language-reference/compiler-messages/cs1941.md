## <a name="example"></a><span data-ttu-id="14022-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="14022-108">Example</span></span>

<span data-ttu-id="14022-109">Aşağıdaki kodu CS1941 oluşturur çünkü `equals` işleci istenir karşılaştırma bir `int` için bir `string`.</span><span class="sxs-lookup"><span data-stu-id="14022-109">The following code generates CS1941 because the `equals` operator is being asked to compare an `int` to a `string`.</span></span>  

```csharp
// cs1941.cs  
using System.Collections;  
using System.Linq;  
class Test  
{  
    static int Main()  
    {  
        var nums = new[] { 1, 2, 3, 4, 5, 6 };  
        var words = new string[] { "lake", "mountain", "sky" };  
        IEnumerable e = from n in nums  
                        join w in words on n equals w // CS1941  
                        select w;  
        return 0;  
    }  
}  
```

<span data-ttu-id="14022-110">Hangi tür çıkarımı başarısız yöntem sorgu yan tümcesi derleme zamanında çevrildiğinde yöntemdir.</span><span class="sxs-lookup"><span data-stu-id="14022-110">The method in which type inference fails is the method that the query clause is translated to at compile time.</span></span>
