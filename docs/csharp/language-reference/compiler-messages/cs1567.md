# <a name="compiler-error-cs1567"></a>Derleyici Hatası CS1567
Win32 kaynağı oluşturulurken hata oluştu: ' dosya '  
  
 Derlemeniz [/Win32Icon](../compiler-options/win32icon-compiler-option.md) derleyici seçeneğini kullandı veya [/win32res](../compiler-options/win32res-compiler-option.md)kullanmadı. Bu, derleyicinin kaynak bilgilerini içeren bir dosya oluşturmasına neden olur, ancak bu, yetersiz disk alanı nedeniyle derleyici dosyayı oluşturamadı ya da başka bir hata.  
  
 Dosya oluşturma sorununu çözemezse, kaynak bilgilerini içeren bir dosya oluşturabilen [/win32res](../compiler-options/win32res-compiler-option.md)' i kullanabilirsiniz.
