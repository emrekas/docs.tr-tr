<span data-ttu-id="bbf97-103">Anonim yöntem ifadesi ifade ağacına dönüştürülemiyor.</span><span class="sxs-lookup"><span data-stu-id="bbf97-103">An anonymous method expression cannot be converted to an expression tree.</span></span>

## <a name="example"></a><span data-ttu-id="bbf97-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="bbf97-108">Example</span></span>

<span data-ttu-id="bbf97-109">Aşağıdaki örnek, CS1946 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="bbf97-109">The following example generates CS1946:</span></span>

```csharp
    using System.Linq.Expressions;  
  
    public delegate void D();  
  
    class Test  
    {  
        static void Main()  
        {  
            Expression<D> tree = delegate() { }; //CS1946  
            // Try using a lambda expression instead.

        }  
    }  
```
