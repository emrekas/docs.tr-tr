## <a name="example"></a><span data-ttu-id="789a4-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="789a4-108">Example</span></span>  
 <span data-ttu-id="789a4-109">Bu kod örneği için bir .dll dosyasını derler, bir .cpp dosyası ve bu .dll öğesini kullanan bir .cs dosyası oluşur.</span><span class="sxs-lookup"><span data-stu-id="789a4-109">This code sample consists of a .cpp file, which compiles to a .dll, and a .cs file, which uses that .dll.</span></span> <span data-ttu-id="789a4-110">Aşağıdaki kod .dll dosyasıdır ve .cs dosyası kod tarafından erişilecek bir özelliğini tanımlar.</span><span class="sxs-lookup"><span data-stu-id="789a4-110">The following code is for the .dll file and defines a property to be accessed by the code in the .cs file.</span></span>  
  
```cpp

// compile with: /clr /LD  
using namespace System;  
public ref class MCPP  
{  
public:  
    property int Prop [int,int]  
    {  
        int get( int i, int b )  
        {  
            return i;  
        }  
    }  
};  
```
