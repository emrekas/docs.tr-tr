# <a name="compiler-error-cs0507"></a><span data-ttu-id="91768-102">Derleyici Hatası CS0507</span><span class="sxs-lookup"><span data-stu-id="91768-102">Compiler Error CS0507</span></span>
<span data-ttu-id="91768-103">'function1': ' erişim ' devralınan üyesi 'function2' erişim değiştiricileri değiştirilemez</span><span class="sxs-lookup"><span data-stu-id="91768-103">'function1' : cannot change access modifiers when overriding 'access' inherited member 'function2'</span></span>  
  
 <span data-ttu-id="91768-104">Bir yöntemi geçersiz kılma erişimi belirtiminde değiştirmeye çalışıldı.</span><span class="sxs-lookup"><span data-stu-id="91768-104">An attempt was made to change the access specification in a method override.</span></span>  
  
## <a name="example"></a><span data-ttu-id="91768-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="91768-105">Example</span></span>  
 <span data-ttu-id="91768-106">Aşağıdaki örnek, CS0507 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="91768-106">The following sample generates CS0507.</span></span>  
  
```csharp  
// CS0507.cs  
abstract public class clx  
{  
   virtual protected void f() {}  
}  
  
public class cly : clx  
{  
   public override void f() {}   // CS0507  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="91768-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="91768-107">Example</span></span>  
 <span data-ttu-id="91768-108">CS0507 olarak işaretlenmiş bir yöntemi geçersiz kılmak bir sınıf çalışırsa da gerçekleşebilir `protected internal` başvurulan meta verilerde tanımlanan.</span><span class="sxs-lookup"><span data-stu-id="91768-108">CS0507 can also occur if a class attempts to override a method marked as `protected internal` defined in referenced metadata.</span></span> <span data-ttu-id="91768-109">Bu durumda geçersiz kılma yöntemi olarak işaretlenmelidir `protected`.</span><span class="sxs-lookup"><span data-stu-id="91768-109">In this situation, the overriding method should be marked as `protected`.</span></span>  
  
```csharp

{  
   virtual protected internal void f() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="91768-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="91768-110">Example</span></span>  
 <span data-ttu-id="91768-111">Aşağıdaki örnek, CS0507 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="91768-111">The following sample generates CS0507.</span></span>
