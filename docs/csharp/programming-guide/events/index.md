 <span data-ttu-id="bab06-110">Olayları, aşağıdaki özelliklere sahiptir:</span><span class="sxs-lookup"><span data-stu-id="bab06-110">Events have the following properties:</span></span>

- <span data-ttu-id="bab06-115">Olaylar genellikle, düğme tıklamaları veya grafik kullanıcı arabirimleri menü seçimleri gibi kullanıcı eylemlerini göstermek için kullanılır.</span><span class="sxs-lookup"><span data-stu-id="bab06-115">Events are typically used to signal user actions such as button clicks or menu selections in graphical user interfaces.</span></span>
