# <a name="c-c-programming-guide"></a>\<c > (C# Programlama Kılavuzu)
## <a name="syntax"></a>Sözdizimi  
  
```xml  
<c>text</c>  
```  
  
#### <a name="parameters"></a>Parametreler  
 `text`  
 Kod olarak belirtmek istediğiniz metin.  
  
## <a name="remarks"></a>Açıklamalar  
 \<c > etiketi metin açıklama içinde kod olarak işaretlenmelidir göstermek için bir yol sağlar. Kullanım [ \<kod >](../../../csharp/programming-guide/xmldoc/code.md) çok satırlı kod olarak belirtmek için.  
  
 Derleme [/doc](../../../csharp/language-reference/compiler-options/doc-compiler-option.md) işlem belgeleri açıklamaları için bir dosya için.
