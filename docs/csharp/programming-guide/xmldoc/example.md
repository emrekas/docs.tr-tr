# <a name="example-c-programming-guide"></a>\<örnek > (C# Programlama Kılavuzu)
## <a name="syntax"></a>Sözdizimi  
  
```xml  
<example>description</example>  
```  
  
## <a name="parameters"></a>Parametreler  
 `description`  
 Kod örneğinin açıklaması.  
  
## <a name="remarks"></a>Açıklamalar  
 Örnek \<> etiketi, bir yöntemin veya diğer kitaplık üyesinin nasıl kullanılacağına ilişkin bir örnek belirtmenize olanak tanır. Bu genellikle [ \<kod >](./code.md) etiketinin kullanılmasını içerir.  
  
 Belge açıklamalarını bir dosyaya işlemek için [/doc](../../language-reference/compiler-options/doc-compiler-option.md) ile derleyin.
