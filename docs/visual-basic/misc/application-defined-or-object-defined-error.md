# <a name="application-defined-or-object-defined-error"></a>Uygulama tanımlı ya da nesne tanımlanan bir hata oluştu
Uygulama tanımlı veya kullanıcı tanımlı bir hata oluştu. Hatasına bakın.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Burada tanımlanır ve bu sorunları gidermek nasıl görmek için hata başvurun.
