# <a name="division-by-zero-visual-basic-run-time-error"></a>Bölme sıfır (Visual Basic çalışma süresi hatası)
Bir ayırıcı kullanılan bir ifade sıfır değerine sahiptir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. İfade değişkenleri yazımını denetleyin. Yanlış yazılmış bir değişken adı, sıfır olarak başlatılır, sayısal bir değişken örtük olarak oluşturabilirsiniz.  
  
2. Önceki değişkenleri ifadede, özellikle yordama bağımsız değişkenler olarak diğer yordamlardan geçirilen işlemleri denetleyin.
