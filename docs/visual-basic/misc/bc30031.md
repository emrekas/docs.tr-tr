# <a name="fully-qualified-names-for-types-in-namespaces-cannot-be-longer-than-number-characters"></a>Tam olarak nitelenmiş adlar için ad alanlarında türleri daha uzun olamaz \<numarası > karakter
Türünün tam adını, belirtilen sayıda karakteri uzun olamaz.  
  
 **Hata Kimliği:** BC30031  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Türün adını kısaltın.
