# <a name="a-delimiter-cannot-be-nothing-or-an-empty-string"></a>Sınırlayıcı, hiçbir şey veya boş dize olamaz.
`TextFieldParser` Çünkü Dosyadan okuma yapamıyor `Delimiters` özelliği `Nothing` ya da boş bir `String` ("").  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   İçin geçerli bir değer sağlamanız `Delimiters`.
