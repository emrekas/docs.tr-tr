# <a name="events-cannot-have-a-return-type"></a>Olaylar, dönüş türüne sahip olamaz
Olaylar, bağımsız değişkenleri kabul edebilir, ancak doğrudan değer döndüremez.  
  
 **Hata Kimliği:** BC30032  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Dönüş türünden kaldırın `Event` deyimi.
