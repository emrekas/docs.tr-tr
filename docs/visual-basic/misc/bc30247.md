# <a name="modifier-is-not-valid-on-a-local-variable-declaration"></a>'\<değiştiricisi >' yerel değişken bildiriminde geçerli değil
A `Dim` deyimi bir yordam içinde geçersiz bir anahtar sözcüğü gibi içerir `Overloads`.  
  
 **Hata Kimliği:** BC30247  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Geçersiz anahtar sözcüğü Kaldır `Dim` deyimi.
