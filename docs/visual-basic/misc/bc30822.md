## <a name="to-correct-this-error"></a><span data-ttu-id="e5afc-107">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="e5afc-107">To correct this error</span></span>  
  
1.  <span data-ttu-id="e5afc-108">Kullanım <xref:System.DBNull> yerine `Null`.</span><span class="sxs-lookup"><span data-stu-id="e5afc-108">Use <xref:System.DBNull> instead of `Null`.</span></span> <span data-ttu-id="e5afc-109">Aşağıdaki örnekte bu gösterir.</span><span class="sxs-lookup"><span data-stu-id="e5afc-109">The following example demonstrates this.</span></span>  
  
    ```  
    Sub TestDBNull()  
        Dim t As DataTable  
        ' Assume the DataGrid is bound to a DataTable.  
        t = CType(DataGrid1.DataSource, DataTable)  
        Dim r As DataRow  
        r = t.Rows(datagrid1.CurrentCell.RowNumber)  
        r.BeginEdit  
        r(1) = System.DBNull.Value ' Assign DBNull to the record.  
        r.EndEdit  
        r.AcceptChanges  
        If r.IsNull(1) Then  
            MsgBox("")  
        End If  
    End Sub  
    ```  
  
2.  <span data-ttu-id="e5afc-110">Kullanma [hiçbir şey](../../visual-basic/language-reference/nothing.md) atamaları ve nesne değişkenleri kullandığınızda karşılaştırmaları için anahtar sözcüğü.</span><span class="sxs-lookup"><span data-stu-id="e5afc-110">Use the [Nothing](../../visual-basic/language-reference/nothing.md) keyword for assignments and comparisons when you use object variables.</span></span> <span data-ttu-id="e5afc-111">Aşağıdaki örnekte bu gösterir.</span><span class="sxs-lookup"><span data-stu-id="e5afc-111">The following example demonstrates this.</span></span>

    ```  
        Dim cls As Object  
        ' cls is Nothing if it has not been assigned using the New keyword.  
        If (cls Is Nothing) Then  
            MsgBox("cls is Nothing")  
        End If  
        cls = Nothing ' Assign Nothing to the class variable cls.  
    End Sub  
    ```
