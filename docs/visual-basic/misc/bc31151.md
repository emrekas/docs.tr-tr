# <a name="element-is-missing-an-end-tag"></a>Öğesinde bir bitiş etiketi eksik
Bir XML öğesi değişmez değeri bir XML bitiş etiketi ya da boş bir etiket tanımlayıcısı eksik '/' (örneğin, `<tagName />`).  
  
 **Hata Kimliği:** BC31151  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- XML etiketi için bitiş etiketi ekleyin veya boş bir etiket tanımlayıcı Ekle ' /' XML etiketi.
