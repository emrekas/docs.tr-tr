# <a name="-is-not-allowed"></a>':' izin verilmiyor
':' izin verilmiyor. Uygun XML adı bu bağlamda kullanılamaz.  
  
 Örneğin, bir hedef işleme yönergesi değişmez değeri geçersiz bir konum içinde tam bir ada sahip bir XML değişmez değer belirtilmedi.  
  
 **Hata Kimliği:** BC31186  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   XML tam adı ile değişmez değer kaldırın veya geçerli bir konuma taşıyın.  
  
     Karakter olduğu konusunda XML adlarında geçerli daha fazla bilgi için bkz: [Genişletilebilir Biçimlendirme Dili (XML) 1.0](https://www.w3.org/TR/xml).
