# <a name="goto-labelname-is-not-valid-because-labelname-is-inside-a-try-catch-or-finally-statement-that-does-not-contain-this-statement"></a>' GoTo \<labelname >' geçerli değil çünkü '\<labelname >' bir 'Try', 'Catch' veya bu deyimi içermeyen ifadesi içinde 'Finally'
Dal oluşturamazsınız bir `Try...Catch...Finally` blok.  
  
 **Hata Kimliği:** BC30754  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Etiket önündeki böylece kodunuzu yeniden yapılandırmayı `Try...Catch...Finally` blok.
