# <a name="initialization-error"></a>Başlatma hatası
Program başlatılırken bir hata oluştu. Bu, hala devam ederken yükleme iptal ediliyor, bir sonucu olabilir.  
  
 **Hata Kimliği:** BC2000  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Yüklemeyi deneyin.
