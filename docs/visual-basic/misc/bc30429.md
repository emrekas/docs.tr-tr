# <a name="end-sub-must-be-preceded-by-a-matching-sub"></a>'End Sub', eşleşen bir 'Sub tarafından' gelmelidir
Bir `End Sub` deyimi görünüyor, kodunuzda hiçbir eşleşen `Sub` , önceki yordamı tanımı.  
  
 **Hata Kimliği:** BC30429  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Kaldırma `End Sub` yedekli ise deyimi.  
  
- Eksik olan kaynağı `Sub` biri eksikse yordamı.  
  
- Taşıma `End Sub` kod uygun yerleştirmek için.
