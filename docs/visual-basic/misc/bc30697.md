# <a name="membername1-cannot-override-membername2-because-they-differ-by-the-types-of-optional-parameters"></a>'\<membername1 >' geçersiz kılınamaz '\<membername2 >', isteğe bağlı parametrelerinin türleri farklı olduğundan
Aşırı yüklenmiş üyelerin bir veya daha fazla gerekli parametreler için farklı veri türleri olmalıdır.  
  
 **Hata Kimliği:** BC30697  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   En az bir gerekli parametresi farklı veri türü olarak tanımlayın.
