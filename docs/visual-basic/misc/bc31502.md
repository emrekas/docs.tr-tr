# <a name="resource-name-resourcename-cannot-be-used-more-than-once"></a>Kaynak adı '\<KaynakAdı >' birden çok kez kullanılamaz
Belirtilen kaynak dinamik bağlantı kitaplığı veya yürütülebilir dosya adı birden çok kez kullanıldı.  
  
 **Hata Kimliği:** BC31502  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Benzersiz adı kaynakları.
