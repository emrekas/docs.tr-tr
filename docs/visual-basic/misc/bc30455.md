# <a name="argument-not-specified-for-parameter-parametername-of-methodname"></a>Parametresi için bağımsız değişken belirtilmemiş '\<parametername >', '\<yöntemAdı >'
Gerekli parametre için bir bağımsız değişken sağlamadınız.  
  
 **Hata Kimliği:** BC30455  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Belirtilen parametre için bir bağımsız değişken sağlayın.
