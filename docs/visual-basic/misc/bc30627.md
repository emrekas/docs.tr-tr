# <a name="option-statements-must-precede-any-declarations-or-imports-statements"></a>Bildirimleri ya da 'Imports' deyimleri 'Seçeneği' deyimleri gelmelidir
`Option` deyimleri, kaynak kodunuzu ilk yorum olmayan ifadeler olmalıdır.  
  
 **Hata Kimliği:** BC30627  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Taşıma `Option` en kaynak kodu, hemen önce `Imports` ve `Namespace` deyimleri.
