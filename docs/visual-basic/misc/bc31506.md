# <a name="typename-cannot-be-used-as-an-attribute-because-it-is-declared-mustinherit"></a>'\<typename >' 'MustInherit' olarak bildirildiğinden öznitelik olarak kullanılamaz
Özel öznitelik sınıfları olarak bildirilemez `MustInherit`.  
  
 **Hata Kimliği:** BC31506  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `MustInherit` değiştiricisi özel öznitelik.
