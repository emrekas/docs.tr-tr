# <a name="typename-cannot-be-used-as-an-attribute-because-it-is-declared-mustinherit"></a>'\<MustInherit ' olarak bildirildiği için ' TypeName > ' öznitelik olarak kullanılamaz
Özel öznitelik sınıfları olarak `MustInherit`bildirilemez.  
  
 **Hata KIMLIĞI:** BC31506  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Özel özniteliklerden `MustInherit` değiştiricisini kaldırın.
