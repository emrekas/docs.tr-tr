# <a name="interfacename1-cannot-implement-methodname-because-there-is-no-matching-method-on-interface-interfacename2"></a>'\<interfacename2>'arabiriminde\<> eşleşen\< \<bir yöntem olmadığından ' interfacename1 > ' ' MethodName > ' uygulayamaz
İlk arabirimin türetildiği arabirimde eşleşen bir yöntem olmadığında bir yöntemi arabirim üzerinde uygulamayı denediniz.  
  
 **Hata KIMLIĞI:** BC30401  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Metodun adını yanlış yazdığınızdan emin olun veya eksik yöntemi sağlayın.
