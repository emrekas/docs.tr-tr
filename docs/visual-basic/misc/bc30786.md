# <a name="declare-statements-in-a-module-cannot-be-declared-specifier"></a>Bir modül deyimlerinde bildirilemez bildirmek '\<belirticisi >'
A `Declare` bildirimi içinde geçersiz bir tanımlayıcısı olan bir `Module` bildirimi. Modüller asla oluşturulamaz, devralmayı desteklemeyen ve arabirimler uygulanamaz.  
  
 **Hata Kimliği:** BC30786  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Tanımlayıcısını kaldırın.
