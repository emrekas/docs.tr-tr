# <a name="ltdeclaration1gt-cannot-override-ltdeclaration2gt-because-they-have-different-access-levels"></a>'&lt;declaration1&gt;'geçersiz kılınamaz'&lt;declaration2&gt;' farklı erişim düzeylerine sahip oldukları
Aynı ada sahip bir devralınan öğe geçersiz kılmak bir yordam veya özellik bildiriminde çalışır, ancak devralınan öğeden farklı bir erişilebilirlik belirtir. Öğenin erişilebilirlik gibi devralınan `Public` veya `Private`, geçersiz kılma olarak korunmalıdır.  
  
 **Hata Kimliği:** BC30266  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Devralınan öğenin eşleşmesi için geçersiz kılma öğesi erişilebilirliğini değiştirin.
