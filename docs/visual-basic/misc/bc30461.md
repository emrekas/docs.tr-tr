# <a name="classes-cannot-be-declared-specifier"></a>Sınıfları bildirilemez '\<belirticisi >'
Bildirme girişiminde bulundunuz bir `Class` veya `Module` ile geçersiz bir tanımlayıcı.  
  
 **Hata Kimliği:** BC30461  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Tanımlayıcısını kaldırın.
