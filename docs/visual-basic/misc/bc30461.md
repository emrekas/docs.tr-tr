# <a name="classes-cannot-be-declared-ltspecifiergt"></a>Sınıfları bildirilemez '&lt;belirticisi&gt;'
Bildirme girişiminde bulundunuz bir `Class` veya `Module` ile geçersiz bir tanımlayıcı.  
  
 **Hata Kimliği:** BC30461  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Tanımlayıcısını kaldırın.
