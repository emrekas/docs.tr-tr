# <a name="implementsclause-cannot-implement-typename-because-typename-is-a-reserved-name"></a>'\<implementsclause >' uygulanamaz '\<typename >' için '\<typename >' ayrılmış bir addır
Bir `Implements` deyimi belirttiğiniz ayrılmış bir ad.  
  
 **Hata Kimliği:** BC31415  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Uygulanmamış arabirim üyesini yeniden adlandırın.
