# <a name="all-parameters-must-be-explicitly-typed-if-any-are"></a>Tüm parametreler varsa açıkça yazılmalıdır
Farklı türde bir işlev bildirimlerinde türü karışık.  
  
 **Hata Kimliği:** BC30529  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Tüm parametreleri açıkça yazdığınızdan emin olun.
