# <a name="mybase-cannot-be-used-with-method-methodname-because-it-is-declared-mustoverride"></a>' MustOverride ' olarak bildirildiği için ' MyBase '\<, ' MethodName > ' yöntemiyle kullanılamaz
`MyBase` Tanımlanmış`MustOverride`bir yöntemle kullanmayı denediniz.  
  
 **Hata KIMLIĞI:** BC30399  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `MustOverride` Bildirimi kaldırın.
