# <a name="mybase-cannot-be-used-with-method-methodname-because-it-is-declared-mustoverride"></a>'MyBase' yöntemiyle kullanılamaz '\<yöntemAdı >' 'MustOverride' olarak bildirildiğinden
Kullanma girişiminde bulundunuz `MyBase` bildirilmiş bir yöntemle `MustOverride`.  
  
 **Hata Kimliği:** BC30399  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Kaldırma `MustOverride` bildirimi.
