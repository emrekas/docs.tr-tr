# <a name="partial-method-methodname-cannot-use-the-implements-keyword"></a>Kısmi yöntemi '\<yöntemAdı >' 'Implements' anahtar sözcüğünü kullanamaz
Kısmi metot tanımına bir arabirimi veya arabirim üyesini uygulayamaz.  
  
 **Hata Kimliği:** BC31439  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Kaldırma `Implements` kısmi yöntem tanımında deyimi.
