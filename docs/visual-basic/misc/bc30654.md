# <a name="return-statement-in-a-function-get-or-operator-must-return-a-value"></a>'Return' deyimi işlev, Get veya işleci bir değer döndürmelidir
`Return` deyimleri, çağıran bir yordam için bir değer döndürmek için kullanılmalıdır. Kullanamazsınız `Return` başlarına deyimleri denetlemek için program akışı.  
  
 **Hata Kimliği:** BC30654  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  İşlev veya yordam döndürebilen bir değer belirtin.  
  
2.  Kullanım `End` güncel süreçte çıkmak program neden deyimi.
