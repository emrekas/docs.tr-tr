# <a name="type-arguments-unexpected"></a><span data-ttu-id="462dd-102">Tür bağımsız değişkenleri beklenmiyor</span><span class="sxs-lookup"><span data-stu-id="462dd-102">Type arguments unexpected</span></span>
<span data-ttu-id="462dd-103">Bir `Implements` yan tümcesinin tür bağımsız değişkenleri için uyguladığı arabirim üyesini sağlar.</span><span class="sxs-lookup"><span data-stu-id="462dd-103">An `Implements` clause supplies type arguments for the interface member it is implementing.</span></span>  
  
 <span data-ttu-id="462dd-104">`Implements` Yan tümcesi yalnızca tanımlamak ve arabirimini uyguladığı üye.</span><span class="sxs-lookup"><span data-stu-id="462dd-104">The `Implements` clause should only identify the interface and member it is implementing.</span></span> <span data-ttu-id="462dd-105">Bunun anlamı genel bir yordam bildiriliyorsa, `Of` yan tümcesi ve tür bağımsız değişkenlerini görünmelidir bildirimi ana bölümünde arabirimi yordamı uygulanmamasının, yaptıkları gibi.</span><span class="sxs-lookup"><span data-stu-id="462dd-105">This means that if you are declaring a generic procedure, the `Of` clause and the type arguments should appear in the main part of the declaration, just as they would if you were not implementing an interface procedure.</span></span>  
  
 <span data-ttu-id="462dd-106">Aşağıdaki kod, bu hata oluşturabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="462dd-106">The following code can generate this error.</span></span>  
  
```  
Public Interface testInterface  
    Sub testSub(Of t)()  
End Interface  
Public Class testClass  
    Implements testInterface  
    Public Sub testSub() Implements testInterface.testSub(Of t)()  
    End Sub  
End Class  
```  
  
 <span data-ttu-id="462dd-107">Yukarıdaki bildirimin `Implements` yan tümcesi, erişim veya yordam değiştiriciler olası eklenmesi dışında arabirim tanımı gibi görünmelidir.</span><span class="sxs-lookup"><span data-stu-id="462dd-107">The declaration preceding the `Implements` clause should look like the interface definition, except for the possible addition of access or procedure modifiers.</span></span> <span data-ttu-id="462dd-108">Aşağıdaki kod, hatayı önler.</span><span class="sxs-lookup"><span data-stu-id="462dd-108">The following code avoids the error.</span></span>  
  
```  
Public Sub testSub(Of t)() Implements testInterface.testSub  
```  
  
 <span data-ttu-id="462dd-109">**Hata Kimliği:** BC32088</span><span class="sxs-lookup"><span data-stu-id="462dd-109">**Error ID:** BC32088</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="462dd-110">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="462dd-110">To correct this error</span></span>  
  
-   <span data-ttu-id="462dd-111">Tür bağımsız değişken listesinden kaldırmak `Implements` yan tümcesi.</span><span class="sxs-lookup"><span data-stu-id="462dd-111">Remove the type argument list from the `Implements` clause.</span></span>  
  
-   <span data-ttu-id="462dd-112">Genel bir arabirim üyesi uyguluyor ve bildirimi ana bölümünde türü bağımsız değişken listesi yerleştirin, önceki `Implements` anahtar sözcüğü.</span><span class="sxs-lookup"><span data-stu-id="462dd-112">If you are implementing a generic member of the interface, then put the type argument list in the main part of the declaration, preceding the `Implements` keyword.</span></span>
