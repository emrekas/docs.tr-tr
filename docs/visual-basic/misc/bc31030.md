# <a name="project-level-conditional-compilation-constant-constantname-is-not-valid-error"></a>Proje düzeyi koşullu derleme sabiti '\<constantname >' geçerli değil: \<hata >
Proje için bir geçersiz koşullu derleme seçeneği belirtildi.  
  
 **Hata Kimliği:** BC31030  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Bu proje ile kullanılan koşullu derleme sabitleri doğrulayın.
