# <a name="argument-argumentname-is-not-a-valid-value"></a>Bağımsız değişken '\<argumentname >', geçerli bir değer değil
Bir bağımsız değişkenin değeri geçerli değil.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Bağımsız değişkenin değerini denetleyin.  
  
2.  Bağımsız değişken bir hesaplamanın sağlanırsa, hesaplama işlemleri değerleri kontrol edin.
