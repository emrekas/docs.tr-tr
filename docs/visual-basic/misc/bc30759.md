# <a name="referenced-membername-has-a-value-of-nothing"></a>Başvurulan '\<membername >' değerini 'Nothing'
Başvurulan öğesi olmayan bir değer atanmadan veya değer atandı `Nothing`.  
  
 **Hata Kimliği:** BC30759  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bu öğenin adını yazdığınızdan ve doğru şekilde ayarlanması doğrulayın.
