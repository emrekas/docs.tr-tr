# <a name="systemruntimeinteropservicesdllimportattribute-cannot-be-applied-to-a-get-or-set"></a>'System.Runtime.InteropServices.DllImportAttribute' bir 'Get' veya 'Set' uygulanamaz
`DllImportAttribute` Özniteliği uygulandığı bir `Get` veya `Set` özellik yordamı.  
  
 **Hata Kimliği:** BC31524  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Kaldırma `DllImportAttribute` gelen `Get` ve `Set` özellik yordamları.
