# <a name="set-statement-must-end-with-a-matching-end-set"></a>'Set' deyimi eşleşen bir 'End Set' ile bitmelidir
`Set` özellik yordamları sonlanmalı bir `End Set` deyimi.  
  
 **Hata Kimliği:** BC30633  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Emin olun `Set` özellik yordamı ile sona erer bir `End Set` oluşturun.
