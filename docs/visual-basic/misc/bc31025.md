# <a name="unable-to-open-key-file-ltfilenamegt-lterrorgt"></a>Anahtar dosyası açılamıyor. '&lt;filename&gt;': &lt;hata&gt;
Bu proje için belirtilen anahtar dosyası açılamadı.  
  
 **Hata Kimliği:** BC31025  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Anahtar dosyası mevcut olduğundan ve başka bir uygulama tarafından kilitlenmediğinden emin olun.
