# <a name="response-file-filename-included-multiple-times"></a>Yanıt dosyası \<filename > birden çok defa eklendi
Bir yanıt dosyası birden çok kez belirtildi. Yanıt dosyaları, derleyici seçenekleri ve kaynak kodu dosyalarını derlemek için içerir.  
  
 **Hata Kimliği:** BC2003  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yedekli yanıt dosya tanımlayıcıları kaldırın.
