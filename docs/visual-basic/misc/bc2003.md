# <a name="response-file-ltfilenamegt-included-multiple-times"></a>Yanıt dosyası &lt;filename&gt; birden çok defa eklendi
Bir yanıt dosyası birden çok kez belirtildi. Yanıt dosyaları, derleyici seçenekleri ve kaynak kodu dosyalarını derlemek için içerir.  
  
 **Hata Kimliği:** BC2003  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yedekli yanıt dosya tanımlayıcıları kaldırın.
