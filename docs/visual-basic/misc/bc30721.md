# <a name="evaluation-of-expression-or-statement-stopped"></a>İfade veya deyimin değerinin durduruldu
**Durdur değerlendirme** menü öğesi seçildiğinde, değerlendirme vermemesine neden oluyor.  
  
 **Hata Kimliği:** BC30721  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Seçmeden devam etmek için yürütmeye izin **Durdur değerlendirme**.
