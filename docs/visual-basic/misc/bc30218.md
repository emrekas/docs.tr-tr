# <a name="lib-expected"></a>'Lib' bekleniyor
A `Declare` deyimi olmadan bir kitaplık adı belirtir `Lib` anahtar sözcüğü.  
  
 **Hata Kimliği:** BC30218  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Ekleme `Lib` kitaplığı adından önce anahtar sözcüğü.
