# <a name="bad-guid-format"></a>Hatalı GUID biçimi
Bir GUID ile yanlış bir biçimde sağlanmış.  
  
 **Hata Kimliği:** BC42035  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   GUID biçimini denetleyin.
