# <a name="xml-comment-cannot-be-applied-more-than-once-on-a-partial-type"></a>XML açıklaması uygulanamaz birden çok kez kısmi \<türü >
XML açıklaması uygulanamaz birden çok kez kısmi \<türü >. Bu XML açıklamaları \<türü > yok sayılacak.  
  
 Bir XML açıklama bloğu kısmi bir türde yalnızca bir kısmını görünür.  
  
 Bir XML açıklama bloğu kısmi bir türde birden fazla bölümü görünürse, her açıklama bloğu için bu uyarı oluşturulur ve en üst düzey XML açıklaması yoksayılacak.  
  
 **Hata Kimliği:** BC42314  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Fazlalık açıklama blokları kaldırın.
