# <a name="end-of-parameter-list-expected"></a>Beklenen parametre listesinin sonuna
A `ParamArray` belirtimi bir veya daha fazla diğer bağımsız değişken belirtimleri bir yordam bildirimi ardında.  
  
 **Hata Kimliği:** BC30192  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Taşıma `ParamArray` bağımsız değişken listesinin sonuna.
