# <a name="xml-comment-block-must-immediately-precede-the-language-element-to-which-it-applies"></a>XML açıklama bloğu uygulandığı dil öğesi hemen önce gelmelidir
XML açıklama bloğu uygulandığı dil öğesi hemen önce gelmelidir. XML açıklaması yoksayılacak.  
  
 Bu hata, bir etiketi yanlış yerleştirilmiş ya da aksi takdirde hatalı ise de gerçekleşir.  
  
 **Hata Kimliği:** BC42300  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Açıklama bloğu uygulandığı dil öğesi öncesinde taşıyın. Fazlalık karakter yanlışlıkla başlangıç etiketi önce eklenmiş değil, emin olun.  
  
2.  Etiket geçerli olduğundan emin olun.
