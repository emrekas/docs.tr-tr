# <a name="implemented-type-must-be-an-interface"></a>Uygulanan türü bir arabirim olmalıdır
Bir `Implements` deyimi bir arabirim belirtmiyor. Bir sınıfı yalnızca arabirim uygulayabilir.  
  
 **Hata Kimliği:** BC30232  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Arabirim adının doğru yazıldığından emin olun.  
  
2.  Deyimi bir sınıf belirtiyorsa, kullanmayı `Inherits` deyimi.
