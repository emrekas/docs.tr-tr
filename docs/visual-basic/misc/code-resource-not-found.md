# <a name="code-resource-not-found"></a>Kod kaynak bulunamadı
Bir yordamda bir kod kaynak için bir çağrı yapıldı, ancak kod kaynağı bulunamadı.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaynak kullanılabilir ve düzgün şekilde başvurulan olup olmadığını denetleyin.
