# <a name="multi-dimensional-array-cannot-be-converted-to-an-expression-tree"></a><span data-ttu-id="5e45a-102">Çok boyutlu dizi bir ifade ağacına dönüştürülemez</span><span class="sxs-lookup"><span data-stu-id="5e45a-102">Multi-dimensional array cannot be converted to an expression tree</span></span>
<span data-ttu-id="5e45a-103">Çoğu ifadeleri ifade ağaçlarına dönüştürülebilir, ancak çok boyutlu diziler olamaz.</span><span class="sxs-lookup"><span data-stu-id="5e45a-103">Most expressions can be converted to expression trees, but multi-dimensional arrays cannot be.</span></span> <span data-ttu-id="5e45a-104">Örneğin, aşağıdaki kod bu hataya neden olur:</span><span class="sxs-lookup"><span data-stu-id="5e45a-104">For example, the following code causes this error:</span></span>  
  
```vb  
Module Module1  
    Sub Main()  
        '' A multi-dimensional array cannot be converted.  
        'Dim expTree As Expressions.Expression(Of Func(Of Object)) = _  
        '    Function() New Integer(1, 1) {{1, 2}, {2, 3}}  
  
        ' A one-dimensional array can be converted.  
        Dim expTree2 As Expressions.Expression(Of Func(Of Object)) = _  
            Function() New Integer() {1, 2, 3}  
    End Sub  
End Module  
```
