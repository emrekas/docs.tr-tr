# <a name="the-keyword-keyword-is-used-to-overload-inherited-members-do-not-use-the-keyword-keyword-when-overloading-sub-new"></a>'\<Anahtar sözcüğü > ' anahtar sözcüğü devralınan üyeleri aşırı yüklemek için kullanılır; ' Sub New '\<öğesini aşırı yüklerken ' anahtar sözcüğü > ' anahtar sözcüğünü kullanmayın
Bir Oluşturucu `Overloads` anahtar sözcüğüyle birlikte bildirilmiştir.  
  
 Visual Basic oluşturucuların devralınmasını veya aşırı yüklenmesini desteklemez.  
  
 **Hata KIMLIĞI:** BC32040  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Tüm Oluşturucu bildirimlerinden anahtar sözcüğünü kaldırın. `Overloads`
