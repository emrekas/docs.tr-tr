# <a name="unable-to-load-referenced-library-ltfilenamegt-lterrorgt"></a>Başvurulan kitaplığı yüklenemiyor '&lt;filename&gt;': &lt;hata&gt;
Bu proje tarafından başvurulan bir kitaplığı yüklenemedi.  
  
 **Hata Kimliği:** BC31011  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Dosyanın belirtilen konumda olduğundan ve başka bir uygulama tarafından kilitlenmediğinden emin olun.
