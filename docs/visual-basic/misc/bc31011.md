# <a name="unable-to-load-referenced-library-filename-error"></a>Başvurulan kitaplığı yüklenemiyor '\<dosya adı >': \<hata >
Bu proje tarafından başvurulan bir kitaplığı yüklenemedi.  
  
 **Hata Kimliği:** BC31011  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Dosyanın belirtilen konumda olduğundan ve başka bir uygulama tarafından kilitlenmediğinden emin olun.
