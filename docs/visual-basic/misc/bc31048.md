# <a name="properties-with-no-required-parameters-cannot-be-declared-default"></a>Özellikleri gerekli parametre olmadan 'Default' olarak bildirilemez
Bir özellik Gerekli Parametreler olmadan ancak ile belirtilen `Default` değiştiricisi.  
  
 **Hata Kimliği:** BC31048  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Varsayılan özellikler için gerekli parametreler bildirin.  
  
- Kaldırma `Default` özelliği tanımından değiştiricisi.
