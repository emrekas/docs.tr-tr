# <a name="withevents-variable-does-not-raise-any-instance-events-that-are-accessible-to-containername"></a>'WithEvents değişkeni olmayan olaylar için erişilebilir olan tüm örnek'\<containername >'
Başvurduğunuz nesne erişilebilir olayları oluşturmaz.  
  
 **Hata Kimliği:** BC30600  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Erişim değiştiricileri olayı değiştirin veya geçerli bağlamda erişilebilir olan bir olay seçin.
