# <a name="for-loop-control-variable-variablename-already-in-use-by-an-enclosing-for-loop"></a>Döngü denetim değişkeni için '\<variablename >' kapsayan bir For döngüsü tarafından zaten kullanılan
İç içe geçmiş `For` döngüler tüm benzersiz denetim değişkenleri kullanmanız gerekir.  
  
 **Hata Kimliği:** BC30069  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Denetim değişkeni adlarından birini değiştirin.
