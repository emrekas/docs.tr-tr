# <a name="for-loop-control-variable-ltvariablenamegt-already-in-use-by-an-enclosing-for-loop"></a>Döngü denetim değişkeni için '&lt;variablename&gt;' kapsayan bir For döngüsü tarafından zaten kullanılan
İç içe geçmiş `For` döngüler tüm benzersiz denetim değişkenleri kullanmanız gerekir.  
  
 **Hata Kimliği:** BC30069  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Denetim değişkeni adlarından birini değiştirin.
