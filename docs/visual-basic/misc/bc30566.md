# <a name="array-initializer-has-too-many-dimensions"></a>Dizi başlatıcıda çok fazla boyuta sahip
Başlatıcı çok boyutlarında diziniz için sağladınız.  
  
 **Hata Kimliği:** BC30566  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaç tane boyutları gerekli olduğunu belirlemek için bir dizi Başlatıcısı denetleyin.  
  
-   Fazladan bağımsız değişken veya bağımsız değişkenleri kaldırın.
