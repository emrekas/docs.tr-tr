# <a name="end-sub-expected"></a>'End Sub' bekleniyor
Bir `End` deyimi sonlandırmasına blok eşleşmesi gerekir. A `Sub` yordam tanımında değil düzgün bir şekilde sonlandırıldı.  
  
 **Hata Kimliği:** BC30026  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Ekleme bir `End Sub` yordamı sonuna deyimi.
