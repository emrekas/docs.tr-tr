# <a name="internal-error-in-the-microsoft-visual-basic-runtime"></a>Microsoft Visual Basic çalışma zamanı iç hata
Microsoft Visual Basic çalışma zamanı'nda iç bir hata oluştu.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Altında hatanın gerçekleştiği durumlarda dikkat edin ve Microsoft Ürün Destek Hizmetleri'ni arayın.
