# <a name="this-inheritance-causes-circular-dependencies-between-type1-typename1-and-its-nested-type2-typename2"></a><span data-ttu-id="28170-102">Bu devralma arasında döngüsel bağımlılıklara neden \<type1 > '\<typename1 >' ve onun iç içe \<type2 > '\<typename2 >'</span><span class="sxs-lookup"><span data-stu-id="28170-102">This inheritance causes circular dependencies between \<type1> '\<typename1>' and its nested \<type2> '\<typename2>'</span></span>
<span data-ttu-id="28170-103">İç içe geçmiş sınıflar arasında döngüsel bağımlılık, bir devralma yapısı, diğer bir deyişle, iki sınıf birbirinden devralan sonuçlanır.</span><span class="sxs-lookup"><span data-stu-id="28170-103">An inheritance structure results in circular dependency among nested classes, that is, two classes inheriting from each other.</span></span>  
  
 <span data-ttu-id="28170-104">Aşağıdaki kod, bu hata iletisi oluşturabilir.</span><span class="sxs-lookup"><span data-stu-id="28170-104">The following code can generate this error message.</span></span>  
  
```  
Public Class c1  
    Inherits c3.c4  
    Public Class c2  
    End Class  
End Class  
Public Class c3  
    Inherits c1.c2  
    Public Class c4  
    End Class  
End Class  
```  
  
 <span data-ttu-id="28170-105">Önceki kodda, sınıf `c1` sınıfından devralan `c4`, ancak `c4` içinde iç içe `c3`, işlevinden devralan `c2`, içinde iç içe geçmiş `c1`.</span><span class="sxs-lookup"><span data-stu-id="28170-105">In the preceding code, class `c1` inherits from class `c4`, but `c4` is nested inside `c3`, which inherits from `c2`, nested inside `c1`.</span></span>  
  
 <span data-ttu-id="28170-106">**Hata Kimliği:** BC30907</span><span class="sxs-lookup"><span data-stu-id="28170-106">**Error ID:** BC30907</span></span>
