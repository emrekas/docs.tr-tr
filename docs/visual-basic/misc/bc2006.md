# <a name="option-optionname-requires-argument"></a>Seçenek \<SeçenekAdı > gerektirir \<bağımsız değişken >

 **Hata Kimliği:** BC2006  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Eksik bağımsız değişken sağlayın.
