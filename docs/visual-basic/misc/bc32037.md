# <a name="next-statement-names-more-variables-than-there-are-matching-for-statements"></a><span data-ttu-id="68804-102">'İleri' deyimi 'For' deyimleri eşleşen var. daha fazla değişken adları</span><span class="sxs-lookup"><span data-stu-id="68804-102">'Next' statement names more variables than there are matching 'For' statements</span></span>
<span data-ttu-id="68804-103">İç içe döngüleri, tek bir sonlandırılır `Next` daha fazla loop değişkenleri aşağıdaki örnekteki gibi iç içe döngüleri kullandığınızdan ifadesi:</span><span class="sxs-lookup"><span data-stu-id="68804-103">Nested loops are terminated with a single `Next` statement that specifies more loop variables than there are loops in the nest, as in the following example:</span></span>  
  
```  
For I = 1 To 10  
   For J = 1 To 5  
      ' ...  
Next J, I, K   ' Next J, I is valid, but there is no loop on K.  
```
