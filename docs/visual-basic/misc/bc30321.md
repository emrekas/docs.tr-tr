# <a name="case-cannot-follow-a-case-else-in-the-same-select-statement"></a>'Case' bir 'Case Else' aynı 'Select' deyimindeki izleyemez.
A `Case Else` deyimi için ilk eşleşme bulunursa çalıştırılacak deyimleri ortaya çıkarır `Case`. A `Case` ifade, sonra bulundu bir `Case Else` aynı `Select` blok.  
  
 **Hata Kimliği:** BC30321  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Taşıma `Case Else` sonra uygun konuma `Case` deyimi.
