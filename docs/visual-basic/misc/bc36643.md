# <a name="array-modifiers-cannot-be-specified-on-lambda-expression-parameters-name-only-on-its-type"></a>Türünü yalnızca üzerinde lambda ifadesi parametre adı dizi değiştiricileri belirtilemez
Lambda ifadeleri dizisi bağımsız değişkenlerinin gönderilebilir, ancak parametre bildirimi, öğe türü içermelidir.  
  
```vb  
' Not valid.  
' Dim arrayExample1 = Function(arrayPara()) 2  
  
' Valid.  
Dim arrayExample2 = Function(arrayPara() As Integer) arrayPara.Length > 0  
Dim arrayexample3 = Function(arrayPara As Integer()) arrayPara.Length > 0  
```  
  
 **Hata Kimliği:** BC36643  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Dizi parametreyi öğelerin türünü belirtin.
