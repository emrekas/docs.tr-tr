# <a name="loop-statements-are-not-valid-in-the-immediate-window"></a>Döngü deyimler yürütme penceresinde geçerli değildir
`Loop` ifadeleri kullanılamaz **hemen** penceresi.  
  
 **Hata Kimliği:** BC30708  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `Loop` kodda hata ayıklamayı deyimlerini.
