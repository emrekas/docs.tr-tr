# <a name="if-block-must-end-with-a-matching-end-if"></a>'#If' bloğu, eşleşen bir '#End If' ile bitmelidir.
`#If` Koşullu derleme yönergesi ' dir. Bir `#If` blok tarafından sonlandırıldı değil bir `#End If` yönergesi.  
  
 **Hata Kimliği:** BC30012  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Ekleme bir `#End If` yönergesini koşullu derleme bloğunun sonu.
