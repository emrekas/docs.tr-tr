# <a name="raiseevent-declaration-must-end-with-a-matching-end-raiseevent"></a>'RaiseEvent' bildirimi eşleşen bir 'End RaiseEvent' ile bitmelidir
A `RaiseEvent` bildirimi sonlanmalı bir `End RaiseEvent` deyimi.  
  
 **Hata Kimliği:** BC31117  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bu maddeyi `RaiseEvent` bildirimi ile sona erer bir `End RaiseEvent` deyimi.
