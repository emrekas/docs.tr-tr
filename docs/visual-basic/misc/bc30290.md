# <a name="local-variable-cannot-have-the-same-name-as-the-function-containing-it"></a>Yerel değişken içeren işlevi ile aynı ada sahip olamaz
A `Dim` deyimi ile aynı ada sahip bir değişken belirtir `Function` içeren yordam.  
  
 **Hata Kimliği:** BC30290  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Değişken bildirimi kaldırın veya değişken adını değiştirin.
