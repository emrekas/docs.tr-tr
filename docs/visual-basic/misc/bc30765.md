# <a name="all-projects-in-a-visual-basic-solution-must-target-the-same-platform-but-the-project-you-are-trying-to-add-targets-a-platform-other-than-the-one-specified-by-your-solution"></a>Bir Visual Basic Çözümdeki tüm projeleri aynı platforma hedeflemesi gerekir, ancak eklemeye çalıştığınız proje çözümünüz tarafından belirtilen dışındaki bir platformu hedefleyen
Bir Visual Basic Çözümdeki tüm projeleri aynı platforma hedeflemesi gerekir, ancak proje çözümünüz tarafından farklı bir platform belirtilen hedefleri ekliyoruz.  
  
 **Hata Kimliği:** BC30765  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Yalnızca çözümünüz tarafından belirtilen aynı platformu hedefleyen projeler ekleyin.
