# <a name="unrecognized-option-optionname-ignored"></a>Tanınmayan seçenek \<SeçenekAdı >; yoksayıldı
Komut satırından geçerli olmayan bir seçenek belirtildi.  
  
 **Hata Kimliği:** BC2007  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Geçerli olanı bulmak için belirtilen seçenekler denetleyin.
