# <a name="method-cannot-have-both-a-paramarray-and-optional-parameters"></a>Yöntemi, hem ParamArray hem de isteğe bağlı parametrelere sahip olamaz
A `ParamArray` bağımsız değişkeni, otomatik olarak isteğe bağlıdır ve yordam bildirimi yalnızca isteğe bağlı bağımsız değişken olmalıdır. Önceki tüm bağımsız değişkenleri gereklidir.  
  
 **Hata Kimliği:** BC30046  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   İsteğe bağlı bağımsız değişkenleri bildiriminde kaldırın.
