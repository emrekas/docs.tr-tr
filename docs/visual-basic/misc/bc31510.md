# <a name="attribute-constant-constantname-cannot-be-the-target-of-an-assignment"></a>Öznitelik sabiti '\<constantname >' atama işleminin hedefi olamaz
Öznitelik içinde bildirilen bir sabit bir değer atamak için girişimde bulunuldu.  
  
 **Hata Kimliği:** BC31510  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Hatalı atamasını kaldırın.  
  
-   Geliştirdiğiniz özel öznitelikler kullanıyorsanız, öznitelik üyesi bir alan olarak yeniden tanımlama.
