# <a name="attribute-constant-constantname-cannot-be-the-target-of-an-assignment"></a>'\<Constantname > ' öznitelik sabiti bir atamanın hedefi olamaz
Bir öznitelikte belirtilen bir değere değer atama girişiminde bulunuldu.  
  
 **Hata KIMLIĞI:** BC31510  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Hatalı atamayı kaldırın.  
  
- Geliştirdiğiniz özel öznitelikler kullanıyorsanız, öznitelik içindeki üyeyi bir alan olarak yeniden tanımlayın.
