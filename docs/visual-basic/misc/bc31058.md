# <a name="conflicts-with-name1-which-is-implicitly-created-for-name2-in-type-name3"></a>\<> '\< \<Name3 > ' türünde ' AD2 > ' için örtük olarak oluşturulan ' name1 > ' ile çakışıyor\<
Bir tür üyesinin adı başka bir örtük olarak oluşturulan üyeyle çakışıyor.  
  
 **Hata KIMLIĞI:** BC31058  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Adlandırma çakışmasını kaldırmak için açıkça belirtilen üyeyi yeniden adlandırın.
