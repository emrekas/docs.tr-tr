# <a name="syntax-error-visual-basic-error"></a>Sözdizimi hatası (Visual Basic hatası)
Kaynak kod satırı belirli olmayan bir sözdizimi hatası oluşturur.  
  
 **Hata Kimliği:** BC30035  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Her kaynak kod satırını kullanılan anahtar sözcükler için belgeleri inceleyin. Örnek kod, bu hata oluşturmadan kaynak satırı karşı belgeleri karşılaştırın.  
  
2. Hatanın nedenini belirleyemez koşullar hakkında bilgi toplamak ve Microsoft Ürün Destek Hizmetleri bildirir.
