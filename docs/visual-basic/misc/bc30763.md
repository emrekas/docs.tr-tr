# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-using-the-ansi-auto-or-unicode-modifier"></a>.NET Compact Framework hedef sürümü, ANSI, otomatik veya Unicode değiştiricisini kullanarak desteklemiyor
.NET Compact Framework'ı birlikte çalıştığınız sürümü desteklemediği `Ansi`, `Auto`, ve `Unicode` değiştiriciler.  
  
 **Hata Kimliği:** BC30763  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Değiştiricisi kaldırın.
