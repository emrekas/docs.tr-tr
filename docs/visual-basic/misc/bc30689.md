# <a name="statement-cannot-appear-outside-of-a-method-bodymultiline-lambda"></a>Dışında bir yöntem gövdesi/çok satırlı lambda deyimi yer alamaz
Executable deyimleri yordamlar kullanılmalıdır.  
  
 **Hata Kimliği:** BC30689  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bir yordam veya alt yordam bildirimi taşıyın.
