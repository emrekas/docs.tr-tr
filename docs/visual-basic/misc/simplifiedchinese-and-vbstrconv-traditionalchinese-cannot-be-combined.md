# <a name="simplifiedchinese-and-vbstrconvtraditionalchinese-cannot-be-combined"></a>SimplifiedChinese a VbStrConv.TraditionalChinese nelze kombinovat
Uygulamanızı birleştirmek çalışıyor `VbStrConv` numaralandırma üyelerini `SimplifiedChinese` ve `TraditionalChinese`, hangi karşılıklı olarak birbirini dışlar.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Ya da kaldırma `VbStrConv.SimplifiedChinese` veya `VbStrConv.TraditionalChinese`.
