# <a name="method-declaration-statements-must-be-the-first-on-a-logical-line"></a>Bildirim deyimleri yöntemi ilk mantıksal bir satırda olmalıdır
A `Function` veya `Sub` deyimi bir iki nokta üst üste (:) deyim ayırıcısı izler. `Function` veya `Sub` , kaynak satırında tek deyim olması gerekir.  
  
 **Hata Kimliği:** BC32009  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Birden çok deyime farklı satırlarına ayırın.
