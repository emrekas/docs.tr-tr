# <a name="character-is-not-valid"></a>Karakter, geçerli değil.
Geçerli bağlam bu karakteri izin vermez. Örneğin, bir tilde (`~`) ya da dikey çubuk (`|`) bildirilen öğe adları parçası olamaz.  
  
 **Hata Kimliği:** BC30037  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Programlama öğesine bir ek karakter yanlışlıkla takılıp denetleyin.  
  
2.  Programlama öğesine yazıldığından veya öğe, bu tür için geçerli olan karakterler kullanarak kontrol edin.
