# <a name="modifier-is-not-valid-on-a-delegate-declaration"></a>'\<değiştiricisi >' Delegate bildiriminde geçerli değil
Bir değiştirici gibi kullanma girişiminde `Shared`, üzerinde bir temsilci bildirimi. Temsilciler, nesne yöntemi belirtimi geçirilen bir oluşturucu tanımlayarak diğer nesnelerin yöntemlerini çağırmak için kullanılan nesneleridir. Delegate bildiriminde bir değiştirici belirtmek için geçerli değil.  
  
 **Hata Kimliği:** BC30385  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Değiştiricisi kaldırın.
