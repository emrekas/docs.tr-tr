# <a name="parameter-parametername-in-extension-method-methodname-defined-in-typename-already-has-a-matching-argument"></a>Parametre '\<parametername >' genişletme yöntemindeki '\<yöntemAdı >' içinde tanımlanan '\<typename >' zaten bir eşleşen bağımsız değişkene sahip
Bir bağımsız değişken adıyla aynı yordam çağrısında birden çok kez geçirilir.  
  
 **Hata Kimliği:** BC36584  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bir bağımsız değişken adları yanlış olup olmadığını denetleyin.  
  
-   Yinelenen bağımsız değişken bağımsız değişken listesinden kaldırın.
