# <a name="option-win32manifest-ignored"></a>/ Win32manifest seçeneği yoksayıldı
/ Win32manifest seçeneği yoksayıldı. Yalnızca hedef bir derleme olduğunda belirtilebilir.  
  
 Visual Basic Derleyicisi geçirilen `/win32manifest` derleyici seçeneği `/target` seçeneği `module`.  
  
 **Hata Kimliği:** BC2034  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Kaldırma `/win32manifest` derleyici seçenek veya seçenek kümesinden `/target` seçeneğini `exe`, `winexe`, veya `library`.
