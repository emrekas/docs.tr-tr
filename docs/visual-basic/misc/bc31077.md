# <a name="assemblyname-is-an-assembly-and-cannot-be-referenced-as-a-module"></a>'\<assemblyname >' bir derlemedir, modül olarak başvurulamaz
Bir derlemeyi bir modül olarak kullanmak için girişimde bulunuldu.  
  
 **Hata Kimliği:** BC31077  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bir modüle başvuru değiştirin.
