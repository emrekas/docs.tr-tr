# <a name="removehandler-declaration-must-end-with-a-matching-end-removehandler"></a>'RemoveHandler' bildirimi eşleşen bir 'End RemoveHandler' ile bitmelidir
A `RemoveHandler` bildirimi sonlanmalı bir `End RemoveHandler` deyimi.  
  
 **Hata Kimliği:** BC31116  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bu maddeyi `RemoveHandler` bildirimi ile sona erer bir `End RemoveHandler` deyimi.
