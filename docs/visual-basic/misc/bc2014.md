# <a name="the-value-value-is-invalid-for-option-optionname"></a>değer '\<değer >' seçeneği için geçersiz '\<SeçenekAdı >'
Geçerli olmayan bir komut satırı seçeneği bir değer sağlandı.  
  
 **Hata Kimliği:** BC2014  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Geçerli olanı bulmak için değerleri kontrol edin.
