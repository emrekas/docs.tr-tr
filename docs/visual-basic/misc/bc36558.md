<span data-ttu-id="28034-112">Aşağıdaki örnek, yeni `AttributeUsage` `ExtensionAttribute` sürümünün hangi program öğelerine uygulanacağını belirtmek için özniteliğini kullanır.</span><span class="sxs-lookup"><span data-stu-id="28034-112">The following example uses the `AttributeUsage` attribute to specify which program elements the new version of `ExtensionAttribute` can apply to.</span></span> <span data-ttu-id="28034-113">Örnek, `AttributeTargets` numaralandırmanın üç üyesini belirtir: `Assembly`, `Class`ve `Method`.</span><span class="sxs-lookup"><span data-stu-id="28034-113">The example specifies three members of the `AttributeTargets` enumeration: `Assembly`, `Class`, and `Method`.</span></span> <span data-ttu-id="28034-114">Bu öğelerden herhangi birinin atlanması bu hataya neden olur.</span><span class="sxs-lookup"><span data-stu-id="28034-114">The omission of any one of these elements will cause this error.</span></span>
