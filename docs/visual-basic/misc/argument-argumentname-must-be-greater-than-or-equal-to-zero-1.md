# <a name="argument-argumentname-must-be-greater-than-or-equal-to-zero"></a>Bağımsız değişken \<argumentname > değerinden büyük veya sıfıra eşit olmalıdır
Bir bağımsız değişken değeri sıfırdan küçüktür. Bağımsız değişken değerleri büyük veya sıfıra eşit olmalıdır.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Bağımsız değişkenin değeri büyük veya sıfıra eşit olduğundan emin olun.  
  
2.  Bağımsız değişken bir hesaplamanın sağlanırsa, hesaplama tarafından işlenmekte olan değerleri kontrol edin.
