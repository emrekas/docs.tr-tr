# <a name="xml-declaration-does-not-allow-attribute-attributename"></a>XML bildirimi 'attributeName' özniteliğine izin vermiyor
Bir XML belgesi değişmez değeri için geçersiz bir öznitelik belirtilmedi. Bir XML belgesi değişmez değeri geçerli bir öznitelik değerleri, XML 1.0 belirtimi temel alır.  
  
 **Hata Kimliği:** BC31154  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Geçersiz öznitelik XML belgesi değişmez değeri kaldırın.
