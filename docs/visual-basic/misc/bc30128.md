# <a name="security-attribute-ltattributenamegt-is-not-valid-lterrorgt"></a>Güvenlik özniteliğinin '&lt;attributename&gt;' geçerli değil: &lt;hata&gt;
Ortak dil çalışma zamanı öznitelik, bir güvenlik bağlamı kabul etmiyor.

**Hata Kimliği:** BC30128

## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için

1. Bildirimden özniteliğini kaldırın veya bağlamı için geçerli bir öznitelik ile değiştirin.

2. Tırnak işaretli hata iletisini inceleyin ve uygun eylemi gerçekleştirin.
