 <span data-ttu-id="83792-104">Sayaç değişkeninde bir `For` döngü tüm aşağıdaki işleçleri destekler tüm veri türü olabilir:</span><span class="sxs-lookup"><span data-stu-id="83792-104">The counter variable in a `For` loop can be of any data type that supports all of the following operators:</span></span>

-   <span data-ttu-id="83792-107">Toplama (`+`)</span><span class="sxs-lookup"><span data-stu-id="83792-107">Addition (`+`)</span></span>  
  
-   <span data-ttu-id="83792-108">Çıkarma (`-`)</span><span class="sxs-lookup"><span data-stu-id="83792-108">Subtraction (`-`)</span></span>

 <span data-ttu-id="83792-111">Veri türlerini de unutmayın `start`, `end`, ve `step` ifadelerinde `For` deyimi sayaç değişkeni veri türünü genişletmeniz gerekir.</span><span class="sxs-lookup"><span data-stu-id="83792-111">Note also that the data types of the `start`, `end`, and `step` expressions in the `For` statement must widen to the data type of the counter variable.</span></span> <span data-ttu-id="83792-112">Sayaç değişkeni, bir kullanıcı tanımlı sınıf veya yapı olup olmadığını ve `start`, `end`, veya `step` farklı türde bir ifadedir, tanımlamanız gerekir `CType` gerekli dönüştürme gerçekleştirmek için dönüştürme işleci.</span><span class="sxs-lookup"><span data-stu-id="83792-112">If the counter variable is a user-defined class or structure and the `start`, `end`, or `step` expression is of a different type, you must define the `CType` conversion operator to accomplish the necessary conversion.</span></span>
