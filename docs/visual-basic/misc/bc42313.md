# <a name="xml-comment-tag-returns-is-not-permitted-on-a-writeonly-property"></a>Bir 'WriteOnly' özelliğinde 'returns' XML açıklama etiketine izin verilmez
Bir 'WriteOnly' özelliğinde 'returns' XML açıklama etiketine izin verilmez. XML açıklaması yoksayılacak.  
  
 XML açıklaması için salt yazılır özellik bildirimi içeremez bir \<döndürür > etiketi.  
  
 **Hata Kimliği:** BC42313  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Desteklenmeyen kaldırmak \<döndürür > etiketi.
