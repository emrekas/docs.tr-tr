# <a name="properties-declared-readonly-cannot-have-a-set"></a>Özellikleri olarak bildirilen bir 'Set' 'ReadOnly' bulunamaz
`Set` Yordamı bir özelliğin değerini yazar. `ReadOnly` Özellikler yazılamaz.  
  
 **Hata Kimliği:** BC30022  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Kaldırma `ReadOnly` from anahtar sözcüğü `Property` deyimi veya kaldırma `Set` yordamdan özellik gövdesi.
