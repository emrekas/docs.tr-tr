# <a name="name-is-ambiguous-in-the-application-objects-list"></a>'\<adı >' uygulama nesneleri belirsiz\<listesi >'
Belirsiz ve bu nedenle çakışan bir ad başka bir adla sağladınız. Visual Basic Derleyicisi, herhangi bir çakışma çözümleme kurallarını yok; adları kendiniz belirsizliğinin gerekir. Liste adı belirsiz olduğu uygulama nesneleri hata iletisi ayrıntılarında sağlanmıştır.  
  
 **Hata Kimliği:** BC30563  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Adı tam olarak nitelendirin.
