# <a name="types-are-not-available-in-this-context"></a>Tür bu bağlamda kullanılamaz
Bir tür içinde başvuru girişiminde bir `#If` yönergesi.  
  
 **Hata Kimliği:** BC30458  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Başvuruyu kaldırın.
