# <a name="keyword-is-not-valid-as-an-identifier"></a>Anahtar sözcük, tanımlayıcı olarak geçerli değil.
Bir anahtar sözcüğü, bir öğe adı gerekli olduğu gerçekleşir. Bir değişken veya yordam gibi bir öğe için bağlam çağırır.  
  
 **Hata Kimliği:** BC30183  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Anahtar sözcüğü bir geçerli öğe adıyla değiştirin.
