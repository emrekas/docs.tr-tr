# <a name="array-dimensions-do-not-match-those-specified-in-the-vbfixedarray-attribute"></a>Dizi boyutları ' VBFixedArray ' özniteliğinde belirtilenler ile eşleşmiyor
Dizinizdeki Boyutlar, `VBFixedArray` özniteliğinde belirttiğiniz boyutlarla eşleşmez.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Dizi `Dim` deyimindeki dizi boyutlarının `VBFixedArray` öznitelik tarafından belirtilen boyutlarla eşleştiğinden emin olun.
