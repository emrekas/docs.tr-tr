 `Public Class testClass(Of t1 As {Structure, t2}, t2 As Class)`  
  
 <span data-ttu-id="27b78-105">Doğrudan kısıtlaması `Structure` dolaylı kısıtlama `Class` tür parametresi için bir çakışmaya neden `t1`, çünkü `Structure` kısıtlaması gerekli karşılık gelen tür bağımsız değişkeni bir değer türü olması gerektiğini sırada `Class` bir başvuru türü olmasını gerektirir.</span><span class="sxs-lookup"><span data-stu-id="27b78-105">The direct constraint `Structure` and the indirect constraint `Class` cause a conflict for type parameter `t1`, because the `Structure` constraint requires that the corresponding type argument be a value type, while `Class` requires that it be a reference type.</span></span>
