# <a name="-expected-declaration"></a>' = ' bekleniyor (bildirim)
Bir `Mid` ifade`#Const` veya derleme yönergesi bir değer ataması içermez.  
  
 **Hata KIMLIĞI:** BC30249  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bir eşittir işareti (`=`) ve dizeye veya derleme sabitine atanacak bir değer ekleyin.
