# <a name="ltargumentnamegt-is-not-a-parameter-of-ltmethoddeclarationgt"></a>'&lt;argumentname&gt;'parametresi değil'&lt;methoddeclaration&gt;'
Yordam çağrısı yordamı tarafından tanımlanmayan bir ek bağımsız değişkeni sağlar.  
  
 **Hata Kimliği:** BC30272  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Gereksiz bağımsız değişken bağımsız değişken listesinden kaldırın.
