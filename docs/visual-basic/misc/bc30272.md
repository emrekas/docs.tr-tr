# <a name="argumentname-is-not-a-parameter-of-methoddeclaration"></a>'\<argumentname >', bir parametre değil '\<methoddeclaration >'
Yordam çağrısı yordamı tarafından tanımlanmayan bir ek bağımsız değişkeni sağlar.  
  
 **Hata Kimliği:** BC30272  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Gereksiz bağımsız değişken bağımsız değişken listesinden kaldırın.
