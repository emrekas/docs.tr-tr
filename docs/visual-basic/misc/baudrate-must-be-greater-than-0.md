# <a name="baudrate-must-be-greater-than-0"></a>Baud hızı 0'dan büyük olmalıdır
`BaudRate` Bağımsız değişkeni için belirtilen `My.Computer.Ports.OpenSerialPort` yöntemi sıfırdan büyük olmalıdır.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Değiştirin `BaudRate` bağımsız değişkeni için pozitif bir sayı.
