# <a name="value-of-type-type1-cannot-be-converted-to-type2-because-the-array-types-have-different-numbers-of-dimensions"></a>Türünün değeri '\<type1 >' olarak değiştirilemez '\<type2 >' Dizi türlerinin boyut sayıları farklı olduğundan
Bir dizi boyutları farklı sayıda diğerine dönüştürmek çalıştınız.  
  
 **Hata Kimliği:** BC30414  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Böylece boyutların sayısı ile eşleşen bir veya iki diziler değiştirin.
