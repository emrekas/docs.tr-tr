# <a name="value-of-type-lttype1gt-cannot-be-converted-to-lttype2gt-because-the-array-types-have-different-numbers-of-dimensions"></a>Türünün değeri '&lt;type1&gt;'olarak değiştirilemez'&lt;type2&gt;' Dizi türlerinin boyut sayıları farklı olduğundan
Bir dizi boyutları farklı sayıda diğerine dönüştürmek çalıştınız.  
  
 **Hata Kimliği:** BC30414  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Böylece boyutların sayısı ile eşleşen bir veya iki diziler değiştirin.
