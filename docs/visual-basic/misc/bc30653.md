# <a name="reference-required-to-module-ltmodulenamegt-containing-the-type-ltmembernamegt"></a>Modül için gereken başvuru '&lt;modulename&gt;'türünü içeren'&lt;membername&gt;'
Modül için gereken başvuru '\<modulename >' türünü içeren '\<membername >'. Projenize ekleyin.  
  
 Proje, Proje başvurularında yer almayan bir derlemeyi başka türden bağımlı bir tür içeriyor.  
  
 **Hata Kimliği:** BC30653  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Projede bağımlı türler için bir proje başvurusu ekleyin.
