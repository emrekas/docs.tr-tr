# <a name="too-many-dll-application-clients"></a>Çok fazla DLL uygulama istemcisi
Visual Basic için dinamik bağlantı kitaplığı (DLL), yalnızca sınırlı sayıda konak uygulamalar tarafından erişim sağlayabilir. Uygulamanız ve Visual Basic konakları (bazıları, uygulamanız tarafından erişilebilecek) olan diğer uygulamaların tümü aynı anda Visual Basic DLL erişmeye çalıştığınız.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Visual Basic erişme açık uygulamaları sayısını azaltın.
