# <a name="the-source-folder-does-not-exist"></a>Kaynak klasör yok
Mevcut bir kaynak klasörü başvuran bir işlem denendi. Bu bir hatalı dosya yolu sağlama sonucu olabilir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Özellikle göreli yollar kullanıyorsanız, doğru yola kullandığınızı kontrol edin.
