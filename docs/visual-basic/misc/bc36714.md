# <a name="expanded-properties-cannot-be-initialized"></a>Genişletilmiş özellikler başlatılamaz.
Otomatik uygulanan bir özellik bildiriminin bir parçası başlatılabilir, ancak genişletilmiş bir özellik olamaz.  
  
 **Hata Kimliği:** BC36714  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Otomatik uygulanan bir özellik kullanabilir veya başlatma alanındaki özellik bildirimini kaldırın.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnekler, hem otomatik olarak uygulanan hem de genişletilmiş özellikleri gösterir. Otomatik uygulanan özellikler atamaları kullanılarak başlatılabilir veya `New` yan tümcesi, ancak genişletilmiş özellikleri olamaz.  
  
```vb  
Class AutoImplementedExample  
    ' An auto-implemented property can be assigned an initial value.  
    Property IDNum As Integer = 33542  
    ' An auto-implemented property can be initialized with New.  
    Property Name As New String("Don Hall")  
End Class  
  
Class ExpandedExample  
    ' Attempting to expand an initialized auto-implemented property  
    ' causes this error.  
    'Property IDNum As Integer = 33542  
    '    Get  
    '    End Get  
    '    Set(ByVal value As Integer)  
    '    End Set  
    'End Property  
  
    ' Instead, you can assign the initial value to the backing field.  
    Private _IDNum As Integer = 33542  
    Property IDNum As Integer  
        Get  
        End Get  
        Set(ByVal value As Integer)  
        End Set  
    End Property  
End Class  
```
