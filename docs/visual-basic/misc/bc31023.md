# <a name="unable-to-parse-xml-lterrorgt"></a>XML ayrıştırılamadı: &lt;hata&gt;
Kullanılan XML dosyasında bir hata var.  
  
 **Hata Kimliği:** BC31023  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   XML düzgün biçimlendirilmiş, geçerli olduğundan ve dosyanın bozuk olmadığını doğrulayın.
