# <a name="missing-required-white-space"></a>Gerekli boşluk eksik
XML değişmez değer gerekli ayırıcı boşluk belirtildi. Bir XML öğesi değişmez değeri sonraki öznitelikleri arasındaki boşluk gerekli ayırıcı boşlukla örneğidir.  
  
 **Hata Kimliği:** BC31173  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Gerekli alan ayırıcı ekleyin.
