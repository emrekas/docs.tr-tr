## <a name="example"></a><span data-ttu-id="5847f-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="5847f-110">Example</span></span>  
 <span data-ttu-id="5847f-111">Aşağıdaki örnekte ilk parametrede belirten `Print` yöntemin genişlettiği `String` veri türü.</span><span class="sxs-lookup"><span data-stu-id="5847f-111">The first parameter in the following example indicates that the `Print` method extends the `String` data type.</span></span>

```  
Public Sub Print (ByVal str As String)  
    Console.WriteLine(str)  
End Sub  
```  
  
 <span data-ttu-id="5847f-112">Ne zaman genişletme yöntemi çağrıldığında gibi parametre `str` yöntemi bağlı `greeting`, örneğini `String` çağrılarının `Print`.</span><span class="sxs-lookup"><span data-stu-id="5847f-112">When the extension method is called as follows, parameter `str` in the method is bound to `greeting`, the instance of `String` that calls `Print`.</span></span> <span data-ttu-id="5847f-113">Derleyicinin kullanacağı `greeting` genişletme yöntemi için bağımsız değişken olarak `Print`.</span><span class="sxs-lookup"><span data-stu-id="5847f-113">The compiler will use `greeting` as the argument to extension method `Print`.</span></span>
