# <a name="named-arguments-are-not-valid-as-array-subscripts"></a>Adlandırılmış bağımsız değişkenler, dizi indisi olarak geçerli değildir.
Bir dizi dizini bir yordam bağımsız değişkeninin ada göre örneğin geçirme söz dizimi kullanılarak belirtilir `Array(Index := 10)`. Bu söz dizimi dizi indisleri için geçerli değil.  
  
 **Hata Kimliği:** BC30075  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Dizi dizini normal bir değişken veya sabit ifade kullanın, örneğin `Array(10)`.
