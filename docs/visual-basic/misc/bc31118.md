# <a name="end-addhandler-must-be-the-first-statement-on-a-line"></a>'Son AddHandler' satırdaki ilk deyim olmalıdır
Bir `End AddHandler` deyimi bir iki nokta üst üste (:) deyim ayırıcısı izler. `End AddHandler` Kaynak satırında tek deyim olması gerekir.  
  
 **Hata Kimliği:** BC31118  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Birden çok deyime farklı satırlarına ayırın.
