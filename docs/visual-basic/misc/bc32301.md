# <a name="project-has-been-closed"></a>Proje kapatıldı
Visual Basic Derleyicisi içinde kod model erişim hatası oluştu.  
  
 **Hata Kimliği:** BC32301  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Yinelenen hata durumunda yeniden görmek için programı derleyin.  
  
2. Hata almaya devam ederseniz, Visual Basic Derleyicisi yeniden yükleyin.  
  
3. Yeniden yüklenmesinden sonra hata oluşmaya devam ederse koşullar hakkında bilgi toplamak ve Microsoft Ürün Destek Hizmetleri bilgilendirir.
