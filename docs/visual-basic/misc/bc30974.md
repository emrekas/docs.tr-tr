# <a name="comment-statements-cannot-be-evaluated-not-in-toc"></a>Açıklama deyimleri (İçindekiler tablosunda değil) değerlendirilemiyor
Bir açıklama deyimi yürütme penceresinde bir hata ayıklama oturumu sırasında girildi. Açıklama deyimleri hemen penceresinde değerlendirilmez.  
  
 **Hata Kimliği:** BC30974
