# <a name="membername-exists-in-multiple-base-interfaces"></a>'\<membername >' birden fazla temel arabirimde bulunuyor
'\<membername >' birden fazla temel arabirimde bulunuyor. Bildiren arabirimin adını kullan '\<membername >' türetilen arabirimin adı yerine 'Implements' yan tümcesindeki.  
  
 Bu arabirim üyeleri aynı ada sahip belirsizlik oluşturma, birden çok arabirimlerinden devralır.  
  
 **Hata Kimliği:** BC31040  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Tanımlama arabirim adını kullanmak `Implements` türetilen arabirimin adı yerine sözleşme maddeleri.
