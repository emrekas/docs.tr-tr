# <a name="cannot-calculate-number-of-periods-using-the-arguments-provided"></a>Sağlanan bağımsız değişkenler kullanarak dönem sayısı hesaplayamıyor
Bir çağrı `NPer` işlevi tüm bağımsız değişkenlerini içermez.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Emin `Rate`, `Prnt` ve `PV` değer işlev çağrısında dahil edilir.
