# <a name="statement-cannot-appear-within-a-method-body"></a>Bir yöntem gövdesi içinde deyim bulunamaz
Bir yordam gibi başka bir yordam bildirimi geçersiz bir deyim içerir. Yordam bildirimleri iç içe olamaz.  
  
 **Hata Kimliği:** BC30289  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yordam bildirimi kaldırın veya kapsayan yordamı taşıyın.
