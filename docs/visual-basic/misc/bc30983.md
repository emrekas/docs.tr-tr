# <a name="type-of-variablename-is-ambiguous-because-the-loop-bounds-and-the-step-variable-do-not-widen-to-the-same-type"></a><span data-ttu-id="66388-102">Tür '\<variablename >' döngü sınırları ve step değişkeni aynı türe genişletmek değil çünkü belirsiz</span><span class="sxs-lookup"><span data-stu-id="66388-102">Type of '\<variablename>' is ambiguous because the loop bounds and the step variable do not widen to the same type</span></span>
<span data-ttu-id="66388-103">Kodunuzu içeren bir `For...Next` döngü içinde derleyici olamaz çıkarımı yapılan döngü denetim değişkeni için bir veri türü aşağıdaki koşulların doğru olduğundan:</span><span class="sxs-lookup"><span data-stu-id="66388-103">Your code contains a `For...Next` loop in which the compiler cannot infer a data type for the loop control variable because the following conditions is true:</span></span>  
  
-   <span data-ttu-id="66388-104">Döngü denetim değişkeni veri türü ile belirtilmemiş bir `As` yan tümcesi.</span><span class="sxs-lookup"><span data-stu-id="66388-104">The data type of the loop control variable is not specified with an `As` clause.</span></span>  
  
-   <span data-ttu-id="66388-105">Döngü sınırları ve step değişkeni en az iki veri türü içeriyor.</span><span class="sxs-lookup"><span data-stu-id="66388-105">The loop bounds and step variable contain at least two data types.</span></span>  
  
-   <span data-ttu-id="66388-106">Birden fazla olası dönüşümü, veri türleri arasında yok.</span><span class="sxs-lookup"><span data-stu-id="66388-106">More than one possible conversion exists between the data types.</span></span>  
  
-   <span data-ttu-id="66388-107">Böylece seçim için döngü denetim değişkeni türü belirsiz adayları arasından en iyi türü yoktur.</span><span class="sxs-lookup"><span data-stu-id="66388-107">There is no best type among the candidates, so that the choice of type for the loop control variable is ambiguous.</span></span>  
  
 <span data-ttu-id="66388-108">Örneğin, aşağıdaki döngü türünde bir döngü bağlı olan `Integer` ve bir döngünün türüne bağlı `UInteger`:</span><span class="sxs-lookup"><span data-stu-id="66388-108">For example, the following loop has one loop bound of type `Integer` and one loop bound of type `UInteger`:</span></span>  
  
```vb  
Dim m As Integer = 1  
Dim n As UInteger = 10  
' Not valid.  
' For i = m To n  
    ' Loop processing.  
' Next  
```  
  
 <span data-ttu-id="66388-109">Olası bir dönüştürme arasında `Integer` ve `UInteger`ve arasındaki olası bir dönüştürmenin `UInteger` ve `Integer`, ancak her ikisini de dönüştürmeleri daraltma ne kadar olan en iyi seçenek.</span><span class="sxs-lookup"><span data-stu-id="66388-109">There is a possible conversion between `Integer` and `UInteger`, and a possible conversion between `UInteger` and `Integer`, but both are narrowing conversions so neither is the best choice.</span></span>  
  
 <span data-ttu-id="66388-110">Sonraki örnekte, üçüncü bir değişken türü `Double` eklenir.</span><span class="sxs-lookup"><span data-stu-id="66388-110">In the next example, a third variable of type `Double` is added.</span></span> <span data-ttu-id="66388-111">Her ikisi de `Integer` ve `UInteger` standart dönüştürmelerine sahip `Double`, dönüştürme yapar `Double` en iyi seçenek.</span><span class="sxs-lookup"><span data-stu-id="66388-111">Both `Integer` and `UInteger` have standard widening conversions to `Double`, which makes conversion to `Double` the best choice.</span></span> <span data-ttu-id="66388-112">Tür çıkarımı yapılan döngü denetim değişkeni için atar `i` veri türü `Double`.</span><span class="sxs-lookup"><span data-stu-id="66388-112">Type inference assigns to loop control variable `i` the data type `Double`.</span></span>  
  
```vb  
Dim stepVar As Double = 1  
' Valid.  
For i = m To n Step stepVar  
    ' Loop processing.  
Next  
```  
  
 <span data-ttu-id="66388-113">**Hata Kimliği:** BC30983</span><span class="sxs-lookup"><span data-stu-id="66388-113">**Error ID:** BC30983</span></span>
