# <a name="nullable-parameters-must-specify-a-type"></a>Parametreleri NULL yapılabilir bir tür belirtmeniz gerekir
Bir parametre (?) boş değer atanabilir tür değiştiricisi içerir, ancak bir tür belirtmiyor.  
  
 **Hata Kimliği:** BC36632  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Boş değer atanabilen parametresi bildirirken bir türü belirtin.
