# <a name="ltqualifiedmethodgt-is-not-accessible-in-this-context-because-it-is-ltmodifiergt"></a>'&lt;qualifiedmethod&gt;'olduğundan bu bağlamda erişilebilir değil'&lt;değiştiricisi&gt;'
Kendisine uygulanan bir değiştirici nedeniyle bu bağlamda erişilebilir değil bir yönteme erişmeye çalıştınız.  
  
 **Hata Kimliği:** BC30390  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Değiştiricisi kaldırın.
