# <a name="qualifiedmethod-is-not-accessible-in-this-context-because-it-is-modifier"></a>'\<qualifiedmethod >' Bu bağlamda erişilebilir değil, çünkü '\<değiştiricisi >'
Kendisine uygulanan bir değiştirici nedeniyle bu bağlamda erişilebilir değil bir yönteme erişmeye çalıştınız.  
  
 **Hata Kimliği:** BC30390  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Değiştiricisi kaldırın.
