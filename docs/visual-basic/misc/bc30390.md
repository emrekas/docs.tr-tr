# <a name="qualifiedmethod-is-not-accessible-in-this-context-because-it-is-modifier"></a>'\<\<Modifier > ' olduğu için ' qualifiedmethod > ' Bu bağlamda erişilebilir değil
Uygulanan bir değiştirici nedeniyle bu bağlamda erişilemeyen bir yönteme erişmeye çalıştınız.  
  
 **Hata KIMLIĞI:** BC30390  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Değiştiricisini kaldırın.
