# <a name="unable-to-open-response-file-ltfilenamegt"></a>Yanıt dosyası açılamıyor. '&lt;filename&gt;'
Erişmeye çalıştığınız dosya kullanılamıyor.  
  
 **Hata Kimliği:** BC2011  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Ağ bağlantınızı hala etkin olduğundan emin olun.  
  
-   Dosya üzerindeki izinlerinizi denetleyin.
