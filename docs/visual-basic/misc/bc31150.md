# <a name="end-tag-tagname-expected"></a>Bitiş etiketi \</tagName > bekleniyor
Bir XML öğesi değişmez değeri bitiş etiketinde ilgili başlangıç etiketiyle eşleşmiyor.  
  
 **Hata Kimliği:** BC31150  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Son XML etiketi XML öğesi değişmez değeri XML başlangıç etiketi eşleşmesi için değiştirin.
