# <a name="using-statements-are-not-valid-in-the-immediate-window"></a>'Using' deyimleri yürütme penceresinde geçerli değildir
`Using` deyimleri yalnızca kaynak kodunda kullanılabilir.  
  
 **Hata Kimliği:** BC30186  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Kaldırma `Using` deyimden **hemen** penceresi.
