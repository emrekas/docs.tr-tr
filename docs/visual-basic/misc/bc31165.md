# <a name="expected-beginning--for-an-xml-tag"></a>Başlangıç beklenen '\<' için bir XML etiketi
XML etiket için bir XML değişmez değeri gerekli başına eksik ' <' karakteri.  
  
 **Hata Kimliği:** BC31165  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Ekle ' <' karakteri XML değişmez değer için XML etiketi başlatma.
