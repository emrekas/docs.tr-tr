# <a name="parameter-ltparameternamegt-already-has-a-matching-argument"></a>Parametre '&lt;parametername&gt;' eşleşen bir bağımsız değişken zaten var
Bir bağımsız değişken adıyla aynı yordam çağrısında birden fazla kez geçirilir.  
  
 **Hata Kimliği:** BC36567  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bir bağımsız değişken adları yanlış olup olmadığını denetleyin.  
  
-   Yinelenen bağımsız değişken bağımsız değişken listesinden kaldırın.
