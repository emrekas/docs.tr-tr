# <a name="internal-compiler-error-code-generator-received-malformed-input"></a>Derleyici iç hatası: Kod Oluşturucu alınan hatalı giriş
Visual Basic derleyici dahili olarak yazılmış dosyalarından birini başarıyla yorumlayamaz.  
  
 **Hata Kimliği:** BC30011  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Yinelenen hata durumunda yeniden görmek için programı derleyin.  
  
2.  Hata almaya devam ederseniz, Visual Basic Derleyicisi yeniden yükleyin.  
  
3.  Yeniden yüklenmesinden sonra hata oluşmaya devam ederse koşullar hakkında bilgi toplamak ve Microsoft Ürün Destek Hizmetleri bilgilendirir.
