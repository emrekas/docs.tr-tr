# <a name="get-not-supported-write-only-property"></a>Desteklenmiyor (salt yazılır özellik) alma
Salt yazılır olduğundan, bir özelliği okunamıyor.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Özelliğini denetleyin ve hangi koşullar altında ayarlanabilir belirleyin.  
  
2. Özellik referansı kaldırın.
