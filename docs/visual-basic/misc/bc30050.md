# <a name="paramarray-parameter-must-be-an-array"></a>ParamArray parametresiyle bir dizi olmalıdır
A `ParamArray` bağımsız değişkeni bir tek boyutlu dizi türü olarak bildirilmiş. Bildirimi sonra parantez içermez `ParamArray` bağımsız değişken adı.  
  
 **Hata Kimliği:** BC30050  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Sonra parantez çifti Ekle `ParamArray` bağımsız değişken adı.
