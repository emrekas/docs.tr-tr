# <a name="array-exceeds-the-limit-of-32-dimensions"></a>Dizi 32 boyut sınırını aşıyor
Dizi boyut 32'den fazla olamaz.  
  
 **Hata Kimliği:** BC30052  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bildirimde dizinin boyut sayısını azaltın.
