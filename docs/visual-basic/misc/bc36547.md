# <a name="anonymous-type-member-or-property-ltpropertynamegt-is-already-declared"></a><span data-ttu-id="6bc56-102">Anonim tür üyesi veya özelliği '&lt;propertyname&gt;' zaten bildirildi</span><span class="sxs-lookup"><span data-stu-id="6bc56-102">Anonymous type member or property '&lt;propertyname&gt;' is already declared</span></span>
<span data-ttu-id="6bc56-103">Bir özellik adı, anonim bir tür bildiriminde yalnızca bir kez oluşturulabilir.</span><span class="sxs-lookup"><span data-stu-id="6bc56-103">A property name can be established only once in the declaration of an anonymous type.</span></span> <span data-ttu-id="6bc56-104">Örneğin, aşağıdaki bildirimi geçerli değil:</span><span class="sxs-lookup"><span data-stu-id="6bc56-104">For example, the following declarations are not valid:</span></span>  
  
```vb  
'' Not valid, because the Label property is assigned to twice.  
' Dim anonType1 = New With {.Label = "Debra Garcia", .Label = .Label & ", President"}  
'' Not valid, because the property name inferred for both properties is  
'' Name.  
' Dim anonType2 = New With {Key product.Name, Key car1.Name}  
```  
  
 <span data-ttu-id="6bc56-105">**Hata Kimliği:** BC36547</span><span class="sxs-lookup"><span data-stu-id="6bc56-105">**Error ID:** BC36547</span></span>
