# <a name="xml-namespace-prefix-prefix-is-reserved-for-use-by-xml-and-the-namespace-uri-cannot-be-changed"></a>XML ad alanı öneki 'öneki' XML ad alanı URI'si değiştirilemez ile kullanım için ayrılmıştır
Bir XML ad alanı öneki gibi ayrılmış `xmlns`, bir XML değişmez değeri kullanılır.  
  
 **Hata Kimliği:** BC31152  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- XML ad alanı öneki ayrılmamış bir değerle değiştirin.
