# <a name="no-accessible-overloaded-methodname-can-be-called-with-these-arguments-without-a-narrowing-conversion"></a>Hiçbir erişilebilir aşırı yüklenmiş '\<yöntemAdı >' bir daraltma dönüşümü olmadan bu bağımsız değişkenlerle çağrılır
Aşırı yüklenmiş bir yöntemi çağrıldı, ancak yöntem bir daraltma dönüşümü olmadan sağlanan bağımsız değişkenler listesi ile eşleştirildi.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Belirtin `Option Strict Off`.  
  
2.  Bağımsız değişkenlerini bir aşırı yüklenmiş yöntemin imzası eşleşecek şekilde değiştirin.
