# <a name="variable-variablename-is-used-before-it-has-been-assigned-a-value-visual-basic-error"></a>Değişken '\<variablename >' (Visual Basic hatası) bir değer atanmadan önce kullanılır.
Bir değer atanmadan önce değişken kullanılıyor, sonuçta elde edilen bir <xref:System.NullReferenceException>.  
  
 **Hata Kimliği:** BC42109  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Değişken değeri atandığından emin olun.
