# <a name="let-and-set-assignment-statements-are-no-longer-supported"></a>'Let' ve 'deyimleri artık desteklenmiyor atama Set'
`Let` ve `Set` atamalarını kullanılamaz.  
  
 **Hata Kimliği:** BC30807  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Atlamak `Let` ve `Set` gelen atama deyimleri; örneğin:  
  
     `X = 2`
