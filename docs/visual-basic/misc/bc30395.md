# <a name="specifier-is-not-valid-on-a-structure-declaration"></a>'\<belirticisi >' Structure bildiriminde geçerli değil
Gibi bir belirleyici kullanmış `Dim` veya `Override`, yani üzerinde geçerli bir `Structure` bildirimi.  
  
 **Hata Kimliği:** BC30395  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Tanımlayıcısını kaldırın.
