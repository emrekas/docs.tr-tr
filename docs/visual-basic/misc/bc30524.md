# <a name="property-propertyname-is-writeonly"></a>Özellik '\<propertyname >' ' WriteOnly '

 **Hata Kimliği:** BC30524  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `WriteOnly` özellik bildirimini gelen tanımlayıcısı.
