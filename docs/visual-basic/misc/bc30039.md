# <a name="loop-control-variable-cannot-be-a-property-or-a-late-bound-indexed-array"></a>Bir özellik ya da geç bağlanan sıralı dizinin döngü denetim değişkeni olamaz
Yinelemek için kullanılan değişkeni bir `For` döngü sayısal veri türünde olmalıdır.  
  
 **Hata Kimliği:** BC30039  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Belirtmek için döngü denetim değişkeni bildirimi değiştirmek `Integer`, `Short`, `Long`, `Byte`, `Single`, `Double`, veya `Decimal`.
