# <a name="constants-must-have-a-value"></a>Sabit bir değere sahip olmalıdır
Bir sabit bir değer atamadan bildirdiğiniz.  
  
 **Hata Kimliği:** BC30438  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- İçin sabit bir değer atayın.
