# <a name="vbstrconvwide-and-vbstrconvnarrow-cannot-be-combined"></a>Argumenty VbStrConv.Wide a VbStrConv.Narrow nelze kombinovat
Birleştirmek, uygulamanız çalışırken `VbStrConv` numaralandırma üyelerini `Wide` ve `Narrow`, hangi karşılıklı olarak birbirini dışlar.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Ya da kaldırma `VbStrConv.Wide` veya `VbStrConv.Narrow`.
