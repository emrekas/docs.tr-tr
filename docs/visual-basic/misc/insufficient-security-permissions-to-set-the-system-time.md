# <a name="insufficient-security-permissions-to-set-the-system-time"></a>Sistem saatini ayarlamak için güvenlik izinleri yetersiz
Güvenlik izinlerini, sistem saati ayarlamak yeterli değil.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Güvenlik izinlerini değiştirin.
