# <a name="end-interface-must-be-preceded-by-a-matching-interface"></a>'Arabirimi end' bir eşleşen 'arabirim tarafından' gelmelidir
Bir `End Interface` deyimi gerçekleşir karşılık gelen olmadan `Interface` deyimi. `End Interface` karşılık gelen tarafından gelmelidir `Interface` deyimi.  
  
 **Hata Kimliği:** BC30252  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Bir önceki olmadığını görmek için onay `Interface` deyimi, yanlış yazılmış veya geçersiz.  
  
2. Emin `Interface` üyeleri hatalı biçimlendirilmiş.
