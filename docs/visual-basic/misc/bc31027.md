# <a name="unable-to-open-file-ltfilenamegt-lterrorgt"></a>Dosya açılamıyor. '&lt;filename&gt;': &lt;hata&gt;

 **Hata Kimliği:** BC31027  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Dosya var ve okunur zarar görmesi veya başka bir uygulama tarafından kilitlenmiş doğrulayın.
