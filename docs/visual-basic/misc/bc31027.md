# <a name="unable-to-open-file-filename-error"></a>Dosya açılamıyor. '\<dosya adı >': \<hata >
Bu projede bir dosya açılırken bir hata oluştu.  
  
 **Hata Kimliği:** BC31027  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Dosya var ve okunur zarar görmesi veya başka bir uygulama tarafından kilitlenmiş doğrulayın.
