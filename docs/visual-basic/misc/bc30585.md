# <a name="event-eventname-cannot-be-handled-because-it-is-not-accessible-from-name"></a>\<'\<Name > ' öğesinden erişilemediğinden ' EventName > ' olayı işlenemiyor
Erişilemeyen bir olayı işlemeye çalıştınız. Örneğin, bir `Handles` değişken paylaşılıyorsa olayları işleyen yöntemin de paylaşılması gerekir.  
  
 **Hata KIMLIĞI:** BC30585  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Olayın erişilebilir olduğundan emin olun.
