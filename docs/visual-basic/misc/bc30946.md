# <a name="stop-request-is-pending"></a>Durdurma isteği bekliyor
Visual Studio hata ayıklayıcısı, yordam çağrısı bir ifade belirtir, ancak iş parçacığı durdurma isteği yoktur.  
  
 Hata ayıklayıcı, etkin olmayan bir iş parçacığında bir yordam çağırma denemez.  
  
 **Hata Kimliği:** BC30946  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Mümkünse, böylece onu yeniden gerçekleşmesini önlemeye iş parçacığı durdurma isteği kaynağını belirler.  
  
-   Sonlandırır ve hata ayıklamayı yeniden başlatın.
