# <a name="too-many-arguments-to-procedurename"></a>Çok fazla bağımsız değişken '\<procedurename >'
Bir yordam çağrısında sağlanan bağımsız değişken yordamı için tanımlanan bağımsız değişken sayısını aşıyor.  
  
 **Hata Kimliği:** BC30057  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yordamı tanımı bakın ve ek bağımsız değişkenler'dan çağrısını kaldırın.
