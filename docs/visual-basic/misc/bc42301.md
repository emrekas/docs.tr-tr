# <a name="only-one-xml-comment-block-is-allowed-per-language-element"></a>Dil öğesi başına yalnızca bir XML açıklama bloğu izin verilir
Birden fazla XML açıklama bloğu bir dil öğesine uygulandı.  
  
 **Hata Kimliği:** BC42301  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Gereksiz XML açıklama bloğu kaldırın.
