# <a name="cannot-save-file-to-temp"></a>TEMP dosyası kaydedilemiyor
Bir bileşen TEMP adlı dizin bulunamıyor veya sürücü veya TEMP dizinini içeren bölüm bilgileri kaydetmek için yeterli alana sahip değil.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. "TEMP" adlı bir dizin oluşturun ve TEMP ortam değişkeni eşit yolunu ayarlayın.  
  
2. Gereksiz dosyaları silerek sürücüsünde olun veya başka bir bölüme TEMP dizinini oluşturmak ve TEMP ortam değişkeni kendi yoluna eşit olarak ayarlayın.
