# <a name="end-raiseevent-must-be-preceded-by-a-matching-raiseevent-declaration"></a>Eşleşen bir 'RaiseEvent' bildirimi 'RaiseEvent End' gelmelidir
Bir `End RaiseEvent` deyimi oluştu karşılık gelen olmadan `RaiseEvent` deyimi. `End RaiseEvent` karşılık gelen tarafından gelmelidir `RaiseEvent` deyimi.  
  
 **Hata Kimliği:** BC31126  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bu maddeyi önceki `RaiseEvent` deyimi geçerli olduğundan ve doğru bir şekilde yazılmış.
