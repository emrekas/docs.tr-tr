# <a name="withevents-variable-name-length-cannot-exceed-1019-characters"></a>WithEvents değişkeni adı uzunluğu 1019 karakterden uzun olamaz
Adı veya her değişken tanımlayıcısı kullanılarak bildirilen `WithEvents` 1019 karakterle sınırlıdır.  
  
 **Hata Kimliği:** BC32205  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Tanımlayıcının uzunluğunu azaltın.
