# <a name="attributes-cannot-be-applied-to-local-variables"></a>Yerel değişkenlere öznitelikler uygulanamaz
Yerel değişkenlere öznitelikler uygulanamaz.  
  
 **Hata Kimliği:** BC30660  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Bu değişken sonraki daha yüksek düzeyde sınıfı gibi bir kapsam veya modül düzeyinde bildirin.
