# <a name="attributes-cannot-be-applied-to-local-variables"></a>Öznitelikler yerel değişkenlere uygulanamıyor
Öznitelikler yerel değişkenlere uygulanamaz.  
  
 **Hata KIMLIĞI:** BC30660  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Bu değişkeni, sınıf veya modül düzeyi gibi bir sonraki daha yüksek kapsam düzeyinde bildirin.
