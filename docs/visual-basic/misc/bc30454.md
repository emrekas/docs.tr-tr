# <a name="expression-is-not-a-method"></a>İfade bir yöntem değildir.
Bir yöntem, yöntem çağrısının hedefi değil.  
  
 **Hata Kimliği:** BC30454  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Yöntem adı yanlış değil emin olun.
