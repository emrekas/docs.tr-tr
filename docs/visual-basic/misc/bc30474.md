# <a name="method1-cannot-override-method2-because-it-is-a-declare-statement"></a>' Declare ' bir ifade olduğundan '\<Method1>','Method2>'öğesinigeçersizkılamaz\<
Bir `Declare` deyimle belirtilen temel sınıf adındaki bir temsilciyi geçersiz kılmaya çalıştınız.  
  
 **Hata KIMLIĞI:** BC30474  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Geçersiz kılınan üyeyi bir `Declare` ifade olmaması için değiştirin.  
  
2. Bu yöntemi geçersiz kılmayı denemeyin.
