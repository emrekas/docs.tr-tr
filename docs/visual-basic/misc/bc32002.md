# <a name="keyword-is-valid-only-within-a-class"></a>'\<anahtar sözcüğü > ' yalnızca bir sınıf içinde geçerlidir
`Me` Veya`MyClass`gibi sınıflarla ilişkili bir anahtar sözcük, sınıf tanımının dışında kullanılır.  
  
 **Hata KIMLIĞI:** BC32002  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Anahtar sözcüğü kullanan kod sınıf örnekleri içeriyorsa, sınıf uygulamasına taşıyın.  
  
- Anahtar sözcüğü kullanan kod sınıflar için geçerli değilse, geçersiz anahtar sözcüğü kaldırın.
