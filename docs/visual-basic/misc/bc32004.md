# <a name="name-is-declared-in-project-projectname1-which-is-not-referenced-by-project-projectname2"></a>'\<adı >' proje içinde bildirilen '\<projectname1 > ', proje tarafından başvurulmadı'\<projectname2 >'
Bir programlama öğesi için geçerli projeye bir başvuru olmayan bir projede tanımlanır.  
  
 **Hata Kimliği:** BC32004  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Geçerli projede istediğiniz öğeyi içeren projeyi bir başvuru ekleyin.
