# <a name="you-must-specify-a-path-that-is-under-the-current-folder-or-one-of-its-sub-folders"></a>Geçerli klasörde veya alt klasörlerinin altında bir yol belirtmeniz gerekir
Belirtilen yol geçerli dizini altında değil. Bu, yanlış bir göreli yol tedarik edilirken dosya yollarını ayrıştırma bir sonucu olabilir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Sağladığınız yolun doğru olduğundan emin olun.
