# <a name="catch-block-never-reached-exception-handled-above-in-the-same-try-statement"></a>'Catch' bloğuna erişilemedi; \<özel durum > yukarıda aynı Try' deyiminde işlendi
A `Catch` kod bloğunda, bir önceki işlendiğinden ulaşılamıyor `Try` blok.  
  
Varsayılan olarak, bu iletiyi bir uyarıdır. Uyarıları gizleme veya uyarıları hata olarak değerlendirmesini hakkında daha fazla bilgi için bkz. [Visual Basic'teki uyarıları yapılandırma](/visualstudio/ide/configuring-warnings-in-visual-basic).
  
 **Hata Kimliği:** BC42031  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Gereksiz bildirimi Kaldır.
