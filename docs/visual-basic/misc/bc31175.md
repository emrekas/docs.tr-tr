# <a name="xml-dtds-are-not-supported"></a>XML DTD desteklenmez.
Bir belge türü tanımı (DTD'nin), bir XML değişmez değer belirtilmedi.  
  
 **Hata Kimliği:** BC31175  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   DTD'nin kaldırın. XML şema tanımı (XSD) şeması bir XML belgesi belirtmek için kullanın.
