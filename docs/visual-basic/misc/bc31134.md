# <a name="addhandler-and-removehandler-method-parameters-cannot-be-declared-byref"></a>'AddHandler' ve 'RemoveHandler' yöntem parametreleri 'ByRef' olarak bildirilemez
Parametreleri `AddHandler` ve `RemoveHandler` yöntemleri ile bildirilemez `ByRef` değiştiricisi.  
  
 **Hata Kimliği:** BC31134  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `ByRef` parametre anahtar sözcüğü.
