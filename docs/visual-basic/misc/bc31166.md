# <a name="expected-closing--for-xml-tag"></a>Kapanış beklenen ' >' için XML etiketi
XML etiket için bir XML değişmez değeri gerekli kapanış eksik ' >' karakteri.  
  
 **Hata Kimliği:** BC31166  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Kapanış Ekle ' >' XML değişmez değer için XML etiketi sonuna karakter.
