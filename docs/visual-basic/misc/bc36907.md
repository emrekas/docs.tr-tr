<span data-ttu-id="16143-103">Tür bağımsız değişkeni, genel hiç parametre yok ya da genel parametresi için bir tür zaten belirtilmedi yok bir genişletme yöntemi çağrısında belirtilmedi.</span><span class="sxs-lookup"><span data-stu-id="16143-103">A type argument has been specified in a call to an extension method that either has no generic parameters or has no generic parameters for which a type is not already specified.</span></span> <span data-ttu-id="16143-104">Örneğin, aşağıdaki kod bu hataya neden olur.</span><span class="sxs-lookup"><span data-stu-id="16143-104">For example, the following code causes this error.</span></span>  
  
```vb  
' The extension method is not generic.  
<Extension()> _  
Sub Example(ByVal str As String)  
    ' Body of the Sub.  
End Sub  
```  
  
```vb  
Dim str = "hi"  
'' The call to Example specifies a type argument.  
'' Not valid.  
'str.Example(Of String)()  
```
