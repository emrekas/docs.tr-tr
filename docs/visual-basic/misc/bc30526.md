# <a name="property-propertyname-is-readonly"></a>Özellik '\<propertyname >' ' ReadOnly '
Bildirilen bir özellik için bir değer atamayı denediniz `ReadOnly`.  
  
 **Hata Kimliği:** BC30526  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Kaldırma `ReadOnly` özellik bildirimini gelen tanımlayıcısı.
