# <a name="unable-to-read-delimited-fields-because-delimiters-is-nothing-or-empty"></a>Ayrılmış alanlar sınırlayıcılar olduğundan hiçbir şey okunamıyor veya boş
`TextFieldParser` Çünkü Dosyadan okuma yapamıyor `Delimiters` özelliği `Nothing` veya boş.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- İçin geçerli bir değer sağlamanız `Delimiters`.
