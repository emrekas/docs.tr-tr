# <a name="attribute-attributename-cannot-be-applied-to-a-method-with-optional-parameters"></a>'\<AttributeName > ' özniteliği isteğe bağlı parametrelere sahip bir yönteme uygulanamaz
Özniteliği yalnızca gerekli bağımsız değişkenleri kullanan veya bağımsız değişken olmayan yöntemlerle kullanılabilir.  
  
 **Hata KIMLIĞI:** BC30645  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. İsteğe bağlı parametreler olmadan yöntemi tanımlayın.  
  
2. İsteğe bağlı parametrelere sahip yöntemlerle kullanılabilecek bir öznitelik kullanın.  
  
3. Bu bağlamda kullanılabilecek özel bir öznitelik tanımlayın.
