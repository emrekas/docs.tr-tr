# <a name="attribute-attributename-cannot-be-applied-to-a-method-with-optional-parameters"></a>Öznitelik '\<attributename >' isteğe bağlı parametreler içeren bir yönteme uygulanamaz
Öznitelik, yalnızca gerekli bağımsız değişken veya bağımsız değişken olmadan kullanın yöntemleriyle kullanılabilir.  
  
 **Hata Kimliği:** BC30645  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  İsteğe bağlı parametreler olmadan yöntemi tanımlayın.  
  
2.  Bu isteğe bağlı parametreleri olan yöntemleri ile kullanılan bir özniteliği kullanın.  
  
3.  Bu bağlamda kullanılabilen özel bir öznitelik tanımlayın.
