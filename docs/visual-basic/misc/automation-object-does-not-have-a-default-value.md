# <a name="automation-object-does-not-have-a-default-value"></a>Otomasyon nesnesi bir varsayılan değeri yok
Visual Basic, belirtilen nesne için varsayılan üye belirlenemiyor.  
  
### <a name="to-correct-the-error"></a>Hatayı düzeltmek için  
  
- Nesnenin belgelerine bakın ve özellik veya yöntem için bir açık belirtimine verin.
