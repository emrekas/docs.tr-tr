# <a name="ltproperty1gt-and-ltproperty2gt-cannot-overload-each-other-because-only-one-is-declared-default"></a>'&lt;property1&gt;'ve'&lt;Özellik2&gt;' yalnızca bir tane 'Default' olarak bildirildiğinden birbirini aşırı yükleyemez
Bir özellik belirtiyorsa `Default`, o adı aşırı yüklenmiş tüm özellikler de belirtmeniz gerekir `Default`.  
  
 **Hata Kimliği:** BC30361  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Aşırı yüklenmiş tüm özellikler bildirildiğinden emin olun `Default`.
