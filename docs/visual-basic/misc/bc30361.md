# <a name="property1-and-property2-cannot-overload-each-other-because-only-one-is-declared-default"></a>'\<property1 >' ve '\<Özellik2 >' yalnızca bir tane 'Default' olarak bildirildiğinden birbirini aşırı yükleyemez
Bir özellik belirtiyorsa `Default`, o adı aşırı yüklenmiş tüm özellikler de belirtmeniz gerekir `Default`.  
  
 **Hata Kimliği:** BC30361  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Aşırı yüklenmiş tüm özellikler bildirildiğinden emin olun `Default`.
