# <a name="expected-closing--for-embedded-expression"></a>Kapanış beklenen ' % >' katıştırılmış ifadesi için
Gerekli kapanış gömülü bir XML Sabit ifadede eksik `%>` karakter.  
  
 **Hata Kimliği:** BC31159  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Gerekli kapatma ekleme `%>` katıştırılmış ifadenin sonuna karakter.
