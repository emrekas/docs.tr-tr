# <a name="expected-closing-gt-for-embedded-expression"></a>Kapanış beklenen ' %&gt;' gömülü ifade
Gerekli kapanış gömülü bir XML Sabit ifadede eksik `%>` karakter.  
  
 **Hata Kimliği:** BC31159  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Gerekli kapatma ekleme `%>` katıştırılmış ifadenin sonuna karakter.
