# <a name="filename-cannot-be-referenced-because-it-is-not-an-assembly"></a>'\<dosya adı >' bir derleme olduğundan başvurulamaz
Bir .NET derlemesi değil bir dosyaya bir başvuru yapıldı.  
  
 **Hata Kimliği:** BC31519  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Derleme konumunu ve adını doğrulayın.  
  
2. Derleme zarar görmüş veya bozuk olmadığını doğrulayın.
