# <a name="modulename-is-a-module-and-cannot-be-referenced-as-an-assembly"></a>'\<modulename >' bir modüldür, derleme olarak başvurulamaz
Bir modül bir derleme olarak kullanmak için girişimde bulunuldu.  
  
 **Hata Kimliği:** BC31076  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bir derlemeye başvuru değiştirin.
