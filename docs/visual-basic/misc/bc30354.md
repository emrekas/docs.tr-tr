# <a name="interface-can-inherit-only-from-another-interface"></a>Arabirimi yalnızca başka bir arabirimden devralabilir
Bir arabirim olmayan bir devralma denediniz.  
  
 **Hata Kimliği:** BC30354  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Temel arabirim veya arabirimleri devralan çalıştığınız gerçekten arabirimleri olduğundan emin olun.
