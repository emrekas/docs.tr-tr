# <a name="lttypenamegt-is-a-type-in-ltcontainernamegt-and-cannot-be-used-as-an-expression"></a>'&lt;typename&gt;'bir türü olan'&lt;containername&gt;' ve ifade olarak kullanılamaz
Kullanılan adlardan biri zaten bir tür adı olarak bildirildi ve bir ifadede kullanılamaz.  
  
 **Hata Kimliği:** BC30691  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Benzersiz üye adı.  
  
-   Üyenin adı köşeli ayraç kullanın.
