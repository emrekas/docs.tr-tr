# <a name="the-value-of-ltargumentnamegt-must-be-a-positive-number"></a>Değerini &lt;argumentname&gt; pozitif bir sayı olmalıdır
Değerini <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.ReserveDiskSpace%2A> özelliği, sıfırdan büyük olmalıdır.  
  
 <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.ReserveDiskSpace%2A> Özellik iletileri günlük dosyasına yazılabilir önce bu boş disk alanı miktarını bayt cinsinden gerekli belirtir  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Ayarlama <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.ReserveDiskSpace%2A> özelliğini pozitif bir sayı.
