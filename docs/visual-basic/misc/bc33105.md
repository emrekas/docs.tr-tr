# <a name="if-operands-cannot-be-named-arguments"></a><span data-ttu-id="ede42-102">'İf' işlenen adlandırılmış bağımsız değişkenler olamaz</span><span class="sxs-lookup"><span data-stu-id="ede42-102">'If' operands cannot be named arguments</span></span>
<span data-ttu-id="ede42-103">Adlı işleneni, bağımsız değişkenleri kullanarak `If` işleci geçerli değil.</span><span class="sxs-lookup"><span data-stu-id="ede42-103">Using named arguments in the operands of the `If` operator is not valid.</span></span> <span data-ttu-id="ede42-104">Aşağıdaki örnek bu hataya neden olur:</span><span class="sxs-lookup"><span data-stu-id="ede42-104">The following example causes this error:</span></span>  
  
```  
Dim i As Integer  
Dim result As String  
' Not valid.  
' result = (If(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="ede42-105">Bu farklıdır `IIf` izin vermez aşağıdaki kodda gösterildiği gibi adlandırılmış bağımsız değişkenler, işlev:</span><span class="sxs-lookup"><span data-stu-id="ede42-105">This differs from the `IIf` function, which does allow named arguments, as shown in the following code:</span></span>  
  
```

```  
  
 <span data-ttu-id="ede42-106">**Hata Kimliği:** BC33105</span><span class="sxs-lookup"><span data-stu-id="ede42-106">**Error ID:** BC33105</span></span>
