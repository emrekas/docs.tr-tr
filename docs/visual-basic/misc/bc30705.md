# <a name="referenced-object-objectname-has-a-value-of-nothing"></a>Başvurulan nesne '\<objectname >' değerini 'Nothing'
Başvurulan nesne ayarlanmamışsa veya değerine ayarlanırsa `Nothing`.  
  
 **Hata Kimliği:** BC30705  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Nesnenin adını doğru yazdığınızdan ve nesne ayarlandığını doğrulayın.
