<span data-ttu-id="4c533-103">İçindeki `Structure` bir LINQ sorgusu, yapının örnek üyesine `Me` veya başvurusunu içerir.</span><span class="sxs-lookup"><span data-stu-id="4c533-103">A LINQ query in a `Structure` includes a reference to `Me` or to an instance member of the structure.</span></span> <span data-ttu-id="4c533-104">İçindeki sorgu ifadelerinde `Me` veya örnek üyelerine başvurulara izin verilmez `Structure`.</span><span class="sxs-lookup"><span data-stu-id="4c533-104">References to `Me` or instance members are not allowed in query expressions within a `Structure`.</span></span>  
  
 <span data-ttu-id="4c533-105">**Hata KIMLIĞI:** BC36535</span><span class="sxs-lookup"><span data-stu-id="4c533-105">**Error ID:** BC36535</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="4c533-106">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="4c533-106">To correct this error</span></span>  
  
1. <span data-ttu-id="4c533-107">Örnek üyesinin veya başvuru `Me` tarafından döndürülen değerin bir kopyasını oluşturun ve aşağıdaki örnekte gösterildiği gibi sorgu ifadesindeki kopyayı kullanın.</span><span class="sxs-lookup"><span data-stu-id="4c533-107">Create a copy of the instance member or the value returned by the reference to `Me` and use the copy in the query expression, as shown in the following example.</span></span>  
  
```vb  
Structure SampleStructure  
    Public SearchValue As Integer  
  
    Public Sub SetSearchValue(ByVal number As Integer)  
      SearchValue = number  
    End Sub  
  
    Public Sub GetData()  
        Dim sv = SearchValue  
        Dim SampleData = New Integer() {1, 2, 3, 4}  
        Dim query = From number In SampleData _  
                    Where number < sv  
    End Sub  
End Structure  
```
