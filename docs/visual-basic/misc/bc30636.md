# <a name="-expected"></a>' >' bekleniyor
Belirtilen kod sözdizimi yanlış. Bu hata genellikle özniteliği ya da XML öğesi değişmez değeri için kapanış açılı ayracı eksik olduğunu gösterir.  
  
 **Hata Kimliği:** BC30636  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kullanılan kod için sözdizimini denetleyin.
