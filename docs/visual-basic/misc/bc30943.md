# <a name="generic-methods-cannot-be-exposed-to-com"></a>Genel yöntemler COM sunulamaz
Bir veya daha fazla genel yordamlar içeren bir .NET Framework bileşeni, bir COM bileşenine aktarılır.  
  
 Bileşen Nesne Modeli (COM) genel türleri desteklemez ve bunlarla işleyemez.  
  
 **Hata Kimliği:** BC30943  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- .NET Framework bileşeni genel yordam ya da yordamlar kaldırın veya COM birlikte çalışması için kullanmayın.
