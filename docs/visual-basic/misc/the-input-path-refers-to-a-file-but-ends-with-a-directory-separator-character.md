# <a name="the-input-path-refers-to-a-file-but-ends-with-a-directory-separator-character"></a>Giriş yolu bir dosya bir dizin ayırıcı karakteri ile biter ancak başvurur
Bir dosya adı verilmesini gerektiren bir işlem denendi, ancak sağlanan adı dizin ayırıcı karakteri ile sona erer (`\`). Bu bir hatalı ayrıştırılmış dosya yolu sonucu olabilir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Emin olmak için sağlanan yolun doğru olduğunu denetleyin.  
  
-   Fazlalık karakteri kaldırın.
