# <a name="evaluation-of-lambda-expressions-is-not-valid-in-the-debugger"></a>Lambda ifadeleri değerlendirme hata ayıklayıcıda geçerli değil
Bir lambda ifadesi girilen **Watch** veya **hemen** penceresi hata ayıklama oturumu sırasında. Windows hata ayıklamaya lambda ifadeleri kullanılamaz.  
  
 **Hata Kimliği:** BC36627  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Lambda ifadeleri, Kod Düzenleyicisi'nde test edin.
