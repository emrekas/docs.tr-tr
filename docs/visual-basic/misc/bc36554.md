<span data-ttu-id="19ae1-103">Genişletme yönteminin ilk parametresi 'ParamArray' uygulanamaz.</span><span class="sxs-lookup"><span data-stu-id="19ae1-103">'ParamArray' cannot be applied to the first parameter of an extension method.</span></span> <span data-ttu-id="19ae1-104">İlk parametre, genişletme türü belirtir.</span><span class="sxs-lookup"><span data-stu-id="19ae1-104">The first parameter specifies which type to extend.</span></span>  
  
 <span data-ttu-id="19ae1-105">Genişletme yönteminin ilk parametresi, yöntemin genişlettiği veri türünü belirtir.</span><span class="sxs-lookup"><span data-stu-id="19ae1-105">The first parameter of an extension method specifies the data type that the method extends.</span></span> <span data-ttu-id="19ae1-106">Bu nedenle, ilk parametresi gereklidir ve isteğe bağlı olamaz.</span><span class="sxs-lookup"><span data-stu-id="19ae1-106">Therefore, the first parameter is required and cannot be optional.</span></span> <span data-ttu-id="19ae1-107">Bir parametre dizisi otomatik olarak isteğe bağlı olduğundan, genişletme yönteminin ilk bağımsız değişken olarak geçerli değil.</span><span class="sxs-lookup"><span data-stu-id="19ae1-107">Because a parameter array is automatically optional, it is not valid as the first argument of an extension method.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="19ae1-111">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="19ae1-111">To correct this error</span></span>  
  
-   <span data-ttu-id="19ae1-112">Parametre dizisi, genişletmek istediğiniz veri türü belirtmezse, bu tür belirten yeni bir ilk parametresi ekleyin.</span><span class="sxs-lookup"><span data-stu-id="19ae1-112">If the parameter array does not specify the data type you want to extend, add a new first parameter that specifies this type.</span></span>  
  
    ```  
    <Extension()>  
    Public Sub AddTo(ByRef str As String, ByVal ParamArray addOns() As String)  
    ' Concatenate the strings in addOns to str.  
    End Sub  
    ```  
  
-   <span data-ttu-id="19ae1-113">Parametre dizisi, genişletmek istediğiniz veri türü belirtirseniz, bu parametre dizi yerine bir bağımsız değişken gerektiren, normal bir dizi azaltmayı düşünün.</span><span class="sxs-lookup"><span data-stu-id="19ae1-113">If the parameter array does specify the data type you want to extend, consider changing it to a regular array, requiring an argument, instead of a parameter array.</span></span> <span data-ttu-id="19ae1-114">Normal bir dizi genişletilebilir.</span><span class="sxs-lookup"><span data-stu-id="19ae1-114">Regular arrays can be extended.</span></span>  
  
    ```  
    <Extension()>  
    Public Function Sum(ByVal ints() As Integer) As Integer  
        Dim total As Integer = 0  
        For Each i As Integer In ints  
            total = total + i  
        Next i  
        Return total  
    End Function  
    ```
