# <a name="expected--for-xml-end-tag"></a>Beklenen ' /' için XML bitiş etiketi
XML bitiş etiketi için bir XML değişmez değeri, gerekli '/' karakteri eksik.  
  
 **Hata Kimliği:** BC31167  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   XML bitiş etiketi başlangıç XML değişmez değer için '/' karakteri ekleyin.
