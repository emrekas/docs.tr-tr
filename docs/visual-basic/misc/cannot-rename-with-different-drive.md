# <a name="cannot-rename-with-different-drive"></a>Farklı bir sürücü ile yeniden adlandırılamıyor
`Rename` Deyimi dosya yolunda bir sürücü harfi belirtemez.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Sürücü harfini dosyasının yolu kaldırın.
