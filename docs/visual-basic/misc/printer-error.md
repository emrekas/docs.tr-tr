# <a name="printer-error"></a>Yazıcı hatası
Yazıcı sırasında bir hata oluştu ancak dosya gönderilen bilgisayara başka hiçbir bilgi döndürülmedi.  
  
### <a name="to-correct-the-error"></a>Hatayı düzeltmek için  
  
- Fiziksel olarak yazıcı inceleyin. Yazıcı bilgisayar arasındaki tüm bağlantıları dolu olduğundan emin olun. Çoğu yazıcılar gibi "Çevrimdışı" veya "yetersiz kağıt." hata bilgileri için bir görüntü sağlayın.
