# <a name="class-in-an-interface-cannot-be-declared-ltspecifiergt"></a>Arabirim içindeki sınıf bildirilemez '&lt;belirticisi&gt;'
Geçersiz bir değiştiricisi, bir arabirimde bildirilen bir sınıf ile belirtildi.  
  
 **Hata Kimliği:** BC31070  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Değiştiriciler gibi kaldırmak `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend`, veya `Overridable`.
