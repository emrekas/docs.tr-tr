# <a name="file-ltfilenamegt-not-found"></a>Dosya '&lt;filename&gt;' bulunamadı
Bir dosya adı yanlış veya dosya yok.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Dosya adının doğru olduğundan ve var olduğundan emin olun.
