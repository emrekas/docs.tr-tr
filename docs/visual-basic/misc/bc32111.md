 <span data-ttu-id="12a01-105">Dolaylı kısıtlaması `Structure` doğrudan kısıtlama `Class` tür parametresi için bir çakışmaya neden `t1`, çünkü `Structure` kısıtlaması gerekli karşılık gelen tür bağımsız değişkeni bir değer türü olması gerektiğini sırada `Class` bir başvuru türü olmasını gerektirir.</span><span class="sxs-lookup"><span data-stu-id="12a01-105">The indirect constraint `Structure` and the direct constraint `Class` cause a conflict for type parameter `t1`, because the `Structure` constraint requires that the corresponding type argument be a value type, while `Class` requires that it be a reference type.</span></span>
