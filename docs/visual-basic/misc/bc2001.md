# <a name="file-filename-could-not-be-found"></a>Dosya \<filename > bulunamadı
Belirtilen dosya bulunamadı. Bu bir hatalı dosya yolu sağlayan bir sonucu olabilir.  
  
 **Hata Kimliği:** BC2001  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Dosya adı ve yolu doğru olup olmadığını denetleyin.
