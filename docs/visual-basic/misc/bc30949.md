# <a name="array-initializer-cannot-be-specified-for-a-non-constant-dimension-use-the-empty-initializer-"></a><span data-ttu-id="76d79-102">Dizi Başlatıcısı sabit olmayan bir boyut için belirtilemez; boş Başlatıcı kullanma '{}'</span><span class="sxs-lookup"><span data-stu-id="76d79-102">Array initializer cannot be specified for a non constant dimension; use the empty initializer '{}'</span></span>
<span data-ttu-id="76d79-103">Derleme zamanında bilinmeyen bir boyut dizisi başlatır.</span><span class="sxs-lookup"><span data-stu-id="76d79-103">An array initializes a dimension that is not known at compile time.</span></span>  
  
 <span data-ttu-id="76d79-104">Aşağıdaki kod bu hatayı üretir.</span><span class="sxs-lookup"><span data-stu-id="76d79-104">The following code generates this error.</span></span>  
  
```  
Dim j As Integer  
Dim intArray As Integer = New Integer(1, j) {{0, 100}, {1,101}}  
```  
  
 <span data-ttu-id="76d79-105">Aşağıdaki kod, hatayı önler.</span><span class="sxs-lookup"><span data-stu-id="76d79-105">The following code avoids the error.</span></span>  
  
```  
Dim intArray As Integer = New Integer(1, j) {}  
For i As Integer = 0 To j  
    intArray(0, i) = i  
    intArray(1, i) = 100 + i  
Next i  
```  
  
 <span data-ttu-id="76d79-106">**Hata Kimliği:** BC30949</span><span class="sxs-lookup"><span data-stu-id="76d79-106">**Error ID:** BC30949</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="76d79-107">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="76d79-107">To correct this error</span></span>  
  
- <span data-ttu-id="76d79-108">Mümkünse, dizi bildirimde sabit bir boyut belirtin.</span><span class="sxs-lookup"><span data-stu-id="76d79-108">If possible, specify a constant dimension in the array declaration.</span></span>  
  
- <span data-ttu-id="76d79-109">Sabit bir boyut belirtemezsiniz, nonconstant boyut bildiği bir döngü kullanarak dizi başlatmanız gerekir.</span><span class="sxs-lookup"><span data-stu-id="76d79-109">If you cannot specify a constant dimension, then you must initialize the array using a loop when the nonconstant dimension becomes known.</span></span>
