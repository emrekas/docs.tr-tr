# <a name="propertyname1-cannot-override-propertyname2-because-they-differ-by-readonly-or-writeonly"></a>' ReadOnly ' veya ' WriteOnly '\<farklı olduğundan 'propertyname1>','propertyName2>'öğesinigeçersizkılamaz\<
Yalnızca `ReadOnly` veya`WriteOnly`' den farklı olan ikinci bir özelliğe sahip bir özelliği geçersiz kılmayı denediniz.  
  
 **Hata KIMLIĞI:** BC30362  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Özelliklerin ve ' `ReadOnly` `WriteOnly`den daha fazlasına göre farklılaştırdığınızdan emin olun.
