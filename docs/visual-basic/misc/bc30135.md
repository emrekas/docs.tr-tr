# <a name="synclock-statements-are-not-valid-in-the-immediate-window"></a>'SyncLock' deyimleri yürütme penceresinde geçerli değildir
`SyncLock` Deyimi iş parçacıkları eşitler ve hata ayıklama bir bağlamda izin verilmiyor.  
  
 **Hata Kimliği:** BC30135  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Sorun değil bir `SyncLock` deyiminde **hemen** penceresi.
