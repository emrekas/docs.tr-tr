# <a name="event-eventname-cannot-be-found"></a>'\<EventName > ' olayı bulunamıyor
Derleyicinin konumlandıramıyor bir olayı işlemeye çalıştınız.  
  
 **Hata KIMLIĞI:** BC30590  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Olay adının doğru yazıldığından emin olun.
