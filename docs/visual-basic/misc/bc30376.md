# <a name="new-cannot-be-used-on-class-classname-because-it-contains-a-mustoverride-member-that-has-not-been-overridden"></a>'New' kullanılamaz sınıf üzerinde\<SınıfAdı >', geçersiz bir 'MustOverride' üyesine içerdiği için
Kullanma girişiminde bulundunuz `New` içeren bir sınıf üzerinde bir `MustOverride` değil kılındı üyesi.  
  
 **Hata Kimliği:** BC30376  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Kaldırma `New` deyimi.
