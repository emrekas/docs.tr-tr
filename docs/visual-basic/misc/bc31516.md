# <a name="type-typename-cannot-be-used-as-an-attribute-because-it-is-not-declared-public"></a>Türü '\<typename >', 'Public' olarak bildirilmediğinden, öznitelik olarak kullanılamaz
Öznitelikler, ortak olmalıdır.  
  
 **Hata Kimliği:** BC31516  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Ekleme `Public` öznitelik sınıfına değiştiricisi.
