# <a name="the-value-of-ltargumentnamegt-must-be-greater-than-or-equal-to-1000"></a>Değerini &lt;argumentname&gt; değerini 1000'e eşit veya daha büyük olmalıdır
Değerini <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.MaxFileSize%2A> özelliği değerini 1000'e eşit veya daha büyük olmalıdır.  
  
 <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.MaxFileSize%2A> Özelliği, yeni bir günlük dosyası oluşturmadan önce bir günlük dosyasına izin verilen bayt maksimum sayısını belirtir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Ayarlama <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.MaxFileSize%2A> özellik değerini 1000'e eşit veya daha büyük bir sayı.
