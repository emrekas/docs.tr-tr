<span data-ttu-id="80c7b-103">Bir lambda ifadesi parametre olmadan bildirilen bir `As` yan tümcesi ile `Option Strict` üzerinde.</span><span class="sxs-lookup"><span data-stu-id="80c7b-103">You have declared a parameter in a lambda expression without using an `As` clause, with `Option Strict` on.</span></span>

' Dim increment1 = Function (n) n + 1  
```  
  
 <span data-ttu-id="80c7b-104">Önceki bildirimi geçerlidir, türü `n` çıkarılan.</span><span class="sxs-lookup"><span data-stu-id="80c7b-104">The previous declaration is valid if the type of `n` can be inferred.</span></span> <span data-ttu-id="80c7b-105">Örneğin, önceki bir lambda ifadesi bir işlev temsilcisi atama `Del`:</span><span class="sxs-lookup"><span data-stu-id="80c7b-105">For example, if you are assigning the previous lambda expression to a function delegate, `Del`:</span></span>

 <span data-ttu-id="80c7b-106">Türü artık `n` parametresinden çıkarılan `p`:</span><span class="sxs-lookup"><span data-stu-id="80c7b-106">Now the type of `n` can be inferred from parameter `p`:</span></span>

```
