# <a name="property-value-is-not-valid"></a>Özellik değeri geçerli değil
Bir nesnenin ya da Denetim özelliklerinden biri, aralık dışında bir değere ayarlandı.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Özellik değeri geçerli bir ayar için değiştirin.
