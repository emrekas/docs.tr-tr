# <a name="type-character-character-cannot-be-used-in-a-declaration-with-an-explicit-type"></a>Türü karakteri '\<karakter >' açık bir türü olan bildirimde kullanılamaz
Bir tür karakteri gibi kullanmış `$`, açıkça belirtmek için bir tür.  
  
 **Hata Kimliği:** BC30302  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Tür karakteri kaldırın veya kaldırma `As <Type>` bildirimi from yan tümcesi.
