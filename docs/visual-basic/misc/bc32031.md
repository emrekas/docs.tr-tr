# <a name="end-sub-must-be-the-first-statement-on-a-line"></a>'End Sub' satırdaki ilk deyim olmalıdır
Bir `End Sub` deyimi bir iki nokta üst üste (:) deyim ayırıcısı izler. `End Sub` Kaynak satırında tek deyim olması gerekir.  
  
 **Hata Kimliği:** BC32031  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Birden çok deyime farklı satırlarına ayırın.
