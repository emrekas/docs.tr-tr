# <a name="notinheritable-classes-cannot-have-members-declared-specifiername"></a>'NotInheritable' sınıflarında bildirilen üyeler olamaz '\<specifiername >'
Geçersiz kılma değiştiricileri ile kullanılamaz `NotInheritable` üyeleri değiştirilemiyor çünkü sınıflar.  
  
 **Hata Kimliği:** BC30607  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Geçersiz kılma değiştiriciler gibi kaldırmak `Overridable`, `NotOverridable`, veya `MustOverride`, sınıf tanımından.
