# <a name="notinheritable-classes-cannot-have-members-declared-specifiername"></a>' NotInheritable ' sınıflarda '\<belirtilmi > ' olarak belirtilen Üyeler olamaz
Geçersiz kılma değiştiricileri, üyeleri geçersiz `NotInheritable` kılınamadığından sınıflarla kullanılamaz.  
  
 **Hata KIMLIĞI:** BC30607  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Sınıf tanımından `Overridable`, `NotOverridable`, veya `MustOverride`gibi geçersiz kılma değiştiricilerini kaldırın.
