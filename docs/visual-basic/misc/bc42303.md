# <a name="xml-comment-cannot-appear-within-a-method-or-a-property"></a>XML açıklaması bir metot veya özellik içinde yer alamaz
XML açıklaması bir metot veya özellik içinde yer alamaz. XML açıklaması yoksayılacak. XML açıklama blokları yöntemleri veya özellikleri içinde yer alamaz.  
  
 **Hata Kimliği:** BC42303  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Açıklama bloğu kaldırın veya daha uygun bir konuma taşıyın.
