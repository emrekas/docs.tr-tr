# <a name="me-cannot-be-the-target-of-an-assignment"></a>' Me ' bir atamanın hedefi olamaz
`Me` Anahtar sözcüğü her zaman kodun yürütülmekte olduğu bir sınıfın belirli örneğine başvurur. Değeri değiştirilemez.  
  
 **Hata KIMLIĞI:** BC30062  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bir değer `Me`atayan ifadeyi kaldırın.
