<span data-ttu-id="a50cf-103">' TypeName ' içinde tanımlanan '\<MethodName > ' genişletme yöntemindeki tür parametrelerinin veri türleri, aynı türe dönüştürülmediğinden, bu bağımsız değişkenlerden çıkarılamıyor.</span><span class="sxs-lookup"><span data-stu-id="a50cf-103">Data type(s) of the type parameter(s) in extension method '\<methodname>' defined in 'typename' cannot be inferred from these arguments because they do not convert to the same type.</span></span> <span data-ttu-id="a50cf-104">Veri türlerinin açıkça belirtilmesi bu hatayı düzeltebilir.</span><span class="sxs-lookup"><span data-stu-id="a50cf-104">Specifying the data type(s) explicitly might correct this error.</span></span>

<span data-ttu-id="a50cf-105">Genel genişletme yöntemine yapılan bir çağrı değerlendirilirken tür parametresinin (veya parametrelerinin) veri türünü (veya türlerini) belirlemekte kullanılacak bir girişimde bulunuldu.</span><span class="sxs-lookup"><span data-stu-id="a50cf-105">An attempt has been made to use type inference to determine the data type (or types) of the type parameter (or parameters) when evaluating a call to a generic extension method.</span></span> <span data-ttu-id="a50cf-106">Derleyici, tüm bağımsız değişkenlerin kısıtlamalarını karşılayan bir veri türü bulamadı.</span><span class="sxs-lookup"><span data-stu-id="a50cf-106">The compiler could not find a data type that meets the constraints of all the arguments.</span></span> <span data-ttu-id="a50cf-107">Bu nedenle, bu hatayı bildirdi.</span><span class="sxs-lookup"><span data-stu-id="a50cf-107">Therefore, it reported this error.</span></span>

> <span data-ttu-id="a50cf-108">Bağımsız değişkenlerin belirtilmesi bir seçenek değil (örneğin, sorgu ifadelerinde sorgu işleçleri için), ikinci tümce olmadan hata iletisi görüntülenir.</span><span class="sxs-lookup"><span data-stu-id="a50cf-108">When specifying arguments is not an option (for example, for query operators in query expressions), the error message appears without the second sentence.</span></span>

<span data-ttu-id="a50cf-109">Aşağıdaki kod hatayı gösterir.</span><span class="sxs-lookup"><span data-stu-id="a50cf-109">The following code demonstrates the error.</span></span>

        '' Not valid. Integer and Date do not convert to the same type.

    End Class
