# <a name="insufficient-security-permissions-to-set-the-system-date"></a>Sistem tarihi ayarlamak için güvenlik izinleri yetersiz
Güvenlik izinlerini sistem tarihi ayarlamak yeterli değil.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Sistem tarihi ayarına izin vermek için güvenlik izinleri değiştirin.
