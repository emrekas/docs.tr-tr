# <a name="byref-parameter-parametername-cannot-be-used-in-a-lambda-expression"></a><span data-ttu-id="478c2-102">'ByRef' parametresinin '\<parametername >' bir lambda ifadesinde kullanılamaz</span><span class="sxs-lookup"><span data-stu-id="478c2-102">'ByRef' parameter '\<parametername>' cannot be used in a lambda expression</span></span>
<span data-ttu-id="478c2-103">Bir lambda ifadesi içinde bildirilen bir `Sub` veya işlev herhangi kullanamaz `ByRef` parametreleri, söz konusu `Sub` veya işlev.</span><span class="sxs-lookup"><span data-stu-id="478c2-103">A lambda expression declared within a `Sub` or function cannot use any `ByRef` parameters of that `Sub` or function.</span></span> <span data-ttu-id="478c2-104">Örneğin, aşağıdaki kod bu hata nedeniyle neden olacak `ByRef` parametre `n` lambda ifadesinde kullanılır.</span><span class="sxs-lookup"><span data-stu-id="478c2-104">For example, the following code will cause this error because the `ByRef` parameter `n` is used in the lambda expression.</span></span>  
  
```  
'' Not valid.   
'Sub ExampleSub(ByRef n As Integer)  
  
'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```  
  
 <span data-ttu-id="478c2-105">**Hata Kimliği:** BC36639</span><span class="sxs-lookup"><span data-stu-id="478c2-105">**Error ID:** BC36639</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="478c2-106">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="478c2-106">To correct this error</span></span>  
  
-   <span data-ttu-id="478c2-107">Ata `ByRef` parametresi için bir yerel değişken ve lambda ifadesindeki, aşağıdaki kodda gösterildiği gibi yerel değişken kullanın:</span><span class="sxs-lookup"><span data-stu-id="478c2-107">Assign the `ByRef` parameter to a local variable, and use the local variable in the lambda expression, as shown in the following code:</span></span>  
  
    ```  
    Sub ExampleSub(ByRef n As Integer)  
  
        Dim temp = n  
        Dim lambda = Function(p As Integer) p + temp  
  
    End Sub  
    ```
