# <a name="exit-function-is-not-valid-in-a-sub-or-property"></a>'Exit işlevi' bir alt veya özelliği geçerli değil
`Exit Function` görünür bir `Sub` yordamı veya `Property` yordamı. Bir `Exit` deyim bloğu eşleşmelidir, hangi gerçekleşir.  
  
 **Hata Kimliği:** BC30067  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Değiştirin `Exit Function` ile `Exit Sub` veya `Exit Property` uygun şekilde deyimi.
