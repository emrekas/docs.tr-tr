# <a name="source-file-ltfilenamegt-specified-multiple-times"></a>Kaynak dosyası &lt;filename&gt; birden çok kez belirtildi
Bir dosyayı birden çok kez belirtildi.  
  
 **Hata Kimliği:** BC2002  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yedek dosya tanımlayıcıları kaldırın.
