# <a name="division-by-zero-occurred-while-evaluating-this-expression"></a>Sıfıra bölme bu ifadesi değerlendirilirken oluştu
Bu ifadenin değerlendirilmesi bir sayı sıfıra bölünme girişiminde bulunuldu.  
  
 **Hata Kimliği:** BC30542  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Sıfır, ifadenin paydası içinde nasıl oluştuğunu belirlemek için sabit ve değişken değerlerini denetleyin.
