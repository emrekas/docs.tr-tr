# <a name="argument-per-is-not-valid"></a>'Başına' bağımsız değişkeni geçerli değil
Bağımsız değişken için sağlanan `Per`, olması gereken bir `Double` ile 1 aralığında ödeme dönemi belirten `NPer`, geçerli değil.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bağımsız değişken geçerli aralığında olduğundan emin olun.
