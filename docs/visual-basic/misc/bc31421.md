# <a name="member-is-already-declared-in-this-structure"></a>'\<üye >' Bu yapıda zaten bildirildi
Bir numaralandırma üyesine, bu yapı içinde birden fazla kez bildirilmiş.  
  
 **Hata Kimliği:** BC31421  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Gereksiz bildirimi Kaldır.
