# <a name="property-ltpropertynamegt-cannot-be-set-to-nothing"></a>Özellik &lt;propertyname&gt; Nothing olarak ayarlanamaz
Özellik dışında bir değer gerektiriyor `Nothing`. Bu, hatalı değeri hesaplama bir sonucu olabilir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Geçerli bir değer sağlayın.
