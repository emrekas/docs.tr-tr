# <a name="overload-resolution-failed-because-no-accessible-method-can-be-called-with-these-argumentserror"></a>Aşırı yükleme çözümlemesi başarısız oldu, çünkü hiçbir erişilebilir '\<yöntemi >' Bu bağımsız değişken ile çağrılabilen:\<hata >
Aşırı yüklenmiş bir yönteme bir çağrı yapılır, ancak derleyicinin bir yöntem ile eşleşen bir bağımsız değişken listesi bulunamıyor.  
  
 **Hata Kimliği:** BC30518  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Sağladığınız bağımsız değişken listesini kontrol edin.
