# <a name="maximum-number-of-errors-has-been-exceeded"></a>En fazla hata sayısı aşıldı
Derleme hataları sayısı 100'dür. Derleyici Hatası raporları boyutunu azaltmak için bu sınır uygular.  
  
 **Hata Kimliği:** BC30041  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bu derleme tarafından oluşturulan diğer hataları gözden geçirin ve olabildiğince çok çözümleyin. Kaynak kodu 100'den fazla hata içeriyorsa, daha fazla hata sonraki derlemede görebilirsiniz.
