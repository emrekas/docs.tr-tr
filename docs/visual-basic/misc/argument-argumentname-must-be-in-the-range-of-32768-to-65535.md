# <a name="argument-argumentname-must-be-in-the-range-of--32768-to-65535"></a>Bağımsız değişken '\<argumentname >' -32768 ile 65535 aralığında olmalıdır
Bir bağımsız değişkenin değeri izin verilen -32.768 65. 535'için aralığının dışında.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Bağımsız değişkenin değeri geçerli aralıkta olduğundan emin olun.  
  
2.  Bağımsız değişken bir hesaplamanın sağlanırsa, hesaplama işlemleri değerleri kontrol edin.
