# <a name="security-attribute-attributename-cannot-be-applied-to-a-module"></a>Güvenlik özniteliğinin '\<attributename >' için bir modül uygulanamaz
Ortak dil çalışma zamanı güvenlik özniteliği bir modül kabul etmiyor.

**Hata Kimliği:** BC30131

## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için

- Bildirimden özniteliği kaldırın.
