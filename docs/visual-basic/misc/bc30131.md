# <a name="security-attribute-attributename-cannot-be-applied-to-a-module"></a>'\<AttributeName > ' güvenlik özniteliği bir modüle uygulanamaz
Ortak dil çalışma zamanı, modüldeki güvenlik özniteliğini kabul etmez.

**Hata KIMLIĞI:** BC30131

## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için

- Özniteliği bildirimden kaldırın.
