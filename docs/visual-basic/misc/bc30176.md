# <a name="only-one-of-public-private-protected-friend-or-protected-friend-can-be-specified"></a>'Public', 'Private', 'Protected', 'Friend' veya 'Protected Friend' yalnızca biri belirtilebilir.
Bir öğe bildiriminde çakışan erişilebilirlikleri oluşur. Yalnızca bir erişilebilirlik belirtilebilir.  
  
 **Hata Kimliği:** BC30176  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bir erişilebilirlik seçin ve diğer anahtar sözcükler bildirimden kaldırın.
