# <a name="exit-property-is-not-valid-in-a-function-or-sub"></a>'Özelliği çıkmak' bir işlev veya alt içinde geçerli değil
`Exit Property` görünür bir `Function` yordamı veya `Sub` yordamı. Bir `Exit` deyim bloğu eşleşmelidir, hangi gerçekleşir.  
  
 **Hata Kimliği:** BC30066  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Değiştirin `Exit Property` ile `Exit Function` veya `Exit Sub` uygun şekilde deyimi.
