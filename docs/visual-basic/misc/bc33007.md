# <a name="end-operator-must-be-preceded-by-a-matching-operator"></a>'Operator End', 'İşleci tarafından' gelmelidir
Bir `End Operator` deyimi görünür eşleşme ile `Operator` kendisinden bildirimi.  
  
 **Hata Kimliği:** BC33007  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Kaldırma `End Operator` yedekli ise deyimi.  
  
- Eksik olan kaynağı `Operator` biri eksikse yordamı.  
  
- Taşıma `End Operator` kod içinde uygun konuma deyimi.
