# <a name="withevents-variable-does-not-raise-any-events"></a>'WithEvents' değişkeni meydana gelen olayları oluşturmaz
Nesne değişkeni ile bildirilen `WithEvents` anahtar sözcüğü, olayları tetikleyebilen bir sınıf örneğini başvurmuyor.  
  
 **Hata Kimliği:** BC30591  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Nesne değişkeni, olayları tetikleyebilen bir sınıf örneğini başvurduğundan emin olun.
