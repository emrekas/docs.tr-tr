# <a name="xml-comment-tag-lttaggt-is-not-permitted-on-a-ltelementgt-language-element"></a>XML açıklama etiketi '&lt;etiketi&gt;' üzerinde izin verilmiyor bir '&lt;öğesi&gt;' dil öğesi
XML açıklama etiketine \<etiket > üzerinde izin verilmiyor bir '\<öğesi >' öğesi. XML açıklaması yoksayılacak.  
  
 Bir XML açıklama etiketi, uygulandığı dil öğesi tarafından desteklenmiyor.  
  
 **Hata Kimliği:** BC42306  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Desteklenmeyen XML açıklama etiketine kaldırın.
