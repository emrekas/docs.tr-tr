# <a name="unable-to-load-dll-filename-error"></a>DLL yüklenemiyor '\<dosya adı >': \<hata >
Bu proje tarafından başvurulan bir DLL yüklenemedi.  
  
 **Hata Kimliği:** BC31013  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Dosyanın belirtilen konumda olduğundan ve başka bir uygulama tarafından kilitlenmediğinden emin olun.
