# <a name="module-statement-must-end-with-a-matching-end-module"></a>'Module' deyimi eşleşen bir 'End Module' ile bitmelidir
Modül blokları sonlanmalı bir `End Module` deyimi.  
  
 **Hata Kimliği:** BC30625  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Ekleme bir `End Module` sonuna deyimi `Module` blok.
