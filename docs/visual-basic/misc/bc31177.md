# <a name="white-space-may-not-appear-here"></a>Boşluk burada yer alamaz
Bir XML öğesi adını değişmez değer boşluk veya alanları dahildir.  
  
 **Hata Kimliği:** BC31177  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Alan veya alanları XML öğesi değişmez değeri adından kaldırın.
