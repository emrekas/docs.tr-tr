# <a name="comma-expected"></a>Virgül bekleniyor
İki bağımsız değişkenleri arasında virgül (,), bir bağımsız değişken listesi eksik.  
  
 **Hata Kimliği:** BC30196  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Tüm bağımsız değişkenlerin virgülle ayrılır emin olun.
