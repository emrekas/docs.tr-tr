# <a name="argument-argumentname-must-be-greater-than-or-equal-to--1"></a>Bağımsız değişken '\<argumentname >' değerinden büyük veya -1 değerine eşit olmalıdır
-1'den daha az bir bağımsız değişkenin değeridir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Bağımsız değişkenin değeri geçerli aralıkta olduğundan emin olun.  
  
2. Bağımsız değişken bir hesaplamanın sağlanırsa, hesaplama işlemleri değerleri kontrol edin.
