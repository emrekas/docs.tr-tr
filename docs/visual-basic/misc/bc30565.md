# <a name="array-initializer-has-too-few-dimensions"></a>Dizi başlatıcıda çok az bir boyuta sahiptir
Diziniz için Başlatıcı yeterli boyutta sağlamadınız.  
  
 **Hata Kimliği:** BC30565  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Kaç tane boyutları gerekli olduğunu belirlemek için bir dizi Başlatıcısı denetleyin.  
  
- Eksik bağımsız değişken veya bağımsız değişkenleri sağlayın.
