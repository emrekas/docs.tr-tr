# <a name="unable-to-apply-security-attributes-to-name-error"></a>Güvenlik öznitelikleri uygulanamadı '\<adı >': \<hata >
Güvenlik öznitelikleri uygulanırken bir hata oluştu. Bu yanlış bir güvenlik öznitelikleri kullanarak bir sonucu olabilir veya derleyici ile ilgili bir sorun olduğunu gösteriyor olabilir.  
  
 **Hata Kimliği:** BC31520  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Doğrulayın ve söz dizimi özniteliği için bağımsız değişkenler.  
  
2.  Hata durumunda yeniden yeniden görmek için programı derleyin.  
  
3.  Hata devam ederse, çalışmanızı kaydedin ve Visual Studio'yu yeniden başlatın.  
  
4.  Hata yeniden oluşursa, Visual Basic yeniden yükleyin.  
  
5.  Yeniden yüklenmesinden sonra hata oluşmaya devam ederse Microsoft Ürün Destek Hizmetleri bildirin.
