<span data-ttu-id="075cd-103">Genel bir arabirim iki veya daha fazla genel arabirimlerinden devralır ve iki inheritances belirli tür bağımsız değişkeni değerlerini çakışabilir.</span><span class="sxs-lookup"><span data-stu-id="075cd-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="075cd-104">Aşağıdaki deyimleri, bu hata oluşturabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="075cd-104">The following statements can generate this error.</span></span>  
  
```  
Public Interface interfaceA(Of u)  
End Interface  
Public Interface interfaceX(Of v)  
    Inherits interfaceA(Of v)  
End Interface  
Public Interface derivedInterface(Of t1, t2)  
    Inherits interfaceA(Of t1), interfaceX(Of t2)  
End Interface  
```  
  
 <span data-ttu-id="075cd-105">Varsa `derivedInterface` oluşturulan ya da her ikisi de aynı türe sağlama uygulanan `t1` ve `t2`, iki sürümünü devralmalıdır `interfaceA` aynı tür bağımsız değişkenleri ile.</span><span class="sxs-lookup"><span data-stu-id="075cd-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceA` with identical type arguments.</span></span> <span data-ttu-id="075cd-106">Bunun yapılması erişmek için hangi sürümü hakkında bir belirsizlik oluşturur.</span><span class="sxs-lookup"><span data-stu-id="075cd-106">Doing so would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="075cd-107">**Hata Kimliği:** BC32123</span><span class="sxs-lookup"><span data-stu-id="075cd-107">**Error ID:** BC32123</span></span>
