# <a name="custom-event-must-end-with-a-matching-end-event"></a>'Özel olay', eşleşen bir 'End Event' ile bitmelidir
A `Custom Event` bildirimi sonlanmalı bir `End Event` deyimi.  
  
 **Hata Kimliği:** BC31114  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bu maddeyi `Custom Event` bildirimi ile sona erer bir `End Event` deyimi.
