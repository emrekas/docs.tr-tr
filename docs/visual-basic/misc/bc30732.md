# <a name="web-methods-cannot-be-evaluated-in-debug-windows"></a>Web yöntemleri hata ayıklama pencerelerinde değerlendirilemiyor
XML Web hizmeti yöntemlerini içinde hata ayıklama penceresine sınanamıyor.  
  
 **Hata Kimliği:** BC30732  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   XML Web hizmeti yöntemlerini Kod Düzenleyicisi'nden test edin.
