# <a name="query-expressions-are-not-valid-in-debug-windows"></a>Sorgu ifadeleri hata ayıklama pencerelerinde geçerli değil
LINQ sorgusu girilen **Watch** veya **hemen** penceresi hata ayıklama oturumu sırasında. Sorgular, hata ayıklama pencerelerinde kullanılamaz.  
  
 **Hata Kimliği:** BC36609
