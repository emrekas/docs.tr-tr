# <a name="namespace-statement-must-end-with-a-matching-end-namespace"></a>'Namespace' deyimi eşleşen bir 'End Namespace' ile bitmelidir
`Namespace` blokları sonlanmalı bir `End Namespace` deyimi.  
  
 **Hata Kimliği:** BC30626  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Ekleme bir `End Namespace` sonuna deyimi `Namespace` blok.
