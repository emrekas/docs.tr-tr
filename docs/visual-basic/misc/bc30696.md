# <a name="ltmembername1gt-and-ltmembername2gt-cannot-overload-each-other-because-they-differ-only-by-the-types-of-optional-parameters"></a>'&lt;membername1&gt;'ve'&lt;membername2&gt;', isteğe bağlı parametrelerinin yalnızca türleri farklı olduğundan birbirini aşırı yükleyemez
Aşırı yüklenmiş üyelerin bir veya daha fazla gerekli parametreler için farklı veri türleri olmalıdır.  
  
 **Hata Kimliği:** BC30696  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   En az bir gerekli parametresi farklı veri türü olarak tanımlayın.
