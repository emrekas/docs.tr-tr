# <a name="membername1-and-membername2-cannot-overload-each-other-because-they-differ-only-by-the-types-of-optional-parameters"></a>'\<membername1 >' ve '\<membername2 >', isteğe bağlı parametrelerinin yalnızca türleri farklı olduğundan birbirini aşırı yükleyemez
Aşırı yüklenmiş üyelerin bir veya daha fazla gerekli parametreler için farklı veri türleri olmalıdır.  
  
 **Hata Kimliği:** BC30696  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- En az bir gerekli parametresi farklı veri türü olarak tanımlayın.
