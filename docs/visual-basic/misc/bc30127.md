# <a name="attribute-attributename-is-not-valid-error"></a>Öznitelik '\<attributename >' geçerli değil: \<hata >
Ortak dil çalışma zamanı, geçerli bağlamda öznitelik kabul etmiyor.  
  
 **Hata Kimliği:** BC30127  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Bildirimden özniteliğini kaldırın veya bağlamı için geçerli bir öznitelik ile değiştirin.  
  
2. Hata iletisini inceleyin ve uygun eylemi gerçekleştirin.
