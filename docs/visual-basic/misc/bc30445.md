# <a name="const-declaration-cannot-have-an-array-initializer"></a>Const bildirimi bir dizi başlatıcısı olamaz
Bir dizi Başlatıcı kullanmayı denediğiniz bir `Const` bildirimi.  
  
 **Hata Kimliği:** BC30445  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Başlatıcı etrafındaki küme ayraçlarını kaldırarak için geçerli bir ifade değiştirin.
