# <a name="warning-number-ltnumbergt-for-the-option-ltoptionnamegt-is-either-not-configurable-or-not-valid"></a>Uyarı numarası '&lt;numarası&gt;'for 'seçeneği&lt;SeçenekAdı&gt;' yapılandırılamaz veya geçersiz
Uyarı Kimliği geçerli değil. Kimlikleri tamsayı değerleri 0'dan büyük olmalıdır.  
  
 **Hata Kimliği:** BC2026  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Geçerli olduğundan emin olmak için sağlanan uyarı kimliği kontrol edin.
