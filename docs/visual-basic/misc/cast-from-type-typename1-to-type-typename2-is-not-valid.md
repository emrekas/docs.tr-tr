# <a name="cast-from-type-typename1-to-type-typename2-is-not-valid"></a>Türden atama '\<typename1 >' türü için '\<typename2 >' geçerli değil
Bir değişken bir türden diğerine dönüştürme girişimi geçerli değil.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Türleri dönüştürme ilgili denetleyin ve anlamlı bir türe dönüştürmeye çalışıyorsunuz. emin olun.
