# <a name="division-by-zero-visual-basic-error"></a>Bölme sıfır (Visual Basic hatası)
Bölen olarak kullanılan bir ifade değeri sıfırdır.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. İfade değişkenleri yazımını denetleyin. Yanlış yazılmış bir değişken örtük olarak sıfır olarak başlatılır, sayısal bir değişken oluşturabilirsiniz.  
  
2. Önceki işlemleri ifade değişkeninde özellikle yordama bağımsız değişkenler olarak diğer yordamlardan geçirilen denetleyin.
