# <a name="argument-ltargumentnamegt-must-be-greater-than-or-equal-to-zero"></a>Bağımsız değişken '&lt;argumentname&gt;' değerinden büyük veya sıfıra eşit olmalıdır
Bu küçük olduğundan sıfır bağımsız değişken geçerli değil.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Bağımsız değişkenler yazımını denetleyin. Yanlış yazılmış bir değişken adı, sıfır olarak başlatılır, sayısal bir değişken örtük olarak oluşturabilirsiniz.  
  
2.  Önceki değişkenleri ifadede, özellikle yordama bağımsız değişkenler olarak diğer yordamlardan geçirilen işlemleri denetleyin.
