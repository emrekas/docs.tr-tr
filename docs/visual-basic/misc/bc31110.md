# <a name="arrays-used-as-attribute-arguments-are-required-to-explicitly-specify-values-for-all-elements"></a>Öznitelik bağımsız değişkenleri olarak kullanılan diziler tüm öğeler için değerleri açıkça belirtmeniz gerekir
Öznitelik bağımsız değişkeni kullanılan bir dizi belirtilmeyen bir değere sahip bir öğe var.  
  
 **Hata Kimliği:** BC31110  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Tüm öğe değerlerini belirtin.
