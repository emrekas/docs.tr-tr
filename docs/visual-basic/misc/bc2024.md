# <a name="the-default-assembly-reference-reference-has-already-been-automatically-added-ignored"></a>Varsayılan derleme başvurusu '\<başvuru >' zaten otomatik olarak eklenen yoksayıldı;
Varsayılan derleme başvurusu eklemek için bir girişimde bulunuldu ancak denemesi göz ardı edilir şekilde derleme zaten, eklendi.  
  
 **Hata Kimliği:** BC2024  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Gereksiz kodu Kaldır.
