# <a name="specified-dll-function-not-found"></a>Belirtilen DLL işlev bulunamadı
Dinamik bağlantı kitaplığı (DLL) kullanıcı Kitaplığı Başvurusu bulundu, ancak belirtilen DLL işlevini DLL bulunamadı.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Geçerli bir sıra işlevi bildiriminde belirtin.  
  
2.  DLL adını ve diğer doğru olduğundan emin olun.
