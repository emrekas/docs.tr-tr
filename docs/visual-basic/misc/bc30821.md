# <a name="rsetstmt-is-not-declared"></a>'\<rsetstmt >' bildirimi yapılmadı
'\<rsetstmt >' değil olarak bildirilir. RSet deyimleri artık desteklenmiyor; Bunun yerine Microsoft.VisualBasic.RSet kullanın.  
  
 Önceki sürümlerde Visual Basic iç çeşitli işlevler taşındı <xref:Microsoft.VisualBasic?displayProperty=nameWithType> ad alanı. Bu işlevleri tüm programlama dilleri için daha genel olarak kullanılabilir hale getirir.  
  
 **Hata Kimliği:** BC30821  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Kullanım `RSet` işlevi `Microsoft.VisualBasic` ad alanı yerine.
