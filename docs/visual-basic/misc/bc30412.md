# <a name="withevents-variables-must-have-an-as-clause"></a>' WithEvents ' değişkenlerinde ' As ' yan tümcesi olmalıdır
Anahtar sözcüğü `As` `WithEvents`ile bir yan tümce sağlamadınız. Değişkeni, olayları yükseltebilen belirli bir sınıf olarak bildirin.  
  
 **Hata KIMLIĞI:** BC30412  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Olayları yükseltecan sınıfı belirten yantümceyiekleyin.`As`
