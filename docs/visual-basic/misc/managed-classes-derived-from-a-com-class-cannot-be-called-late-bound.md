# <a name="managed-classes-derived-from-a-com-class-cannot-be-called-late-bound"></a>Bir COM sınıfından türetilmiş yönetilen sınıflar, geç bağlanan çağrılamaz.
Bir COM sınıfından türetilen bir yönetilen sınıf geç bağlanan çağrı yapmak çalışıldı; Böyle çağrılar desteklenmiyor.  
  
### <a name="to-correct-the-problem"></a>Sorunu düzeltmek için  
  
- Çağrı yapma erken bağlanmış.
