# <a name="array-initializer-is-missing-number-elements"></a>Dizi Başlatıcı eksik \<numarası > öğeleri
Diziniz için yeterli sayıda öğe başlatıcısında sağlamadınız.  
  
 **Hata Kimliği:** BC30567  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Eksik bağımsız değişken veya bağımsız değişkenleri sağlayın.  
  
-   Kullanım `ReDim` dizinin boyutunu değiştirmek için.
