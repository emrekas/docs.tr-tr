# <a name="array-initializer-is-missing-number-elements"></a>Dizi başlatıcıda eksik \<sayı > öğeleri
Diziniz için başlatıcıda yeterli sayıda öğe sağlamadınız.  
  
 **Hata KIMLIĞI:** BC30567  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Eksik bağımsız değişkeni veya bağımsız değişkenleri sağlayın.  
  
- Dizinin `ReDim` boyutunu değiştirmek için kullanın.
