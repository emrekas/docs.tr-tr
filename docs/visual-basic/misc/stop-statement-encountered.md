# <a name="stop-statement-encountered"></a>Stop deyimi karşılaşıldı
A `Stop` deyimi yürütmeyi askıya alır, karşılaşıldı.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Kaldırma `Stop` deyimi.
