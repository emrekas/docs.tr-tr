# <a name="number-of-indices-is-less-than-the-number-of-dimensions-of-the-indexed-array"></a>Dizin sayısı sıralı dizinin boyut sayısını küçüktür
Bir dizi öğesine erişmek için kullanılan dizinlerin sayısı tam olarak diğer bir deyişle, boyut için bildirilen sayısını dizinin boyut ile aynı olmalıdır.  
  
 **Hata Kimliği:** BC30105  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Dizi boyut sayısını toplam alt simgeler eşittir kadar alt simgeler dizi başvuru ekleyin.
