# <a name="end-operator-must-be-the-first-statement-on-a-line"></a>'Son işleci' satırdaki ilk deyim olmalıdır
Bir `End Operator` deyimi takip eden bir iki nokta üst üste (`:`) deyim ayırıcısı. `End Operator` Kaynak satırında tek deyim olması gerekir.  
  
 **Hata Kimliği:** BC33006  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Birden çok deyime farklı satırlarına ayırın.
