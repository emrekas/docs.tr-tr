# <a name="project-level-conditional-compilation-constant-is-not-valid-error"></a>Proje düzeyi koşullu derleme sabiti geçerli değil: \<hata >
Proje için bir geçersiz koşullu derleme seçeneği belirtildi.  
  
 **Hata Kimliği:** BC31031  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Bu proje ile kullanılan koşullu derleme sabitleri doğrulayın.
