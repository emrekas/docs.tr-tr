# <a name="methods-in-a-module-cannot-implement-interface-members"></a>Modül içindeki yöntemler arabirim üyeleri uygulayamaz
Arabirim üyeleri sınıf veya yapı tarafından uygulanmalıdır.  
  
 **Hata Kimliği:** BC31083  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Arabirim üyeleri sınıflar veya yapılar uygulayın.
