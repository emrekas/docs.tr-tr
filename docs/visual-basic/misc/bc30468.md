# <a name="type-declaration-characters-are-not-valid-in-this-context"></a>Tür bildirimi karakterler, bu bağlamda geçerli değildir.
$ Gibi bu bağlamda geçerli olmayan bir karakter kullanma girişimi.  
  
 **Hata Kimliği:** BC30468  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Karakteri kaldırın.
