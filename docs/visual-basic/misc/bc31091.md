# <a name="import-of-type-lttypenamegt-from-assembly-or-module-ltnamegt-failed"></a>İçeri aktarma türü '&lt;typename&gt;'derlemesi veya Modülü' nden&lt;adı&gt;' başarısız oldu
Tüm başvurulan kitaplıkları birbiriyle uyumlu olduğundan emin olun.  
  
 **Hata Kimliği:** BC31091  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Başvurulan tür içeren dosyaları bozuk olmadığını doğrulayın.
