# <a name="import-of-type-typename-from-assembly-or-module-name-failed"></a>İçeri aktarma türü '\<typename >' derleme veya modülünden '\<adı >' başarısız oldu
Tüm başvurulan kitaplıkları birbiriyle uyumlu olduğundan emin olun.  
  
 **Hata Kimliği:** BC31091  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Başvurulan tür içeren dosyaları bozuk olmadığını doğrulayın.
