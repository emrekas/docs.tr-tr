# <a name="statement-cannot-appear-within-a-property-body"></a>Özellik gövdesi içinde deyim bulunamaz
Özellik gövdesi içinde deyim bulunamaz. Son kabul özelliği.  
  
 Beklenmeyen bir deyimi içinde bir özellik ile karşılaşıldı.  
  
 **Hata Kimliği:** BC30634  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Doğrulayın, özelliği doğru söz dizimini kullanır ve ile sona erer bir `End Property` oluşturun.
