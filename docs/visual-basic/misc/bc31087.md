# <a name="array-modifiers-cannot-be-specified-on-both-a-variable-and-its-type"></a>Bir değişken hem türü dizi değiştiricileri belirtilemez.
Dizi değiştiriciler değişkeni hem türünü belirten dizi yok.  
  
 **Hata Kimliği:** BC31087  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Dizi değiştirici tür belirticisi ' kaldırın.
