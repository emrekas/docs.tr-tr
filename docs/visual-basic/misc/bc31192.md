# <a name="error-embedding-win32-manifest-manifest-file-ltfilegt-cannot-be-found"></a>Hata: Win32 bildirimi ekleme Bildirim dosyası &lt;dosya&gt; bulunamıyor
Belirtilen dosya `/win32manifest` derleyici seçeneği bulunamıyor.  
  
 **Hata Kimliği:** BC31192  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Geçerli dosya adı ve yolu sağlayan `/win32manifest` derleyici seçeneği.
