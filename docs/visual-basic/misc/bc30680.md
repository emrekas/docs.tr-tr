# <a name="end-region-must-be-preceded-by-a-matching-region"></a>Bir eşleşen '#Region' '#End Region' gelmelidir
İle `#Region` genişletmek veya daraltmak için anahat oluşturma özelliği Visual Studio Kod Düzenleyicisi'nin kullanırken kod bloğunu belirtebilirsiniz. Başındaki ve sonundaki `#Region` deyimleri aynı kod bloğu içinde olması gerekir.  
  
 **Hata Kimliği:** BC30680  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- INSERT `#Region` karşılık gelen önce uygun yerde `#End Region` deyimi.
