# <a name="the-stream-passed-to-textfieldparser-cannot-be-read"></a>TextFieldParser için geçirilen akış okunamıyor
[TextFieldParser nesnesi](../../visual-basic/language-reference/objects/textfieldparser-object.md) kendisine geçirilen akış okunamıyor. Bu, bir metin dosyası değil bir dosyayı okumaya çalışırken bir sonucu olabilir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Dosya bir metin dosyası olduğundan emin olmak için kontrol edin.
