# <a name="region-statement-must-end-with-a-matching-end-region"></a>'#Region' deyimi eşleşen bir '#End Region' ile bitmelidir
Kullanım `#Region` Genişlet veya daralt anahat oluşturma özelliği, Visual Studio Kod Düzenleyicisi'ni kullanarak bir kod bloğunu belirtmek için. Başındaki ve sonundaki `#Region` deyimleri aynı kod bloğu içinde olması gerekir.  
  
 **Hata Kimliği:** BC30681  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  INSERT `#End Region` koddan sonra uygun yerde `#Region` deyimi.
