# <a name="attribute-value-is-not-valid-expecting-value1-or-value2"></a>Öznitelik değeri geçerli değil; 'value1' veya 'value2' bekleniyor
Bir XML belgesi değişmez değer özniteliği için geçersiz değer belirtildi. Bir XML belgesi değişmez değeri geçerli bir öznitelik değerleri, XML 1.0 belirtimi temel alır.  
  
 **Hata Kimliği:** BC31182  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Öznitelik değeri XML 1.0 belirtimi uyarınca geçerli bir değere değiştirin.
