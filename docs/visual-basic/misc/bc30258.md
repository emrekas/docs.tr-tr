# <a name="classes-can-inherit-only-from-other-classes"></a>Sınıflar, yalnızca diğer sınıflardan devralınabilir.
Bir `Inherits` deyimi, bir sınıf dışında bir tür kendi temel türü belirtir.  
  
 **Hata Kimliği:** BC30258  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Önceden tanımlanmış bir sınıf için temel türünü değiştirin.
