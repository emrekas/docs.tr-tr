# <a name="the-path-has-not-been-set"></a>Yolu ayarlanmadı
Gerekli yolun sağlamadan taşıma veya bir dosya kopyalama gibi bir dosya işlemi girişiminde bulunuldu. Bu işlem, yolu yanlış ayrıştırması neden olabilir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Dosya yolu sağlanmış ve doğru olduğunu kontrol edin.
