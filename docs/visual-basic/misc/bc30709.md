# <a name="variable-declaration-statements-are-not-valid-in-the-immediate-window"></a>Değişken bildirimi deyimler yürütme penceresinde geçerli değildir
Değişken bildirimlerini kullanılamaz **hemen** penceresi.  
  
 **Hata Kimliği:** BC30709  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Değişkenleri bildirmeyin **hemen** penceresi.
