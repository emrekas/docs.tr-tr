# <a name="catch-must-end-with-a-matching-end-try"></a>'Catch' eşleşen bir 'End Try' ile bitmelidir
A `Catch` deyimi olmadan eşleşen kodunuzda görünen `End Try` deyimi. `Catch` deyimleri tamamlanmış, ile bir `End Try` deyimi.  
  
 **Hata Kimliği:** BC30441  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Kaldırma `Catch` deyimi.  
  
2.  Ekleme bir `End Try` blok sonlandırma için deyimi.
