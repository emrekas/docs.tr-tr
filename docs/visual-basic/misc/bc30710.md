# <a name="end-of-expression-expected"></a>İfade Sonlandırılmamış
Aksi halde tam ifadenin sonunda fazladan karakterler bulundu.  
  
 **Hata Kimliği:** BC30710  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Nesnesi veya ifade söz dizimini doğrulayın.
