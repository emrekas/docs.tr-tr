<span data-ttu-id="b0852-105">Tür çıkarımı, genel yordama yönelik bir çağrı değerlendirilirken tür parametresinin veya parametrelerinin veri türünü veya türlerini belirlemekte kullanılmak üzere bir girişimde bulunuldu.</span><span class="sxs-lookup"><span data-stu-id="b0852-105">An attempt has been made to use type inference to determine the data type or types of the type parameter or parameters when evaluating a call to a generic procedure.</span></span> <span data-ttu-id="b0852-106">Derleyici, tüm bağımsız değişkenlerin kısıtlamalarını karşılayan bir veri türü bulamadı.</span><span class="sxs-lookup"><span data-stu-id="b0852-106">The compiler could not find a data type that meets the constraints of all the arguments.</span></span> <span data-ttu-id="b0852-107">Bu nedenle, bu hatayı bildirdi.</span><span class="sxs-lookup"><span data-stu-id="b0852-107">Therefore, it reported this error.</span></span>
