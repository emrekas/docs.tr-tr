# <a name="paramarray-parameter-must-be-a-one-dimensional-array"></a>ParamArray parametresi tek boyutlu bir dizi olmalıdır
A `ParamArray` bağımsız değişkeni bir tek boyutlu dizi türü olarak bildirilmiş. Bildirim için birden fazla boyut belirtir `ParamArray` bağımsız değişken.  
  
 **Hata Kimliği:** BC30051  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Aşağıdaki parantez içinde tüm virgül kaldırmak `ParamArray` bağımsız değişken adı.
