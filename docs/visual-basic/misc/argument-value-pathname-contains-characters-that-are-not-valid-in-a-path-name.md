# <a name="argument-value-pathname-contains-characters-that-are-not-valid-in-a-path-name"></a>Bağımsız değişken değeri '\<yol adı >', yol adında geçersiz karakterler içeriyor
Bir yol adı geçerli olmayan bir veya daha fazla karakter içeriyor.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Karakter yolu adından kaldırın.
