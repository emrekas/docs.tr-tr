# <a name="no-input-sources-specified"></a>Belirtilen giriş kaynağı yok
Derleme sırasında hiç giriş kaynağı belirtilmedi.  
  
 **Hata Kimliği:** BC2008  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bir kaynak kodu dosyası dahil ettiğiniz emin olun.
