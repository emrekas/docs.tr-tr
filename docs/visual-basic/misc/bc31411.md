# <a name="typename-must-be-declared-mustinherit-because-it-contains-methods-declared-mustoverride"></a>'\<typename >', 'MustOverride' olarak bildirilen yöntemler içerdiğinden 'MustInherit' olarak bildirilmelidir
İçeren türleri `MustOverride` üyeleri bildirilmelidir `MustInherit`.  
  
 **Hata Kimliği:** BC31411  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Ekleme `MustInherit` değiştirici türü.
