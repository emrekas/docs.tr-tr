# <a name="non-intrinsic-type-names-are-not-allowed-in-conditional-compilation-expressions"></a>Koşullu derleme deyimlerinde olmayan iç tür adlarına izin verilmemektedir
Koşullu derleme ifade iç olmayan bir türe başvuruyor, ancak yalnızca iç türlere izin verilir.  
  
 Visual Basic'te iç türü hakkında daha fazla bilgi için bkz. [veri türleri özeti](../../visual-basic/language-reference/keywords/data-types-summary.md).  
  
 **Hata Kimliği:** BC31426  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Geçerli olmayan bir tür, bir iç türüyle değiştirin.
