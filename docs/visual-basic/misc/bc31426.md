# <a name="non-intrinsic-type-names-are-not-allowed-in-conditional-compilation-expressions"></a>Koşullu derleme ifadelerinde iç olmayan tür adlarına izin verilmez
Koşullu derleme ifadesi, iç olmayan bir türe başvuruyor, ancak yalnızca iç türlere izin veriliyor.  
  
 Visual Basic iç türü hakkında daha fazla bilgi için bkz. [veri türleri Özeti](../../visual-basic/language-reference/keywords/data-types-summary.md).  
  
 **Hata KIMLIĞI:** BC31426  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- İç olmayan türü bir iç türle değiştirin.
