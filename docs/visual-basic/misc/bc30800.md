# <a name="method-arguments-must-be-enclosed-in-parentheses"></a>Yöntem bağımsız değişkenleri, parantez içine alınmalıdır
Yordam çağrılarını kuralları Visual Basic'in daha yeni sürümlerde daha basittir. Tüm bağımsız değişkenler parantez içine alınması gerekir.  
  
 **Hata Kimliği:** BC30800  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bağımsız değişken listesi ayraç içine alın; Örneğin:  
  
    ```  
    MySub(ArrayIndex, NewValue)  
    ```
