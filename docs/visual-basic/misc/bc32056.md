# <a name="type-parameter-not-allowed-in-implements-clause"></a><span data-ttu-id="dc176-102">Tür parametresi 'Implements' yan tümcesinde izin verilmiyor</span><span class="sxs-lookup"><span data-stu-id="dc176-102">Type parameter not allowed in 'Implements' clause</span></span>
<span data-ttu-id="dc176-103">Bir `Implements` yan tümcesinde bir genel tür, tür parametresi uygulanacak üye olarak belirtir.</span><span class="sxs-lookup"><span data-stu-id="dc176-103">An `Implements` clause in a generic type specifies a type parameter as the member to be implemented.</span></span>  
  
 <span data-ttu-id="dc176-104">Bir `Implements` yan tümcesi bir arabirim ve üye belirtmeniz gerekir.</span><span class="sxs-lookup"><span data-stu-id="dc176-104">An `Implements` clause must specify an interface and a member.</span></span> <span data-ttu-id="dc176-105">Arabirimi için bir tür parametresini geçirebilirsiniz, ancak bunu olamaz üyesine geçirmek ya da üye adı olarak kullanın.</span><span class="sxs-lookup"><span data-stu-id="dc176-105">It can pass a type parameter to the interface, but it cannot pass it to the member, nor use it as the name of the member.</span></span>  
  
 <span data-ttu-id="dc176-106">Aşağıdaki deyimleri, bu hata oluşturabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="dc176-106">The following statements can generate this error.</span></span>  
  
```  
Class c1(Of t)  
    Implements i1(Of t)  
    Public Sub doSomething() Implements t  
End Class  
```  
  
 <span data-ttu-id="dc176-107">**Hata Kimliği:** BC32056</span><span class="sxs-lookup"><span data-stu-id="dc176-107">**Error ID:** BC32056</span></span>
