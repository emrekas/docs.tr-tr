# <a name="ltdeclaration1gt-cannot-override-ltdeclaration2gt-because-it-is-declared-shared"></a>'&lt;declaration1&gt;'geçersiz kılınamaz'&lt;declaration2&gt;' 'Shared' olarak bildirildiğinden
Aynı ada sahip bir devralınan öğe geçersiz kılmak bir yordam veya özellik bildiriminde çalışır, ancak devralınan öğesi olarak belirtilen `Shared`. Bunlar kılınamaması paylaşılan öğeleri herhangi bir sınıf örneği ile ilişkili değildir.  
  
 **Hata Kimliği:** BC30268  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `Shared` anahtar sözcüğü devralınan öğesi ya da geçersiz kılma bildirimi kaldırın.
