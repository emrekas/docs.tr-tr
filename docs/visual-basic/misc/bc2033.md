# <a name="error-embedding-win32-manifest-option-win32manifest-conflicts-with-nowin32manifest"></a>Hata: Win32 bildirimi ekleme / Win32manifest seçeneği/nowin32manifest ile çakışıyor
Hem Visual Basic Derleyicisi geçirildi bir `/win32manifest` derleyici seçeneği ve `/nowin32manifest` derleyici seçeneği. Visual Basic Derleyicisi için bu seçeneklerden yalnızca birini geçirilebilir.  
  
 **Hata Kimliği:** BC2033  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Ya da kaldırma `/win32manifest` derleyici seçeneği veya `/nowin32manifest` derleyici seçeneği.
