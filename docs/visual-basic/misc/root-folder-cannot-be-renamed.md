# <a name="root-folder-cannot-be-renamed"></a>Kök klasör yeniden adlandırılamıyor
Kök dizini yeniden adlandırılamıyor girişiminde bulunuldu. Bu bir hatalı ayrıştırılmış dosya yolunun bir sonucu olabilir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Doğru olduğundan emin olmak için dosya yolunu denetleyin.
