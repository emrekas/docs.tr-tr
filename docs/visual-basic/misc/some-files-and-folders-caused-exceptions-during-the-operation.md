# <a name="some-files-and-folders-caused-exceptions-during-the-operation"></a>Bazı dosya ve klasörleri işlemi sırasında özel durum nedeni
Dizinleri ve dosyaları bir işlem denendi ancak kullanıcı gerekli izinlere sahip değil. özel durum neden olabilir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Tüm dosya ve klasörlere erişmesi gereken izinleri ekleyin.
