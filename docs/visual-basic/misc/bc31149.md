# <a name="duplicate-xml-attribute-attributename"></a>XML özniteliği 'attributeName' yinelenen
Belirtilen bir XML öğesi için sabit bir öznitelik XML öğesi değişmez değeri için bir önceden tanımlanmış bir öznitelik ile eşleşir. XML öğesi değişmez değerlerinde öznitelik adları benzersiz olmalıdır.  
  
 **Hata Kimliği:** BC31149  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Öznitelik XML öğesi değişmez değeri için benzersiz bir adla yeniden adlandırın.
