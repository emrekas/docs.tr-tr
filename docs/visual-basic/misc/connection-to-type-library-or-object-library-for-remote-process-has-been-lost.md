# <a name="connection-to-type-library-or-object-library-for-remote-process-has-been-lost"></a>Kitaplık veya uzak işlem için Nesne Kitaplığı bağlantısı kesildi
Uzak işlem nesne kitaplığı veya tür kitaplığı için bağlantınız kesildi.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Yeniden `Application` nesne.  
  
2.  Bu Yardım konusu girdiğiniz hata iletişim kutusunda tıklatın **Tamam** görüntülenecek **başvuruları** iletişim kutusu. "Eksik solunda" sözcüğüyle kayıp başvuru görünür.  
  
3.  Kayıp başvuru silin.  
  
4.  İçinde **başvuruları** için iletişim kutusunda, onay kutusunu seçin `Application` nesne.
