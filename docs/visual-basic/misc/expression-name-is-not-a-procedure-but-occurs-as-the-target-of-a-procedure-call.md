# <a name="expression-name-is-not-a-procedure-but-occurs-as-the-target-of-a-procedure-call"></a>İfade '\<adı >' bir yordam değil, ancak bir yordam çağrısı hedefi olarak gerçekleşir.
Bir yordam bir yordam değil bir ifade için çağrısı yapıldı.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Yordamın adını yazsanız değil emin olun.  
  
2.  Yordam sağlayın, <`name`> çağırır.
