# <a name="operator-is-not-valid-for-type-lttypenamegt"></a>İşleç türü için geçerli değil '&lt;typename&gt;'
Bir işleç türleri için uygun değil.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   İşlecin üzerinde işlem yapılan türleri için uygun olduğundan emin olun.
