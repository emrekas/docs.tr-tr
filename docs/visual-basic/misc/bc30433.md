# <a name="methods-in-a-module-cannot-be-declared-ltspecifiergt"></a>Modül içindeki yöntemler bildirilemez '&lt;belirticisi&gt;'
Yöntemi içinde geçerli bir tanımlayıcı kullandığınız bir `Module` deyimi. Modüller asla oluşturulamaz, devralmayı desteklemeyen ve arabirimler uygulanamaz.  
  
 **Hata Kimliği:** BC30433  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Tanımlayıcısını kaldırın.
