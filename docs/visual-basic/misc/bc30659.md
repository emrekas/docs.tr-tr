# <a name="property-or-field-name-does-not-have-a-valid-attribute-type"></a>'\<Name > ' özelliğinin veya alanının geçerli bir öznitelik türü yok
Öznitelik başka uygun oluşturuculara sahip olsa da, geçerli bir öznitelik türü kabul olmayan bir öznitelik Oluşturucu kullanıyor olabilirsiniz.  
  
 **Hata KIMLIĞI:** BC30659  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Bu özniteliğin kullanımını kontrol edin.
