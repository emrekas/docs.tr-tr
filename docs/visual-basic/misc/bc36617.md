# <a name="aggregate-function-name-cannot-be-used-with-a-type-character"></a>Toplama işlevi adı bir tür karakteri ile kullanılamaz
Türü bir karakteri bir toplama işlevi adı sonunda eklenmiştir.  
  
 **Hata Kimliği:** BC36617  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Toplama işlevi adından tür karakteri kaldırın.
