# <a name="specifier1-type-cannot-inherit-from-a-specifier2-type-because-it-expands-the-access-of-the-base-type"></a>\<specifier1 > \<türü > devralamaz bir \<specifier2 > \<türü > temelinin erişimini genişlettiğinden \<türü >
Bir ortak sınıf olarak işaretlenmiş bir taban sınıftan devralın çalıştınız `Private` veya `Friend`.  
  
 **Hata Kimliği:** BC30509  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Temel sınıf bildirme `Public` veya devralan sınıf bildirme `Private` veya `Friend`.
