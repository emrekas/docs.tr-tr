# <a name="ltspecifier1gt-lttypegt-cannot-inherit-from-a-ltspecifier2gt-lttypegt-because-it-expands-the-access-of-the-base-lttypegt"></a>&lt;specifier1&gt; &lt;türü&gt; devralamaz bir &lt;specifier2&gt; &lt;türü&gt; temelinin erişimini genişlettiğinden &lt;türü&gt;
Bir ortak sınıf olarak işaretlenmiş bir taban sınıftan devralın çalıştınız `Private` veya `Friend`.  
  
 **Hata Kimliği:** BC30509  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Temel sınıf bildirme `Public` veya devralan sınıf bildirme `Private` veya `Friend`.
