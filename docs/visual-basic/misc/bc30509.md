# <a name="specifier1-type-cannot-inherit-from-a-specifier2-type-because-it-expands-the-access-of-the-base-type"></a>\<specifier1 > \<türü >, temel \< \<türün erişimini genişlettiğinden \<> bir specifier2 > türünden devralma >
`Private` Veya`Friend`olarak işaretlenmiş bir temel sınıftan bir ortak sınıfa devralma girişiminde bulunuldu.  
  
 **Hata KIMLIĞI:** BC30509
