# <a name="argument-cannot-be-less-than-zero"></a>Bağımsız değişken sıfırdan küçük olamaz
Bir değer sıfır belirtilmiş, ancak bağımsız değişken sıfırdan küçük olamaz daha az olmasıdır.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Nasıl söz konusu değer, doğru olduğundan emin olmak için hesaplanır denetleyin.
