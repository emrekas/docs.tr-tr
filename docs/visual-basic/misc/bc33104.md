# <a name="if-operator-requires-either-two-or-three-operands"></a>'If' işleci iki veya üç işlenen gerektirir
`If` İşleci yalnızca iki veya üç bağımsız değişken kabul eder.  
  
 **Hata Kimliği:** BC33104  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   İnceleme `If` işleçleri sayısı sınırlı veya iki veya üç bağımsız değişkenler için genişletilmiş olup olmadığını görmek için ifade.  
  
-   Başka bir koşullu yapı, gibi kullanmayı göz önünde bulundurun bir `If...Then...Else` deyimi veya bir iç içe `If...Then...Else` bağımsız değişkenleri düzenlemek için deyimi.
