# <a name="unable-to-access-member"></a>Üye erişilemiyor
Belirtilen üye erişilemedi.  
  
 **Hata Kimliği:** BC30749  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Üye tanımlandı ve, üye adının doğru yazıldığından olduğunu doğrulayın.  
  
2. Modülde tanımlanmış üyelerinden erişmeyi deneyin. Bazı durumlarda, burada bildirildikleri modülleri henüz yüklenmedi çünkü hata ayıklama ortamında üyeleri bulamıyor.
