# <a name="the-remote-server-machine-does-not-exist-or-is-unavailable-visual-basic"></a>Uzak Sunucu makinesi yok veya olduğundan kullanılamıyor (Visual Basic)
Kullanılamıyor veya yok, uzak bir sunucuya bağlanmaya çalışıyorsunuz. Kayıp ağ bağlantınızı tarafından bu hataya neden olabilir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Ağa yeniden bağlanıp tekrar deneyin.
