<span data-ttu-id="01191-103">XML değişmez değer bildirimi, Visual Basic Derleyicisi için belirsiz bir konumda bir ifadede kullanılır.</span><span class="sxs-lookup"><span data-stu-id="01191-103">An XML literal declaration is used in an expression in a location that is ambiguous to the Visual Basic compiler.</span></span> <span data-ttu-id="01191-104">Diğer bir deyişle, Visual Basic Derleyicisi belirlenemiyor olup az-karakteri (<) karşılaştırma işleci veya bir XML sabit değerinin başlangıcı olarak istenenden.</span><span class="sxs-lookup"><span data-stu-id="01191-104">That is, the Visual Basic compiler cannot determine whether the less-than character (<) is intended as a comparative operator or the start of an XML literal.</span></span> <span data-ttu-id="01191-105">Aşağıdaki kod, bir örnek sağlar:</span><span class="sxs-lookup"><span data-stu-id="01191-105">The following code provides an example:</span></span>  

```vb  
' Generates an error.  
Dim queryResult = From element In elements _  
                  Where <sample>Value</sample> = "Value" _  
                  Select element  
```  
  
 <span data-ttu-id="01191-106">**Hata Kimliği:** BC31198</span><span class="sxs-lookup"><span data-stu-id="01191-106">**Error ID:** BC31198</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="01191-107">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="01191-107">To correct this error</span></span>

```vb  
                  Select element  
```
