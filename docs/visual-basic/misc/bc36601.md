# <a name="type-characters-cannot-be-used-in-range-variable-declarations"></a>Tür karakterleri aralığı değişken bildirimlerinde kullanılamaz
LINQ sorgusu aralık değişkeni bildiriminde adını, sonunda bir tür karakteri eklenmiştir.  
  
 **Hata Kimliği:** BC36601  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Aralık değişkeni bildiriminden türü karakteri kaldırın.
