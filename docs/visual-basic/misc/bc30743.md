# <a name="type-lttypenamegt-is-not-defined-or-the-module-containing-it-is-not-loaded-in-the-debugging-session"></a>Türü '&lt;typename&gt;' tanımlı değil veya hata ayıklama oturumunda içeren Modülü yüklü değil
Bir tür kullanılabilir değil ya da tanımlanmamış başvuruldu.  
  
 **Hata Kimliği:** BC30743  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Tür adının doğru yazıldığından emin olun.  
  
2.  Geçerli sınıf veya Modül içindeki tür tanımlar.
