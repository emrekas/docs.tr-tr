# <a name="set-not-permitted"></a>İzin kümesi
Özellik ayarlarını ya da çalışma zamanında ayarlanamaz veya başka yalnızca belirli koşullar altında ayarlanabilir değiştirilmeye çalışıldı. Örneğin, değiştirme denediniz `Appearance`, `ControlBox`,`MinButton`, veya `MaxButton` çalıştırma veya form için özellik ayarları denedi ayarlamak `Visible` özelliğini `False` son kalan üst menüde alt menü görünür.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Özelliğini denetleyin ve hangi koşullar altında ayarlanabilir belirleyin.
