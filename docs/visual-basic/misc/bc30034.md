# <a name="bracketed-identifier-is-missing-closing-"></a>Kapanış köşeli parantez içindeki tanımlayıcısı eksik ']'
Kaçan bir adı köşeli ayraç eşleşen çiftleri gerçekleşmelidir.  
  
 **Hata Kimliği:** BC30034  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Atlanan adın sonunda kapanış köşeli ayraç yerleştirin.
