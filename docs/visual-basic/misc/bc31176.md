# <a name="character-character-0xhexadecimal-is-not-allowed-in-xml-literal"></a>Karakter 'character' (0 x\<onaltılık >) XML değişmez değer izin verilmiyor
Desteklenmeyen bir karakter, bir XML değişmez değer kullanıldı.  
  
 **Hata Kimliği:** BC31176  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Desteklenmeyen karakteri kaldırın.
