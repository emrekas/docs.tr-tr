# <a name="event-name-length-cannot-exceed-1011-characters"></a>Olay adının uzunluğu 1011 değerinin kodunu karakterden uzun olamaz
Adı veya her olay tanıtıcısı ile bildirilen bir `Event` deyimi 1011 değerinin kodunu karakterle sınırlıdır.  
  
 **Hata Kimliği:** BC32204  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Tanımlayıcının uzunluğunu azaltın.
