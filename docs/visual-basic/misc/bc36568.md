# <a name="ltelementnamegt-is-not-a-method-parameter"></a>'&lt;elementname&gt;' bir yöntem parametresi değil
Yordam çağrısı yöntemi tarafından tanımlanmayan bir ek bağımsız değişkeni sağlar.  
  
 **Hata Kimliği:** BC36568  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Gereksiz bağımsız değişken bağımsız değişken listesinden kaldırın.
