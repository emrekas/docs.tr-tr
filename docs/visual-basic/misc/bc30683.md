# <a name="inherits-statement-must-precede-all-declarations-in-a-class"></a>' Inherits ' deyimleri bir sınıftaki tüm bildirimlerden önce gelmelidir
`Inherits` Deyimler sınıf tanımlarının başında olmalıdır.  
  
 **Hata KIMLIĞI:** BC30683  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Inherits` Deyiminizi sınıfınızın en üstüne taşıyın.
