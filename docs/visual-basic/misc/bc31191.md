# <a name="error-embedding-win32-manifest-manifest"></a>Win32 Bildirim gömülüyor hata: \<bildirim >
Belirtilen dosya `/win32manifest` derleyici seçeneği eklenemiyor. Bu, yetersiz güvenlik ayrıcalıkları bildirim dosyasına bir sonucu olabilir.  
  
 **Hata Kimliği:** BC31191  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Visual Basic derleyici komutu çalıştıran kimliğin Win32 bildirim dosyası erişimi olduğundan emin olun.
