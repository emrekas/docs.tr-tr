# <a name="method-methodname-cannot-be-declared-partial-because-only-one-method-methodname-can-be-marked-partial"></a>Yöntemi '\<yöntemAdı >' 'Partial' bildirilemez yalnızca bir yöntem '\<yöntemAdı >' 'Partial' olarak işaretlenmiş
Aynı ada ve imzaya sahip iki kısmi yöntem bildiriminin sahip olamaz.  
  
 **Hata Kimliği:** BC31433  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Adı veya bir kısmi yöntem bildiriminin imzasının değiştirin.
