<span data-ttu-id="89a7d-103">Bir bildirim veya atama ifadesi içeren bir [,](../../visual-basic/language-reference/statements/of-clause.md) genel olmayan bir tür çağrılırken yan tümcesi.</span><span class="sxs-lookup"><span data-stu-id="89a7d-103">A declaration or assignment statement includes an [Of](../../visual-basic/language-reference/statements/of-clause.md) clause when invoking a non-generic type.</span></span>  
  
 <span data-ttu-id="89a7d-104">Kendi tanımına göre bir *genel tür* bir sınıf, yapı, arabirim, yordam ya da bir veya daha fazla, veri türlerinde çalışır temsilci belirtebilirsiniz *tür parametrelerindeki*.</span><span class="sxs-lookup"><span data-stu-id="89a7d-104">By its definition, a *generic type* is a class, structure, interface, procedure, or delegate that operates on data types you can specify through one or more *type parameters*.</span></span> <span data-ttu-id="89a7d-105">Bir türü kullanarak kod oluşturur, bu genel türden sağlayan bir *tür bağımsız değişkeni* her tür parametresi için.</span><span class="sxs-lookup"><span data-stu-id="89a7d-105">When the using code creates a type from this generic type, it supplies a *type argument* to each type parameter.</span></span> <span data-ttu-id="89a7d-106">Tür oluşturma işleminin bir parçası olarak, karşılık gelen bir tür parametresi oluşturulan kodda her geçtiği her tür bağımsız değişkeni değiştirir.</span><span class="sxs-lookup"><span data-stu-id="89a7d-106">As part of the creation of the type, each type argument replaces every occurrence of its corresponding type parameter in the generated code.</span></span>  
  
 <span data-ttu-id="89a7d-107">Tür parametreleri ile tanımlanan bir `Of` yan tümcesi içinde parantez ve tür bağımsız değişkenleri kullanarak sağlanacak olan bir `Of` parantez içinde yan tümcesi.</span><span class="sxs-lookup"><span data-stu-id="89a7d-107">Type parameters are defined with an `Of` clause inside parentheses, and type arguments are supplied using an `Of` clause inside parentheses.</span></span> <span data-ttu-id="89a7d-108">`Of` Yan tümcesi yalnızca genel türleri ile işlem yapılırken kullanılır.</span><span class="sxs-lookup"><span data-stu-id="89a7d-108">The `Of` clause is used only when dealing with generic types.</span></span>  
  
 <span data-ttu-id="89a7d-109">Genel olmayan türler, tür parametreleri kabul etmemeyi ve böyle bir türü çağırdığınızda, herhangi bir tür bağımsız değişkeni belirtemezsiniz.</span><span class="sxs-lookup"><span data-stu-id="89a7d-109">Non-generic types do not accept type parameters, and you cannot specify any type arguments when you invoke such a type.</span></span>
