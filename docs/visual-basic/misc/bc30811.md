# <a name="redim-statements-can-no-longer-be-used-to-declare-array-variables"></a>'ReDim' deyimleri artık dizi değişkenleri bildirmek için kullanılabilir
`ReDim` yalnızca var olan bir dizinin boyutunu değiştirmek için kullanılabilir.  
  
 **Hata Kimliği:** BC30811  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bunlar bildirildiğinde diziler boyutunu belirtin; Örneğin:  
  
    ```  
    Dim X(20) As Integer  
    ```
