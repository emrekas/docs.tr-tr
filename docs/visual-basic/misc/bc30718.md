# <a name="select-statements-are-not-valid-in-the-immediate-window"></a>'Select' deyimleri yürütme penceresinde geçerli değildir
`Select` deyimleri yalnızca kaynak kodunda kullanılabilir.  
  
 **Hata Kimliği:** BC30718  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Kaldırma `Select` kodda hata ayıklama gelen deyimleri.
