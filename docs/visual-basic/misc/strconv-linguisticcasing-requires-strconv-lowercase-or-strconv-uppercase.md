# <a name="strconvlinguisticcasing-requires-strconvlowercase-or-strconvuppercase"></a>'StrConv.LinguisticCasing' 'StrConv.LowerCase' veya 'StrConv.UpperCase' gerektirir
Kullanma girişiminde bulundunuz `StrConv.LinguisticCasing`, geçerli yalnızca birlikte `StrConv.LowerCase` veya `StrConv.UpperCase`.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Kullanım `StrConv.LowerCase` veya `StrConv.UpperCase` birlikte `StrConv.LinguisticCasing`.
