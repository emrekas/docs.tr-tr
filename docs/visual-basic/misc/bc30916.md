# <a name="type-lttypenamegt-is-not-supported-because-it-either-directly-or-indirectly-inherits-from-itself"></a>Türü '&lt;typename&gt;', doğrudan veya dolaylı olarak kendisinden devraldığından desteklenmiyor
Bir sınıf veya arabirim ya da kendi içinden veya başka bir sınıf ya da en sonunda ondan devralır arabirimi devralır.  
  
 Visual Basic döngüsel devralmaya desteklemez.  
  
 **Hata Kimliği:** BC30916  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Devralma yapısı değiştirin, böylece bir temel sınıf veya hiçbir diğer sınıf veya arabirim devralmayan arabirimi üzerine inşa edilmiştir.
