# <a name="method1-cannot-override-method2-because-they-differ-by-the-default-values-of-optional-parameters"></a>isteğe bağlı parametrelerinin varsayılan değerleri farklı olduğundan\<'Method1>','Method2>'öğesinigeçersizkılamaz\<
İsteğe bağlı parametrelerinin varsayılan değerlerinde birinciden farklı olan bir yöntemi geçersiz kılmayı denediniz, yani imzaları farklıdır. Bir tür, aynı ada ve imzaya sahip bir yöntemi bildirerek ve açıklamayı `Overrides` değiştiriciyle işaretleyerek devralınan geçersiz kılınabilir bir yöntemi geçersiz kılabilir.  
  
 **Hata KIMLIĞI:** BC30307  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- İki yöntemin de aynı imzaya sahip olduğundan emin olun.
