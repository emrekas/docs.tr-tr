# <a name="name-necessary-for-compiling-this-construct-cannot-be-found"></a>'\<adı >', bu yapı derleme bulunamıyor için gerekli
Derleme için gerekli olan bir öğe bulunamıyor.  
  
 **Hata Kimliği:** BC30930  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Gerekli yardımcı öğe sağlayın.  
  
2.  Kod geçerli yönetilen kod olduğundan emin olun.
