# <a name="methods-declared-overrides-cannot-be-declared-overridable-because-they-are-implicitly-overridable"></a>Örtük olarak geçersiz kılınabilir oldukları için bildirilen yöntemler 'Overrides' 'Overridable' olarak bildirilemez
Çoğu yöntemlerinden farklı olarak ile işaretlenmiştir `Overrides` değiştiricisi varsayılan olarak geçersiz kılınabilir.  
  
 **Hata Kimliği:** BC30730  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Atlamak `Overridable` değiştiricisi yöntemlerinden ile işaretli `Overrides` değiştiricisi.
