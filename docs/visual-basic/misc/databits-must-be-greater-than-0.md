# <a name="databits-must-be-greater-than-0"></a>DataBits 0'dan büyük olmalıdır
`DataBits` Bağımsız değişkeni için belirtilen `My.Computer.Ports.OpenSerialPort` yöntemi sıfırdan büyük olmalıdır.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Değiştirin `DataBits` bağımsız değişkeni için pozitif bir sayı.
