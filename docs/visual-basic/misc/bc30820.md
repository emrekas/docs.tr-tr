# <a name="lsetstmt-is-not-declared"></a>'\<lsetstmt >' bildirimi yapılmadı
'\<lsetstmt >' değil olarak bildirilir. LSet deyimleri artık desteklenmiyor; Bunun yerine Microsoft.VisualBasic.LSet kullanın.  
  
 Önceki sürümlerde Visual Basic iç çeşitli işlevler taşındı <xref:Microsoft.VisualBasic?displayProperty=nameWithType> ad alanı. Bu işlevleri tüm programlama dilleri için daha genel olarak kullanılabilir hale getirir.  
  
 **Hata Kimliği:** BC30820  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kullanım `LSet` işlevi `Microsoft.VisualBasic` ad alanı yerine.
