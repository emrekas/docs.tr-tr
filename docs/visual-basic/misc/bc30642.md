# <a name="optional-and-paramarray-cannot-be-combined"></a>'Optional' ve 'ParamArray' birleştirilemez
`Optional` Ve `ParamArray` değiştirici karşılıklı olarak birbirini dışlar.  
  
 **Hata Kimliği:** BC30642  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Seçeneklerinden birini belirtin `ParamArray` veya `Optional`, ikisini birden belirtmeyin.
