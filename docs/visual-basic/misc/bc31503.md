# <a name="typename-cannot-be-used-as-an-attribute-because-it-is-not-a-class"></a>'\<typename >', bir sınıf olmadığından öznitelik olarak kullanılamaz
Bir sınıf özniteliği olarak değil bir türü kullanmak için girişimde bulunuldu.  
  
 **Hata Kimliği:** BC31503  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Türetilen sınıflar olarak özel öznitelikler tanımlamak `System.Attribute`.
