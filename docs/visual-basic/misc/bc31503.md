# <a name="typename-cannot-be-used-as-an-attribute-because-it-is-not-a-class"></a>'\<TypeName > ' bir sınıf olmadığından öznitelik olarak kullanılamaz
Öznitelik olarak sınıf olmayan bir tür kullanılmaya çalışıldı.  
  
 **Hata KIMLIĞI:** BC31503  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Özel öznitelikleri öğesinden `System.Attribute`türetilen sınıflar olarak tanımlayın.
