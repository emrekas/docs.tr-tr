# <a name="file-io-with-type-typename-is-not-valid"></a>Dosya g/ç türü ile '\<typename >' geçerli değil
Desteklenmeyen bir tür bir dosyaya yazmak çalışıyorsunuz.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Desteklenmeyen tür nerede oluştuğunu belirlemek için yapısını kontrol edin ve ardından geçerli bir tür olarak değiştirin.
