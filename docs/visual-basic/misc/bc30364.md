# <a name="sub-new-cannot-be-declared-modifier"></a>' Sub New ', '\<Modifier > ' olarak bildirilemez
Geçerli olmayan bir değiştirici `Sub New` `Overrides`ile bir (gibi) bildirmeye çalıştınız.  
  
 **Hata KIMLIĞI:** BC30364  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Değiştiricisini kaldırın.
