# <a name="sub-new-cannot-be-declared-ltmodifiergt"></a>'Sub New' olarak bildirilemez '&lt;değiştiricisi&gt;'
Bildirme girişiminde bulundunuz bir `Sub New` bir değiştiricisi gibi `Overrides`, yani geçerli değil.  
  
 **Hata Kimliği:** BC30364  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Değiştiricisi kaldırın.
