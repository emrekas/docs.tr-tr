# <a name="type-characters-are-not-allowed-in-label-identifiers"></a>Etiket tanımlayıcılarının türü karakterlere izin verilmez
Bir deyim etiket en az bir tanımlayıcı türü karakteri içerir.  
  
 Bir deyimi etiketi geçerli bir Visual Basic adı olmalıdır. İzin verilen karakter tanımlayıcı türü karakterlerin hiçbirini içermez (`%`, `&`, `@`, `!`, `#`, ve `$`). Bkz: [bildirilen öğe adları](../../visual-basic/programming-guide/language-features/declared-elements/declared-element-names.md).  
  
 **Hata Kimliği:** BC31395  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Tür tanımlayıcı karakteri veya karakterleri deyimi etiketten Kaldır.
