# <a name="specified-registry-key-is-not-valid-because-it-contains-two-or-more-consecutive-backslashes"></a>İki veya daha fazla ardışık ters eğik çizgiler içerdiğinden belirtilen kayıt defteri anahtarı geçerli değil
Bir yol ile belirtilen bir kayıt defteri anahtarı, iki veya daha fazla ardışık ters eğik çizgi içerir. Bu işlem, yolu oluşturmak için birkaç dizelerini birleştirme ve yanlışlıkla çok fazla ters eğik çizgi dahil olmak üzere bir sonucu olabilir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Nerede ve neden fazladan eğik eklenme belirlemek için belirtilen kayıt defteri anahtarını inceleyin.
