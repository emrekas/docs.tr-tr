# <a name="functionname-is-not-declared-visual-basic-error"></a>'\<functionname >' bildirilmedi (Visual Basic hatası)
'\<functionname >' değil olarak bildirilir. Dosya g/ç işlevselliği Microsoft.VisualBasic ad alanında kullanılabilir.  
  
 **Hata Kimliği:** BC30815  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Tanımlanan dosya g/ç işlevleri `My.Computer.FileSystem`, `System.IO` ad alanı, Visual Basic çalışma zamanı işlevleri veya `Microsoft.VisualBasic.FileSystem` nesne.
