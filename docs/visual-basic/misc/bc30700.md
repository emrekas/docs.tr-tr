# <a name="side-effects-not-valid-during-expression-evaluation-in-this-context"></a>Yan etkileri sırasında ifade değerlendirmesi bu bağlamda geçerli değil
Bir nesnenin durumunu değiştiren deyimleri bu bağlamda izin verilmez.  
  
 **Hata Kimliği:** BC30700  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Diğer öğelerinin durumunu değiştirmeden ifadeler değerlendirilir, böylece kodunuzu değiştirin.
