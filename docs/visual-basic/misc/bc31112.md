# <a name="statement-cannot-appear-within-an-event-body"></a>Bir olay gövdesi içinde deyim bulunamaz
Bir olay gövdesi içinde deyim bulunamaz. Olay kabul sonu.  
  
 Bir olay gövdesi içinde geçersiz bir deyim bir içinde görünür.  
  
 **Hata Kimliği:** BC31112  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Ekleme bir `End Event` deyiminden önce.
