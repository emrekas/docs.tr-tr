# <a name="possible-problem-detected-while-building-assembly-error"></a>Derleme oluşturulurken olası sorun algılandı: \<hata >
Visual Basic derleyici tarafından adlandırılan ALink aracını derleme oluşturulurken bir hata bildirir. Olası nedenlerinden biri, imzalanmamış bir derlemeye başvuru yapma imzalı bir derleme olmasıdır.  
  
 Bu ileti bir uyarıdır. Derleyici, derleme oluşturmak devam ediyor. Uyarıları gizleme veya uyarıları hata olarak değerlendirmesini daha fazla bilgi için bkz. Lütfen [Visual Basic'teki uyarıları yapılandırma](/visualstudio/ide/configuring-warnings-in-visual-basic).  
  
 **Hata Kimliği:** BC40009  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Tırnak işaretli hata iletisini inceleyin ve uygun eylemi gerçekleştirin.  
  
2.  Yinelenen hata durumunda yeniden görmek için programı derleyin.  
  
3.  Hata almaya devam ederseniz, Visual Basic Derleyicisi yeniden yükleyin.  
  
4.  Yeniden yüklenmesinden sonra hata oluşmaya devam ederse koşullar hakkında bilgi toplamak ve Microsoft Ürün Destek Hizmetleri bilgilendirir.
