# <a name="date-constant-is-not-valid"></a>Sabit tarih geçerli değil
Bir tarih sabit geçersiz bir tarih ile bildirilir.  
  
 **Hata Kimliği:** BC31085  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Sabiti kullanılan tarih düzeltin.
