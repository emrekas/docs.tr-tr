# <a name="xml-comment-must-be-the-first-statement-on-a-line"></a>XML açıklaması satırdaki ilk deyim olmalıdır
XML açıklaması satırdaki ilk deyim olmalıdır. XML açıklaması yoksayılacak.  
  
 XML açıklaması izin başka bir dil öğesi sonra yerleştirilir.  
  
 **Hata Kimliği:** BC42302  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- XML açıklamasında, yeni bir satıra Taşı.
