# <a name="derivedtypename-cannot-inherit-from-type-constructedbasetypename-because-it-expands-the-access-of-type-internaltypename-outside-the-assembly"></a><span data-ttu-id="a9b1e-102">'\<derivedtypename >' öğesinden devralamaz \<türü > '\<constructedbasetypename >' türünün erişimini genişlettiğinden '\<internaltypename >' derleme dışından</span><span class="sxs-lookup"><span data-stu-id="a9b1e-102">'\<derivedtypename>' cannot inherit from \<type> '\<constructedbasetypename>' because it expands the access of type '\<internaltypename>' outside the assembly</span></span>
<span data-ttu-id="a9b1e-103">Türetilen bir sınıfta veya arabirimde bir temel sınıf veya arabirim türü bağımsız değişkeni olarak kullanarak kısıtlanmış bir tür erişim düzeyini genişletmek çalışır.</span><span class="sxs-lookup"><span data-stu-id="a9b1e-103">A derived class or interface attempts to expand the access level of a restricted type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="a9b1e-104">Aşağıdaki kod, bu hata oluşturabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="a9b1e-104">The following code can generate this error.</span></span>  
  
```  
Public Class baseClass(Of t)  
End Class  
Public Class derivedClass  
    Inherits baseClass(Of restrictedStructure)  
End Class  
Friend Structure restrictedStructure  
    Dim firstMember As Integer  
End Structure  
```  
  
 <span data-ttu-id="a9b1e-105">Kod derleme dışından erişmesine izin verilmiyor `restrictedStructure`.</span><span class="sxs-lookup"><span data-stu-id="a9b1e-105">Code outside the assembly is not allowed to access `restrictedStructure`.</span></span> <span data-ttu-id="a9b1e-106">Ancak, `derivedClass` ona başvuran herhangi bir koddan erişilebilir.</span><span class="sxs-lookup"><span data-stu-id="a9b1e-106">However, `derivedClass` can be accessed from any code that can reference it.</span></span> <span data-ttu-id="a9b1e-107">Bu nedenle, `derivedClass` devralamaz `baseClass` kullanılıyorsa `restrictedStructure` bir tür bağımsız değişkeni olarak, maruz bırakabileceğinden `restrictedStructure` kodunda herhangi bir derleme.</span><span class="sxs-lookup"><span data-stu-id="a9b1e-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `restrictedStructure` as a type argument, because that could expose `restrictedStructure` to code in any assembly.</span></span>  
  
 <span data-ttu-id="a9b1e-108">**Hata Kimliği:** BC30922</span><span class="sxs-lookup"><span data-stu-id="a9b1e-108">**Error ID:** BC30922</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="a9b1e-109">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="a9b1e-109">To correct this error</span></span>  
  
- <span data-ttu-id="a9b1e-110">Türetilmiş tür kısıtlanmış türdeki erişim düzeyini genişletmeyen şekilde arabirimler ve sınıflar erişim düzeyini ayarlayın.</span><span class="sxs-lookup"><span data-stu-id="a9b1e-110">Adjust the access levels of the classes or interfaces so the derived type does not expand the access level of the restricted type.</span></span>  
  
     <span data-ttu-id="a9b1e-111">-veya-</span><span class="sxs-lookup"><span data-stu-id="a9b1e-111">-or-</span></span>  
  
- <span data-ttu-id="a9b1e-112">Erişim düzeyleri ayarlayamazsınız, kısıtlanmış türdeki temel sınıfı veya arabirimi oluştururken bir tür bağımsız değişkeni kullanmayın.</span><span class="sxs-lookup"><span data-stu-id="a9b1e-112">If you cannot adjust the access levels, do not use the restricted type as a type argument when constructing the base class or interface.</span></span>
