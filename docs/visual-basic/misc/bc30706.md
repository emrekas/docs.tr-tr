# <a name="expression-or-statement-is-not-valid-in-debug-windows"></a>İfade veya deyimin hata ayıklama pencerelerinde geçerli değil
`Try...Catch...Finally` deyimleri, hata ayıklama bağlamda kullanılamaz.  
  
 **Hata Kimliği:** BC30706  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Kaldırma `Try...Catch...Finally` kodda hata ayıklama gelen deyimleri.
