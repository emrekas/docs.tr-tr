# <a name="readonly-and-writeonly-cannot-be-combined"></a>'ReadOnly' ve 'WriteOnly' birleştirilemez
`ReadOnly` Ve `WriteOnly` değiştirici karşılıklı olarak birbirini dışlar.  
  
 **Hata Kimliği:** BC30635  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Seçeneklerinden birini belirtin `ReadOnly` veya `WriteOnly`, ikisini birden belirtmeyin.
