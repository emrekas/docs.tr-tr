# <a name="option-ltspecifiergt-statement-can-only-appear-once-per-file"></a>' Seçeneği &lt;belirticisi&gt;' deyimi yalnızca görüntülenebilir kez dosya
`Option Compare`, `Option Explicit`, Ve `Option Strict` deyimleri yalnızca kullanılabilir bir kez aynı kaynak dosyada.  
  
 **Hata Kimliği:** BC30225  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yinelenen kaldırma `Option` deyimi.
