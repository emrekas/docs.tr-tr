# <a name="option-specifier-statement-can-only-appear-once-per-file"></a>' Seçeneği \<belirticisi >' deyimi yalnızca görüntülenebilir kez dosya
`Option Compare`, `Option Explicit`, Ve `Option Strict` deyimleri yalnızca kullanılabilir bir kez aynı kaynak dosyada.  
  
 **Hata Kimliği:** BC30225  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Yinelenen kaldırma `Option` deyimi.
