# <a name="xml-literals-and-xml-axis-properties-are-not-available-during-this-debugging-session-because-they-are-not-used-in-this-assembly"></a>Bu derlemede kullanılmadığından XML sabit değerleri ve XML eksen özellikleri bu hata ayıklama oturumu sırasında kullanılabilir değil
XML değişmez değeri ya da XML axis özelliği olarak başvuruldu **Watch** veya **hemen** penceresi Visual Basic'te hangi XML'de özellikleri kullanılabilir değil hata ayıklama oturumu sırasında. Bu durum, XML, Visual Basic özellikleri kullanmaz veya bir yayın yapısı bir derleme için geçerlidir.  
  
 **Hata Kimliği:** BC31196  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Derleme hata ayıklama yapısını kullanarak yeni bir hata ayıklama oturumu başlatın.
