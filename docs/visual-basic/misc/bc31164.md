# <a name="expected-matching-closing-double-quote-for-xml-attribute-value"></a>Kapanış çift tırnak işareti XML öznitelik değeri için eşleştirme bekleniyor
Kapanış çift tırnak işareti ayrılmış bir öznitelik değeri eksik tırnak işareti karakteri.  
  
 **Hata Kimliği:** BC31164  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Öznitelik değeri sınırlandırmak için kapanış çift tırnak işareti karakteri Ekle.
