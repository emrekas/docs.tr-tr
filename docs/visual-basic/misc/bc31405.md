# <a name="optional-parameters-cannot-have-structure-types"></a>İsteğe bağlı parametreleri yapı türleri olamaz
İsteğe bağlı parametreler, yapı türleri olamaz.  
  
 **Hata Kimliği:** BC31405  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Yapı türü parametreleri gerekli olarak bildirin.  
  
2.  Bir yapının gerçekleşmesi için bir statik sınıf üyesi bildirin.
