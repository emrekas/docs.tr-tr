# <a name="method1-cannot-override-method2-because-they-differ-by-optional-parameters"></a>isteğe\<bağlı parametreleri farklı olduğundan ' Method1\<> ', ' Method2 > ' öğesini geçersiz kılamaz
İsteğe bağlı parametrelerinin değerlerinden farklı olan bir yöntemi geçersiz kılmayı denediniz, yani imzaları farklı. Bir tür, aynı ada ve imzaya sahip bir yöntemi bildirerek ve açıklamayı `Overrides` değiştiriciyle işaretleyerek devralınan geçersiz kılınabilir bir yöntemi geçersiz kılabilir.  
  
 **Hata KIMLIĞI:** BC30308  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- İki yöntemin de aynı imzaya sahip olduğundan emin olun.
