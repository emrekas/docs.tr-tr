# <a name="goto-ltlabelnamegt-is-not-valid-because-ltlabelnamegt-is-inside-a-synclock-statement-that-does-not-contain-this-statement"></a>' GoTo &lt;labelname&gt;' geçerli değil çünkü '&lt;labelname&gt;', bu deyimi içermeyen bir 'SyncLock' deyiminin içinde yer
Dal oluşturamazsınız bir `SyncLock` blok.  
  
 **Hata Kimliği:** BC30755  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Etiket önündeki böylece kodunuzu yeniden yapılandırmayı `SyncLock` blok.
