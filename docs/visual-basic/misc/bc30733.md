# <a name="membername-is-already-declared-by-procedurename-which-was-generated-for-this-containername"></a>'\<membername >' tarafından bildirilen '\<procedurename >', oluşturulduğu için bu \<containername >

Ortak dil çalışma zamanı tarafından bildirilen bir üye ile aynı ada sahip bir üye bildirildi.

**Hata Kimliği:** BC30733

## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için

- Çakışma kaldırmak için üyenin adını değiştirin.
