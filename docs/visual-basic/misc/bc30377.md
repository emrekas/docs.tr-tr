# <a name="method-methodname-is-already-declared-in-interface-interfacename"></a>Yöntem '\<yöntemAdı >' arabiriminde zaten bildirilmiş '\<InterfaceName >'
Bu çoğaltma bir arabirim yönteminin ikinci bir kez bildirilen.  
  
 **Hata Kimliği:** BC30377  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Yinelenen bildirimini kaldırın.
