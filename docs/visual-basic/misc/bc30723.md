# <a name="method-call-did-not-return-a-value"></a>Yöntem çağrısı bir değer döndürmedi.
Çağrılan yöntemi bir değer, beklenen şekilde döndürmedi.  
  
 **Hata Kimliği:** BC30723  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Çağrılan yöntemi bir değer döndüren bir işlev olduğundan emin olun.
