# <a name="sub-new-cannot-be-declared-overrides"></a>'Sub New' 'Overrides' bildirilemez
Bir oluşturucu, devralınan bir oluşturucunun geçersiz kılmaları gösterir. Oluşturucular geçersiz kılınamaz.  
  
 **Hata Kimliği:** BC30283  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Kaldırma `Overrides` from anahtar sözcüğü `Sub` bildirimi.
