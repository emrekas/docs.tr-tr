# <a name="an-embedded-expression-may-not-be-used-here"></a>Gömülü deyim burada kullanılamaz
Katıştırılmış bir ifade desteklenmeyen bir konumda bir XML değişmez değer eklenmiştir. Katıştırılmış ifadeler bir XML ad alanı değeri belirtmek için kullanılamayacağını unutmayın.  
  
 **Hata Kimliği:** BC31172  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Katıştırılmış bir ifade desteklenmeyen konumundan kaldırın.
