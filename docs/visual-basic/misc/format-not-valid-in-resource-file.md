# <a name="format-not-valid-in-resource-file"></a>Kaynak dosya biçimi geçerli değil
Projenizdeki bir kaynak dosyası bir Windows kaynak dosyası için standart biçim izlemez veya kaynak dosyasını ve Visual Basic projesi arasında bir uyumsuzluk var.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Kaynak kaynağı ve kaynak dosyalarını yeniden oluşturun ve Windows yazılım geliştirme Seti'nde belgelenen söz dizimi izliyor.  
  
2.  Geçersiz kaynak dosyasını projeden kaldırın.
