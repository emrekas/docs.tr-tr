# <a name="local-variable-variablename-cannot-be-referred-to-before-it-is-declared"></a>Yerel değişken '\<variablename >', bildirilmeden önce çıkarsanamaz
Bir yordam içinde bir değişken, bildiriminden önce kullanıldı.  
  
 **Hata Kimliği:** BC32000  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Değişken için ilk başvuruyu önceleyen bildirimi taşıyın.
