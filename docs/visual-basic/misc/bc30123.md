# <a name="end-statements-are-not-valid-in-the-immediate-window"></a>'End' deyimleri yürütme penceresinde geçerli değildir
`Stop` Ve `End` deyimleri yürütmeyi askıya almak ve bir hata ayıklama bağlamında izin verilmez.  
  
 **Hata Kimliği:** BC30123  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Sorun değil bir `End` veya `Stop` deyiminde **hemen** penceresi.
