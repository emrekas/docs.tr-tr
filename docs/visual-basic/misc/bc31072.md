# <a name="warning-treated-as-error-error"></a>Uyarı hata olarak kabul edilir: \<hata >
Çalışan bir derleme ile derlendiğinde bir uyarı ile karşılaşıldı **uyarıları hata olarak değerlendir** seçeneği açık.  
  
 **Hata Kimliği:** BC31072  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Uyarı düzeltin.  
  
2.  İle derleme **uyarıları hata olarak değerlendir** seçeneği devre dışı bırakılmış.
