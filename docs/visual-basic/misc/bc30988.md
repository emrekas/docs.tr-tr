# <a name="type-or-with-expected"></a><span data-ttu-id="dbd2b-102">Tür veya ' with ' bekleniyor</span><span class="sxs-lookup"><span data-stu-id="dbd2b-102">Type or 'With' expected</span></span>
<span data-ttu-id="dbd2b-103">Bir sınıfın örneğini bildirdiğinizde, `New` anahtar kelimesinin ardından bir tür adı veya tarafından `With`gelmelidir.</span><span class="sxs-lookup"><span data-stu-id="dbd2b-103">When you declare an instance of a class, the `New` keyword must be followed by a type name or by `With`.</span></span> <span data-ttu-id="dbd2b-104">Örneğin, aşağıdaki deyimler `client` `Customer` sınıfının bir örneği olarak bildirir.</span><span class="sxs-lookup"><span data-stu-id="dbd2b-104">For example, the following statements each declare `client` to be an instance of the `Customer` class.</span></span> <span data-ttu-id="dbd2b-105">Tür adı `Customer` aşağıdaki gibidir `New`.</span><span class="sxs-lookup"><span data-stu-id="dbd2b-105">The type name `Customer` follows `New`.</span></span>  
  
```  
' Dim client As New Customer()  
' The next declaration uses an object initializer.  
Dim client As New Customer() With {.Name = "Litware, Inc."}  
```  
  
 <span data-ttu-id="dbd2b-106">Visual Basic 2008 ' den başlayarak, bir nesneyi anonim bir türün örneği olacak şekilde bildirebilirsiniz, bu durumda bir veri türü belirtmezsiniz.</span><span class="sxs-lookup"><span data-stu-id="dbd2b-106">Beginning with Visual Basic 2008, you can declare an object to be an instance of an anonymous type, in which case you do not specify a data type.</span></span> <span data-ttu-id="dbd2b-107">Anonim tür bildirimlerinde anahtar sözcüğü `With` aşağıdaki gibidir. `New`</span><span class="sxs-lookup"><span data-stu-id="dbd2b-107">In anonymous type declarations, the keyword `With` follows `New`.</span></span>  
  
