# <a name="type-or-with-expected"></a><span data-ttu-id="064f5-102">Yazın veya 'With' bekleniyor</span><span class="sxs-lookup"><span data-stu-id="064f5-102">Type or 'With' expected</span></span>
<span data-ttu-id="064f5-103">Bir sınıf örneği bildirdiğinizde `New` anahtar sözcüğü, bir tür adı tarafından ya da gelmelidir `With`.</span><span class="sxs-lookup"><span data-stu-id="064f5-103">When you declare an instance of a class, the `New` keyword must be followed by a type name or by `With`.</span></span> <span data-ttu-id="064f5-104">Örneğin, aşağıdaki deyimleri her bildirmek `client` örneğini olmasını `Customer` sınıfı.</span><span class="sxs-lookup"><span data-stu-id="064f5-104">For example, the following statements each declare `client` to be an instance of the `Customer` class.</span></span> <span data-ttu-id="064f5-105">Tür adı `Customer` izleyen `New`.</span><span class="sxs-lookup"><span data-stu-id="064f5-105">The type name `Customer` follows `New`.</span></span>  
  
```  
' Dim client As New Customer()  
' The next declaration uses an object initializer.  
Dim client As New Customer() With {.Name = "Litware, Inc."}  
```  
  
 <span data-ttu-id="064f5-106">Visual Basic 2008'den itibaren bu durumda bir veri türü belirtmeyin bir nesne, anonim bir türün bir örneği olarak bildirebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="064f5-106">Beginning with Visual Basic 2008, you can declare an object to be an instance of an anonymous type, in which case you do not specify a data type.</span></span> <span data-ttu-id="064f5-107">Anonim türde bildirimlerden, anahtar sözcüğü `With` izleyen `New`.</span><span class="sxs-lookup"><span data-stu-id="064f5-107">In anonymous type declarations, the keyword `With` follows `New`.</span></span>  
  
```  
Dim person = New With {.Name ="Mike Nash", .Age = 27}  
```
