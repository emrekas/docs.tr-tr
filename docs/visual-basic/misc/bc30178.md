# <a name="specifier-is-duplicated"></a>Tanımlayıcısı yineleniyor
Bir anahtar sözcüğü, aynı bildirimde birden çok kez görünür.  
  
 **Hata Kimliği:** BC30178  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yinelenen anahtar sözcüğü görünümlerini bildirimden kaldırın.
