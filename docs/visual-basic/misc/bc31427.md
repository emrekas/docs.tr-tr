# <a name="syntax-error-in-conditional-compilation-expression"></a>Koşullu derleme ifadesinde sözdizimi hatası
Koşullu derleme ifadesi özel olmayan bir sözdizimi hatası oluşturuyor.  
  
 **Hata KIMLIĞI:** BC31427  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Koşullu derleme ve ifadede kullanılan anahtar sözcükler için belgeleri inceleyin. Belgelerdeki örnek kodu, bu hatayı oluşturan kaynak satıra göre karşılaştırın.  
  
2. Hatanın nedenini belirleyemezseniz, bu koşullara ilişkin bilgileri toplayın ve Microsoft Ürün Destek Hizmetleri 'ne bildirin.
