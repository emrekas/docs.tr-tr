# <a name="syntax-error-in-conditional-compilation-expression"></a>Koşullu derleme ifadede sözdizimi hatası
Koşullu derleme ifade belirli olmayan bir sözdizimi hatası oluşturur.  
  
 **Hata Kimliği:** BC31427  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  İfadede kullanılan anahtar sözcükleri ve koşullu derleme için belgeleri inceleyin. Örnek kod, bu hata oluşturmadan kaynak satırı karşı belgeleri karşılaştırın.  
  
2.  Hatanın nedenini belirleyemez koşullar hakkında bilgi toplamak ve Microsoft Ürün Destek Hizmetleri bildirir.
