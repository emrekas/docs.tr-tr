# <a name="unable-to-get-type-information-for-typename"></a>Tür bilgileri alınamıyor. '\<typename >'
Bir tür, ortak dil çalışma zamanı tarafından yüklenmemiş başvuruldu.  
  
 **Hata Kimliği:** BC30751  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Türü geçerli kapsam içinde tanımlanan şekilde kodunuzu yeniden yapılandırabilirsiniz.
