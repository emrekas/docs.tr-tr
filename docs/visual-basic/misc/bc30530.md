# <a name="parameter-cannot-have-the-same-name-as-its-defining-function"></a>Parametre, onu tanımlayan işlevle aynı ada sahip olamaz
Bir parametre hem de onu tanımlayan işlevle aynı ada kullandınız.  
  
 **Hata Kimliği:** BC30530  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- İşlevin içindeki parametre adları kontrol edin ve çakışan parametrenin adını değiştirin.
