# <a name="currency-is-no-longer-a-supported-type-use-the-decimal-type-instead"></a>'Para birimi' artık desteklenen bir türdür; Bunun yerine 'Decimal' türünü kullanın
`Currency` Veri türü tarafından değiştirilmiştir `Decimal` veri türü.  
  
 **Hata Kimliği:** BC30803  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kullanım `Decimal` veri türü yerine `Currency`.
