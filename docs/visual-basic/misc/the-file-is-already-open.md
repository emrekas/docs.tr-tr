# <a name="the-file-is-already-open"></a>Dosya zaten açılmış
Zaten açık olan bir dosyayı açma girişiminde bulunuldu.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Açmak denemeden önce dosyayı kapatın.  
  
     -veya-  
  
- Yedekli kaldırmak `FileOpen` kodunuzdan deyimi.
