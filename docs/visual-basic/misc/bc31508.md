# <a name="cannot-find-net-framework-directory-error"></a>.NET Framework dizin bulunamıyor: \<hata >
Dizin burada [!INCLUDE[dnprdnshort](~/includes/dnprdnshort-md.md)] depolanmış olan derlemeleri bulunamadı.  
  
 **Hata Kimliği:** BC31508  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Yinelenen hata durumunda yeniden görmek için programı derleyin.  
  
2.  Hata devam ederse, çalışmanızı kaydedin ve Visual Studio'yu yeniden başlatın.  
  
3.  Hata almaya devam ederseniz, Visual Basic yeniden yükleyin.  
  
4.  Yeniden yüklenmesinden sonra hata oluşmaya devam ederse Microsoft Ürün Destek Hizmetleri bildirin.
