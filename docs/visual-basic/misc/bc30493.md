# <a name="constructor-must-be-declared-as-a-sub-not-as-a-function"></a>Oluşturucu bir işlev olarak değil bir Sub'olarak bildirilmelidir
Bildirme girişiminde bulundunuz bir `Function New`. Oluşturucular olarak bildirilmesi gerekir `Sub New`.  
  
 **Hata Kimliği:** BC30493  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Kullanım `Sub` yerine `Function`.
