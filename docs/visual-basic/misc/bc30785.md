# <a name="parameter-specifier-is-duplicated"></a>Parametre tanımlayıcısı yineleniyor
Bir parametre tanımlayıcısı birden çok kez parametresi uygulanır.  
  
 **Hata Kimliği:** BC30785  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Yinelenen tanımlayıcıları kaldırın.
