# <a name="cannot-implement-interface-interfacename1-because-its-implementation-could-conflict-with-the-implementation-of-another-implemented-interface-interfacename2-for-some-type-arguments"></a><span data-ttu-id="bb2e4-102">Arabirim uygulayamaz '\<interfacename1 >' ile başka bir uygulanan arabirimi uygulaması uygulanması çakışma hatası nedeniyle '\<interfacename2 >' için bazı tür bağımsız değişkenleri</span><span class="sxs-lookup"><span data-stu-id="bb2e4-102">Cannot implement interface '\<interfacename1>' because its implementation could conflict with the implementation of another implemented interface '\<interfacename2>' for some type arguments</span></span>
<span data-ttu-id="bb2e4-103">Bir sınıf bildirimi içeren bir `Implements` iki veya daha fazla arabirimi, ancak en az bir arabirimlerin genel ve iki uygulamalarının olduğunu belirten bir ifade çakışan belirli tür bağımsız değişkenleri değerleri.</span><span class="sxs-lookup"><span data-stu-id="bb2e4-103">A class declaration includes an `Implements` statement that specifies two or more interfaces, but at least one of the interfaces is generic and two of the implementations could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="bb2e4-104">Aşağıdaki deyimleri, bu hata oluşturabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="bb2e4-104">The following statements can generate this error.</span></span>  
  
```  
Public Interface iFace1  
    Sub testSub(ByVal arg As String)  
End Interface  
Public Interface iFace2(Of t)  
    Sub testSub(ByVal arg As t)  
End Interface  
Public Class testClass  
    Implements iFace1, iFace2(Of String)  
End Class  
```  
  
 <span data-ttu-id="bb2e4-105">Çünkü `iFace2` kullanılarak oluşturulan `String`, `testClass` iki sürümünü de uygulamalıdır `testSub` aynı imzaya sahip.</span><span class="sxs-lookup"><span data-stu-id="bb2e4-105">Because `iFace2` is constructed using `String`, `testClass` must implement two versions of `testSub` with identical signatures.</span></span> <span data-ttu-id="bb2e4-106">Bunun yapılması erişmek için hangi sürümü hakkında bir belirsizlik oluşturur.</span><span class="sxs-lookup"><span data-stu-id="bb2e4-106">Doing so would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="bb2e4-107">**Hata Kimliği:** BC32072</span><span class="sxs-lookup"><span data-stu-id="bb2e4-107">**Error ID:** BC32072</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="bb2e4-108">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="bb2e4-108">To correct this error</span></span>  
  
- <span data-ttu-id="bb2e4-109">Çakışma olmasını yönelik genel arabirimi sağlanan tür bağımsız değişkeni olarak değiştirin.</span><span class="sxs-lookup"><span data-stu-id="bb2e4-109">Change the type argument supplied to the generic interface so that there is no conflict.</span></span>  
  
     <span data-ttu-id="bb2e4-110">-veya-</span><span class="sxs-lookup"><span data-stu-id="bb2e4-110">-or-</span></span>  
  
- <span data-ttu-id="bb2e4-111">Kaldırma `Implements` uygulama çakışma ile sonuçlanan arabirimlerin deyimi.</span><span class="sxs-lookup"><span data-stu-id="bb2e4-111">Remove from the `Implements` statement one of the interfaces resulting in the implementation conflict.</span></span>
