<span data-ttu-id="478be-103">Bir sınıf bildirimi iki veya `Implements` daha fazla arabirimi belirten bir ifade içerir, ancak en az bir arabirimler geneldir ve uygulamalardan ikisi tür bağımsız değişkenlerinin belirli değerleri için çakışabilir.</span><span class="sxs-lookup"><span data-stu-id="478be-103">A class declaration includes an `Implements` statement that specifies two or more interfaces, but at least one of the interfaces is generic and two of the implementations could conflict for certain values of type arguments.</span></span>

<span data-ttu-id="478be-104">Aşağıdaki deyimler bu hatayı oluşturabilir.</span><span class="sxs-lookup"><span data-stu-id="478be-104">The following statements can generate this error.</span></span>

```vb
Public Interface iFace1
    Sub testSub(ByVal arg As String)
End Interface
Public Interface iFace2(Of t)
    Sub testSub(ByVal arg As t)
End Interface
Public Class testClass
    Implements iFace1, iFace2(Of String)
End Class
```

<span data-ttu-id="478be-105">Kullanılarak `iFace2` `testSub` oluşturulduğu için ,`testClass` aynı imzalara sahip iki sürümünün uygulanması gerekir. `String`</span><span class="sxs-lookup"><span data-stu-id="478be-105">Because `iFace2` is constructed using `String`, `testClass` must implement two versions of `testSub` with identical signatures.</span></span> <span data-ttu-id="478be-106">Bunun yapılması, hangi sürümün erişecağına bir belirsizlik üretir.</span><span class="sxs-lookup"><span data-stu-id="478be-106">Doing so would produce an ambiguity about which version to access.</span></span>

<span data-ttu-id="478be-107">**Hata KIMLIĞI:** BC32072</span><span class="sxs-lookup"><span data-stu-id="478be-107">**Error ID:** BC32072</span></span>
