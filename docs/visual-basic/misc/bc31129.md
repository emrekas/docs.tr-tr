# <a name="raiseevent-is-already-declared"></a>'RaiseEvent' zaten bildirilmiş
Birden fazla `RaiseEvent` bildirimi bir özel olay bildiriminde görüntülenmektedir. A `RaiseEvent` bildirim olayı oluşturmak için kullanılan bir yordamı bildirir.  
  
 **Hata Kimliği:** BC31129  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yedekli kaldırmak `RaiseEvent` deyimi.
