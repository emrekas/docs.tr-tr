# <a name="type-argument-lttypeargumentnamegt-is-declared-mustinherit-and-does-not-satisfy-the-new-constraint-for-type-parameter-lttypeparameternamegt"></a><span data-ttu-id="89034-102">Tür bağımsız değişkeni '&lt;typeargumentname&gt;'' MustInherit' olarak bildirildi ve tür parametresi için 'New' kısıtlamasını karşılamaz'&lt;typeparametername&gt;'</span><span class="sxs-lookup"><span data-stu-id="89034-102">Type argument '&lt;typeargumentname&gt;' is declared 'MustInherit' and does not satisfy the 'New' constraint for type parameter '&lt;typeparametername&gt;'</span></span>
<span data-ttu-id="89034-103">Genel bir tür ile çağrılan bir `MustInherit` ancak karşılık gelen tür parametresi ile bildirilen bir tür bağımsız değişkeni, sınıfta `New` kısıtlaması.</span><span class="sxs-lookup"><span data-stu-id="89034-103">A generic type is invoked with a `MustInherit` class as a type argument, while the corresponding type parameter is declared with the `New` constraint.</span></span>  
  
 <span data-ttu-id="89034-104">`New` Kısıtlaması gerekli karşılık gelen tür bağımsız değişkeni geçirilen türü nesnelerinin oluşturulması desteklemesi gerekir.</span><span class="sxs-lookup"><span data-stu-id="89034-104">The `New` constraint requires that the type passed in the corresponding type argument must support the creation of objects.</span></span> <span data-ttu-id="89034-105">Ancak, bir *soyut* diğer bir deyişle, bir sınıf olarak bildirilen sınıf `MustInherit`, herhangi bir nesne oluşturulamıyor çünkü hiçbir oluşturucu kullanıma sunmuyor.</span><span class="sxs-lookup"><span data-stu-id="89034-105">However, an *abstract* class, that is, a class declared as `MustInherit`, does not expose any constructors because you cannot create any objects from it.</span></span>  
  
 <span data-ttu-id="89034-106">**Hata Kimliği:** BC32082</span><span class="sxs-lookup"><span data-stu-id="89034-106">**Error ID:** BC32082</span></span>

1.  <span data-ttu-id="89034-108">Tür bağımsız değişkeni içinde kullanılan sınıfı soyut ve ardından kaldırmak gereksinimi yoksa `MustInherit` bildiriminden from anahtar sözcüğü.</span><span class="sxs-lookup"><span data-stu-id="89034-108">If the class used in the type argument does not need to be abstract, then remove the `MustInherit` keyword from its declaration.</span></span>

3.  <span data-ttu-id="89034-110">Geçirilen türünden herhangi bir nesne oluşturun ve ardından kaldırmak ilgili tür parametresindeki gereksinimi yoksa `New` bildiriminden kısıtlamasından.</span><span class="sxs-lookup"><span data-stu-id="89034-110">If the corresponding type parameter does not need to create any objects from the type passed to it, then remove the `New` constraint from its declaration.</span></span>
