# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-latebinding"></a>.NET Compact Framework'ün hedeflenen sürümünü geç bağlamayı desteklemiyor
.NET Compact Framework'ı çalıştığınız sürümü geç bağlamayı desteklemiyor.  
  
 **Hata Kimliği:** BC30762  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. İşlevler, abonelerimizin veya özelliklerine nesnesi olarak bildirilen bir değişken çağırma kaçının.  
  
2. Nesne değişkeni bir dizi kullanmaktan kaçının.  
  
3. Sözlük üye erişimi ifadeleri ile nesne değişkenleri kullanmaktan kaçının.
