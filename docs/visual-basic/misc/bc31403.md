# <a name="imports-alias-ltnamegt-conflicts-with-ltname2gt-declared-in-the-root-namespace"></a>Imports diğer&lt;adı&gt;'ile çakışıyor'&lt;name2&gt;' kök ad alanı bildirimi
Kullanılan diğer ad `Imports` ifadesi, başka bir adı kök ad alanında bildirilen çakışıyor.  
  
 **Hata Kimliği:** BC31403  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Tarafından kullanılan diğer adı için benzersiz bir ad seçin `Imports` deyimi.
