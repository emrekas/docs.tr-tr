# <a name="ltmodifiergt-is-not-valid-on-a-declare"></a>'&lt;değiştiricisi&gt;' Declare üzerinde geçerli değil
A `Declare` deyimi içeren bir geçersiz anahtar sözcüğü gibi `ReadOnly`.  
  
 **Hata Kimliği:** BC30244  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Geçersiz anahtar sözcüğü Kaldır `Declare` deyimi.
