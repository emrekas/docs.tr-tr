# <a name="expression-expected"></a>İfade bekleniyor
Operatör gibi `+` veya `=` , sağ işlenen için bir ifade gerektirir ancak hiçbir tanınabilir ifadesi görünür.  
  
 **Hata Kimliği:** BC30201  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- İşlecinin sağ işlenen için geçerli bir ifade girin.
