<span data-ttu-id="e8c4d-103">Öznitelik bağımsız değişkeni için kullanılan bir ifade, karşılık gelen öznitelik parametresinden farklı bir veri türü değerlendirir ve Visual Basic öznitelik bağımsız değişkenleri için gerekli tür dönüştürmeye izin vermez.</span><span class="sxs-lookup"><span data-stu-id="e8c4d-103">An expression used for an attribute argument evaluates to a data type different from that of the corresponding attribute parameter, and Visual Basic does not allow the required type conversion for attribute arguments.</span></span>  
  
 <span data-ttu-id="e8c4d-104">Bir öznitelik, uygulandığı öğe için meta veriler sağlar ve derleyicinin derleme zamanında tüm meta verileri oluşturabiliyor olması gerekir.</span><span class="sxs-lookup"><span data-stu-id="e8c4d-104">An attribute provides metadata for the element it is applied to, and the compiler must be able to construct all the metadata at compile time.</span></span> <span data-ttu-id="e8c4d-105">Bu nedenle, her özniteliğin derleme zamanında sabit değerler kullanması gerekir ve bu nedenle her öznitelik bağımsız değişkeni bir derleme zamanı sabit değeri olarak değerlendirilmelidir.</span><span class="sxs-lookup"><span data-stu-id="e8c4d-105">For this reason, every attribute must use values that are constant at compile time, and therefore every attribute argument must evaluate to a compile-time constant value.</span></span>  
  
 <span data-ttu-id="e8c4d-106">Belirli tür dönüştürmeleri derleme zamanında sabit değer üretemiyor.</span><span class="sxs-lookup"><span data-stu-id="e8c4d-106">Certain type conversions cannot produce values that are constant at compile time.</span></span> <span data-ttu-id="e8c4d-107">Örneğin, bir `String` `Double` veya bir `Date` ' a dönüştürmek, çalışma zamanında yerel ayara göre değişir.</span><span class="sxs-lookup"><span data-stu-id="e8c4d-107">For example, converting a `String` to a `Double` or a `Date` depends on the locale setting at run time.</span></span> <span data-ttu-id="e8c4d-108">Türetilmiş bir tür `Object`dizisi gibi diğer dönüşümler, derleyiciye öznitelik bağımsız değişkenlerinde izin vermelerine izin vermeyen çeşitli sorunlar sunar.</span><span class="sxs-lookup"><span data-stu-id="e8c4d-108">Other conversions, such as an array of a derived type to an array of `Object`, present a variety of problems that do not permit the compiler to allow them on attribute arguments.</span></span>
