# <a name="this-system-does-not-contain-support-for-the-japanese-locale"></a>Bu sistem, Japonca yerel ayar için destek içermiyor.
Kullanmaya çalıştığınız `VbStrConv.Japanese` numaralandırma üyesi, ancak sisteminizi Japonca yerel ayarını desteklemez.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Japonca yerel ayarında desteklemek için makine yükseltebilir veya uygulamayı yerel olarak destekleyen bir makinede çalıştırın.
