# <a name="declaration1-cannot-override-declaration2-because-it-is-declared-notoverridable"></a>' NotOverridable ' olarak bildirildiği için '\<declaration1>','declaration2>'öğesinigeçersizkılamaz\<
Yordam veya özellik bildirimi aynı ada sahip devralınmış bir öğeyi geçersiz kılmaya çalışır, ancak devralınmış öğe olarak `NotOverridable`belirtilir.  
  
 **Hata KIMLIĞI:** BC30267  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Devralınan öğenin bildiriminden anahtar sözcüğünü kaldırın veya geçersiz kılma bildirimini kaldırın. `NotOverridable`
