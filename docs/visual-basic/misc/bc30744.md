# <a name="referenced-array-element-has-a-value-of-nothing"></a>Başvurulan bir dizi öğesinin değerine sahip 'Nothing'
Başvurulan dizi öğesi ayarlanmamış ya da daha önce ayarlandığı `Nothing`.  
  
 **Hata Kimliği:** BC30744  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Dizideki değerleri olarak doğru şekilde doğrulayın.
