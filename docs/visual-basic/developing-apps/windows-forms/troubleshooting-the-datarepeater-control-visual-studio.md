## <a name="datarepeater-keyboard-and-mouse-events-are-not-raised"></a><span data-ttu-id="b5be5-104">DataRepeater klavye ve fare olayları harekete Geçirilmemesi</span><span class="sxs-lookup"><span data-stu-id="b5be5-104">DataRepeater Keyboard and Mouse Events Are Not Raised</span></span>

 <span data-ttu-id="b5be5-110">Özel durum olduğunda <xref:System.Windows.Forms.Control.Padding%2A> özelliği ayarlandığında büyük bir kenarlarına kullanıma sunmak için yeterli değer <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> denetimi.</span><span class="sxs-lookup"><span data-stu-id="b5be5-110">The exception to this is when the <xref:System.Windows.Forms.Control.Padding%2A> property is set to a large enough value to expose the edges of the <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> control.</span></span> <span data-ttu-id="b5be5-111">Bu durumda, kullanıma sunulan kenar boşluğunu tıklayarak fare olayları ortaya koyar.</span><span class="sxs-lookup"><span data-stu-id="b5be5-111">In this case, clicking in the exposed margin will raise mouse events.</span></span>
