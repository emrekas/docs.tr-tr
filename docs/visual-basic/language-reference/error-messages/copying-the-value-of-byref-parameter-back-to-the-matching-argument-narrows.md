 <span data-ttu-id="f4894-109">Varsa bir `ByRef` bağımsız değişken değeri yordama kopyalanır ve bağımsız değişken ile parametre aynı tür dönüştürme gerekli değildir.</span><span class="sxs-lookup"><span data-stu-id="f4894-109">If a `ByRef` argument value is copied into the procedure and the argument and parameter are of the same type, no conversion is necessary.</span></span> <span data-ttu-id="f4894-110">Ancak, Visual Basic türleri farklı ise, her iki yönde dönüştürmeniz gerekir.</span><span class="sxs-lookup"><span data-stu-id="f4894-110">But if the types are different, Visual Basic must convert in both directions.</span></span> <span data-ttu-id="f4894-111">Türlerinden biri, sınıf veya yapı türü ise, Visual Basic, için ve diğer türden dönüştürmeniz gerekir.</span><span class="sxs-lookup"><span data-stu-id="f4894-111">If one of the types is your class or structure type, Visual Basic must convert it both to and from the other type.</span></span> <span data-ttu-id="f4894-112">Bu dönüştürmeler birini genişletme, ters dönüştürme daraltma.</span><span class="sxs-lookup"><span data-stu-id="f4894-112">If one of these conversions is widening, the reverse conversion might be narrowing.</span></span>
