# <a name="cannot-create-activex-component"></a>ActiveX Bileşeni oluşturulamıyor
Tasarım zamanında bir formunda bir ActiveX denetimi yerleştirin veya, bir ActiveX denetimi ile bir projeye form ekleme çalışıldı, ancak ilgili bilgilerin kayıt defterinde bulunamadı.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kayıt defteri bilgileri bozuk veya silinmiş. ActiveX denetimi yeniden yükleyin veya denetim satıcısına başvurun.
