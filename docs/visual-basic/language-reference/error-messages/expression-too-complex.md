# <a name="expression-too-complex"></a>İfade çok karmaşık
Bir kayan nokta ifadesi, çok fazla iç içe geçmiş alt ifadeler içeriyor.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   İfade çok ayrı ifadelere hata oluşmasını önlemek için gerektiği şekilde bölün.
