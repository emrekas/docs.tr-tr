# <a name="expression-does-not-produce-a-value"></a>İfade bir değer üretmiyor
Arama gibi bir değer üreten bağlamında bir değer üretmiyor ifade kullanmayı denediniz bir `Sub` bir bağlamda burada bir `Function` beklenir.  
  
 **Hata Kimliği:** BC30491  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- İfade bir değer üreten bir değiştirin.
