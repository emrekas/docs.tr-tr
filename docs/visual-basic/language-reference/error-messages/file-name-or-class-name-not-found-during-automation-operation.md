# <a name="file-name-or-class-name-not-found-during-automation-operation-visual-basic"></a>Otomasyon işlemi sırasında dosya adı veya sınıf adı bulunamadı (Visual Basic)
Dosya adı veya bir sınıf için belirtilen ad `GetObject` işlevi bulunamadı.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Adlarını denetleyin ve yeniden deneyin. İçin kullanılan adı olduğundan emin olun `class` parametre sistemi ile kayıtlı eşleşiyor.
