# <a name="this-key-is-already-associated-with-an-element-of-this-collection"></a>Bu anahtar bu koleksiyonun bir öğesi ile zaten ilişkili
Koleksiyon üyesi zaten başka bir koleksiyonun üyesi tanımlar için bir anahtar belirtilmiş. Bir anahtar belirtilen bir dizedir `Add` belirli bir koleksiyonun üyesi benzersiz olarak tanımlayan yöntemi.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bu üye için farklı bir anahtar kullanın.
