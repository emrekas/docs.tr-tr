# <a name="file-not-found-visual-basic-run-time-error"></a>Dosya bulunamadı (Visual Basic Çalışma Süresi Hatası)
Burada belirtilen dosya bulunamadı. Hata aşağıdaki olası nedenleri vardır:  
  
- Bir deyim var olmayan bir dosyasına başvuruyor.  
  
- Bir dinamik bağlantı kitaplığı (DLL) bir yordam çağırma çalışıldı, ancak belirtilen kitaplık `Lib` yan tümcesi `Declare` deyimi bulunamıyor.  
  
- Bir proje açın ya da var olmayan bir metin dosyası yükleme çalışıldı.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Dosya adını ve yolunu yazımını denetleyin.
