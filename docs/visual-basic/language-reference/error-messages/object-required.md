# <a name="object-required-visual-basic"></a>Gerekli nesne (Visual Basic)
Özellikleri ve yöntemleri başvuruları genellikle açık nesne niteleyicisi gerektirir. Böyle bir durumda budur.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Nesne özelliği veya yöntemi başvuruları geçerli nesne niteleyicisi olup olmadığını denetleyin. Bir nesne niteleyicisi bir sağlamadı belirtin.  
  
2.  Nesne niteleyicisinin yazım denetimi ve nesnenin bir parçası, başvurduğunuzdan program görünür olduğundan emin olun.  
  
3.  Bir ana uygulama için bir yol sağlanırsa **Dosya Aç** komutu, değişkenlerinde doğru olup olmadığını denetleyin.  
  
4.  Nesnenin belgeleri denetleyin ve eylemin geçerli olduğundan emin olun.
