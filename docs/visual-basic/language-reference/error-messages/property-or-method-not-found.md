# <a name="property-or-method-not-found"></a>Özellik veya yöntem bulunamadı
Başvurulan nesne yöntemi veya nesne özelliği tanımlı değil.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Nesne adı yanlış. Hangi özelliklere ve yöntemlere bir nesne için tanımlanmış görmek için Nesne Tarayıcısı'nı görüntüleyin. Kullanılabilir özellikler ve yöntemler bir listesini görüntülemek için uygun nesne kitaplığı seçin.
