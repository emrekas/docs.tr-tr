# <a name="device-io-error"></a>Cihaz G/Ç hatası
Programınızı yazıcı veya disk sürücüsünü gibi bir cihazı kullanırken bir giriş veya çıkış hata oluştu.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Cihazın düzgün çalıştığından emin olun ve sonra işlemi yeniden deneyin.
