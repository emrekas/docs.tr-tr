# <a name="subscript-out-of-range-visual-basic"></a>Alt simge aralık dışı (Visual Basic)
İzin verilen aralığın dışında kaldığından, bir dizi alt simgesi geçerli değil. Her zaman bir boyut için alt simge en düşük değer olan 0 ve en yüksek alt simge değer tarafından döndürülür `GetUpperBound` o boyut için yöntemi.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Alt simge, bu geçerli aralığa denk gelen şekilde değiştirin.
