# <a name="some-subkeys-cannot-be-deleted"></a>Bazı alt anahtarlar silinemiyor
Bir kayıt defteri anahtarını silme girişiminde bulunuldu, ancak bazı alt anahtarlar silinemiyor çünkü işlem başarısız oldu. Yetersiz izinler nedeniyle genellikle budur.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Belirtilen alt anahtarlarını silmek için yeterli izinlere sahip olduğunuzdan emin olun.
