# <a name="constant-expression-not-representable-in-type-39lttypenamegt39"></a>Sabit ifade türünde gösterilemez &#39; &lt;typename&gt;&#39;
Hedef türü uymayan bir sabit değerlendirilecek çalıştığınız aralığı genellikle taşma olduğundan.  
  
 **Hata Kimliği:** BC30439  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Hedef türü sabit işleyebilen bir değiştirin.
