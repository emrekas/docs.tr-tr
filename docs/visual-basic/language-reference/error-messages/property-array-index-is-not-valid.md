# <a name="property-array-index-is-not-valid"></a>Özellik dizisi dizini geçerli değil
Sağlanan değer için bir özellik dizisi dizini geçerli değil.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Dizininizi belirtilen özellik için geçerli aralıkta olduğundan emin olmak için bileşenin belgelerine bakın.
