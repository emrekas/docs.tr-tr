# <a name="class-does-not-support-automation-or-does-not-support-expected-interface"></a>Sınıf Otomasyonu veya beklenen arabirimi desteklemiyor
İçinde belirtilen sınıf `GetObject` veya `CreateObject` işlev çağrısı bir programlama arabirimi ifşa veya bir proje için .exe, .dll dosyasından değiştirilmiş ya da tam tersi.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Bu nesnenin sınıfını otomasyonla kullanımı ile ilgili sınırlamalar için nesneyi oluşturan uygulamanın belgelerine bakın.  
  
2.  Bir proje için .exe veya .dll dosyasından değiştirdiyseniz, eski .dll veya .exe el ile kaydı gerekir.
