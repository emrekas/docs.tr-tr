# <a name="path-not-found"></a>Yol bulunamadı
Bir dosya erişim veya disk erişimi işlemi sırasında işletim sistemi belirtilen yolu bulamadı. Bir dosya yolu, sürücü belirtimi yanı sıra dizinleri ve dosyayı bulmak için geçmesi gereken alt dizinleri içerir. Göreli veya mutlak bir yol olabilir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Yol belirtmek ve doğrulayın.
