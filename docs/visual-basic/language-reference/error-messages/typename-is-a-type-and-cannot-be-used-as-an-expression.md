# <a name="39lttypenamegt39-is-a-type-and-cannot-be-used-as-an-expression"></a>&#39;&lt;TypeName&gt; &#39; türüdür ve ifade olarak kullanılamaz
Bir tür adı bir ifade gerekli olduğu gerçekleşir. Bir ifade değişkenleri, sabitleri, sabit değerleri, özellikleri, bileşiminden oluşmalı ve `Function` yordam çağrıları.  
  
 **Hata Kimliği:** BC30108  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Tür adı'nı kaldırın ve geçerli öğeleri kullanılarak ifade oluşturun.
