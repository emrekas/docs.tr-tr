# <a name="need-property-array-index"></a>Need özelliği dizi dizini
Bu özellik değeri, tek bir değer yerine dizi oluşur. Erişmeye çalıştığınız özellik dizisi dizini belirtilmedi.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Aralık için dizinleri dizisi için uygun bir bulmak için bileşenin belgelerine bakın. Özellik erişim raporunuza uygun bir dizin belirtin.
