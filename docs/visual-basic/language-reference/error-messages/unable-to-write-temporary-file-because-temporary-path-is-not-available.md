# <a name="unable-to-write-temporary-file-because-temporary-path-is-not-available"></a>Geçici yol kullanılabilir olmadığından geçici dosya yazılamıyor
Visual Basic, geçici dosyaların depolandığı yolu belirleyemedi.  
  
 **Hata Kimliği:** BC30698  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Visual Studio'yu yeniden başlatın.  
  
2. Sorun devam ederse, Visual Studio'yu yeniden yükleyin.
