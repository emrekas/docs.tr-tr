# <a name="attributes-visual-basic"></a>Öznitelikler (Visual Basic)
Visual Basic nesneleri yönetilmeyen kod ile birlikte çalışmak sağlayan çeşitli öznitelikler ve modül üyeleri modül adı erişilmesini sağlayan bir öznitelik sağlar. Aşağıdaki tabloda, Visual Basic tarafından kullanılan öznitelikleri listeler.  
  
|||  
|---|---|  
|<xref:Microsoft.VisualBasic.ComClassAttribute>|Bir sınıf bir COM nesnesi olarak gösterilmesine izin veren meta verileri eklemek için derleyicinin sağlar.|  
|<xref:Microsoft.VisualBasic.HideModuleNameAttribute>|Yalnızca modül için gerekli nitelik kullanılarak erişilecektir modülü üyeleri sağlar.|  
|<xref:Microsoft.VisualBasic.VBFixedArrayAttribute>|Bir dizi bir yapının veya yerel olmayan değişken bir sabit uzunluklu dizi olarak değerlendirilmesi gerektiğini gösterir.|  
|<xref:Microsoft.VisualBasic.VBFixedStringAttribute>|Uzunluğu düzeltilen gibi bir dize değerlendirilmesi gerektiğini gösterir.|
