# <a name="byref-visual-basic"></a>ByRef (Visual Basic)
Bağımsız değişken çağrılan yordam çağıran koddaki bağımsız değişken arka plandaki bir değişkenin değerini değiştirebilirsiniz şekilde geçirildiğini belirtir.  
  
## <a name="remarks"></a>Açıklamalar  
 `ByRef` Bu bağlamda değiştirici kullanılabilir:  
  
 [Declare Deyimi](../../../visual-basic/language-reference/statements/declare-statement.md)  
  
 [Function Deyimi](../../../visual-basic/language-reference/statements/function-statement.md)  
  
 [Sub Deyimi](../../../visual-basic/language-reference/statements/sub-statement.md)
