# <a name="directories-and-files-summary-visual-basic"></a>Dizinler ve Dosyalar Özeti (Visual Basic)
Visual Basic dili anahtar sözcükleri ve çalışma zamanı kitaplığı üyeleri amaç tarafından düzenlenir ve kullanın.  
  
 Kullanarak `My` özelliği size daha fazla verimlilik ve performans bu üyeleri kullanmaktan dosya g/ç işlemleri. Daha fazla bilgi için bkz. <xref:Microsoft.VisualBasic.FileIO.FileSystem>.  
  
|**Eylem**|**Dil öğesi**|  
|----------------|--------------------------|  
|Bir dizin veya klasör değiştirin.|<xref:Microsoft.VisualBasic.FileSystem.ChDir%2A>|  
|Sürücüyü değiştirin.|<xref:Microsoft.VisualBasic.FileSystem.ChDrive%2A>|  
|Bir dosyaya kopyalayın.|<xref:Microsoft.VisualBasic.FileSystem.FileCopy%2A>|  
|Bir dizin veya klasör olun.|<xref:Microsoft.VisualBasic.FileSystem.MkDir%2A>|  
|Bir dizin veya klasör kaldırın.|<xref:Microsoft.VisualBasic.FileSystem.RmDir%2A>|  
|Dosya, dizin veya klasör yeniden adlandırın.|<xref:Microsoft.VisualBasic.FileSystem.Rename%2A>|  
|Geçerli bir yol döndürür.|<xref:Microsoft.VisualBasic.FileSystem.CurDir%2A>|  
|Bir dosyanın tarih/saat damgasının döndürür.|<xref:Microsoft.VisualBasic.FileSystem.FileDateTime%2A>|
