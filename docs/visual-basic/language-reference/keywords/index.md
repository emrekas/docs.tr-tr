Aşağıdaki tablolar, tüm Visual Basic dili anahtar sözcükleri listeler.  
  
## <a name="reserved-keywords"></a>Ayrılmış anahtar sözcükler  
 Aşağıdaki anahtar sözcükler *ayrılmış*, yani bunları programlama değişkenleri ya da yordamlar gibi öğeler için ad olarak kullanamazsınız. Bu kısıtlama adı köşeli ayraç içinde kapsayan tarafından atlayabilirsiniz (`[]`). Daha fazla bilgi için bkz: "Kaçan adlar" [bildirilen öğe adları](../../../visual-basic/programming-guide/language-features/declared-elements/declared-element-names.md).  
  
> [!NOTE]
>  Kaçış adları, kodunuzun okunmasını zorlaştırabilir ve bulmak zor olabilir Zarif hatalara yol açabilir çünkü kullanmanız önerilmez.
