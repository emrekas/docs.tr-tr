# <a name="registry-summary-visual-basic"></a>Kayıt Defteri Özeti (Visual Basic)
Visual Studio dil anahtar sözcükleri ve çalışma zamanı kitaplığı üyeleri amaç tarafından düzenlenir ve kullanın.  
  
 Kullanarak `My` özelliği daha fazla verimlilik ve performans bu öğeleri kayıt işlemlerinde size sağlar. Daha fazla bilgi için bkz. <xref:Microsoft.VisualBasic.MyServices.RegistryProxy>.  
  
|**Eylem**|**Dil öğesi**|  
|----------------|--------------------------|  
|Program ayarlarını silin.|<xref:Microsoft.VisualBasic.Interaction.DeleteSetting%2A>|  
|Program ayarlarını oku.|<xref:Microsoft.VisualBasic.Interaction.GetSetting%2A>, <xref:Microsoft.VisualBasic.Interaction.GetAllSettings%2A>|  
|Program ayarlarını kaydetme.|<xref:Microsoft.VisualBasic.Interaction.SaveSetting%2A>|
