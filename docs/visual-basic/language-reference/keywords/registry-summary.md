# <a name="registry-summary-visual-basic"></a>Kayıt Defteri Özeti (Visual Basic)

 Kullanarak `My` özelliği daha fazla verimlilik ve performans bu öğeleri kayıt işlemlerinde size sağlar. Daha fazla bilgi için bkz. <xref:Microsoft.VisualBasic.MyServices.RegistryProxy>.
