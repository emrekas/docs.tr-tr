# <a name="myapplicationinfo-object"></a>My.Application.Info Nesnesi
Uygulama sürüm numarası, açıklama, yüklenen derlemeler ve benzeri gibi ilgili bilgileri almak için özellikleri sağlar.  
  
## <a name="remarks"></a>Açıklamalar  
 Yöntemleri ve özellikleri hakkında bilgi için `My.Application.Info` nesne, bkz: <xref:Microsoft.VisualBasic.ApplicationServices.AssemblyInfo>.  
  
> [!NOTE]
>  Özelliklerini kullanabilirsiniz <xref:System.Diagnostics.FileVersionInfo?displayProperty=nameWithType> diskte dosya hakkında bilgi edinmek için sınıf.  
  
## <a name="requirements"></a>Gereksinimler  
 **Namespace:** <xref:Microsoft.VisualBasic.ApplicationServices>  
  
 **Sınıf:** <xref:Microsoft.VisualBasic.ApplicationServices.AssemblyInfo>  
  
 **Derleme:** Visual Basic Çalışma Zamanı Kitaplığı (Microsoft.VisualBasic.dll içinde)
