# <a name="mycomputerkeyboard-object"></a>My.Computer.Keyboard Nesnesi

 Yöntemleri ve özellikleri hakkında bilgi için `My.Computer.Keyword` nesne, bkz: <xref:Microsoft.VisualBasic.Devices.Keyboard>.
