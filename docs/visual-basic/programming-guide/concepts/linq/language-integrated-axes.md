# <a name="language-integrated-axes-in-visual-basic-linq-to-xml"></a>Dille tümleşik eksenler (LINQ to XML) Visual Basic'te
Bu bölümde, XML erişmek kolay hale getirmek için doğrudan Visual Basic diline yerleşik özellikler açıklanmaktadır. XML belgeleri için LINQ örneklerin çoğu bu tümleşik bir Visual Basic eksenler kullanın.  
  
## <a name="in-this-section"></a>Bu Bölümde  
  
|Konu|Açıklama|  
|-----------|-----------------|  
|[XML Alt Axis Özelliği](../../../../visual-basic/language-reference/xml-axis/xml-child-axis-property.md)|Aşağıdakilerden birinin alt öğelere erişim sağlar: bir <xref:System.Xml.Linq.XElement> nesnesi bir <xref:System.Xml.Linq.XDocument> nesnesi, koleksiyonu <xref:System.Xml.Linq.XElement> nesneleri ya da bir koleksiyonu <xref:System.Xml.Linq.XDocument> nesneleri. Bu eksen eşdeğerdir <xref:System.Xml.Linq.XContainer.Elements%2A> ekseni.|  
|[XML Descendant Axis Özelliği](../../../../visual-basic/language-reference/xml-axis/xml-descendant-axis-property.md)|Aşağıdaki alt öğeleri için erişim sağlar: bir <xref:System.Xml.Linq.XElement> nesnesi bir <xref:System.Xml.Linq.XDocument> nesne ya da bir koleksiyonu <xref:System.Xml.Linq.XElement> nesneleri. Bu eksen eşdeğerdir <xref:System.Xml.Linq.XContainer.Descendants%2A> ekseni.|  
|[XML Özniteliği Axis Özelliği](../../../../visual-basic/language-reference/xml-axis/xml-attribute-axis-property.md)|Bir özniteliğin erişim sağlayan bir <xref:System.Xml.Linq.XElement> nesne. Bu eksen için kabaca <xref:System.Xml.Linq.XElement.Attribute%2A> ekseni. Bu eksen farklıdır <xref:System.Xml.Linq.XElement.Attribute%2A> , eksen döndürür özniteliğinin değeri olmayan bir <xref:System.Xml.Linq.XAttribute> nesne.|  
|[Extension Indexer Özelliği](../../../../visual-basic/language-reference/xml-axis/extension-indexer-property.md)|Bir koleksiyondaki öğelere erişim sağlar.|
