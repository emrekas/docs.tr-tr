# <a name="differences-between-passing-an-argument-by-value-and-by-reference-visual-basic"></a><span data-ttu-id="e16c4-102">Değere ve Başvuruya Göre Bağımsız Değişken Geçirme Arasındaki Farklar (Visual Basic)</span><span class="sxs-lookup"><span data-stu-id="e16c4-102">Differences Between Passing an Argument By Value and By Reference (Visual Basic)</span></span>
<span data-ttu-id="e16c4-103">Bir yordam için bir veya daha fazla değişken geçtiğinizde, her bağımsız değişken çağıran koddaki temel programlama öğesine karşılık gelir.</span><span class="sxs-lookup"><span data-stu-id="e16c4-103">When you pass one or more arguments to a procedure, each argument corresponds to an underlying programming element in the calling code.</span></span> <span data-ttu-id="e16c4-104">Bu temel alınan öğenin değeri veya buna bir başvuru geçirebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="e16c4-104">You can pass either the value of this underlying element, or a reference to it.</span></span> <span data-ttu-id="e16c4-105">Bu olarak bilinir *mekanizması geçirme*.</span><span class="sxs-lookup"><span data-stu-id="e16c4-105">This is known as the *passing mechanism*.</span></span>  
  
## <a name="passing-by-value"></a><span data-ttu-id="e16c4-106">Değere göre geçirme</span><span class="sxs-lookup"><span data-stu-id="e16c4-106">Passing by Value</span></span>  
 <span data-ttu-id="e16c4-107">Bağımsız değişken geçirmeniz *değeriyle* belirterek [ByVal](../../../../visual-basic/language-reference/modifiers/byval.md) anahtar sözcüğü için yordam tanımında karşılık gelen parametre.</span><span class="sxs-lookup"><span data-stu-id="e16c4-107">You pass an argument *by value* by specifying the [ByVal](../../../../visual-basic/language-reference/modifiers/byval.md) keyword for the corresponding parameter in the procedure definition.</span></span> <span data-ttu-id="e16c4-108">Visual Basic programlama temel alınan öğenin değeri bu geçirme mekanizması kullandığınızda, yordamda yerel bir değişken içine kopyalar.</span><span class="sxs-lookup"><span data-stu-id="e16c4-108">When you use this passing mechanism, Visual Basic copies the value of the underlying programming element into a local variable in the procedure.</span></span> <span data-ttu-id="e16c4-109">Yordam kodu çağıran kod temel öğeye herhangi bir erişimi yok.</span><span class="sxs-lookup"><span data-stu-id="e16c4-109">The procedure code does not have any access to the underlying element in the calling code.</span></span>  
  
## <a name="passing-by-reference"></a><span data-ttu-id="e16c4-110">Başvuruya göre geçirme</span><span class="sxs-lookup"><span data-stu-id="e16c4-110">Passing by Reference</span></span>  
 <span data-ttu-id="e16c4-111">Bağımsız değişken geçirmeniz *başvuruya göre* belirterek [ByRef](../../../../visual-basic/language-reference/modifiers/byref.md) anahtar sözcüğü için yordam tanımında karşılık gelen parametre.</span><span class="sxs-lookup"><span data-stu-id="e16c4-111">You pass an argument *by reference* by specifying the [ByRef](../../../../visual-basic/language-reference/modifiers/byref.md) keyword for the corresponding parameter in the procedure definition.</span></span> <span data-ttu-id="e16c4-112">Bu geçirme mekanizması kullandığınızda, Visual Basic yordamı bir doğrudan başvuru temel programlama öğesine çağıran kodu sağlar.</span><span class="sxs-lookup"><span data-stu-id="e16c4-112">When you use this passing mechanism, Visual Basic gives the procedure a direct reference to the underlying programming element in the calling code.</span></span>  
  
## <a name="passing-mechanism-and-element-type"></a><span data-ttu-id="e16c4-113">Geçirme mekanizması ve öğe türü</span><span class="sxs-lookup"><span data-stu-id="e16c4-113">Passing Mechanism and Element Type</span></span>  
 <span data-ttu-id="e16c4-114">Seçim mekanizması geçirme sınıflandırmasını temel öğe türü ile aynı değil.</span><span class="sxs-lookup"><span data-stu-id="e16c4-114">The choice of passing mechanism is not the same as the classification of the underlying element type.</span></span> <span data-ttu-id="e16c4-115">Değere veya başvuruya göre geçirme ne Visual Basic için yordamının kodunu sağladığı için ifade eder.</span><span class="sxs-lookup"><span data-stu-id="e16c4-115">Passing by value or by reference refers to what Visual Basic supplies to the procedure code.</span></span> <span data-ttu-id="e16c4-116">Bir değer türü veya başvuru türü bir programlama öğesi bellekte depolanma için ifade eder.</span><span class="sxs-lookup"><span data-stu-id="e16c4-116">A value type or reference type refers to how a programming element is stored in memory.</span></span>  
  
 <span data-ttu-id="e16c4-117">Ancak, öğe türü ve geçirme mekanizması birbirleriyle ilişkilidir.</span><span class="sxs-lookup"><span data-stu-id="e16c4-117">However, the passing mechanism and element type are interrelated.</span></span> <span data-ttu-id="e16c4-118">Bir başvuru türü başka bir yerde bellekteki verileri için bir işaretçi değeridir.</span><span class="sxs-lookup"><span data-stu-id="e16c4-118">The value of a reference type is a pointer to the data elsewhere in memory.</span></span> <span data-ttu-id="e16c4-119">Temel alınan öğenin kendisinin erişemiyor olsa bile bir başvuru türü değerle geçirdiğinizde, yordamının kodunu temel öğesinin veri işaretçisi olduğunu gösterir.</span><span class="sxs-lookup"><span data-stu-id="e16c4-119">This means that when you pass a reference type by value, the procedure code has a pointer to the underlying element's data, even though it cannot access the underlying element itself.</span></span> <span data-ttu-id="e16c4-120">Örneğin, bir dizi değişkenini öğesiyse yordamının kodunu değişkene erişimi yok, ancak dizi üyeleri erişebilir.</span><span class="sxs-lookup"><span data-stu-id="e16c4-120">For example, if the element is an array variable, the procedure code does not have access to the variable itself, but it can access the array members.</span></span>

|<span data-ttu-id="e16c4-127">Değer türü (yalnızca bir değer içeriyor)</span><span class="sxs-lookup"><span data-stu-id="e16c4-127">Value type (contains only a value)</span></span>|<span data-ttu-id="e16c4-128">Yordamı, değişken veya diğer üyelerinin değiştiremezsiniz.</span><span class="sxs-lookup"><span data-stu-id="e16c4-128">The procedure cannot change the variable or any of its members.</span></span>|<span data-ttu-id="e16c4-129">Yordamı, değişken ve üyelerini değiştirebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="e16c4-129">The procedure can change the variable and its members.</span></span>|
