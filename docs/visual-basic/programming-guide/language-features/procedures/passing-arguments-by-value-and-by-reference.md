## <a name="choice-of-passing-mechanism"></a><span data-ttu-id="94616-113">Seçim mekanizması geçirme</span><span class="sxs-lookup"><span data-stu-id="94616-113">Choice of Passing Mechanism</span></span>

- <span data-ttu-id="94616-138">Yordamı çağıran koddaki temel alınan öğeyi değiştirmek için orijinal gereksinimi varsa, karşılık gelen parametre bildirmek [ByRef](../../../../visual-basic/language-reference/modifiers/byref.md).</span><span class="sxs-lookup"><span data-stu-id="94616-138">If the procedure has a genuine need to change the underlying element in the calling code, declare the corresponding parameter [ByRef](../../../../visual-basic/language-reference/modifiers/byref.md).</span></span>  
  
- <span data-ttu-id="94616-139">Temel alınan öğeyi çağıran koddaki yordamı doğru kod yürütmeyi bağlıdır, parametre bildirmek `ByRef`.</span><span class="sxs-lookup"><span data-stu-id="94616-139">If the correct execution of the code depends on the procedure changing the underlying element in the calling code, declare the parameter `ByRef`.</span></span> <span data-ttu-id="94616-140">Değere göre geçirirseniz veya çağırma kodunun kılıyorsa `ByRef` mekanizması bağımsız değişken parantez içine alarak geçirme yordam çağrısı beklenmeyen sonuçlara neden.</span><span class="sxs-lookup"><span data-stu-id="94616-140">If you pass it by value, or if the calling code overrides the `ByRef` passing mechanism by enclosing the argument in parentheses, the procedure call might produce unexpected results.</span></span>
