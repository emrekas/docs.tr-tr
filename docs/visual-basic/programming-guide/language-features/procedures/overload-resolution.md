# <a name="overload-resolution-visual-basic"></a><span data-ttu-id="48167-102">Aşırı Yükleme Çözümü (Visual Basic Başvurusu)</span><span class="sxs-lookup"><span data-stu-id="48167-102">Overload Resolution (Visual Basic)</span></span>
<span data-ttu-id="48167-103">Visual Basic Derleyicisi aşırı yüklenmiş sürümlerinde tanımlanan bir yordam çağrısı karşılaştığında derleyici çağırmak için aşırı yüklemeleri, karar vermeniz gerekir.</span><span class="sxs-lookup"><span data-stu-id="48167-103">When the Visual Basic compiler encounters a call to a procedure that is defined in several overloaded versions, the compiler must decide which of the overloads to call.</span></span> <span data-ttu-id="48167-104">Bunu aşağıdaki adımları uygulayarak yapar:</span><span class="sxs-lookup"><span data-stu-id="48167-104">It does this by performing the following steps:</span></span>  
  
1. <span data-ttu-id="48167-105">**Erişilebilirlik.**</span><span class="sxs-lookup"><span data-stu-id="48167-105">**Accessibility.**</span></span> <span data-ttu-id="48167-106">Bu, çağrıyı yapan kod bunu çağırma engelleyen erişim düzeyine sahip hiçbir aşırı yüklemeyle ortadan kaldırır.</span><span class="sxs-lookup"><span data-stu-id="48167-106">It eliminates any overload with an access level that prevents the calling code from calling it.</span></span>  
  
2. <span data-ttu-id="48167-107">**Parametre sayısı.**</span><span class="sxs-lookup"><span data-stu-id="48167-107">**Number of Parameters.**</span></span> <span data-ttu-id="48167-108">Bu, çağrıyı sağlanan daha farklı bir dizi parametre tanımlar hiçbir aşırı yüklemeyle ortadan kaldırır.</span><span class="sxs-lookup"><span data-stu-id="48167-108">It eliminates any overload that defines a different number of parameters than are supplied in the call.</span></span>  
  
3. <span data-ttu-id="48167-109">**Parametre veri türleri.**</span><span class="sxs-lookup"><span data-stu-id="48167-109">**Parameter Data Types.**</span></span> <span data-ttu-id="48167-110">Derleyici, örnek yöntemler tercih üzerinde genişletme yöntemleri sağlar.</span><span class="sxs-lookup"><span data-stu-id="48167-110">The compiler gives instance methods preference over extension methods.</span></span> <span data-ttu-id="48167-111">Herhangi bir örnek yöntemi yalnızca yordam çağrısı eşleştirilecek dönüştürmeleri genişletme gerektiren bulunursa, tüm uzantı yöntemleri bırakılır ve derleyici yalnızca örnek yöntemi adayları ile devam eder.</span><span class="sxs-lookup"><span data-stu-id="48167-111">If any instance method is found that requires only widening conversions to match the procedure call, all extension methods are dropped and the compiler continues with only the instance method candidates.</span></span> <span data-ttu-id="48167-112">Bu tür bir örnek yöntem bulunursa, örneği ve genişletme yöntemleri ile devam eder.</span><span class="sxs-lookup"><span data-stu-id="48167-112">If no such instance method is found, it continues with both instance and extension methods.</span></span>  
  
     <span data-ttu-id="48167-113">Bu adımda, çağırma bağımsız değişkenlerinin veri türleri için aşırı yüklemesi'içinde tanımlanan parametre türleri olarak değiştirilemez hiçbir aşırı yüklemeyle ortadan kaldırır.</span><span class="sxs-lookup"><span data-stu-id="48167-113">In this step, it eliminates any overload for which the data types of the calling arguments cannot be converted to the parameter types defined in the overload.</span></span>  
  
4. <span data-ttu-id="48167-114">**Daraltma dönüştürmeleri.**</span><span class="sxs-lookup"><span data-stu-id="48167-114">**Narrowing Conversions.**</span></span> <span data-ttu-id="48167-115">Bu çağırma bağımsız değişken türleri için tanımlanan parametre türleri bir daraltma dönüşümü gerektirir hiçbir aşırı yüklemeyle ortadan kaldırır.</span><span class="sxs-lookup"><span data-stu-id="48167-115">It eliminates any overload that requires a narrowing conversion from the calling argument types to the defined parameter types.</span></span> <span data-ttu-id="48167-116">Bu tür denetimi olup geçiş geçerlidir ([Option Strict deyimi](../../../../visual-basic/language-reference/statements/option-strict-statement.md)) olan `On` veya `Off`.</span><span class="sxs-lookup"><span data-stu-id="48167-116">This is true whether the type checking switch ([Option Strict Statement](../../../../visual-basic/language-reference/statements/option-strict-statement.md)) is `On` or `Off`.</span></span>  
  
5. <span data-ttu-id="48167-117">**En az genişletme.**</span><span class="sxs-lookup"><span data-stu-id="48167-117">**Least Widening.**</span></span> <span data-ttu-id="48167-118">Derleyici çiftler halinde kalan aşırı göz önünde bulundurur.</span><span class="sxs-lookup"><span data-stu-id="48167-118">The compiler considers the remaining overloads in pairs.</span></span> <span data-ttu-id="48167-119">Her bir çifti için tanımlanan parametrelerin veri türlerini karşılaştırır.</span><span class="sxs-lookup"><span data-stu-id="48167-119">For each pair, it compares the data types of the defined parameters.</span></span> <span data-ttu-id="48167-120">Tüm aşırı yüklemeler türlerinde diğer karşılık gelen türlerine genişletmek, derleyici ikinci ortadan kaldırır.</span><span class="sxs-lookup"><span data-stu-id="48167-120">If the types in one of the overloads all widen to the corresponding types in the other, the compiler eliminates the latter.</span></span> <span data-ttu-id="48167-121">Diğer bir deyişle, en az genişletme miktarını gerektiren aşırı korur.</span><span class="sxs-lookup"><span data-stu-id="48167-121">That is, it retains the overload that requires the least amount of widening.</span></span>  
  
6. <span data-ttu-id="48167-122">**Tek aday.**</span><span class="sxs-lookup"><span data-stu-id="48167-122">**Single Candidate.**</span></span> <span data-ttu-id="48167-123">Aşırı yüklemeler tek kadar çiftler halinde aşırı kalır ve aşırı yükleyen çağrısı çözümler considering devam eder.</span><span class="sxs-lookup"><span data-stu-id="48167-123">It continues considering overloads in pairs until only one overload remains, and it resolves the call to that overload.</span></span> <span data-ttu-id="48167-124">Derleyici, tek bir aday için aşırı azaltılamaz, bir hata oluşturur.</span><span class="sxs-lookup"><span data-stu-id="48167-124">If the compiler cannot reduce the overloads to a single candidate, it generates an error.</span></span>
