# <a name="xml-literals-and-the-xml-10-specification-visual-basic"></a>XML Değişmez Değerleri ve XML 1.0 Belirtimi (Visual Basic)
Visual Basic'te XML değişmez sözdiziminin Genişletilebilir Biçimlendirme Dili (XML) 1.0 belirtimi çoğunu destekler. XML 1.0 belirtimi hakkında daha fazla ayrıntı için bkz: [Genişletilebilir Biçimlendirme Dili (XML) 1.0](https://www.w3.org/TR/xml) W3C Web sitesinde.  
  
## <a name="what-visual-basic-does-not-support"></a>Hangi Visual Basic desteklemez  
  
-   XML değişmez bir belge türü tanımı (DTD'nin) içeremez.  
  
-   Bir XML belgesi değişmez değeri bir XML belgesi bildirimi ile başlamalıdır.  
  
-   XML değişmez değer tek satırda 65.535'den fazla karakter içeremez.  
  
-   XML ad alanı öneklerini, öğe adları ve öznitelik adları çok 1024 karakter içeremez.  
  
## <a name="extra-features-that-visual-basic-supports"></a>Visual Basic destekleyen ek özellikler  
  
-   Belge ve öğe değişmez değerlerine izin gömülü ifade söz dizimi geçerli XML değil.
