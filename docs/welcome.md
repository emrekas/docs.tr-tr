Bkz: [.NET Core ile çalışmaya başlama](core/get-started.md) .NET Core uygulamaları oluşturma hakkında bilgi edinmek için.

Uygulamalarınızı, Android, iOS, Linux, macOS ve Windows üzerinde çalıştırabilirsiniz.
