# <a name="staticresource-markup-extension"></a><span data-ttu-id="d55fe-102">StaticResource Biçimlendirme Uzantısı</span><span class="sxs-lookup"><span data-stu-id="d55fe-102">StaticResource Markup Extension</span></span>
<span data-ttu-id="d55fe-103">İçin herhangi bir değer sağlar [!INCLUDE[TLA2#tla_xaml](../../../../includes/tla2sharptla-xaml-md.md)] önceden tanımlanan bir kaynağa başvuru bakarak özelliği özniteliği.</span><span class="sxs-lookup"><span data-stu-id="d55fe-103">Provides a value for any [!INCLUDE[TLA2#tla_xaml](../../../../includes/tla2sharptla-xaml-md.md)] property attribute by looking up a reference to an already defined resource.</span></span> <span data-ttu-id="d55fe-104">Bu kaynak için arama davranışı önceden geçerli yüklenen kaynakları arar yükleme zamanı arama, benzer [!INCLUDE[TLA2#tla_xaml](../../../../includes/tla2sharptla-xaml-md.md)] sayfasında yanı sıra diğer uygulama kaynakları ve kaynak değeri olarak oluşturur çalışma zamanı nesneleri özellik değeri.</span><span class="sxs-lookup"><span data-stu-id="d55fe-104">Lookup behavior for that resource is analogous to load-time lookup, which will look for resources that were previously loaded from the markup of the current [!INCLUDE[TLA2#tla_xaml](../../../../includes/tla2sharptla-xaml-md.md)] page as well as other application sources, and will generate that resource value as the property value in the run-time objects.</span></span>
