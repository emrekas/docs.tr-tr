## <a name="software-rendering-pipeline"></a><span data-ttu-id="6ddf9-110">Yazılımla işlenen ardışık düzen</span><span class="sxs-lookup"><span data-stu-id="6ddf9-110">Software Rendering Pipeline</span></span>

### <a name="graphics-rendering-tiers"></a><span data-ttu-id="6ddf9-118">Grafik İşleme Katmanları</span><span class="sxs-lookup"><span data-stu-id="6ddf9-118">Graphics Rendering Tiers</span></span>

 <span data-ttu-id="6ddf9-121">Bunu başarmak için [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] zamanında sistemin grafik kapasitesini belirlemek için işlevsellik sağlar.</span><span class="sxs-lookup"><span data-stu-id="6ddf9-121">To achieve this, [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] provides functionality to determine the graphics capability of a system at runtime.</span></span> <span data-ttu-id="6ddf9-122">Grafik yeteneği, üç işleme yeteneği katmanları biri olarak ekran kartı gruplayarak belirlenir.</span><span class="sxs-lookup"><span data-stu-id="6ddf9-122">Graphics capability is determined by categorizing the video card as one of three rendering capability tiers.</span></span> [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] <span data-ttu-id="6ddf9-123">kullanıma sunan bir [!INCLUDE[TLA#tla_api](../../../../includes/tlasharptla-api-md.md)] işleme yeteneği katmanını sorgulamak bir uygulama sağlar.</span><span class="sxs-lookup"><span data-stu-id="6ddf9-123">exposes an [!INCLUDE[TLA#tla_api](../../../../includes/tlasharptla-api-md.md)] that allows an application to query the rendering capability tier.</span></span> <span data-ttu-id="6ddf9-124">Uygulamanızı daha sonra donanım tarafından desteklenen işleme katmanına bağlı olarak çalışma zamanında farklı kod yollarını da alabilir.</span><span class="sxs-lookup"><span data-stu-id="6ddf9-124">Your application can then take different code paths at run time depending on the rendering tier supported by the hardware.</span></span>  
  
 <span data-ttu-id="6ddf9-125">En çok katmanı işleme düzeyleri etkileyen grafik donanımının özellikleri şunlardır:</span><span class="sxs-lookup"><span data-stu-id="6ddf9-125">The features of the graphics hardware that most impact the rendering tier levels are:</span></span>  
  
- <span data-ttu-id="6ddf9-126">**Video RAM** birleştirme grafik için kullanılan arabellek sayısı ve boyutu grafik donanımının video bellek miktarı belirler.</span><span class="sxs-lookup"><span data-stu-id="6ddf9-126">**Video RAM** The amount of video memory on the graphics hardware determines the size and number of buffers that can be used for compositing graphics.</span></span>  
  
- <span data-ttu-id="6ddf9-127">**Piksel gölgelendirici** piksel gölgelendirici efektleri piksel başına temelinde hesaplar işlevi bir grafik olduğundan.</span><span class="sxs-lookup"><span data-stu-id="6ddf9-127">**Pixel Shader** A pixel shader is a graphics processing function that calculates effects on a per-pixel basis.</span></span> <span data-ttu-id="6ddf9-128">Görüntülenen grafiklerin çözümleme bağlı olarak, her görüntü çerçevesi için işlenmesi gereken birkaç milyon piksel olabilir.</span><span class="sxs-lookup"><span data-stu-id="6ddf9-128">Depending on the resolution of the displayed graphics, there could be several million pixels that need to be processed for each display frame.</span></span>  
  
- <span data-ttu-id="6ddf9-129">**Köşe gölgelendirici** köşe gölgelendiricisi olan nesnenin köşe verileri matematiksel işlemler gerçekleştiren işlevi bir grafik.</span><span class="sxs-lookup"><span data-stu-id="6ddf9-129">**Vertex Shader** A vertex shader is a graphics processing function that performs mathematical operations on the vertex data of the object.</span></span>  
  
- <span data-ttu-id="6ddf9-130">**Çoklu doku desteği** bir 3B grafik nesnede karıştırma işlemi sırasında iki veya daha fazla farklı dokular uygulama özelliği başvurduğu çoklu doku desteği.</span><span class="sxs-lookup"><span data-stu-id="6ddf9-130">**Multitexture Support** Multitexture support refers to the ability to apply two or more distinct textures during a blending operation on a 3D graphics object.</span></span> <span data-ttu-id="6ddf9-131">Çoklu doku desteği derecesini grafik donanımının çoklu doku birimleri sayısına göre belirlenir.</span><span class="sxs-lookup"><span data-stu-id="6ddf9-131">The degree of multitexture support is determined by the number of multitexture units on the graphics hardware.</span></span>  
  
 <span data-ttu-id="6ddf9-132">Piksel gölgelendirici, köşe gölgelendiricisi ve çoklu doku özellikleri belirli tanımlamak için kullanılan [!INCLUDE[TLA2#tla_dx](../../../../includes/tla2sharptla-dx-md.md)] , buna karşılık, farklı işleme katmanları tanımlamak için kullanılan sürüm düzeylerini [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)].</span><span class="sxs-lookup"><span data-stu-id="6ddf9-132">The pixel shader, vertex shader, and multitexture features are used to define specific [!INCLUDE[TLA2#tla_dx](../../../../includes/tla2sharptla-dx-md.md)] version levels, which, in turn, are used to define the different rendering tiers in [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)].</span></span>

- <span data-ttu-id="6ddf9-135">**Katman 0 işleme** donanım grafik ivmesi.</span><span class="sxs-lookup"><span data-stu-id="6ddf9-135">**Rendering Tier 0** No graphics hardware acceleration.</span></span> <span data-ttu-id="6ddf9-136">[!INCLUDE[TLA2#tla_dx](../../../../includes/tla2sharptla-dx-md.md)] Sürüm düzeyi sürüm 7.0 azdır.</span><span class="sxs-lookup"><span data-stu-id="6ddf9-136">The [!INCLUDE[TLA2#tla_dx](../../../../includes/tla2sharptla-dx-md.md)] version level is less than version 7.0.</span></span>  
  
- <span data-ttu-id="6ddf9-137">**Katman 1 işleme** kısmi grafik donanım hızlandırma.</span><span class="sxs-lookup"><span data-stu-id="6ddf9-137">**Rendering Tier 1** Partial graphics hardware acceleration.</span></span> <span data-ttu-id="6ddf9-138">[!INCLUDE[TLA2#tla_dx](../../../../includes/tla2sharptla-dx-md.md)] Büyüktür veya eşittir sürüm 7.0 sürüm düzeyi ve **daha az** sürüm 9.0 daha.</span><span class="sxs-lookup"><span data-stu-id="6ddf9-138">The [!INCLUDE[TLA2#tla_dx](../../../../includes/tla2sharptla-dx-md.md)] version level is greater than or equal to version 7.0, and **lesser** than version 9.0.</span></span>  
  
- <span data-ttu-id="6ddf9-139">**Katman 2 işleme** grafik donanım hızlandırmayı çoğu grafik özellikleri kullanın.</span><span class="sxs-lookup"><span data-stu-id="6ddf9-139">**Rendering Tier 2** Most graphics features use graphics hardware acceleration.</span></span> <span data-ttu-id="6ddf9-140">[!INCLUDE[TLA2#tla_dx](../../../../includes/tla2sharptla-dx-md.md)] Sürüm düzeyi 9.0 sürümüne eşit veya daha büyük.</span><span class="sxs-lookup"><span data-stu-id="6ddf9-140">The [!INCLUDE[TLA2#tla_dx](../../../../includes/tla2sharptla-dx-md.md)] version level is greater than or equal to version 9.0.</span></span>
