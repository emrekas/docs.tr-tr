 <span data-ttu-id="505d4-171">[(Yerel Görüntü Oluşturucu](../../../docs/framework/tools/ngen-exe-native-image-generator.md) (NGEN) bütünleştirilmiş kodları yerel koda derleyen ve yerel bilgisayarda yerel görüntü önbelleğine yükler.</span><span class="sxs-lookup"><span data-stu-id="505d4-171">The [(Native Image Generator](../../../docs/framework/tools/ngen-exe-native-image-generator.md) (NGEN) compiles assemblies to native code and installs them in the native image cache on the local computer.</span></span> <span data-ttu-id="505d4-172">NGEN, .NET Native gibi yerel kod üretir olsa da, ancak bunu .NET Native bazı önemli fark vardır:</span><span class="sxs-lookup"><span data-stu-id="505d4-172">However, although NGEN, like .NET Native, produces native code, it differs from .NET Native in some significant ways:</span></span>
