# <a name="2578---trycatchexceptionfromcatchorfinally"></a>2578 - TryCatchExceptionFromCatchOrFinally
## <a name="properties"></a>Özellikler  
  
|||  
|-|-|  
|Kimlik|2578|  
|anahtar sözcükler|WFActivities|  
|Düzey|Uyarı|  
|Kanal|Microsoft Windows uygulaması sunucu-uygulamalar/hata ayıklama|  
  
## <a name="description"></a>Açıklama  
 Gösteren bir Catch veya Finally etkinlik bir özel durum oluşturdu.  
  
## <a name="message"></a>İleti  
 Catch veya Finally TryCatch etkinlik '%1' ile ilişkili olan etkinliği bir özel durum oluşturdu.
