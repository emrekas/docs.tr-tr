# <a name="runtime-activities-in-wf"></a>Çalışma zamanı WF etkinlikleri
[!INCLUDE[netfx_current_short](../../../includes/netfx-current-short-md.md)] kalıcılığı ve sonlandırma gibi iş akışı çalışma zamanı özellikleri erişmek için çeşitli sistem tarafından sağlanan etkinlikler sağlar.  
  
|Etkinlik|Açıklama|  
|--------------|-----------------|  
|<xref:System.Activities.Statements.Persist>|İş akışı durumuna kalıcı açıkça ister.|  
|<xref:System.Activities.Statements.TerminateWorkflow>|Çalışan iş akışı örneği sonlandırır.|  
|<xref:System.Activities.Statements.NoPersistScope>|Çocuk etkinliklerinin kalıcı engelleyen bir kapsayıcı etkinliği.|
