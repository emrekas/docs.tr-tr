# <a name="1143---flowchartnextnull"></a>1143 - FlowchartNextNull
## <a name="properties"></a>Özellikler  
  
|||  
|-|-|  
|Kimlik|1143|  
|anahtar sözcükler|WFActivities|  
|Düzey|Bilgiler|  
|Kanal|Microsoft Windows uygulaması sunucu-uygulamalar/hata ayıklama|  
  
## <a name="description"></a>Açıklama  
 Sonraki düğümü bir akış grafiğindeki null olduğunu gösterir. Akış Çizelgesi yürütme sona erecek.  
  
## <a name="message"></a>İleti  
 Akış ' %1'/FlowStep - sonraki düğüme NULL'dur. Akış Çizelgesi yürütme sona erecek.
