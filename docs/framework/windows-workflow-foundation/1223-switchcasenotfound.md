# <a name="1223---switchcasenotfound"></a>1223 - SwitchCaseNotFound
## <a name="properties"></a>Özellikler  
  
|||  
|-|-|  
|Kimlik|1223|  
|anahtar sözcükler|WFActivities|  
|Düzey|Bilgiler|  
|Kanal|Microsoft Windows uygulaması sunucu-uygulamalar/hata ayıklama|  
  
## <a name="description"></a>Açıklama  
 Bir anahtar etkinliği örneği bir etkinlik ile eşleşen bir ifade sonucu bulunamadı gösterir.  
  
## <a name="message"></a>İleti  
 '%1' Switch etkinliği ifade sonucu eşleşen servis talebi bir etkinlik bulunamadı.
