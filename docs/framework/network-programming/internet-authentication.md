# <a name="internet-authentication"></a><span data-ttu-id="7399c-102">İnternet Kimlik Doğrulaması</span><span class="sxs-lookup"><span data-stu-id="7399c-102">Internet Authentication</span></span>
<span data-ttu-id="7399c-103"><xref:System.Net> Sınıfları istemci kimlik doğrulama mekanizmaları, temel standart Internet kimlik doğrulama yöntemleri de dahil olmak üzere çeşitli destek, Özet, negotiate, NTLM ve Kerberos kimlik doğrulaması, hem de oluşturabileceğiniz özel yöntemler.</span><span class="sxs-lookup"><span data-stu-id="7399c-103">The <xref:System.Net> classes support a variety of client authentication mechanisms, including the standard Internet authentication methods basic, digest, negotiate, NTLM, and Kerberos authentication, as well as custom methods that you can create.</span></span>  
  
 <span data-ttu-id="7399c-104">Kimlik doğrulama kimlik bilgileri depolanır <xref:System.Net.NetworkCredential> ve <xref:System.Net.CredentialCache> uygulayan sınıflar <xref:System.Net.ICredentials> arabirimi.</span><span class="sxs-lookup"><span data-stu-id="7399c-104">Authentication credentials are stored in the <xref:System.Net.NetworkCredential> and <xref:System.Net.CredentialCache> classes, which implement the <xref:System.Net.ICredentials> interface.</span></span> <span data-ttu-id="7399c-105">Bu sınıflardan birine kimlik bilgileri sorgulandığında örneğini döndürür. **NetworkCredential** sınıfı.</span><span class="sxs-lookup"><span data-stu-id="7399c-105">When one of these classes is queried for credentials, it returns an instance of the **NetworkCredential** class.</span></span> <span data-ttu-id="7399c-106">Kimlik doğrulama işlemi tarafından yönetilen <xref:System.Net.AuthenticationManager> sınıfı ve gerçek kimlik doğrulama işlemi uygulayan bir kimlik doğrulama modülü sınıfı tarafından gerçekleştirilen <xref:System.Net.IAuthenticationModule> arabirimi.</span><span class="sxs-lookup"><span data-stu-id="7399c-106">The authentication process is managed by the <xref:System.Net.AuthenticationManager> class, and the actual authentication process is performed by an authentication module class that implements the <xref:System.Net.IAuthenticationModule> interface.</span></span> <span data-ttu-id="7399c-107">Özel kimlik doğrulama modülü ile kaydetmelisiniz **bulunan** kullanılmadan önce; modüller için temel, Özet, negotiate, NTLM ve Kerberos kimlik doğrulama yöntemleri varsayılan olarak kayıtlı.</span><span class="sxs-lookup"><span data-stu-id="7399c-107">You must register a custom authentication module with the **AuthenticationManager** before it can be used; modules for the basic, digest, negotiate, NTLM, and Kerberos authentication methods are registered by default.</span></span>  
  
 <span data-ttu-id="7399c-108">**NetworkCredential** URI tarafından tanımlanan tek bir Internet kaynağına ile ilişkili kimlik bilgileri depolar ve yanıt olarak yapılan tüm çağrıların döndürür <xref:System.Net.NetworkCredential.GetCredential%2A> yöntemi.</span><span class="sxs-lookup"><span data-stu-id="7399c-108">**NetworkCredential** stores a set of credentials associated with a single Internet resource identified by a URI and returns them in response to any call to the <xref:System.Net.NetworkCredential.GetCredential%2A> method.</span></span> <span data-ttu-id="7399c-109">**NetworkCredential** sınıf genellikle sınırlı sayıda Internet kaynaklarına erişen uygulamaları veya her durumda aynı kimlik bilgileri kümesi kullanan uygulamalar tarafından kullanılır.</span><span class="sxs-lookup"><span data-stu-id="7399c-109">The **NetworkCredential** class is typically used by applications that access a limited number of Internet resources or by applications that use the same set of credentials in all cases.</span></span>  
  
 <span data-ttu-id="7399c-110">**CredentialCache** sınıfı depolayan çeşitli Web kaynaklar için kimlik bilgileri koleksiyonu.</span><span class="sxs-lookup"><span data-stu-id="7399c-110">The **CredentialCache** class stores a collection of credentials for various Web resources.</span></span> <span data-ttu-id="7399c-111">Zaman <xref:System.Net.CredentialCache.GetCredential%2A> yöntemi çağrıldığında **CredentialCache** uygun bir dizi kimlik bilgisi, Web kaynağı ve istenen kimlik doğrulama şeması URI'si tarafından belirlenen şekilde döndürür.</span><span class="sxs-lookup"><span data-stu-id="7399c-111">When the <xref:System.Net.CredentialCache.GetCredential%2A> method is called, **CredentialCache** returns the proper set of credentials, as determined by the URI of the Web resource and the requested authentication scheme.</span></span> <span data-ttu-id="7399c-112">Farklı kimlik doğrulama düzenleri çeşitli Internet kaynakları kullanan uygulamalar kullanma avantajını yakalayabilirler **CredentialCache** tüm kimlik bilgilerini depolar ve bunların istendiği gibi sağlar sınıfını.</span><span class="sxs-lookup"><span data-stu-id="7399c-112">Applications that use a variety of Internet resources with different authentication schemes benefit from using the **CredentialCache** class, since it stores all the credentials and provides them as requested.</span></span>  
  
 <span data-ttu-id="7399c-113">Bir Internet kaynağına kimlik doğrulaması istediğinde <xref:System.Net.WebRequest.GetResponse%2A?displayProperty=nameWithType> yöntemi gönderir <xref:System.Net.WebRequest> için **bulunan** birlikte bir istek için kimlik bilgileri.</span><span class="sxs-lookup"><span data-stu-id="7399c-113">When an Internet resource requests authentication, the <xref:System.Net.WebRequest.GetResponse%2A?displayProperty=nameWithType> method sends the <xref:System.Net.WebRequest> to the **AuthenticationManager** along with the request for credentials.</span></span> <span data-ttu-id="7399c-114">İsteğin şu işlem göre doğrulanır:</span><span class="sxs-lookup"><span data-stu-id="7399c-114">The request is then authenticated according to the following process:</span></span>  
  
1. <span data-ttu-id="7399c-115">**Bulunan** çağrıları <xref:System.Net.IAuthenticationModule.Authenticate%2A> yöntemi her kayıtlı sırada kayıtlı kimlik doğrulama modülü.</span><span class="sxs-lookup"><span data-stu-id="7399c-115">The **AuthenticationManager** calls the <xref:System.Net.IAuthenticationModule.Authenticate%2A> method on each of the registered authentication modules in the order they were registered.</span></span> <span data-ttu-id="7399c-116">**Bulunan** döndürmez ilk modülü kullanır **null** kimlik doğrulama işlemini gerçekleştirmek için.</span><span class="sxs-lookup"><span data-stu-id="7399c-116">The **AuthenticationManager** uses the first module that does not return **null** to carry out the authentication process.</span></span> <span data-ttu-id="7399c-117">İşlemin ayrıntılarını ilgili kimlik doğrulama modülü türüne bağlı olarak değişir.</span><span class="sxs-lookup"><span data-stu-id="7399c-117">The details of the process vary depending on the type of authentication module involved.</span></span>  
  
2. <span data-ttu-id="7399c-118">Kimlik doğrulama işlemi tamamlandıktan sonra kimlik doğrulama modülü döndürür bir <xref:System.Net.Authorization> için **WebRequest** Internet kaynağına erişmek için gereken bilgileri içerir.</span><span class="sxs-lookup"><span data-stu-id="7399c-118">When the authentication process is complete, the authentication module returns an <xref:System.Net.Authorization> to the **WebRequest** that contains the information needed to access the Internet resource.</span></span>  
  
 <span data-ttu-id="7399c-119">Bazı kimlik doğrulama düzeni bir kaynak için bir istek yapmadan kullanıcı kimlik doğrulaması yapabilir.</span><span class="sxs-lookup"><span data-stu-id="7399c-119">Some authentication schemes can authenticate a user without first making a request for a resource.</span></span> <span data-ttu-id="7399c-120">Bir uygulama, bu nedenle sunucu için en az bir gidiş dönüş ortadan kaynak kullanıcıyla preauthenticating tarafından zamandan tasarruf edebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="7399c-120">An application can save time by preauthenticating the user with the resource, thus eliminating at least one round trip to the server.</span></span> <span data-ttu-id="7399c-121">Veya, daha sonra kullanıcıya daha duyarlı olmamız için program başlatma sırasında kimlik doğrulaması gerçekleştirebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="7399c-121">Or, it can perform authentication during program startup in order to be more responsive to the user later.</span></span> <span data-ttu-id="7399c-122">Ön kimlik doğrulaması kullanabilir kimlik doğrulama düzenleri <xref:System.Net.IAuthenticationModule.PreAuthenticate%2A> özelliğini **true**.</span><span class="sxs-lookup"><span data-stu-id="7399c-122">Authentication schemes that can use preauthentication set the <xref:System.Net.IAuthenticationModule.PreAuthenticate%2A> property to **true**.</span></span>
