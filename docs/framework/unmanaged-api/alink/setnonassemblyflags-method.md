# <a name="setnonassemblyflags-method"></a>SetNonAssemblyFlags Yöntemi

## <a name="syntax"></a>Sözdizimi  
  
```  
HRESULT SetNonAssemblyFlags(  
    AssemblyFlags afFlags  
) PURE;  
```  
  
## <a name="parameters"></a>Parametreler  
 `afFlags`  
 ALINK bayraklar.
