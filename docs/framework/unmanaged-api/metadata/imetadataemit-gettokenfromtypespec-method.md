# <a name="imetadataemitgettokenfromtypespec-method"></a>IMetaDataEmit::GetTokenFromTypeSpec Metodu
Bir meta veri türü belirtilen meta veri imzası belirtecini alır.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
HRESULT GetTokenFromTypeSpec (   
    [in]  PCCOR_SIGNATURE       pvSig,   
    [in]  ULONG                 cbSig,   
    [out] mdTypeSpec            *ptypespec   
);  
```  
  
## <a name="parameters"></a>Parametreler  
 `pvSig`  
 [in] Tanımlanan imzası.  
  
 `cbSig`  
 [in] Bayt sayısı `pvSig`.  
  
 `ptypespec`  
 [out] `mdTypeSpec` Atanan simgesi.
