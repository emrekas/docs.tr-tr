# <a name="imetadatainfo-interface"></a>IMetaDataInfo Arabirimi
Meta veri eşleme bilgilerini bir disk dosyasından belleğe alır bir yöntem sağlar.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[GetFileMapping Yöntemi](../../../../docs/framework/unmanaged-api/metadata/imetadatainfo-getfilemapping-method.md)|Eşleşen dosya ve tür eşlemesi bir bellek bölgesini alır.|
