# <a name="imetadatatablesgetblob-method"></a>IMetaDataTables::GetBlob Metodu
Bir işaretçi ikili büyük nesne (BLOB) için belirtilen sütun dizini alır.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
HRESULT GetBlob (  
    [in]  ULONG          ixBlob,  
    [out] ULONG          *pcbData,  
    [out] const void     **ppData  
);  
```  
  
#### <a name="parameters"></a>Parametreler  
 `ixBlob`  
 [in] Bellek adresi alınmaya başlanacağı `ppData`.  
  
 `pcbData`  
 [out] Bayt cinsinden boyutu için bir işaretçi, `ppData`.  
  
 `ppData`  
 [out] İkili veriler için bir işaretçi işaretçisi alınır.
