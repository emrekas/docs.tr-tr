## <a name="syntax"></a>Sözdizimi  
  
```idl  
HRESULT DefineKickoffMethod(    [in] mdToken kickoffMethod);  
```  
  
## <a name="parameters"></a>Parametreler  
  
|Parametre|Açıklama|  
|---------------|-----------------|  
|`kickoffMethod`||

 Döndürür `HRESULT`.
