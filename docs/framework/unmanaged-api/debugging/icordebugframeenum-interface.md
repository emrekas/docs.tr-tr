# <a name="icordebugframeenum-interface1"></a>Icordebugframeenum arabirimi1
Icordebugenum yöntemlerini uygular ve Icordebugframe dizilerini numaralandırır.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[Next Yöntemi](../../../../docs/framework/unmanaged-api/debugging/icordebugframeenum-next-method.md)|Belirtilen sayıda alır `ICorDebugFrame` örneklerden geçerli konumunda başlayan bir numaralandırma.|  
  
## <a name="remarks"></a>Açıklamalar  
  
> [!NOTE]
>  Bu arabirim makineler arası veya çapraz işlem uzaktan çağrılan desteklemez.
