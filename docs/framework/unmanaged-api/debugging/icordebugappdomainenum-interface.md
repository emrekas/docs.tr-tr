# <a name="icordebugappdomainenum-interface"></a>ICorDebugAppDomainEnum Arabirimi

Sağlar `Next` belirtilen sayıda döndüren yöntemi `ICorDebugAppDomainEnum` sonraki konumdan başlayarak değerleri. Bu arabirim "ICorDebugEnum" sınıfıdır.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[Next Yöntemi](../../../../docs/framework/unmanaged-api/debugging/icordebugappdomainenum-next-method.md)|Koleksiyondan geçerli imleç konumdan başlayarak belirtilen sayıda uygulama etki alanları alır.|  
  
## <a name="remarks"></a>Açıklamalar  
  
> [!NOTE]
>  Bu arabirim makineler arası veya çapraz işlem uzaktan çağrılan desteklemez.
