# <a name="icordebugtypeenum-interface"></a>ICorDebugTypeEnum Arabirimi
"ICorDebugEnum" yöntemlerini uygular ve "ICorDebugType" dizilerini numaralandırır.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[Next Yöntemi](../../../../docs/framework/unmanaged-api/debugging/icordebugtypeenum-next-method.md)|Belirtilen sayıda alır `ICorDebugType` örneklerden geçerli konumunda başlayan bir numaralandırma.|  
  
## <a name="remarks"></a>Açıklamalar  
  
> [!NOTE]
>  Bu arabirim makineler arası veya çapraz işlem uzaktan çağrılan desteklemez.
