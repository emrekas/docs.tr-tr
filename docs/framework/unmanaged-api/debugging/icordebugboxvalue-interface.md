# <a name="icordebugboxvalue-interface"></a>ICorDebugBoxValue Arabirimi

Kutulanmış değer sınıf nesnesini temsil eden "ICorDebugHeapValue" öğesinin bir alt sınıfı.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[GetObject Yöntemi](../../../../docs/framework/unmanaged-api/debugging/icordebugboxvalue-getobject-method.md)|Kutulanmış "ICorDebugObjectValue" örneğine bir arabirim işaretçisi alır.|  
  
## <a name="remarks"></a>Açıklamalar  
  
> [!NOTE]
> Bu arabirim, çapraz makine ya da çapraz işlem için uzaktan çağrılmakta değil.
