|Yöntem|Açıklama|  
|------------|-----------------|  
|[GetLength Yöntemi](../../../../docs/framework/unmanaged-api/debugging/icordebugstringvalue-getlength-method.md)|Bu tarafından başvurulan dizedeki karakter sayısını alır `ICorDebugStringValue`.|
