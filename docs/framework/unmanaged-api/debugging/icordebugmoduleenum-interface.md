# <a name="icordebugmoduleenum-interface1"></a>Icordebugmoduleenum arabirimi1
Icordebugenum yöntemlerini uygular ve Icordebugmodule dizilerini numaralandırır.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[Next Yöntemi](../../../../docs/framework/unmanaged-api/debugging/icordebugmoduleenum-next-method.md)|Belirtilen sayıda alır `ICorDebugModule` örneklerden geçerli konumunda başlayan bir numaralandırma.|  
  
## <a name="remarks"></a>Açıklamalar  
  
> [!NOTE]
>  Bu arabirim makineler arası veya çapraz işlem uzaktan çağrılan desteklemez.
