# <a name="icordebugbreakpointenum-interface1"></a>Icordebugbreakpointenum arabirimi1
Icordebugenum yöntemlerini uygular ve Icordebugbreakpoint dizilerini numaralandırır.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[Next Yöntemi](../../../../docs/framework/unmanaged-api/debugging/icordebugbreakpointenum-next-method.md)|Belirtilen sayıda alır `ICorDebugBreakpoint` örneklerden geçerli konumunda başlayan bir numaralandırma.|  
  
## <a name="remarks"></a>Açıklamalar  
  
> [!NOTE]
>  Bu arabirim makineler arası veya çapraz işlem uzaktan çağrılan desteklemez.
