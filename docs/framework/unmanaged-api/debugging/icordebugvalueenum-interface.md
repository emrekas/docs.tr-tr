# <a name="icordebugvalueenum-interface1"></a>Icordebugvalueenum arabirimi1
"ICorDebugEnum" yöntemlerini uygular ve "ICorDebugValue" dizilerini numaralandırır.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[Next Yöntemi](../../../../docs/framework/unmanaged-api/debugging/icordebugvalueenum-next-method.md)|Belirtilen sayıda alır `ICorDebugValue` örneklerden geçerli konumunda başlayan bir numaralandırma.|  
  
## <a name="remarks"></a>Açıklamalar  
  
> [!NOTE]
>  Bu arabirim makineler arası veya çapraz işlem uzaktan çağrılan desteklemez.
