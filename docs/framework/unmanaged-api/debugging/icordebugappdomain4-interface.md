# <a name="icordebugappdomain4-interface"></a>ICorDebugAppDomain4 Arabirimi
Icordebugappdomain arabirimi COM çağrılabilir sarmalayıcısı yönetilen bir nesneye almak için mantıksal olarak genişletir.  
  
## <a name="method"></a>Yöntem  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[GetObjectForCCW Yöntemi](../../../../docs/framework/unmanaged-api/debugging/icordebugappdomain4-getobjectforccw-method.md)|COM çağrılabilir sarmalayıcı (CCW) işaretçisinden bir yönetilen bir nesne alır.|
