Bu yöntem .NET Framework'ün geçerli sürümde uygulanmadı.  
  
## <a name="syntax"></a>Sözdizimi  
  
```cpp  
HRESULT GetCodeBase (  
    [in] ULONG32  cchName,  
    [out] ULONG32 *pcchName,  
    [out, size_is(cchName), length_is(*pcchName)]   
        WCHAR szName[]  
);  
```
