# <a name="icordebugtypegetrank-method"></a>ICorDebugType::GetRank Metodu
Bir dizi türü boyut sayısını alır.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
HRESULT GetRank (  
    [out] ULONG32   *pnRank  
);  
```  
  
## <a name="parameters"></a>Parametreler  
 `pnRank`  
 [out] Boyutların sayısı için bir işaretçi.
