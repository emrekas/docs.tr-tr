# <a name="deprecated-clr-hosting-interfaces-and-coclasses"></a>Kullanım Dışı CLR Barındırma Arabirimleri ve Yardımcı Sınıfları
Bu bölümde, yönetilmeyen arabirimler açıklanmaktadır. konakları ortak dil çalışma zamanı (CLR) tümleştirme için kullanabileceğiniz .NET Framework sürüm 1.0 ve 1.1 uygulamalarına. Bu arabirimler, yapılandırmak ve çalışma zamanını bir işleme yüklemek bir konak için yöntemler sağlar.  
  
## <a name="in-this-section"></a>Bu Bölümde  
 Iappdomainsetup  
 Yapılandırmak konak için yöntemler sağlar bir <xref:System.AppDomain>.  
  
 [ICeeFileGen Sınıfı](../../../../docs/framework/unmanaged-api/hosting/iceefilegen-class.md)  
 (Kullanım dışı) Bir yerel taşınabilir yürütülebilir (PE) dosyası oluşturmak için işlevsellik sağlar.  
  
 [ICorRuntimeHost Arabirimi](../../../../docs/framework/unmanaged-api/hosting/icorruntimehost-interface.md)  
 CLR ayarlarını yapılandırmak konak için yöntemler sağlar.
