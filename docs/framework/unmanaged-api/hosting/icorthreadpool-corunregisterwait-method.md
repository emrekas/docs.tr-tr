# <a name="icorthreadpoolcorunregisterwait-method"></a>ICorThreadpool::CorUnregisterWait Yöntemi

## <a name="syntax"></a>Sözdizimi  
  
```cpp  
HRESULT CorUnregisterWait (  
    [in] HANDLE hWaitObject,  
    [in] HANDLE CompletionEvent,  
    [out] BOOL* result  
);  
```
