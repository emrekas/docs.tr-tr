# <a name="itypenamegettypeargumentcount-method"></a>ITypeName::GetTypeArgumentCount Yöntemi

## <a name="syntax"></a>Sözdizimi  
  
```  
HRESULT GetTypeArgumentCount (  
    [out, retval] DWORD* pCount  
);  
```
