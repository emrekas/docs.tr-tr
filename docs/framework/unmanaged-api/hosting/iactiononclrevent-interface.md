# <a name="iactiononclrevent-interface"></a>IActionOnCLREvent Arabirimi
Sağlar [Iactiononclrevent::ONEVENT](../../../../docs/framework/unmanaged-api/hosting/iactiononclrevent-onevent-method.md) bir çağrı kullanılarak kaydedilmiş olayları geri çağırmaları gerçekleştiren yöntemi [Iclroneventmanager::registeractiononevent](../../../../docs/framework/unmanaged-api/hosting/iclroneventmanager-registeractiononevent-method.md) yöntemi.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[OnEvent Yöntemi](../../../../docs/framework/unmanaged-api/hosting/iactiononclrevent-onevent-method.md)|Kayıtlı bir olay için bir geri arama gerçekleştirir.|
