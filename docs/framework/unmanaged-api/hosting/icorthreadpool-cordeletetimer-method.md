# <a name="icorthreadpoolcordeletetimer-method"></a>ICorThreadpool::CorDeleteTimer Yöntemi

## <a name="syntax"></a>Sözdizimi  
  
```  
HRESULT CorDeleteTimer (  
    [in]  HANDLE Timer,  
    [in]  HANDLE CompletionEvent,  
    [out] BOOL*  result

```
