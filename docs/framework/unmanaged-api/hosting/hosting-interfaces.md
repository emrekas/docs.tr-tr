# <a name="hosting-interfaces"></a>Barındırma Arabirimleri
Bu bölümde, yönetilmeyen arabirimler açıklanmaktadır. konaklar, ortak dil çalışma zamanı (CLR) uygulamalarıyla tümleştirmeleri için kullanabilirsiniz.  
  
 .NET Framework sürüm 2.0 barındırma arabirimleri, .NET Framework sürüm 1.0 ve 1.1 arabirimler yerini alır. Arabirimleri iki kümesi arasında önemli farklılıklar vardır. Bunları karıştırma beklenmeyen davranışlara neden olabilir ve önerilmez.  
  
 .NET Framework 3.0 ve 3.5 sürümlerini .NET Framework sürüm 2.0 barındırma arabirimleri kullanır ve tüm barındırma özellikleri tanıtan değil.  
  
 .NET Framework 4 barındırma arabirimleri, .NET Framework 2.0 arabirimleri yerini alır.
