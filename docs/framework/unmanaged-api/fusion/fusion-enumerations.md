# <a name="fusion-enumerations"></a>Fusion Numaralandırmaları
Bu bölümde API fusion kullandığı yönetilmeyen numaralandırmaları açıklar.  
  
## <a name="in-this-section"></a>Bu Bölümde  
 [ASM_CACHE_FLAGS Sabit Listesi](../../../../docs/framework/unmanaged-api/fusion/asm-cache-flags-enumeration.md)  
 Tarafından temsil edilen bir derleme kaynağını gösterir [Iassemblycacheıtem](../../../../docs/framework/unmanaged-api/fusion/iassemblycacheitem-interface.md) genel derleme önbelleğinde.  
  
 [ASM_CMP_FLAGS Sabit Listesi](../../../../docs/framework/unmanaged-api/fusion/asm-cmp-flags-enumeration.md)  
 Sürüm, yapı, kültür, imza ve tarafından Karşılaştırılacak iki derlemelerin benzeri belirten [Iassemblyname::ısequal](../../../../docs/framework/unmanaged-api/fusion/iassemblyname-isequal-method.md) yöntemi.  
  
 [ASM_DISPLAY_FLAGS Sabit Listesi](../../../../docs/framework/unmanaged-api/fusion/asm-display-flags-enumeration.md)  
 Sürüm, derleme, kültür, imza ve benzeri, derlemenin görünen adı tarafından alınabilir gösterir [Iassemblyname::getdisplayname](../../../../docs/framework/unmanaged-api/fusion/iassemblyname-getdisplayname-method.md) yöntemi.  
  
 [ASM_NAME Sabit Listesi](../../../../docs/framework/unmanaged-api/fusion/asm-name-enumeration.md)

 [AssemblyComparisonResult Sabit Listesi](../../../../docs/framework/unmanaged-api/fusion/assemblycomparisonresult-enumeration.md)

 Özniteliklerini belirtir bir `IAssemblyName` nesnesi tarafından oluşturulduğunda [CreateAssemblyNameObject](../../../../docs/framework/unmanaged-api/fusion/createassemblynameobject-function.md) işlevi.
