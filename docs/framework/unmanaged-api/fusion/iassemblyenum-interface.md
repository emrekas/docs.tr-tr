|Yöntem|Açıklama|  
|------------|-----------------|  
|[Clone Yöntemi](../../../../docs/framework/unmanaged-api/fusion/iassemblyenum-clone-method.md)|Bu basit bir kopyasını oluşturur `IAssemblyEnum` nesne.|
