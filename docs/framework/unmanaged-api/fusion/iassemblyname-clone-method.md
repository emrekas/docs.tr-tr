## <a name="syntax"></a>Sözdizimi  
  
```cpp  
HRESULT Clone (  
    [out] IAssemblyName **pName  
);  
```  
  
## <a name="parameters"></a>Parametreler

 dışı Bu `IAssemblyName` nesnenin döndürülen kopyası.
