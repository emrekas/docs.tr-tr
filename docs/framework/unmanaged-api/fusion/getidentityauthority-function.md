# <a name="getidentityauthority-function"></a>GetIdentityAuthority İşlevi

## <a name="syntax"></a>Sözdizimi  
  
```cpp  
HRESULT GetIdentityAuthority (  
    [out] IIdentityAuthority   **ppIIdentityAuthority  
 );
```  
