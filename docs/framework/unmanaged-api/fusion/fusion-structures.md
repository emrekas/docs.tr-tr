# <a name="fusion-structures"></a>Fusion Yapıları
Bu bölümde API fusion kullandığı yönetilmeyen yapıları açıklar.  
  
## <a name="in-this-section"></a>Bu Bölümde  
 [ASSEMBLY_INFO Yapısı](../../../../docs/framework/unmanaged-api/fusion/assembly-info-structure.md)  
 Genel derleme önbelleğinde kayıtlı bir derlemeyle ilgili bilgiler içerir.  
  
 [FUSION_INSTALL_REFERENCE Yapısı](../../../../docs/framework/unmanaged-api/fusion/fusion-install-reference-structure.md)  
 Uygulama genel derleme önbelleğinde yüklü olduğu bir derlemeye bir uygulama sağlar. bir başvuruyu temsil eder.  
  
 [IDENTITY_ATTRIBUTE Yapısı](../../../../docs/framework/unmanaged-api/fusion/identity-attribute-structure.md)  
 Hakkında meta veri özniteliğinin bilgilerini içeren bir [Idefinitionıdentity](../../../../docs/framework/unmanaged-api/fusion/idefinitionidentity-interface.md) örneği.  
  
 [IDENTITY_ATTRIBUTE_BLOB Yapısı](../../../../docs/framework/unmanaged-api/fusion/identity-attribute-blob-structure.md)  
 İlişkili bir uzaklık bilgilerini içeren [ıdentıty_attrıbute](../../../../docs/framework/unmanaged-api/fusion/identity-attribute-structure.md) yapısı.
