## <a name="lifetime-of-collectible-assemblies"></a><span data-ttu-id="d34d9-110">Toplanabilir derlemeler ömrü</span><span class="sxs-lookup"><span data-stu-id="d34d9-110">Lifetime of collectible assemblies</span></span>

<span data-ttu-id="d34d9-111">Toplanabilir derleme ömrünü içerdiği türleri ve bu türlerden oluşturulan nesnelere başvurular varlığını tarafından denetlenir.</span><span class="sxs-lookup"><span data-stu-id="d34d9-111">The lifetime of a collectible assembly is controlled by the existence of references to the types it contains and the objects that are created from those types.</span></span> <span data-ttu-id="d34d9-112">Bir veya daha fazlasını mevcut olduğu sürece ortak dil çalışma zamanının bir derlemeyi kaldırmaz (`T` derlemede tanımlanan herhangi bir tür):</span><span class="sxs-lookup"><span data-stu-id="d34d9-112">The common language runtime does not unload an assembly as long as one or more of the following exist (`T` is any type that is defined in the assembly):</span></span>

- <span data-ttu-id="d34d9-114">Bir dizi örneği `T`.</span><span class="sxs-lookup"><span data-stu-id="d34d9-114">An instance of an array of `T`.</span></span>

   > [!IMPORTANT]
   > <span data-ttu-id="d34d9-118">Bütünleştirilmiş kod parçalarını temsil eden tüm nesneleri serbest bırakmanız gerekir.</span><span class="sxs-lookup"><span data-stu-id="d34d9-118">You must release all objects that represent parts of the assembly.</span></span> <span data-ttu-id="d34d9-119"><xref:System.Reflection.Emit.ModuleBuilder> Tanımlayan `T` bir başvuru tutar <xref:System.Reflection.Emit.TypeBuilder>ve <xref:System.Reflection.Emit.AssemblyBuilder> nesne başvuru tutar <xref:System.Reflection.Emit.ModuleBuilder>, bu nesnelere başvuruları serbest bırakılması gerekir.</span><span class="sxs-lookup"><span data-stu-id="d34d9-119">The <xref:System.Reflection.Emit.ModuleBuilder> that defines `T` keeps a reference to the <xref:System.Reflection.Emit.TypeBuilder>, and the <xref:System.Reflection.Emit.AssemblyBuilder> object keeps a reference to the <xref:System.Reflection.Emit.ModuleBuilder>, so references to these objects must be released.</span></span> <span data-ttu-id="d34d9-120">Varlığı bile bir <xref:System.Reflection.Emit.LocalBuilder> veya <xref:System.Reflection.Emit.ILGenerator> oluşumunu kullanılan `T` kaldırma engeller.</span><span class="sxs-lookup"><span data-stu-id="d34d9-120">Even the existence of a <xref:System.Reflection.Emit.LocalBuilder> or an <xref:System.Reflection.Emit.ILGenerator> used in the construction of `T` prevents unloading.</span></span>

- <span data-ttu-id="d34d9-121">Statik başvuru `T` başka bir dinamik olarak tanımlanan bir tür tarafından `T1` olan hala erişilebilir kodu yürüterek.</span><span class="sxs-lookup"><span data-stu-id="d34d9-121">A static reference to `T` by another dynamically defined type `T1` that is still reachable by executing code.</span></span> <span data-ttu-id="d34d9-122">Örneğin, `T1` öğesinden türetilen `T`, veya `T` yöntemi içindeki bir parametre türü olabilir `T1`.</span><span class="sxs-lookup"><span data-stu-id="d34d9-122">For example, `T1` might derive from `T`, or `T` might be the type of a parameter in a method of `T1`.</span></span>

- <span data-ttu-id="d34d9-128">Derlemenin bir modülde tanımlanan statik bir yöntem için temsilci.</span><span class="sxs-lookup"><span data-stu-id="d34d9-128">A delegate to a static method that is defined in a module of the assembly.</span></span>

<span data-ttu-id="d34d9-129">Yalnızca bu listedeki bir öğe yalnızca bir tür veya derlemedeki bir yöntemi varsa, çalışma zamanı derleme kaldıramıyor.</span><span class="sxs-lookup"><span data-stu-id="d34d9-129">If only one item from this list exists for only one type or one method in the assembly, the runtime cannot unload the assembly.</span></span>

> <span data-ttu-id="d34d9-130">Listedeki tüm öğeler için sonlandırıcılar çalıştırılana kadar çalışma zamanı derleme gerçekten kaldırmaz.</span><span class="sxs-lookup"><span data-stu-id="d34d9-130">The runtime does not actually unload the assembly until finalizers have run for all items in the list.</span></span>

## <a name="restrictions-on-collectible-assemblies"></a><span data-ttu-id="d34d9-133">Toplanabilir derlemeler kısıtlamaları</span><span class="sxs-lookup"><span data-stu-id="d34d9-133">Restrictions on collectible assemblies</span></span>
