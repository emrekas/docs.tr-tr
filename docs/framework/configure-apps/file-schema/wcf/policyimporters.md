# <a name="policyimporters"></a>\<PolicyImporters >
Bu yapılandırma öğesi bağlamalar hakkında özel ilke onaylamalarını denetleyen tüm ilke içe aktarımlarını belirler. Her alt öğe, bağlama özellikleriyle`policyImporter`ilgili özel ilke onaylamalarını aramak için kullanılan bir < > ve onaylama için gereken özellikleri uygulayan özel bir bağlama öğesi ekler.
