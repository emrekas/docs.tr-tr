# <a name="ltpolicyimportersgt"></a>&lt;policyImporters&gt;
Bu yapılandırma öğesi bağlamalar hakkında özel ilke onaylamalarını içe denetleyen tüm ilke ımporters belirtir. Her alt öğesi olan bir <`policyImporter`> yanı sıra arama özellikleri bağlama hakkında özel ilke onaylamalarını onay gerektiren özellikleri uygulayan bir özel bağlama öğesi eklemek için kullanılır.
