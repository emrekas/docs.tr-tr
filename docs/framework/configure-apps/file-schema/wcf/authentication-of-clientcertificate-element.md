## <a name="example"></a><span data-ttu-id="23019-193">Örnek</span><span class="sxs-lookup"><span data-stu-id="23019-193">Example</span></span>  
 <span data-ttu-id="23019-194">Aşağıdaki kod, `<authentication>` öğesinde bir X. 509.952 sertifikası ve özel bir doğrulama türü belirtir.</span><span class="sxs-lookup"><span data-stu-id="23019-194">The following code specifies an X.509 certificate and a custom validation type in the `<authentication>` element.</span></span>  
  
```xml  
<serviceBehaviors>
  <behavior name="myServiceBehavior">
    <clientCertificate>
      <certificate findValue="www.cohowinery.com"
                   storeLocation="CurrentUser"
                   storeName="TrustedPeople"
                   x509FindType="FindByIssuerName" />
      <authentication customCertificateValidatorType="MyTypes.Coho"
                      certificateValidationMode="Custom"
                      revocationMode="Offline"
                      includeWindowsGroups="false"
                      mapClientCertificateToWindowsAccount="true" />
    </clientCertificate>
  </behavior>
```  
