# <a name="claimtyperequired"></a>\<claimTypeRequired >
Gelen güvenlik belirteçleri için gerekli talep kümesini belirtir.  
  
 \<system.identityModel>  
\<identityConfiguration >  
\<claimTypeRequired >  
  
## <a name="syntax"></a>Sözdizimi  
  
```xml  
<system.identityModel>  
  <identityConfiguration>  
    <claimTypeRequired>  
    </claimTypeRequired>  
  </identityConfiguration>  
</system.identityModel>  
```  
  
## <a name="attributes-and-elements"></a>Öznitelikler ve Öğeler  
 Öznitelikler, alt ve üst öğeler aşağıdaki bölümlerde açıklanmaktadır.  
  
### <a name="attributes"></a>Öznitelikler  
 None

|Öğe|Açıklama|  
|-------------|-----------------|  
|[\<claimType >](../../../../../docs/framework/configure-apps/file-schema/windows-identity-foundation/claimtype.md)|Gelen güvenlik belirteçleri için tek bir isteğe bağlı veya gerekli talep belirtir.|  
  
### <a name="parent-elements"></a>Üst Öğeler  
  
|Öğe|Açıklama|  
|-------------|-----------------|
