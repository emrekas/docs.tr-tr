# <a name="windows-workflow-foundation-configuration-schema"></a>Windows Workflow Foundation Yapılandırma Şeması
Windows Workflow Foundation (WF) yapılandırma öğeleri, iş akışı uygulamaları yapılandırmanızı sağlar. Bir iş akışı uygulama için izleme ve izleme yanı sıra, yapılandırabilirsiniz. İzleme ve izleme hakkında daha fazla bilgi için bkz. [iş akışı izleme ve izleme](../../../../../docs/framework/windows-workflow-foundation/workflow-tracking-and-tracing.md). İş akışı hizmetleri için Windows Communication Foundation (WCF) yapılandırma öğelerini de kullanabilirsiniz. WCF hakkında daha fazla ayrıntı için bkz. [WCF yapılandırma şeması](../../../../../docs/framework/configure-apps/file-schema/wcf/index.md).  
  
 Yapılandırma dosyaları XML olarak biçimlendirilir olduğundan, bir metin Düzenleyicisi'ni kullanarak el ile düzenlemek istiyorsanız, XML ile bilgi sahibi olmanız gerekir. Aksi takdirde içine sorunları unfound XML öğesi etiketi veya öznitelik gibi çalışabilir. Bunun nedeni, XML öğesi etiketleri ve öznitelikleri duyarlıdır.
