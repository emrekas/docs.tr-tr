 <span data-ttu-id="4d93d-182">Aşağıdaki komut, komut sözdiziminin bir açıklamasını ve InstallUtil.exe'ye ilişkin seçenekleri görüntüler.</span><span class="sxs-lookup"><span data-stu-id="4d93d-182">The following command displays a description of the command syntax and options for InstallUtil.exe.</span></span>
