## <a name="connection-string-injection-attacks"></a><span data-ttu-id="62c84-110">Bağlantı dizesi ekleme saldırıları</span><span class="sxs-lookup"><span data-stu-id="62c84-110">Connection String Injection Attacks</span></span>  
 <span data-ttu-id="62c84-111">Dinamik dize birleştirme kullanıcı girişini temel alarak bağlantı dizeleri oluşturmak için kullanılan bir bağlantı dizesi ekleme saldırısına oluşabilir.</span><span class="sxs-lookup"><span data-stu-id="62c84-111">A connection string injection attack can occur when dynamic string concatenation is used to build connection strings that are based on user input.</span></span> <span data-ttu-id="62c84-112">Dize doğrulanmış ve kötü amaçlı bir metin veya kaçış karakteri değil ise, bir saldırganın hassas verileri veya sunucudaki diğer kaynakları içeriklerine erişebilir.</span><span class="sxs-lookup"><span data-stu-id="62c84-112">If the string is not validated and malicious text or characters not escaped, an attacker can potentially access sensitive data or other resources on the server.</span></span> <span data-ttu-id="62c84-113">Örneğin, bir saldırganın bir noktalı virgül sağlayarak ve ek bir değer ekleme saldırısı.</span><span class="sxs-lookup"><span data-stu-id="62c84-113">For example, an attacker could mount an attack by supplying a semicolon and appending an additional value.</span></span> <span data-ttu-id="62c84-114">Bağlantı dizesi "son bir WINS" algoritması kullanılarak ayrıştırılır ve tehlikeli giriş için meşru bir değer ile değiştirilir.</span><span class="sxs-lookup"><span data-stu-id="62c84-114">The connection string is parsed by using a "last one wins" algorithm, and the hostile input is substituted for a legitimate value.</span></span>  
  
 <span data-ttu-id="62c84-115">Bağlantı dizesi Oluşturucusu sınıfları kararın ortadan kaldırabilir ve söz dizimi hataları ve güvenlik açıklarına karşı korumak için tasarlanmıştır.</span><span class="sxs-lookup"><span data-stu-id="62c84-115">The connection string builder classes are designed to eliminate guesswork and protect against syntax errors and security vulnerabilities.</span></span> <span data-ttu-id="62c84-116">Bunlar, yöntemleri ve her bir veri sağlayıcısı tarafından izin verilen bilinen anahtar/değer çiftleri için karşılık gelen özellikleri sağlar.</span><span class="sxs-lookup"><span data-stu-id="62c84-116">They provide methods and properties corresponding to the known key/value pairs permitted by each data provider.</span></span> <span data-ttu-id="62c84-117">Her sınıf, eş anlamlılar sabit bir koleksiyonunu tutar ve iyi bilinen bir karşılık gelen anahtar adı bir eş anlamlı çevirebilir.</span><span class="sxs-lookup"><span data-stu-id="62c84-117">Each class maintains a fixed collection of synonyms and can translate from a synonym to the corresponding well-known key name.</span></span> <span data-ttu-id="62c84-118">Geçerli anahtar/değer çiftleri için önleme denetimleri yapıldıktan ve geçersiz bir çifti bir özel durum oluşturur.</span><span class="sxs-lookup"><span data-stu-id="62c84-118">Checks are performed for valid key/value pairs and an invalid pair throws an exception.</span></span> <span data-ttu-id="62c84-119">Ayrıca, eklenen değerleri güvenli bir şekilde ele alınır.</span><span class="sxs-lookup"><span data-stu-id="62c84-119">In addition, injected values are handled in a safe manner.</span></span>  
  
 <span data-ttu-id="62c84-120">Aşağıdaki örnek, gösterir nasıl <xref:System.Data.SqlClient.SqlConnectionStringBuilder> eklenmiş bir ek değer işleme `Initial Catalog` ayarı.</span><span class="sxs-lookup"><span data-stu-id="62c84-120">The following example demonstrates how the <xref:System.Data.SqlClient.SqlConnectionStringBuilder> handles an inserted extra value for the `Initial Catalog` setting.</span></span>

