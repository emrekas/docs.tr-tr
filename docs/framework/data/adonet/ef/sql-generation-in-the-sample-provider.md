# <a name="sql-generation-in-the-sample-provider"></a>Örnek Sağlayıcısında SQL Oluşturma
[Entity Framework örnek sağlayıcı](https://code.msdn.microsoft.com/windowsdesktop/Entity-Framework-Sample-6a9801d0) , [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)]tarafından desteklenen ADO.NET veri sağlayıcılarının yeni bileşenlerini gösterir.  SQL Server 2005 veritabanı ile birlikte çalışarak System. Data. SqlClient ADO.NET 2,0 Veri Sağlayıcısı için sarmalayıcı olarak uygulanır.  
  
 Örnek sağlayıcının SQL oluşturma modülü (örneğin, DmlSqlGenerator.cs dosyası dışında SQL oluşturma klasörü altında bulunur), bir giriş DbQueryCommandTree alır ve tek bir SQL SELECT açıklaması üretir.
