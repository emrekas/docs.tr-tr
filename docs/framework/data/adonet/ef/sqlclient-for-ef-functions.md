# <a name="sqlclient-for-entity-framework-functions"></a>Entity Framework için SqlClient İşlevleri
Entity Framework SQL Server (SqlClient) için .NET Framework veri sağlayıcısı, matematik ve toplama hesaplamaları gerçekleştirmek için işlevler `System.DateTime` ve işlemler ve `string` işlemler için işlevler sağlar. Bu işlevler `SQLServer` ad alanıdır.  
  
 Herhangi bir sağlayıcı ile çalışması gereken işlevlerin listesi için bkz. [kurallı işlevler](./language-reference/canonical-functions.md).  
  
 Kurallı işlevlerin SQL Server işlevlere nasıl eşlenmesiyle ilgili bilgi için, bkz. [kavramsal model kurallı ve SQL Server Işlevleri eşleme](conceptual-model-canonical-to-sql-server-functions-mapping.md).
