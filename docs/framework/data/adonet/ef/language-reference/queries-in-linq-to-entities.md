 <span data-ttu-id="fa3e4-108">Bir LINQ Sorgu işlemi üç eylemden oluşur: sorguyu veri kaynağından veya kaynaklarından elde etmek ve sorgu oluşturun.</span><span class="sxs-lookup"><span data-stu-id="fa3e4-108">A LINQ query operation consists of three actions: obtain the data source or sources, create the query, and execute the query.</span></span>
