2.  <span data-ttu-id="ce3c2-119">Komut isteminde satır sonları olmadan aşağıdaki komutu yürütün:</span><span class="sxs-lookup"><span data-stu-id="ce3c2-119">At the command prompt, execute the following command without line breaks:</span></span>

