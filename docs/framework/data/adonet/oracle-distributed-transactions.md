# <a name="oracle-distributed-transactions"></a>Oracle Dağıtılmış İşlemleri
<xref:System.Data.OracleClient.OracleConnection> Nesne bir işlem etkin olduğunu belirlerse, varolan bir dağıtılmış işlemde otomatik olarak kaydeder. Bağlantı açılamıyor veya bağlantı havuzundan alınan otomatik bir işlem kaydı gerçekleşir. Belirterek mevcut işlemlerde otomatik kayıt devre dışı bırakabilirsiniz  
  
```  
Enlist=false  
```  
  
 için bir bağlantı dizesi parametresi olarak bir <xref:System.Data.OracleClient.OracleConnection>.
