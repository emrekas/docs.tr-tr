# <a name="how-to-represent-computed-columns"></a>Nasıl yapılır: Hesaplanan Sütunları Temsil Etme
İçeriği hesaplama sonucu olan bir <xref:System.Data.Linq.Mapping.ColumnAttribute> sütunu temsil etmek için bir özniteliğinde özelliğinikullanın.<xref:System.Data.Linq.Mapping.ColumnAttribute.Expression%2A> [!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)]  
  
 Kod örnekleri için bkz <xref:System.Data.Linq.Mapping.ColumnAttribute.Expression%2A>.
