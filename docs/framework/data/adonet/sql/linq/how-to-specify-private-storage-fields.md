# <a name="how-to-specify-private-storage-fields"></a>Nasıl yapılır: Özel Depolama Alanları Belirtme
Temel bir depolama alanının adını <xref:System.Data.Linq.Mapping.DataAttribute> belirlemek için özniteliğinde özelliğinikullanın.<xref:System.Data.Linq.Mapping.DataAttribute.Storage%2A> [!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)]  
  
 Kod örnekleri için bkz <xref:System.Data.Linq.Mapping.DataAttribute.Storage%2A>.  
  
### <a name="to-specify-the-name-of-an-underlying-storage-field"></a>Temel alınan bir depolama alanının adını belirtmek için  
  
1. <xref:System.Data.Linq.Mapping.DataAttribute.Storage%2A> Özelliği<xref:System.Data.Linq.Mapping.ColumnAttribute> özniteliğine ekleyin.  
  
2. <xref:System.Data.Linq.Mapping.DataAttribute.Storage%2A> Özelliğin değeri olarak alanın adını atayın.
