-   <span data-ttu-id="c922e-113">Kimlik doğrulaması, SQL Server'ın kullandığı bir asıl sunucu değerlendiren kimlik bilgileri göndererek erişim isteğinde oturum açmadan işlemidir.</span><span class="sxs-lookup"><span data-stu-id="c922e-113">Authentication is the process of logging on to SQL Server by which a principal requests access by submitting credentials that the server evaluates.</span></span> <span data-ttu-id="c922e-114">Kimlik doğrulaması, kullanıcı veya kimlik doğrulaması gerçekleştirilen işlem kimliğini oluşturur.</span><span class="sxs-lookup"><span data-stu-id="c922e-114">Authentication establishes the identity of the user or process being authenticated.</span></span>
