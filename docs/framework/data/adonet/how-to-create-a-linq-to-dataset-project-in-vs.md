## <a name="to-enable-linq-to-dataset-functionality"></a><span data-ttu-id="a5394-109">LINQ DataSet işlevselliğini etkinleştirmek için</span><span class="sxs-lookup"><span data-stu-id="a5394-109">To enable LINQ to DataSet functionality</span></span>

1. <span data-ttu-id="a5394-111">Başvuruları Ekle **System.Core**, **System.Data**, ve **System.Data.DataSetExtensions**.</span><span class="sxs-lookup"><span data-stu-id="a5394-111">Add references to **System.Core**, **System.Data**, and **System.Data.DataSetExtensions**.</span></span>

   <span data-ttu-id="a5394-112">İçinde **Çözüm Gezgini**, sağ **başvuruları** düğümünü seçip alt **Başvuru Ekle**.</span><span class="sxs-lookup"><span data-stu-id="a5394-112">In **Solution Explorer**, right-click on the **References** node and select **Add Reference**.</span></span> <span data-ttu-id="a5394-113">İçinde **başvuru Yöneticisi** iletişim kutusunda **System.Core**, **System.Data**, ve **System.Data.DataSetExtensions**.</span><span class="sxs-lookup"><span data-stu-id="a5394-113">In the **Reference Manager** dialog box, select **System.Core**, **System.Data**, and **System.Data.DataSetExtensions**.</span></span> <span data-ttu-id="a5394-114">**Tamam**’ı seçin.</span><span class="sxs-lookup"><span data-stu-id="a5394-114">Select **OK**.</span></span>

   ```csharp
   using System.Data;
   using System.Linq;
   ```
