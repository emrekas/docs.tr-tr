# <a name="dataadapter-datatable-and-datacolumn-mappings"></a><span data-ttu-id="6850f-102">DataAdapter DataTable ve DataColumn Eşlemeleri</span><span class="sxs-lookup"><span data-stu-id="6850f-102">DataAdapter DataTable and DataColumn Mappings</span></span>
<span data-ttu-id="6850f-103">A **DataAdapter** sıfır veya daha fazla koleksiyonu içeren <xref:System.Data.Common.DataTableMapping> nesneler, **TableMappings** özelliği.</span><span class="sxs-lookup"><span data-stu-id="6850f-103">A **DataAdapter** contains a collection of zero or more <xref:System.Data.Common.DataTableMapping> objects in its **TableMappings** property.</span></span> <span data-ttu-id="6850f-104">A **DataTableMapping** verileri arasındaki ana eşlemeyi döndürülen bir veri kaynağında bir sorgudan sağlar ve bir <xref:System.Data.DataTable>.</span><span class="sxs-lookup"><span data-stu-id="6850f-104">A **DataTableMapping** provides a master mapping between the data returned from a query against a data source, and a <xref:System.Data.DataTable>.</span></span> <span data-ttu-id="6850f-105">**DataTableMapping** adı yerine geçirilebilir **DataTable** için ad **dolgu** yöntemi **DataAdapter**.</span><span class="sxs-lookup"><span data-stu-id="6850f-105">The **DataTableMapping** name can be passed in place of the **DataTable** name to the **Fill** method of the **DataAdapter**.</span></span> <span data-ttu-id="6850f-106">Aşağıdaki örnek, oluşturur bir **DataTableMapping** adlı **AuthorsMapping** için **yazarlar** tablo.</span><span class="sxs-lookup"><span data-stu-id="6850f-106">The following example creates a **DataTableMapping** named **AuthorsMapping** for the **Authors** table.</span></span>  
  
```vb  
workAdapter.TableMappings.Add("AuthorsMapping", "Authors")  
```  
  
```csharp  
workAdapter.TableMappings.Add("AuthorsMapping", "Authors");  
```  
  
 <span data-ttu-id="6850f-107">A **DataTableMapping** sütun adları kullanmanızı sağlayan bir **DataTable** veritabanı farklı olan.</span><span class="sxs-lookup"><span data-stu-id="6850f-107">A **DataTableMapping** enables you to use column names in a **DataTable** that are different from those in the database.</span></span> <span data-ttu-id="6850f-108">**DataAdapter** sütunlarla tablo güncelleştirildiğinde eşleşmesi için eşleme kullanır.</span><span class="sxs-lookup"><span data-stu-id="6850f-108">The **DataAdapter** uses the mapping to match the columns when the table is updated.</span></span>  
  
 <span data-ttu-id="6850f-109">Belirtmezseniz, bir **TableName** veya **DataTableMapping** ad çağrılırken **dolgu** veya **güncelleştirme** yöntemi  **DataAdapter**, **DataAdapter** arayan bir **DataTableMapping** "Tablo" adlı.</span><span class="sxs-lookup"><span data-stu-id="6850f-109">If you do not specify a **TableName** or a **DataTableMapping** name when calling the **Fill** or **Update** method of the **DataAdapter**, the **DataAdapter** looks for a **DataTableMapping** named "Table".</span></span> <span data-ttu-id="6850f-110">Bu, **DataTableMapping** yok **TableName** , **DataTable** "Tablo".</span><span class="sxs-lookup"><span data-stu-id="6850f-110">If that **DataTableMapping** does not exist, the **TableName** of the **DataTable** is "Table".</span></span> <span data-ttu-id="6850f-111">Varsayılan belirtebilirsiniz **DataTableMapping** oluşturarak bir **DataTableMapping** "Tablo" adı.</span><span class="sxs-lookup"><span data-stu-id="6850f-111">You can specify a default **DataTableMapping** by creating a **DataTableMapping** with the name of "Table".</span></span>  
  
 <span data-ttu-id="6850f-112">Aşağıdaki kod örneği oluşturur bir **DataTableMapping** (gelen <xref:System.Data.Common> ad alanı) için belirtilen varsayılan eşleme hale getirir **DataAdapter** "Tablo" adlandırma tarafından.</span><span class="sxs-lookup"><span data-stu-id="6850f-112">The following code example creates a **DataTableMapping** (from the <xref:System.Data.Common> namespace) and makes it the default mapping for the specified **DataAdapter** by naming it "Table".</span></span> <span data-ttu-id="6850f-113">Örnek sorgu sonucu Birinci tablodaki sütunların sonra eşler ( **müşteriler** tablosu **Northwind** veritabanı) daha kullanıcı dostu adlarında bir dizi **Northwind müşterileri**  tablosundaki <xref:System.Data.DataSet>.</span><span class="sxs-lookup"><span data-stu-id="6850f-113">The example then maps the columns from the first table in the query result (the **Customers** table of the **Northwind** database) to a set of more user-friendly names in the **Northwind Customers** table in the <xref:System.Data.DataSet>.</span></span> <span data-ttu-id="6850f-114">Eşlenmemiş sütunlar için veri kaynağından sütunun adı kullanılır.</span><span class="sxs-lookup"><span data-stu-id="6850f-114">For columns that are not mapped, the name of the column from the data source is used.</span></span>  
  
```vb  
Dim mapping As DataTableMapping = _  
  adapter.TableMappings.Add("Table", "NorthwindCustomers")  
mapping.ColumnMappings.Add("CompanyName", "Company")  
mapping.ColumnMappings.Add("ContactName", "Contact")  
mapping.ColumnMappings.Add("PostalCode", "ZIPCode")  
  
adapter.Fill(custDS)  
```  
  
```csharp  
DataTableMapping mapping =   
  adapter.TableMappings.Add("Table", "NorthwindCustomers");  
mapping.ColumnMappings.Add("CompanyName", "Company");  
mapping.ColumnMappings.Add("ContactName", "Contact");  
mapping.ColumnMappings.Add("PostalCode", "ZIPCode");  
  
adapter.Fill(custDS);  
```  
  
 <span data-ttu-id="6850f-115">Daha gelişmiş durumlarda, aynı istediğinize karar **DataAdapter** farklı eşlemeleri farklı tablolarla yüklemeyi destekleyecek kadar.</span><span class="sxs-lookup"><span data-stu-id="6850f-115">In more advanced situations, you may decide that you want the same **DataAdapter** to support loading different tables with different mappings.</span></span> <span data-ttu-id="6850f-116">Bunu yapmak için ek eklemeniz yeterlidir **DataTableMapping** nesneleri.</span><span class="sxs-lookup"><span data-stu-id="6850f-116">To do this, simply add additional **DataTableMapping** objects.</span></span>  
  
 <span data-ttu-id="6850f-117">Zaman **dolgu** yönteme örneği bir **veri kümesi** ve **DataTableMapping** adı bu ada sahip bir eşleme varsa, aksi takdirde bir  **DataTable** ile adı kullanılır.</span><span class="sxs-lookup"><span data-stu-id="6850f-117">When the **Fill** method is passed an instance of a **DataSet** and a **DataTableMapping** name, if a mapping with that name exists it is used; otherwise, a **DataTable** with that name is used.</span></span>  
  
 <span data-ttu-id="6850f-118">Aşağıdaki örnekler bir **DataTableMapping** adıyla **müşteriler** ve **DataTable** adını **BizTalkSchema**.</span><span class="sxs-lookup"><span data-stu-id="6850f-118">The following examples create a **DataTableMapping** with a name of **Customers** and a **DataTable** name of **BizTalkSchema**.</span></span> <span data-ttu-id="6850f-119">Bu örnek için SELECT deyimi tarafından döndürülen satırlar sonra eşler **BizTalkSchema** **DataTable**.</span><span class="sxs-lookup"><span data-stu-id="6850f-119">The example then maps the rows returned by the SELECT statement to the **BizTalkSchema** **DataTable**.</span></span>  
  
```vb  
Dim mapping As ITableMapping = _  
  adapter.TableMappings.Add("Customers", "BizTalkSchema")  
mapping.ColumnMappings.Add("CustomerID", "ClientID")  
mapping.ColumnMappings.Add("CompanyName", "ClientName")  
mapping.ColumnMappings.Add("ContactName", "Contact")  
mapping.ColumnMappings.Add("PostalCode", "ZIP")  
  
adapter.Fill(custDS, "Customers")  
```  
  
```csharp  
ITableMapping mapping =   
  adapter.TableMappings.Add("Customers", "BizTalkSchema");  
mapping.ColumnMappings.Add("CustomerID", "ClientID");  
mapping.ColumnMappings.Add("CompanyName", "ClientName");  
mapping.ColumnMappings.Add("ContactName", "Contact");  
mapping.ColumnMappings.Add("PostalCode", "ZIP");  
  
adapter.Fill(custDS, "Customers");  
```

## <a name="handling-multiple-result-sets"></a><span data-ttu-id="6850f-125">Birden çok sonuç kümesi işleme</span><span class="sxs-lookup"><span data-stu-id="6850f-125">Handling Multiple Result Sets</span></span>  
 <span data-ttu-id="6850f-126">Varsa, **SelectCommand** birden fazla tabloyu döndürür **doldurun** tabloları için artımlı değerlerle tablo adları otomatik olarak oluşturduğu **veri kümesi**ile başlayan Belirtilen tablo adı ve devam etmeden biçiminde **TableName** *N*ile başlayan **TableName1**.</span><span class="sxs-lookup"><span data-stu-id="6850f-126">If your **SelectCommand** returns multiple tables, **Fill** automatically generates table names with incremental values for the tables in the **DataSet**, starting with the specified table name and continuing on in the form **TableName** *N*, starting with **TableName1**.</span></span> <span data-ttu-id="6850f-127">Tablo eşlemeleri için tabloda belirtilen bir adı otomatik olarak oluşturulan tablo adını eşleştirmek için kullanabileceğiniz **veri kümesi**.</span><span class="sxs-lookup"><span data-stu-id="6850f-127">You can use table mappings to map the automatically generated table name to a name you want specified for the table in the **DataSet**.</span></span> <span data-ttu-id="6850f-128">Örneğin, bir **SelectCommand** iki tablo döndüren **müşteriler** ve **siparişler**, aşağıdaki çağrısını sorun **dolgu**.</span><span class="sxs-lookup"><span data-stu-id="6850f-128">For example, for a **SelectCommand** that returns two tables, **Customers** and **Orders**, issue the following call to **Fill**.</span></span>

adapter.Fill(customersDataSet, "Customers")  
```  
  
 <span data-ttu-id="6850f-129">İki tablo oluşturulan **veri kümesi**: **Müşteriler** ve **Customers1**.</span><span class="sxs-lookup"><span data-stu-id="6850f-129">Two tables are created in the **DataSet**: **Customers** and **Customers1**.</span></span> <span data-ttu-id="6850f-130">Tablo eşlemeleri ikinci tablo adı emin olmak için kullanabileceğiniz **siparişler** yerine **Customers1**.</span><span class="sxs-lookup"><span data-stu-id="6850f-130">You can use table mappings to ensure that the second table is named **Orders** instead of **Customers1**.</span></span> <span data-ttu-id="6850f-131">Bunu yapmak için kaynak tablosu harita **Customers1** için **veri kümesi** tablo **siparişler**, aşağıdaki örnekte gösterildiği gibi.</span><span class="sxs-lookup"><span data-stu-id="6850f-131">To do this, map the source table of **Customers1** to the **DataSet** table **Orders**, as shown in the following example.</span></span>

adapter.TableMappings.Add("Customers1", "Orders")  
adapter.Fill(customersDataSet, "Customers")  
```
