# <a name="entity-key"></a><span data-ttu-id="392cf-102">Varlık anahtarı</span><span class="sxs-lookup"><span data-stu-id="392cf-102">entity key</span></span>
<span data-ttu-id="392cf-103">Bir *Varlık anahtarı* olduğu bir [özelliği](../../../../docs/framework/data/adonet/property.md) veya özellikleri kümesi bir [varlık türü](../../../../docs/framework/data/adonet/entity-type.md) kimliğini belirlemek için kullanılır.</span><span class="sxs-lookup"><span data-stu-id="392cf-103">An *entity key* is a [property](../../../../docs/framework/data/adonet/property.md) or a set of properties of an [entity type](../../../../docs/framework/data/adonet/entity-type.md) that are used to determine identity.</span></span> <span data-ttu-id="392cf-104">Bir varlık anahtarı oluşturan özelliklerini tasarım zamanında seçilir.</span><span class="sxs-lookup"><span data-stu-id="392cf-104">The properties that make up an entity key are chosen at design time.</span></span> <span data-ttu-id="392cf-105">Varlık anahtar özelliklerin değerlerini içinde bir varlık türü örneği benzersiz şekilde tanımlamalıdır bir [varlık kümesi](../../../../docs/framework/data/adonet/entity-set.md) çalışma zamanında.</span><span class="sxs-lookup"><span data-stu-id="392cf-105">The values of entity key properties must uniquely identify an entity type instance within an [entity set](../../../../docs/framework/data/adonet/entity-set.md) at run time.</span></span> <span data-ttu-id="392cf-106">Bir varlık kümesindeki örneklerin benzersizliği garanti etmek için bir varlık anahtarı oluşturan özellikleri seçilmelidir.</span><span class="sxs-lookup"><span data-stu-id="392cf-106">The properties that make up an entity key should be chosen to guarantee uniqueness of instances in an entity set.</span></span>  
  
 <span data-ttu-id="392cf-107">Bir varlık anahtarı için bir özellikler kümesini için gereksinimleri şunlardır:</span><span class="sxs-lookup"><span data-stu-id="392cf-107">The following are the requirements for a set of properties to be an entity key:</span></span>  
  
-   <span data-ttu-id="392cf-108">Bir varlık kümesi içinde hiçbir iki varlık anahtarları aynı olabilir.</span><span class="sxs-lookup"><span data-stu-id="392cf-108">No two entity keys within an entity set can be identical.</span></span> <span data-ttu-id="392cf-109">Diğer bir deyişle, herhangi iki varlık için bir varlık kümesindeki bir anahtar oluşturan tüm özellikler için değerleri aynı olamaz.</span><span class="sxs-lookup"><span data-stu-id="392cf-109">That is, for any two entities within an entity set, the values for all of the properties that constitute a key cannot be the same.</span></span> <span data-ttu-id="392cf-110">Ancak, bazı (Tümü değil) bir varlığı değerleri olan anahtar aynı olabilir.</span><span class="sxs-lookup"><span data-stu-id="392cf-110">However, some (but not all) of the values that make up an entity key can be the same.</span></span>  
  
-   <span data-ttu-id="392cf-111">Varlık anahtarı NULL olmayan, sabit, kümesinden oluşmalıdır [ilkel tür özellikleri](../../../../docs/framework/data/adonet/entity-data-model-primitive-data-types.md).</span><span class="sxs-lookup"><span data-stu-id="392cf-111">An entity key must consist of a set of non-nullable, immutable, [primitive type properties](../../../../docs/framework/data/adonet/entity-data-model-primitive-data-types.md).</span></span>  
  
-   <span data-ttu-id="392cf-112">Belirtilen varlık türü için bir varlık anahtarı oluşturan özelliklerini değiştiremezsiniz.</span><span class="sxs-lookup"><span data-stu-id="392cf-112">The properties that make up an entity key for a given entity type cannot change.</span></span> <span data-ttu-id="392cf-113">Belirtilen varlık türü için olası Varlık anahtarı birden fazla izin veremez; vekil anahtarlar desteklenmez.</span><span class="sxs-lookup"><span data-stu-id="392cf-113">You cannot allow more than one possible entity key for a given entity type; surrogate keys are not supported.</span></span>  
  
-   <span data-ttu-id="392cf-114">Bir devralma hiyerarşisinde bir varlığı söz konusu olduğunda kök varlığın varlık anahtarı oluşturan tüm özelliklerini içermelidir ve varlık anahtarı üzerinde kök varlık türü olarak tanımlanmış olması gerekir.</span><span class="sxs-lookup"><span data-stu-id="392cf-114">When an entity is involved in an inheritance hierarchy, the root entity must contain all the properties that make up the entity key, and the entity key must be defined on the root entity type.</span></span> <span data-ttu-id="392cf-115">Daha fazla bilgi için [varlık veri modeli: Devralma](../../../../docs/framework/data/adonet/entity-data-model-inheritance.md).</span><span class="sxs-lookup"><span data-stu-id="392cf-115">For more information, see [Entity Data Model: Inheritance](../../../../docs/framework/data/adonet/entity-data-model-inheritance.md).</span></span>
