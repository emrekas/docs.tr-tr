## <a name="to-request-the-default-service-document-by-using-internet-explorer"></a><span data-ttu-id="e1b18-109">Internet Explorer'ı kullanarak varsayılan hizmet belgesi istemek için</span><span class="sxs-lookup"><span data-stu-id="e1b18-109">To request the default service document by using Internet Explorer</span></span>

1. <span data-ttu-id="e1b18-110">Internet Explorer'da gelen **Araçları** menüsünde **Internet Seçenekleri**, tıklayın **içerik** sekmesinde **ayarları**, temizleyin **Akış görüntülenirken kapatma**.</span><span class="sxs-lookup"><span data-stu-id="e1b18-110">In Internet Explorer, from the **Tools** menu, select **Internet Options**, click the **Content** tab, click **Settings**, and clear **Turn on feed viewing**.</span></span>

    > <span data-ttu-id="e1b18-113">Ham XML verileri olarak akışa tarayıcınızın görüntüleyemiyorsanız yine de sayfası için kaynak kodu olarak akış görüntüleyebilir olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="e1b18-113">If your browser cannot display the feed as raw XML data, you should still be able to view the feed as the source code for the page.</span></span>

## <a name="to-access-entity-set-resources-from-a-web-browser"></a><span data-ttu-id="e1b18-118">Varlık erişmek için bir Web tarayıcısından kaynakları ayarlayın</span><span class="sxs-lookup"><span data-stu-id="e1b18-118">To access entity set resources from a Web browser</span></span>

1. <span data-ttu-id="e1b18-119">Web tarayıcınızın adres çubuğuna aşağıdaki URI girin:</span><span class="sxs-lookup"><span data-stu-id="e1b18-119">In the address bar of your Web browser, enter the following URI:</span></span>

    ```
    http://localhost:12345/northwind.svc/Customers
    ```

     <span data-ttu-id="e1b18-120">Bu, Northwind örnek veritabanındaki tüm müşteriler bir dizi döndürür.</span><span class="sxs-lookup"><span data-stu-id="e1b18-120">This returns a set of all customers in the Northwind sample database.</span></span>

2. <span data-ttu-id="e1b18-121">Web tarayıcınızın adres çubuğuna aşağıdaki URI girin:</span><span class="sxs-lookup"><span data-stu-id="e1b18-121">In the address bar of your Web browser, enter the following URI:</span></span>

    ```
    http://localhost:12345/northwind.svc/Customers('ALFKI')
    ```

     <span data-ttu-id="e1b18-122">Belirli bir müşteri için bu varlık örneğini döndürür `ALFKI`.</span><span class="sxs-lookup"><span data-stu-id="e1b18-122">This returns an entity instance for the specific customer, `ALFKI`.</span></span>

3. <span data-ttu-id="e1b18-123">Web tarayıcınızın adres çubuğuna aşağıdaki URI girin:</span><span class="sxs-lookup"><span data-stu-id="e1b18-123">In the address bar of your Web browser, enter the following URI:</span></span>

    ```
    http://localhost:12345/northwind.svc/Customers('ALFKI')/Orders
    ```

     <span data-ttu-id="e1b18-124">Bu müşteriler ve siparişler bir dizi belirli bir müşteri için tüm siparişleri döndürmek için arasındaki ilişkiyi erişir `ALFKI`.</span><span class="sxs-lookup"><span data-stu-id="e1b18-124">This traverses the relationship between customers and orders to return a set of all orders for the specific customer `ALFKI`.</span></span>

4. <span data-ttu-id="e1b18-125">Web tarayıcınızın adres çubuğuna aşağıdaki URI girin:</span><span class="sxs-lookup"><span data-stu-id="e1b18-125">In the address bar of your Web browser, enter the following URI:</span></span>

    ```
    http://localhost:12345/northwind.svc/Customers('ALFKI')/Orders?$filter=OrderID eq 10643
    ```

     <span data-ttu-id="e1b18-126">Bu belirli bir müşteriye ait siparişler filtreler `ALFKI` böylece belirli bir sırada sağlanan üzerinde göre döndürülen `OrderID` değeri.</span><span class="sxs-lookup"><span data-stu-id="e1b18-126">This filters orders that belong to the specific customer `ALFKI` so that only a specific order is returned based on the supplied `OrderID` value.</span></span>
