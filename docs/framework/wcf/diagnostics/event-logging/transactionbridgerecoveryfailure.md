# <a name="transactionbridgerecoveryfailure"></a>TransactionBridgeRecoveryFailure
Kimliği: 146  
  
 Önem derecesi: Hata  
  
 Kategori: TransactionBridge  
  
## <a name="description"></a>Açıklama  
 Bu olay MSDTC TransactionBridge Kurtarma sırasında başarısız olduğunu gösterir. MSDTC hizmetinin sona erdi için önemli bir koşulu budur. Özel durum, işlem adı ve işlem kimliği, olay listeler biçimindeki telefon numarasıdır.
