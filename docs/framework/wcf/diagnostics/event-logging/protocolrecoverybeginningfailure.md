# <a name="protocolrecoverybeginningfailure"></a>ProtocolRecoveryBeginningFailure
Numarasını 144  
  
 İnin Hata  
  
 Alan Işlem Köprüsü  
  
## <a name="description"></a>Açıklama  
 Bu olay, MSDTC WS-Atomik Işlem protokolünün kurtarma başlangıcında başarısız olduğunu gösterir. Sonuç olarak, WS Atomik Işlem işlevselliği devre dışıdır. Olayda protokol KIMLIĞI, protokol adı, özel durum, işlem adı ve işlem KIMLIĞI listelenmektedir.
