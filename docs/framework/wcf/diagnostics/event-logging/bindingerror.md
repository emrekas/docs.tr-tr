# <a name="bindingerror"></a>BindingError
Kimliği: 159  
  
 Önem derecesi: Hata  
  
 Kategori: SMSvcHost  
  
## <a name="description"></a>Açıklama  
 Bu olay belirli bir sitenin belirli bağlama ayrıştırılırken bir hata oluştu, bu nedenle protokolü için site geçici olarak devre dışıdır gösterir. Olay bağlama, kaynak, özel durum, işlem adı ve işlem kimliği listeler
