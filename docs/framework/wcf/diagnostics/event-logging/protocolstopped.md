# <a name="protocolstopped"></a>ProtocolStopped
Kimliği: 151  
  
 Önem derecesi: Hata  
  
 Kategori: TransactionBridge  
  
## <a name="description"></a>Açıklama  
 Bu olay, WS-Atomic işlem protokolü hizmetinin durdurulduğunu gösterir. Olay Protokolü kimliği, protokol adı, işlem adı ve işlem kimliği listeler biçimindeki telefon numarasıdır.
