# <a name="failfastexception"></a>FailFastException
Kimliği: 103  
  
 Önem derecesi: Hata  
  
 Kategori: İzleme  
  
## <a name="description"></a>Açıklama  
 İşlem sırasında hızla başarısız bir özel durum oluştu. Özel durum, işlem adı ve işlem kimliği, olay listeler
