# <a name="webhosthttperror"></a>WebHostHttpError
Kimliği: 110  
  
 Önem derecesi: Hata  
  
 Kategori: ServiceModel  
  
## <a name="description"></a>Açıklama  
 Bu olay, bir Web barındırma HTTP özel durumu oluştuğunu gösterir. Olayı Gönderen bilgisi HtmlErrorMessage, özel durum, işlem adı ve işlem kimliği listeler
