# <a name="piiloggingon"></a>PiiLoggingOn
Kimliği: 107  
  
 Önem derecesi: Hata  
  
 Kategori: İzleme  
  
## <a name="description"></a>Açıklama  
 Bu olay, PII günlüğü açıldı olduğunu gösterir. Hassas bilgileri günlüğe kaydedilir. Olay işlem kimliği ve işlem adını listeler.
