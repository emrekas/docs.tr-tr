# <a name="unhandledstatemachineexceptionrecorddescription"></a>UnhandledStateMachineExceptionRecordDescription
Numarasını 136  
  
 İnin Hata  
  
 Alan Işlem Köprüsü  
  
## <a name="description"></a>Açıklama  
 Bu olay, bir durum makinesi bir olayı işlerken işlenmeyen bir özel durumun oluşturulduğunu gösterir. Olay işlem KIMLIĞI, durum makine adı, geçerli durum, geçmiş, kayıt KIMLIĞI, özel durum, işlem adı ve işlem KIMLIĞI listeler.
