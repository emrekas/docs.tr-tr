# <a name="fatalunexpectedstatemachineevent"></a>FatalUnexpectedStateMachineEvent
Numarasını 137  
  
 İnin Hata  
  
 Alan Işlem Köprüsü  
  
## <a name="description"></a>Açıklama  
 Bu olay bir durum makinesinin beklenmeyen bir olayı işlemeye çalıştı olduğunu gösterir. Olay önemli olarak kabul edildi. Olay işlem KIMLIĞI, durum makinesi, geçerli durum, geçmiş, olay adı, olay ayrıntıları, işlem adı ve işlem KIMLIĞI listeler.
