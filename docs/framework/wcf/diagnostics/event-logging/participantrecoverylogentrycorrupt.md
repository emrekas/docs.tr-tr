# <a name="participantrecoverylogentrycorrupt"></a>ParticipantRecoveryLogEntryCorrupt
Numarasını 138  
  
 İnin Hata  
  
 Alan Işlem Köprüsü  
  
## <a name="description"></a>Açıklama  
 Bu olay katılımcı kurtarma günlüğü girişinin bozuk olduğunu ve seri durumdan çıkarılmadığını gösterir. Veri kaybı bu hatadan kaynaklanabilir. Olayda Işlem KIMLIĞI, kurtarma verileri (Base64 kodlamalı), özel durum, işlem adı ve işlem KIMLIĞI listelenmektedir.
