# <a name="participantrecoverylogentrycorrupt"></a>ParticipantRecoveryLogEntryCorrupt
Kimliği: 138  
  
 Önem derecesi: Hata  
  
 Kategori: TransactionBridge  
  
## <a name="description"></a>Açıklama  
 Bu olay bir katılımcı kurtarma günlük girdisi bozuk ve seri durumdan çıkarılmış hale getirilemedi gösterir. Bu hatadan veri kaybına neden olabilir. Olay listeleri işlem kimliği, kurtarma verileri (şifreli Base64), özel durum, işlem adı ve işlem kimliği
