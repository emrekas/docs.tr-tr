# <a name="unknownlisteneradaptererror"></a>UnknownListenerAdapterError
Kimliği: 161  
  
 Önem derecesi: Hata  
  
 Kategori: SMSvcHost  
  
## <a name="description"></a>Açıklama  
 Bu olay, bir Windows İşlem Etkinleştirme Hizmeti (WAS) bildirim işlenirken beklenmeyen bir hata dinleyicisi bağdaştırıcısı oluştuğunu gösterir. İşlem sonlandırıldı. Protokol, özel durum, işlem adı ve işlem kimliği, olay listeler
