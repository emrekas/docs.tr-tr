|||  
|-|-|  
|Kimlik|3408|  
|anahtar sözcükler|HTTP|  
|Düzey|Ayrıntılı|  
|Kanal|Microsoft Windows uygulaması sunucu-uygulamalar/hata ayıklama|  
  
## <a name="description"></a>Açıklama  
 Http ileti işleyicisi gelen isteği zaman uyumsuz olarak işlemeye başlatıldığında bu olay yayılır.  
  
## <a name="message"></a>İleti  
 HTTP ileti işleyicisi, gelen isteği zaman uyumsuz işlenmesine başlandı.
