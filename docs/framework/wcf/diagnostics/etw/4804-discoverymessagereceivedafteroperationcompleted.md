## <a name="properties"></a>Özellikler  
  
|||  
|-|-|  
|Kimlik|4804|  
|anahtar sözcükler|Bulma|  
|Düzey|Bilgiler|  
|Kanal|Microsoft Windows uygulaması sunucu-uygulamalar/hata ayıklama|  
  
## <a name="description"></a>Açıklama  
 Bu olay, bulma ileti işlemi tamamlandıktan sonra alındığında yayınlanır.  
  
## <a name="message"></a>İleti  
 MessageID ile bir %1 ileti = karşılık gelen %3 işlemi tamamlandıktan sonra olduğundan, '%2' tarafından DiscoveryClient bırakıldı.
