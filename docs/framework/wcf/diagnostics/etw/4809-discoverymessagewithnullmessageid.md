|||  
|-|-|  
|Kimlik|4809|  
|anahtar sözcükler|Bulma|  
|Düzey|Uyarı|  
|Kanal|Microsoft Windows uygulaması sunucu-uygulamalar/hata ayıklama|  
  
## <a name="description"></a>Açıklama  
 Gerekli MessageID özelliği ileti üst bilgisi içermiyordu bulma ileti bırakıldı, bu olay yayılır.  
  
## <a name="message"></a>İleti  
 Bir %1 ileti, ileti üst bilgisi gerekli MessageID özelliği içermediğinden atıldı.
