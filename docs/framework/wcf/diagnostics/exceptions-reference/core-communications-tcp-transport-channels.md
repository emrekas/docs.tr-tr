# <a name="core-communications-tcp-transport-channels"></a>Temel İletişimler: TCP Taşıma Kanalları
Bu konu, TCP taşıma kanalları tarafından oluşturulan tüm özel durumları listeler.  
  
## <a name="exception-list"></a>Özel durum listesi  
  
|Kaynak kodu|Kaynak dizesi|  
|-------------------|---------------------|  
|SocketCloseReadTimeout|Belirtilen yuvanın uzak uç nokta belirtilen ayrılan süre içinde bir kapatma isteğini yanıtlamadı. Bu, uzak uç nokta Kapat alma işlemi Receive işlevinden EOF sinyalini (null) aldıktan sonra çağrısı yapmıyor mümkündür. Bu işlem için ayrılan süre daha uzun bir zaman aşımı değerinin bir bölümü olabilir.|
