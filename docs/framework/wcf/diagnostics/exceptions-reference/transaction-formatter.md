# <a name="transaction-formatter"></a>İşlem Biçimlendirici
Bu konu, işlem biçimlendirici tarafından oluşturulan tüm özel durumları listeler.  
  
## <a name="exception"></a>Özel Durum  
  
|Kaynak kodu|Kaynak dizesi|  
|-------------------|---------------------|  
|TMCommunicationError|Dağıtılmış İşlem Yöneticisi ile iletişim kurulurken bir hata oluştu.|  
|WhereaboutsNoHostName|MSDTC konum bilgilerini bir ana bilgisayar adını içermiyordu.|  
|WhereaboutsReadFailed|MSDTC konum bilgilerini seri olmayan hale getirilemedi.|  
|WhereaboutsSignatureMissing|Standart konumu imzası MSDTC konum bilgileri eksik.|  
|WsatUriCreationFailed|Kayıt Hizmeti adresi MSDTC konum bilgileri oluşturulamadı.|
