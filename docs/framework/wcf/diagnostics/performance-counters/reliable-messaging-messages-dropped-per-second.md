# <a name="reliable-messaging-messages-dropped-per-second"></a>Saniyede Bırakılan Güvenli Mesajlaşma İletileri
Sayaç adı: Saniye başına bırakılan güvenilir Mesajlaşma oturumları.  
  
## <a name="description"></a>Açıklama  
 Bu hizmet, bir saniye içinde bırakılmış olan güvenilir Mesajlaşma iletileri toplam sayısı.  
  
 Bu sayaç performans sayacı türüdür [PERF_COUNTER_COUNTER](https://go.microsoft.com/fwlink/?LinkID=94649), değeri aşağıdaki formül kullanılarak hesaplanır.  
  
 (1 - N 0 N) / ((D 1 - D 0) / F)
