# <a name="queued-poison-messages-per-second"></a>Saniyede Kuyruğa Alınan Zehirli İleti
Sayaç adı: Saniyede kuyruğa alınmış zehirli iletiler.  
  
## <a name="description"></a>Açıklama  
 Bir saniyede sıraya alınan aktarım sırasında bu hizmeti tarafından zarar görmüş işaretlenmiş iletilerinin sayısı.  
  
 Bu sayaç performans sayacı türüdür [PERF_COUNTER_COUNTER](https://go.microsoft.com/fwlink/?LinkID=94649), değeri aşağıdaki formül kullanılarak hesaplanır.  
  
 (1 - N 0 N) / ((D 1 - D 0) / F)
