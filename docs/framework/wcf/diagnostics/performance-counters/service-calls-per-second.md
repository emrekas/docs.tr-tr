## <a name="description"></a>Açıklama  
 Bu hizmete bir saniye içinde çağrı sayısı.  
  
 Bu sayaç performans sayacı türüdür [PERF_COUNTER_COUNTER](https://go.microsoft.com/fwlink/?LinkID=94649), değeri aşağıdaki formül kullanılarak hesaplanır.  
  
 (1 - N 0 N) / ((1 D -D 0) / F) (N) pay tıklarının sayısını geçen son örnek zaman aralığı boyunca paydası (D) temsil son örnek aralığı sırasında gerçekleştirilen işlemlerin sayısını temsil eder ve F işaretleri sıklığını olduğu yerde.
