# <a name="calls-duration"></a>Çağrı Süresi
Sayaç adı: Çağrı Süresi  
  
## <a name="description"></a>Açıklama  
 Bu işlem çağrıları ortalama süresi. Ortalama süresi Bu denklemi temel alınarak hesaplanır: (N1-N0)/(D1-D0).  
  
> [!WARNING]
>  Zaman uyumsuz bir WCF Hizmeti üzerinde kullanıldığında bir çağrı süresi sayacı her zaman -1 döndürür.
