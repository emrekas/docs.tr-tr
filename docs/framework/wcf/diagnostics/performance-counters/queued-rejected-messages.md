# <a name="queued-rejected-messages"></a>Kuyruğa Alınan Reddedilmiş İletiler
Sayaç adı: Kuyruğa alınmış iletiler reddetti.  
  
## <a name="description"></a>Açıklama  
 Sıraya alınan aktarım sırasında bu hizmeti tarafından reddedilen ileti sayısı.  
  
 Bkz: [Poison ileti işleme](https://go.microsoft.com/fwlink/?LinkID=96233) iletileri ne zaman reddedilir hakkında daha fazla bilgi.
