# <a name="queue-dropped-messages"></a>Kuyruktan Bırakılan İletiler
Sayaç adı: Sıraya alınan iletileri bırakıldı.  
  
## <a name="description"></a>Açıklama  
 Sıraya alınan aktarım sırasında bu hizmeti tarafından bırakılan ileti sayısı.  
  
 Bkz: [Poison ileti işleme](https://go.microsoft.com/fwlink/?LinkId=96233) iletileri ne zaman bırakılan hakkında daha fazla bilgi.
