# <a name="endpoint-calls-faulted"></a>Uç noktası: Hatalı Çağrılar
Sayaç adı: Hatalı çağrılar.  
  
## <a name="description"></a>Açıklama  
 Bu uç noktaya yönelik döndürmüş çağrı sayısı. Windows Communication Foundation (WCF) uygulamaları, hizmet yöntemleri işleme hata bilgilerini SOAP hata iletileri kullanarak iletişim kurar. SOAP hatalarının bir hizmet işlemi için meta veriler bulunur ve bu nedenle istemcilerin yürütülmesi daha sağlam ve etkileşimli hale getirmek için kullanabileceği bir hataya sözleşmesi oluşturma iletisi türleridir. SOAP hataları XML formundaki istemcilere ifade olduğundan, yüksek düzeyde birlikte çalışabilir.
