# <a name="systemservicemodeltxasyncabort"></a>System.ServiceModel.TxAsyncAbort
Belirtilen işlem zaman uyumsuz olarak durduruldu.  
  
## <a name="description"></a>Açıklama  
 Geçerli işlem, başka bir katılımcı iptal için gerçekleşen zaman aşımları oylama veya bir işlem katılımcı içinde başka bir hata nedeniyle durduruldu.  
  
## <a name="troubleshooting"></a>Sorun giderme  
 Bu iptal gerçek iptal nedenini belirlemek için beklenmeyen ise tüm sistem günlüklerini kontrol edin.
