# <a name="systemservicemodeltxcompletionstatuscompletedforasyncabort"></a>System.ServiceModel.TxCompletionStatusCompletedForAsyncAbort
Belirtilen işlem için belirtilen işlem nedeniyle zaman uyumsuz iptal tamamlandı.  
  
## <a name="description"></a>Açıklama  
 Geçerli işlem, başka bir katılımcı iptal için gerçekleşen zaman aşımları oylama veya bir işlem katılımcı içinde başka bir hata nedeniyle durduruldu.  
  
## <a name="troubleshooting"></a>Sorun giderme  
 Bu iptal beklenmedikse gerçek iptal nedenini belirlemek için tüm sistem günlüklerini kontrol edin.
