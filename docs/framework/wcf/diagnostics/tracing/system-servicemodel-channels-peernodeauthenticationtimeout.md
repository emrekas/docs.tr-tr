# <a name="systemservicemodelchannelspeernodeauthenticationtimeout"></a>System.ServiceModel.Channels.PeerNodeAuthenticationTimeout
Belirtilen süre sınırı içinde güvenlik el sıkışması başarısız oldu.  
  
## <a name="description"></a>Açıklama  
 Bu izleme, güvenli komşu bağlantısı kurmaya çalışırken ortaya çıkar.  
  
## <a name="troubleshooting"></a>Sorun giderme  
 Tüm Komşuları uygun güvenlik kimlik bilgilerine sahip olması ve 1 dakikadan az tüm iletileri cevaplayin emin olun.
