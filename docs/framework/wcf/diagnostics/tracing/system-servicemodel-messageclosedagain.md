# <a name="systemservicemodelmessageclosedagain"></a>System.ServiceModel.MessageClosedAgain
System.ServiceModel.MessageClosedAgain  
  
## <a name="description"></a>Açıklama  
 Bir ileti yeniden kapatıldı.  
  
 Bir ileti yalnızca bir kez kapatılmalıdır. Kullanıcı uzantı kodunda yayılan bu izleme, kullanıcı uzantı kodu zaten kapatılmış olan bir ileti kapatmakta olduğunu gösterir. Ürün kodu yayılan bu izleme, kullanıcı uzantı kodu potansiyel olarak bir ileti çok erken kapatma olması olduğunu gösterir.
