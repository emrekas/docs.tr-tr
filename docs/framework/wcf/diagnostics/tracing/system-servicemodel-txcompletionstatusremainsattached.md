# <a name="systemservicemodeltxcompletionstatusremainsattached"></a>System.ServiceModel.TxCompletionStatusRemainsAttached
Belirtilen işlem için belirtilen işlem için InstanceContext iliştirilmiş olarak kalır.  
  
## <a name="description"></a>Açıklama  
 Bir ServiceOperation tamamlanır ancak işlemi tamamlamak için ayarlanmadı izlenen.
