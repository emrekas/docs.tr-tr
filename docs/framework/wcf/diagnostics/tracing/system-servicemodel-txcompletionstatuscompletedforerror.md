# <a name="systemservicemodeltxcompletionstatuscompletedforerror"></a>System.ServiceModel.TxCompletionStatusCompletedForError
Belirtilen işlem için belirtilen işlem yürütme işlenmeyen bir özel durum nedeniyle tamamlandı.  
  
## <a name="description"></a>Açıklama  
 Geçerli işlemi tamamlamak için girişimi sırasında bir hata oluştuğunda izlenen. Bir yanıt önce böyle veya hata çağırana gönderilir.  
  
## <a name="troubleshooting"></a>Sorun giderme  
 Özel durum iletisi ve herhangi bir eyleme dönüştürülebilir öğe için izlenen iletisini inceleyin.
