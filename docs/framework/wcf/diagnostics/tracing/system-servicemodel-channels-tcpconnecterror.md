# <a name="systemservicemodelchannelstcpconnecterror"></a>System.ServiceModel.Channels.TcpConnectError
TCP bağlantı işlemi başarısız oldu.  
  
## <a name="description"></a>Açıklama  
 Bu uyarı düzeyi izleme bir TCP uç noktasına bağlanmak için bir hata gösterir. Belirtilen IP adresi ve bağlantı noktasında uzak uç noktanın yanıt vermiyorsa bu durum gerçekleşebilir. Bu izleme, sonraki yoksayılabilir bağlanmaya başka bir geçerli IP adresleri (IPv4 veya IPv6 adresleri veya gibi belirli bir ana bilgisayar adını temsil eden diğer IP adresleri) başarısız. Özel durum'içinde bu izleme, hata hakkında ek bilgiler ortaya çıkarabilir.
