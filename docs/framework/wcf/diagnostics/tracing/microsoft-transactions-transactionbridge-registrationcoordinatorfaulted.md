# <a name="microsofttransactionstransactionbridgeregistrationcoordinatorfaulted"></a>Microsoft.Transactions.TransactionBridge.RegistrationCoordinatorFaulted
WS-AT protokolü hizmetini düzenleyicisiyle bir kayıt iletiye yanıt olarak bir hata aldı.  
  
## <a name="description"></a>Açıklama  
 Yerel TransactionManager, üstün TransactionManager döndürülen bir hata nedeniyle kaydı mümkün değilse, izlenen.  
  
## <a name="troubleshooting"></a>Sorun giderme  
 Döndürülen hata izleme iletisi inceleyin.
