# <a name="systemservicemodelchannelspeerneighborstatechangefailed"></a>System.ServiceModel.Channels.PeerNeighborStateChangeFailed

## <a name="description"></a>Açıklama  
 Bu izleme, bir komşu'nın durumunu değiştirmek için başarısız bir girişimden sonra gerçekleşir. Bu, bir beklenmeyen bir iç hata veya bozuk bellek belirtir.
