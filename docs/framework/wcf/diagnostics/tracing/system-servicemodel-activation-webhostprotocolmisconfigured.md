# <a name="systemservicemodelactivationwebhostprotocolmisconfigured"></a>System.ServiceModel.Activation.WebHostProtocolMisconfigured
Windows İşlem Etkinleştirme Hizmeti (WAS) protokolü yanlış yapılandırılmış.  
  
## <a name="description"></a>Açıklama  
 Aktarım Protokolü hatalı yapılandırıldığında, bu hata oluşur.  
  
## <a name="troubleshooting"></a>Sorun giderme  
 Denetleyin ve taşıma web.config dosyasına yapılandırma parametrelerini düzeltin.
