# <a name="systemservicemodelchannelspeerneighbornotaccepted"></a>System.ServiceModel.Channels.PeerNeighborNotAccepted
Rovnocenný soused kabul edilmedi.  
  
## <a name="description"></a>Açıklama  
 Bu izleme, bir komşu bağlantı işlenirken gerçekleşir. Olası nedenleri eşleşmeyen güvenlik ayarları, zaman aşımı aşıldı, bir bağlantı veya geçici ağ sorunları kabul etmeyen bir komşu.  
  
## <a name="troubleshooting"></a>Sorun giderme  
 Eşleşmeyen güvenlik ayarları, zaman aşımı aşıldı, komşu'nın bir bağlantı veya geçici ağ sorunlarından kabul etme becerisinin denetleyin.
