# <a name="microsofttransactionstransactionbridgeenlisttransactionfailure"></a>Microsoft.Transactions.TransactionBridge.EnlistTransactionFailure
Sağlanan düzenleme bağlamı kullanarak bir işleme kaydolunamadı WS-AT protokolü hizmeti başarısız oldu.  
  
## <a name="description"></a>Açıklama  
 MSDTC verilen 2pc protokolü için bir işlem üzerinde listeleme işlenemediğinde izlenen.  İşlem artık yok, kaydetme artık izin verilmez veya çok fazla sayıda kayıtlar zaten mevcut olduğundan bu durum ortaya çıkabilir.  
  
## <a name="troubleshooting"></a>Sorun giderme  
 Eyleme dönüştürülebilir herhangi bir öğe olup olmadığını belirlemek için izleme iletisi durumu dizede inceleyin.
