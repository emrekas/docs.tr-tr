# <a name="systemservicemodelcommunicationobjectabortfailed"></a>System.ServiceModel.CommunicationObjectAbortFailed
İletişim nesnesi iptal edilemedi.  
  
## <a name="description"></a>Açıklama  
 İletişim nesnesi, kendi iptal metodu yürütülürken bir özel durum oluştu çünkü durdurma başarısız oldu. İzleme yakındaki başka bir özel durumun ayrıntılarını içerebilir.
