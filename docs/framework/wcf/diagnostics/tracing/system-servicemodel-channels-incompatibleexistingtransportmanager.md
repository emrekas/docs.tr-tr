# <a name="systemservicemodelchannelsincompatibleexistingtransportmanager"></a>System.ServiceModel.Channels.IncompatibleExistingTransportManager
Mevcut bir uyumsuz aktarım Yöneticisi, belirtilen URI için bulunamadı.  
  
## <a name="description"></a>Açıklama  
 Bu bilgilendirici izleme, eşleşen bir kayıt açıldı, ancak mevcut aktarım Yöneticisi'ni uyumsuz dinleyiciyi bulunamadığını gösterir. Dinleyici başka bir aktarım Yöneticisi'ni kullanın ve başarısız olursa bir özel durum dener.
