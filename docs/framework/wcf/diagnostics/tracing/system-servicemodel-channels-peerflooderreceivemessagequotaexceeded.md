# <a name="systemservicemodelchannelspeerflooderreceivemessagequotaexceeded"></a>System.ServiceModel.Channels.PeerFlooderReceiveMessageQuotaExceeded
İletilerin gelen alma hızı çok yüksektir.  
  
## <a name="description"></a>Açıklama  
 Bu izleme, gelen iletileri işlemek çalışılırken oluşur. Bu komşu için belirlediğiniz kotayı aştığı için bir ileti için belirli bir komşu iletilemedi. Bekleyen iletiler, bir biriktirme listesi için bir komşu temizlemek bir yanıt vermeyen komşu başarısız olduğunda gerçekleşir.  
  
## <a name="troubleshooting"></a>Sorun giderme  
 Kafes içinde gönderilen iletileri oranı azaltın.
