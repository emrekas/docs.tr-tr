# <a name="systemservicemodeltxreleaseserviceinstanceoncompletion"></a>System.ServiceModel.TxReleaseServiceInstanceOnCompletion
Hizmet örneği, işlem tamamlandığında yayımlanan '{0}' ReleaseServiceInstanceOnTransactionComplete ServiceBehaviorAttribute ayarlanmış olduğu için true.  
  
## <a name="description"></a>Açıklama  
 Mevcut hizmet örneğinin yayınlandığı veya geçerli etkin işlem tamamlandığında nedeniyle atıldı ve ReleaseServiceInstanceOnTransactionComplete ayarlandığında izlenen `true`.
