# <a name="systemservicemodeltxcompletionstatuscompletedforsetcomplete"></a>System.ServiceModel.TxCompletionStatusCompletedForSetComplete
Belirtilen işlem için belirtilen işlem nedeniyle SetTransactionComplete çağrısı tamamlandı.  
  
## <a name="description"></a>Açıklama  
 Kullanıcı OperationContext aracılığıyla geçerli işlemi tamamlamak açıkça karar verdiyse izlenen.
