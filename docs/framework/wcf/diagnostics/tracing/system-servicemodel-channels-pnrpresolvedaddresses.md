# <a name="systemservicemodelchannelspnrpresolvedaddresses"></a>System.ServiceModel.Channels.PnrpResolvedAddresses
Pnrp Çözümleyicisi çözümleme işlemi tamamlandı.  
  
## <a name="description"></a>Açıklama  
 Bu uyarı izleme hiçbir adresine Pnrp çözümleyici bulunamadı gösterir. Bu izleme, PNRP çözümleyici tarafından çözme işlemi sırasında oluşur.
