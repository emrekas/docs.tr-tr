# <a name="systemservicemodelservicehosttimeoutonclose"></a>System.ServiceModel.ServiceHostTimeoutOnClose
System.ServiceModel.ServiceHostTimeoutOnClose  
  
## <a name="description"></a>Açıklama  
 Bir ServiceHost kapatma işlemini zaman aşımına uğradı. Bu, bazı oturumdaki kanallar'ın düzgün bir şekilde istemci tarafından kapatılan değil ve kesilmiş olabilir gösterir. Kapatma zaman aşımını ayarlamak için varsayılan zaman değiştirin.
