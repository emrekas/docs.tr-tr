# <a name="systemservicemodeltxcompletionstatusabortedonsessionclose"></a>System.ServiceModel.TxCompletionStatusAbortedOnSessionClose
Bu oturumu kapatıldı ve TransactionAutoCompleteOnSessionClose OperationBehaviorAttribute false olarak ayarlanmış tamamlanmamış olduğundan belirtilen işlem durduruldu.  
  
## <a name="description"></a>Açıklama  
 Geçerli etkin oturum kapatıldığında, işlem tamamlanmadı ve TransactionAutoCompleteOnSessionClose ayarlandığında izlenen `false`.  
  
## <a name="troubleshooting"></a>Sorun giderme  
 Bu izleme araştırılması gereken olası bir uygulama hatası gösterir.
