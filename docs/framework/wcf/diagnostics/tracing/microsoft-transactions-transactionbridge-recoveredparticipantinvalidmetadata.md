# <a name="microsofttransactionstransactionbridgerecoveredparticipantinvalidmetadata"></a>Microsoft.Transactions.TransactionBridge.RecoveredParticipantInvalidMetadata
Katılımcı kurtarma günlüğü girişinde bulunan uç nokta başvurusu geçersiz ya da uyumsuz meta veri içeriyor.  
  
## <a name="description"></a>Açıklama  
 Sistem, kurtarılan bir katılımcı listesi için proxy oluşturulamıyor.  
  
## <a name="troubleshooting"></a>Sorun giderme  
 Sorun devam ederse, Microsoft ile iletişime geçin.
