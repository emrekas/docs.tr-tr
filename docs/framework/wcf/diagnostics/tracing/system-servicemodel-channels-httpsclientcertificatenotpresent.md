# <a name="systemservicemodelchannelshttpsclientcertificatenotpresent"></a>System.ServiceModel.Channels.HttpsClientCertificateNotPresent
İstemci sertifikası gereklidir. Sertifika, istekte bulundu.  
  
## <a name="description"></a>Açıklama  
 Bu izleme, HTTPS dinleyicisi bir istemci sertifikası ile ilişkilendirilmemiş bir HTTPS isteği aldığını gösterir. Dinleyici tüm HTTPS isteklerinde istemci sertifikaları gerektirmek için yapılandırılıldıktan sonra dinleyici istemcinin orijinallik sertifikası doğrulanamadı.
