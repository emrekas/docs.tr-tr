# <a name="systemservicemodelchannelshttpsclientcertificateinvalid"></a>System.ServiceModel.Channels.HttpsClientCertificateInvalid
İstemci sertifikası geçersiz.  
  
## <a name="description"></a>Açıklama  
 Bu izleme, HTTPS dinleyicisi tarafından olan istemci tarafından sağlanan sertifika geçersiz bulundu gösterir. Bu sertifikayı kullanarak istemci özgünlüğünü doğrulamak HTTPS dinleyicisi çalışıyordu. Sertifika, Sertifika Yetkilisi hizmetini barındıran sunucu tarafından tanınmıyorsa geçersiz olabilir.
