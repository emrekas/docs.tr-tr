# <a name="systemservicemodelmanualflowthrottlelimitreached"></a>System.ServiceModel.ManualFlowThrottleLimitReached
System.ServiceModel.ManualFlowThrottleLimitReached  
  
## <a name="description"></a>Açıklama  
 Sistem, ManualFlowControlLimit kısıtlama için ayarladığı sınırına ulaştı. Kısıtlama değeri, geçerli olduğu ServiceHost veya InstanceContext, ManualFlowControlLimit özelliğini değiştirerek değiştirilebilir.  
  
 Bu izleme, el ile akış denetimi sınırına 0 olarak başlangıçta azaltıldığında yayınlanır. 0 sonraki değişiklikleri izlenen değil. Akış denetimi sınırına örnek bağlamı, her bağlam için bir kez izlenen.
