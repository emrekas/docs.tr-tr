# <a name="microsofttransactionstransactionbridgeregistrationcoordinatorfailed"></a>Microsoft.Transactions.TransactionBridge.RegistrationCoordinatorFailed
WS-AT protokolü hizmetini ilgili düzenleyiciye bir kayıt iletisi gönderilemedi  
  
## <a name="description"></a>Açıklama  
 Yerel TransactionManager, üstün TransactionManager ileti göndermek için bağlantı kurma sorunu nedeniyle kaydı mümkün değilse, izlenen.  
  
## <a name="troubleshooting"></a>Sorun giderme  
 İleti gönderme hatası neden olan özel durumu için izleme iletisi inceleyin
