# <a name="systemservicemodelactivationwebhostserviceclosefailed"></a>System.ServiceModel.Activation.WebHostServiceCloseFailed
Bir hizmet düzgün bir şekilde kapatıldı ve iptal gerçekleşir.  
  
## <a name="description"></a>Açıklama  
 Bu hata kodu yalnızca günlük dosyasında görünür. Hizmet durdurma zaten çağrıldıktan sonra kapatmak çalıştığınızda, genellikle bir programlama hatası gibi gösterir.  
  
## <a name="troubleshooting"></a>Sorun giderme  
 Uygulama kaynak kodunu kontrol edin.
