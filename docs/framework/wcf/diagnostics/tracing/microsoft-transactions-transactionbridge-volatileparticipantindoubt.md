# <a name="microsofttransactionstransactionbridgevolatileparticipantindoubt"></a>Microsoft.Transactions.TransactionBridge.VolatileParticipantInDoubt
WS-AT protokolü hizmeti, tanınmayan bir geçici katılımcıdan Prepared ya da yeniden yürütme iletisi aldı. Bir hata şüpheli hareketin sonucu olduğunu bildirir, katılımcı döndürüldü.  
  
## <a name="description"></a>Açıklama  
 Yerel hareket yöneticisi zaten unutmuş bir geçici liste Prepared ya da yeniden yürütme iletisi aldığında izlenen.  
  
## <a name="troubleshooting"></a>Sorun giderme  
 Olası nedenler, geç geçici katılımcıdan gelen iletileri inceleyin.
