# <a name="operating-system-resources-required-by-wcf"></a>WCF Tarafından Gerektirilen İşletim Sistemi Kaynakları
Windows Communication Foundation (WCF), işlev işletim sistemi tarafından sağlanan çeşitli kaynaklar bağlıdır. Aşağıdaki tabloda, bu kaynakları listeler.  
  
|Kaynak|Açıklama|  
|--------------|-----------------|  
|Microsoft Dağıtılmış İşlem Düzenleyicisi (MSDTC)|OleTx işlemleri desteklemek için gereklidir.|  
|Internet Information Services (IIS)|IIS uygulamanızı barındırmak için kullanmak istiyorsanız gereklidir.|  
|Windows İşlem Etkinleştirme Hizmeti (WAS)|WAS uygulamanızı barındırmak için kullanmak istiyorsanız gereklidir.|
