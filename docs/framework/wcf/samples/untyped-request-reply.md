 <span data-ttu-id="123f1-105">Bu örnek, [Başlarken](../../../../docs/framework/wcf/samples/getting-started-sample.md)' i temel alır.</span><span class="sxs-lookup"><span data-stu-id="123f1-105">This sample is based on the [Getting Started](../../../../docs/framework/wcf/samples/getting-started-sample.md).</span></span> <span data-ttu-id="123f1-106">Hizmet sözleşmesi bir ileti türünde bağımsız değişken olarak alan ve bir ileti döndüren bir işlemi tanımlar.</span><span class="sxs-lookup"><span data-stu-id="123f1-106">The service contract defines one operation that takes in a message type as an argument and returns a message.</span></span> <span data-ttu-id="123f1-107">İşlem, ileti gövdesinden toplamı hesaplamak için gerekli tüm verileri toplar ve sonra toplamı dönüş iletisinde gövde olarak gönderir.</span><span class="sxs-lookup"><span data-stu-id="123f1-107">The operation collects all required data to compute the sum from the message body and then sends the sum as body in the return message.</span></span>

```csharp
[OperationContract(Action = CalculatorService.RequestAction, ReplyAction = CalculatorService.ReplyAction)]  
Message ComputeSum(Message request);  
```  
  
 <span data-ttu-id="123f1-108">Hizmette, işlem giriş iletisinde geçirilen tamsayıların dizisini alır ve toplamı hesaplar.</span><span class="sxs-lookup"><span data-stu-id="123f1-108">On the service, the operation retrieves the array of integers passed in the input message and then computes the sum.</span></span> <span data-ttu-id="123f1-109">Yanıt iletisi göndermek için örnek, uygun ileti sürümü ve eylemiyle yeni bir ileti oluşturur ve hesaplanan toplamı gövde olarak ekler.</span><span class="sxs-lookup"><span data-stu-id="123f1-109">To send a response message, the sample creates a new message with the appropriate message version and Action and adds the computed sum as its body.</span></span> <span data-ttu-id="123f1-110">Aşağıdaki örnek kod bunu gösterir.</span><span class="sxs-lookup"><span data-stu-id="123f1-110">The following sample code demonstrates this.</span></span>  
  
```csharp
public Message ComputeSum(Message request)  
{  
    //The body of the message contains a list of numbers which will be   
    //read as a int[] using GetBody<T>  
    int result = 0;  
  
    int[] inputs = request.GetBody<int[]>();  
    foreach (int i in inputs)  
    {  
        result += i;  
    }  
  
    Message response = Message.CreateMessage(request.Version,   
                                      ReplyAction, result);  
    return response;  
}  
```  
  
 <span data-ttu-id="123f1-111">İstemci, uzak hizmete bir ara sunucu oluşturmak için [ServiceModel meta veri yardımcı programı Aracı (Svcutil. oluşturduğu proxy kanalında yeni bir kapsam oluşturur.</span><span class="sxs-lookup"><span data-stu-id="123f1-113">Thus, it creates a new <xref:System.ServiceModel.OperationContextScope> scoped to data-stu-id="123f1-114">The client passes an input array as the body to the request message and then invokes the `ComputeSum` on the proxy.</span></span> <span data-ttu-id="123f1-115">İstemci daha sonra yanıt iletisindeki `GetBody<T>` yöntemine erişerek, geçirildiği girişlerin toplamını alır.</span><span class="sxs-lookup"><span data-stu-id="123f1-115">The client then retrieves the sum of the inputs it passed by accessing the `GetBody<T>` method on the reply message.</span></span> <span data-ttu-id="123f1-116">Aşağıdaki örnek kod bunu gösterir.</span><span class="sxs-lookup"><span data-stu-id="123f1-116">The following sample code demonstrates this.</span></span>  
  
```csharp
using (new OperationContextScope(client.InnerChannel))  
{  
    // Call the Sum service operation.  
    int[] values = { 1, 2, 3, 4, 5 };  
    Message request = Message.CreateMessage(  
        OperationContext.Current.OutgoingMessageHeaders.MessageVersion,   
        RequestAction, values);  
    Message reply = client.ComputeSum(request);  
    int response = reply.GetBody<int>();  
  
    Console.WriteLine("Sum of numbers passed (1,2,3,4,5) = {0}",   
                                                       response);  
}  
```
