## <a name="enabling-port-sharing"></a><span data-ttu-id="1fd1d-123">Bağlantı noktası paylaşımı etkinleştirme</span><span class="sxs-lookup"><span data-stu-id="1fd1d-123">Enabling Port Sharing</span></span>  
 <span data-ttu-id="1fd1d-124">Aşağıdaki kod, sunucuda etkinleştirme bağlantı noktası paylaşımı gösterir.</span><span class="sxs-lookup"><span data-stu-id="1fd1d-124">The following code demonstrates enabling port sharing on the server.</span></span> <span data-ttu-id="1fd1d-125">Bir örneğini başlatır `ICalculator` hizmet sabit bir bağlantı noktası ile rastgele bir URI yolu.</span><span class="sxs-lookup"><span data-stu-id="1fd1d-125">It starts an instance of the `ICalculator` service on a fixed port with a random URI path.</span></span> <span data-ttu-id="1fd1d-126">İki hizmet de aynı bağlantı noktasını paylaşabilirsiniz olsa da, böylece NetTcp bağlantı noktası paylaşma hizmeti için doğru uygulama iletileri yönlendirebilirsiniz genel uç nokta adresleri hala benzersiz olması gerekir.</span><span class="sxs-lookup"><span data-stu-id="1fd1d-126">Even though two services can share the same port, their overall endpoint addresses still must be unique so that the NetTcp Port Sharing Service can route messages to the correct application.</span></span>  

```csharp
// Configure a binding with TCP port sharing enabled  
NetTcpBinding binding = new NetTcpBinding();  
binding.PortSharingEnabled = true;  
  
// Start a service on a fixed TCP port  
ServiceHost host = new ServiceHost(typeof(CalculatorService));  
ushort salt = (ushort)new Random().Next();  
string address = $"net.tcp://localhost:9000/calculator/{salt}";
host.AddServiceEndpoint(typeof(ICalculator), binding, address);  
host.Open();  
```

## <a name="running-the-sample"></a><span data-ttu-id="1fd1d-129">Örneği çalıştırma</span><span class="sxs-lookup"><span data-stu-id="1fd1d-129">Running the Sample</span></span>  
 <span data-ttu-id="1fd1d-130">İletiler için bağlantı noktası paylaşımı hizmetleri doğru yönlendirilir denetlemek için test İstemcisi'ni kullanabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="1fd1d-130">You can use the test client to check that messages are correctly routed to services sharing the port.</span></span>  

```csharp
class client  
{  
   static void Main(string[] args)  
   {  
      Console.Write("Enter the service number to test: ");  
      ushort salt = ushort.Parse(Console.ReadLine());  
      string address = $"net.tcp://localhost:9000/calculator/{salt}";
      ChannelFactory<ICalculator> factory = new ChannelFactory<ICalculator>(new NetTcpBinding());  
      ICalculator proxy = factory.CreateChannel(new EndpointAddress(address));  
  
      // Call the Add service operation.  
      double value1 = 100.00D;  
      double value2 = 15.99D;  
      double result = proxy.Add(value1, value2);  
      Console.WriteLine("Add({0},{1}) = {2}", value1, value2, result);  
  
      // Call the Subtract service operation.  
      value1 = 145.00D;  
      value2 = 76.54D;  
      result = proxy.Subtract(value1, value2);  
      Console.WriteLine("Subtract({0},{1}) = {2}", value1, value2, result);  
  
      // Call the Multiply service operation.  
      value1 = 9.00D;  
      value2 = 81.25D;  
      result = proxy.Multiply(value1, value2);  
      Console.WriteLine("Multiply({0},{1}) = {2}", value1, value2, result);  
  
      // Call the Divide service operation.  
      value1 = 22.00D;  
      value2 = 7.00D;  
      result = proxy.Divide(value1, value2);  
      Console.WriteLine("Divide({0},{1}) = {2}", value1, value2, result);  
  
      Console.WriteLine();  
      Console.WriteLine("Press <ENTER> to terminate client.");  
      Console.ReadLine();  
  
      factory.Close();  
   }  
}  
```

 <span data-ttu-id="1fd1d-131">Hizmetin her örneği, kendi benzersiz numarasını ve adresi yazdırır.</span><span class="sxs-lookup"><span data-stu-id="1fd1d-131">Each instance of the service prints out its unique number and address.</span></span> <span data-ttu-id="1fd1d-132">Örneğin, service.exe çalıştırdığınızda, aşağıdaki metni görebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="1fd1d-132">For instance, you may see the following text when you run service.exe.</span></span>
