## <a name="locking-configuration-file-elements"></a><span data-ttu-id="fa210-136">Kilitleme yapılandırma dosyası öğeleri</span><span class="sxs-lookup"><span data-stu-id="fa210-136">Locking configuration file elements</span></span>
