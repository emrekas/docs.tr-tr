 <span data-ttu-id="0476a-112">İstemcide, bir kanal örneği ayarlara göre kanal özelliği iki modda çalışabilir <xref:System.ServiceModel.Channels.IContextManager.Enabled%2A>.</span><span class="sxs-lookup"><span data-stu-id="0476a-112">On the client, an instance of a channel can operate in two modes based on the settings on the channel property, <xref:System.ServiceModel.Channels.IContextManager.Enabled%2A>.</span></span>

<span data-ttu-id="0476a-134">Bu özellik SOAP üst bilgi veya bağlamı, bağlama yapılandırmasına bağlıdır içeren HTTP tanımlama bilgisi dönüştürülür.</span><span class="sxs-lookup"><span  <span data-ttu-id="0476a-136">Bağlam değişimi Protokolü yeni kullanıma sunuyor `wsc:Context` bağlam yaymak için HTTP tanımlama bilgileri kullanılmaz, bağlam bilgilerini temsil etmek için SOAP üstbilgisi.</span><span class="sxs-lookup"><span data-stu-id="0476a-136">The context exchange protocol introduces a new `wsc:Context` SOAP header to represent the context information when HTTP cookies are not used to propagate the context.</span></span> <span data-ttu-id="0476a-137">Herhangi bir sayıda alt öğeleri, her bir dize anahtarı ve dize içerik için içerik üstbilgisi şemasının sağlar.</span><span class="sxs-lookup"><span data-stu-id="0476a-137">The context header schema allows for any number of child elements, each with a string key and string content.</span></span> <span data-ttu-id="0476a-138">Bağlam üst bilgisi bir örnek verilmiştir.</span><span class="sxs-lookup"><span data-stu-id="0476a-138">The following is an example of a context header.</span></span>
