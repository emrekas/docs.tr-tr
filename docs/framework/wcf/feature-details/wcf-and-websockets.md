# <a name="wcf-and-websockets"></a>WCF ve WebSockets
.NET Framework 4.5, Windows Communication Foundation'da WebSockets için destek sunuyor.  WebSockets standart HTTP 80 ve 443 bağlantı noktalarını çift yönlü iletişimi sağlayan bir verimli, standartlara dayalı teknolojisidir. Standart HTTP bağlantı kullanımını WebSockets aracılarla web üzerinden iletişim kurmasına izin verin.  İki yeni standart bağlamalar iletişim WebSocket aktarımı üzerinden desteklemek üzere eklendi. <xref:System.ServiceModel.NetHttpBinding> ve <xref:System.ServiceModel.NetHttpsBinding>. WebSockets özgü ayarlar yapılandırılabilir <xref:System.ServiceModel.Channels.HttpTransportBindingElement> erişerek <xref:System.ServiceModel.Channels.HttpTransportBindingElement.WebSocketSettings%2A> özelliği.
