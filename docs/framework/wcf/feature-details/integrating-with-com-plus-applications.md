# <a name="integrating-with-com-applications"></a>COM Uygulamaları ile Tümleştirme
Windows Communication Foundation (WCF) dağıtılmış uygulamalar oluşturmak için zengin bir ortam sağlar. COM +'da barındırılan bir bileşen tabanlı uygulama mantığı önemli ölçüde yatırımınız varsa, WCF, yeniden yazmak zorunda kalmadan yerine mevcut mantığınızı genişletmek için de kullanabilirsiniz. Bu bölüm içindeki konuları COM + ile WCF nasıl kullanılacağını açıklar.  
  
## <a name="in-this-section"></a>Bu Bölümde  
 [COM+ Uygulamaları ile Tümleştirme Genel Bakış](../../../../docs/framework/wcf/feature-details/integrating-with-com-plus-applications-overview.md)  
 Ne zaman ve nasıl tümleştireceğinizi COM + bileşenleri hakkında genel bir bakış sağlar.  
  
 [Nasıl yapılır: COM + hizmet modeli yapılandırma aracı kullanın](../../../../docs/framework/wcf/feature-details/how-to-use-the-com-service-model-configuration-tool.md)  
 COM + hizmet modeli yapılandırma komut satırı aracı (ComSvcConfig.exe), WCF hizmetleri kullanıma sunulan istediğiniz uygulama arabirimi yapılandırmak için kullanmayı açıklar.  
  
 [Nasıl yapılır: COM + hizmet ayarlarını yapılandırma](../../../../docs/framework/wcf/feature-details/how-to-configure-com-service-settings.md)  
 Bir WCF hizmeti olarak bir COM nesnesi yapılandırma açıklanmaktadır.  
  
 [Nasıl yapılır: COM + tümleştirme uygulaması dağıtma](../../../../docs/framework/wcf/feature-details/how-to-deploy-a-com-integration-application.md)  
 COM + tümleştirme uygulaması taşınacağı açıklanmaktadır.
