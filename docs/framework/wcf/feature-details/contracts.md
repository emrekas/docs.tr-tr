## <a name="service-contracts"></a><span data-ttu-id="ef54b-118">Hizmet Sözleşmeleri</span><span class="sxs-lookup"><span data-stu-id="ef54b-118">Service Contracts</span></span>  
 <span data-ttu-id="ef54b-119">Bir hizmet sözleşmesini hakkında bilgi sağlayan bir açıklamadır:</span><span class="sxs-lookup"><span data-stu-id="ef54b-119">A service contract is a statement that provides information about:</span></span>  
  
- <span data-ttu-id="ef54b-120">Bir hizmet işlemlerinde gruplandırmasıdır.</span><span class="sxs-lookup"><span data-stu-id="ef54b-120">The grouping of operations in a service.</span></span>

- <span data-ttu-id="ef54b-122">Bu iletileri veri türleri.</span><span class="sxs-lookup"><span data-stu-id="ef54b-122">The data types of these messages.</span></span>  
  
- <span data-ttu-id="ef54b-123">İşlemleri konumu.</span><span class="sxs-lookup"><span data-stu-id="ef54b-123">The location of the operations.</span></span>

 <span data-ttu-id="ef54b-125">Örneğin, bir satın alma siparişi sözleşme olabilir bir `CreateOrder` sipariş bilgileri bir girişi kabul eden bir işlem türleri ve siparişi tanımlayıcısı dahil olmak üzere, başarı veya başarısızlık durumu bilgilerini döndürür.</span><span class="sxs-lookup"><span data-stu-id="ef54b-125">For example, a purchase order contract might have a `CreateOrder` operation that accepts an input of order information types and returns success or failure information, including an order identifier.</span></span> <span data-ttu-id="ef54b-126">Ayrıca olabilir bir `GetOrderStatus` işlemi, bir siparişi tanımlayıcısı kabul eden ve sipariş durumu bilgilerini döndürür.</span><span class="sxs-lookup"><span data-stu-id="ef54b-126">It might also have a `GetOrderStatus` operation that accepts an order identifier and returns order status information.</span></span> <span data-ttu-id="ef54b-127">Bu tür bir hizmet sözleşmesini belirtmeniz gerekir:</span><span class="sxs-lookup"><span data-stu-id="ef54b-127">A service contract of this sort would specify:</span></span>

- <span data-ttu-id="ef54b-130">Bu iletiler gerçekleştirebilirsiniz veriler.</span><span class="sxs-lookup"><span data-stu-id="ef54b-130">The data that these messages can carry.</span></span>

### <a name="understanding-the-hierarchy-of-requirements"></a><span data-ttu-id="ef54b-146">Hiyerarşi gereksinimlerini anlama</span><span class="sxs-lookup"><span data-stu-id="ef54b-146">Understanding the Hierarchy of Requirements</span></span>
