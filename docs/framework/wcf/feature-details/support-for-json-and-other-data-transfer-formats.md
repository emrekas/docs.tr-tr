# <a name="support-for-json-and-other-data-transfer-formats"></a>JSON ve Diğer Veri Aktarma Biçimleri için Destek
JSON (JavaScript nesne gösterimi) olduğundan, AJAX (zaman uyumsuz JavaScript ve XML) ile istemci tarayıcıları arasında verileri küçük miktarda hızlı değişimleri sağlayan verimli veri kodlama biçimi-etkin Web Hizmetleri.
