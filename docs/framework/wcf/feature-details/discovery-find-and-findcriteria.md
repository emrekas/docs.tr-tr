## <a name="findresponse"></a><span data-ttu-id="b871b-140">FindResponse</span><span class="sxs-lookup"><span data-stu-id="b871b-140">FindResponse</span></span>  
 <span data-ttu-id="b871b-141"><xref:System.ServiceModel.Discovery.FindResponse> sahip bir <xref:System.ServiceModel.Discovery.FindResponse.Endpoints%2A> içeren bus'dan eşleşen ağ üzerindeki hizmet tarafından gönderilen koleksiyon özelliği.</span><span class="sxs-lookup"><span data-stu-id="b871b-141"><xref:System.ServiceModel.Discovery.FindResponse> has an <xref:System.ServiceModel.Discovery.FindResponse.Endpoints%2A> collection property that contains any replies sent by matching services on the network.</span></span> <span data-ttu-id="b871b-142">Hizmet yanıtlandı, boş bir koleksiyondur.</span><span class="sxs-lookup"><span data-stu-id="b871b-142">If no services replied, the collection is empty.</span></span> <span data-ttu-id="b871b-143">Bir veya daha fazla hizmet yanıt verdi, her yanıt depolanan bir <xref:System.ServiceModel.Discovery.EndpointDiscoveryMetadata> adresi, sözleşme ve hizmetle ilgili bazı ek bilgiler içeren nesne.</span><span class="sxs-lookup"><span data-stu-id="b871b-143">If one or more services replied, each reply is stored in an <xref:System.ServiceModel.Discovery.EndpointDiscoveryMetadata> object, which contains the address, contract, and some additional information about the service.</span></span>  
  
 <span data-ttu-id="b871b-144">Aşağıdaki örnek kodda bir bulma işlemi gösterilmektedir.</span><span class="sxs-lookup"><span data-stu-id="b871b-144">The following example shows how to perform a find operation in code.</span></span>  
  
```  
// Create DiscoveryClient  
DiscoveryClient discoveryClient = new DiscoveryClient(new UdpDiscoveryEndpoint());  
  
// Create FindCriteria  
FindCriteria findCriteria = new FindCriteria(typeof(IPrinterService));  
findCriteria.Scopes.Add(new Uri("http://www.contoso.com/building1/floor1"));  
findCriteria.Duration = TimeSpan.FromSeconds(10);   
  
// Find ICalculatorService endpoints              
FindResponse findResponse = discoveryClient.Find(findCriteria);  
  
Console.WriteLine("Found {0} ICalculatorService endpoint(s).", findResponse.Endpoints.Count)  
```
