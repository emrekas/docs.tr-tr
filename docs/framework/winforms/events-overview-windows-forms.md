 <span data-ttu-id="85ef5-119">Bu olay modeli kullanan *Temsilciler* bunları işlemek için kullanılan yöntemleri için olaylar bağlamak için.</span><span class="sxs-lookup"><span data-stu-id="85ef5-119">This event model uses *delegates* to bind events to the methods that are used to handle them.</span></span> <span data-ttu-id="85ef5-120">Temsilci işleyicisi yöntemi belirterek olay bildirimi için kaydetmek diğer sınıflar sağlar.</span><span class="sxs-lookup"><span data-stu-id="85ef5-120">The delegate enables other classes to register for event notification by specifying a handler method.</span></span> <span data-ttu-id="85ef5-121">Olay gerçekleştiğinde temsilci ilişkili yöntemi çağırır.</span><span class="sxs-lookup"><span data-stu-id="85ef5-121">When the event occurs, the delegate calls the bound method.</span></span> <span data-ttu-id="85ef5-122">Temsilcileri tanımlama hakkında daha fazla bilgi için bkz. [olayları](../../standard/events/index.md).</span><span class="sxs-lookup"><span data-stu-id="85ef5-122">For more information about how to define delegates, see [Events](../../standard/events/index.md).</span></span>

 <span data-ttu-id="85ef5-123">Temsilciler, tek bir yöntemi veya çok noktaya yayın başvurulan birden çok yöntem bağlanabilir.</span><span class="sxs-lookup"><span data-stu-id="85ef5-123">Delegates can be bound to a single method or to multiple methods, referred to as multicasting.</span></span> <span data-ttu-id="85ef5-124">Bir olay için temsilci oluştururken siz (veya Windows) genellikle çok noktaya yayın olay oluşturursunuz.</span><span class="sxs-lookup"><span data-stu-id="85ef5-124">When creating a delegate for an event, you (or the Windows) typically create a multicast event.</span></span> <span data-ttu-id="85ef5-125">Nadir bir özel durum, birden çok kez olay mantıksal olarak tekrarlayın değil belirli yordam (örneğin, bir iletişim kutusunu görüntüleme) sonuçlanır bir olay olabilir.</span><span class="sxs-lookup"><span data-stu-id="85ef5-125">A rare exception might be an event that results in a specific procedure (such as displaying a dialog box) that would not logically repeat multiple times per event.</span></span> <span data-ttu-id="85ef5-126">Çok noktaya yayın temsilci oluşturma hakkında daha fazla bilgi için bkz: [nasıl yapılır: Temsilcileri (çok noktaya yayın temsilcileri) birleştirme](~/docs/csharp/programming-guide/delegates/how-to-combine-delegates-multicast-delegates.md).</span><span class="sxs-lookup"><span data-stu-id="85ef5-126">For information about how to create a multicast delegate, see [How to: Combine Delegates (Multicast Delegates)](~/docs/csharp/programming-guide/delegates/how-to-combine-delegates-multicast-delegates.md).</span></span>

 <span data-ttu-id="85ef5-129">Uygulama tarafından bir olay kaydedildiğinde, bu olay için temsilci çağırarak denetim olayı başlatır.</span><span class="sxs-lookup"><span data-stu-id="85ef5-129">When an event is recorded by the application, the control raises the event by invoking the delegate for that event.</span></span> <span data-ttu-id="85ef5-130">Temsilci sırayla ilişkili yöntemi çağırır.</span><span class="sxs-lookup"><span data-stu-id="85ef5-130">The delegate in turn calls the bound method.</span></span> <span data-ttu-id="85ef5-131">(Bir çok noktaya yayın temsilcisi) en yaygın durumda temsilci her ilişkili yöntem çağırma listesinde sırasıyla bir-çok bildirim sağlayan çağırır.</span><span class="sxs-lookup"><span data-stu-id="85ef5-131">In the most common case (a multicast delegate) the delegate calls each bound method in the invocation list in turn, which provides a one-to-many notification.</span></span> <span data-ttu-id="85ef5-132">Bu strateji, denetimin olay bildirimi için hedef nesnelerin listesini korumak gerekmez anlamına gelir — tüm kaydı ve bildirimi temsilci işler.</span><span class="sxs-lookup"><span data-stu-id="85ef5-132">This strategy means that the control does not need to maintain a list of target objects for event notification—the delegate handles all registration and notification.</span></span>

 <span data-ttu-id="85ef5-133">Temsilciler çok bir bildirim sağlayan aynı yönteme bağlı birden fazla olayın da etkinleştirin.</span><span class="sxs-lookup"><span data-stu-id="85ef5-133">Delegates also enable multiple events to be bound to the same method, allowing a many-to-one notification.</span></span> <span data-ttu-id="85ef5-134">Örneğin, bir düğme tıklama olayı ve bir menü komutu – tıklama olayı hem de daha sonra aynı şekilde bu ayrı olayları işlemek için tek bir yöntemi çağırır aynı temsilci çağırabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="85ef5-134">For example, a button-click event and a menu-command–click event can both invoke the same delegate, which then calls a single method to handle these separate events the same way.</span></span>

 <span data-ttu-id="85ef5-135">Temsilciler ile kullanılan bir bağlama mekanizması dinamik: bir temsilci imzası olan, olay işleyicisinin eşleşen herhangi bir yöntemi için çalışma zamanında bağlanabilir.</span><span class="sxs-lookup"><span data-stu-id="85ef5-135">The binding mechanism used with delegates is dynamic: a delegate can be bound at run time to any method whose signature matches that of the event handler.</span></span> <span data-ttu-id="85ef5-136">Bu özellik ile ayarlayın veya bağımlı bir yöntem bir koşula bağlı ve bir olay işleyicisi bir denetimi dinamik olarak eklemek için değiştirin.</span><span class="sxs-lookup"><span data-stu-id="85ef5-136">With this feature, you can set up or change the bound method depending on a condition and to dynamically attach an event handler to a control.</span></span>
