# <a name="splitter-control-windows-forms"></a>Bölümlendirici Denetimi (Windows Forms)
Windows Forms `Splitter` denetimleri çalışma zamanında yerleşik denetimleri yeniden boyutlandırmak için kullanılır. `Splitter` Denetim genellikle, veri bölmeleri farklı zamanlarda değişen genişlikler bilgisi içeren Windows Gezgini gibi, mevcut uzunluklara sahip denetimlerle birlikte formlarda kullanılır.  
  
> [!NOTE]
> , Ve önceki sürümlerin `Splitter` denetimine işlevsellik ekler ve bu, `Splitter` isterseniz hem geriye dönük uyumluluk hem de gelecekte kullanılmak üzere korunur. <xref:System.Windows.Forms.SplitContainer>
