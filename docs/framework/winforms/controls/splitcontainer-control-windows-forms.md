# <a name="splitcontainer-control-windows-forms"></a>SplitContainer Denetimi (Windows Forms)
Windows Forms `SplitContainer` denetimi bileşik olarak düşünülebilir; taşınabilir bir çubukla ayrılmış iki paneldir. Fare işaretçisi çubuğun üzerindeyken, işaretçiyi çubuğun taşınabilir olduğunu göstermek için şeklini değiştirir.  
  
> [!NOTE]
> **Araç kutusunda**bu denetim, Visual Studio 'nun <xref:System.Windows.Forms.Splitter> önceki sürümünde bulunan denetimin yerini almıştır. Denetim, denetimin <xref:System.Windows.Forms.Splitter> üzerinde çok tercih edilir. `SplitContainer` Sınıfı, var olan uygulamalarla uyumluluk için .NET Framework dahil edilmiştir, ancak yeni projeler için `SplitContainer` denetimi kullanmanızı kesinlikle öneririz. <xref:System.Windows.Forms.Splitter>  
  
 `SplitContainer` Denetim, karmaşık kullanıcı arabirimleri oluşturmanıza olanak sağlar; genellikle, bir panelde seçim diğer panelde hangi nesnelerin gösterileceğini belirler. Bu düzenleme bilgileri görüntülemek ve göz atmak için oldukça etkilidir. İki bölme olması, alanlarda bilgi toplamanıza olanak sağlar ve çubuk veya "ayırıcı", kullanıcıların panoları yeniden boyutlandırmasını kolaylaştırır.
