# <a name="how-to-add-and-remove-tabs-with-the-windows-forms-tabcontrol-using-the-designer"></a>Nasıl yapılır: Tasarımcı Kullanarak Windows Forms TabControl ile Sekme Ekleme ve Kaldırma
Formunuza bir <xref:System.Windows.Forms.TabControl> denetim yerleştirdiğinizde, varsayılan olarak iki sekme içerir. Tasarımcıyı kullanarak sekmeler ekleyebilir veya kaldırabilirsiniz.

 Aşağıdaki yordam, bir <xref:System.Windows.Forms.TabControl> denetim içeren bir form ile **Windows uygulama** projesi gerektirir. Böyle bir projeyi ayarlama hakkında daha fazla bilgi için bkz [. nasıl yapılır: Windows Forms bir uygulama projesi](/visualstudio/ide/step-1-create-a-windows-forms-application-project) oluşturun ve [şunları yapın: Windows Forms](how-to-add-controls-to-windows-forms.md)denetimleri ekleyin.

## <a name="to-add-or-remove-a-tab-using-the-designer"></a>Tasarımcıyı kullanarak sekme ekleme veya kaldırma

- Denetimin akıllı etiketinde **Sekme Ekle** veya **sekmeyi kaldır** ' a tıklayın.

     -veya-

     **Özellikler** penceresinde, **TabPage koleksiyonu düzenleyicisini**açmak için <xref:System.Windows.Forms.TabControl.TabPages%2A> özelliğinin yanındaki![üç nokta düğmesini (... Özellikler penceresi](./media/visual-studio-ellipsis-button.png)) tıklayın... **Ekle** veya **Kaldır** düğmesine tıklayın.
