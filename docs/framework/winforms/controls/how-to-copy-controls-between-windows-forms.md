# <a name="how-to-copy-controls-between-windows-forms"></a>Nasıl yapılır: Windows Formları Arasında Denetimleri Kopyalama
Bir denetimi, aynı forma, projedeki başka bir form üzerine veya diğer çözümleri kullanmak için Pano'ya kopyalanabilir.  
  
### <a name="to-copy-a-control"></a>Bir denetimi kopyalamak için  
  
1. İstediğiniz denetimi seçin ve **Düzenle** menüsünde **kopyalama**.  
  
 Bu denetim, artık söz konusu denetim türünü kabul eden herhangi bir biçimde yapıştırılabilir. Ayrıca, Denetim panoya eklendi.
