# <a name="printpreviewcontrol-control-windows-forms"></a>PrintPreviewControl Denetimi (Windows Forms)
Windows Forms `PrintPreviewControl` bir belge yazdırıldığında görünecek görüntülemek için kullanılır. Bu nedenle genellikle kullandığınız hiç düğme yok veya diğer kullanıcı arabirimi öğeleri bu denetime sahiptir `PrintPreviewControl` yalnızca kendi Yazdır-Önizle kullanıcı arabirimi yazmak istiyorsanız. Standart kullanıcı arabirimi istiyorsanız kullanmak bir <xref:System.Windows.Forms.PrintPreviewDialog> denetimi.
