# <a name="hscrollbar-and-vscrollbar-controls-windows-forms"></a>HScrollBar ve VScrollBar Denetimleri (Windows Forms)
Windows Forms kaydırma çubuğu denetimleri yatay kaydırma tarafından veya içinde bir uygulamanın veya denetimin dikey olarak uzun bir liste aracılığıyla kolay gezinti öğeleri veya büyük miktarda bilgiyi sağlamak için kullanılır. Kaydırma çubukları Windows arabiriminin ortak bir öğe var.  
  
## <a name="in-this-section"></a>Bu Bölümde  
 [HScrollBar ve VScrollBar Denetimlerine Genel Bakış](hscrollbar-and-vscrollbar-controls-overview-windows-forms.md)
