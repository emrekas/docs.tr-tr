# <a name="develop-a-composite-windows-forms-control"></a>Bileşik Windows Forms denetimi geliştirme

Temel sınıf <xref:System.Windows.Forms.UserControl>, alt denetimler için klavye yönlendirmesi sağlar ve bu sayede alt denetimlerin odağı alabilmesini sağlar. Bir kullanıcı denetimi örneği için bkz <xref:System.Windows.Forms.UserControl> [. nasıl yapılır: Windows Forms denetimlerinde](how-to-apply-attributes-in-windows-forms-controls.md)öznitelikleri uygulayın.
