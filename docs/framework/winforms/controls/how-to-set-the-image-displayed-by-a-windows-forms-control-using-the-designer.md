# <a name="how-to-set-the-image-displayed-by-a-windows-forms-control-using-the-designer"></a>Nasıl yapılır: Tasarımcıyı kullanarak Windows Forms denetimi tarafından gösterilecek resmi ayarlama

Birçok Windows Forms denetimi görüntü görüntüleyebilir. Görüntü, **Kaydet** komutunu gösteren düğmedeki bir disk simgesi gibi denetimin amacını açıklığa kavuşturan bir simge olabilir. Ya da simge istediğiniz görünümü denetlemek için bir arka plan görüntüsü olabilir.

## <a name="to-set-the-image-displayed-by-a-control"></a>Bir denetim tarafından görüntülenecek resmi ayarlamak için

1. **Özellikler** penceresinde, denetimin **görüntüsünü** veya **BackgroundImage** özelliğini seçin ve sonra![ **seçme kaynağını** göstermek için üç nokta (Visual Studio](./media/visual-studio-ellipsis-button.png)'da üç nokta düğmesi) seçin iletişim kutusu.

2. Göstermek istediğiniz görüntüyü seçin.
