# <a name="how-to-move-a-toolstrip-out-of-a-toolstripcontainer-onto-a-form"></a>Nasıl yapılır: Bir ToolStripContainer Kapsayıcısından bir ToolStrip Öğesini bir Forma Taşıma
<xref:System.Windows.Forms.ToolStrip> Bir<xref:System.Windows.Forms.ToolStripContainer> formun dışına taşımak için aşağıdaki yordamı kullanın.

## <a name="to-move-a-toolstrip-out-of-a-toolstripcontainer-onto-a-form"></a>Bir ToolStripContainer 'ın ToolStrip dışına bir form üzerine taşımak için

1. Öğesini seçin <xref:System.Windows.Forms.ToolStrip>.

2. CTRL + X tuşlarına basarak kesmeyi kesin veya sağ <xref:System.Windows.Forms.ToolStrip> tıklayıp bağlam menüsünden **Kes** ' i seçin. <xref:System.Windows.Forms.ToolStrip>

3. Formu seçin.

4. CTRL + V tuşlarına basarak yapıştırın veya **Düzenle** menüsünden Yapıştır ' ı seçin. <xref:System.Windows.Forms.ToolStrip>

5. Öğesinin özelliğini **en üst**olarak ayarlayın. <xref:System.Windows.Forms.ToolStrip> <xref:System.Windows.Forms.ToolStrip.Dock%2A>
