# <a name="etw-events-in-the-net-framework"></a>.NET Framework'te ETW Olayları
Olay izleme için Windows (ETW), Windows işletim sistemleri tarafından sağlanan yüksek performanslı, düşük ek yük, ölçeklenebilir bir izleme sistemidir. .NET Framework tarafından sağlanan profil oluşturma ve hata ayıklama desteğini tamamlayan ve çeşitli senaryoları sorunlarını gidermek için kullanılabilir.  
  
 .NET Framework'te ETW olay izleme (CLR), ortak dil çalışma için kullanılabilir [görev paralel Kitaplığı](../../../docs/standard/parallel-programming/task-parallel-library-tpl.md), ve [paralel LINQ (PLINQ)](../../../docs/standard/parallel-programming/parallel-linq-plinq.md).

 [Ortak Dil Çalışma Zamanı Modülünde ETW Olayları](../../../docs/framework/performance/etw-events-in-the-common-language-runtime.md)
