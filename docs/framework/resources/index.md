## <a name="packaging-and-deploying-resources"></a><span data-ttu-id="78293-123">Paketleme ve Dağıtma Kaynakları</span><span class="sxs-lookup"><span data-stu-id="78293-123">Packaging and Deploying Resources</span></span>

 <span data-ttu-id="78293-132">Daha fazla bilgi için [kaynakları paketleme ve dağıtma](../../../docs/framework/resources/packaging-and-deploying-resources-in-desktop-apps.md) makalesi.</span><span class="sxs-lookup"><span data-stu-id="78293-132">For more information, see the [Packaging and Deploying Resources](../../../docs/framework/resources/packaging-and-deploying-resources-in-desktop-apps.md) article.</span></span>  
  
## <a name="retrieving-resources"></a><span data-ttu-id="78293-133">Kaynakları Alma</span><span class="sxs-lookup"><span data-stu-id="78293-133">Retrieving Resources</span></span>  
 <span data-ttu-id="78293-134">Çalışma zamanında bir uygulama, <xref:System.Globalization.CultureInfo.CurrentUICulture%2A?displayProperty=nameWithType> özelliği tarafından belirtilen kültürü temel alan, iş parçacığını düzeyinde uygun yerelleştirilmiş kaynakları yükler.</span><span class="sxs-lookup"><span data-stu-id="78293-134">At run time, an app loads the appropriate localized resources on a per-thread basis, based on the culture specified by the <xref:System.Globalization.CultureInfo.CurrentUICulture%2A?displayProperty=nameWithType> property.</span></span> <span data-ttu-id="78293-135">Bu özellik değeri aşağıdaki şekilde türetilir:</span><span class="sxs-lookup"><span data-stu-id="78293-135">This property value is derived as follows:</span></span>  
  
-   <span data-ttu-id="78293-136">Yerelleştirilmiş kültürü temsil eden bir <xref:System.Globalization.CultureInfo> nesnesini doğrudan <xref:System.Threading.Thread.CurrentUICulture%2A?displayProperty=nameWithType> özelliğine atayarak.</span><span class="sxs-lookup"><span data-stu-id="78293-136">By directly assigning a <xref:System.Globalization.CultureInfo> object that represents the localized culture to the <xref:System.Threading.Thread.CurrentUICulture%2A?displayProperty=nameWithType> property.</span></span>  
  
-   <span data-ttu-id="78293-137">Eğer bir kültür açıkça atanmamışsa, <xref:System.Globalization.CultureInfo.DefaultThreadCurrentUICulture%2A?displayProperty=nameWithType> özelliğinden varsayılan iş parçacığı UI kültürünü alarak.</span><span class="sxs-lookup"><span data-stu-id="78293-137">If a culture is not explicitly assigned, by retrieving the default thread UI culture from the <xref:System.Globalization.CultureInfo.DefaultThreadCurrentUICulture%2A?displayProperty=nameWithType> property.</span></span>  
  
-   <span data-ttu-id="78293-138">Varsayılan iş parçacığı UI kültürü açıkça, yerel bilgisayardaki geçerli kullanıcının kültürünü alarak atanmamışsa.</span><span class="sxs-lookup"><span data-stu-id="78293-138">If a default thread UI culture is not explicitly assigned, by retrieving the culture for the current user on the local computer.</span></span> <span data-ttu-id="78293-139">Windows üzerinde çalışan .NET uygulamalarını bunu Windows çağırarak [ `GetUserDefaultUILanguage` ](/windows/desktop/api/winnls/nf-winnls-getuserdefaultuilanguage) işlevi.</span><span class="sxs-lookup"><span data-stu-id="78293-139">.NET implementations running on Windows do this  by calling the Windows [`GetUserDefaultUILanguage`](/windows/desktop/api/winnls/nf-winnls-getuserdefaultuilanguage) function.</span></span>  
  
 <span data-ttu-id="78293-140">Geçerli UI kültürünün nasıl ayarlandığı hakkında daha fazla bilgi için <xref:System.Globalization.CultureInfo> ve <xref:System.Globalization.CultureInfo.CurrentUICulture%2A?displayProperty=nameWithType> başvuru sayfalarına bakın.</span><span class="sxs-lookup"><span data-stu-id="78293-140">For more information about how the current UI culture is set, see the <xref:System.Globalization.CultureInfo> and <xref:System.Globalization.CultureInfo.CurrentUICulture%2A?displayProperty=nameWithType> reference pages.</span></span>  
  
 <span data-ttu-id="78293-141">Ardından, <xref:System.Resources.ResourceManager?displayProperty=nameWithType> sınıfını kullanarak geçerli UI kültürünün veya belirli bir kültürün kaynaklarını alabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="78293-141">You can then retrieve resources for the current UI culture or for a specific culture by using the <xref:System.Resources.ResourceManager?displayProperty=nameWithType> class.</span></span> <span data-ttu-id="78293-142">Ancak <xref:System.Resources.ResourceManager> sınıfı, kaynakları almak için en yaygın olarak kullanılır <xref:System.Resources?displayProperty=nameWithType> ad alanı, kaynakları almak için kullanabileceğiniz ek türleri içerir.</span><span class="sxs-lookup"><span data-stu-id="78293-142">Although the <xref:System.Resources.ResourceManager> class is most commonly used for retrieving resources, the <xref:System.Resources?displayProperty=nameWithType> namespace contains additional types that you can use to retrieve resources.</span></span> <span data-ttu-id="78293-143">Bu güncelleştirmeler şunlardır:</span><span class="sxs-lookup"><span data-stu-id="78293-143">These include:</span></span>  
  
-   <span data-ttu-id="78293-144">Bir derlemede gömülü olan veya bir tek başına ikili .resources dosyasında depolanan kaynakları numaralandırmanızı sağlayan <xref:System.Resources.ResourceReader> sınıfı.</span><span class="sxs-lookup"><span data-stu-id="78293-144">The <xref:System.Resources.ResourceReader> class, which enables you to enumerate resources embedded in an assembly or stored in a standalone binary .resources file.</span></span> <span data-ttu-id="78293-145">Çalışma zamanında kullanılabilir olan kaynakların tam adını bilmediğinizde bu yararlı olur.</span><span class="sxs-lookup"><span data-stu-id="78293-145">It is useful when you don't know the precise names of the resources that are available at run time.</span></span>  
  
-   <span data-ttu-id="78293-146">Kaynakları bir XML (.resx) dosyasından almanızı sağlayan <xref:System.Resources.ResXResourceReader> sınıfı.</span><span class="sxs-lookup"><span data-stu-id="78293-146">The <xref:System.Resources.ResXResourceReader> class, which enables you to retrieve resources from an XML (.resx) file.</span></span>  
  
-   <span data-ttu-id="78293-147">Geri dönüş kurallarını gözlemlemeden belirli bir kültüre ait kaynakları almanızı sağlayan <xref:System.Resources.ResourceSet> sınıfı.</span><span class="sxs-lookup"><span data-stu-id="78293-147">The <xref:System.Resources.ResourceSet> class, which enables you to retrieve the resources of a specific culture without observing fallback rules.</span></span> <span data-ttu-id="78293-148">Kaynaklar bir derlemede veya bir tek başına ikili .resources dosyasında depolanabilir.</span><span class="sxs-lookup"><span data-stu-id="78293-148">The resources can be stored in an assembly or a standalone binary .resources file.</span></span> <span data-ttu-id="78293-149">Kaynakları başka bir kaynaktan almak için aynı zamanda <xref:System.Resources.IResourceReader> sınıfını kullanmanızı sağlayan bir <xref:System.Resources.ResourceSet> uygulaması geliştirebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="78293-149">You can also develop an <xref:System.Resources.IResourceReader> implementation that enables you to use the <xref:System.Resources.ResourceSet> class to retrieve resources from some other source.</span></span>
