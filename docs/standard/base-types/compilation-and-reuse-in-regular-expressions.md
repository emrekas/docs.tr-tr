## <a name="the-regular-expressions-cache"></a><span data-ttu-id="ffa35-119">Normal ifadeler önbellek</span><span class="sxs-lookup"><span data-stu-id="ffa35-119">The Regular Expressions Cache</span></span>  
 <span data-ttu-id="ffa35-120">Performansı artırmak için bir uygulama önbellek derlenmiş normal ifadeler, normal ifade altyapısı tutar.</span><span class="sxs-lookup"><span data-stu-id="ffa35-120">To improve performance, the regular expression engine maintains an application-wide cache of compiled regular expressions.</span></span> <span data-ttu-id="ffa35-121">Önbellek, yalnızca statik yöntem çağrılarında kullanılan normal ifade desenleri depolar.</span><span class="sxs-lookup"><span data-stu-id="ffa35-121">The cache stores regular expression patterns that are used only in static method calls.</span></span> <span data-ttu-id="ffa35-122">(Örnek yöntemleri için sağlanan normal ifade deseni önbelleğe alınmaz.)  <span data-ttu-id="ffa35-123">Önbelleğe alınan normal ifadelerin sayısı değeri tarafından belirlenir `static` (`Shared` Visual Basic'te) <xref:System.Text.RegularExpressions.Regex.CacheSize%2A?displayProperty=nameWithType> özelliği.</span><span class="sxs-lookup"><span data-stu-id="ffa35-123">The maximum number of cached regular expressions is determined by the value of the `static` (`Shared` in Visual Basic) <xref:System.Text.RegularExpressions.Regex.CacheSize%2A?displayProperty=nameWithType> property.</span></span> <span data-ttu-id="ffa35-124">Varsayılan olarak, en fazla 15 derlenmiş normal ifadeler normal ifade altyapısının önbelleğe alır.</span><span class="sxs-lookup"><span data-stu-id="ffa35-124">By default, the regular expression engine caches up to 15 compiled regular expressions.</span></span> <span data-ttu-id="ffa35-125">Derlenmiş normal ifadeler önbellek boyutunu aşıyor, yakın zamanda kullanılmamış normal ifade atılır ve yeni bir normal ifade önbelleğe alınır.</span><span class="sxs-lookup"><span data-stu-id="ffa35-125">If the number of compiled regular expressions exceeds the cache size, the least recently used regular expression is discarded and the new regular expression is cached.</span></span>

- <span data-ttu-id="ffa35-127">Statik bir yöntemi kullanarak <xref:System.Text.RegularExpressions.Regex> normal ifade tanımlamak için nesne.</span><span class="sxs-lookup"><span data-stu-id="ffa35-127">By using a static method of the <xref:System.Text.RegularExpressions.Regex> object to define the regular expression.</span></span> <span data-ttu-id="ffa35-128">Başka bir statik yöntem çağrısında zaten tanımlanmış bir normal ifade deseni kullanıyorsanız, normal ifade altyapısı önbellekten alır.</span><span class="sxs-lookup"><span data-stu-id="ffa35-128">If you are using a regular expression pattern that has already been defined in another static method call, the regular expression engine will retrieve it from the cache.</span></span> <span data-ttu-id="ffa35-129">Aksi halde altyapısı normal ifadeyi derlemek ve önbelleğe ekleyin.</span><span class="sxs-lookup"><span data-stu-id="ffa35-129">If not, the engine will compile the regular expression and add it to the cache.</span></span>  
  
- <span data-ttu-id="ffa35-130">Mevcut bir yeniden tarafından <xref:System.Text.RegularExpressions.Regex> normal ifade desenine gerekli olduğu sürece nesne.</span><span class="sxs-lookup"><span data-stu-id="ffa35-130">By reusing an existing <xref:System.Text.RegularExpressions.Regex> object as long as its regular expression pattern is needed.</span></span>
