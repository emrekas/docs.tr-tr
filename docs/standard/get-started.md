# <a name="get-started"></a>Başlarken

.NET ile çalışmaya başlamanın birkaç yolu vardır. .NET çok büyük bir platform olduğundan, bu belgede, her biri farklı bir perspektiften .NET ile nasıl başlatılacağınız hakkında daha fazla makale vardır.

## <a name="get-started-using-net-languages"></a>.NET dillerini kullanmaya başlama

* Başlarken [ C# makaleleri ve](../csharp/getting-started/index.md) [ C# öğreticiler](../csharp/tutorials/index.md) , merkezli bir C#şekilde başlamak için çeşitli yollar sağlar.

* Başlarken öğreticileri, Visual Studio, Visual Studio Code veya komut satırı araçlarıyla F#kullanabileceğiniz üç temel yol sağlar. [ F# ](../fsharp/tutorials/getting-started/index.md)

* [Visual Basic](../visual-basic/getting-started/index.md) Başlarken makaleleri, Visual Studio 'da Visual Basic kullanmaya yönelik kılavuzlar sağlar.

## <a name="get-started-using-net-core"></a>.NET Core kullanmaya başlama

* [.NET Core Ile çalışmaya başlamak](../core/get-started.md) , farklı işletim sistemlerinde .NET Core ile çalışmaya başlama ve farklı araçları kullanma hakkında genel bir bakış sunar.

* [.NET Core öğreticileri](../core/tutorials/index.md) , işletim sisteminizi ve tercih ettiğiniz araçları kullanarak .NET Core ile çalışmaya başlamanıza kadar çeşitli yollar ayrıntılandırır.
