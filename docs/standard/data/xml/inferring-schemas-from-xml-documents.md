## <a name="the-schema-inference-process"></a><span data-ttu-id="e3868-104">Şema çıkarımı işlemi</span><span class="sxs-lookup"><span data-stu-id="e3868-104">The Schema Inference Process</span></span>  
 <span data-ttu-id="e3868-105"><xref:System.Xml.Schema.XmlSchemaInference> Sınıfının <xref:System.Xml.Schema?displayProperty=nameWithType> ad alanı, bir veya daha fazla XML Şeması Tanım Dili (XSD) şemaları yapısının bir XML belgesi oluşturmak için kullanılır.</span><span class="sxs-lookup"><span data-stu-id="e3868-105">The <xref:System.Xml.Schema.XmlSchemaInference> class of the <xref:System.Xml.Schema?displayProperty=nameWithType> namespace is used to generate one or more XML Schema definition language (XSD) schemas from the structure of an XML document.</span></span> <span data-ttu-id="e3868-106">Oluşturulan şemaları, orijinal XML belgeyi doğrulamak için kullanılabilir.</span><span class="sxs-lookup"><span data-stu-id="e3868-106">The generated schemas may be used to validate the original XML document.</span></span>  
  
 <span data-ttu-id="e3868-107">Bir XML belgesi tarafından işlenen <xref:System.Xml.Schema.XmlSchemaInference> sınıfı <xref:System.Xml.Schema.XmlSchemaInference> sınıfı tanımlayan öğeler ve öznitelikler XML belgesi şema bileşenleri hakkında varsayımlar hale getirir.</span><span class="sxs-lookup"><span data-stu-id="e3868-107">As an XML document is processed by the <xref:System.Xml.Schema.XmlSchemaInference> class, the <xref:System.Xml.Schema.XmlSchemaInference> class makes assumptions about the schema components that describe the elements and attributes in the XML document.</span></span> <span data-ttu-id="e3868-108"><xref:System.Xml.Schema.XmlSchemaInference> Sınıfı da çıkarsar şema bileşenleri kısıtlanmış yolla en kısıtlayıcı bir türde bir belirli öğesi veya özniteliği için'çıkarımını yapma.</span><span class="sxs-lookup"><span data-stu-id="e3868-108">The <xref:System.Xml.Schema.XmlSchemaInference> class also infers schema components in a constrained way by inferring the most restrictive type for a particular element or attribute.</span></span> <span data-ttu-id="e3868-109">XML belgesi hakkında daha fazla bilgi toplandıktan gibi bu kısıtlamalar, daha az kısıtlayıcı türlerin çıkarımını yapma tarafından gevşetiliyormuş.</span><span class="sxs-lookup"><span data-stu-id="e3868-109">As more information about the XML document is gathered, these constraints are loosened by inferring less restrictive types.</span></span> <span data-ttu-id="e3868-110">Çıkarımı yapılan en az kısıtlayıcı türü `xs:string`.</span><span class="sxs-lookup"><span data-stu-id="e3868-110">The least restrictive type that can be inferred is `xs:string`.</span></span>  
  
 <span data-ttu-id="e3868-111">Örneğin, bir XML belgesi aşağıdaki parçası'ı alın.</span><span class="sxs-lookup"><span data-stu-id="e3868-111">Take, for example, the following piece of an XML document.</span></span>  
  
```xml

    <child>Two</child>  
</parent>  
<parent attribute1="A">  
```  
  
 <span data-ttu-id="e3868-112">Yukarıdaki örnekte, zaman içinde `attribute1` öznitelik değeri ile karşılaşıldığında `6` tarafından <xref:System.Xml.Schema.XmlSchemaInference> işlemi varsayılır türünde olmasını `xs:unsignedByte`.</span><span class="sxs-lookup"><span data-stu-id="e3868-112">In the example above, when the `attribute1` attribute is encountered with a value of `6` by the <xref:System.Xml.Schema.XmlSchemaInference> process, it is assumed to be of type `xs:unsignedByte`.</span></span> <span data-ttu-id="e3868-113">Zaman ikinci `parent` öğesi tarafından karşılaştı <xref:System.Xml.Schema.XmlSchemaInference> sürecini kısıtlaması izin vermek için gevşekleştirildiğini türüne değiştirerek `xs:string` çünkü değerini `attribute1` özniteliktir artık `A`.</span><span class="sxs-lookup"><span data-stu-id="e3868-113">When the second `parent` element is encountered by the <xref:System.Xml.Schema.XmlSchemaInference> process, the constraint is loosened by modifying the type to `xs:string` because the value of the `attribute1` attribute is now `A`.</span></span> <span data-ttu-id="e3868-114">Benzer şekilde, `minOccurs` özniteliği için tüm `child` şemada çıkarılan öğeleri izin vermek için gevşekleştirildiğini için `minOccurs="0"` ikinci üst öğenin alt öğeleri olduğundan.</span><span class="sxs-lookup"><span data-stu-id="e3868-114">Similarly, the `minOccurs` attribute for all the `child` elements inferred in the schema are loosened to `minOccurs="0"` because the second parent element has no child elements.</span></span>
