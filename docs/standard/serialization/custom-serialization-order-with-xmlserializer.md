# <a name="custom-serialization-order-with-xmlserializer"></a>XmlSerializer ile Özel Serileştirme Düzeni
[Örneği indirin](https://download.microsoft.com/download/4/7/B/47B2164C-E780-4B10-8DE4-2CB5B886E0A6/Technologies/Serialization/Xml%20Serialization/CustomOrder.zip.exe)  
  
 Bu örnek serileştirilmiş ve seri durumdan çıkarılmış öğelerin sırasını XML Serileştirmenin denetlemek nasıl gösterir.  
  
 Serileştirme hakkında daha fazla bilgi için kaynak kodu ve build.proj dosyalarını yorumlarda inceleyin.  
  
### <a name="to-build-the-sample-using-the-command-prompt"></a>Komut istemi kullanarak örneği oluşturmak için  
  
1. Komut istemi açın ve örnek için dile özgü alt birine gidin.  
  
2. Tür **msbuild CustomOrder.sln** komut satırına.  
  
### <a name="to-build-the-sample-using-visual-studio"></a>Visual Studio kullanarak örneği oluşturmak için  
  
1. Açık [!INCLUDE[fileExplorer](../../../includes/fileexplorer-md.md)] ve örnek için dile özgü alt birine gidin.  
  
2. Visual Studio'da dosyayı açmak CustomOrder.sln simgesini çift tıklatın.  
  
3. İçinde **derleme** menüsünde **Çözümü Derle**.  
  
4. Uygulama örneği varsayılan \bin veya \bin\debug'dır alt üzerine inşa edilmiştir.
