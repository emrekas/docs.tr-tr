### <a name="working-with-the-orchestration-client"></a><span data-ttu-id="7aa99-124">Düzenleme istemcisi ile çalışma</span><span class="sxs-lookup"><span data-stu-id="7aa99-124">Working with the Orchestration client</span></span>

<span data-ttu-id="7aa99-125">Azure işlevleri'nde tetiklenen işlemlerinin başka stilleri karşılaştırıldığında düzenlemeleri benzersizdir.</span><span class="sxs-lookup"><span data-stu-id="7aa99-125">Orchestrations are unique when compared to other styles of triggered operations in Azure Functions.</span></span> <span data-ttu-id="7aa99-126">Dayanıklı işlevler ele saat veya gün tamamlamak için bile işlevlerin yürütülmesini sağlar.</span><span class="sxs-lookup"><span data-stu-id="7aa99-126">Durable Functions enables the execution of functions that may take hours or even days to complete.</span></span> <span data-ttu-id="7aa99-127">Bu türden bir davranış, çalışan bir düzenleme durumunu denetlemek izinlere gerek ile sıd'lerde sonlandırmak veya dış olaylar bildirim göndermek gelir.</span><span class="sxs-lookup"><span data-stu-id="7aa99-127">That type of behavior comes with the need to able to check the status of a running orchestration, preemptively terminate, or send notifications of external events.</span></span>

<span data-ttu-id="7aa99-128">Böyle durumlarda, dayanıklı işlevler uzantısını sağlar `DurableOrchestrationClient` ile etkileşimde bulunmanızı sağlar sınıfını işlevleri düzenlenir.</span><span class="sxs-lookup"><span data-stu-id="7aa99-128">For such cases, the Durable Functions extension provides the `DurableOrchestrationClient` class that allows you to interact with orchestrated functions.</span></span> <span data-ttu-id="7aa99-129">Kullanarak düzenleme istemcisi için size erişim `OrchestrationClientAttribute` bağlama.</span><span class="sxs-lookup"><span data-stu-id="7aa99-129">You get access to the orchestration client by using the `OrchestrationClientAttribute` binding.</span></span> <span data-ttu-id="7aa99-130">Genellikle, bu öznitelik başka bir tetikleyici türü ile gibi dahil bir `HttpTrigger` veya `ServiceBusTrigger`.</span><span class="sxs-lookup"><span data-stu-id="7aa99-130">Generally, you would include this attribute with another trigger type, such as an `HttpTrigger` or `ServiceBusTrigger`.</span></span> <span data-ttu-id="7aa99-131">Kaynak işlevi tetiklendikten sonra düzenleme istemcisi bir düzenleyici işlevi başlatmak için kullanılabilir.</span><span class="sxs-lookup"><span data-stu-id="7aa99-131">Once the source function has been triggered, the orchestration client can be used to start an orchestrator function.</span></span>

```csharp
[FunctionName("KickOff")]
public static async Task<HttpResponseMessage> Run(
    [HttpTrigger(AuthorizationLevel.Function, "POST")]HttpRequestMessage req,
    [OrchestrationClient ] DurableOrchestrationClient<orchestrationClient>)
{
    OrderRequestData data = await req.Content.ReadAsAsync<OrderRequestData>();

    string instanceId = await orchestrationClient.StartNewAsync("PlaceOrder", data);

    return orchestrationClient.CreateCheckStatusResponse(req, instanceId);
}
```

> <span data-ttu-id="7aa99-140">*Hata işleme kodunu konuyu uzatmamak amacıyla bırakıldı '*</span><span class="sxs-lookup"><span data-stu-id="7aa99-140">*Error handling code has been left out for brevity*</span></span>

```csharp
[FunctionName("PlaceOrder")]
public static async Task<string> PlaceOrder([OrchestrationTrigger] DurableOrchestrationContext context)
{
    OrderRequestData orderData = context.GetInput<OrderRequestData>();

    await context.CallActivityAsync<bool>("CheckAndReserveInventory", orderData);
    await context.CallActivityAsync<string>("ProcessPayment", orderData);

    string trackingNumber = await context.CallActivityAsync<string>("ScheduleShipping", orderData);
    await context.CallActivityAsync<string>("EmailCustomer", trackingNumber);

    return trackingNumber;
}
```
