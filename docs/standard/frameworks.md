## <a name="how-to-specify-target-frameworks"></a><span data-ttu-id="91d0d-195">Hedef Çerçeve belirtme</span><span class="sxs-lookup"><span data-stu-id="91d0d-195">How to specify target frameworks</span></span>

<span data-ttu-id="91d0d-196">Hedef Çerçeve proje dosyanızda belirtilir.</span><span class="sxs-lookup"><span data-stu-id="91d0d-196">Target frameworks are specified in your project file.</span></span> <span data-ttu-id="91d0d-197">Tek hedef Framework'ü belirtildiğinde kullanın **TargetFramework** öğesi.</span><span class="sxs-lookup"><span data-stu-id="91d0d-197">When a single target framework is specified, use the **TargetFramework** element.</span></span> <span data-ttu-id="91d0d-198">Aşağıdaki konsol uygulama projesi dosyası, .NET Core 2.2 hedef gösterilmektedir:</span><span class="sxs-lookup"><span data-stu-id="91d0d-198">The following console app project file demonstrates how to target .NET Core 2.2:</span></span>

```xml
<Project Sdk="Microsoft.NET.Sdk">

  <PropertyGroup>
    <OutputType>Exe</OutputType>
    <TargetFramework>netcoreapp2.2</TargetFramework>
  </PropertyGroup>

</Project>
```

<span data-ttu-id="91d0d-199">Birden çok hedef çerçeve belirttiğinizde, her hedef çerçeve için derlemeleri koşullu başvurabilir.</span><span class="sxs-lookup"><span data-stu-id="91d0d-199">When you specify multiple target frameworks, you may conditionally reference assemblies for each target framework.</span></span> <span data-ttu-id="91d0d-200">Kodunuzda, koşullu olarak karşı bu derlemeler ile önişlemci sembolleri kullanarak derleyebilirsiniz *if-then-else* mantığı.</span><span class="sxs-lookup"><span data-stu-id="91d0d-200">In your code, you can conditionally compile against those assemblies by using preprocessor symbols with *if-then-else* logic.</span></span>

<span data-ttu-id="91d0d-201">Aşağıdaki kitaplık projesi dosyası .NET Standard API'leri, hedefler (`netstandard1.4`) ve API'leri .NET Framework'ün (`net40` ve `net45`).</span><span class="sxs-lookup"><span data-stu-id="91d0d-201">The following library project file targets APIs of .NET Standard (`netstandard1.4`) and APIs of the .NET Framework (`net40` and `net45`).</span></span> <span data-ttu-id="91d0d-202">Çoğul kullanın **TargetFrameworks** birden fazla hedef Framework'e sahip öğe.</span><span class="sxs-lookup"><span data-stu-id="91d0d-202">Use the plural **TargetFrameworks** element with multiple target frameworks.</span></span> <span data-ttu-id="91d0d-203">Not nasıl `Condition` öznitelikler, kitaplık için iki .NET Framework Tfm'ler derlendiğinde uygulama paketleri içerir:</span><span class="sxs-lookup"><span data-stu-id="91d0d-203">Note how the `Condition` attributes include implementation-specific packages when the library is compiled for the two .NET Framework TFMs:</span></span>

```xml
<Project Sdk="Microsoft.NET.Sdk">

  <PropertyGroup>
    <TargetFrameworks>netstandard1.4;net40;net45</TargetFrameworks>
  </PropertyGroup>

  <!-- Conditionally obtain references for the .NET Framework 4.0 target -->
  <ItemGroup Condition=" '$(TargetFramework)' == 'net40' ">
    <Reference Include="System.Net" />
  </ItemGroup>

  <!-- Conditionally obtain references for the .NET Framework 4.5 target -->
  <ItemGroup Condition=" '$(TargetFramework)' == 'net45' ">
    <Reference Include="System.Net.Http" />
    <Reference Include="System.Threading.Tasks" />
  </ItemGroup>

</Project>
```

<span data-ttu-id="91d0d-204">Kitaplık veya uygulama içinde her hedef çerçeve için derlemek için koşullu bir kod yazın:</span><span class="sxs-lookup"><span data-stu-id="91d0d-204">Within your library or app, you write conditional code to compile for each target framework:</span></span>

```csharp
public class MyClass
{
    static void Main()
    {
#if NET40
        Console.WriteLine("Target framework: .NET Framework 4.0");
#elif NET45  
        Console.WriteLine("Target framework: .NET Framework 4.5");
#else
        Console.WriteLine("Target framework: .NET Standard 1.4");
#endif
    }
```
