## <a name="syntax"></a><span data-ttu-id="c35d6-105">Sözdizimi</span><span class="sxs-lookup"><span data-stu-id="c35d6-105">Syntax</span></span>

```fsharp

| pattern1 [ when condition ] -> result-expression1
```
