# <a name="structures"></a><span data-ttu-id="6a057-103">Yapılar</span><span class="sxs-lookup"><span data-stu-id="6a057-103">Structures</span></span>

<span data-ttu-id="6a057-104">A *yapısı* küçük miktarda veri ve basit davranışı sahip türleri bir sınıf daha verimli olabilir compact nesne türüdür.</span><span class="sxs-lookup"><span data-stu-id="6a057-104">A *structure* is a compact object type that can be more efficient than a class for types that have a small amount of data and simple behavior.</span></span>

## <a name="syntax"></a><span data-ttu-id="6a057-105">Sözdizimi</span><span class="sxs-lookup"><span data-stu-id="6a057-105">Syntax</span></span>

```fsharp
[ attributes ]
type [accessibility-modifier] type-name =
    struct
        type-definition-elements-and-members
    end
// or
[ attributes ]
[<StructAttribute>]
type [accessibility-modifier] type-name =
    type-definition-elements-and-members
```
